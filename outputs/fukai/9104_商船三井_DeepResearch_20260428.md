## 4. REIT設立計画 詳細分析（最重要セクション）

### 背景：田村新体制とElliottの要求

2026年4月1日に新CEOに就任した田村城太郎氏（57歳・MOL生え抜き）のもと、BLUE ACTION 2035フェーズ2が始動。Elliott Investment Management（運用総額約$79.8B）が2026年3月18日に商船三井の「significant stake（重大株式）」取得を発表し、以下を要求：

1. **株式自社買戻し：3年間で約3,000億円（$20億）**
2. **不動産ポートフォリオの見直し**：ダイビルの再上場・REIT化
3. **Daibiru（8806、2022年に完全子会社化・非上場化）の資本解放**
4. **船舶・不動産の含み益（合計数千億円規模）の顕在化**

### ダイビル（Daibiru Corp.）資産分析

| 項目 | 内容 |
|------|------|
| 設立 | 1923年（大阪）。旧・大阪建物株式会社 |
| MOLへの関係 | 2022年にMOLが完全TOBで非上場化。MOL100%子会社 |
| 主要資産 | 東京・大阪中心地のオフィスビル（大手町、淀屋橋等）。シドニー「135 King Street」タワー、ロンドン「Capital House」等海外も保有 |
| 簿価（推計） | 含み益考慮前：約1兆円前後の不動産総資産（推計） |
| 含み益 | GoldmanSachs試算：約6,000億円（国内不動産のみ） |
| 外部公表値 | ダイビル自身が1株3,000円（推計市場価格）を参考値として開示（TOB時のAVI指摘より） |

### REIT化による株主価値向上メカニズム

1. **NAV顕在化**：6,000億円超の含み益が連結BPS（¥7,870）に未反映。REIT化でこの含み益が顕在化されれば修正BPS：推計¥9,500〜¥10,500/株
2. **PBR修正**：修正BPS¥9,500ベース×PBR1.0倍→理論株価¥9,500（現値比+63%）
3. **安定CF創出**：不動産REIT化で毎期安定した配当収入・スポンサーフィーを獲得→海運の変動CFを補完
4. **資本効率向上**：非コアの不動産資産をREIT（外部資金調達）で拡大しつつMOL本体のROEを改善
