### 2.1 スタグフレーション耐性の数理的裏付け

#### スタグフレーションとは何か（深井定義）
スタグフレーション＝**物価上昇（コストプッシュ）＋景気停滞（需要減）**の同時発生。
企業分析上の問題は：「原材料コストが上がる中、消費者への価格転嫁が可能か、かつ需要が維持されるか」。

#### キリンHDの5つの耐性メカニズム

**① 必需品カテゴリー効果（Price Inelasticity）**

アルコール飲料・機能性飲料・医薬品は景気感応度が低い。
需要の価格弾力性（|ε|）を推計すると：

```
国内酒類（ビール等）   |ε| ≈ 0.3〜0.5  （非常に低弾力）
医薬品（協和キリン）   |ε| ≈ 0.1〜0.2  （保険適用で需要安定）
機能性飲料（iMUSE等）  |ε| ≈ 0.6〜0.8  （健康意識高まりで耐性）
→ 加重平均 |ε| ≈ 0.35〜0.50（景気後退時も需要急落しない）
```

消費財平均の|ε|≈0.8〜1.2と比較して、キリンHDの事業ポートフォリオは**構造的に景気変動に鈍感**。

**② 価格転嫁力の実績検証**

```
FY2022: 原材料費+15%上昇 → 製品値上げ実施 → 売上高増 + 営業利益維持
FY2023: 追加値上げ（麦芽・ホップ・アルミ缶コスト対応）→ 消費者受容確認
FY2024: プレミアム製品比率上昇 → 売上単価+3〜5%改善
FY2025E: 価格転嫁効果フル寄与 → マージン回復局面
```

**数値化：価格転嫁率（Pass-Through Rate）**

```
コスト増加分を価格に転嫁できた割合：
= （価格改定による売上増加）/（原材料コスト増加額）
推計：70〜80%（業界平均50〜60%を大幅上回る）
→ ブランド力とシェアの高さの証左
```

**③ 円安メリットの構造（海外セグメント）**

```
円安感応度試算：
豪ドル（AUD）円換算：
  AUD/JPY が1円円安 → 年間約40〜50億円の営業利益増加（推計）
  
海外売上高比率：約30%
→ 円安15%進行（115円→132円想定）で：
  営業利益押し上げ効果 ≈ +120〜180億円/年（推計）

スタグフレ局面では通常円安→海外利益の円換算が自動ヘッジ機能として機能
```

**④ 原油高の間接影響と対応力**

WTI原油：112.97 USD/bbl（直近）、ブレント：111.11 USD/bblという高水準。
キリンHDへの影響経路と対応：

```
【影響経路】
原油高 → 包材（PET・アルミ）コスト上昇 → COGS押し上げ
原油高 → 輸送・物流コスト上昇 → SGA押し上げ
原油高 → 円安促進 → 海外利益の円換算増

【定量試算】
原材料・包材コストへの原油感応度：売上高の約3〜4%がエネルギー関連コスト
WTI $100→$113（+13%）の影響：COGS増加 ≈ 0.4〜0.5%の営業利益率押し下げ
一方、円安効果（海外利益増）：+0.6〜0.8%の押し上げ
→ ネット：わずかにプラス（円安メリットがコストデメリットを上回る）

【結論】原油高はキリンHDにとってネットニュートラル〜わずかにポジティブ
```

**⑤ 医薬セグメントの景気無相関性（最強の耐性）**

```
協和キリン（KHK）の主要製品：
- クリースビータ（FGF23関連疾患）：患者数が景気に無関係
- フェソロデックス（乳がん）：治療継続が最優先
- 新規パイプライン（腎臓病・がん）：保険適用→需要安定

医薬セグメントは景気後退時でも売上0%成長〜プラス成長
→ キリンHDのポートフォリオの「アンカー」として機能
```

## 4. DCFバリュエーション

### 4.1 WACC計算

#### Step 1：リスクフリーレート
```
Rf = 2.380%（日本10年国債、2026-04-06 09:28 JST、社長確認値）
```

#### Step 2：β（ベータ）
```
キリンHD（2503）の推計β：
  食品・飲料ディフェンシブ + 医薬 + 海外事業の複合
  比較対象：アサヒGHD β≈0.80、味の素 β≈0.65、協和キリン β≈0.70
  
  深井推計β：0.75
  （ディフェンシブ性と海外エクスポージャーのバランスで0.8より低め）
```

#### Step 3：ERP（エクイティリスクプレミアム）
```
日本株ERP：5.5%（Damodaran 2026年推計ベース、日本市場標準値）
```

#### Step 4：株主資本コスト（Re）
```
Re = Rf + β × ERP
   = 2.380% + 0.75 × 5.5%
   = 2.380% + 4.125%
   = 6.505%
```

#### Step 5：負債コスト（Rd）
```
Rd（税引前）：1.5%（日本大企業の社債レート水準）
実効税率（Tc）：30%（日本法人税実効税率）
Rd × (1-Tc) = 1.5% × 0.70 = 1.05%
```

#### Step 6：資本構成
```
時価総額（E）：約¥1兆9,558億（¥2,540 × 770M株）
有利子負債（D）：約¥8,000億
E + D ≈ ¥2兆7,558億

E/(D+E) = ¥1兆9,558億 / ¥2兆7,558億 ≈ 70.97%
D/(D+E) = ¥8,000億 / ¥2兆7,558億 ≈ 29.03%
```

#### Step 7：WACC計算
```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.7097 × 6.505%] + [0.2903 × 1.05%]
     = 4.617% + 0.305%
     = 4.922% ≈ 4.92%
```
