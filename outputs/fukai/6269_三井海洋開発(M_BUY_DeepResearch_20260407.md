# 1. 企業概要・事業分析

## 1-1. 会社概要

三井海洋開発（MODEC: Mitsui Ocean Development & Engineering Company）は1968年設立、東証プライム上場（6269）の海洋石油・ガス生産設備専業メーカー兼オペレーターである。本社は東京（港区）、グループ連結従業員は約3,500名（2025年度）。

主力事業は**FPSO（Floating Production, Storage and Offloading：浮体式生産貯蔵積出設備）** の設計・建造（EPC）から長期オペレーション（O&M）までの一貫提供。世界的に見てもFPSO専業オペレーターとしてSBM Offshore（蘭）、Technip Energies（仏）と並ぶ上位3社の一角を占める。

## 1-2. セグメント構成

```
売上高構成（FY2025 概算）
├── FPSOオペレーション (O&M)    ≈ 67%  【高安定・高マージン】
├── EPC（建造・据付）           ≈ 25%  【変動大・完工時収益集中】
├── FSOオペレーション           ≈  5%  【成熟・安定】
└── その他（MOPU/FSO等）        ≈  3%
```

**地域別売上**（推定）

| 地域 | 売上比率 | 主要顧客 |
|------|---------|---------|
| ブラジル（ペトロブラス中心） | 約55% | Petrobras, Shell Brasil |
| アジア太平洋 | 約20% | INPEX, PTTEP, Jadestone |
| アフリカ・北海 | 約15% | TotalEnergies, Eni |
| 中南米（ブラジル除く） | 約10% | ExxonMobil（ガイアナ）|

## 1-3. 競合優位性（Competitive Moat）

1. **FPSO専業深海オペレーションの実績蓄積** — 1,000m超深海でのFPSO稼働実績はSBM Offshoreと並ぶ世界最高水準。ペトロブラスの厳格なHSSE（健康・安全・環境）基準への適合ノウハウが参入障壁を形成。

2. **長期固定契約（15〜20年）のバックログ** — 一度契約が締結されると解除コストが極めて高く（FPSO1基の月次チャーターレートは$3〜6億円規模）、顧客ロックインが強固。

3. **ブラジル深海石油（プレソルト）特化** — 水深2,000〜3,000mの技術的難易度がコモディティ化を防ぎ、価格競争を回避。

4. **三井グループの信用力** — 三井物産が主要株主（約20%保有）。大型EPC契約における資金調達・与信力がSPC（特別目的会社）ファイナンスに直結。

# 2. 社長指示重点分析

## ① FPSO事業モデルの詳細：EPC vs O&M 収益構造分解

### EPC（Engineering, Procurement & Construction）フェーズ

```
【EPCフェーズの特性】
期間: 通常3〜5年（契約締結→設計→建造→据付→初生産）
収益認識: 進捗度ベース（IFRS15）
典型的なFPSO1基のEPC規模: $1.0〜2.5 billion
MODECの利益マージン（EPC）: 概ね3〜8%（リスク内包型）
リスク: 資材費高騰・工期遅延・為替変動
```

EPCフェーズは**完工時の損益振れが大きい**のが特徴。2019〜2021年のコロナ禍・鋼材高騰局面でMODECはEPCコスト超過により営業赤字を計上した経緯がある（2020年度：営業損失約▲150億円）。これがバリュエーション上の「EPCリスクディスカウント」として今日まで残存している。

**重要：** 現在のMODECはEPCリスク管理を大幅強化（固定価格EPC→コストプラスへの契約形態転換、サブコン管理厳格化）しており、2023年以降の新規EPC案件は損益改善が顕著。

### O&M（Operation & Maintenance）フェーズ

```
【O&Mフェーズの特性】
期間: 初生産後15〜20年（場合によりオプション延長あり）
収益認識: 月次定額チャーターレート（原油価格連動条項なし＝固定ベース）
典型的なFPSO1基のO&M年収: $50〜150 million（規模・仕様依存）
MODECのO&Mマージン: 20〜30%（EBITDA）
リスク: 設備故障・稼働率低下・保険コスト
```

**原油価格感応度の定量化（O&Mフェーズ）:**

| 原油価格帯 | MODECのO&M収益への直接影響 | 説明 |
|-----------|--------------------------|------|
| WTI < $40 | **ほぼゼロ（直接）** | チャーターレートは固定。ただし顧客の財務悪化→長期的な契約延長交渉リスク |
| WTI $40〜$60 | 軽微（間接）| 顧客が既存生産継続を優先→稼働率維持 |
| WTI $60〜$100 | **中立〜若干プラス** | 顧客の追加投資余力増加→延長オプション行使確率上昇 |
| WTI > $100 | **プラス（主にEPC・新規受注）** | 新規FPSO発注加速→バックログ積み上げ |

> **結論①:** O&M収益（売上の約67%）は原油価格変動から**構造的に遮断**されている。原油価格感応度は主にEPC新規受注タイミングと契約更新時の交渉力に限定される。現在WTI $113/bblは新規EPC受注加速の強力な後押し。

### Gordon Growth Model（GGM）チェック

```
適正PBR = ROE × (1-Plowback) / (Re - g_sustainble)
         または ROE / Re（純利益全額配当仮定の簡易上限）

ROE: 30.6%
Re: 7.06%
適正PBR上限 = 30.6% / 7.06% = 4.33倍

現在PBR 4.64倍 vs 適正上限 4.33倍
→ 若干プレミアム（約7%）だが、ターミナル成長率g=1.5%考慮で正当化可能域
```

**修正GGM（g=1.5%考慮）:**
```

```

**BPSベース理論株価:**
```

```
