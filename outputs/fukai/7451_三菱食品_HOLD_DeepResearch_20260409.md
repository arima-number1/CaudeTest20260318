## 3. 食品インフレ×消費二極化の影響定量化

### 3-1. 原材料・物流コスト上昇の仕入価格転嫁進捗

```
【食品インフレの転嫁メカニズム（三菱食品における作用）】

原材料コスト上昇
    → メーカーが卸売価格を引き上げ（仕入価格↑）
    → 三菱食品が小売・外食に転嫁（販売価格↑）
    → 転嫁ラグ（3〜6ヶ月）の間、粗利率が一時的に低下
    → 転嫁完了後は売上高増・粗利額維持→粗利率横ばい
```

| 評価項目 | 内容 | 推定影響 |
|---------|------|---------|
| 食品値上げ品目数（FY2026/3） | 約1.5〜2万品目（累積） | 売上高押し上げ +3〜4% |
| 転嫁ラグによる粗利率圧迫 | 転嫁完了前の時期に粗利率▲0.05〜0.10%pt | 利益影響 ▲10〜15億円（一時的） |
| 転嫁完了後の回復 | FY2026/3後半には転嫁完了で安定 | 粗利率回復 +0.05〜0.08%pt |
| **純影響（年度通算）** | 売上高増加効果が転嫁ラグコストを上回る | **ネット・ポジティブ** |

### 3-2. 消費者節約志向の影響

| 消費動向 | 内容 | 三菱食品への影響 |
|---------|------|---------------|
| 家計調査：食料支出▲1.4%（実質） | 数量減少（単価上昇で名目は横ばい） | 取扱数量・ケース数の微減 |
| PB（プライベートブランド）需要増 | SM・CVSのPB拡大：SM全体で20〜25% | NB卸の取扱量漸減リスク（年▲1〜2%） |
| 外食需要の二極化 | ファストフード堅調・高級外食好調・中価格帯苦戦 | 業務用卸は選別的に影響 |
| CVS食品需要 | 堅調（共働き増・単身世帯増が追い風） | CVS向け（売上30〜35%）は安定 |

**プライベートブランド拡大リスクの定量化（試算）：**

- SM向け売上（推定）：約9,000〜9,500億円
- NBからPBへのシフト率：年+1〜2%
- 影響額：9,000億円 × 1〜2% = **▲90〜180億円/年の売上減少リスク**
- ただし、SM向けPBの原材料・資材調達でも三菱食品が関与できるケースあり（部分的オフセット）
- **純リスク影響（営業利益換算）：▲1〜3億円/年**（粗利率0.9%換算）→限定的

### 感応度計算の詳細前提

```
各シナリオのWACC計算式（β=0.55、ERP=5.50%固定）：

Re = Rf + 0.55 × 5.50% = Rf + 3.025%
WACC = 0.75 × Re + 0.25 × 1.5% × 0.70 = 0.75 × Re + 0.2625%

TV = 155 × 1.01 ÷ (WACC − 0.01)

理論株価変動の主因はターミナルバリュー（全体の85〜90%）
→ WACCの0.25%pt変動で理論株価が±5〜6%変動
→ 金利1%上昇シナリオ（日銀追加利上げ継続）で▲18%下押し
```

> 📌 **深井コメント：** 金利感応度が高い（WACC+1%で▲18%）のは、三菱食品のFCFが安定的・緩成長型であり、ターミナルバリューの比重が大きいためです。日銀の4月利上げ「かなりの確率」（貝塚氏発言）が実現する場合（Rf+0.25〜0.50%相当）、理論株価は▲5〜9%の下押し圧力を受けます。ただし食品卸の実需は金利よりも消費動向に支配的で、**業績への直接影響は限定的**（後述⑧参照）。
