## 1. 企業概要・事業分析

### 1-1. 企業概要

**ゆうちょ銀行（Japan Post Bank Co., Ltd. / 7182.T）**は、日本郵政グループの金融子会社として2007年に民営化・分社化された銀行持株。総資産規模**約230兆円**は国内最大級（三菱UFJFGに次ぐ水準）。預金残高は**約195〜200兆円**という国内断トツのリテール預金基盤を有する。

### 1-2. セグメント・収益構造

```
総資産 約230兆円
├── 国内債券（国債・地方債）：約60〜80兆円（約28〜35%）
├── 外国証券（外債・外株）：約60〜70兆円（約26〜30%）
├── 国内株式・投資信託：約20〜30兆円（約10〜13%）
├── 貸出金：約40兆円（約17%）
└── その他資産：残余
```

**収益の約85〜90%は運用利息収益**（資金利益）が占め、手数料収益は限定的。典型的な「投資勘定型銀行」であり、通常の商業銀行（融資中心）と異なる事業モデル。

### 1-3. 競合優位性

| 強み | 詳細 |

## 4. DCFバリュエーション

### 4-1. WACC計算

**前提パラメータ**

| パラメータ | 値 | 根拠 |
|-----------|-----|------|
| Rf（リスクフリーレート） | **2.467%** | 日本10年国債（JGB10Y）2026-04-15 |
| β（ベータ） | **0.65** | 銀行・金融セクター。ゆうちょは運用型銀行で市場感応度は商業銀行より低め |
| ERP（エクイティリスクプレミアム） | **5.5%** | 日本市場標準（Damodaran参照、新興国調整なし） |
| **Re（株主資本コスト）** | **2.467% + 0.65 × 5.5% = 6.04%** | CAPM |
| D/E比率 | 負債（預金等）依存度極高だが、Equity Valuationなので Re のみ使用 | 銀行は預金を「負債」として扱うが、DCFでは株主価値直接算出 |
| **WACC（≒Re、銀行モデル）** | **6.04%** | 銀行はEquity Cash Flowモデル採用 |

> **深井注**: 銀行のDCFは通常「Dividend Discount Model（配当割引モデル）」または「Excess Return Model」を使用。一般的なWACC/FCFモデルは適合しない（負債が「仕入れ」に相当するため）。本分析はDDMベースで実施。

### 4-2. DDM（配当割引モデル）バリュエーション

**前提**

| パラメータ | 基本ケース | 強気ケース | 弱気ケース |
|-----------|----------|----------|----------|
| 現在配当（D0） | ¥50/株 | ¥55/株 | ¥45/株 |
| 短期成長率（g1, 1〜5年） | 2.0% | 4.0% | 0.5% |
| 長期成長率（g2, 6年〜永続） | 1.0% | 1.5% | 0.5% |
| Re（割引率） | 6.04% | 5.5% | 6.8% |

**DDM計算（基本ケース）**

```
2段階DDM：

Phase 1（1〜5年）：成長率g1 = 2.0%
D1 = ¥50 × 1.02 = ¥51.0
D2 = ¥52.0
D3 = ¥53.1
D4 = ¥54.1
D5 = ¥55.2

Phase 1 PV = Σ Dt/(1.0604)^t = ¥51.0/1.0604 + ¥52.0/1.124 + ¥53.1/1.192 + ¥54.1/1.264 + ¥55.2/1.340
= ¥48.1 + ¥46.3 + ¥44.5 + ¥42.8 + ¥41.2 = ¥222.9

Phase 2（永続価値, 6年〜）：
Terminal Value（TV） = D6 / (Re - g2) = ¥55.2 × 1.01 / (0.0604 - 0.01)
= ¥55.75 / 0.0504 = ¥1,106.5

TV のPV = ¥1,106.5 / (1.0604)^5 = ¥1,106.5 / 1.340 = ¥825.7

【基本ケース DDM理論株価】= ¥222.9 + ¥825.7 = ¥1,048.6
```

### 4-3. PBRアプローチ（Residual Income Model補完）

PBRアプローチ（Gordon Growth Model by ROE）：

```
理論PBR = (ROE - g) / (Re - g)
= (5.3% - 1.0%) / (6.04% - 1.0%)
= 4.3% / 5.04%
= **0.853倍**

理論株価（PBRアプローチ）= BPS × 理論PBR
= ¥2,622 × 0.853 = ¥2,237
```

### 4-4. PERアプローチ

```
理論PER = 配当性向 / (Re - g)
配当性向推計 = ¥50/¥120 = 41.7%

理論PER = 41.7% / (6.04% - 1.0%) = 41.7% / 5.04% = **8.27倍**

理論株価（PERアプローチ）= EPS × 理論PER
= ¥120 × 8.27 = ¥992

※ 実力EPS¥105使用時：¥105 × 8.27 = ¥868
```

### 4-5. 総合バリュエーション（マルチプル加重）

| 手法 | 理論株価 | ウェイト | 加重値 |
|------|---------|---------|-------|
| DDM（配当割引） | ¥1,049 | 25% | ¥262 |
| PBRアプローチ | ¥2,237 | 45% | ¥1,007 |
| PERアプローチ（実力EPS） | ¥868 | 30% | ¥260 |
| **加重平均理論株価** | — | 100% | **¥1,529** |

> ⚠️ **重要注記（深井）**: DDM・PERベースの理論株価は¥1,000〜¥1,100と現在値から▲60%以上の割高を示す。ただしPBRアプローチは¥2,237と比較的高い。銀行セクターにおいてはPBR/残余利益モデルが最も実務的に使用されるため、**PBRアプローチ¥2,237を基本ケースのコアターゲットとして採用し、DDM/PERの低水準を「下方リスクの重し」として機能させる**。
