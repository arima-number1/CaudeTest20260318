## 1. 企業概要・事業分析

### 1-1. 基本プロフィール

安田倉庫は1923年創業、東京・品川区を本拠地とする老舗総合物流企業。東京湾岸（品川・江東・大田）を中心に都内一等地の倉庫・土地を大量保有する「不動産含み益銘柄の典型」として知られる。時価総額は概算**約450億円**（株価¥2,402×発行済株式数 約1,875万株）と小型で、機関投資家の組み入れ対象になりにくい。

> **発行済株式数推計**: BPS¥3,528・純資産簿価≒EPS×PER÷ROE で逆算
> EPS ¥110 → 純利益÷発行株数
> 純資産 = BPS × 発行株数 → ROE = EPS/BPS = 110/3,528 = **3.12%**
> ※ 提供ROE 3.9%との差は連結調整・優先株等の可能性。公式値3.9%を使用。
> 時価総額 = ¥2,402 × PBR逆算純資産ベース → 純資産簿価 / PBR = 時価総額
> **純資産簿価推計**: BPS¥3,528 × 発行株数≒ → 時価総額/PBR = 450億÷0.68 ≈ **661億円（純資産簿価）**

**〈安田倉庫優位点〉**
1. **含み益/時価総額比率が桁外れに高い（78〜83%）**: 三井倉庫HDの倍以上。「不動産の安さ」が顕著。

3. **小型株プレミアム**: CGコード対応で資産活用を迫られた場合、時価総額が小さいほど株価インパクトが大きい（自社株買い100億円で株価2倍超も理論上可能）。

**〈三井倉庫HD優位点〉**
1. フォワーディング規模が20倍超で地政学特需の絶対額が大きい
2. 配当・自社株買いの実績あり（資本政策の信頼性高）
3. 流動性が高く機関投資家が参入可能
4. ROEが高く事業の収益力で劣後しない

**深井総合評価**: 「リターンの非対称性（アップサイド大・ダウンサイド限定）」では**安田倉庫が優位**。ただし流動性リスクと大株主リスクが甚大であり、**両社を組み合わせる（コア: 三井倉庫HD、サテライト: 安田倉庫）が最も合理的な戦略**。

### ④ ホルムズ混乱局面でのフォワーディング恩恵試算

### ⑤ 「株価が上がらない」構造的原因の解明

#### (A) 時価総額・流動性：機関投資家が入れるサイズか

| 指標 | 安田倉庫 | 機関投資家参入水準目安 |
|-----|---------|-------------------|
| 時価総額 | 約450億円 | 国内中小型ファンド: 500億円以上が多い |
| 日次出来高（推計） | **1,000〜3,000株/日** | 機関参入に必要: 最低1万株/日以上 |
| 出来高金額（推計） | **約240〜720万円/日** | 機関参入ボーダー: 5,000万円/日 |

**判定: 機関投資家の参入は現実的に不可能**。TOPIXのSmall Index組み入れはあるが、パッシブの機械的買いのみで積極的バリュー投資家は入れない。

#### (B) 大株主構造：創業家・安田系グループの保有状況

| 推定大株主グループ | 推定保有比率 |
|-----------------|-----------|
| 創業家・関連財団・持株会（推計） | 20〜35%程度 |
| 取引先・持合い先（商社・メーカー） | 10〜20%程度 |
| 自社（自己株式） | 若干 |
| **安定株主合計（推計）** | **30〜55%** |

安定株主が50%超を占める可能性があり、**実質的な流通株式比率は40〜50%以下**。アクティビストが委任状争奪戦を仕掛けても勝算は低い。これが最大の構造的ブレーキ。

#### (C) 資本政策の欠如

- **自社株買い**: 近年の実績は極めて限定的（年間数億円規模、もしくはゼロ）
- **増配姿勢**: 配当性向は低水準（20〜30%程度）で純利益の大半を内部留保
- **資産売却**: 倉庫・土地の売却事例は存在するが、戦略的資産活用の姿勢なし
- **IR活動**: 投資家向け説明会・英語IR発信なし→外国人機関投資家の認知度ゼロ

**判定**: 「含み益はあるが使う気がない」→これが株価低迷の本質。

#### (D) 事業の成長性：「安定だが成長期待ゼロ」

- 倉庫業界は人口・GDP連動で**年率0〜2%成長**が上限
- 安田倉庫の売上成長率は過去5年で年率**+1〜3%程度**
- Eコマース物流センター開発（竹中・プロロジス等）に対する設備競争で劣後
- ROE 3.9%はコスト・オブ・エクイティ（後述WACC計算: 約7〜8%）を大幅に下回っており**価値破壊状態**

### ⑧ DCF FairValue算出

#### (A) 事業価値（DCF）

**前提条件**

| パラメータ | 採用値 | 根拠 |
|---------|-------|-----|
| Rf（リスクフリーレート） | **2.429%** | JGB10Y（財務省日次R8.4.6） |
| ERP（株式リスクプレミアム） | **6.0%** | Damodaran日本市場推計値 |
| β | **0.60** | 倉庫業（小型・不動産保有）の推計β |
| Re（株主資本コスト） | **2.429% + 0.60×6.0% = 6.03%** | — |
| 負債比率（D/E） | **15%（D/(D+E))** | 自己資本比率60%仮定 |
| 税引前負債コスト（Rd） | **1.5%** | 低信用スプレッド |
| 実効税率（Tc） | **30%** | 日本標準 |
| WACC | **= 0.85×6.03% + 0.15×1.5%×0.70 = 5.13% + 0.16% ≈ 5.29%** | — |
| 永続成長率（g） | **1.0%** | 国内物流業・低成長 |

**WACC計算詳細**

```
E/(D+E) = 0.85, D/(D+E) = 0.15
Re = 2.429% + 0.60 × 6.0% = 2.429% + 3.60% = 6.029%
Rd×(1-Tc) = 1.5% × 0.70 = 1.05%
WACC = 0.85 × 6.029% + 0.15 × 1.05%
     = 5.125% + 0.158%
     = 5.28%（≒5.3%）
```

**FCF推計**

| 年度 | EBITDA推計 | D&A | EBIT | 税引後NOPAT | 設備投資 | FCF（推計） |
|-----|----------|-----|-----|-----------|---------|-----------|
| FY2026E | 38億円 | 12億円 | 26億円 | 18.2億円 | 8億円 | **22億円** |
| FY2027E | 40億円 | 12億円 | 28億円 | 19.6億円 | 8億円 | **23.6億円** |
| FY2028E | 42億円 | 13億円 | 29億円 | 20.3億円 | 9億円 | **24.3億円** |
| FY2029E | 43億円 | 13億円 | 30億円 | 21億円 | 9億円 | **25億円** |
| FY2030E | 44億円 | 13億円 | 31億円 | 21.7億円 | 9億円 | **25.7億円** |

**DCF計算**

```
割引率: WACC = 5.28%
ターミナルバリュー（TV）= FCF2030E × (1+g) / (WACC-g)
                       = 25.7億 × 1.01 / (5.28%-1.0%)
                       = 25.96億 / 4.28%
                       = 約606億円

DCF合計:
PV(FCF1-5): ≈ 22/1.0528 + 23.6/1.0528² + 24.3/1.0528³ + 25/1.0528⁴ + 25.7/1.0528⁵
            ≈ 20.9 + 21.3 + 20.8 + 20.4 + 19.9
            = 約103億円

PV(TV): 606億 / 1.0528⁵ ≈ 606 / 1.293 ≈ 469億円

事業価値（EV）= 103 + 469 = 約572億円
- 純有利子負債（推計△40億円、ネットキャッシュ）: +40億円
= 株主価値（事業） ≈ 612億円

一株当たり事業価値 = 612億 ÷ 発行株数（≒1,875万株）
                  ≈ ¥3,264/株
```

### ⑨ リスクシナリオ

#### リスク1: ホルムズ混乱早期収束

- **影響**: フォワーディング売上+8〜25% → 0%に逆戻り
- **EPS影響**: ▲¥8〜¥15（EPS¥110比▲7〜14%）
- **株価影響**: ▲¥50〜¥150（▲2〜6%程度、軽微）
- **深井評価**: フォワーディング特需はそもそも「+α」にすぎず、消えても致命傷にならない

#### リスク2: 倉庫老朽化・建替えコスト

- 品川・江東の主要倉庫の多くは建設後**30〜50年以上**
- 建替えコスト: 工業系倉庫で15〜20万円/㎡ × 6万㎡ = **90〜120億円**
- 含み益の「土地分（689億円）」は建替えで消えないが、稼働停止期間中の収益喪失が課題
- **含み益修繕費相殺リスク**: 老朽化が進む建物部分（簿価80〜100億円）は修繕・建替えで**30〜80億円のキャッシュアウト**が見込まれる
- **対応策**: 建替え→賃料収入で回収が基本。ただし現経営陣が「建替え→REIT化→含み益顕在化」の一歩を踏み出す動機が弱い

#### リスク3: 大株主による資産固定化の継続

- これが最大・最長期間のリスク
- 創業家・安定株主が経営権を維持する限り、含み益の顕在化は**インセンティブがない**
- 「倉庫を売れば事業が成り立たない」という経営論理もある
- CGコード改訂があっても、非上場化・MBO（管理下に移す）の可能性すら否定できない
- **極端な場合**: 非上場化→含み益を内輪で享受→少数株主は割安のまま締め出し

#### リスク4: 流動性リスク（最重要・実務的リスク）

- 日次出来高が数百〜数千株のため、**急いで売れない**
- 地政学悪化・市場急落時に買い手がいない→実効的な損切りが困難
- 小型株は市場全体のリスクオフ局面で**割安が更に割安になる罠**（バリュートラップ）
