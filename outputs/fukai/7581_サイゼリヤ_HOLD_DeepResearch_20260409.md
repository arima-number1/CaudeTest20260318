# 3. 「売れすぎコスト問題」の構造分析（最重要）

## 3-1. パラドクスの全体構造図

```
【逆レバレッジ・パラドクス】

売上  ↑↑（SSS+6〜8%）
      ↓
食材消費量 ↑↑（計画大幅超過）
      ↓
通常調達キャパシティの天井（工場能力不足）
      ↓
   ┌─────────────────┐
   │ 特別調達ルート発動     │
   │ ・スポット購入（割高）  │
   │ ・緊急航空輸送        │
   │ ・中間業者介在        │
   └─────────────────┘
      ↓
食材原価率 上昇（+1.5〜2.0%pt 推計）
      ↓
売上↑ × 原価率↑ = OP伸び率 < 売上伸び率
      ↓
「売れているのに利益が出ない」構造の完成
```

## 3-2. 特別調達コストの詳細推計

**前提計算：**

- FY2026/8通期売上高：¥1,900億円（推計）
- 通常食材原価率：約35%（¥665億円）
- 特別調達コスト上乗せ：+0.4%pt相当
- 上乗せ額：¥1,900億 × 0.4% = **¥7.6億円**（≒¥7〜8億円の推計と整合）

```
原価率感応度：
  通常原価率 35.0% → 特別調達後 35.4%（+0.4%pt）
  売上¥1,900億 → 原価増加：¥7.6億円
  これが通期OP¥12億下方修正の主因として整合的
```

| インバウンド消費 | ★（中） | 中 | 都市部店舗限定 |

### 家計調査との矛盾整理

**家計調査（2026年2月）：実質消費+0.5%・食料▲1.4%**

```
矛盾のロジック：
「食料支出全体が▲1.4%」でも「サイゼリヤSSS+6〜8%」は矛盾しない

理由：
①食料支出の節約≠外食の節約（自炊→外食のシフト現象は非線形）
②物価高による食料全体支出の名目減少の中で、外食では「安い外食」に集中
③食品スーパー・中高価格帯外食→サイゼリヤへのダウンシフト（「サイゼシフト」）
④実質消費+0.5%は可処分所得の回復を示唆。ただし食料品の値上がりで
  同額でも量が減っているだけ（デフレではなくスタグ消費の中の勝ち組）
```

**結論**：食料支出▲1.4%の中でSSSが+6〜8%成長できているのは、**「安いほうに消費が集中するリバランス効果」**。これはサイゼリヤの低価格ポジションの強さを逆説的に証明している。

## 5-1. 北京店舗の減速：原因と規模

### 北京減速の複合要因分析

```
北京店舗減速 = 景気要因 + 競合要因 + 構造要因の合計

①景気・所得要因（寄与：▲40%）
  中国の若年失業率（16〜24歳）：高止まり
  不動産デフレによる資産効果の喪失
```
