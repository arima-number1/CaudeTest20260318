## 5. DCFバリュエーション修正（1Q実績反映）

### WACC・前提条件

| パラメータ | 数値 |
|-----------|------|
| JGB10Y（リスクフリーレート） | 1.50% |
| ERP（株式リスクプレミアム） | 5.50% |
| β | 1.10 |
| **Ke（株主資本コスト）** | **1.50% + 5.50%×1.10 = 7.55%** |
| 自己資本比率 | 91.6% → 実質**Equity Only**でWACC≒Ke |
| **WACC** | **≒7.55%** |

### FCFの推計

**Step1：1Q実績ベースの年率換算FCF**
- 1Q純利益：38M → **年率換算：152M**
- 減価償却費（推定）：年間30M → 1Q換算7.5M
- 設備投資（推定）：年間△20M → 1Q換算△5M
- 運転資本増減：1Q自己資本比率91.6%の極めて安定した構造から軽微と仮定
- **1Q換算FCF：38 + 7.5 - 5 = 40.5M → 年率換算FCF：162M**

**Step2：DCFモデル（3段階）**

| フェーズ | 期間 | 成長率 | 根拠 |
|---------|------|--------|------|
| 高成長期 | 1〜3年 | **12%** | 1Q実績の利益成長率（+54.9%）を保守的に圧縮 |
| 移行期 | 4〜7年 | **6%** | 業界平均水準への収束 |
| 永続成長期 | 8年〜 | **2.5%** | 名目GDP成長率水準 |

**Step3：DCF計算**

```
■ 高成長期FCF（1〜3年、g=12%）
  Y1: 162M × 1.12¹ = 181.4M → PV: 181.4/1.0755 = 168.7M
  Y2: 181.4M × 1.12  = 203.2M → PV: 203.2/1.0755² = 175.5M
  Y3: 203.2M × 1.12  = 227.5M → PV: 227.5/1.0755³ = 182.6M
  小計PV: 526.8M

■ 移行期FCF（4〜7年、g=6%）
  Y4: 241.2M → PV: 180.5M
  Y5: 255.7M → PV: 177.9M
  Y6: 271.0M → PV: 175.4M
  Y7: 287.3M → PV: 172.9M
  小計PV: 706.7M

■ 永続成長価値（ターミナルバリュー、g=2.5%）
  TV = FCF_Y8 / (WACC - g)
     = (287.3M × 1.025) / (0.0755 - 0.025)
     = 294.5M / 0.0505
     = 5,831M
  PV of TV: 5,831M / 1.0755⁷ = 3,508M

■ 企業価値合計
  EV = 526.8 + 706.7 + 3,508 = 4,741M

■ 純有利子負債調整（自己資本比率91.6%・実質無借金）
  Net Debt ≒ 0（現金リッチ）→ Equity Value = 4,741M
```

### 理論株価（分割前・分割後）

| 指標 | 数値 |
|------|------|
| Equity Value | **4,741百万円** |
| 発行済株数（分割前） | **3,025千株（推定）** |
| **理論株価（分割前）** | **4,741M ÷ 3,025千株 ≒ 1,567円** |
| **理論株価（分割後）** | **1,567円 ÷ 2 ≒ 784円** |

### バリュエーションレンジ（感応度）

| シナリオ | 高成長期g | 理論株価（分割前） | 理論株価（分割後） |
|---------|---------|----------------|----------------|
| Bear | 8% | **1,280円** | **640円** |
| **Base** | **12%** | **1,567円** | **784円** |

> **現在株価水準との比較**：仮に市場価格が800〜900円（分割前）近辺であれば、Base想定でも**約75〜95%の上昇余地**。Bullシナリオでは2倍超。
