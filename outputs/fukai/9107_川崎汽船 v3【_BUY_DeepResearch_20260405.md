## 4. DCFバリュエーション

### 4-1. WACC算出前提

#### ① リスクフリーレート（絶対厳守）
```
Rf = 2.380%
出典: 日本10年国債（JGB10Y）
取得日時: 2026-04-05 18:02 JST（社長確認値）
参照: https://jp.investing.com/rates-bonds/japan-10-year-bond-yield-streaming-chart
```

#### ② パラメータ設定

| パラメータ | 値 | 根拠 |
|-----------|---|------|
| Rf | 2.380% | JGB10Y（社長確認値） |
| β（川崎汽船） | 1.30 | 海運セクター平均。市況感応度高く1.2〜1.4レンジ中央 |
| ERP（日本株） | 5.50% | 竹村TC標準。Damodaran参考・日本リスクプレミアム |
| Re（株主資本コスト） | 2.380% + 1.30×5.50% = **9.53%** | CAPMベース |
| Rd（負債コスト） | 2.80% | 格付BBB+相当・現行借入金利推定 |
| Tc（実効税率） | 30.0% | 日本法人標準 |
| D/（D+E）（負債比率） | 約31% | ネット有利子負債¥2,100億／総資本¥6,810億 |
| E/（D+E）（自己資本比率） | 約69% | 上記の残余 |

#### ③ WACC計算

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.69 × 9.53%] + [0.31 × 2.80% × (1-0.30)]
     = 6.576% + 0.607%
     = 7.18%
```

**WACC = 7.18%**

| 前提 | フェーズ1（FY2027〜2030） | フェーズ2（FY2031〜2035） | ターミナル |

| FCF成長率 | +3.0% | +1.5% | +1.0% |
| 想定FCF基点 | ¥900億（3Q年率換算） | — | — |
| 永続成長率（g） | — | — | 1.0% |

### 4-3. DCF計算（5年明示期間＋TV）

#### フェーズ1 FCF系列（FY2027〜2031）

| 年度 | FCF（億円） | 割引係数（7.18%） | PV（億円） |

#### ターミナルバリュー（TV）

```
TV = FCF_2032 × (1+g) / (WACC - g)
   = 1,044 × 1.01 / (0.0718 - 0.01)
   = 1,054.4 / 0.0618

```

```
EV（事業価値） = PV合計 + TV現在価値
```

## 6. PBR1.0倍超障壁：ROE乖離分析

### 6-1. PBRとROEの関係式

```
PBR = ROE / Re（持続的均衡の場合）

現在のPBR0.98倍が示す市場の暗示ROE：
0.98 = 暗示ROE / 9.53%
暗示ROE = 0.98 × 9.53% = 9.34%

実際のROE（FY2026/3予想）：約28〜32%

乖離 = 実際ROE(30%) - 暗示ROE(9.34%) = +20.66%ポイント
```

### 6-2. なぜPBR1.0倍超に届かないか

| 阻害要因 | 内容 |
|---------|------|
| ① 利益の持続性懸念 | ONE運賃市況は不可逆的に高水準ではないとの市場認識 |
| ② シクリカル株ペナルティ | 海運株は「ピーク値では買えない」慣行的割引 |
| ③ ONE非連結リスク | 持分法のため利益の質（可視性）が低いと見なされる |
| ④ 地政学プレミアム | 紅海リスク・中国景気・米中貿易戦争の不確実性 |
| ⑤ 金利上昇局面 | Rf上昇→ Re上昇→ 理論PBR低下圧力（後述感応度参照） |

### 6-3. PBR1.0倍超の条件

```
PBR > 1.0 の条件：ROE > Re（=9.53%）

FY2026/3 ROE予想30%超 → 条件は充足
問題は「持続性の信頼」

PBR1.5倍達成には：市場が「ROE 15%以上が3〜5年持続」と確信することが必要
→ 自社株買い・DOE政策・ONE安定配当が証明手段
```

## 10. DOE下値支持線

### 10-1. DOE政策の枠組み

```
DOE（Dividend on Equity）= DPS / BPS × 100

BPS：¥2,753（PBR0.98倍・株価¥2,698から逆算）
```

### 10-2. DOE水準別 DPS・配当利回り

| DOE水準 | DPS（計算値） | 配当利回り（¥2,698ベース） | 評価 |
|---------|------------|----------------------|-----|
| DOE 3.0% | ¥83 | 3.07% | 最低ライン |
| DOE 4.0% | ¥110 | 4.08% | 保守的下限 |
| DOE 5.0% | ¥138 | 5.11% | 現実的下限 |
| DOE 6.0% | ¥165 | 6.12% | 中期目標水準 |
| DOE 7.0% | ¥193 | 7.15% | 強気水準 |
| **現行DPS¥300** | **DOE換算** | **10.9%** | **現在は高水準** |

### 10-3. 下値支持線の解釈

```
仮にDOE5.0%を下限として設定した場合：
→ DPS≥¥138が保証される
→ 配当利回り5.11%が株価下値支持として機能
→ 配当利回り5%水準の株価：¥138÷5% = ¥2,760（現在株価¥2,698に近い）

DOE5%設定はPBR0.98倍・¥2,698近辺に強い配当サポートを形成
```

**DOE政策の明示は、市場に「BPSが下がっても配当水準が連動して下がる」という透明性を与え、過剰な悲観売りを抑制する。¥2,698近辺が配当利回り面でも底値圏であることを示す。**
