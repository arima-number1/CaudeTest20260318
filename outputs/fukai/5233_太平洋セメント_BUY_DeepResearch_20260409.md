# 2. 社長指示重点分析

## ① PBR0.66倍の構造的原因分析（最重要）

### 1-1. PBR割れの多因子分解

現在のPBR = **0.66倍**（BPS ¥5,719、株価 ¥3,758）

PBRはROEと株主資本コスト（Re）の比率で理論的に決まる：

```
PBR = ROE / Re × (持続的成長を前提にした調整係数)

簡易式: PBR ≈ (ROE - g) / (Re - g)

現状:
  ROE = 3.7%
  Re  ≈ 8.0〜9.5%（後述WACC算出参照）
  g   ≈ 1.5%（長期成長率 名目GDP成長率ベース）

  理論PBR = (3.7% - 1.5%) / (9.0% - 1.5%)
           = 2.2% / 7.5%
           = 0.293倍

→ 理論PBRは0.29倍程度になるはずだが、実際は0.66倍
```

> **深井解釈:** 市場は0.29倍の「ROEだけで見た理論値」より高い0.66倍をつけている。
> これは①石灰石鉱山・不動産含み益（NAV上乗せ）、②将来ROE改善への期待、③廃棄物処理事業の隠れた収益性、を市場が一定程度織込んでいることを意味する。
> **つまりPBR0.66倍は「純粋な低ROEの市場評価（0.29倍）+ 資産含み益プレミアム（+0.37倍分）」の合成値だ。**

**【深井の最終判定: グレーゾーン寄りの「条件付き本物バリュー」】**

1. 石灰石鉱山の埋蔵量価値（後述③で詳述）が帳簿価額を大幅に上回る
