## 🔑 回転率67%「不足」の意味と4/27受渡リスク

```
TOPIX リバランス Buy: 310,600株 × ¥1,636 = ¥5.08億（ADV 91%）
PO受渡:              4,764,200株 × ¥1,635 = ¥77.9億

合計4/27受渡需要: 約¥83億（ADV 1,640%相当の巨大フロー）

回転率67% = 貸株可能株数 < 現在の売り残
→ ブローカーが受渡できない可能性
→ 受渡不能（Fail）発生 → 強制買い戻し（翌営業日）
→ 月曜寄りの需給はPO+リバランス+受渡Failの三重圧力
```

4/27(月)の想定動き: 寄り付きで受渡Fail分の買い戻しが出れば一時的に株価サポートとなるが、それはFail解消であり実需の買いではない。Fail解消後は再び売り圧力。
