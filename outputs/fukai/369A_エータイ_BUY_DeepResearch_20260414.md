## セクション4：DCFバリュエーション

### 4-1. DCF前提条件

> **⚠️ 以下の前提は公開情報・業界標準値を使用。独自推計は最小化し、根拠を全て明示。**

#### 基本前提

| パラメータ | 値 | 根拠・出典 |
|-----------|-----|----------|
| **リスクフリーレート (Rf)** | **2.467%** | 日本10年国債（財務省 2026-04-14） |
| **ERP（株式リスクプレミアム）** | **5.5%** | 日本市場長期平均（Damodaran推計ベース） |
| **β（ベータ）** | **0.85** | 小型・内需株（葬祭・終活関連）の業種特性。市場ベータ~1.0より低い（ディフェンシブ性あり） |
| **株主資本コスト (Re)** | **2.467% + 0.85 × 5.5% = 7.14%** | CAPM |
| **負債コスト (Rd)** | **1.5%** | 小型優良企業の銀行借入金利（推計） |
| **実効税率 (Tc)** | **30%** | 日本法人税・地方税合算標準 |
| **資本構成 E/(D+E)** | **85%** | コンサル・仲介業は負債少（推計。要BS確認） |
| **WACC** | **0.85×7.14% + 0.15×1.5%×(1-0.30) = 6.23%** | 計算明示 |
| **永続成長率 (g)** | **2.0%** | 高齢化構造需要×インフレ勘案（日本名目成長率上限） |

#### WACC計算詳細

```
Re = 2.467% + 0.85 × 5.5% = 2.467% + 4.675% = 7.142%
Rd × (1-Tc) = 1.5% × 0.70 = 1.05%

WACC = 0.85 × 7.142% + 0.15 × 1.05%
      = 6.071% + 0.158%
      = 6.23%
```
