## 1. 企業概要・事業分析

### 1-1. ゼンムテック（338A）とは何者か

ゼンムテックは**ゼロトラストネットワークアクセス（ZTNA）に特化した国内純粋プレイSaaS企業**。社名「全夢（ゼンム）」は「全てのアクセスを信頼しない」というゼロトラスト哲学を体現している（深井推察）。

東証グロース市場上場（2024年）、時価総額約77億円の小型成長株。

### 1-2. プロダクト構成（推計）

| セグメント | 推計売上比率 | 特徴 |
|-----------|------------|------|
| ZTNAクラウドプラットフォーム（SaaS） | 約70% | サブスクリプション課金。ARR成長の主エンジン |
| セキュリティコンサルティング | 約20% | 導入支援・監査。SaaSへのブリッジ役 |
| 政府・防衛向けオンプレミス | 約10% | 高単価・低成長。但し防衛需要拡大で比率上昇期待 |

### 1-3. 競合優位性（Competitive Moat）

**① 国産ゼロトラストの認定・準拠優位性**
- NISC（内閣サイバーセキュリティセンター）・デジタル庁の国産優先調達方針と整合
- 外資大手（Zscaler等）は安全保障上の懸念から政府調達で不利な立場に置かれる可能性

**② 日本語・日本語法規環境への深い適合**
- 個人情報保護法・サイバーセキュリティ基本法への準拠設計
- 中小・中堅企業向けの日本語サポート体制（外資にとって最大の弱点）

**③ 切り替えコスト（スイッチングコスト）の高さ**
- ZTNAは企業のID認証基盤と深く統合→一度導入すると3-5年は乗り換えられない

## 2. 社長指示の重点分析（深井優先度1）

### 2-1. ① AI×セキュリティ：陳腐化リスク vs ZTNA需要拡大機会（両面定量分析）

#### 🔴 AIがセキュリティを陳腐化するリスク（ネガティブシナリオ）

| リスク要因 | 定量的影響推計 | 確率評価 |
|-----------|-------------|---------|
| AIがゼロトラスト判定を自動化→汎用ツールに組み込み | 既存ZTNAベンダーのARR成長率▲5-10pt下押し | **低（15%）** |
| 大手クラウド（AWS/Azure/GCP）がZTNA機能を無料バンドル | 中小企業市場の獲得困難→TAMが▲20-30%縮小 | **中（25%）** |
| 生成AIを悪用した高度な偽ID攻撃がZTNAを突破 | ブランドダメージ→解約率上昇（NRR▲5-10pt） | **低中（20%）** |

**陳腐化リスクの総合確率加重インパクト**：ARR成長率▲3-8pt（確率加重後）

#### 🟢 AIがZTNA需要を増大させる機会（ポジティブシナリオ）

| 機会要因 | 定量的影響推計 | 確率評価 |
|---------|-------------|---------|
| 生成AI（Claude/GPT等）を企業内で活用するための安全なアクセス制御需要が爆発的増加 | エンタープライズZTNA市場+35-50%成長加速 | **高（70%）** |
| AIエージェントのAPI間通信を制御するZTNAの新ユースケース創出 | 新カテゴリ「AI-to-AI ZTNA」でARR単価+20-30%期待 | **中高（45%）** |
| サイバー攻撃のAI高度化→防御側もAI×ZTNAが必須に（軍拡競争） | セキュリティ予算全体+15-25%増加→ZTNA予算も比例 | **高（65%）** |
| 政府のAI活用推進×セキュリティ義務化の同時進行 | 公共調達で国産ZTNA優先→ゼンムテック直接受益 | **中高（50%）** |

**機会の総合確率加重インパクト**：ARR成長率+12-22pt（確率加重後）

#### 📊 AIインパクトの純効果（ネット）

```
期待値（ネット）= 機会+12〜22pt − リスク▲3〜8pt = 純効果：+9〜14pt

→ AIは中長期でゼンムテックに「強い追い風」。陳腐化より需要創出が優勢。
```

**深井の結論**：AIによるZTNA陳腐化リスクは現時点では限定的。むしろClaude等の生成AIが企業内に浸透するほど、「誰が何のAIに何のデータを見せていいか」を管理するZTNAの必要性は**逆説的に急増**する。これは社長指示の「逆説的急増」という表現と完全に一致する。

### 4-2. DCFモデル（基本ケース：5年予測 + 残存価値）

**FCF成長前提（基本ケース）**

| FY2029E | 25% | ¥3.4B | 18% | ¥612M |
| FY2030E | 20% | ¥4.1B | 20% | ¥820M |

**DCF計算**

```
割引率（WACC）= 10.64%

PV(FY2026) = ¥120M / 1.1064^1 = ¥108.5M
PV(FY2027) = ¥252M / 1.1064^2 = ¥206.0M
PV(FY2028) = ¥405M / 1.1064^3 = ¥299.6M
PV(FY2029) = ¥612M / 1.1064^4 = ¥410.0M
PV(FY2030) = ¥820M / 1.1064^5 = ¥496.4M

FCFの合計PV = ¥1,520.5M

残存価値（Terminal Value）：
TV = FCF(2030) × (1+g) / (WACC - g)
   = ¥820M × 1.030 / (0.1064 - 0.030)
   = ¥844.6M / 0.0764
   = ¥11,054M

PV(TV) = ¥11,054M / 1.1064^5 = ¥6,692M

企業価値（EV）= ¥1,520.5M + ¥6,692M = ¥8,212M

純負債調整（推計▲¥100M：現金保有が負債超過と仮定）：
株主価値 = ¥8,212M + ¥100M = ¥8,312M

発行済株式数 ≒ 271万株

理論株価 = ¥8,312M / 271万株 = ¥3,067
```
