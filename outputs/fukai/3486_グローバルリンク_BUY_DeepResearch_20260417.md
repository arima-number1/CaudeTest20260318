## 1. 企業概要・事業分析

### 1-1. ビジネスモデル概要

グローバルリンクマネジメント（GLM）は**東京都心部（23区中心）における投資用ワンルーム・コンパクトマンションの企画・開発・販売**に特化した純粋プレーヤーである。

```
【GLMのバリューチェーン】
土地仕入（23区・駅徒歩5分圏）
    ↓
自社企画・設計（GLM独自スペック）
    ↓
建設（外部ゼネコン活用）
    ↓
販売（富裕層・法人投資家・海外投資家）
    ↓
オプション：管理・賃貸管理サービス（ストック収益）
```

**同社の競合優位性（ソースオブエッジ）:**

| 優位性 | 詳細 | 持続性 |
|--------|------|--------|
| 土地仕入ネットワーク | 23区内の小規模・変形地への特化ノウハウ | 高（10年以上の実績） |
| 投資家データベース | 累計購入者への反復販売モデル | 高（スイッチングコスト） |
| 商品スペック設計力 | 入居率・利回り最大化特化の間取り・設備 | 中〜高 |
| 回転速度 | 着工から24-30ヶ月での販売完了 | 中（金利環境依存） |

### 1-2. セグメント構成

```
売上構成（推計）:
├── 投資用マンション販売:    約90-92%（コア）
├── 管理・賃貸収益（ストック）: 約5-7%（成長中）
└── その他:                  約1-3%
```

**ROE34.7%の源泉分解（デュポン分析）:**

```
ROE = 純利益率 × 資産回転率 × 財務レバレッジ

推計:
純利益率:      約6.5-7.0%
資産回転率:    約1.2-1.4回（在庫回転型ビジネス）
財務レバレッジ: 約3.5-4.0倍（不動産開発業として標準的）

ROE = 7% × 1.3 × 3.8 ≈ 34.6% ✓（実績34.7%と整合）
```

> **深井所見**: ROEの高さはレバレッジ単独ではなく、純利益率と資産回転率の両輪が機能している点が特筆すべき質の高さ。「偽物の高ROE（レバレッジのみ）」ではない。

## 2. 社長指示重点分析：「不動産に異変でも起きているのかってぐらい安い」

### 2-1. PER6.9倍×ROE34.7%の「矛盾」を解剖する

**グレアム・ドッドのバリュエーション公式:**

理論PER ＝ ROE ÷ 株主資本コスト

```
株主資本コスト（Re）≈ 8.5-10.0%（後述WACC参照）

理論PER = 34.7% ÷ 9.0% = 38.6倍

現実PER = 6.9倍

乖離倍率 = 38.6 ÷ 6.9 = 5.6倍の「割安」
```

**→ 結論: この乖離は「異常」であり、市場は何らかの構造的リスクを価格に織り込んでいる。その正体を以下で特定する。**

---

### 2-2. 割安の真因候補①：JGB金利上昇リスク

**JGB10年金利 = 2.409%（2026-04-17現在）の意味**

```
【不動産投資家の購入意思決定ロジック】

期待利回り（グロス）: 東京23区ワンルーム ≈ 4.5-5.5%
管理費・空室損失控除後（ネット）: ≈ 3.0-4.0%

金利スプレッド（投資妙味）:
= ネット利回り - 借入金利
= 3.5% - 2.0%（投資ローン金利） = 1.5%（現在）

JGBが3.0%に上昇した場合:
投資ローン金利 ≈ 3.5-4.0%
スプレッド = 3.5% - 3.8% = ▲0.3%（逆ザヤ）
→ 投資需要が急速に萎縮
```

**金利感応度（GLM売上への影響推計）:**

| JGB水準 | 投資ローン金利 | 投資需要への影響 | GLM売上影響 |
|---------|-------------|---------------|------------|
| 2.0%以下 | 1.5-2.0% | 非常に旺盛 | +10〜+15% |
| 2.409%（現在） | 2.0-2.5% | 堅調 | ベースライン |
| 2.75% | 2.5-3.0% | 慎重化 | ▲5〜▲10% |
| 3.00% | 3.0-3.5% | 急減速 | ▲15〜▲25% |
| 3.50%超 | 3.5-4.0% | 市場停滞 | ▲30%超 |

**→ 現在の2.409%は「危険水域」ではないが、市場は3%突破シナリオを確率的に織り込んでいる。この「金利恐怖プレミアム」がPER圧縮の主因と判定。**

---

### 2-3. 割安の真因候補②：在庫リスク（棚卸資産の積み上がり）

不動産開発業における在庫（棚卸資産）は**金利コストを毎月消費する「燃えるキャッシュ」**である。

```
【在庫リスクの定量化】

棚卸資産（推計）: ≈ ¥250-350億円
平均在庫金利コスト: 2.0-2.5%/年
月次金利負担: ≈ ¥0.4-0.7億円/月

在庫回転日数（正常値）: 180-240日
在庫回転日数（要警戒）: 300日超
在庫回転日数（危機）:   360日超
```

**過去の不動産デベロッパー在庫クラッシュ事例:**
- 2008年リーマンショック：アーバンコーポレイション等、在庫膨張→倒産
- 2022年金利上昇局面：米国新興デベロッパーで在庫損失が表面化

**GLMの在庫リスク評価:**

```
ポジティブ要因:
✓ 東京23区特化→空室率2%台で需要の底堅さ確認
✓ 1棟あたり規模が小さく（20-50戸規模）、リスク分散
✓ 専有面積25-35㎡の実需性高い商品特性

ネガティブ要因:
✗ 完成在庫の絶対額が2026年8月決算まで非開示
✗ BOJ利上げ観測期に購入決定を先送りした顧客の存在
✗ 土地仕入ペースを落とさない場合、在庫累増リスク
```

**→ 在庫リスクは「実在する懸念」だが、東京23区の需給構造を考慮すると「致命的リスク」ではなく「管理可能なリスク」と判定。現時点での織り込みは過剰と見る。**

---

### 2-4. 割安の真因候補③：BOJ政策リスクと見送り確定の意味

```
BOJ 2026年4月会合：利上げ見送り確定
→ 次回利上げ時期コンセンサス: 2026年Q3-Q4（最短）

GLMへの影響タイムライン:
4月：見送り確定 → 購入決定の心理的障壁除去
5-7月：春の引渡しシーズン → 販売実績確認
8月：本決算公表 → 在庫・EPS確認の分水嶺
Q3-Q4：次回利上げ判断 → リスク再評価
```

**BOJ見送りによる「時間的猶予」の価値:**

現在の状況は「嵐の前の凪」であり、GLMにとって：
1. 既存在庫の消化に3-6ヶ月の追加時間
2. 購入を躊躇していた顧客の購入再開
3. 市場センチメントの改善による株価PER正常化

**→ BOJ利上げ見送り確定は短期カタリストとして有効。ただし次回利上げまでの「期限付き猶予」であることを認識した時間軸管理が必須。**

## 4. DCFバリュエーション

### 4-1. 前提条件

```
【WACC算出前提】

■ リスクフリーレート（Rf）
  Rf = 2.409%（JGB10年 2026-04-17 財務省）← 絶対遵守

■ エクイティリスクプレミアム（ERP）
  ERP = 5.5%（日本株市場平均：Damodaran 2026年推計採用）

■ ベータ（β）
  β = 1.35（推計：不動産小型株セクター平均 1.2-1.5の中央値）
  ※ 小型株・不動産セクター・金利感応度を加味して高めに設定

■ 株主資本コスト（Re）
  Re = Rf + β × ERP
     = 2.409% + 1.35 × 5.5%
     = 2.409% + 7.425%
     = 9.834% ≈ 9.8%

■ 負債コスト（Rd）
  Rd = 2.5%（不動産開発向け銀行融資金利推計）
  税率（Tc）= 30%（法定実効税率）
  税引後Rd = 2.5% × (1-0.30) = 1.75%

■ 資本構成（E/D比率）
  純資産（E）≈ ¥94億
  有利子負債（D）≈ ¥300億（推計：不動産開発業標準）
  E/(D+E) ≈ 24%
  D/(D+E) ≈ 76%

■ WACC計算
  WACC = [0.24 × 9.8%] + [0.76 × 1.75%]
        = 2.352% + 1.330%
        = 3.682% ≈ 3.7%

  ※ 不動産開発業の高レバレッジ構造により、WACCは低位
  ※ ただしエクイティホルダーの実質コストは9.8%で評価
```

> **深井注記**: DCFは「株主価値」を算出するため、エクイティDCF（FCFE/Re法）と企業価値DCF（FCFF/WACC法）の両面から確認する。

Year 2:  ¥310 × 1.08 = ¥335

Year 4:  ¥361 × 1.08 = ¥390
