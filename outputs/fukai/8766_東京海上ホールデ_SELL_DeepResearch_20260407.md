### 2.2 バフェット提携の実態検証

#### 提携の構造（浅井第31弾言及の背景）

バークシャー・ハサウェイ（以下BH）と東京海上HDの関係は、**直接的な資本提携（相互持合い）ではなく、間接的・戦略的関係**である点を明確にする必要がある。

**事実確認：**

| 項目 | 実態 |
|------|------|
| BHの東京海上株式保有 | **確認されていない**（BHの13-F/年次報告に東京海上株記載なし） |
| 東京海上のBH株保有 | **確認されていない** |
| 再保険取引 | 可能性あり（非開示） |
| バフェット発言 | 日本の損保・商社株全般への好意的言及（特定銘柄としての東京海上推奨発言なし） |

**「バフェットプレミアム」の実態：**

バフェットが日本株（特に商社5社）への大規模投資を行ったことで、「日本の優良企業＝バフェット好み」という連想が働き、東京海上HDにも**間接的なハロー効果**が生じていると解釈するのが適切。直接提携ではなく**マーケットナラティブによるプレミアム**である。

#### バフェットプレミアムの定量試算

**Gordon Growth Model（GGM）による適正PBR算出：**

```
適正PBR = ROE × (配当性向) / (Re - g)
```

前提：
- ROE = 20.7%
- 配当性向 = 40%（推定）
- Re（株主資本コスト） = 7.5%（後述WACC算出参照）
- g（成長率） = 3.5%（基本シナリオ）

```
適正PBR = 20.7% × 40% / (7.5% - 3.5%)
         = 8.28% / 4.0%
         = 2.07倍
```

**現在PBR（2.61倍）との乖離：** +0.54倍 → **金額換算：¥2,800 × 0.54 = ¥1,512/株の過剰プレミアム**

> **「バフェットプレミアム」の適正水準：** GGM上の適正PBRは2.07倍。現在の2.61倍との差0.54倍はバフェットプレミアム・需給プレミアム・希少性プレミアムが混在する「説明困難な超過」。この超過プレミアムは¥1,500/株程度であり、ナラティブが剥落した場合の下落余地を示す。

### 2.5 DCF フェアバリュー算出（金利シナリオ別 × バフェットプレミアム有無別）

#### WACC算出前提

**① 株主資本コスト（Re）**

| パラメータ | 値 | 根拠 |
|-----------|-----|------|
| Rf | **2.386%** | 日本10年国債（指示値） |
| β | **0.85** | 大手損保・ディフェンシブ系 |
| ERP | **5.5%** | 日本市場標準 |
| **Re** | **2.386% + 0.85 × 5.5% = 7.061%** | |

**② 負債コスト（Rd）**

| パラメータ | 値 |
|-----------|-----|
| 有利子負債 | 約¥8,000億（劣後債中心） |
| Rd | 1.8% |
| 実効税率（Tc） | 30% |
| Rd×(1-Tc) | 1.26% |

**③ 資本構成**

| 項目 | 金額 | 比率 |
|------|------|------|
| 株主資本（E） | ¥10.5兆（時価総額） | 93% |
| 有利子負債（D） | ¥0.8兆 | 7% |

**④ WACC計算**

```
WACC = 93% × 7.061% + 7% × 1.26%
     = 6.567% + 0.088%
     = 6.655% ≒ 6.66%
```

#### DCFモデル前提（基本シナリオ）

| 前提項目 | 基本シナリオ | 強気シナリオ | 弱気シナリオ |
|---------|------------|------------|------------|
| 現在EPS | ¥550 | ¥550 | ¥550 |
| FCF成長率（Year1-5） | 5.5% | 8.0% | 2.0% |
| FCF成長率（Year6-10） | 4.0% | 6.0% | 1.5% |
| 永続成長率（g） | 2.5% | 3.5% | 1.5% |

| バフェットプレミアム | なし(0%) | あり(+10%) | なし(0%) |

#### DCF計算結果

**【基本シナリオ】（バフェットプレミアムなし）**

```
Phase 1（1〜5年）FCF成長5.5%：
Year1: ¥550 × 1.055 = ¥580.3
Year2: ¥611.9
Year3: ¥645.6
Year4: ¥681.1
Year5: ¥718.6
PV（WACC=6.66%）= ¥2,523

Phase 2（6〜10年）FCF成長4.0%：
PV合計 = ¥1,789

Terminal Value（g=2.5%）:
TV = ¥718.6 × (1+4.0%)^5 × (1+2.5%) / (6.66%-2.5%)
   = ¥875.0 × 1.025 / 0.0416
   = ¥21,555
PV of TV = ¥21,555 / (1.0666)^10 = ¥11,432

合計PV ≒ ¥2,523 + ¥1,789 + ¥11,432 = ¥15,744（純資産価値ベース）

→ 一株換算DCF理論株価（基本） ≒ ¥5,800
```

**【強気シナリオ】（バフェットプレミアム+10%）**

```
成長加速＋WACC低下＋永続成長3.5%：
DCF理論株価 ≒ ¥6,900 × 1.10 = ¥7,590
→ バフェットプレミアム込み上限 ≒ ¥6,900（プレミアムなし） / ¥7,590（プレミアム10%）
```

> ⚠️ 強気シナリオでバフェットプレミアム込みでもようやく現在値水準に達する。つまり**現在株価は「強気×バフェットプレミアム最大」の組み合わせでのみ正当化される。**

