## 2-1. Disintermediation（中抜き）の構造的背景

### 2-1-a. Anthropic / OpenAI のエンタープライズ直販戦略の現状

```
【AI大手の直販加速トレンド】

Anthropic Claude Enterprise（直販）
  → Fortune500クラスへの直接API契約
  → "Bedrock経由 vs 直接契約"の二層構造が形成中
  → 日本市場: 2025年以降、国内大企業への直販営業を本格化

OpenAI Enterprise
  → ChatGPT Enterprise: 中大規模企業への直接SaaS販売
  → Azure OpenAI経由 vs OpenAI直接 の競合が激化

Microsoft Copilot（365統合）
  → SIer不要で既存M365ライセンスに自動追加
  → IT部門が直接管理可能 → SIer関与余地が縮小

Google Gemini / Vertex AI
  → GCP直販エンジニアによる直接導入支援
```

**核心**: AI製品は「ソフトウェアをSIerが代理販売して実装する」従来モデルではなく、**「ベンダーが直接契約し、APIで自己実装させる」モデルへの移行**が進んでいる。

【Disintermediationモデル】
顧客企業 ────────────────→ Anthropic/OpenAI（直販）
              ↑SIerが完全に排除される

【ハイブリッド移行期（現在）】
顧客企業 → SIer（実装支援のみ）→ AI直販ベンダー
              ↑マージンは残るが激減（5〜10%程度）

# 5. DCFバリュエーション（シナリオ別）

## 5-1. WACC算出

### 株主資本コスト（Re）の算出

```
■ 使用パラメータ:
  Rf（リスクフリーレート）= 2.380%（JGB10Y 社長確認値）
  β（ベータ）= 0.75（中小SIer / スタンダード市場 / 流動性低め）
              ※ 東証SIer小型株平均β 0.65〜0.85 の中央値を採用
  ERP（株式リスクプレミアム）= 6.0%（日本市場標準推計）
              ※ Damodaran 日本ERP参考 / 市場コンセンサス

  Re = Rf + β × ERP
     = 2.380% + 0.75 × 6.0%
     = 2.380% + 4.500%
     = 6.880%
```

### WACC算出

```
■ 資本構成（推定）:
  自己資本比率: 約55〜65%（SIer平均 / 低負債構造）
  → E/(D+E) = 60%
  → D/(D+E) = 40%

  負債コスト（Rd）: 1.5%（日本の銀行借入 低金利環境）
  実効税率（Tc）: 30%（日本法人税標準）

  WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
        = [0.60 × 6.880%] + [0.40 × 1.500% × (1-0.30)]
        = 4.128% + 0.420%
        = 4.548%
        ≒ 4.55%（ベース WACC）
```
