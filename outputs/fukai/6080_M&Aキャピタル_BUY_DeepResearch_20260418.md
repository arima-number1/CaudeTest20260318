## 1. 企業概要・事業分析

### 1-1. ビジネスモデルの本質

M&Aキャピタルパートナーズ（以下「M&Aキャピ」）は、**中堅・中小企業M&Aの専任アドバイザリー（FA方式）に特化した純粋プレイ銘柄**。仲介方式ではなく**FAアドバイザリー方式**を採用している点が競合との最大差別化。

**ビジネスモデル詳細：**

```
【収益構造】
売り手企業 ——相談——→ M&Aキャピ（FA）——マッチング——→ 買い手企業
                          ↓
              成功報酬（Success Fee）のみ
              = 売上高の大部分が成功報酬
              = 固定費は人件費（コンサルタント）のみ
              = レバレッジ型ビジネス：成約増加が直接EPS増加に直結
```

| 項目 | M&Aキャピ | 業界平均（中小仲介） |
|------|----------|-----------------|
| 主要顧客規模 | 売上高10〜500億円（中堅中小） | 売上高1〜30億円（小規模） |
| 平均案件規模（譲渡額） | ¥20〜150億円 | ¥1〜10億円 |
| 手数料率 | 3〜8%（レーマン変形） | 5〜10%（小規模高率） |
| 平均手数料額/件 | **¥1.8〜2.5億円** | ¥300〜800万円 |
| FA方式採用 | **○（売り手専任）** | △（仲介が主流） |
| 年間成約件数 | 80〜95件 | 数百〜数千件 |

**資産軽量モデルの圧倒的収益性：**
- 固定資産ほぼゼロ（コンサルタント×オフィスのみ）
- FCFマージン：推定25〜30%
- ROE：11.1%（提供値）← **実態はもっと高いはずだが、自社株買い・現金蓄積で希薄化している**

### 1-2. 競合優位性（構造的モート）

```
1. 大型案件処理能力（最重要）
   → 100億円超案件に対応できるコンサルタント数・DD知識
   → 中小仲介業者には人材・ノウハウの両面で参入不可能

2. 売り手専任FA（利益相反の排除）
   → 仲介方式では売り手・買い手双方から受領→利益相反リスク

3. ブランド・実績（参照案件の積み上げ）
   → 上場企業オーナー・経営者コミュニティ内での紹介案件
   → 「M&Aキャピに頼めば高く売れる」という口コミが最強の営業

4. 人材密度（1コンサルあたり収益が高い）
   → コンサルタント数：推定200〜250名
   → 1人あたり売上高：¥5,000〜6,000万円（業界最高水準）
```

## 2. 社長指示重点分析：Minimum Tax（Pillar2）特需の全解剖

### 2-1. Pillar2 グローバル最低税率：日本適用スケジュールと対象企業規模

#### ■ 制度概要

```
OECD Pillar2（GloBE Rules）
= 年間連結売上高7.5億ユーロ（約1,200億円）超の多国籍企業に
  実効税率15%以上を義務付け
= 適用開始：日本は2024年4月1日以降開始事業年度から適用済
= 具体的には「IIR（所得合算ルール）」が2024年度に発動
```

#### ■ 日本企業の対象規模と「Minimum Tax回避型M&A」のメカニズム

**なぜMinimum Tax導入でM&Aが加速するのか？**

```
【メカニズム1：タックスヘイブン子会社の清算・売却急増】
→ 実効税率15%未満の子会社（低税率国拠点）は
  IIRによりトップアップ税課税 → 節税メリット消滅
→ 対応策：低税率子会社を売却 or グループ再編
→ M&Aキャピへの案件持ち込み増加

【メカニズム2：国内M&Aによるグループ最適化（実効税率調整）】
→ 日本国内で実効税率が低い企業を買収し
  グループ全体の平均税率を15%超に引き上げる
→ 期末（3月・9月）に向けた成約ラッシュ

【メカニズム3：中堅・同族企業オーナーの「売り時」意識】
→ 大手グループからの買収打診が増加
→ 「今が高く売れる」タイミングとして売り手が前向きに
→ M&Aキャピの売り手FA案件として流入
```

**対象企業数の試算（深井推計）：**

| カテゴリー | 推定社数 | M&Aキャピへの影響 |
|----------|---------|----------------|
| 連結売上1,200億円超の日本多国籍企業 | **約600〜800社** | 大型案件のバイヤー側として積極的に買収実行 |
| 上記グループ傘下で売却検討中の事業・子会社 | **約200〜400件（年間）** | 直接の売り案件として流入 |
| 中堅・中小オーナー企業（大手グループの買収対象） | **無数（推定3,000〜5,000社が2〜3年で顕在化）** | M&Aキャピの主戦場 |

> **深井見解：** 2024年4月に制度は適用開始されているが、企業側の税務対応・意思決定・アドバイザー選定・DD実施のリードタイムを考えると、**実際のM&A成約として数字に出るのが2Q（7〜9月2026）から——これが芦澤氏のタイムラインの根拠。** 1Q（4〜6月）は検討フェーズ・アドバイザー選定フェーズ。

**M&Aキャピが推奨No.1である決定的理由：**

```
1. 「大型案件×Minimum Taxは最高の組み合わせ」
   → Minimum Taxで動く企業は連結売上1,200億円超の大企業グループ
   → 彼らの子会社・事業売却は譲渡額50〜200億円規模
   → これはM&Aキャピのど真ん中。ストライク・日本M&Aは小さすぎて取れない

2. 「FA方式＝高い売値＝売り手が選ぶ」
   → 大企業グループに売る案件でFA方式は売り手利益を最大化
   → 仲介方式の競合は利益相反懸念でリジェクトされるリスク

3. 「PER21.3倍は特需前の低評価」
   → FY25ピークEPS¥190〜200想定に対し、現在株価¥3,350 → 実質PER16〜18倍
   → 特需を織り込んでいない割安水準
```
