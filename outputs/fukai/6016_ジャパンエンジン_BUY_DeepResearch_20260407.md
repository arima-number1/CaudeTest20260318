### ⑦ PBR・BPS・ROE（yfinance確認値）ベースのバリュエーション

#### yfinance確認値の整合性チェック

```
【yfinance確認値（絶対正）】
  現在株価：¥13,640
  BPS：¥795

        レバレッジ・資本構成の複雑性を示す可能性あり、yfinance値を優先使用）
```

#### PBR・ROE・COE（株主資本コスト）による理論PBR分析

```
【Gordon Growth Model 変形によるPBR理論値】

理論PBR = (ROE - g) / (COE - g) × 1
※ COE = Re（株主資本コスト）

Re = Rf + β × ERP
  Rf = 2.429%（絶対厳守）
  β = 1.20（防衛・重工系、市場平均よりやや高め）
  ERP = 6.5%（日本株式リスクプレミアム標準）
  Re = 2.429% + 1.20 × 6.5% = 2.429% + 7.8% = 10.229%

ROE = 7.6%（yfinance確認値）
g（長期成長率） = 3.5%（防衛需要・海運回復・脱炭素3軸織り込み）

理論PBR = (7.6% - 3.5%) / (10.229% - 3.5%)
         = 4.1% / 6.729%
         = 0.61倍

現在PBR 17.16倍 vs 理論PBR 0.61倍 → 大幅乖離

【解釈】
この乖離は以下を示唆：
1. ROE 7.6%は現時点の低水準であり、今後の防衛需要拡大で向上する期待
2. バランスシートが薄い（BPS ¥795）ため、PBRが高倍率になりやすい構造
3. 市場は将来のROE改善（10〜15%水準への上昇）を先行織り込んでいる
4. テーマ株・希少性プレミアム（舶用エンジン国内唯一の上場企業）

将来ROEが15%に改善した場合の理論PBR：
= (15% - 3.5%) / (10.229% - 3.5%)
= 11.5% / 6.729%
= 1.71倍

理論株価：¥795 × 1.71倍 = ¥1,359 → PERベースとの大幅乖離を示す
※PERベース（¥510 × 26.7倍 = ¥13,617）と比較するとPBRベース理論値は保守的すぎる

【深井分析】
PBR/ROEフレームワーク単体では不十分。
J-ENGは資産軽量型ビジネスモデルでなく、重厚長大型製造業であるため、
DCF（収益価値）とPERの複合アプローチが適切。
```
