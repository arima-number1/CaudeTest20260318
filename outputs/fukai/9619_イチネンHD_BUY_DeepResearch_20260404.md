## 1. 企業概要・事業分析

### 1-1. 会社概要

イチネンホールディングス（旧：イチネン）は、**自動車リース・メンテナンス事業を中核**としながら、生活環境サービス（防錆・防蟻処理など）、駐車場運営、化学品販売、農業資材など多角的な事業を展開する純粋持株会社。創業1930年代にさかのぼる老舗企業で、総合BtoB生活インフラサービスグループとしての性格を持つ。

- **本社:** 東京都
- **市場:** 東証プライム
- **従業員数:** 連結 約3,000〜3,500名（グループ計）
- **主要子会社:** イチネンオート（カーリース）、イチネンケミカルズ（化学品）、イチネンマテリアルズ（農業資材関連）等

### 1-3. 今回のトリガー：三菱商事アグリサービス2社の子会社化

#### 【概要】
イチネンHDが三菱商事グループ傘下のアグリサービス関連2社（詳細社名は開示内容による）を株式取得し、子会社化。農業資材・アグリサービス分野への**本格的かつ戦略的参入**を宣言。

#### 【買収の戦略的意義】

```
【バリューチェーン拡張イメージ】

従来:
イチネンHD ──→ 農業資材販売（小規模）

買収後:
イチネンHD ──→ 三菱商事アグリサービスA社（農業資材調達・卸）
            ──→ 三菱商事アグリサービスB社（農業サービス・ソリューション）
            ──→ 農業資材販売（スケールアップ）
            ──→ 農家・農業法人向けトータルサービス（新展開）
```

#### 【シナジー効果（推計）】

| シナジー項目 | 内容 | 効果規模（推計） |
|------------|------|----------------|
| 売上上乗せ | 2社の売上連結取込 | +20〜40億円/年（推計） |
| 仕入ルート強化 | 三菱商事ネットワーク活用 | 原価率改善余地 |
| 顧客基盤 | 農業法人・JAルートの拡大 | クロスセル機会創出 |
| ブランド力 | 三菱商事ブランドの活用 | 信用力・営業力向上 |

#### 【農業資材市場の構造的追い風】

- **食料安全保障**への国内回帰意識の高まり
- **スマート農業**推進による農業資材需要の高度化・増加
- **農業法人化**の進展による購買の組織化・大型化
- 政府の農業振興政策（農林水産省 農業基本法改正等）

## 4. DCFバリュエーション

### 4-1. 前提条件（全開示）

#### マクロ前提

| パラメータ | 設定値 | 根拠・出所 |
|-----------|-------|-----------|
| **リスクフリーレート (Rf)** | **1.5%** | **JGB10年物（2026年4月時点）** |
| **エクイティリスクプレミアム (ERP)** | **5.5%** | 日本市場標準（Damodaran日本ERP参照） |
| **β（ベータ）** | **0.85** | 東証データ参照。多角化・ディフェンシブ性を反映し市場β(<1) |
| **株式コスト (Ke)** | **6.175%** | Ke = Rf + β × ERP = 1.5% + 0.85 × 5.5% |
| **負債コスト（税引後）** | **1.2%** | 短期借入・長期借入金利約1.8%×（1-実効税率33%） |
| **資本構成（E/V）** | **55%** | 時価総額÷（時価総額＋有利子負債）推計 |
| **資本構成（D/V）** | **45%** | |
| **WACC** | **3.93%** | WACC = Ke×E/V + Kd×D/V = 6.175%×55% + 1.2%×45% |
| **永続成長率 (g)** | **1.0%** | 日本長期GDP成長率＋農業資材成長を保守的に反映 |

```
【WACC計算式の詳細】
WACC = 6.175% × 0.55 + 1.2% × 0.45
     = 3.396% + 0.540%
     = 3.94%（≒3.9%で計算）
```

### 4-3. DCF計算（基本シナリオ）

#### 予測期間のPV計算

| 年度 | FCFF（億円） | ディスカウントファクター（WACC=3.94%） | PV（億円） |
|------|------------|--------------------------------------|-----------|
| FY2025E | 16.4 | 0.962 | 15.8 |
| FY2026E | 20.1 | 0.925 | 18.6 |
| FY2027E | 22.8 | 0.890 | 20.3 |
| FY2028E | 25.8 | 0.857 | 22.1 |
| FY2029E | 27.8 | 0.824 | 22.9 |
| **予測期間PV合計** | | | **99.7** |

#### ターミナルバリュー（TV）計算

```
TV = FCFF(FY2029) × (1 + g) / (WACC - g)
   = 27.8 × (1 + 1.0%) / (3.94% - 1.0%)
   = 28.08 / 2.94%
   = 955.8億円

TV現在価値 = 955.8 × 0.824
           = 787.6億円
```

#### 企業価値（EV）→株主価値計算

```
事業価値（EV）= 予測期間PV + TV現在価値
             = 99.7 + 787.6
             = 887.3億円

株主価値 = EV - 純有利子負債 + 非事業用資産
        = 887.3 - 175.0（推計ネット有利子負債）+ 20.0（持分法投資等）
        = 732.3億円

発行済株式数 ≈ 約950〜1,000万株（自己株除く、推計）

理論株価（基本） = 7,323,000万円 ÷ 960万株
               ≈ 2,430〜2,500円
```

> 📌 **重要注記:** 発行済株式数・ネット有利子負債は公開情報（有報・決算短信）で確認要。上記は推計値です。

## 6. メインシナリオ vs リスクシナリオ

### 6-1. 強気シナリオ（Bull Case）— 確率: 25%

#### 前提条件
- 三菱商事アグリサービス2社との統合が順調に進み、想定以上のシナジー効果
