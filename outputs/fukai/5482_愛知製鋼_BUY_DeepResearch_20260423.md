## 2. 社長指示重点分析

### 2-1. 🎯 値上げ¥10,000/ton のOPMインパクト定量試算

#### Step 1：販売数量の推定

```
特殊鋼鋼材売上高（FY2025推定）：¥2,100億円
平均単価（特殊鋼）             ：¥100,000〜120,000円/ton（推定）
  ※特殊鋼は普通鋼の3〜5倍単価。¥110,000/ton中央値採用
販売トン数推定                 ：¥2,100億 ÷ ¥110,000 ≒ 約190万ton/年
  ※うち値上げ対象（構造用鋼・ばね鋼・軸受け鋼）：約70% → 133万ton
```

#### Step 2：値上げによる売上・利益インパクト

```
値上げ幅：¥10,000/ton（最低値・保守的採用）

【年間売上増加額】
  133万ton × ¥10,000/ton = ¥133億円/年

【年間コスト増加額（スクラップ+10%）】
  スクラップ使用量推定：200万ton/年（鉄スクラップ→特殊鋼）
  スクラップ現在価格：¥45,000〜50,000円/ton（国内相場）
  コスト増：200万ton × ¥47,500 × 10% = ¥95億円/年

【純増OPM（値上げ売上増 − コスト増）】
  ¥133億円 − ¥95億円 = +¥38億円/年（純利益押上）
  → 税引き後（Tc=30%）: +¥26.6億円
  → EPS押上効果: ¥26.6億 ÷ 発行済株式数（約3,500万株推定）≒ +¥76/株

【OPMへの影響】
  FY2025ベース営業利益：¥82億円（推定）
  値上げ後営業利益：¥82億 + ¥38億 = ¥120億円（+46%）
  OPM：2.6% → 3.8%（+1.2pt）
```
