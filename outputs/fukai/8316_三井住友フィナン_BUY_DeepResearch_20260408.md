### ① プライベートクレジット・エクスポージャーの定量把握（最重要）

#### （A）MFS（Market Financial Solutions）破綻によるSMBCの直接損失試算

**事実整理：**

| 項目 | 内容 |
|------|------|
| 破綻企業名 | Market Financial Solutions（MFS）英国 |
| 事業内容 | 英国モーゲージ（ブリッジローン・住宅ローン）ノンバンク貸出 |
| 破綻時期 | 2026年2〜3月（報告） |
| SMBCエクスポージャー | £100M（約133億円 ＠¥133/$、£1=¥189前後） |
| 性質 | ウェアハウスライン（短期与信ファシリティ）と推定 |

**担保・優先劣後構造の分析：**

MFSのようなモーゲージ系ノンバンクへのウェアハウスラインは、一般的に以下の構造を持つ：

```
【典型的なウェアハウスライン構造】

SMBCポジション（シニア優先）
─────────────────────────────
担保: MFS保有モーゲージ債権（LTV 70-80%水準）
担保比率: 原資産の80-90%まで融資（ヘアカット10-20%）
優先順位: 第一順位担保権（Senior Secured）
MFS出資部分: 劣後（損失吸収バッファー）

回収シナリオ：
・担保実現価格が元本比80%以上 → SMBCほぼ全額回収
・担保実現価格が60〜80%     → 部分損失（最大£20M程度）
・担保実現価格が60%未満     → 最大損失£40M（約53億円）
```

**損失試算（深井推計）：**

| シナリオ | 担保実現率 | 想定損失（£） | 想定損失（¥） |
|---------|-----------|-------------|-------------|
| 楽観 | 90%以上 | £10M未満 | 13億円未満 |
| ベース | 80% | £20M | 約27億円 |
| 悲観 | 65% | £35M | 約47億円 |
| 最悪 | 50% | £50M | 約67億円 |

**⚠️ 重要判断：**
SMFGの年間純利益（通期ガイダンス9,500億円）対比で、最悪損失£50M（約67億円）は
**純利益の0.7%に過ぎない。**これは**実質的に誤差の範囲**である。

ただし、英国不動産市場の流動性低下が続く場合、担保実現に時間を要する可能性があり、
引当計上のタイミングが投資家心理に一時的な影響を与える可能性は排除できない。

### ⑧ DCF FairValue算出（3ケース）

#### 前提共通パラメータ

```
【WACC算出前提】

リスクフリーレート（Rf）：2.411%（JGB10Y 2026/4/8）
エクイティリスクプレミアム（ERP）：5.5%（新興国調整後日本市場標準）
β（8316）：0.95（銀行セクター平均、金利感応性を反映）

株主資本コスト（Re）：
Re = Rf + β × ERP
   = 2.411% + 0.95 × 5.5%
   = 2.411% + 5.225%
   = 7.636%

資本構成（推計）：
E/（D+E）：14%（株式時価総額 / 総資産規模で調整）
D/（D+E）：86%（預金・借入等）
負債コスト（Rd）：1.2%（調達コスト推計）
実効税率（Tc）：30%

WACC = 14% × 7.636% + 86% × 1.2% × (1-30%)
      = 1.069% + 0.724%
      = 1.793%

※ 銀行業のWACCは総資産ベースで計算するため、
  製造業等と異なり低WACCが正常。
  代替的にROE/PBRアプローチを主軸とし、DCFはサポートに使用。
```

> **深井注記：** 銀行業のDCFは「超過利益モデル（EBO/DDM）」が適切。
> 以下ではDDM（配当割引モデル）を主軸として算出する。
