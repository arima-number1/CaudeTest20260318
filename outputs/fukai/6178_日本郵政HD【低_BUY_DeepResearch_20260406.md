## 2. 【社長指示最優先】不動産資産の実態価値・含み益試算 / ゆうちょ・かんぽ分離バリュー影響

> **社長指示:** 「不動産資産の実態価値をDCF・含み益両面から算出。ゆうちょ・かんぽ分離のバリューへの影響も必須」

### 2.1 不動産資産の全体像

日本郵政グループが保有する不動産は、主に以下3層で構成される：

```
【不動産資産 3層構造】

Layer 1: 郵便局舎（全国2.4万局）
  → 多くが自社所有または長期賃借物件
  → 都市部物件の簿価は1960-1990年代取得 → 含み益が巨大

Layer 2: JP不動産（日本郵政グループの不動産管理子会社）
  → 大手町・丸の内等のオフィスビル、物流施設
  → 「KITTE」（旧東京中央郵便局跡地）等の商業施設

Layer 3: 郵便局土地（特に主要都市の一等地）
  → 東京・大阪・名古屋・福岡等の駅前
  → 開発余地が最も大きいレイヤー
```

### 2.4 PBR乖離の構造的分析

```
現在PBR: 0.54倍
BPS（帳簿上）: ¥3,381

理論的に「正当化されるPBR」= ROE ÷ 株主資本コスト（Re）
= 4.4% ÷ 8.5%（後述Re）= 0.52倍

→ 現在のPBR 0.54倍は、ROE=4.4%・Re=8.5%の前提では
  むしろ「ほぼ適正評価」に近い水準
  
→ 「本物の割安」にするためには:
  ① ROE改善（6〜8%水準へ）、または
  ② 不動産含み益の顕在化（NAV折り込み）、または
  ③ 両方の複合カタリストが必要
```

## 4. DCFバリュエーション

### 4.1 WACC算出

#### 資本構成・コスト前提

| パラメータ | 数値 | 根拠・出典 |
|----------|------|----------|
| **Rf（リスクフリーレート）** | **2.380%** | JGB10Y（社長確認値 2026-04-06） |
| **ERP（株式リスクプレミアム）** | **5.50%** | 日本株式市場標準値（Damodaran 2025） |
| **β（ベータ）** | **1.10** | 金融持株会社・規制業種・低流動性調整 |
| **Re（株主資本コスト）** | **8.43%** | Re = 2.380% + 1.10 × 5.50% |
| **Rd（負債コスト）** | **1.80%** | 長期社債利回り水準（Arated / 日本金融機関） |
| **Tc（実効税率）** | **30.0%** | 日本法人実効税率標準 |
| **E/（D+E）（株式比率）** | **65%** | グループBS（金融負債調整後） |
| **D/（D+E）（負債比率）** | **35%** | 同上 |

#### WACC計算

```
Re = 2.380% + 1.10 × 5.50% = 2.380% + 6.050% = 8.430%

WACC = [0.65 × 8.430%] + [0.35 × 1.80% × (1 - 0.30)]
     = 5.480% + 0.441%
     = 5.921%
     ≒ 5.92%
```
