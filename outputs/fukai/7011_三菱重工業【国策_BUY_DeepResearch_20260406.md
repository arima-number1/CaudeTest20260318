#### ■ 「停戦利確→押し目」のメカニズム分析

```
【株価のドライバー分解】

MHI株価 = 本源的価値（DCF） + 地政学プレミアム + モメンタム

地政学プレミアム（現状推定）:
  中東緊張: +3〜5%程度の上乗せ
  極東リスク（台湾・北朝鮮）: +8〜12%程度の上乗せ
  合計: +11〜17%程度が「リスクプレミアム乗せ」分

→ 中東停戦のみ解消: 最大-5〜-8%の一時的調整

→ 「極東リスク」は停戦で消えない（むしろ資源が極東集中で増大）
  → この調整が「押し目」の本質
```

### 2-3. 有事ベータ推計（通常時β vs 有事時β）

#### ■ ベータ推計方法論

```
【ベータ推計アプローチ】

① 通常時ベータ（Normalizedβ）:
   TOPIX vs MHI株価 の60ヶ月回帰（長期平均）
   推定値: β ≈ 1.15〜1.25
   → 市場感応度は高め（重工業・製造業として標準的）
   → 本DCFで使用: β = 1.20

② 地政学イベント時ベータ（Event-drivenβ）:
   過去の北朝鮮ミサイル発射・台湾緊張時の日次騰落率を
   TOPIX日次騰落率と比較した「イベント感応度」

   ─────────────────────────────────────────
   イベント種別        MHI騰落  TOPIX騰落  推定β(event)
   ─────────────────────────────────────────
   北朝鮮ミサイル発射    +3〜5%   -0.5〜+1%   3.0〜5.0
   台湾海峡緊張激化     +5〜8%   -1〜+2%     2.5〜4.0
   防衛予算増額発表     +8〜15%  +0〜+3%     3.0〜5.0
   中東停戦合意        -3〜6%   +0.5〜+2%   ▲2.0〜▲3.0
   日銀サプライズ利上げ  -2〜4%   -3〜-5%     0.5〜1.0
   ─────────────────────────────────────────

③ 「有事ベータ」の実態解釈:
   → 防衛株の有事ベータは「市場と逆方向」に動く場合がある
   → 有事時: 市場全体が売られる中でMHIは買われる
     = 「ネガティブ有事ベータ」（ヘッジ機能）

【有事ベータ区分まとめ】
─────────────────────────────────────────
シナリオ          通常βに対する 修正係数  実効ベータ
─────────────────────────────────────────
平時（TOPIX連動）      β = 1.20         1.20
北朝鮮ミサイル          β = -3.0〜-5.0  ※市場逆行
台湾有事警戒            β = -2.0〜-4.0  ※市場逆行
防衛予算サプライズ増     β = 3.0〜5.0    ※単独急騰
停戦・地政学鎮静        β = 2.0〜3.0    ※市場と同方向だが過大に下落
─────────────────────────────────────────
※「市場逆行」= 市場全体下落時にMHIが上昇するため、
  厳密には「負の相関」となり伝統的ベータ定義外だが
  ポートフォリオのヘッジとして機能する
```

## 4. DCFバリュエーション

### 4-1. WACC計算

#### ■ 前提パラメータ

| パラメータ | 数値 | 根拠・出典 |
|----------|------|----------|
| **Rf** | **2.380%** | JGB10Y / 社長確認値 2026-04-06 |
| **β（通常時）** | **1.20** | 推定（TOPIX 60ヶ月回帰） |
| **ERP（株式リスクプレミアム）** | **6.00%** | 日本株ERP実務標準値 |
| **Re（株主資本コスト）** | **9.58%** | Rf + β×ERP = 2.380% + 1.20×6.00% |
| **Rd（負債コスト）** | **1.80%** | MHI社債利回り推定 |
| **Tc（実効税率）** | **30%** | 日本法人税実効税率 |
| **D/(D+E)** | **35%** | 負債比率推定 |
| **E/(D+E)** | **65%** | 株式比率推定 |

#### ■ WACC計算

```
WACC = E/(D+E) × Re + D/(D+E) × Rd × (1-Tc)
     = 0.65 × 9.58% + 0.35 × 1.80% × (1-0.30)
     = 6.227% + 0.441%
     = 6.668%
     ≈ 6.67%
```

### 4-2. DCFモデル（基本ケース）

```
  永続成長率(g): 1.50%（日本GDP成長率+インフレ目標折衷）
```
