## 2. 社長指示重点分析

### 2-1. 大和証券TP¥2,200→¥3,600（+63.6%）格上げの根拠徹底解明

#### ＜格上げ前（旧TP¥2,200）の想定前提＞
旧TPは明らかに以下を前提にしていたと推察される：
- 自動車向け（ICE）が主力として継続し、EV転換でのダメージを過大評価
- 中国需要の回復シナリオで保守的試算
- 航空宇宙成長を「オプション」として除外
- PER倍率：約15〜17倍（旧EPS¥130〜145を想定）
- 結果：¥130×17倍 ≒ ¥2,210 → TP¥2,200と整合

#### ＜新TP¥3,600の根拠を逆算解明＞

大和証券の格上げは以下の3軸の同時発火と見られる：

**軸①：EPS再評価（利益モデルの上方修正）**
```
旧想定EPS: ¥130（自動車依存・航空宇宙除外）
新想定EPS: ¥175〜185（航空宇宙成長込み・FY2027E）
  └ 航空宇宙寄与: +¥20〜25
  └ 北米底打ち: +¥10〜15
  └ コスト削減・ミックス改善: +¥10〜15
  └ DR流動性プレミアム解消: PER押し上げ要因

大和TP逆算: ¥3,600 ÷ PER想定 ＝
  ¥3,600 ÷ ¥180（FY2027E EPS想定） ＝ PER 20.0倍
  → 現在PER 20倍と整合。「EPS¥180×PER20倍＝¥3,600」が大和の計算式と確信
```

**軸②：バリュエーション倍率の正常化（PER）**
- 旧評価：PER 16〜17倍（EV転換不確実性プレミアム込み）
- 新評価：PER 20〜21倍（航空宇宙成長株として再評価 + DR流動性改善）
- 比較対象の変化：競合の切削工具メーカー（サンドビック・ケナメタル）はPER 22〜25倍
- OSGがグローバル同業並みに評価されれば自然にPER20倍台

**軸③：DR（預託証書）新規実施の需給インパクト**
- 海外機関投資家が東証への直接投資なしにOSGへアクセス可能に
- フロートの拡大 → 外国人持株比率の上昇 → バリュエーション格差縮小
- 過去事例（DRを新規実施した中規模日本株）でPER改善幅：平均+2〜4倍
- これだけで¥300〜600の株価押し上げ効果が試算可能

**【格上げ根拠の結論】**
```
大和証券の価値上昇ロジック：
  FY2027E EPS: ¥175〜185（航空宇宙30%成長込み）
  × PER: 20倍（DR実施後の流動性正常化）
  = TP: ¥3,500〜3,700
  中央値 ¥3,600 ← これが旧TP¥2,200比+63.6%の正体
```

## 4. DCFバリュエーション

### 4-1. DCF前提条件（全明示）

```
【絶対固定値】
  現在株価:    ¥3,049（2026-04-23 Yahoo Finance取得）

  ERP（株式リスクプレミアム）: 5.5%（日本株標準、Damodaran 2026推計）

    ※ 競合京セラβ≒1.10、サンドビック相当β≒1.20から補間

    Re = 2.392% + 1.15 × 5.5%

【WACC計算】
```
