## 1. 企業概要・事業分析

### 1-1. 太平洋セメントとは

太平洋セメント（5233.T）は国内セメント最大手（シェア約35〜40%）。旧・秩父小野田セメントと日本セメントの合併により1998年設立。売上高は約9,000〜9,500億円規模（FY2025）。

**セグメント構成（概要）**

| セグメント | 売上比率（推計） | 主な製品・地域 |
|-----------|--------------|-------------|
| セメント・生コン（国内） | 約45〜50% | 普通ポルトランドセメント・混合セメント・生コン |
| 海外（北米・アジア・豪州） | 約25〜30% | 米国（Giant Cement / Calaveras）・アジア・オセアニア |
| 建材・環境事業 | 約15% | 骨材・廃棄物処理・太陽光 |
| その他（商事・サービス） | 約10% | — |

### 1-2. 競合優位性

1. **国内シェア断トツ**：住友大阪セメント・宇部三菱セメントを大きく引き離す規模の経済
2. **廃棄物処理（環境事業）との融合**：セメントキルンで廃棄物・バイオマスを燃料化し、燃料コスト削減と廃棄物処理収入の二重取り
3. **北米事業の安定収益**：Giant Cement（米東部）・Calaveras（米西部）はインフラ法恩恵を享受
4. **石灰石資源の自社保有**：原料面での価格変動リスクが低い

### 1-3. 直近カタリスト

- 能登半島復興需要（2026〜2028年）
- 米国インフラ投資法（IIJA）執行本格化（米国子会社の追い風）
- 国内防衛インフラ（基地整備・滑走路延伸等）
- **LPG代替燃料（廃棄物・石炭・水素）への転換進捗**（最重要ウォッチポイント）

## 2. 社長指示重点分析 ｜「LPG高騰・調達難」

> **社長：これが今回レポートの核心です。丁寧に分析します。**

### 2-1. セメントキルンとLPGの関係

セメント製造の核心工程は「クリンカー焼成」。ロータリーキルンを1,400〜1,500℃に加熱する必要があり、燃料としてLPG（液化石油ガス）・石炭・重油・廃棄物等を使用する。

**太平洋セメントのエネルギー構成（推計）**

| 燃料種 | 比率（推計） | 単価変動リスク |
|--------|------------|------------|
| 石炭（一般炭） | 約35〜40% | 中（豪州・インドネシア炭） |
| LPG（プロパン・ブタン） | 約20〜25% | **高（ホルムズ直撃）** |
| 廃棄物・バイオマス | 約25〜30% | 低（処理収入で逆ザヤ解消） |
| 重油・その他 | 約10〜15% | 中 |

### 2-2. ホルムズ封鎖シナリオの定量インパクト

**LPG輸入の中東依存度（日本全体）**

- 日本のLPG輸入の**約80%が中東産**（サウジアラビア・UAE・カタール）
- ホルムズ海峡はその主要ルート。封鎖・準封鎖状態では**スポット価格が現在の1.5〜3倍**に急騰するリスク

**コスト感応度試算（深井試算）**

```
前提：太平洋セメント年間エネルギー費 ≒ 1,800〜2,000億円（推計）
     LPG比率：約22% → LPG費用 ≒ 400〜440億円/年

シナリオ別LPG価格変動インパクト：
  +25%上昇 → 追加コスト +100〜110億円 → OPM ▲1.1〜1.2%pt
  +50%上昇 → 追加コスト +200〜220億円 → OPM ▲2.2〜2.4%pt
  +100%上昇（ホルムズ完全封鎖） → 追加コスト +400〜440億円 → OPM ▲4.4〜4.9%pt
  ※売上高9,000億円で除した場合
```

**→ ホルムズ完全封鎖なら、現在の営業利益（推計430〜540億円）がほぼ消失するインパクト。**

### 2-3. 調達難リスク（質・量両面）

- **スポット調達への依存増大**：長期契約外の緊急調達はスポット価格で行われ、さらに高コスト
- **船腹不足・保険コスト急騰**：中東情勢緊迫化→タンカー保険料上昇→CFRコスト激増
- **代替調達先の限界**：米国産LPG（シェール由来）・豪州産は増量可能だが、設備・インフラ制約で短期切り替えに限界

### 2-4. 太平洋セメントの防衛策（現状評価）

| 防衛策 | 進捗・評価 |
|--------|---------|
| 廃棄物・バイオマス燃料比率引き上げ | ◎ 構造的に進行中。最も有効な長期策 |
| 石炭比率引き上げ（短期代替） | ○ 可能だが石炭価格も連動上昇リスク |
| 重油・灯油への一時切り替え | △ キルン設備制約あり。全面切替は困難 |
| LPG長期契約比率引き上げ | △ 現状不明。緊急契約は割高 |
| 価格転嫁（値上げ） | ○ 過去に実績あり。ただしタイムラグ半年〜1年 |
| 操業度調整（生産量絞り込み） | △ 固定費負担増で逆効果の面も |

**社長への直言：最悪ケース（ホルムズ封鎖長期化）では、廃棄物燃料への急速転換が唯一の実効的防衛策。転換の進捗速度が株価命運を分ける。次の決算説明会でこの質問を経営陣に直撃すべき。**

### 3-2. 財務指標の解読

**純利益（逆算）**
- EPS ¥461 × 発行株数（推計）≒ **1.18億株** → 純利益 **≒ 544億円**

**純資産（逆算）**
- BPS ¥5,719 × 発行株数（推計）≒ **1.18億株** → 純資産 **≒ 6,749億円**

**ROE分析（デュポン分解）**
```
ROE = 純利益率 × 総資産回転率 × 財務レバレッジ
3.7% ≒ 純利益率(約5.8%) × 回転率(約0.48) × レバレッジ(約1.33)
→ 回転率の低さ（重装備型製造業の宿命）と利益率の薄さが低ROEの根本原因
```

**ROE 3.7%の評価**
- TOPIX平均ROE（約8〜10%）を大幅に下回る
- PBR0.65倍（解散価値以下）はこの低ROEを正直に反映
- ROEがWACC（後述：約6〜7%）を下回っており、**理論上は価値破壊状態**
