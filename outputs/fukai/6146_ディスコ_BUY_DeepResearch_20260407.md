### 1-2. 事業セグメント構成

```

├── 精密加工装置（装置売上）：～75%

**注目ポイント①：消耗品ビジネスの安定性**
ダイサー用ダイシングブレードは毎月消費される消耗品であり、装置稼働率に連動して売上が立つ。HBMやNAND生産が増加すれば消耗品需要も自動的に拡大するストック型モデル。**装置売上の景気敏感性をコンシューマブルが下支え**する構造が競合との差別化要素。

**注目ポイント②：KIRU・KEZURU・MIGAKU（KKM戦略）**
ディスコは「切る・削る・磨く」の全工程で世界最高精度を誇る。HBMの多層積層パッケージング（TSV：Through Silicon Via）では、ウェーハの超薄型化（50μm以下）が不可欠であり、グラインダー需要が構造的に拡大している。

### ③ DRAM2倍超・Samsung8倍増益との乖離メカニズム解明

#### 装置メーカーの「6〜12ヶ月ラグ」構造

```
【需要爆発→株価上昇のタイムライン】

DRAM価格上昇・Samsung増益確認
        ↓（発表：2025年Q4〜2026年Q1）
Micron・Samsung・SK Hynixが設備投資計画決定
        ↓（+1〜3ヶ月）
ディスコへ発注内示・正式受注
        ↓（+2〜4ヶ月）
製造・検品・納品
        ↓（+2〜3ヶ月）
顧客側で検収完了 → ディスコの売上計上
        ↓
財務数値に反映 → アナリストが確認 → 株価上昇

合計ラグ：発注から売上計上まで6〜12ヶ月
```

**現在地の特定（2026年4月時点）：**

```
2025年Q3〜Q4：DRAM価格反転・Samsung増益確認
    → ディスコへの発注内示：2025年Q4〜2026年Q1（現在進行中）

2026年Q1〜Q2：正式受注・バックログ積み上げフェーズ
    → バックログは急増しているが、まだP/Lに反映されていない
    → 4/22決算の「受注残・受注高」開示が最大注目点

2026年Q3〜Q4：納品・売上計上加速フェーズ
    → ここで初めてP/Lに顕在化
    → FY2027（2027年3月期）業績ジャンプアップが本命シナリオ
```

**結論：「需要爆発は本物だが、P/Lへの反映は6〜12ヶ月後」**
現在の株価¥63,840は「現在の財務数値」でなく「将来のバックログ消化」を先取りすべき水準。市場がこのラグを正しく認識できていないことが、現時点での割安感の源泉。
