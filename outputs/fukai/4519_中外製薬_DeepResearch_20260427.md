### 3-4. 進捗率24%の解釈（最重要）

**結論: Q1進捗率24%は「正常な季節性の範囲内」であり、構造的問題ではない。**

| 確認事項 | 内容 |
|---------|------|
| 文脈 | Q1（1〜3月）の実績が通期予想の何%か = 「四半期進捗率」 |
| 売上進捗率 | 23.9%（売上¥3,217億÷通期予想¥1兆3,450億） |
| Core OP進捗率 | 24.4%（Core OP¥1,633億÷通期予想¥6,700億） |
| 季節性判断 | Q1は薬価改定影響（4月改定前の駆け込み需要消滅）が下期比較で不利。また海外輸出は下期偏重の傾向あり |

**比較: 過去の1Q進捗率（概算）**:
- FY2025 Q1進捗率: 売上¥2,885億÷通期¥1兆2,579億 = 22.9%（参考）
- FY2026 Q1進捗率23.9%は前年比+1.0pt改善

**結論**: 進捗率24%は前年を上回っており、構造問題ではない。むしろQ1好調の証左。問題は「通期ガイダンスを据え置いた（上方修正しなかった）こと」への市場の失望。

## 4. 事業概要・競合優位性

### 4-1. ロシュとの関係

中外製薬はロシュの持分約60%の子会社。関係の核心は3層構造:

1. **日本販売権**: ロシュが開発したバイオ医薬品（ハーセプチン・アバスチン・パージェータ等）の日本独占販売
2. **海外向け製品輸出（最重要）**: ヘムライブラ・ネムルビオ等の中外製薬開発品をロシュ（またはロシュ子会社）へ輸出→売上収益の最大柱
3. **ロイヤルティ収入**: 中外製薬の特許化合物技術に基づくロシュへのライセンス料。FY2026予想¥2,172億(+25.8%)はネムルビオ（ガルデルマ）のロイヤルティ急増が主因

**リスク**: ロシュの戦略変更（医薬品ポートフォリオ見直し）で輸出数量が変動するリスクあり。ただし現在は関係良好でヘムライブラ輸出加速。

### 4-2. 主要3製品のTAM・成長性・競合

**ヘムライブラ（Hemlibra / emicizumab）**
- 適応: 血友病A（第VIII因子の代替二重特異性抗体）
- TAM: グローバル血友病A市場¥8,000〜9,000億（成長率+12%/年）
- 競合: フィッシュバイン（ノボノルディスク）、遺伝子治療薬（ロシュ/スパーク）
- 優位性: 皮下注射・1ヵ月投与可能・既存阻害抗体保有患者にも使用可能
- リスク: 遺伝子治療の台頭（2026〜2027年以降に本格競合の可能性）

**バビースモ（Vabysmo / faricimab）**
- 適応: 加齢黄斑変性（AMD）・糖尿病黄斑浮腫（DME）
- TAM: グローバル抗VEGF市場¥1.5兆（成長率+8%/年）
- 競合: ルセンティス（ノバルティス/一般名廃版方向）、アイリーア（バイエル/再生医療研）、ターリーア（ノバルティス）
- 優位性: 二重特異性抗体（抗VEGF×抗Ang-2）→投与間隔延長（最大16週）が差別化点
- パイプライン展望: 眼内持続放出製剤化や適応拡大（DIE疾患等）の可能性

**ネムルビオ（Nemluvio / nemolizumab）**
- 適応: 結節性痒疹（PN）・アトピー性皮膚炎（AD）
- TAM: グローバル皮膚科抗IL-31/抗IL-4/抗IL-13市場¥5,000億超
- 競合: デュピクセント（サノフィ）、コセンティクス（ノバルティス）
- 優位性: IL-31Rαへの選択的結合→独自の掻痒制御。デュピクセントと作用機序が異なり補完的
- ロイヤルティ: ガルデルマがグローバル（日本・台湾除く）の開発・販売権→ロイヤルティ収入が急増中

### 4-4. 中東・ホルムズ情勢の製薬業への影響

中外製薬への直接影響は**限定的（Neutral〜軽微Negative）**：
- 製薬業のナフサ依存度は化学・石化より低い
- バイオ医薬品（抗体製品）は石油由来原料依存度が低い
- ただし①培養培地・一次容器（バイアル）のガラス・プラスチック原料価格上昇リスク ②輸送コスト上昇が軽微に影響
- 国内薬価改定サイクルは内的要因であり、ホルムズ状況とは独立

**Bearが確率20%にとどまる理由**:
- ヘムライブラはQ1で+26%加速中（後退の兆候なし）
- ロシュとの関係は安定的（ロシュ自身の業績好調が継続）

- OPM49%超の高採算性により、多少の販売減も利益圧迫は限定

- EV = 0.30×¥10,800 + 0.50×¥9,200 + 0.20×¥6,400 = ¥3,240+¥4,600+¥1,280 = **¥9,120**
