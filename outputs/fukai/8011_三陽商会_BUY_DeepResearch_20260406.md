### 2-2. 「PBR1倍超」が正当化される条件 — 理論的フレームワーク

PBRは以下の恒等式で分解できる：

```
PBR = ROE × PER
    = ROE × (1 / (Re - g))   ← Gordon Growth Modelとの接続

または

PBR > 1  ⟺  ROE > Re（株主資本コスト）
```

**三陽商会の場合:**

| 指標 | 数値 |
|------|------|
| ROE（実績） | 8.2% |
| Re（推計、後述） | 約7.5〜8.5% |

> **結論①: ROE（8.2%）≈ Re（推計7.5〜8.5%）**
> 理論上、PBRは1.0倍前後に収束するのが均衡値。現在の1.09倍は**わずかなプレミアム**に留まる。
> 「業績が良いわけでもない」という社長の直感は数値的に正しい。

### 2-6. 【核心判断】PBR1.09倍 — 正当化できるか？

```
判定スコアカード:
┌─────────────────────────────────────────────────┐
│  ROE vs 資本コスト比較:  ROE 8.2% ≈ Re 7.5〜8.5%  → △ 均衡〜やや低い   │
│  業績トレンド:           緩慢改善。「良い」とは言えない   → ▼ 弱い      │
│  事業再構築進捗:         生存完了、成長未達             → △ 中立        │
│  株主還元:               低水準だが改善方向             → △ 中立        │
│  成長期待:               売上高成長率2〜4%止まり        → ▼ 弱い        │
│  東証PBR圧力:            政策的下支え効果              → ○ ポジティブ   │
│  バランスシート健全性:    自己資本比率は一定水準維持     → ○ ポジティブ   │
│  DCF理論株価との乖離:     後述（割高方向の可能性）       → ▼ 弱い        │
└─────────────────────────────────────────────────┘
```

**深井の核心判断:**
> **「PBR1.09倍は、現状の業績・還元水準から見て、ギリギリ正当化できるかどうかの境界線上にある」**
>
> ROEが資本コストと拮抗している状況でPBR1.0倍は理論的均衡値であり、
> 1.09倍のプレミアムは「将来の改善期待」と「東証圧力による政策的プレミアム」で
> 説明される。しかし、その期待が実現しない場合はPBR1倍割れ（株価 < ¥3,724）への
> 回帰リスクがある。
>
> **社長の直感「業績良くないのに買う理由があるか」に対する深井の答え：**
> **現時点では積極的な買い根拠は薄い。** 割安感はなく、むしろ緩やかな割高感がある。

## 4. DCFバリュエーション

### 4-1. WACC計算の前提

#### β（ベータ）設定

```
業種: 日本アパレル・小売
日経平均・TOPIXに対するβ: 0.85〜1.00（推計）
※ アパレル業界は景気感応度は中程度。ただし構造変化リスクがある。
採用β: 0.90
```

#### ERP（株式リスクプレミアム）設定

```
日本株ERP: 5.5%（Damodaran参考値・日本市場標準）
```

#### 株主資本コスト（Re）計算

```
Re = Rf + β × ERP
   = 2.380% + 0.90 × 5.5%
   = 2.380% + 4.950%
   = 7.330%
```

#### 負債コスト・資本構成

```
有利子負債比率（推計）: 約15〜20%（自己資本比率40〜50%のため）
負債コスト（Rd）: 約0.8〜1.2%（低金利環境・大企業格付け）
実効税率（Tc）: 30%
D/(D+E): 15%
E/(D+E): 85%
```

#### WACC計算

```
WACC = E/(D+E) × Re + D/(D+E) × Rd × (1 - Tc)
     = 0.85 × 7.330% + 0.15 × 1.0% × (1 - 0.30)
     = 6.231% + 0.105%
     = 6.336%

採用WACC: 6.3%（四捨五入）
```
