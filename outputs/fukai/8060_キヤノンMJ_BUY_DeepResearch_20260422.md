### 3.2 Q1 決算の重要ポイント

**なぜQ1 OPが+40.7%という異例の伸びを示したか？（3つの仮説）**

```
仮説①【構造的：ITソリューション高成長】 → 確度：高
  → 企業のAI投資加速（AIワークステーション・GPU搭載サーバー需要）
  → ソフト・サービス比率拡大でミックス改善
  → OP margin 7.86% → 10.77%の2.91pp拡大はソフト混入比率増加と整合

仮説②【一時的：前年同期コスト高の反動】 → 確度：中
  → Q1 FY2025に発生した特殊コスト（物流・人件費等）の剥落
  → 一時的であれば、Q2以降に利益率の正常化（下押し）が生じる可能性
  → ※この仮説が正しければBear caseに転換

仮説③【季節性：法人IT投資の期初集中】 → 確度：中
  → 日本企業の期初（4月会計年度始まり）に合わせ1Q（1-3月）に商談完了
  → Canon MJは3月末に法人向け大口契約が集中する傾向
  → この場合、Q2以降は通常利益率に戻る（全年保守ガイダンスと整合）
```

**重要判断**: 仮説①が支配的であれば年間通期ガイダンスは大幅上振れ確実。仮説②③が支配的なら¥600億ガイダンスで折り合う。現状データからは①③のハイブリッドが最有力。

### 5.2 市場環境（IT投資トレンド）

**フォローウィンド（追い風要因）**:
1. **企業AI投資の加速**: GPU搭載ワークステーション・AIサーバー需要が急拡大。Canon MJはCanon AI撮影ソリューション・AIワークフロー製品を展開

4. **セキュリティ需要**: サイバー攻撃増加でエンドポイントセキュリティ・ゼロトラスト需要旺盛

1. **国内IT予算縮小リスク**: 景気後退局面ではIT投資が後回しになるリスク
