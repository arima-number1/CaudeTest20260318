### 2-2. US事業の損益分岐点試算（社長指示必須項目）

#### 前提条件の整理

メルカリUSの公開情報ベースの推定値を使用する。

```
【US事業コスト構造推定（2025年度実績ベース）】
・固定費（人件費・オフィス・開発費）：推定 ¥150〜200億/年
・変動費（マーケティング・決済手数料）：GMVの推定 5〜7%
・売上（Take Revenue）：GMV × Take Rate（推定8〜10%）
・現状GMV（US）：推定 $5〜7億USD相当（¥750〜1,050億 ＠150円換算）
・現状Take Revenue：推定 ¥60〜100億/年
・現状営業赤字：推定 ▲¥100〜150億/年（連結決算の開示ベース）
```

#### 損益分岐点試算

**損益分岐点（BEP）の定義式：**

```
BEP GMV = 固定費 ÷（Take Rate − 変動費率）
```

| 前提 | 値 |
|------|-----|
| 固定費（年間） | ¥175億（保守的中値） |
| Take Rate（US） | 9.0% |
| 変動費率 | 5.5% |
| **BEP GMV** | **¥175億 ÷ (9.0% − 5.5%) = ¥175億 ÷ 3.5% = 約¥5,000億** |

**現状US GMV（推定）：¥750〜1,050億 → BEPまでの距離：約 4〜4.5倍のGMV拡大が必要**

#### US GMV成長シナリオと黒字化タイムライン

| シナリオ | 年間GMV成長率 | BEP達成時期 | 必要期間 |
|---------|------------|------------|---------|
| 弱気 | +15%/年 | 2031〜2032年 | 約6〜7年 |
| 基本 | +30%/年 | 2029〜2030年 | 約4〜5年 |
| 強気 | +50%/年 | 2027〜2028年 | 約2〜3年 |

```
【US黒字化の感応度】
・固定費を¥130億まで削減（構造改革）した場合：BEP GMV ≒ ¥3,700億（▲26%短縮）
・Take Rateを10.5%に改善した場合：BEP GMV ≒ ¥3,500億（▲30%短縮）
・両施策組み合わせ（Bull想定）：BEP GMV ≒ ¥2,600億（基本比▲48%短縮）
　→ 黒字化タイムラインが 2026〜2027年度に前倒し可能
```

**深井コメント：US事業の鍵は「固定費の規律ある削減」×「Take Rate改善」の組み合わせ。単純なGMV拡大だけでは遅すぎる。2027年度黒字化が株価の次のレイティングチェンジトリガーとなる。**

## 💹 4. DCFバリュエーション

### 4-1. WACC算出前提

#### β・ERP・株主資本コスト

```
【WACC前提パラメータ】
リスクフリーレート（Rf）  : 2.380%（JGB10Y、社長確認値）
エクイティリスクプレミアム : 6.0%（日本市場標準、Damodaran 2025近似）
β（メルカリ）            : 1.35（成長株・IT・高ボラ銘柄として設定）
                           ※ 東証IT高成長株の典型β範囲：1.2〜1.5

株主資本コスト（Re）       :
  Re = 2.380% + 1.35 × 6.0%
     = 2.380% + 8.10%
     = 10.48%

負債コスト（Rd）          : 1.0%（低金利・優良企業水準）
実効税率（Tc）            : 30%（日本法人税標準）
負債比率 D/(D+E)          : 15%（資本構成：成長企業・低レバレッジ想定）
株式比率 E/(D+E)          : 85%

WACC = [0.85 × 10.48%] + [0.15 × 1.0% × (1 - 30%)]
     = 8.908% + 0.105%
     = 9.013%
     ≈ 9.0%（計算上使用）
```

#### ターミナル成長率

```
g（永続成長率） = 2.5%（日本GDP成長+リユース市場構造成長）
ターミナルバリュー計算式：TV = FCF(n+1) ÷ (WACC - g) = FCF × (1+g) ÷ (9.0% - 2.5%)
```
