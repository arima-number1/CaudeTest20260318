### 4-A. 前提条件（全項目明記）

### 4-B. DCF計算詳細（基本ケース）

**WACC精緻計算：**
```
Re = 2.423% + 1.10 × 6.0% = 2.423% + 6.600% = 9.023%
Rd × (1-Tc) = 1.5% × (1-0.30) = 1.05%

WACC = (0.90 × 9.023%) + (0.10 × 1.05%)
     = 8.121% + 0.105%
     = 8.226% ≒ 8.23%
```

**5年FCFプロジェクション（基本ケース）：**

| 年度 | FCF推計（億円） | 割引係数（@8.23%） | 現在価値（億円） |
|------|--------------|-----------------|--------------|
| FY2026 | 9.0 | 0.9240 | 8.32 |
| FY2027 | 9.5 | 0.8538 | 8.11 |
| FY2028 | 10.5 | 0.7888 | 8.28 |
| FY2029 | 11.5 | 0.7288 | 8.38 |
| FY2030 | 12.5 | 0.6733 | 8.42 |
| **FCF現在価値合計** | | | **41.51** |

**ターミナルバリュー（TV）計算：**
```
TV = FCF2030 × (1+g) / (WACC - g)
   = 12.5億円 × 1.015 / (0.0823 - 0.015)
   = 12.6875 / 0.0673
   = 188.5億円

TV現在価値 = 188.5 × 0.6733 = 126.9億円
```

**企業価値（EV）：**
```
EV = FCF現在価値合計 + TV現在価値
   = 41.5億円 + 126.9億円
   = 168.4億円

ネットデット（推定）≒ ▲10〜+5億円（ほぼ無借金、現預金考慮）
→ 株式価値 ≒ 168.4 + 10(純現金) = 178.4億円（中央値）

発行済株式数（推定）= 950万株
理論株価（DCFベース）= 178.4億円 ÷ 950万株 ≒ ¥1,878/株
```

> ⚠️ **深井補足注記：** 純粋DCFでは¥1,878と算出されるが、これはFCFベースのみの評価。SECカーボンは**資産価値（BPS¥3,927）が極めて厚い**ため、DCFと純資産価値（NAV）のブレンド評価が適切。

### 4-C. NAV（純資産価値）ブレンド評価

```
【評価ウェイト】
DCF（事業価値）   : 40%重み → ¥1,878
PBR基準（NAV）   : 40%重み → BPS¥3,927 × 目標PBR0.75倍 = ¥2,945
EPS倍率（PER法） : 20%重み → EPS¥191 × 目標PER14倍 = ¥2,674

【ブレンド理論株価（基本ケース）】
= (¥1,878 × 0.40) + (¥2,945 × 0.40) + (¥2,674 × 0.20)
= ¥751.2 + ¥1,178.0 + ¥534.8
= ¥2,464 〜（保守的安全域考慮）

【安全域（Margin of Safety）+20%バッファ適用後】
= ¥2,464 × 1.20 = ¥2,957 ≒ ¥2,980（基本ケース中央値）
```

**理論株価レンジ：¥2,820〜¥3,150（基本ケース中央値：¥2,980）**
