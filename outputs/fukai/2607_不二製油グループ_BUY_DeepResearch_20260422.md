## 1. 企業概要・事業分析

### 1-1. 会社概要

**不二製油グループ本社（2607.T）** は、チョコレート用油脂・製菓素材・植物性食品素材のグローバルメーカー。売上高は連結で★約3,500〜3,800億円規模（FY2025推計）。国内外に40以上の生産拠点を持ち、B2Bの食品原料メーカーとして食品・菓子・外食チェーンに素材供給する。

### 1-2. セグメント構成（推計）

| セグメント | 売上構成比（推計） | 主力製品 | 成長ドライバー |
|-----------|----------------|---------|--------------|
| チョコレート・製菓用油脂 | ★約50〜55% | CBE（カカオバター代替脂）、テンパリング型チョコ | 高カカオ相場→CBE代替需要、Blommer統合効果 |
| 植物性食品（大豆・代替タンパク） | ★約20〜25% | 大豆たん白、植物性チーズ素材、DEALI | 代替タンパクブーム、DEALI新ブランド |
| 乳化油脂・業務用マーガリン | ★約15〜20% | 業務用マーガリン、フィリング用油脂 | 外食チェーン向け安定需要 |
| その他・海外 | ★残余 | Blommer（北米）、東南アジア | Blommer M&A効果 |

### 1-3. 直近カタリスト（投資判断の核心）

**① DEALI（植物ミルク新ブランド）— 本日2026-04-22 HPローンチ**
- 不二製油グループが自社ブランドとして植物性ミルク市場に直接参入するB2C/B2B混合展開
- 従来のB2B素材供給からブランド価値創出への戦略転換を象徴
- 短期的な売上インパクトは軽微だが、**成長ストーリーの「見える化」**という点で株価にはポジティブなセンチメント効果

**② CBE価格リスク — IRが明言した「最大の毒」**
- カカオバター原料価格の高騰局面でCBE（代替品）需要が急増→CBE製品の価格が上昇した局面が続いた
- しかしIR担当・千野氏が「**今後価格が下がるリスクはある**」と明言
- 短期契約中心のため、CBE市況下落が即座に売上・利益に反映されるリスク
- これが現在、PER33.9倍の高バリュエーションの最大の下方修正要因

**③ Blommer事業（北米）コンパウンド伸長**
- 2019年に約600億円で買収した北米最大のチョコレート原料メーカー
- 前期比でコンパウンド（複利的）に伸長継続とIRが確認
- 北米大手食品メーカーとの深耕が進んでいる

**④ 5/12決算 × 5/13説明会**
- 次の情報更新ポイント。CBE価格の実績影響、Blommer通期着地、DEALI販管費先行の影響を確認
- **この決算を通過するまでは「ウォッチリスト維持」が基本スタンス**

## 4. DCFバリュエーション

### 4-1. 前提条件（全項目明記）

| パラメータ | 採用値 | 根拠・出典 |
|-----------|-------|-----------|
| **リスクフリーレート（Rf）** | **2.392%** | 日本10年国債（JGB10Y）財務省日次 R8.4.21 ← 絶対使用値 |
| **エクイティリスクプレミアム（ERP）** | **5.5%** | 日本株式市場標準（Damodaran 2025年度推計：日本5.0〜6.0%の中央値） |
| **ベータ（β）** | **0.75** | 不二製油は食品セクター（ディフェンシブ）+海外事業拡大（やや上乗せ）。食品セクター平均β≒0.65〜0.80の中央値 |
| **株主資本コスト（Re）** | **Re = 2.392% + 0.75 × 5.5% = 6.517%** | 上記計算 |
| **負債コスト（Rd）** | **1.5%** | 日本大手食品メーカーの借入金利水準（低金利環境下） |
| **実効税率（Tc）** | **30%** | 日本法人実効税率標準 |
| **資本構成（E/（D+E））** | **E: 65% / D: 35%** | 食品セクター平均的レバレッジ（BPS・時価総額から推計） |
| **WACC** | 計算式参照 | 下記計算 |
| **永続成長率（g）** | **1.0%** | 日本食品セクター長期成長率（インフレ0.5〜1.5%の中央値、植物性食品の成長を勘案） |

### 4-2. WACC計算

```
Re = Rf + β × ERP
   = 2.392% + 0.75 × 5.5%
   = 2.392% + 4.125%
   = 6.517%

WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.65 × 6.517%] + [0.35 × 1.5% × (1-0.30)]
     = 4.236% + 0.368%
     = 4.604%

ベースケースWACC = 4.60%（小数第2位四捨五入）
```

### 4-3. DCFモデル（5年予測 + 永続価値）

**ベースケース前提**:
- NOPAT（税引後営業利益）成長率: 年5%（FY2026-FY2028）→ 年3%（FY2029-FY2030）
- FCF = NOPAT × (1 - 再投資率)。再投資率30%
- ターミナル成長率 g = 1.0%

| 年度 | FCF推計（億円）★ | 割引係数（WACC=4.60%） | PV（億円） |
|------|----------------|----------------------|-----------|
| FY2026 | 70 | 0.956 | 66.9 |
| FY2027 | 73 | 0.914 | 66.7 |
| FY2028 | 77 | 0.874 | 67.3 |
| FY2029 | 79 | 0.836 | 66.0 |
| FY2030 | 82 | 0.799 | 65.5 |
| **予測期間PV合計** | — | — | **332.4** |

**ターミナルバリュー（TV）計算**:
```
TV = FCF_2030 × (1+g) / (WACC - g)
   = 82 × 1.01 / (0.046 - 0.010)
   = 82.82 / 0.036
   = 2,300.6億円

TV現在価値 = 2,300.6 × 0.799 = 1,838.2億円
```

**企業価値（EV）計算**:
```
EV = 予測期間PV + TV現在価値
   = 332.4 + 1,838.2
   = 2,170.6億円

発行済み株式数（推計） = 純利益★¥85億円 ÷ EPS¥102 ≒ 8,333万株
（別途確認：時価総額 = ¥3,447 × 推計株数から逆算）

時価総額推計 = EV - ネット有利子負債（★推計▲200億円）
             = 2,170.6 - 200
             = 1,970.6億円

理論株価 = 1,970.6億円 ÷ 8,333万株
         = ¥2,365〜... 
```
