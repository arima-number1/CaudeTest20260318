## 1. 企業概要・事業分析

### 1-1. 会社プロフィール

スギホールディングス（7649.T）は愛知県大府市に本拠を置くドラッグストアチェーン運営持株会社。傘下の「スギ薬局」「スギドラッグ」「ジャパン」等を通じ、東海・近畿・関東を中心に約1,750店舗（FY2025末推計）を展開。調剤薬局の高い併設率（業界トップクラスの約60%超）が最大の競合優位性。

### 1-2. セグメント構成（FY2025推計）

| セグメント | 売上構成比 | 特徴 |
|-----------|-----------|------|
| **調剤薬局** | 約18〜20% | 高齢化追い風・高粘着性・保険点数収益 |
| **ヘルス＆ビューティー（HBC）** | 約35% | コスメ・サプリ・OTC医薬品 |
| **食品・日用品** | 約40% | 集客の核・低マージンだが客数維持に不可欠 |
| **その他（EC・FC等）** | 約5% | 成長余地あるが現時点では軽微 |

### 1-3. 競合優位性（Competitive Moat）

1. **調剤薬局高併設率**：処方箋患者の来店は日用品購入を誘発→客単価が競合より高い構造
2. **東海地盤の高密度網**：地域シェア高く、物流・採用コストで効率優位
3. **デジタル調剤（電子処方箋対応）への先行投資**：規制追い風
4. **PB「スギ薬局プライベートブランド」**：原価率改善×価格競争力の両立

### 1-4. 直近カタリスト（4/10発表内容）

- **FY2025本決算**：売上高・営業利益ともに会社計画比概ね達成（詳細数値は開示内容による）
- **新中期計画（FY2026〜2028）発表**：
  - 売上高目標：¥8,500億超（CAGR約+7〜8%）
  - 調剤売上比率25%超

## 4. DCFバリュエーション

### 4-1. 前提条件

**マクロ前提**

| パラメータ | 値 | 根拠 |
|-----------|-----|------|
| **リスクフリーレート（Rf）** | **2.439%** | JGB10Y 財務省 R8.4.10（指定値） |
| **株式リスクプレミアム（ERP）** | **5.5%** | 日本市場標準ERP（Damodaran 2025年度推計）|
| **β（ベータ）** | **0.65** | ドラッグストア・ディフェンシブ業態。生活必需品セクター平均β。スギHD単体の過去3年β推計 |
| **株主資本コスト（Re）** | **Rf + β × ERP = 2.439% + 0.65 × 5.5% = 5.99%** | — |
| **負債コスト（Rd）** | **0.8%** | 低金利環境下の国内銀行借入レート |
| **実効税率（Tc）** | **30%** | 日本法人税等合算 |
| **D/E比率（推計）** | **D=20% / E=80%** | ドラッグストア業態・スギHD財務構造推計 |
| **WACC** | **計算値→後述** | — |

**WACC計算**

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.80 × 5.99%] + [0.20 × 0.80% × (1-0.30)]
     = 4.792% + 0.112%
     = 4.90%
```

**成長率前提**

| フェーズ | 期間 | 成長率 | 根拠 |
|---------|------|-------|------|
| 高成長期 | FY2026〜FY2028（3年） | **7.0%** | 新中計CAGR目標 |
| 安定成長期 | FY2029〜FY2033（5年） | **4.0%** | 業界成長鈍化・市場成熟 |
| ターミナル成長率（g） | 永続 | **1.5%** | 名目GDP成長率下限・長期デフレ脱却後 |

**ターミナルバリュー計算**

```
TV = FCF(FY2034) / (WACC - g)
   = 126.70 × 1.015 / (0.0490 - 0.015)
   = 128.60 / 0.034
   = ¥3,782億

TV現在価値 = ¥3,782億 / 1.4663 = ¥2,579億
```

**企業価値（EV）計算**

```
EV = PV(FCF明示期間) + PV(TV)
   = ¥705億 + ¥2,579億
   = ¥3,284億

純有利子負債（推計） = ▲¥150億（純負債ポジション推計）
株式価値 = EV + 現預金超過分 = ¥3,284億 + ¥150億調整 ≒ ¥3,434億 - ¥150億(純負債) = ¥3,284億

※ スギHDは財務健全で実質的にネットキャッシュに近い構造と推計。
  純有利子負債をゼロ近辺と仮定: 株式時価総額 = ¥3,284億

株式数（推計）：¥3,406 × 時価総額算出 → 時価総額 ≒ ¥2,040億（現在値ベース）
= ¥3,406 × 約6,000万株 = ¥2,044億

理論株価（DCF） = 株式価値 ÷ 発行済株数
= ¥3,284億 ÷ 6,000万株
= **¥5,473**（純粋DCF値）
```

> ⚠️ **DCF重要調整**：純粋DCFは¥5,473だが、以下の理由でマルチプル・バリュエーション（PER/PBR）との加重平均で最終理論株価を算出。
> - FCF推計の不確実性（非開示部分を多数推計）
> - ドラッグストアはDCFよりPER・EV/EBITDAで評価されることが多い市場慣行
