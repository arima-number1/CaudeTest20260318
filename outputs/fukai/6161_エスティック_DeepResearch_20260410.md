## 3. 低PBR×高自己資本の構造分析

### 3-1. PBR理論値vs実態

```
WACC計算:
  Re = JGB10Y 2.425% + β0.75 × ERP 5.5% = 6.55%

理論PBR = ROE / Re = 9.7% / 6.55% = 1.48倍
現在PBR = 0.957倍

乖離幅: 1.48x - 0.96x = -0.52倍分の「低評価ディスカウント」
割安根拠: ROEがWACCを上回っているにもかかわらずBPS割れ
```

**なぜ割安放置されているか（3つの理由）**:

① **スタンダード市場・超小型株**: 時価総額¥120億、ADV¥12.3M/日は機関投資家の投資対象ギリギリ以下。アナリストカバレッジがなく、情報伝達が遅い  
② **Q3進捗の見た目の悪さ**: 3Q累計OP▲21.5%というヘッドラインが印象を悪化させた。季節性の説明が不足  
③ **東証スタンダード市場のPBR改善プレッシャー**: プライム市場ほど強くないが、2026年度から全市場に拡大。認知度の高まりにより「PBR1倍割れ企業」への圧力が波及中

### 3-2. 自己資本86.1%が持つ意味

高い自己資本比率は通常は「過剰資本」として低ROEの理由にもなるが、エスティックの場合は**無借金×高利益率の組み合わせ**であり、以下の潜在価値がある：

- 現金¥28.7億（時価総額比23.9%）が丸ごと「実質ディスカウント」として株価に含まれている
- キャッシュリッチ企業への自社株買い・増配圧力は東証要請の文脈でも正当化される
- 有事（中東情勢悪化・景気後退）でも財務破綻リスクは皆無

## 5. 株主還元分析

### 5-1. 配当政策

```
現在:
  配当 ¥29/株（FY2025: ¥28→FY2026: ¥29、¥1増配）
  配当性向: 28.1%（EPS¥103ベース）
  配当利回り: 2.84%

増配余地:
  配当性向28.1% → 業界平均30〜40%まで引き上げ余地
  EPS¥103 × 35% = ¥36/株 → 利回り3.53%
  EPS¥103 × 40% = ¥41/株 → 利回り4.02%
```

**東証PBR改善要請の文脈で、増配は最も実行しやすいカード**。現金¥28.7億を持ちながら年間配当総額は¥3.4億（11.77M株 × ¥29）にすぎない。増配余力は十分。

### 5-2. 自社株買い可能性

現時点で自社株買いプログラムの記録なし。ただし：

- 東証プレッシャー + 低流動性株の需給改善効果として**最も効果的な株主還元手段**

4/28決算説明で「自社株買い検討」の一言があれば短期で大幅高の可能性あり（ADV少なく需給インパクト大）。
