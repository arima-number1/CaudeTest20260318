## 4. Fair Value算出と感応度分析

### 4-1. WACC計算

```
Rf（無リスク金利）  = 2.380%（固定・変更不可）
β（海運セクター）   = 1.40（市況敏感・有事プレミアム含む）
ERP（株式リスクプレミアム） = 5.5%
WACC = 2.380% + 1.40 × 5.5% = 2.380% + 7.700% = 10.08%
```

注: β = 1.40は日本海運セクターの過去5年平均（1.2〜1.5）の中央値。ホルムズ危機下では実効β上昇。

### 4-2. PERアプローチ

```
FY2027E EPS（深井）    = 858円
トラフPER（海運8x）    → FV = 858 × 8.0 = 6,864円
ノーマルPER（10x）     → FV = 858 × 10.0 = 8,580円
ピークPER（12x）       → FV = 858 × 12.0 = 10,296円
使用PER（10x）         → FV基準 = 8,580円
```

### 4-3. DCF補完

```
FY2026〜FY2030 コア営業CF
  FY2026: 2,200億円（会社目標950億+ONE配当他）
  FY2027: 2,700億円
  FY2028: 2,000億円
  FY2029〜FY2030: 1,800億円（正常化後）
ターミナルバリュー成長率: g = 1.5%
WACC: 10.08%

DCF概算FV（発行株式数約3.14億株前提）:
純利益ベース5年DCF + TV = 概算¥7,800〜¥8,500
```

### 4-4. PBR補完

```
BPS（現在）  = 7,870円
目標PBR      = 1.05倍（Elliott圧力でPBR1倍回復シナリオ）
→ FV = 7,870 × 1.05 = 8,264円
```

### 4-5. 総合Fair Value
