# 1. 企業概要・事業分析

## 1.1 会社プロフィール

住友化学は1913年創業（住友グループ中核）、売上高約2.4兆円規模の日本最大級の総合化学メーカーのひとつ。医薬品（住友ファーマ連結）、石化、精密化学、農業、電池材料など多岐にわたるポートフォリオを展開。本社大阪。

## 1.2 セグメント構成（概算）

| セグメント | 売上構成比（概算） | 主要製品・サービス | 収益性（EBIT Margin目安） |
|-----------|-----------------|-----------------|------------------------|
| **石油化学** | 約30% | エチレン、プロピレン、フェノール | 低〜中（スプレッド依存、3〜6%） |
| **エネルギー・機能材料** | 約20% | **リチウムイオン電池用セパレータ（LiBS）**、光学フィルム | 中〜高（8〜14%） |
| **情報電子化学** | 約15% | **半導体フォトレジスト**、CMP材、液晶材料 | 高（12〜18%） |
| **健康・農業関連** | 約20% | 農薬原体（メソミル等）、動物用医薬 | 中（7〜12%） |
| **医薬品（住友ファーマ）** | 約15% | ラツーダ（特許切れ影響）、オルゴビクス等 | 変動大（過去赤字要因） |

### セグメント特記事項

**✅ LiBS（リチウムイオン電池セパレータ）**
- 住友化学の「Licipo」ブランド：乾式・湿式両方式を保有
- EV市場の回復・中国電池メーカーへの供給再拡大が2025年後半から顕在化
- 旭化成（ハイポア）・東レと並ぶ国内3強の一角
- **グローバルシェア推定：約10〜15%（湿式）**

**✅ 半導体フォトレジスト**
- KrF・ArF・EUVレジストを全てラインナップ（JSR・信越化学に次ぐ国内有力サプライヤー）
- 台湾TSMC・韓国SKハイニックスへの供給実績
- 先端ロジック（2nm以降）向けEUVレジスト開発投資継続中
- 半導体設備投資サイクル回復（2025年〜）の直接受益者

**⚠️ 医薬セグメント（住友ファーマ）**
- ラツーダ特許切れ（2023年）による米国売上急減が2023〜2024年の最大ドラッグ要因
- 2025年以降：構造改革（人員削減・パイプライン集約）が進捗、損失縮小フェーズへ
- **2000億円上方修正の主因ではないが、下振れ要因剥落が全社PLに貢献**

# 4. DCFバリュエーション

## 4.1 前提条件（Parameter設定）

### リスクフリーレート（絶対厳守）

```
Rf = 2.380%
（日本10年国債 JGB10Y、2026-04-06 05:17 JST取得、社長確認値）
出所: https://jp.investing.com/rates-bonds/japan-10-year-bond-yield-streaming-chart
```

### その他パラメータ

| パラメータ | 設定値 | 根拠 |
|-----------|------|------|
| **Rf（リスクフリーレート）** | **2.380%** | JGB10Y（社長確認値、厳守） |
| **ERP（株式リスクプレミアム）** | **5.50%** | 日本株ERP（Damodaran 2025年推計：5.3〜5.7%の中央値） |
| **β（ベータ）** | **1.25** | 東証プライム化学セクター平均1.1〜1.4。住友化学は医薬・素材混在で高め設定 |
| **Re（株主資本コスト）** | **2.380% + 1.25 × 5.50% = 9.255%** | CAPM計算値 |
| **Rd（負債コスト）** | **1.80%** | 住友化学の社債・借入金利実績（低格付け化学大手水準） |
| **Tc（実効税率）** | **30.0%** | 日本法人税等（実効税率平均） |
| **E/(D+E)（株式比率）** | **55%** | BS推計（純資産1.3兆円/総資産3.6兆円ベース） |
| **D/(D+E)（負債比率）** | **45%** | 同上 |
| **WACC** | **5.648%** | 下記計算 |
| **永続成長率（g）** | **1.00%** | 日本名目GDP成長率（1〜1.5%）の保守値 |

### WACC計算

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.55 × 9.255%] + [0.45 × 1.80% × (1 - 0.30)]
     = 5.090% + 0.567%
     = 5.657% ≈ 5.66%
```

> 計算精度を考慮し **WACC = 5.66%** を採用。

## 4.3 DCF計算（基本シナリオ）

### 予測期間PV

| 年度 | FCFF（億円） | 割引係数（WACC=5.66%） | PV（億円） |
|------|------------|---------------------|---------|
| FY2025E | 800 | 0.9464 | 757 |
| FY2026E | 1,010 | 0.8957 | 905 |
| FY2027E | 1,255 | 0.8477 | 1,064 |
| FY2028E | 1,337 | 0.8023 | 1,073 |
| FY2029E | 1,411 | 0.7593 | 1,071 |
| **合計PV** | | | **4,870** |

### ターミナルバリュー（TV）

```
TV = FCFF_FY2029 × (1 + g) / (WACC - g)
   = 1,411 × (1 + 0.010) / (0.0566 - 0.010)
   = 1,425 / 0.0466
   = 30,579億円

TV現在価値 = 30,579 × 0.7593 = 23,219億円
```

### Enterprise Value → 株式価値

```
EV = 予測期間PV + TV現在価値
   = 4,870 + 23,219
   = 28,089億円

純有利子負債（ネットデット）= 約7,000〜8,000億円（推定）
少数株主持分等控除    = 約1,000〜1,500億円

株式価値 = 28,089 - 7,500 - 1,200 = 19,389億円

発行済株式数（概算）= 約39億〜40億株

理論株価（基本）= 19,389億円 ÷ 39.5億株 ≈ 491円
```
