> **📌 発行済株式数の推計根拠:**
> BPS ¥3,728 × 株式数 = 純資産額。
> ROE 5.4% → 純利益/純資産 = 5.4%
> EPS ¥172 → 純利益/株式数 = ¥172

> EPS ¥172 / ROE 5.4% ≈ ¥3,185 と BPS ¥3,728の差はROEの分母（期首/期末）の違いで説明可能
> 純資産総額 = BPS × 株式数 → 同業規模比較で¥5〜8億の純資産を持つ場合: **株式数 = ¥7億/¥3,728 ≈ 187,800株**

> EPS¥172でPER12.5倍 = 株価¥2,150 ✅
> 純利益総額 = 株価¥2,150 × 株式数 / PER12.5 = 株式数 × ¥172
> PBR0.58倍: 株価¥2,150 / BPS¥3,728 = 0.577 ✅
> この整合性から: **株式数は市場全体で整合している**
> 仮に時価総額を¥40億と仮定 → 株式数 = ¥40億/¥2,150 ≈ 186万株
> → 純資産 = ¥3,728 × 186万株 = **約¥69億**の純資産
