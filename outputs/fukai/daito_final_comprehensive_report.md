### サブリースモデルの構造と持続可能性

大東建託の収益モデルは「土地オーナーへのアパート建設請負 → 一括借上（サブリース）→ 賃貸管理」という垂直統合型の3フェーズ構造に加え、近年は **Build Set（自社土地取得・建設・売却）** という第4フェーズが加わりつつある。

**新フェーズ — Build Set:** 大東建託が土地を自ら購入し、賃貸住宅を建設した後に既存アパートオーナーへ売却するスキーム。通常の建設請負（土地オーナー起点）とは異なり、同社が起点となることで受注の安定性を高める効果がある。2026年3月単月で「工事関連別途30億円」として計上されており、フロー受注の実態把握にこの概念の適用が必要。

**持続可能性の総評:**
- 管理戸数129万戸のプール効果で入居率97.98%を維持する構造的優位性は依然として強固。
- Build Set事業の伸長は、地主依存型の従来モデルを補完するものとして評価できる。
- 2026年3月末入居率が前月比+28bp上昇したことは、新規供給減・退去減という外部環境の追い風も含む。次期（4月以降）の季節調整後の動向確認が必要。

## Section G: DCF理論株価の算出

### 前提条件（WACC算出）

```
【リスクフリーレート】
  JGB 10Y（日本10年国債利回り）: 1.5%（2026年4月時点）

【エクイティリスクプレミアム（ERP）】
  5.5%（Damodaran日本市場推計ベース）

【ベータ値（β）】
  0.85（住宅大手・ディフェンシブ特性を反映し市場平均より低め）

【Cost of Equity（Ke）】
  Ke = 1.5% + 5.5% × 0.85 = 6.175% ≈ 6.2%

【負債コスト（税引後）】
  Kd(after-tax) = 0.5%（有利子負債倍率0.26倍の低水準を反映）

【資本構成（時価ベース）】
  E/V = 80%、D/V = 20%

【WACC】
  WACC = 6.2% × 80% + 0.5% × 20% = 5.06% ≈ 5.0%
```
