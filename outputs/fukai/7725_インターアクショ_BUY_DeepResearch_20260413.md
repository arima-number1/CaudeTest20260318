**シェア奪取の意味するところ（深井分析）:**

| 論点 | 深井の見解 |

| **なぜインターアクションが選ばれたか** | 光源均一性・検査精度の優位性。iPhone 17系以降のカメラ画素数急増（200MP超要求）に対応できる精度が内製光源技術にある |

## 4. DCFバリュエーション

### 4-1. 前提条件

```
【絶対厳守パラメーター】
リスクフリーレート (Rf):  2.439%（日本10年国債 2026/4/10）
ERP（株式リスクプレミアム）: 6.5%（日本市場標準値）
β（ベータ）:              1.40（小型精密機器メーカー・半導体装置関連
                                 の過去3年実績から推計）
株主資本コスト (Re):
  Re = Rf + β × ERP
  Re = 2.439% + 1.40 × 6.5%
  Re = 2.439% + 9.10%
  Re = 11.54%

資本構成: 無借金経営に近い小型製造業として推計
  E/(D+E) = 90%、D/(D+E) = 10%
  負債コスト (Rd): 1.5%（短期借入金主体）
  実効税率 (Tc): 30%

WACC = [0.90 × 11.54%] + [0.10 × 1.5% × (1-0.30)]
WACC = 10.386% + 0.105%
WACC = 10.49%（≒10.5%）

ターミナル成長率 (g): 2.0%（国内精密機器製造業の長期成長率）
```
