### 手法①：シングルステージDDM（ベア基準）

```
D₁ = ¥180 × 1.03 = ¥185.4
g  = 3.0%（定常成長・保守的）
r  = 8.098%

FV = ¥185.4 / (0.08098 - 0.030) = ¥185.4 / 0.05098 = ¥3,636
```
→ **FV①：¥3,636**（原分析の¥3,627と整合。ただしg=3%は過度に保守的）

---

### 手法②：二段階DDM（ベース・中期計画反映）

```
Stage 1（FY2026〜2028、3年間）：g = 7.7%（中期計画売上CAGR準拠）
  D₁ = ¥180 × 1.077 = ¥193.9
  D₂ = ¥193.9 × 1.077 = ¥208.8
  D₃ = ¥208.8 × 1.077 = ¥224.9

Stage 2（FY2029以降）：g = 3.0%（定常成長）
  TV₃ = ¥224.9 × 1.03 / (0.08098 - 0.030) = ¥231.6 / 0.05098 = ¥4,543

PV計算（r = 8.098%）：
  PV(D₁) = ¥193.9 / 1.081  = ¥179.4
  PV(D₂) = ¥208.8 / 1.168  = ¥178.8
  PV(D₃) = ¥224.9 / 1.263  = ¥178.1
  PV(TV₃)= ¥4,543 / 1.263  = ¥3,597

FV = 179.4 + 178.8 + 178.1 + 3,597 = ¥4,133
```
→ **FV②：¥4,133**

### 手法③：FY2028 P/E逆算（中期計画到達時割引現在価値）

```
FY2028 OP目標    ¥5,500M

```
→ **FV③：¥4,570**

### 手法④：PBR/ROEジャスティファイド（NAVFシナリオ感応度）

```
正当化PBR = (ROE - g) / (r - g)

ROE 5.7%（現状）: (0.057-0.030) / (0.08098-0.030) = 0.027/0.051 = 0.529x → FV = ¥2,009（参考値のみ）
ROE 8.0%（目標）: (0.080-0.030) / (0.08098-0.030) = 0.050/0.051 = 0.980x → FV = ¥3,724
ROE 10.0%（NAVF強制型）:(0.100-0.030)/(0.08098-0.030) = 0.070/0.051 = 1.373x → FV = ¥5,216
```
→ **NAVF成否でFV¥3,724〜¥5,216の幅**
