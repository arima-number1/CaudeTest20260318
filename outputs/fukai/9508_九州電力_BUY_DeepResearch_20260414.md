#### 2-1. 決算前エントリーの妥当性判断

**結論: 「買える」— ただし原子力稼働リスクをヘッジした上での分割エントリーが必須**

**判断根拠:**

**① 現在のバリュエーションが決定的に割安**

- PBR **0.76倍**（BPS¥2,386に対し株価¥1,804）
- PER **5.0倍**（EPS¥364）— 電力セクター平均PER8〜12倍と比較して**40〜60%のディスカウント**
- 仮にPER8倍で評価されれば株価 = ¥364 × 8 = **¥2,912**（現在比+61%）

**② 原子力4基高稼働→決算でのEPS上振れ確度が高い**

原子力の限界コストは化石燃料の**1/3〜1/4**。玄海3・4号＋川内1・2号の4基稼働が続けば、FY2025のEPSは**¥350〜¥420**レンジで着地する可能性が高く、「決算サプライズ≒上方修正」シナリオが現実的。

**③ ホルムズリスク = 九州電力には「逆説的プラス」の側面**

WTI¥97・ブレント¥98水準の高原油環境下で原子力高稼働企業は**相対的勝者**。
化石燃料依存の火力主体企業が痛む中、九州電力の原子力比率の高さが**ヘッジ機能**を果たす。

**④ 決算前の需給**

## セクション4：DCFバリュエーション

### 4-1. WACC算出

**前提条件（全て明示）:**

| パラメータ | 値 | 根拠・出所 |
|----------|---|----------|
| Rf（リスクフリーレート）| **2.467%** | JGB10Y / 財務省 2026-04-13（指定値） |
| β（ベータ）| **0.55** | 電力公益株の典型値（低ボラティリティ）。九州電力の過去5年β実績は0.45〜0.65レンジ |
| ERP（エクイティリスクプレミアム）| **5.5%** | 日本市場標準（Damodaran 2026年1月更新値参考） |
| **Re（株主資本コスト）** | **= 2.467% + 0.55 × 5.5% = 2.467% + 3.025% = 5.49%** | CAPM |
| Rd（負債コスト・税前）| **1.5%** | 電力会社社債格付A〜AA水準、現在金利環境 |
| 実効税率 Tc | **30%** | 日本法人税・実効税率標準 |
| **Rd × (1-Tc)** | **= 1.5% × 0.70 = 1.05%** | 税後負債コスト |
| E（株式時価総額）| 約**4,300億円**（¥1,804 × 2.4億株）| 推計 |
| D（有利子負債）| 約**3.5兆円** | 業界公開情報 |
| D+E（企業価値ベース）| 約**3.93兆円** | — |
| E/(D+E) | **≒ 10.9%** | — |
| D/(D+E) | **≒ 89.1%** | — |

**WACC計算:**

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.109 × 5.49%] + [0.891 × 1.05%]
     = 0.599% + 0.936%
     = 1.535%
```

> **WACC = 約1.54%**

> ⚠️ **電力会社特有の注意点**: 電力会社はD/E比率が極めて高く（設備産業）、WACCは負債コストに強く引っ張られる。株主資本コスト(Re=5.49%)は財務レバレッジ調整後でも妥当。

> **天然ガス価格（Henry Hub ¥2.61 USD/MMBtu）は比較的低位安定**だが、日本向けLNGはアジアスポット価格（Henry Hubの3〜5倍）で取引される。ホルムズ封鎖ならアジアLNG価格が**$20〜$40/MMBtu**に急騰するリスク。九州電力の場合、原子力4基稼働でLNG依存を相対的に下げているため、**純火力主体企業（中国電力・四国電力等）より耐久力が高い**。
