### 2-3. 低PBRの本質診断：「衰退ディスカウント」vs「本物のバリュー」

#### 【現状確認】

```
現在株価：    ¥756
BPS（純資産）：¥822
PBR：         0.92倍
ROE：         5.6%
PER：         16.5倍
EPS：         ¥46
```

#### 【PBR 0.92倍の分解診断】

**理論PBR = ROE / Re（株主資本コスト）**

仮にRe = 8.0%（後述DCF前提）とすると：

```
理論PBR = 5.6% / 8.0% = 0.70倍
```

**現在のPBR 0.92倍は理論値（0.70倍）を上回っている。**

これは市場が「現在ROE 5.6%は将来改善する」または「不動産等の含み益がある」と評価していることを示唆する。

#### 【「衰退ディスカウント」の根拠】

```
[ディスカウント要因]
  ① 地方メディアの構造的衰退 → 広告収益の長期逓減
  ② 全国プラットフォームとの競争激化 → 値下げ圧力
  ③ 中国地方の人口・経済縮小 → TAM（全体市場）の縮小
  ④ ROE 5.6% → 資本コスト（推計8%）を下回る → 価値破壊状態
  ⑤ 成長投資余地の限界 → 再投資利益率の低さ
```

#### 【「本物のバリュー」の根拠】

```
[バリュー要因]
  ① PBR 0.92倍 → 純資産比9%ディスカウント（清算価値以下）
  ② 不動産含み益（後述NAV分析）→ 実質解散価値はBPS以上
  ③ 人材・葬祭の成長が広告縮小を相殺しつつある
  ④ 低PBR是正を求める東証プレッシャー（東証スタンダード改革）
  ⑤ フリーキャッシュフロー黒字継続（資本配分余地）
  ⑥ 自社株買い実施余地（余剰現金保有）
```

#### 【深井の本質診断】

> **「両方が正しい」ハイブリッド状況**
>
> KG情報のPBR 0.92倍は、①構造的衰退への一定のディスカウントと、②不動産含み益・非広告転換進捗への評価が**相殺**した結果と見る。
> 純粋な「衰退銘柄」なら PBR 0.5〜0.6倍まで売られるはずが、0.92倍に留まっているのは「転換が進んでいる」証拠。
> 一方、PBR 1.0倍超への回復には **ROEの8%超え（資本コスト超過）** が必要であり、現状のROE 5.6%では正当化しにくい。
> **「緩やかなバリュー」として配当+自社株買いを享受しながら非広告転換の加速を待つ」投資スタイルが適切。**

```
  β（ベータ）：               0.85（地方メディア・サービス複合、推計）
  ERP（株式リスクプレミアム）： 6.5%（日本株：Damodaran推計ベース）

  Re = 2.386% + 0.85 × 6.5%
     = 2.386% + 5.525%
     = 7.911% ≈ 7.9%
```

#### WACCの算出

```
  E（株式時価総額）： ¥756 × 7,800千株 = ¥5,897M
  D（有利子負債）：  ¥1,800M
  E+D：             ¥7,697M
  E/(D+E)：         76.6%
  D/(D+E)：         23.4%

  Rd（負債コスト）：  1.5%（地方銀行借入金利推計）
  Tc（実効税率）：    30%

  WACC = 76.6% × 7.9% + 23.4% × 1.5% × (1-30%)
       = 6.051% + 0.246%
       = 6.30%
```
