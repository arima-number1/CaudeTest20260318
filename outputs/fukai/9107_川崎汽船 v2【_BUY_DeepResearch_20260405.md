### 2.1 PBR0.98倍の意味と資本効率改善余地

PBR1.0倍 = 株価がBPSと等しい水準

→ PBR1.0倍はほぼ「目前」。市場がROE改善を確認すれば自然達成可能。

### 2.3 DOE観点の下値支持線分析

**DOE（Dividend on Equity：純資産配当率）**は、配当の持続性と株価下値支持を分析する上で有効な指標です。

```
DOEの定義:
DOE(%) = 配当総額 ÷ 自己資本 × 100
       = DPS ÷ BPS × 100
       = 配当利回り × PBR

【川崎汽船のDOE推計】
DPS（FY2026/3予想）: 約¥200〜250（推定）
BPS（3Q時点）: ¥2,753
DOE: ¥200〜250 ÷ ¥2,753 = 7.3〜9.1%
```

**DOE固定による下値支持線計算:**

仮にDOE = 7.0%を「暗黙のコミットメント」とした場合：

```
DPS固定 = ¥2,753 × 7.0% = ¥193（切上で¥200）
配当利回り 目標 3.5% → 下値支持線 = ¥200 ÷ 3.5% = ¥5,714
配当利回り 目標 4.0% → 下値支持線 = ¥200 ÷ 4.0% = ¥5,000
配当利回り 目標 5.0% → 下値支持線 = ¥200 ÷ 5.0% = ¥4,000
配当利回り 目標 7.0% → 下値支持線 = ¥200 ÷ 7.0% = ¥2,857
```

**保守的下値支持ゾーン:**

| 配当利回り水準 | 下値支持線（¥） | 現在値比（¥2,698） |
|-------------|--------------|-----------------|
| 3.5%（強気） | ¥5,714 | +112% |
| 4.0%（基本） | ¥5,000 | +85% |
| 5.0%（中立） | ¥4,000 | +48% |
| 7.0%（売られすぎ） | ¥2,857 | +6% |
| **7.4%（現在値）** | **¥2,698** | **±0%（現在値）** |

> **深井解釈:** 現在株価¥2,698は配当利回り約7.4%相当。これは海運株としても高利回りゾーンにあり、DOE観点での**下値支持は強固**。自己資本が厚い（BPS¥2,753 ≒ 株価）ためDOE維持コストも相対的に重い一方、ONE社配当で賄える水準。

### 2.4 誤算ロジックの欠陥特定と再発防止プロセス

#### 2.4.1 欠陥特定（根本原因分析）

**欠陥①: BPS参照時点のズレ**
```
前回誤り: FY2025/3期末（年度末）のBPSを使用
→ 当時BPSは¥1,600〜1,800程度（ONE好業績前）
正値: FY2026/3 3Q末BPS ≒ ¥2,753
→ ONEの好業績で自己資本が急増していた事実を反映していなかった
```

**欠陥②: ONE社持分法利益の純資産積み上げ効果を見落とし**
```
誤認識: ONE社の利益 = 川崎汽船PLに「見えない利益」
正認識: 持分法利益は配当として受取らない分が「投資有価証券」として
        BSに累積 → 自己資本を急増させる
→ この「見えない自己資本膨張」を追跡していなかった
```

**欠陥③: 自社株買いによる株式数変化の未更新**
```
誤認識: 発行済株式数を旧データのまま使用
正認識: 自社株買いで流通株式数が減少 → BPS計算の分母が変化
```

**欠陥④: データソース確認の一元化不備**
```
誤認識: IR資料・決算短信・Bloombergデータの参照時点が混在
正認識: 同一決算期・同一時点のデータセットで一貫性確認が必要
```

#### 2.4.2 再発防止プロセス（即日適用）

```
【チェックリスト v2.0（深井式 PBR計算フロー）】

STEP 1: データ取得
□ 決算短信（最新四半期）から「期末自己資本合計」を直接取得
□ 発行済株式数（自己株式除く）を同一決算書から確認
□ BPS = 自己資本合計 ÷ 発行済株式数（自己株式控除後）

STEP 2: 株価確認

□ 株価の出所・取得日時を明記

STEP 3: PBR計算

STEP 4: クロスチェック
□ 会社HPの「株式情報」ページのPBR表示と照合

## 4. DCFバリュエーション

### 4.1 前提条件（全明記）

#### A. コスト・オブ・キャピタル前提

| パラメーター | 値 | 根拠・出所 |
|-----------|-----|---------|
| **リスクフリーレート（Rf）** | **2.380%** | JGB10Y（日本10年国債）社長確認値、2026-04-05 17:53 JST取得 |
| **エクイティリスクプレミアム（ERP）** | **6.0%** | Damodaran 2025年推定（日本市場）|
| **ベータ（β）** | **1.15** | 東証海運セクター平均・川崎汽船過去5年実績 |
| **株主資本コスト（Re）** | **2.380% + 1.15 × 6.0% = 9.28%** | CAPM |
| **負債コスト（Rd）** | **1.5%** | 直近社債・借入金利（低水準を維持）|
| **実効税率（Tc）** | **30%** | 日本法人税率相当 |
| **資本構成（E/V）** | **85%** | 時価総額÷（時価総額＋有利子負債）推計 |
| **負債比率（D/V）** | **15%** | 上記の逆数 |

**WACC計算:**
```
WACC = E/V × Re + D/V × Rd × (1-Tc)
     = 0.85 × 9.28% + 0.15 × 1.5% × (1-0.30)
     = 7.888% + 0.1575%
     = 8.05%（≒8.0%）
```

|------|------|

| 成長率（フェーズ1: 1〜5年） | +3.0%/年（ONE安定・ドライバルク回復） |
| 成長率（フェーズ2: 6〜10年） | +1.5%/年（成熟化） |
| 永続成長率（g） | **1.5%** |
