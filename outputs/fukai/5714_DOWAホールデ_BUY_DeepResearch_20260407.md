### 1-3. 競合優位性（Moat分析）

**① 都市鉱山リサイクルの希少性**
- 貴金属・レアメタルの湿式精錬技術は国内最高水準
- 秋田製錬（100%子会社）の一貫精錬体制により、スクラップ→地金→電子材料まで垂直統合
- 廃電子機器からのパラジウム・インジウム回収は競合の追随困難

**② 電子材料の技術障壁**
- 高純度亜鉛（5N/6N）は半導体製造工程での代替困難性が高い
- 銀粉は粒径制御技術が差別化要因（DOWA製は±5nm精度）

**③ 熱処理の地理的分散**
- 国内外100拠点超の熱処理ネットワーク
- 自動車OEM各社との長期サプライヤー契約が収益安定化

## 4. DCFバリュエーション

### 4-1. WACC計算

#### パラメータ設定

| パラメータ | 値 | 根拠 |
|-----------|---|------|
| **リスクフリーレート（Rf）** | **2.429%** | JGB10Y確定値 |
| **株式リスクプレミアム（ERP）** | 5.5% | 日本株市場ERP（Damodaran推計2025〜2026） |
| **β（ベータ）** | 1.05 | 非鉄金属・コモディティ連動セクター（過去3年推計） |
| **株主資本コスト（Re）** | **Re = 2.429% + 1.05 × 5.5% = 8.204%** | |
| **負債コスト（Rd）** | 1.2% | 格付A相当・銀行借入金利水準 |
| **実効税率（Tc）** | 30% | 日本法人税等 |
| **税引後負債コスト** | 1.2% × (1-30%) = **0.84%** | |
| **資本構成 E/(D+E)** | **78%** | 推計（自己資本3,700億/総資本4,800億） |
| **負債比率 D/(D+E)** | **22%** | |

#### WACC計算

```
WACC = E/(D+E) × Re + D/(D+E) × Rd × (1-Tc)
     = 0.78 × 8.204% + 0.22 × 0.84%
     = 6.399% + 0.185%
     = 6.584%

【WACC = 6.58%（基本ケース）】
```
