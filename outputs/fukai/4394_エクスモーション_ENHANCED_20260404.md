## 2. 【★最重要】上方修正の「Why」徹底解剖

### 2-0. はじめに — この分析が「なぜ」重要か

市場参加者の多くは「上方修正 → 株価上昇」という単純反応をする。しかし、真のアルファは「その上方修正が持続するか否か」を見極めることにある。一過性の特需や原価の偶発的な低下によるものであれば、株価の上昇余地は限定的だ。逆に、ビジネスモデルの構造的改善が背景にあるならば、アナリスト全体の来期以降の推計修正は時間差で発生し、先行して織り込んだ投資家に超過リターンをもたらす。

本章では以下の3ステップで「Why」を徹底解剖する：

- **Step 1:** 3ヶ月前（期初ガイダンス時点）との定量比較
- **Step 2:** 改善要因の3分解（具体的数値付き）
- **Step 3:** 「一過性」か「構造的」かの最終判定

### 2-2. Step 2: 「何が想定外に改善したのか」— 3要因徹底解剖

#### 要因1: サブスクリプション比率の構造的改善（最重要・最大寄与）

**背景と定量的インパクト**

エクスモーションのビジネスモデルは、製造業向けAI/IoTソフトウェアのライセンス販売から、サポート・保守契約のサブスクリプション型（SaaS的）収益への移行を進めてきた。

```
【営業レバレッジの仕組み】

エクスモーションの固定費構造（推計）:
- 研究開発費:  約¥40-50M/年（人件費中心、売上変動に連動しにくい）
- 一般管理費:  約¥30-35M/年（上場コスト含む固定費）
- 合計固定費:  約¥70-85M/年

→ 固定費はほぼ一定
→ 増分売上の60-70%以上が営業利益に流れる（限界利益率の高さ）
```

| 指標 | FY2024 Q1（推計） | FY2025 Q1（実績） | 変化 |
|------|------------------|------------------|------|
| 推計サブスク比率 | ~55-60% | ~65-70% | +10pp |
| 推計サブスク収益 | ~¥190-205M相当 | ~¥233-250M相当 | +¥28-45M |
| 固定費（推計） | ~¥20M/Q | ~¥20M/Q | 変化なし |
| 増分利益転換率 | — | ~60-70% | — |

**具体的なインパクト試算:**

```
追加サブスクリプション収益（推計）: +¥28M（売上増分）
うち利益転換分（限界利益率65%仮定）: +¥18M → これが営業利益増加の「コア」

前年比 OP 増加額: ¥57M - ¥37M = +¥20M
うちサブスク比率改善寄与: ~¥12-15M（全体増加の60-75%）
```

**なぜ「構造的」か:**

サブスクリプション収益の増加は、一度顧客が導入したシステムを解約することが稀なSaaS的特性（高い顧客粘着性）に基づく。自動車Tier-1や電子機器メーカーの生産ラインに組み込まれたMES（製造実行システム）やAI品質管理ソフトウェアは、切り替えコスト（スイッチングコスト）が極めて高い。

```
推計チャーンレート（解約率）: <5%/年
→ 一度獲得したサブスク収益は毎年自動的に積み上がる
→ 来期以降も同様の「底上げ」効果が継続する = STRUCTURAL
```

#### 要因2: 大型案件の前倒し計上（寄与あり・一過性要素含む）

**背景**

ソフトウェア企業における「収益認識」のタイミングは、四半期間で利益をシフトさせる。エクスモーションの場合、大型製造業顧客への新規ライセンス導入（例：新工場立ち上げ時のシステム一括導入）は、プロジェクト完了時に一括計上される傾向がある。

**Q1に前倒し計上されたと推定される案件の規模:**

```
売上高の期初ガイダンス比超過: +¥28M（推計）
うちサブスク増加分: -¥15M（構造的）
残差（大型案件前倒し推計）: +¥13M程度

この¥13Mの案件が当初Q2以降に計画されていたとすれば:
- Q2以降の利益は若干「前借り」となる
- ただし、Q1の固定費吸収が早まることで通期では大きく影響しない
```

**案件候補（業界知識に基づく推測）:**

エクスモーションの主要顧客である自動車Tier-1サプライヤー（デンソー関連企業、アイシン系列等）は、2025年末〜2026年初頭にかけてEV対応の生産ライン改修プロジェクトを多数実施しており、そのタイミングでMES/予知保全ソフトウェアの新規導入が加速した可能性が高い。

**一過性vs構造的の判定:**

```
前倒し計上: 一過性（Q2以降の計上が減少する可能性）
ただし、新規案件獲得そのものは継続（バックログ積み上がり）
→ 案件計上タイミングは一過性だが、案件総量の増加は構造的

判定: 「一過性・構造的混在」（寄与の30%が一過性）
```

#### 要因3: 原価低減（外注費・クラウドインフラコスト削減）

**背景と数値**

エクスモーションのAI/IoTソフトウェア運用には、クラウドインフラコスト（AWS/Azure等）と外注エンジニアコストが発生する。2025年以降、以下の改善が推計される：

```
【コスト改善の内訳（推計）】

(A) クラウドインフラコスト最適化:
    - 2024年中のアーキテクチャ見直し（マルチクラウド戦略、コンテナ化）
    - 推計削減額: ¥3-5M/年 → Q1では ¥1-1.5M
    - 性質: 構造的（一度最適化すれば継続的効果）

(B) 外注エンジニア費用の内製化:
    - 2025年新卒・中途採用強化による内製化推進
    - 外注比率低下により、外注費削減 ¥2-4M/Q（推計）
    - 性質: 構造的（採用コストは初年度に先行）

(C) その他変動費正常化:
    - 前年Q1はシステム移行関連の一時費用が発生していた可能性
    - 推計特殊費用の消滅: +¥1-2M/Q 効果
    - 性質: 一過性（前年の費用がなくなったにすぎない）

合計推計コスト削減効果: ¥4-7M/Q
→ このうち利益改善寄与: ほぼ全額（¥4-7M）
```

**粗利益率へのインパクト:**

```
前年Q1 推計粗利益率: ~55-58%
今Q1 推計粗利益率:   ~60-63%（+2-5pp）
→ 売上¥358Mに対して +¥7-18Mの利益改善効果
（うちクラウド・外注コスト分: ¥4-7M、残差は売上ミックス改善）
```

### 2-3. Step 3: 「一過性」か「構造的」か — 最終判定

| 要因 | 寄与額（推計） | 構造的 | 一過性 | 判定 |
|------|--------------|--------|--------|------|
| サブスク比率改善 | ¥12-15M | ✅ 主 | — | 構造的（★★★） |
| 大型案件前倒し | ¥5-8M | △ 部分 | ✅ 部分 | 混在（★★） |
| 原価低減 | ¥4-7M | ✅ 主 | △ 一部 | 構造的（★★） |
| **合計超過利益** | **¥20M（前年比）** | **~70%** | **~30%** | — |

#### 最終判定: 70% 構造的 ／ 30% 一過性

```
■ 深井 最終判定:

今回の上方修正ペース（OP +54.9% YoY）の内訳:

構造的改善分（~70% = ¥14M相当）:
  → SaaS的サブスクリプション収益の積み上がり
  → 営業レバレッジの発現（固定費が変わらず増分利益が直撃する構造）
  → クラウド・外注コストの構造的低下

一過性分（~30% = ¥6M相当）:
  → 大型案件の前倒し計上タイミング
  → 前年の特殊費用消滅効果

結論: この上方修正は実力ベースの改善が主体
```

#### 修正後「真の実力」推計

構造的改善のみ（一過性を除外）ベースで、FY2026通期OP を推計すると：

```

構造的改善の寄与（年間換算）:   +¥14M × 4Q = +¥56M（上限）
→ ただしQ4偏重の季節性を考慮: +¥40-50M が現実的な追加分

実力ベース通期OP推計:
  - 保守的シナリオ: ¥220M（ガイダンス+7.8%）
  - 中間シナリオ:   ¥225M（ガイダンス+10.3%）
  - 強気シナリオ:   ¥230M（ガイダンス+12.7%）

→ 深井 最終推計: ¥220-230M（「確実な10%超え」を基本線とする）
```

> **アルファの源泉:** 市場（コンセンサス）はまだガイダンス通り¥204Mベースで動いている。「実力¥220-230M」を先取りして保有することが超過リターンにつながる。

## 4. 株式分割＋増配統合評価

### 4-1. 株式分割（1:2）の詳細

| 項目 | 詳細 |
|------|------|
| 分割比率 | 1株 → 2株（1:2） |
| 効力発生日 | **2026年6月1日** |
| 基準日 | 2026年5月31日 |
| 目的 | 株式の流動性向上・個人投資家層の拡大 |

**株価への影響:**

```
分割前 現在株価（仮定）: ¥1,500/株
分割後 理論株価:         ¥750/株（機械的な半値）
分割後 目標株価（深井）:  ¥784/株（ベースケース）
分割後 目標株価（強気）:  ¥1,025/株（ブルケース）

→ 分割自体は株価中立だが、流動性向上と
　 個人投資家の参入増加が「流動性プレミアム」をもたらす可能性
```

### 4-2. 配当政策の大幅改善

| 配当 | 分割前換算 | 分割後換算 | 前年比 |
|------|----------|----------|--------|
| 中間配当 | ¥10/株 | ¥5/株 | **新設** |
| 期末配当 | ¥20/株 | ¥10/株 | — |
| **年間合計** | **¥30/株** | **¥15/株** | **前年¥20→+50%増配** |

```
配当利回り計算（分割後）:
現在株価（仮定 分割後¥750）:   ¥750
年間配当（分割後）:             ¥15
配当利回り:                     2.0%

目標株価ベースでの利回り:       ¥15 ÷ ¥784 = 1.91%
```

**増配の意義分析:**

1. **中間配当の新設:** これまで期末一括支払いだった配当を、中間・期末の2回に分割。これは「年間を通じたキャッシュフローへの自信」の表れ。
2. **+50%増配:** 単純な増益率を超える増配率は、経営陣の「来期以降も高水準を維持できる」というコミットメントを示す。
3. **分割後でも増配:** 分割調整後でも実質的な増配を維持することで、既存株主への還元を明確にしている。

### 4-3. 株式分割後の株主構成変化予測

```
現在の個人株主比率（推計）: ~30-35%
分割後1年での予測:          ~45-55%

→ 個人投資家の参入により:
  - 需要基盤の拡大
```
