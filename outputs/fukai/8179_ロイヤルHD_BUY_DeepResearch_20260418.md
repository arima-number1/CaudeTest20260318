## セクション4 — バリュエーション（正しい数値で再算出）

### WACC（Rf = 2.380%固定）

| パラメータ | 数値 |
|-----------|------|
| Rf | 2.380% |
| β | 0.161（Yahoo Finance実測値。低ベータ） |
| ERP | 5.5% |
| Re | 2.380% + 0.161 × 5.5% = **3.266%** |
| Rd | 1.5%（D/E低く借入金利低位） |
| E/(D+E) | 72% |
| **WACC** | **72% × 3.266% + 28% × 1.5% × 0.7 = 2.352% + 0.294% = 2.646%** |

> **注：β=0.161は極めて低い。外食・ホテル複合業態の防衛的性格を反映。低WACCはDCF上プラスに働く。**

### EPS成長モデル

| 期 | EPS | 根拠 |
|----|-----|------|
| FY2025実績 | ¥57.48 | 決算短信（公式） |
| FY2026E（会社ガイダンス） | ¥57.87 | 決算短信（公式） |
| FY2026E（賃上げ調整後） | **¥54.3** | ガイダンス▲賃上げ未織込み推計¥3.6 |
| FY2027E | **¥65** | ホテル継続成長+外食コスト正常化 |

### フェアバリュー

| 手法 | FV | 根拠 |
|------|-----|------|
| FY2026E EPS¥54.3 × PER 22x | ¥1,195 | コスト増環境での適正倍率 |
| FY2026E EPS¥54.3 × PER 25x（現状） | ¥1,358 | ホテル成長プレミアム込みの現状倍率 |
| FY2027E EPS¥65 × PER 22x × 割引6.5% | ¥1,345 | 1年先業績現在価値 |
| GGM（ROE 10.8%・Re 3.27%・g1%） | ¥1,422 | BPS¥555.56 × PBR理論2.56x |
| **加重平均FV** | **¥1,380** | |

**GGM（ゴードン成長モデル）：**
```
PBR理論値 = (ROE - g) / (Re - g)
= (10.8% - 1.0%) / (3.266% - 1.0%)
= 9.8% / 2.266%
= 4.33倍

→ ただしβ=0.161の低リスク株として Re=3.27%は低水準。
  より市場平均的なRe=6.5%で計算すると：
  (10.8% - 1.0%) / (6.5% - 1.0%) = 9.8% / 5.5% = 1.78倍
  → ¥555.56 × 1.78 = ¥988（保守的FV）

→ 実態はRe=3.27%〜6.5%の間。中央値Re=4.9%：
  9.8% / 3.9% = 2.51倍 → ¥1,394
```
