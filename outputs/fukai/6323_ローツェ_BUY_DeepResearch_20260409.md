# 1. 企業概要・事業分析

## 1-1. ローツェの本質：「半導体工場の血管」

ローツェ（本社：岡山県総社市）は、半導体製造工程において**ウエハーを汚染ゼロで自動搬送するシステム**を専業とする世界トップメーカー。半導体装置メーカーではなく「搬送インフラ専業」という独自ポジションを持つ。

**核心製品とシェア：**
- **FOUP（Front Opening Unified Pod）搬送装置**：クリーンルーム内のウエハー輸送カプセル。世界シェア推定**30〜40%**（トップ水準）
- **ウエハーソーター**：ウエハーの仕分け・方向合わせ装置。世界シェア推定**30%以上**
- **EFEM（Equipment Front End Module）**：装置と搬送系のインターフェース
- **AMHS（Automated Material Handling System）**：工場全体の自動搬送ライン

# 4. DCFバリュエーション

## 4-1. WACC計算の前提

```
【WACC計算パラメータ】

リスクフリーレート（Rf）：2.374%（JGB10Y・2026-04-09）
株式リスクプレミアム（ERP）：6.0%（日本株ERP・標準値）
β（推定）：1.25（半導体装置セクター・東証プライム）
  → 半導体装置はシクリカル×成長株でβ1.1〜1.4が妥当範囲
  → 1.25を採用

株主資本コスト（Re）：
  Re = Rf + β × ERP
  Re = 2.374% + 1.25 × 6.0%
  Re = 2.374% + 7.5%
  Re = 9.874% ≒ 9.9%

【資本構成】
時価総額（E）：¥566,900M
有利子負債（D）：¥23,781M
E/(D+E) = ¥566,900M ÷ ¥590,681M = 95.97%
D/(D+E) = ¥23,781M ÷ ¥590,681M = 4.03%

負債コスト（Rd）：1.5%（推定・日本の銀行借入レート）
法人税率（Tc）：30%

【WACC計算】
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
WACC = [0.9597 × 9.874%] + [0.0403 × 1.5% × 0.70]
WACC = 9.477% + 0.042%
WACC = 9.52% ≒ 9.5%
```
