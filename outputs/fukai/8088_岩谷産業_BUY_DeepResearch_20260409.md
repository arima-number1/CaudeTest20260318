### ①LPG価格スパイクの業績インパクト試算（最重要）

#### 1-a. 前提整理：岩谷産業のLPG事業構造

```
【LPG収益フロー概略図】

サウジアラムコCP価格（USD/t）
        ↓ （為替換算・海上運賃加算）
    仕入原価（輸入LPG）
        ↓
  岩谷産業 卸売・小売
        ↓ （原料費調整制度により翌月〜翌々月転嫁）
    消費者・産業向け販売価格
        ↓
    販売マージン（スプレッド）
```

**主要前提パラメータ**

| 項目 | 数値 | 出所・根拠 |
|------|------|----------|
| 国内LPG販売量（岩谷推定） | 約230万t/年 | 業界データ・シェア推定 |
| うちプロパン比率 | 約65% ≒ 150万t | 業界標準 |
| うちブタン比率 | 約35% ≒ 80万t | 業界標準 |
| USD/JPY想定レート | ¥150/USD | 現市況 |
| LPG中東依存度（日本全体） | 約90%超（岩谷単体） | 業界公知 |
| 海上運賃・保険・諸経費 | $50〜80/t | 業界推定 |
| 販売マージン（スプレッド） | 約¥30,000〜40,000/t | 過去決算推定 |

> **補足（重要）**: 「日本の中東LPG依存度はわずか3.7%」という浅井コメントは**米国産LPG輸入依存度に関する記述**として理解（米国産が約60%、カナダ21%）。岩谷産業の調達構造は**中東（サウジ・カタール）主体**であり、CP価格直撃を受ける。

#### 1-b. CP価格変動の利益インパクト計算式

```
ΔCP（プロパン）= $750 - $545 = +$205/t
ΔCP（ブタン）  = $800 - $540 = +$260/t

税込み仕入増加コスト（FY2027/3 通年、転嫁前）:
プロパン: 150万t × $205 × ¥150 = ¥461億
ブタン  :  80万t × $260 × ¥150 = ¥312億
合計（仕入コスト増）= ▲¥773億/年（転嫁前グロス）

ただし：
・原料費調整制度により翌月〜2ヶ月以内に小売価格転嫁
・転嫁後は「コスト増＝価格増」でニュートラル（スプレッド維持）
・転嫁ラグ期間（1〜2ヶ月）のみ一時的マージン圧迫発生
```

#### 1-c. 原料費調整制度の仕組みと転嫁タイムラグ

```
【原料費調整制度メカニズム】

Step 1: サウジCP価格通知（毎月初旬）
         ↓ （約2〜4週間）
Step 2: 岩谷産業が国内卸売価格を改定（翌月1日付）
         ↓ （小売業者への通知・契約改定）
Step 3: 小売価格への反映（翌月〜翌々月）

→ 転嫁ラグ: 最大1〜2ヶ月（通常1.5ヶ月平均）
```

**タイムラグ期間の一時的マージン圧迫額試算**

```
月間仕入コスト増加額（上昇分フル転嫁前）:
= ¥773億 ÷ 12ヶ月 = 約¥64億/月

1.5ヶ月ラグ分のマージン圧迫:
= ¥64億 × 1.5ヶ月 = 約▲¥96億（一時的）

※ ただし4月卸売値+¥23,000/t引上げ済（転嫁先行措置）
  先行転嫁分: 230万t × ¥23,000 × 1/12 ≒ ¥44億/月 相当
  → 実質的な未転嫁ラグ損失: 約¥96億 − ¥44億 = ▲約¥52億（1QのQ限定）
```
