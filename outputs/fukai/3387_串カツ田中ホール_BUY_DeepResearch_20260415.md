## 補論：ROE-PBRフレームワークによるチェック

**理論的適正PBR = ROE ÷ 株主資本コスト（Re）**

```
ROE（確定値）= 12.7%
Re（計算値） = 7.10%

理論的適正PBR = 12.7% ÷ 7.10% = 1.79倍

現実のPBR（確定値） = 6.90倍

乖離率 = 6.90 ÷ 1.79 = 3.86倍のオーバーバリュー
```

> **この乖離の意味：** 市場は「ROEが現在の12.7%から将来的に大幅に上昇する（20〜30%超）」または「成長オプション価値が巨大」と見込んでいる。しかし、外食チェーンの業態特性上、ROEが20%超に持続的に達する可能性は低い。**PBR6.9倍は構造的に正当化困難。**
