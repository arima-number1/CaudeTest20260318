## 2. 【社長緊急命題】LPG優位性の数理検証

### 2.1 問題の本質定義

**「LPG原料プラント優位性によるナフサ競合Beat構造」**——
これは以下の問いに分解される：

```
① LPGクラッカー vs ナフサクラッカーのコスト差（スプレッド）は実在するか？
② その差は三井化学のEBITDAに定量的にどう反映されるか？
③ 市場（株価）はその優位性を正当に評価しているか？
④ 逆転トリガーは何か？
```

---

### 2.2 LPG vs ナフサ：原料コスト構造の解剖

#### 2.2.1 原料価格の構造的差異

**ナフサ（Naphtha）：**
- 原油連動性が高い（原油価格×0.85〜0.95の係数）
- 国際スポット価格に晒される（アジアNaphtha価格）
- 日本では100%輸入依存

**LPG（プロパン/ブタン）：**
- 原油・天然ガス双方から分離（分散的供給構造）
- 中東・北米産のFOB価格（CP：コントラクトプライス）が基準
- 日本への海上輸送コスト加算だが、**シェール革命後の北米LPG余剰**で構造的下落

#### 2.2.2 スプレッド定量化モデル

以下は深井推計モデル（公開データ・業界レポートに基づく）：

```
【基準パラメーター（2025年度平均推計）】

原油価格想定        : WTI $75/bbl = Brent $77/bbl
アジアナフサ価格     : $620〜650/MT
中東プロパンCP      : $540〜580/MT
ブタンCP            : $510〜550/MT
LPG平均（C3/C4混合）: 約$555/MT（プロパン70%：ブタン30%仮定）
```

```
【エチレン・プロピレン収率差（クラッキング理論収率）】

              ナフサ系     LPGプロパン系    差異
─────────────────────────────────────────────
エチレン収率   28〜32%      38〜42%        +10pt（LPG優位）
プロピレン収率 14〜16%       8〜12%        -4pt（ナフサ優位）
ブタジエン収率  4〜 5%       1〜 2%        -3pt（ナフサ優位）
原料単価/MT   $635/MT      $555/MT        -$80/MT（LPG優位）
─────────────────────────────────────────────
```

> **⚠️ 重要な非対称性：**
> エチレンはプロピレンより市場価格が高く、LPGの高エチレン収率は有利。
> ただしプロピレン不足（プロピレンデフィシット問題）が副作用として発生。

#### 2.2.3 スプレッド計算（深井モデル）

```
【ナフサクラッカー 原料コスト/エチレン換算MT】

原料費         : $635/MT ÷ 30%収率 = $2,117/MT-エチレン相当
副産品回収価値 : プロピレン15%×$850 + ブタジエン4.5%×$1,100 + その他
               ≈ $127.5 + $49.5 + $30 = △$207/MT
ネット原料費   : $2,117 - $207 = $1,910/MT-エチレン

【LPGクラッカー（プロパン）原料コスト/エチレン換算MT】

原料費         : $555/MT ÷ 40%収率 = $1,388/MT-エチレン相当
副産品回収価値 : プロピレン10%×$850 + ブタジエン1.5%×$1,100 + 水素等
               ≈ $85 + $16.5 + $20 = △$121.5/MT
ネット原料費   : $1,388 - $121.5 = $1,267/MT-エチレン

【LPG優位スプレッド（対ナフサ系）】

$1,910 - $1,267 = $643/MT-エチレン ≈ ¥96,000/MT-エチレン（150円/ドル換算）

これはエチレン市場価格（約$1,100〜1,200/MT）の53〜58%に相当する
「構造的コストアドバンテージ」であり、極めて有意。
```

### 2.3 三井化学固有の構造（市原コンビナート分析）

#### 2.3.1 市原工場のLPG活用実態

```
三井化学 市原地区NCC構成（推計）：
─────────────────────────────────────────
総クラッキング能力      : 約65万MT/年（エチレン）
LPG原料比率（推計）     : 約40〜50%（ナフサ・LPGフレキシブル運用）
LPG専用設備投資実施     : ○（2010年代に原料フレキシブル化改造）
プロパンデハイドロ(PDH)  : 保有（プロピレン自製体制）
─────────────────────────────────────────
```

> **深井コメント：**
> 三井化学の真の優位性は「LPG専業」ではなく**「ナフサ/LPGフレキシブルスイッチング能力」**にある。
> 原料相場に応じて最適原料を選択できる「オプション価値」が、
> 純粋ナフサ系の競合（三菱ケミカル・住友化学）には存在しない。

#### 2.3.2 プロピレンデフィシット対策（PDH設備）

```
【LPG系のアキレス腱：プロピレン不足】

プロパンクラッキング増 → プロピレン副産が減少
→ PP（ポリプロピレン）・酸化プロピレン等の原料不足リスク

三井化学の対策：
・PDH（プロパンデハイドロジェネーション）設備保有
  プロパン → プロピレンを直接製造（LPG原料でプロピレンも自製）
・これにより「LPG高収率＋プロピレン自製」の二重優位を実現

競合比較：
┌────────────────┬──────────┬──────────┬──────────┐
│                │三井化学  │三菱ケミカル│住友化学  │
├────────────────┼──────────┼──────────┼──────────┤
│LPGフレキシブル  │  ◎       │  △        │  ×       │
│PDH保有         │  ○       │  △        │  ×       │
│ナフサ依存度     │  中      │  高        │  極高    │
│原料コスト優位   │  高      │  中        │  低      │
└────────────────┴──────────┴──────────┴──────────┘
```

### 2.5 「市場の誤解」の正体と逆転トリガー

#### 2.5.1 市場が見落としている3つの構造

```
【市場の誤解①：「化学品全体のコモディティ化」という一括り評価】
実態：三井化学はLPGフレキシブル化＋高機能材料シフトで
     コモディティサイクルから部分的に脱却済み
     → 市場はPER/PBRで競合と同一視しているが実態は異なる

【市場の誤解②：「住友化学・三菱ケミカルとの同一業種ディスカウント】
実態：両社の財務劣化（特に住友の実質危機水準）が
     セクター全体のセンチメントを押し下げている
     → 同業種同士の"汚染効果"で三井が不当に売られている可能性

【市場の誤解③：「LPGコスト優位は一時的」という先入観】
実態：中東・北米シェール由来LPG供給増は構造的トレンドであり、
     少なくとも5〜10年は継続する見通し（IEA・EIA予測）
     → 投資家のLPG優位認識は「サイクル的」と誤解されている
```

#### 2.5.2 逆転トリガーの特定（優先順位付き）

```
★★★ 最高優先度トリガー
T1：三菱ケミカル/住友化学の追加減損・格下げ発表
   → セクター内の相対価値再評価が起きる最速のカタリスト
   → 「なぜ三井だけ健全か」の市場理解が進む

★★☆ 高優先度トリガー
T2：LPG-ナフサスプレッドの$100/MT超え（原油急騰局面）
   → 三井の決算プレスに「LPGコスト優位による利益貢献」の
     定量記載が増える → アナリストカバレッジ更新 → 株価反応

T3：FY2026/3月期 通期ガイダンスで営業利益1,000億円超の見通し
   → 市場コンセンサスを上回ればバリュエーション水準切り上げ

★☆☆ 中優先度トリガー
T4：自動車軽量化・EV向け高機能材料受注の大型化（具体的数値付き）
T5：IR/決算説明会でのLPG優位性の明示的コミュニケーション強化
```
