### 2-3. PBR1.51倍は割高か正当か

#### 日本メガバンク比較

| 銘柄 | PBR（現在） | ROE | PBR正当性 |
|------|-----------|-----|----------|
| 三菱UFJ（8306） | **1.51倍** | 6.1% | 後述分析参照 |
| 三井住友FG（8316） | 約1.30〜1.40倍（推定） | 約7〜8% | やや割安 |
| みずほFG（8411） | 約0.95〜1.10倍（推定） | 約6〜7% | 依然ディスカウント |
| JPMorgan Chase | 約2.0〜2.2倍 | 約15〜17% | プレミアム正当 |
| HSBC | 約0.9〜1.1倍 | 約10〜12% | 新興市場ディスカウント |

#### PBRの理論的正当性：Gordon Growth Model（株主資本コスト vs ROE）

```
正当PBR = (ROE - g) / (Re - g)
```

| パラメータ | 値 |
|-----------|-----|
| ROE | 6.1%（所与） |
| Re（株主資本コスト） | 7.5%（後述WACC計算ベース） |
| g（長期成長率） | 2.0%（名目GDP成長率） |

```
正当PBR = (6.1% - 2.0%) / (7.5% - 2.0%) = 4.1% / 5.5% = 0.745倍
```

> **これは現在のPBR1.51倍を大幅に下回る。**

#### しかし、なぜ市場は1.51倍を許容しているのか

Gordon Growth Modelは静的ROEを前提とするが、MUFGのROEは**構造的改善トレンド**にある：

| 項目 | 現在 | 2〜3年後期待 |
|------|------|------------|
| ROE | 6.1% | 7.5〜8.5%（利上げ効果+政策株売却+自社株買い） |
| 正当PBR（ROE=8.0%、Re=7.5%、g=2.0%） | — | **1.09倍** |
| 正当PBR（ROE=8.5%、Re=7.5%、g=2.0%） | — | **1.18倍** |

**追加要因：解散価値プレミアム**

MUFGのBPS¥1,885に含まれる資産の多くは時価評価済み金融資産であり、製造業のような簿価乖離は少ない。加えて、**東証のPBR1倍以上要請**が官製フロアとして機能しており、機械的な売り圧力が遮断されている。

**結論②：** Gordon Growth Modelによる理論PBRは0.74〜1.18倍であり、現在の**1.51倍は10〜20%程度の「将来ROE改善期待プレミアム」を上乗せした水準**。過度に割高とは言い切れないが、ROE改善が遅れればプレミアムは剥落しうる。ただしPBRが1.51倍だからといってショートの根拠にはならない——東証圧力×自社株買いが実質的な下値サポートとして機能する。

### 2-4. バフェット×日本株文脈でのメガバンク評価

バークシャー・ハサウェイは日本の5大商社株を保有し続けており（直近持分6〜9%）、日本株への長期投資姿勢を示している。メガバンクへの直接保有は確認されていないが、商社株保有の文脈で以下の波及効果がある：

**結論③：** バフェット文脈はメガバンク株の「センチメントフロア」として機能。ショートを仕掛けた際にこのナラティブが復活すると踏み上げリスクが高まる。

### 4-1. WACC計算前提

銀行のDCFは事業DCFではなく**株主資本DCF（Equity DCF）**または**DDM（配当割引モデル）**が適切。ここでは**超過収益モデル（Excess Return Model）**と**DDM**の2軸で算出する。

#### WACC / 株主資本コスト（Re）

| パラメータ | 値 | 根拠 |
|-----------|-----|------|
| Rf（無リスク金利） | **2.409%** | JGB10Y（2026/04/17 財務省） |
| ERP（株式リスクプレミアム） | **5.5%** | 日本市場標準（Damodaran 2025推計） |
| β（MUFG、過去5年月次） | **0.95** | 金融株標準。市場連動性高いが防衛的側面もあり |
| Re = Rf + β × ERP | **2.409% + 0.95 × 5.5%** | = **7.634%** ≒ **7.6%** |

> **銀行セクターのDCFにおいてはWACCではなくReを直接適用するEquity DCFを採用。**

#### Re = **7.6%**（株主資本コスト）

### 4-2. DDM（配当割引モデル）

#### 2段階DDM前提

| フェーズ | 期間 | DPS成長率 | 根拠 |
|---------|------|----------|------|
| Phase 1（高成長） | FY2026〜FY2030（5年） | **+8%/年** | BOJ利上げ継続、自社株買い、増配（¥50→¥73推計） |
| Phase 2（定常成長） | FY2031以降（永続） | **2.5%** | 名目GDP成長率+やや上乗せ |

| 年度 | DPS（¥） | 割引率7.6% | PV（¥） |
|------|----------|-----------|---------|
| FY2026 | 54.0 | 1.076 | 50.2 |
| FY2027 | 58.3 | 1.158 | 50.3 |
| FY2028 | 63.0 | 1.246 | 50.5 |
| FY2029 | 68.0 | 1.341 | 50.7 |
| FY2030 | 73.5 | 1.443 | 50.9 |
| **Phase 1 合計PV** | | | **252.6** |

**Terminal Value（TV）= DPS2031 / (Re - g)**

```
DPS2031 = 73.5 × 1.025 = ¥75.3
TV = 75.3 / (7.6% - 2.5%) = 75.3 / 5.1% = ¥1,476
PV(TV) = 1,476 / 1.443 = ¥1,023
```

**DDM理論株価 = 252.6 + 1,023 = ¥1,276**

> ⚠️ DDMは配当のみをキャッシュフローとするため、自社株買いを含む**総還元ベース**に修正する。

#### 修正：総還元DDM（配当+自社株買いEPS換算）

自社株買い¥4,000億 ÷ 発行済株数（約113.5億株）= **¥35/株の実質追加配当相当**

修正DPS（Phase1）= ¥54 + ¥35 = **¥89**として再計算

```
Phase1 PV合計（DPS¥89基準、+8%成長）:
FY2026: 89.0/1.076 = 82.7
FY2027: 96.1/1.158 = 83.0
FY2028: 103.8/1.246 = 83.3
FY2029: 112.1/1.341 = 83.6
FY2030: 121.1/1.443 = 83.9
合計: 416.5

TV基準DPS = 121.1 × 1.025 = 124.1
TV = 124.1 / 5.1% = 2,433
PV(TV) = 2,433 / 1.443 = 1,686
```

**修正DDM理論株価 = 416.5 + 1,686 = ¥2,103**

### 4-3. 超過収益モデル（Excess Return Model）

```
理論株価 = BPS + PV(超過ROE × BPS)
超過ROE = ROE - Re = 6.1% - 7.6% = ▲1.5%（現在は超過収益マイナス）
```
