## 🎯 2. 社長指示最重点分析：「今から買うのは遅いか」— 株価の織込み度検証

> **社長指示優先度2（2026/04/06）：「肥料高騰の織込み度検証。乗り遅れリスクと継続成長余地を数値で判定せよ」**

**「相当程度は織り込み済み。ただし構造的成長テーマとして『終わった話』ではない。現値での新規フルポジションは勧めにくく、¥4,200前後の押し目を待つのが最善」**

### 2-2. 肥料高騰テーマの「織込み度」を数値で検証

#### ステップ①：PERで見る織込み度

| 指標 | 現在値 | 同業・過去水準 | 判断 |
|------|--------|--------------|------|
| PER | **17.3倍** | 過去5年平均：約15〜18倍 | ほぼ適正〜やや割高圏 |
| 種苗セクターグローバル平均 | 18〜22倍（Corteva等） | — | 国際比では割安余地あり |

→ **PER17.3倍は過去平均の上端。「肥料高騰特需」はすでに相当EPS予想に織り込まれている可能性が高い。**

#### ステップ②：PBRで見る織込み度

```
PBR = 1.17倍（現在）
BPS = ¥3,931

PBR の意味するもの:
  純資産に対し17%のプレミアム = 成長期待が「小〜中程度」
  ROE 7.0% × PER17.3倍 = PBR 1.21倍（理論整合性OK、概ね適正）

歴史的文脈:
  PBR1.17倍は「割安でも割高でもない中庸」水準
  肥料高騰テーマのピーク時（2022〜2024推定）はPBR1.5〜2.0倍に達した可能性
  → 現在はテーマ過熱が剥落した「適正水準」に戻っている
```

#### ステップ③：ROEとコスト・オブ・エクイティの比較

```
ROE = 7.0%
株主資本コスト（Re推定）= 2.380% + 0.75 × 5.5% = 6.5%
（β推定値：0.75 — ディフェンシブ種苗株）

ROE(7.0%) > Re(6.5%) → わずかに超過収益を創出
超過収益スプレッド = +0.5%pt（非常に薄い）

⇒ 現在の株価は「超過収益が永続する」とは織り込んでいない
⇒ 「肥料高騰で爆発的ROE拡大」シナリオは織り込まれていない
⇒ つまり、肥料高騰による業績急拡大が「まだ数字に出ていない」なら上昇余地あり
   逆に、すでに業績に反映済みなら現在のPERが割高感を示す
```

#### ステップ④：肥料価格と業績の連動性分析

**天然ガス（Henry Hub）現在値：2.79 USD/MMBtu**

```
肥料価格（特に窒素系）との関係:
  天然ガス ──→ アンモニア ──→ 尿素・硝安 → 肥料

  現在のHenry Hub: $2.79/MMBtu
  → 歴史的には「低〜中水準」（ピーク時2022年: $8〜9/MMBtu）
  → 肥料コストは既にピークから大幅緩和

重要な逆説:
  サカタのタネにとって「肥料高騰」の影響は以下の二面性がある:

  【ポジティブ面（テーマ側）】
  肥料高 → 農業コスト上昇 → 高収量・高効率品種への需要増大
  → サカタの付加価値種子の購買インセンティブ↑

  【ネガティブ面（コスト側）】
  農業資材セグメントの仕入コスト上昇
  農家の購買力低下 → 高価格種苗への支出抑制リスク

  現在の天然ガス$2.79は「ポジティブ面が剥落し、コスト面も緩和」の状態
  → 肥料高騰テーマとしての株式市場の注目度は低下局面
```
