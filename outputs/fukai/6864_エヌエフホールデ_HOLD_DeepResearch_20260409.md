### ⑤ 現在のバリュエーション分析

#### PBR・PERの矛盾問題

```
【データ整合性確認】
EPS（リアルタイム取得）：▲3円（赤字）
BPS（リアルタイム取得）：¥1,750

PER = 「取得失敗」
ミーティングメモ記載「PER27倍」との整合：
  → EPS▲3円（赤字）では本来PERは計算不能（赤字）
  → PER27倍は「過去の黒字EPS」または「会社予想EPS」に基づく可能性
  → または別の期のデータ混在の可能性
  → 深井判断：EPS▲3円の赤字局面では「PER27倍」は意味を持たない
  → 現状は「純粋なPBR評価」が適切
```

#### PBR0.76倍の妥当性

```
【PBR妥当性評価】

理論的適正PBR = ROE ÷ 株主資本コスト（Re）

ROE = ▲2.3%（赤字）
Re（推計）= Rf + β × ERP = 2.374% + 0.80 × 5.5% = 6.77%

理論PBR = ▲2.3% ÷ 6.77% = ▲0.34倍 → 理論上は0倍以下（解散価値以下が理論値）

ただし純資産（清算価値）担保があるため実際は0倍にはならない
→ 赤字企業のPBR妥当水準：解散価値の70〜85%程度
→ 適正PBR下限推計：0.55〜0.70倍
→ 現在PBR0.76倍は「赤字企業としてはやや割高」または「量子テーマプレミアム含む」
```
