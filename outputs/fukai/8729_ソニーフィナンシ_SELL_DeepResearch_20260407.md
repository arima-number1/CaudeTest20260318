### 2-7. 不動産含み益NAV算出

#### 不動産資産の推定

```
ソニーフィナンシャルグループ（主にソニー生命）の不動産保有：
  帳簿価額（推定）：約¥500〜800億円
  ├── 本社ビル（東京・港区）：約¥200〜300億円
  ├── その他事業用不動産：約¥200〜300億円
  └── 運用不動産（一部）：約¥100〜200億円

含み益推定（帳簿価額対比で20〜40%プレミアム）：
  含み益 = ¥500〜800億 × 25% = ¥125〜200億円（推定）
```

#### NAV算出

```
NAV = 純資産（簿価）+ 不動産含み益（税引後）+ 債券評価損（税引後）

ベースケース（現状金利）：
  純資産（簿価）   = ¥97/株 × 7.5億株 = ¥728億円
  不動産含み益（税後）= ¥125〜200億 × 70% = ¥88〜140億円
  債券評価損（税後） = ▲¥200〜▲¥400億円（現在金利水準での推定含み損）

  NAV合計 = ¥728 + ¥88〜140 − ¥200〜400 = ¥616〜528億円
  NAV/株 = ¥82〜71円/株

金利+100bp シナリオNAV：
  純資産（簿価）   = ¥728億円
  不動産含み益（税後）= ¥88〜140億円
  債券評価損（税後） = ▲¥8,232億円（前述試算）
  → NAV は大幅マイナス（理論上▲¥7,000億円超の毀損）
  → ただしOCI処理のため直接的な倒産リスクにはならないが、株式評価は理論上大幅ディスカウント
```

## 4. DCFバリュエーション

### 4-1. WACC計算

#### β推定


#### ERP（株式リスクプレミアム）

```

採用ERP：5.5%（Damodaran Japan推定ベース）
```

#### 株主資本コスト（Re）

```
Re = Rf + β × ERP
   = 2.386% + 0.90 × 5.5%
   = 2.386% + 4.95%
   = 7.336%
```

#### 負債コスト・資本構成

```
生命保険会社の「負債」は保険負債が主体（金融負債と異なる）
財務負債比率：推定D/(D+E) = 10〜15%（金融債務は小さい）
採用：D/(D+E) = 12%、E/(D+E) = 88%
Rd（負債コスト）：2.5%
```

### 4-4. DCFバリュエーション計算

#### ベースケース（WACC=6.7%、g=1.0%、正常化FCF=¥100）

```

  Year 1: ¥100 × 1.02 / 1.067 = ¥95.6
  Year 2: ¥102 × 1.02 / 1.067² = ¥91.3

  FCF Year 6 = ¥108 × 1.020 × 1.010 = ¥111.3
  TV = ¥111.3 / (6.70% - 1.00%) = ¥111.3 / 5.70% = ¥1,952.6
  TV PV = ¥1,952.6 / 1.067⁵ = ¥1,413.8
```
