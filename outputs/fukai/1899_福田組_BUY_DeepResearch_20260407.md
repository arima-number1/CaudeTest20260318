## 2-1. 低PBRの原因解剖：構造的問題 vs サイクル的問題

### 現状確認
```
PBR:   0.80倍（リアルタイム取得値）
BPS:   ¥10,845
現在株価: ¥8,640
純資産ディスカウント: ▲¥2,205/株（▲20.3%）
```

### 建設コスト上昇の実態分析

建設コスト上昇が利益率を圧迫している**証拠構造**を以下で解剖する。

#### A) コスト上昇の内訳（業界データ）

| コスト項目 | 上昇率（2020年比） | 影響度 | 転嫁難易度 |
|-----------|-----------------|--------|-----------|
| 鉄鋼（H形鋼・鉄筋） | +35〜45% | ★★★★ | 中（公共工事は単価改定あり） |
| セメント・コンクリート | +20〜30% | ★★★ | 中 |
| 労務費（大工・鳶・型枠） | +25〜40% | ★★★★★ | 低（民間契約では困難） |
| アスファルト・重油系 | +40〜55% | ★★★ | 高（公共単価連動） |
| 木材・合板 | +20〜35%（一部落ち着き） | ★★ | 中 |
| 設備・電気機器 | +15〜25% | ★★ | 低〜中 |

**WTI原油$112.79**は建設用重機燃料・アスファルト原料コストに上昇圧力。ただし公共工事のインフレスライド条項で一定程度カバーされる。

#### B) 構造的問題 vs サイクル的問題の判定マトリックス

```
┌─────────────────────────────────────────────────────┐
│           深井判定：「サイクル的問題が主因、構造的問題は副因」   │
└─────────────────────────────────────────────────────┘

【サイクル的問題（回復可能）】
────────────────────────────────
✓ 資材高騰→公共工事単価改定ラグ（1〜2年で解消）
✓ 受注した工事の採算悪化（フロー問題：新規受注から改善）
✓ 2024年問題による工期延長コスト（2025年以降は適正化）
✓ 金属・セメント価格の一部高原回復
✓ 能登復興→受注量増加で固定費吸収効率改善

【構造的問題（長期課題）】
────────────────────────────────
△ 少子化による建設技術者の慢性不足（長期的コスト上昇圧力）
△ 地方ゼネコンの競争激化（準大手の地方侵食）
△ 環境規制（カーボンニュートラル対応投資必要）

【判定根拠】
ROE 6.4% → 資本コスト(Re推定6〜8%)との乖離が小さい
→ 純資産毀損ではなく「利益率の一時的圧縮」が主因
→ 受注量・価格転嫁が改善すれば2〜3年でROE8〜10%回復射程
```

### 低PBRの真の原因構造

```
低PBR 0.80倍の原因ツリー
│
├─① 利益率の一時的圧縮（サイクル的・70%ウエイト）
│   └─建設コスト上昇×価格転嫁ラグ→営業利益率3〜4%台
│
├─② 資本効率の低さ（構造的・20%ウエイト）
│   └─ROE 6.4% < 株主資本コスト推定6.5〜7.5%
│     →PBR=ROE/Re の理論上1.0倍未満は合理的
│
└─③ 情報開示・IR戦略の弱さ（構造的・10%ウエイト）
    └─不動産含み益・NAVの未開示
      アクティビスト未介入→株主還元加速への外部圧力なし
```

**深井コメント:** PBR 0.80倍の最大要因は「建設コスト上昇による一時的な利益率圧縮」と「ROEが資本コストをわずかに下回る構造」の複合。純資産の質自体は悪くなく、むしろ含み益を内包する可能性が高い。
