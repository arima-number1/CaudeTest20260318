## 4. 核心テーマ②：需要の価格弾力性──おむつは売れ続けるか

### 4.1 価格弾力性の国・セグメント別マトリクス

```
【おむつ需要の価格弾力性分析（深井推定）】

                弾力性（絶対値）
          低 ←────────────→ 高
          |                    |
日本 ─── ●────────────────────
（乳児）  弾力性≈0.1〜0.2
（大人）  弾力性≈0.15〜0.25

インド ───────────●───────────
（普及率が低く          弾力性≈0.4〜0.6
 まだ「必需品化」途上）  → 価格上昇でも
                        数量は伸び続ける

インドネシア ──────────────────●
（現地ブランド代替が   弾力性≈0.7〜0.9
 容易な市場）         → 値上げ → 代替品に流出

中国 ────────────────────────●
（既に過剰競争市場）   弾力性≈0.8〜1.1
                      → 値上げ不可能。むしろ
                        価格を下げないと負け続ける
```

### 4.2 日本市場：「防衛的消費財」としての確固たる地位

**乳児用おむつ（日本）**:
- 出生数: 2025年推定70万人台（過去最低水準更新中）
- 市場規模: 約¥1,440億円（微減）
- ユニ・チャームシェア: 約52% → **安定的**
- 値上げ後の行動: 消費者はブランド内でグレードを下げるが、他社には切り替えない
- 理由: 「肌荒れしたらどのブランドのせいか」が不明確なため乗り換えコストが心理的に高い

**大人用おむつ（日本）**:
- 市場規模: 約¥1,150億円（年率+4%成長）
- 高齢化により需要は**構造的に右肩上がり**
- 価格弾力性は乳児用より低い（介護者・医療機関が購入→費用は社会保障で一部負担）
- ユニ・チャームの利益率改善の主役は**大人用おむつ事業**

**深井の結論**: 日本市場は「値上げしても売れ続ける」典型例。

### 4.3 中国市場：「値上げ不可能」の地政学的チキンレース

```
【中国市場の実態（2024〜2026年）】

ユニ・チャームの中国実売動向:
  2021年12月（ピーク）→ 2024年12月: 累計▲20%超（実質ベース）

崩壊の要因:
  ① 2024年「衛生材料風評被害」: SNSで製品品質疑惑→小売・卸が在庫凍結
  ② 中国消費者の「質倹生活（省倹消費）」: 日本ブランドプレミアム剥落
  ③ 花王式（怡丽佳儿）、百亿补贴式などの国内ブランドが価格×品質で肉薄

→ 中国では値上げは「自殺行為」。むしろ値下げプレッシャーがある。
→ ユニ・チャームは2025年以降、EC重点×マーケティング投資で回復を図る。
→ 回復は2026年からとされるが、これはイラン戦争前の計画である。
```

### 4.4 インド市場：「値上がりしても普及率向上で数量が伸びる」奇跡

- インドのおむつ普及率: 現在わずか3〜5%（日本は95%以上）
- MamyPoko（ユニ・チャーム）がPampers（P&G）を抜いて**インドNo.1ブランド**に
- ラジャスタン州第3工場稼働（2025年2月）で供給能力増強
- インド市場は「価格が多少上がっても需要が増える」──普及率が低いため一定の値上げは問題にならない
- **深井の評価**: インドは2026〜2028年の成長エンジン。ここが崩れなければユニ・チャームの長期テーゼは維持される。

## 5. 核心テーマ③：6279瑞光への波及──装置発注蒸発リスク定量化

### 5.1 ユニ・チャームと瑞光の関係性

```
【ユニ・チャーム・瑞光の需給関係】

ユニ・チャーム
  └─ 瑞光は「主要機械サプライヤー」（Zuiko公式IR情報）
  └─ 乳児用・大人用おむつ製造機 → 全量または大部分が瑞光製と推定
  └─ ユニ・チャームの設備投資計画 ≈ 瑞光の受注計画 と直結
```

### 5.2 ユニ・チャームFY2026設備投資計画の衝撃

```
【設備投資 vs 減価償却の比較（FY2026）】

FY2026 設備投資計画（会社発表）: ¥4,000億円... ではなく
→ 正確には: **¥400億円（¥40bn）**

FY2026 減価償却費: **¥470億円（¥47bn）**

純設備投資（Capex - D&A）= ¥400億円 - ¥470億円 = **▲¥70億円（純マイナス！）**
```

> **深井の警告**: CapexがD&Aを下回ることは、**既存設備の更新すら完全には行わない**ことを意味する。ユニ・チャームは資産を「食いつぶしながら」、その資金を戦略的M&AやDX投資に振り向けている。新規ラインへの投資余力は極めて限定的。

### 5.3 「脱製造業」戦略が瑞光に与える構造的破壊

**第13次中期経営計画（FY2026〜FY2030）の核心転換**:

```
【従来のビジネスモデル（旧）】
ユニ・チャームが自社工場を建設
    ↓
瑞光の製造機を購入・設置
    ↓
自社ライン増設 → 瑞光への継続発注

【新戦略「脱製造業」（新）】
新興国展開では「OEM/ODM」パートナー企業に製造委託
    ↓
ユニ・チャーム自身は工場を建てない
    ↓
瑞光への装置発注が「原理的に発生しない」
```

**第13次中計の数値目標（FY2026〜FY2030）**:
- 5年間の成長投資総額: ¥3,900〜4,300億円（年平均¥780〜860億円）
- うち「戦略的M&Aリザーブ」: ¥1,500〜1,600億円
- 設備投資: 年平均¥400〜450億円（推定）
- 減価償却: 年平均¥480〜500億円（推定）
- **純設備投資は5年間を通じてマイナスまたはほぼゼロ維持の公算**

### 5.4 装置発注減少の定量推計

```
【ユニ・チャームから瑞光への発注影響モデル（深井試算）】

前提:
・ユニ・チャームは過去5〜7年、毎年10〜20ライン分を瑞光から調達（推定）
・1ライン = 瑞光売上換算で約5〜15億円（設置・調整含む）
・過去のユニ・チャーム向け売上 = 瑞光総売上の推定20〜35%
  （瑞光売上211億円の20〜35% = 年間42〜74億円）

シナリオ別影響試算（FY2027以降・年次ベース）:

①「脱製造業」影響（戦略的要因）:
  OEM転換比率: 新規展開の30〜50%をOEM化
  発注減少率: ▲15〜25%
  瑞光売上影響: ▲6〜19億円/年

②原材料高騰による「採算悪化→凍結」（短期的要因）:
  前回分析済み: ▲20〜35億円（FY2027）
  ただし既発注分は履行される可能性が高い

③二要因の重複:
  実質的な影響は単純合算ではなく、
  「新規発注が戦略理由でも採算理由でも出ない」という
  相乗効果になる可能性が高い

統合影響推定（FY2027〜FY2028の瑞光売上への押し下げ）:
  ▲25〜50億円/年（従前比較）

瑞光FY2027目標270億円に対する乖離:
  目標270億円 → 実質推定220〜245億円
```

> **深井の判断**: 瑞光の「中計目標270億円」は、最大顧客であるユニ・チャームの「脱製造業」という構造転換を全く織り込んでいない。中計は根本から組み直しが必要だ。前回のSTRONG SELL判断（FV¥800）は**追確認**される。

### 5.5 サウジアラビア工場リスク（追加リスク）

ユニ・チャームはサウジアラビアに**3つの製造工場**を持つ（Unicharm Gulf Hygienic Industries）。これらはイラン戦争の最前線エリアに近接している。

