# 1. 企業概要・事業分析

## 1.1 ホールディングス体制移行の意義と「Play fashion!」プラットフォーマー戦略

2026年3月に「アダストリア」から「アンドエスティHD」へと商号変更しホールディングス体制へ移行した。この移行は単なるリブランディングではなく、**事業ポートフォリオの再定義を明示するシグナル**である。

従来の「ファッション小売業」という自己定義を脱し、**「Play fashion! プラットフォーマー」**として：

1. **グループ内ブランドを束ねる販路インフラ（and ST）**
2. **グループ外ブランドのDTC支援（B2B2C）**
3. **グローバル展開の持株会社的統治**

という三層構造を法人格上も明確化した。ホールディングス化により、事業別の資本効率評価・投資配分の透明化・将来的な事業分離・上場オプションが生まれる。特に**and STの外部開放（グループ外ショップ53社）**はHD化なくして推進困難であり、構造的必然性がある。

### 評価

> ホールディングス体制移行は「プラットフォーマー化」という中計2030の根幹戦略を実装するための**不可欠な組織的インフラ整備**である。短期的に経営管理コストは増加するが、中長期の企業価値創造において正当化される。評価：**戦略的合理性は高い（◎）**

## 1.3 and STのプラットフォーム化が競争優位性に与える構造的影響

```
【プラットフォーム経済の構造的優位性】

  グループ内ブランド（グローバルワーク、レプシィム、ラコレ等）
           ↓
       and ST（ファッションDTC OS）
           ↑
  グループ外ブランド（53社→拡大中）
           ↓
    約2,170万会員データベース
           ↓
  データドリブン在庫最適化・レコメンデーション
```

**構造的優位性の源泉：**

1. **ネットワーク外部性**：グループ外ブランド増加→会員利便性向上→会員増→ブランド誘致力強化というポジティブフィードバックが始動
2. **データの護城河**：2,170万会員の購買データは競合が短期に複製不能。在庫精度向上（≒値引きロス低減）に直結
3. **クロスセル機能**：1会員が複数ブランドを回遊→LTV向上
4. **収益多様化**：グループ外GMV手数料収入という「家賃モデル」が確立すると、自社ブランドの業績変動に対するバッファーになる

**現時点の限界：**
- グループ外GMV45億円（45/462 = 9.7%）はまだ「補完」の域。プラットフォーム事業単独で損益が見えない
- Zozotown（会員9,200万人超）とのスケール差は依然大きい

**深井の評価：**
> and STは「アパレル版プラットフォーム」の萌芽であり、完成すれば企業価値の再評価（マルチプル拡張）につながる。ただし現時点では「プラットフォーム企業」より「大型アパレル小売＋育ちかけのプラットフォーム」という評価が適切。**5年後の本格的プラットフォーム化を前提としたバリュエーションは時期尚早**だが、オプション価値として織り込む余地あり。

# 2. FY2026/2決算精査

## 2.1 売上総利益率54.6%（計画未達）の構造的原因分析

| 項目 | FY2025/2 | FY2026/2 | 計画 | 実績差 |
|------|---------|---------|------|--------|
| 売上総利益率 | 54.7% | 54.6% | 55.0%程度（推定） | **-0.4pt未達** |
| 販管費率 | 49.4% | 49.1% | — | -0.3pt改善 |
| 営業利益率 | 5.3% | 5.4% | — | +0.1pt |

**未達の構造的原因（2要因に分解）：**

### 要因①：在庫値引き（Markdown）
- FY2026/2前半（1Q-3Q）はグローバルワーク低調に伴う**在庫消化圧力が発生**
- 特に主力3社の営業利益が-9.5%というのは、在庫の強引な値引き処分が粗利を毀損したことを示す
- グローバルワーク売上構成比22%で+2.2%しか伸びなかったことは「低速消化→値引き促進」の構図

### 要因②：計画超のポイント付与
- and STのアクティブ会員拡大施策として**ポイント付与率を引き上げた**と推察
- 売上高計上後のポイント引当は実質的な粗利の圧縮として機能
- アクティブ会員約780万人への施策は顧客囲い込みという長期投資だが、**短期粗利には逆風**

### 深井の評価
> -0.4ptの未達は「在庫値引き（構造的）」と「ポイント施策（戦略的）」の合成。前者は在庫管理精度向上で解消方向、後者は顧客LTVへの先行投資と評価できる。ただし**計画精度の低さ**（特に粗利率予測）は経営の問題点として記録しておく。4Qの正価販売比率上昇（3月月次で確認）は改善の兆しを示す。

## 2.2 4Q単独営業利益+255.5%YoYの質的評価

**「前年の低ハードル」vs「真の改善」論点整理：**

### 前年（FY2025/2 4Q）の状況
- 3Qまで営業減益が継続していた年度
- 4Q（2024年12月-2025年2月）は暖冬影響による秋冬物低調が推測される
- すなわち**前年比較基準（ベース）が極めて低い**

### FY2026/2 4Qの実態
1. **売上の確認**：4Q単独 売上+5.4%（前年比）→それ自体は正常水準
2. **グローバルワーク「緩やかに回復」**：3Q以降の記述は実態改善を示す
3. **来期3月月次（+5.6%全店）**：4Qの回復が連続性を持って続いていることを示す
4. **正価販売比率上昇**：3月月次で客単価-3.2%かつ客数+9.0%は、「安売りで客を増やした」ではなく「正規価格でも来客増」の構図

### 深井の判断
> 4Q+255.5%の約**3分の2は低ベース効果**、約**3分の1は真の改善**と評価する。決定的に重要なのは、4Qの回復が「一時的な滑り込み」でなく、**FY2027/2 3月月次（+5.6%）として翌期に引き継がれている**という事実だ。この連続性が確認できる以上、4Q回復の信頼性は一定程度高い。ただし「2Q・3Qでの低調→4Qで挽回」というパターンの繰り返しリスクは注視すべき。

## 5.3 3ケースDCF計算

### 【基本ケース】Terminal growth rate g = 2.0%

```
WACC = 6.636%、g = 2.0%

割引率計算：
PV(FCF合計) = Σ FCFt / (1+WACC)^t

t=1: 79億 / 1.06636^1 = 74.1億
t=2: 101億 / 1.06636^2 = 88.8億
t=3: 121億 / 1.06636^3 = 100.0億
t=4: 146億 / 1.06636^4 = 113.1億
t=5: 169億 / 1.06636^5 = 122.5億

PV(FCF合計) = 498.5億円

Terminal Value = FCF5 × (1+g) / (WACC - g)
= 169億 × 1.02 / (0.06636 - 0.02)
= 172.4億 / 0.04636
= 3,718.5億円

PV(TV) = 3,718.5億 / (1.06636^5) = 2,695.2億円

Enterprise Value = 498.5 + 2,695.2 = 3,193.7億円

Equity Value = EV + 現預金 - 有利子負債
= 3,193.7 + 249 - 0 = 3,442.7億円

理論株価 = 3,442.7億円 ÷ 46.2百万株
= ¥7,452（注：見直し後→後述）
```

5. **流動性・機関投資家行動**

> **深井の見立て：** 株価不反応は「短期的なカタリスト不足」と「中計乖離への失望」の合成であり、**本源的価値の毀損ではない**。中期投資家にとってはこの「不反応」こそがエントリー機会を提供している。

# 9. 金利感応度分析（Sensitivity Analysis）

## WACC計算式の確認

```
Re = Rf + β × ERP
   = Rf + 0.85 × 5.0%
   = Rf + 4.25%

WACC = Re（無借金経営）
```

## 金利感応度テーブル

| Rfシナリオ | Rf(%) | WACC(%) | 理論株価（JPY）| 現在値比乖離率 |
|-----------|-------|---------|--------------|------------|
| **-1.00%** | **1.386%** | **5.636%** | **¥4,893** | **+67.6%** |
| **-0.50%** | **1.886%** | **6.136%** | **¥4,318** | **+47.9%** |
| **-0.25%** | **2.136%** | **6.386%** | **¥4,067** | **+39.3%** |

| **+0.50%** | **2.886%** | **7.136%** | **¥3,470** | **+18.9%** |
| **+1.00%** | **3.386%** | **7.636%** | **¥3,156** | **+8.1%** |

※ 各シナリオはTerminal growth rate g = 2.0%（基本ケース）を固定

### 計算根拠（WACC→理論株価の変換）

Terminal Valueは `FCF5 × (1+g) / (WACC-g)` の構造であり、WACCが低下するほど分母が小さくなり（0.04636→0.03636等）、TV/EVが指数関数的に拡大する。

### 重要観察

1. **WACC+1.00%（Rf=3.386%）でも理論株価¥3,156**、現在値比+8.1%の割安を維持
2. 金利が更に上昇してRf=4.386%（WACC=8.636%）になると理論株価は¥2,919に接近
3. **日銀の利上げペース（+0.25〜0.50%/年程度）の範囲内であれば割安は維持される**
4. 無借金経営はWACCの上昇抵抗力を高める（有利子負債があれば金利コスト増として直撃）

## 11.2 円安進行シナリオ（USD/JPY=158円）

### 仕入コストへの影響

FY2027/2会社計画策定時の想定レート（推計）は概ね**145〜150円/USD**と推定される。これがUSD/JPY=158円（+5〜8%円安）に進行した場合の影響を試算する。

```
【円安影響試算：USD/JPY=158円】

前提：FY2027/2 売上原価（推計）= 3,140億円 × 45% = 1,413億円
     うち外貨建て仕入比率（推計）= 60%（アジア生産比率）
     → 外貨建て仕入額（推計）= 848億円

想定レート（推計）145円/USD → 実勢158円/USD
円安影響率 = (158-145)/145 = +8.97%（≒+9%）

仕入コスト増加額（推計）= 848億円 × 9% × 為替パスルー率（推計70%）
　　　　　　　　　　　　= 848億円 × 6.3%
　　　　　　　　　　　　= ≒ +53.4億円
```

> **為替パスルー率70%**：外貨建て仕入のうち100%が即座にコストに反映されるわけではない（期中在庫・FOB価格交渉等でバッファーあり）。70%はアパレルSPA業界の実務的な感応度として設定。

### アジア生産比率と為替感応度

アンドエスティHDは生産拠点を中国（推計30%）・バングラデシュ（推計25%）・ベトナム（推計20%）・その他アジア（推計15%）に分散させている（推計）。中国の人民元はUSDに対してペッグ性が高く、バングラデシュ・ベトナムのUSD建て決済比率も高い。結果として、円安はほぼ直線的に仕入コストに転嫁される構造を持つ。

|-----------------|------------|---------|
| USD建て（中国・バングラ・ベトナム） | 60% | 高 |
| EUR建て（欧州素材） | 5% | 中 |
| 円建て（国内素材・賃加工） | 35% | なし |

# Section 12: 4Q定性変化考察

## 12.1 FY2026/2 4Q（11月〜2月）営業利益+255.5%の背景分析

### 4Q急回復の構造要因：深掘り

FY2026/2の4Q単体営業利益は約68億円（推計）、前年同期比**+255.5%**という異例の急回復を記録した。この数字を額面通りに受け取るのではなく、4つの構造要因に分解して解析する。

---

**① 秋冬物の価格消化率改善（在庫適正化の果実）**

FY2025/2（前年）4Qにグローバルワークを中心とした秋冬物の大量値引き処分が発生し、粗利率を大きく毀損した。FY2026/2は春先から秋冬仕入量をコントロールし、シーズン末（1〜2月）の正価消化率が大幅に改善した。

```
【在庫コントロール効果（推計）】
FY2025/2 4Q：値引き処分比率（推計）= 30%以上
FY2026/2 4Q：値引き処分比率（推計）= 20%程度
  → 粗利率改善効果（推計）= +2〜3pt
  → 金額換算（4Q売上829億円ベース）= +16〜25億円（推計）
```

これが4Q急回復の最大要因と推定される。在庫コントロールの成功は、**商品企画精度の向上（and STのデータ活用）**と**仕入量の意図的絞り込み**の複合成果である。

---

**② グローバルワーク秋冬企画の立て直し**

FY2025/2の失敗は、「アウターのデザイントレンド読み違いと過剰SKU展開」にあったとされる。FY2026/2では①SKU数を絞り込み、②コアアイテム（ベーシックフリース・機能性インナー）への集中、③and STデータを用いた発注量の最適化、が実施された。

結果、グローバルワークの4Q既存店は推計で**+5〜8%**（全社4Q月次の平均を上回る水準）に回復したと推定される。ブランド毀損から1年での急回復は、アンドエスティHDの商品開発PDCA速度の高さを示す証左である。

---

**③ コスト管理（販管費コントロール）の貢献**

4Qは売上高が通期最大（約829億円、通期の約28%）となる一方、固定費は年間均等発生する。この「売上レバレッジ」が最大化するのが4Qの構造的特性である。加えてFY2026/2では以下のコスト抑制が追加的に寄与した。

| コスト項目 | FY2025/2 4Q（推計） | FY2026/2 4Q（推計） | 変化 |
|-----------|------------------|------------------|------|
| 物流コスト | 高止まり | 福岡DC稼働効果で改善 | ▼ |
| 広告宣伝費 | 高水準 | and ST中心にデジタル比率向上 | ▼ |
| 減損・特損（Velvet等） | 軽微 | 計上済み（特損として区分） | — |
| 採用・教育費 | 増加傾向 | 安定化 | → |

---

**④ 前年同期（FY2025/2 4Q）の異常な低迷をベースとした反動効果**

FY2025/2 4Qの低迷は「単なる不調」ではなく、グローバルワーク在庫処分・米国Velvet関連損失・台湾・韓国などアジア事業の苦戦が重なった**複合的な異常値**であった。

```
【ベース効果の定量化（推計）】
FY2025/2 4Q 営業利益（推計）: 約19億円（異常低水準）
FY2026/2 4Q 営業利益（推計）: 約68億円
差額                         : +49億円

うち：構造的改善（実力値向上）: 推計+30億円（61%）
　　 ベース効果（前年が異常低）: 推計+19億円（39%）
```

**重要な含意**：4Q+255.5%の約**4割はベース効果**に起因する。FY2027/2 4Qは「前年が高水準」となるため、同四半期の前年比較は自然と穏やかになる。**通期業績は4Qのベース効果が薄れる分、1Q〜3Qの改善継続が鍵**となる。

---

> 🔷 **深井の評価：4Q+255.5%の実質は「+150〜200%」、それでも十分に強い**
>
> ベース効果を除いた実力ベースの改善幅は推計+150〜200%程度と見積もられる。それでも「在庫コントロール→粗利率回復→コスト効率向上」の3連鎖が同時に機能したことは、経営の質的改善を示す強いシグナルだ。特に注目すべきは、これが「一過性のコスト削減」ではなく「商品企画プロセスの構造改善」に根ざしている点である。and STデータを活用した発注精度の向上は、来期以降も持続的な粗利率改善をもたらすと期待する。

## 12.2 トゥデイズスペシャル（TS）減損とHD化の影響

### のれん23.34億円減損の一時性 vs 構造的ブランド価値棄損の見極め

トゥデイズスペシャル（TS）は「食とライフスタイル」を融合させたセレクトショップ型ブランドとして展開されてきた。FY2026/2に計上されたのれん・無形資産減損額23.34億円の性質を以下で評価する。

**一時的要因（楽観シナリオ）**：
- 店舗収益性の一時的悪化（コロナ後の百貨店立地苦戦）
- HD化に伴う会計的再評価（組織変更時ののれん見直し）
- 食品EC・オムニチャネル展開の先行投資フェーズ

**構造的要因（懸念シナリオ）**：
- 「食×ファッション」のコンセプト融合ビジネスの収益化困難
- 高価格帯セレクト市場の競争激化（北欧暮らしの道具店・212キッチンストア等との競合）
- and ST会員との客層ミスマッチ（TSはコアターゲットが40〜50代、and STは20〜30代）

```
【減損後のTS資産価値（推計）】
減損前のれん・無形資産（推計）: 35〜40億円
減損計上額                    : 23.34億円
減損後残存価値（推計）         : 12〜17億円

→ 残存価値が示すのは、ブランドとして「完全清算」ではなく
  「縮小均衡での事業継続」が会計的に正当化された範囲
```

**深井の見立て**：23.34億円の減損は**「一時性60%：構造的40%」**の混合と評価する。TSは今後、and STエコシステムの中でライフスタイル提案特化ブランドとして再定義される方向で位置づけられる。TSが持つ「食・器・暮らし」領域の高感度編集力は、アパレル主体のand ST会員に対するクロスセル導線として機能し、グループ全体の客単価押し上げに寄与する可能性を持つ。減損処理によってのれんの過大評価リスクは概ね払拭されており、今後はゼロベースの収益貢献を問われるフェーズへと移行した。TSに対する市場の懸念は「のれん追加減損リスク」から「利益貢献の再証明」へとシフトしており、投資家としては次の1〜2期の損益推移を厳格にモニタリングすることが求められる。

## 13. 不動産含み益・NAV考察

## 13.1 アパレル企業における不動産資産――なぜアンドエスティHDにNAV視点が必要か

アパレル企業のバリュエーションは通常、EV/EBITDAやPERが中心となる。しかしアンドエスティHDの場合、以下の理由からNAV（Net Asset Value）視点を補助軸として加えることが有効である。

第一に、**差入保証金（敷金）の規模**が相対的に大きい点。全国900店舗超を運営するグループ企業として、退去時に返還を受ける敷金残高は推計150〜200億円に上ると見られる。これはBSに資産計上されているものの、流動性が低く市場が適正評価しにくいオフバランス的な性格を持つ。

第二に、**福岡物流センター売却という先行事例**が存在する点。FY2024/2に34億円の売却益を計上したこの取引は、グループが必要に応じて不動産資産をキャッシュ化できることを実証した。

第三に、**TSのれん・無形資産の減損（23.34億円）**によって、BSの資産質が改善した点。過大評価されていた無形資産が整理されたことで、修正NAVの信頼性が高まった。

## 13.2 主要不動産関連資産の推計

| 資産区分 | 規模（推計） | 含み益/損 | 備考 |
|---------|-----------|---------|------|
| 差入保証金（敷金） | 150〜200億円 | —（帳簿価額と一致） | 退去時返還前提、流動性低 |
| 福岡物流センター | 売却完了 | +34億円（FY2024計上済） | キャッシュ化実績 |
| のれん・無形資産（TS） | 減損後12〜17億円（推計） | 棄損確定：▲23.34億円 | 追加減損リスクは限定的と判断 |
| 米国Velvet関連資産 | 売却完了 | 売却損計上済（FY2026/2） | 事業撤退完了 |
| 自社所有不動産（推定） | 未開示・限定的 | 不明 | アセットライト経営の結果 |

**重要な解釈**：差入保証金150〜200億円は含み益ではなく、あくまで「確定的に返還される性格の資産」として、修正NAV計算において液体資産に近い扱いが可能である。ただし退去タイミングは不確定であり、全額即時現金化は不可能である点は留意が必要だ。

### 「アセットライト＝高ROEの源泉」という逆説

一般的にNAVが大きい企業は「含み益が豊富」として評価される。しかしアンドエスティHDの場合、**不動産をほとんど自社保有しないアセットライト構造**こそが高ROEの源泉であり、これは逆説的にNAV分析でポジティブに評価されるべき点だ。

自社保有不動産が少ない＝固定資産の膨張がない＝資産回転率が高い＝ROEが向上する、というロジックは、and STプラットフォームという「データ資産＝無形の高収益資産」によって代替されている。差入保証金という形で退去時に返還される資産が積み上がっていることは、財務の安全弁として機能しつつも、日常的な投資対象としては適切にアンダーウェイトされてきた。

## 13.4 まとめ：NAVから見たInvestment Thesis補強

アンドエスティHDの投資価値の本質は、**アセットライト×データ資産×2,170万会員という無形の競争優位**にある。不動産含み益の絶対額は限定的であるが、それ自体がアセットライト経営の証明であり、資本効率の高さを裏付ける。NAV分析の結論として：

- 修正NAVは¥3,500〜¥4,200のレンジを支持
- 現在株価¥2,919はNAV下限に対しても▲17%のディスカウント状態
- 「不動産含み益が小さい」ことを弱点と見るのではなく、**資産の質＝データ・ブランド・会員基盤という無形固定資産への転換**として評価するべきである

NAV視点は単独では投資判断の決め手にはならないが、DCFベースの理論株価¥3,847と整合的な結論を導いており、**バリュエーション乖離の多面的な裏付け**として機能する。

4Q FY2026/2の営業利益+255.5% YoYという劇的な回復は、暖冬在庫の正常化・Velvet売却完了・経費コントロールの三位一体によってもたらされた。これが一過性で終わらない証拠が、翌期最初の月次データ（3月）における全店+5.6%・客数+9.0%という数字だ。業績回復のバトンは確実に次のフェーズへ渡されている。

### 第3柱：構造的競争優位（and ST×2,170万会員×外部開放加速）

and STプラットフォームの2,170万会員は、競合他社が容易に複製できない資産である。外部ブランドへのECプラットフォーム開放が進めば、GMVの拡大はアンドエスティHD自体のアパレル販売とは独立した収益軸を生み出す。これは従来の「アパレル専業企業」の枠を超えた**プラットフォームバリュー**であり、市場はまだこの価値を十分に織り込んでいない。

## 14.4 深井の最終メッセージ

アンドエスティHDへの投資の本質は、「回復局面のアパレル企業を買う」という単純な話ではない。これは**HD化という組織変革と、and STというデジタルプラットフォームの二重の転換点に、割安な入り口で参加できる稀有な機会**だと私は認識している。¥2,919という株価は、FY2026/2の苦難――暖冬・Velvet問題・TSのれん減損――を全て知った上でついた値段だ。悪材料の多くはすでに株価に反映されており、今後の株価上昇のトリガーは「悪材料の消滅」ではなく「構造的成長の顕在化」になる。

HD化×and STプラットフォームという中長期変革ストーリーは、2〜3年のスパンで見たときに、アンドエスティHDを「アパレル企業」から「ライフスタイルプラットフォーム企業」へと変貌させる可能性を秘めている。2,170万人の会員データを保有し、外部ブランドへのプラットフォーム開放が進めば、GMVの拡大は自社アパレル販売の季節変動に左右されない安定収益軸を生む。市場がこの変化を「アパレルの粗利改善」としか見ていない間は、構造的な割安が続く。これは長期投資家にとっての特権的な時間軸だ。

中東リスクとグローバルワークの低迷は、短期的には確かに重荷である。しかし逆の見方をすれば、この二つのリスクが解消された瞬間に、業績には**対称的な上方レバレッジ**がかかる。合繊コストが正常化すれば粗利率は数十bp単位で改善し、グローバルワークがブランド再生に成功すれば、グループ全体の月次は一段上の成長軌道に乗る。今の株価はそのレバレッジをゼロと見積もっているが、実際には相当の潜在的押し上げ力が内在している。問題が解決されていない今こそが、その恩恵を安価に仕込める局面だ。

¥2,919で買う意味を一言で言えば、「転換の代価が最も安い瞬間に乗ること」だ。4Q FY2026/2の+255.5%という営業利益の急回復、3月月次の客数+9.0%という足元のデータ、そして不動産売却・Velvet処理による財務整理の完了――これらが重なったこの瞬間は、中期的に振り返ったとき、アンドエスティHDの投資ストーリーにおける「最も説明しやすい買い場」として記録されるはずだ。私が¥3,847を基本目標株価とし、強気シナリオで¥4,200を掲げるのは、楽観的な夢物語ではなく、FY2027/2計画数値と合理的なマルチプルから機械的に導かれた結論である。あとはカタリストが一つずつ積み重なるのを、冷静に見届ければいい。
