## 6. DCFバリュエーション（大阪テーマシナリオ別）

### 6.1 WACC計算

**前提パラメータ:**

| パラメータ | 値 | 根拠 |
|----------|-----|------|
| Rf（リスクフリーレート） | **2.380%** | 日本10年国債（社長確認値） |
| β（ベータ） | 0.75 | 中小建設株・地場ゼネコン平均（市場感応度低め） |
| ERP（株式リスクプレミアム） | 5.5% | 日本市場標準（Damodaran2025値） |
| Re（株主資本コスト） | **2.380% + 0.75 × 5.5% = 6.505%** | — |
| Rd（負債コスト） | 1.5% | 中小建設業の借入金利水準 |
| Tc（実効税率） | 30% | 日本法人税率 |
| E/(D+E)（株式比率） | 90% | 低レバレッジ企業 |
| D/(D+E)（負債比率） | 10% | 有利子負債少 |

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1−Tc)]
     = [0.90 × 6.505%] + [0.10 × 1.5% × (1−0.30)]
     = 5.855% + 0.105%
     = 5.96% ≒ 6.0%
```
