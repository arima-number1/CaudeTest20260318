## 3. アンソロピックショック考察（最重要）

### 3-1. 「アンソロピックショック」とは何か

社長が定義する「アンソロピックショック」とは、Anthropic社が以下を積極展開することにより、グロービングのような国内AI企業が受ける影響を指す：

| Anthropic製品 | 展開内容 | グロービングへの影響方向 |
|--------------|---------|----------------------|
| Claude 3.5 / Claude 4系 | API料金継続引き下げ・能力向上 | **複合（コスト低下↑・競合↓）** |
| Claude Code | コーディング自動化で人月単価圧縮 | **競合リスク（一部）** |
| Claude API / Bedrock統合 | AWS/AWSパートナー経由で国内普及 | **補完（採用促進）** |
| Claude for Enterprise | 大企業直販・Anthropicコンサル強化 | **競合リスク（高い）** |
| Claude.ai Pro/Team | 個人・チーム向け普及 | **中立〜わずかに競合** |

### 3-2. 競合か補完か — 深井の最終判断

**結論：現段階では「70%補完・30%競合」の構造と推察される。ただし3〜5年で比率が変化するリスクあり。**

#### 補完要因（70%）

**① AI技術の民主化がコンサル需要を創出**
AnthropicがClaude APIの能力を向上させると、日本企業が「使いたいが使い方が分からない」「既存システムに統合できない」という需要が爆発的に増加する。グロービングのような実装支援企業への引き合いは**短期（0〜2年）では増加する**と推察される。

**② 実装工数はAPIの能力向上で削減されない領域が大きい**
Claude CodeやGPT-4oがコーディングを自動化しても、日本企業特有の課題（レガシーシステム統合、社内承認プロセス、規制対応、セキュリティ要件）への対応は引き続き人的専門知識が必要である。この領域は**短中期では代替困難**と判断する。

**③ Anthropicの日本市場リソースの限界**
Anthropic社は日本法人設立・直販体制の構築途上である（2025年時点）。日本市場の文化的複雑性・言語障壁・コンプライアンス要件対応には時間を要する。国内パートナー（グロービング等）へのチャネル依存は当面継続すると推察される。

**④ 「知る・使える」から「使いこなせる・変革できる」へのギャップ**
Anthropicが提供するのは「道具」であり、グロービングが提供するのは「その道具で企業の業務を変革する能力」である。このギャップは当面埋まらない。

#### 競合要因（30%）

**① Claude for Enterprise の直販化リスク（3〜5年）**
AnthropicがFortune 500・日経225クラスの大企業に直接コンサルチームを組んで乗り込む動きが顕在化した場合、グロービングの大型案件獲得機会は競合圧力を受ける。これは**3〜5年のリスクシナリオ**として重みをおく。

**② 料金のゼロ方向への収斂リスク**
Anthropic / OpenAI / Googleが競争激化の中でAPI料金を大幅引き下げた場合、グロービングのコンサルティング単価に対する顧客の心理的抵抗が高まる（「なぜ月数百万払うのか、自社でできないか」という経営者の発想転換）。

**③ Claude Code による開発コスト圧縮**
Claude Code（AIによるコード生成）が日本語・日本固有パターンに対応が高度化した場合、グロービングのエンジニア稼働工数が競争上の優位性を失う。**現時点では限定的だが要モニタリング**。

- 2Qサプライズ後に通期を据え置く場合、下期への「バッファ」を残している可能性が高く、これは**隠れた上方修正余地**として評価できる
