## 1. 企業概要・事業分析

### 1-1. 会社プロフィール

放電精密加工研究所（Hoden Seimitsu Kako Kenkyusho Co., Ltd.）は、**放電加工技術**を核とした精密部品・金型加工の専門メーカーである。1953年創業という長い歴史を持ち、東証スタンダード市場に上場。本社は神奈川県川崎市に置く。

- **従業員数:** 約600〜700名（連結推計）
- **主要顧客:** 自動車・電子・半導体・医療機器メーカー
- **主力技術:** 放電加工（EDM）、精密プレス、金型製造、表面処理

### 1-2. セグメント・プロダクト構成

```

### 1-3. 競合優位性（Competitive Moat）

**① 技術的参入障壁（★★★★★）**
放電加工は職人的ノウハウの蓄積が必要であり、70年超の加工ノウハウは容易に模倣困難。特に**マイクロEDM（微細放電加工）**領域での技術力は国内トップクラス水準。

**② 顧客粘着性（★★★★☆）**
精密部品は品質認定に長期間を要するため、スイッチングコストが高く、一度採用されると長期的な取引関係が継続する傾向。

**③ 多品種小ロット対応力（★★★★☆）**
半導体・医療分野で求められる多品種少量生産への柔軟対応が可能。大手では不採算となる案件を吸収できる。

**④ 地理的優位性（★★★☆☆）**
国内主要製造業クラスター（神奈川・東海）への近接性。ただし海外展開は限定的で、逆に言えば円安の恩恵を受けやすい。

## 4. DCFバリュエーション

### 4-1. 前提条件（全明記）

```
【DCF前提パラメータ】

■ 資本コスト関連
  リスクフリーレート (Rf)    : 1.5%  ← JGB10年物（2026年4月時点）
  エクイティリスクプレミアム  : 5.5%  （国内市場・Damodaran参照）
  ベータ (β)               : 0.85  （小型精密製造業・推計）
                                     ※市場感応度は低め（ディフェンシブ気味）
  株主資本コスト (Ke)        : 1.5% + 0.85 × 5.5% = 6.18% → 6.2%
  
  負債コスト（税引前）        : 1.8%  （現在の銀行借入金利水準）
  実効税率                   : 30%
  負債コスト（税引後）        : 1.26%
  
  D/E比率（推計）            : 約35/65
  WACC                      : 6.2% × 0.65 + 1.26% × 0.35 = 4.47% → 約4.5%

■ FCF成長率仮定
  予測期間                   : 10年（FY2026〜FY2035）
  フェーズ1（FY2026-2030）  : FCF成長率 +6〜8%/年（需要回復・利益率改善）
  フェーズ2（FY2031-2035）  : FCF成長率 +3〜4%/年（成熟期移行）
  永続成長率 (g)             : 1.5%  （日本のインフレ・名目成長率参照）
  
■ 基準FCF（FY2026E）
  基本シナリオ               : ¥400M
  強気シナリオ               : ¥550M
  弱気シナリオ               : ¥250M
  
■ 発行済株式数（推計）        : 約4,430〜4,500千株
```

### 4-2. WACCの詳細計算

```
【WACC計算シート】

Ke = Rf + β × ERP
   = 1.5% + 0.85 × 5.5%
   = 1.5% + 4.675%
   = 6.175% ≈ 6.2%

Kd(after-tax) = 1.8% × (1 - 0.30) = 1.26%

資本構成: 
  E/(D+E) = 65% （自己資本比率55%ベースに調整）
  D/(D+E) = 35%

WACC = 6.2% × 65% + 1.26% × 35%
     = 4.03% + 0.44%
     = 4.47% ≈ 4.5%
```

### 4-3. DCFモデル（基本シナリオ）

```
【フェーズ1: 高成長期（FY2026〜FY2030）】
  FY2026E FCF: ¥400M
  FY2027E FCF: ¥432M （+8%）

【フェーズ2: 安定成長期（FY2031〜FY2035）】

  FY2032E FCF: ¥606M （+5%）
  FY2033E FCF: ¥636M （+5%）
  FY2034E FCF: ¥668M （+5%）
  FY2035E FCF: ¥701M （+5%）
  小計PV（@WACC4.5%）: 約¥1,850M

【ターミナルバリュー（TV）】
  TV = FCF2035 × (1+g) / (WACC - g)
     = ¥701M × 1.015 / (0.045 - 0.015)
     = ¥711M / 0.03
     = ¥23,700M
  TV現在価値（10年割引）: ¥23,700M / (1.045)^10
                        ≈ ¥23,700M / 1.553
                        ≈ ¥15,260M

【企業価値（EV）合計】
  EV = ¥1,900M + ¥1,850M + ¥15,260M = ¥19,010M

【株主価値】
  株主価値 = EV - 有利子負債 + 現金

【1株理論株価（基本）】
  ¥17,510M ÷ 4,450千株 ≈ ¥3,936
  
  ⚠️ DCF高値警戒: TVの比率が高い。感応度分析で調整。
