### 2-1. 水道法改正の概要と応用地質への直撃度

**【改正の骨子】**

2018年改正水道法（施行: 2019年〜段階的）および**2024〜2025年の広域化・官民連携推進強化**により、全国約1,400の水道事業者（主に市町村）に対して以下が実質義務化された：

```
① 水道施設の計画的な老朽化更新（管路更新率の数値目標設定）
② 広域化・統合に向けた施設実態調査（広域連携調査）
③ コンセッション方式導入前の施設診断（PFI/PPP前提調査）
④ 危機管理計画の見直し（震災・渇水・汚染シナリオ別）
⑤ 水源地・地下水源の保全調査義務強化
```

**これらは全て応用地質のコア技術領域と直結する。**

---

### 2-2. 需要規模の定量試算

**全国水道管路の現状（公開統計ベース）:**
- 総延長: 約72万km
- 法定耐用年数超過管路: 約18万km（全体の約25%、2024年時点）
- 年間更新量: 約5,000〜6,000km（更新率0.7〜0.8%）
- **必要更新速度**: 耐用年数を全管路で守るには年間約1.5〜2.0%（約1万km超）が必要
- **現状ペースとの乖離**: 約2.5〜3倍の更新加速が社会的要請

**調査・診断コスト試算:**
```
管路更新1kmあたり調査費用 ≈ 200〜500万円（地盤・水質・管路診断込み）
年間追加更新目標 ≈ 5,000〜8,000km（現状からの加速分）
追加調査市場規模（概算）= 5,000km × 300万円 = 1,500億円/年（業界合計）
```

**応用地質のシェア想定:**
- 地盤・地下水分野でのシェア: 業界トップ水準（推計15〜20%）
- 水道調査市場での取込可能額: **150〜300億円/年（段階的）**
- 直近の当該関連売上推計: **60〜80億円/年**
- **3〜5年での倍増シナリオは現実的**

### 2-3. 4657（環境管理センター）との同軸比較

| 比較軸 | 応用地質（9755） | 環境管理センター（4657） |
|--------|----------------|----------------------|
| **コア技術** | 地質・地盤・地下水構造解析 | 水質分析・環境測定・土壌汚染調査 |
| **水道関連の切り口** | 管路下の地盤診断・漏水起因地盤沈下・地下水源調査 | 水質基準適合検査・排水処理効果測定 |
| **官公庁依存度** | 高（70%超） | 中〜高（60〜65%推計） |
| **技術の希少性** | ◎（地質専門技術者の絶対数制約） | ○（分析機器・資格者は競合多数） |
| **広域化受益の方向性** | 上流（施設実態把握・設計前調査）で受益 | 中流〜下流（検査・モニタリング）で受益 |
| **価格決定力** | 強い（代替困難な専門性） | やや弱い（競合多・価格競争あり） |
| **投資判断上の含意** | 上流の地盤・構造調査=プロジェクト必須工程=受注安定 | 検査業務=単価競争リスクあり |

**深井の見解:**
> 環境管理センター（4657）と応用地質（9755）は水道法改正の**異なるフェーズ**で受益する。応用地質は「何が埋まっているか・地盤はどうか」という**プロジェクト着手前の必須調査**を担うため、工事発注の上流に位置する。これは仕事が来なければ着手できない「ゲートキーパー」ポジションであり、**需要増時の価格決定力・受注量の両面で優位**。環境管理センターは検査・測定の下流工程であり、競合が多い。水道法改正受益の**純度・質**では応用地質が上回る。

### 2-4. 地下水脈解析需要の新展開

近年注目されているのが**地下水ガバナンス**の変化。

- **気候変動による地下水位変動**: 渇水・洪水の両極化により、自治体の地下水源モニタリング投資が増加
- **熊本型地下水保全モデルの全国展開**: 熊本県での成功事例（半導体工場誘致×地下水保全の両立）を受け、全国自治体が地下水マスタープラン策定に乗り出す動き
- **データセンター×地下水冷却**: 北海道・東北でのデータセンター立地急増に伴う地下水利用可能性調査需要

応用地質は地下水専門技術者を多数保有しており、**この新規需要領域は既存技術の応用展開**として低コストで取り込める。

### 2-5. 管路診断の技術革新と応用地質のポジション

```
【次世代管路診断技術トレンド】

従来方式: 掘削→目視確認→判断（コスト高・時間長）
         ↓
新技術①: 音響式非破壊検査（AE法）× 地盤データ統合
新技術②: ドローン×赤外線熱画像×GIS
新技術③: AIによる腐食予測モデル（地質データ×管材データ×水質データ）
```

応用地質はこれら新技術の**地盤データ連携レイヤー**を担う独占的ポジションにある。管路診断単体は他社も参入可能だが、「地盤変動が管路にどう影響するか」という**統合解析**は応用地質の特許技術・ノウハウ領域。

```

  理論株価 = (FCF5年分PV + TV現在価値 + ネットキャッシュ) ÷ 発行済株数

TV = FCF_t+1 / (WACC - g)
   WACCが上がるほどTVが圧縮され理論株価が低下

例: Rf=3.386%時（+1.00%シナリオ）
  WACC = 6.558%
  TV = 35.1 / (0.06558 - 0.015) = 35.1 / 0.05058 = ¥694億円
  EV = 115 + 694 × 0.7271 + 30 = ¥650億円
  理論株価 ≒ ¥650億 ÷ 780万株 ≒ ¥2,680
```
