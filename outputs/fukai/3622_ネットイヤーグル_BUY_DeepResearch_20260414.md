## 2. 社長指示重点分析：「アクティビスト介入×プライム→スタンダード降格×4/28決算迫る。優先度1」

### 2-1. アクティビスト介入の構造分析

**時価総額¥80〜90億円の小型デジタルエージェンシーへのアクティビスト介入**は、以下の「解散価値/隠れ資産」の観点から動機が説明できる。

```
アクティビスト介入の経済学（ネットイヤーの場合）
─────────────────────────────────────────
純資産（BPS¥288 × 推計株数1,500〜1,600万株）= 約¥43〜46億円
現在時価総額: 約¥82〜87億円
PBR: 1.88倍

アクティビスト視点での「解散価値ギャップ」:
→ 現預金・有形資産が純資産の大部分を占めるなら、
  「稼ぐ力が弱い＝事業を清算してキャッシュを株主に返せ」論が成立
→ ただしPBR1.88倍は解散価値に対して割高であり、
  アクティビストが要求するのは「株主還元強化 or 経営効率化 or 売却」
─────────────────────────────────────────
```

**考えられるアクティビスト要求内容（典型パターン）**:
1. **大規模自社株買い・特別配当**（最もシンプル）
2. **外部資本との統合・M&A賛成票**（戦略的売却）
3. **経営陣刷新・社外取締役増員**（ガバナンス要求）
4. **MBO支援**（非公開化による効率経営）

### 2-2. プライム→スタンダード降格の需給インパクト

**降格の機械的インパクト**:
- TOPIXから除外される可能性（プライム残存条件を満たさない場合）→ **TOPIXインデックスファンドの強制売り**
- プライム市場上場基準（時価総額100億円・流通株式数・利益要件）を維持できなかった事実が、**機関投資家のユニバース除外**につながる

## 4. DCFバリュエーション

### 4-1. 前提条件

```
【前提パラメータ】
────────────────────────────────────────
リスクフリーレート（Rf）: 2.467%（JGB10Y 2026-04-14）
株式リスクプレミアム（ERP）: 5.5%（Damodaran日本ERP概算）
ベータ（β）: 0.85（小型デジタルサービス株。低流動性でβは市場β0.75〜0.95を採用）
────────────────────────────────────────
株主資本コスト（Re）:
  Re = Rf + β × ERP
  Re = 2.467% + 0.85 × 5.5%
  Re = 2.467% + 4.675%
  Re = 7.14%
────────────────────────────────────────
資本構成（無借金 or 低借金企業と仮定）:
  ネットイヤーはIT軽資産型。有利子負債は限定的と推定
  E/(D+E) = 95% / D/(D+E) = 5%（推計）
  Rd（負債コスト）= 1.5%（短期借入金金利推計）
  実効税率（Tc）= 30%
────────────────────────────────────────
WACC:
  WACC = 0.95 × 7.14% + 0.05 × 1.5% × (1-0.30)
  WACC = 6.783% + 0.0525%
  WACC = 6.84%（≒6.8%として使用）
────────────────────────────────────────
ターミナル成長率（g）:
  基本シナリオ: 1.5%（日本名目GDP成長率≒インフレ1〜2%）
  強気シナリオ: 2.5%
  弱気シナリオ: 0.5%
────────────────────────────────────────
DCF対象: 株主に帰属するFCF（FCFE）
  FCFEベース: EPS¥22 × 株数 → 純利益ベースで近似
  FCFマージン適用純利益（FY2025ベース）: ¥3.2〜3.5億円
  1株FCFE（FY2025）: ¥22（EPS一致と仮定）
  5年間成長率（予測期間）: 基本シナリオ年率5%
```

### 4-2. DCF計算（基本シナリオ）

```
【基本シナリオ DCF（1株ベース）】
Re = 7.14%（≒株主資本コスト：純資本構造のためWACCに近接）
g（ターミナル）= 1.5%

Year 1: ¥22.0 × 1.05 = ¥23.1   PV = ¥23.1 / 1.0714 = ¥21.6
Year 2: ¥23.1 × 1.05 = ¥24.3   PV = ¥24.3 / 1.0714² = ¥21.1
Year 3: ¥24.3 × 1.05 = ¥25.5   PV = ¥25.5 / 1.0714³ = ¥20.7
Year 4: ¥25.5 × 1.05 = ¥26.8   PV = ¥26.8 / 1.0714⁴ = ¥20.3
Year 5: ¥26.8 × 1.05 = ¥28.1   PV = ¥28.1 / 1.0714⁵ = ¥19.9

5年PV合計: ¥103.6

ターミナルバリュー（Year5末）:
TV = ¥28.1 × (1 + 1.5%) / (7.14% - 1.5%)
TV = ¥28.5 / 5.64%
TV = ¥505.7

ターミナルバリューPV:
PV(TV) = ¥505.7 / 1.0714⁵ = ¥358.4

【基本シナリオ 理論株価】:
FV = ¥103.6 + ¥358.4 = ¥462 ← 基本シナリオ
```
