## 4. DCFバリュエーション

### 4-1. バリュエーション前提条件（全項目明記）

```
■ 絶対厳守パラメータ
  Rf（リスクフリーレート） = 2.380%（JGB10Y、社長確認値）
  取得日時: 2026-04-05 18:30 JST
  参照元: https://jp.investing.com/rates-bonds/japan-10-year-bond-yield-streaming-chart

■ 追加パラメータ（深井設定）
  ERP（株式リスクプレミアム）= 5.50%（日本株市場標準）
  β（ベータ）                = 1.35（航空業界平均。コロナ後の資本構造・高レバレッジ反映）
  Re（株主資本コスト）        = Rf + β × ERP
                             = 2.380% + 1.35 × 5.50%
                             = 2.380% + 7.425%
                             = 9.805%

  負債コスト前（Rd）          = 1.85%（実効借入金利推計）
  実効税率（Tc）              = 30.0%
  負債コスト後（Rd×(1-Tc)）  = 1.85% × 0.70 = 1.295%

  資本構造（時価ベース）:
    E（株式時価総額）         = 9,800億円
    D（有利子負債）           = 12,800億円
    D+E                       = 22,600億円
    E/(D+E)                   = 43.4%
    D/(D+E)                   = 56.6%

  WACC = E/(D+E) × Re + D/(D+E) × Rd × (1-Tc)
       = 0.434 × 9.805% + 0.566 × 1.295%
       = 4.255% + 0.733%
       = 4.988% ≒ 5.00%

  ターミナル成長率（g）        = 1.00%（航空業界の長期実質成長＋インフレ）
  予測期間                    = 5年（FY2026〜FY2030）
```

### 5-2. ⑤ DCF理論値が高くても空売りが有効な理由（核心論点）

> **「理論値4,155円なのになぜ空売りか？」** — この矛盾を解消する。

```
【Deep理由1: タイムラグと収益崩壊の非線形性】
  DCF理論値はあくまで「正常化後」の企業価値。
  ホルムズ封鎖が6〜12ヶ月継続した場合:
  ・FCF FY2026がマイナス転落（深井弱気試算: -200〜-500億円）
  ・ターミナル価値計算の分母（WACC-g）は変わらずとも、
    FCFベースが崩壊すればTVの基点も消える。
  → 理論値は最大で基本シナリオの50〜60%に収縮する。

【Deep理由2: 株式市場のカタリスト反応速度】
  ・原油スパイク・地政学ニュースは即日リプライシング
  ・一方、ANAの燃料費ヘッジ解消・損失計上は四半期遅行
  ・市場は「見通し悪化」を先行して株価に折り込む
  → DCF理論値が高い局面でも、カタリスト発生後6ヶ月は
    株価が理論値の60〜70%水準まで売られる過去事例あり。

【Deep理由3: レバレッジ増幅効果（高債務企業特有）】
  Net D/E = 1.15倍（高レバレッジ）
  FCFが300億円→▲200億円に転落する場合:
  ・債務返済能力への懸念 → 格付け見直しリスク
  ・社債スプレッド拡大 → Rdの上昇 → WACC上昇 → 理論値下落
  ・ネガティブスパイラル: 理論値自体も連動して崩れる

【Deep理由4: 信用コスト（貸株金利）の非対称性】
  空売りの貸株コストが年2〜4%としても、
  下値△40〜△50%のリターンと比較すると十分ペイする。
  （§8で詳述）

【結論】
  理論値が現在株価を大幅に上回っていても、
  「地政学ショック → FCF崩壊 → 理論値収縮 → 株価先行下落」
  の連鎖が発動するリスクシナリオ下では、
  空売りは理論的に整合する戦術。
  浅井スコア-95はその蓋然性が高まっている状態を示す。
```
