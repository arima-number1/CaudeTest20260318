```

## 5. DCFバリュエーション（詳細）

### 5.1 WACC計算

#### ステップ1: 株主資本コスト（Re）

```
【絶対厳守】Rf = 2.380%（JGB10Y、社長確認値）

・β = 0.85（過去3年 週次リターン、市場：TOPIX）
  ※ E&P セクター β は通常 0.8〜1.1
  ※ INPEX は国策色・LNG長期契約で市場平均より低β
・ERP = 5.50%（日本市場ERP、Damodaran 2025推計）

Re = Rf + β × ERP
   = 2.380% + 0.85 × 5.50%
   = 2.380% + 4.675%
   = 7.055%
```

#### ステップ2: 負債コスト（Rd）

```
・加重平均借入金利 = 1.20%（低コスト調達、国策企業特性）
・実効税率 Tc = 40%
・税引後負債コスト = 1.20% × (1 − 0.40) = 0.72%
```

#### ステップ3: 資本構成

```
・株式時価総額（E）: ¥4,675 × 9.7億株 = 約¥4.5兆円
  → ¥45,348億円
・有利子負債（D）  : 約¥14,000億円（ネット）
・D + E            : 約¥59,348億円
・E/(D+E)          : 76.4%
・D/(D+E)          : 23.6%
```

#### ステップ4: WACC

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1−Tc)]
     = [0.764 × 7.055%] + [0.236 × 0.72%]
     = 5.390% + 0.170%
     = 5.560%
     → 採用WACC: 5.56%
```

*注: FCF成長の鈍化はWTI価格の平均回帰（$113→$100付近）を保守的に反映*

#### フェーズ2: ターミナルバリュー（TV）

```
永続成長率（g）= 1.50%
  ※ 長期石油・ガス需要の緩やかな成長 + インフレ調整
  ※ エネルギー転換リスクを考慮し保守的設定

ターミナルFCF = FY2028E FCF × (1 + g)
              = 3,800 × 1.015
              = 3,857億円

TV = 3,857 ÷ (WACC − g)
   = 3,857 ÷ (5.56% − 1.50%)
   = 3,857 ÷ 4.06%
   = 95,000億円（概算）

PV(TV) = 95,000 × 0.7635（5年割引係数）
        = 72,533億円
```

#### エンタープライズバリュー（EV）と理論株価

```
EV = PV(FCF明示期) + PV(TV)

理論株価 = 78,641億円 ÷ 9.7億株
