## 🔑 最重要論点 — Q4に¥16.3億が必要だが達成困難

```
通期経常利益予想: ¥26億
Q3累計(9M)実績: ¥9.72億
Q4(1月-3月)の必要額: ¥26.0億 - ¥9.72億 = ¥16.28億

vs. Q3(10-12月)単四半期 run rate:
Q3 YTD ¥9.72億 ÷ 3四半期 = ¥3.24億/四半期

Q4必要額 = run rate の約5倍
```

**なぜ達成困難か:**
1. ホルムズ封鎖(2月28日〜)による電力コスト上昇継続
2. 電力補助金終了(4月〜)で1〜3月も電力費負担が増大
3. 鉄スクラップH2型¥50,000/t（20ヵ月ぶり高値）が継続
4. 中国安価鋼材流入で鋼材売価に上値圧力
5. Q4単独での特殊利益（土地売却等）がなければ達成困難

**結論:** 通期経常利益は¥13〜18億着地（予想¥26億から大幅未達）の可能性が高い。これが決算前に買わない最大の理由。
