```
- マーケティング部門は仕入れコストも上がる（スプレッドが固定でないリスク）

```

### PBR1.44倍は理論的下値支持線か？

```
理論的下値支持線の検証：

【ROEとPBRの理論的整合性】
PBR = ROE / Ke × (1 + 長期成長率考慮)
ROE = 10.8% / Ke（株主資本コスト） = 5.6%（後述DCFより）
理論PBR = 10.8% / 5.6% × 調整 ≒ 1.93倍（理論値）

→ 現在1.44倍は理論値の約75%。
→ 理論的には「割安」だが、LNGリスクプレミアムで正当化される部分もある。

【PBR1.0倍（BPS=¥4,477）が絶対的下値支持線】
→ 現在株価¥6,467 / BPS¥4,477 = 1.44倍
→ PBR1.0倍 = ¥4,477（約▲31%下落しない限り「純資産割れ」にはならない）
→ 公益インフラ株でPBR1.0倍割れは「非常事態」。日本の公益規制が維持される限り、
   BPS割れは過度な売られすぎを意味する。

【直近の実勢支持線】
→ PBR1.2倍 = ¥5,372（配当利回り2.2%水準、機関投資家の買い意欲出る水準）
→ PBR1.3倍 = ¥5,820（現値から▲10%、過去の底値近辺）
→ 深井の損切りライン候補：¥5,500〜5,800レンジ

結論：PBR1.44倍は理論的下値支持線ではなく「現在の適正値近辺」。
下値支持線は¥5,372（PBR1.2倍）〜¥5,820（PBR1.3倍）のレンジ。
```
