## 2. 投資テーゼ：なぜ今ルネサスか

### テーゼ①：52週最安値圏からの回復局面 + 米系証券が目標¥3,600維持

2026/03/31に52週安値¥2,136をつけた直後（3日後）の現在株価¥2,271。底打ち確認後の初期回復局面にある可能性が高い。米系大手証券が目標株価¥3,600で強気評価を継続している事実は、機関投資家の下値支持として機能する。

```
52週高値:   ¥3,138（2026/02/09）
52週安値:   ¥2,136（2026/03/31）← 直近底値
現在株価:   ¥2,271（2026/04/03）
底値からの回復率: +6.3%（まだ初期回復段階）
米系大手目標: ¥3,600（現在株価比 +58.5%）
```

### テーゼ②：MS日本10兆円DC投資 → 産業・インフラ向け（51%セグメント）に直撃追い風

ルネサスの売上の51%を占める「産業・インフラ・IoT」セグメントは、データセンター建設に不可欠な部品を多数供給している。

**具体的な受益製品：**
- **電源管理IC（PMIC）**：DCのラックサーバー電源回路に必須。AIサーバー1台あたりの搭載数は従来の2〜3倍
- **産業用マイコン（RZシリーズ）**：DC内の冷却システム制御・電力監視に使用
- **リアルタイムクロック（RTC）**：全てのサーバーマザーボードに搭載

MS10兆円という投資規模が意味するのは「数十万台規模のAIサーバー新設」。これはルネサスのPMIC・産業用MCUへの直接的な追加発注を意味する。

### テーゼ③：車載マイコン世界首位 × EV/ADAS需要の構造的成長

ルネサスは車載マイコン（MCU）で世界シェア約30%を持つ揺るぎないリーダー。EV化・ADAS（先進運転支援システム）の普及により、1台の自動車に搭載されるマイコン数は急増している。

**車載向けの成長ドライバー：**
- **R-Car SoC**（ADAS・インフォテインメント統合プロセッサ）：トヨタ・日産・ホンダ等への採用拡大
- **電動化対応MCU**：EVの電動パワートレイン制御に特化した製品ラインナップ強化
- **機能安全規格 ISO 26262 対応**：競合の参入障壁として機能

原油$111/bblの高止まりでEV化が加速すれば、ルネサスの車載セグメント（48%）の成長が再加速する。

### テーゼ④：特約店契約終了 → 直販シフト → マージン改善

2026年4月にリョー菱洋HDとの特約店契約を終了。これは表面上はネガティブに見えるが、実態は「中間マージンを削除した直販強化」という戦略的なマージン改善施策だ。

```
従来フロー: ルネサス → 特約店 → エンドユーザー（中間マージン発生）
新フロー:   ルネサス → エンドユーザー（直販）

効果: GPM（粗利率）が数パーセントポイント改善する可能性
     ルネサスの現在のGPM約50%が51〜53%へ改善余地
```

## 3. 懸念点と反論

### 懸念①：PERが非開示 = 業績への不確実性

PERが「非表示」になっている理由として、業績予想の精度が低いまたは赤字転落の可能性が考えられる。

**反論：** ルネサスは半導体の在庫調整サイクルを経て、2024〜2025年は回復基調にあった。PER非表示は「予想値の設定を保守的に控えている」ことが多く、実態は黒字継続と推定。BPS¥1,347に対してPBR1.69倍は「市場が将来の収益力を評価している」ことの現れ。

### 懸念②：「パワー半導体、規模拡大の段階にない」発言

柴田社長の発言は「今すぐパワー半導体に大型投資しない」という設備投資抑制の意思表示。

**反論：** これはネガティブではなく、むしろ**資本規律（Capital Discipline）の証拠**。過剰投資を避け、FCF（フリーキャッシュフロー）をM&Aや自社株買いに充当する姿勢は株主にとってポジティブ。

### 懸念③：52週高値¥3,138から▲27.6%の急落

2026年2〜3月の急落の背景には、自動車向け半導体の在庫調整懸念と、円高リスクへの警戒があったと推定。

**反論：** 3/31に¥2,136で底打ちし、4/3時点で¥2,271へ回復中。「底値確認後の初期回復局面」に入った可能性があり、ここからの上昇余地が大きい。

### EPS × PER による理論株価

| シナリオ | EPS想定 | 適用PER | 理論株価 |
|---------|--------|--------|---------|

| Bear | ¥90 | 20倍 | **¥1,800** |

### 加重 Fair Value

```
FV = ¥3,600 × 30% + ¥2,800 × 50% + ¥1,800 × 20%
   = ¥1,080 + ¥1,400 + ¥360
   = ¥2,840 ≒ ¥2,800（端数調整）

現在株価 ¥2,271.5 → アップサイド +23.3%（base）〜+58.5%（bull）
```

## 8. 浅井（Fact）× 不労（Flow）× 豆田（Risk）コメント

**浅井（Fact）：**
朝刊PDF確認済みファクト：MS日本10兆円投資でDC建設ラッシュ確定。ルネサスの産業・インフラ向け（51%）への直接波及は「連想」ではなく「実需」。Q1決算（5月）でこれが数字に現れるか監視。

**不労（Flow）：**
52週安値¥2,136（3/31）から3日で+6%回復。底値圏での出来高が増加していれば「大口の買い集め」の可能性あり。米系大手証券の¥3,600目標は機関投資家のポジション構築の「錨」として機能する。4月第2〜3週の出来高動向を注視。

**豆田（Risk）：**
最大リスクは「自動車向け在庫調整の長期化」。ルネサスの車載向けは48%であり、ここが想定より回復しない場合は業績を大きく押し下げる。また円高転換（日銀利上げ＋停戦成立）リスク：海外売上80%の同社は円安恩恵銘柄であり、急激な円高は要注意。PBR1.69倍は暴落耐性がそこまで高くないため、損切ライン厳守が必須。
