### 総還元性向25〜30%方針（初明示）

今回の決算で初めて「総還元性向25〜30%」を目指す方針を明文化。

| FY3/2026 実績配当 | ¥50（配当性向44.4%・総還元性向含む） |

| 自社株買い（後発事象） | **27億円（上限）・1,500千株・5/1〜9/30** |

- BPS¥2,156.96 × 75.8%自己資本比率 = 財務余力は十分。自社株買い拡大余地は大きい

### DCF改訂（来期赤字を反映）

**前提: Rf=2.380%（固定）、ERP=5.5%、β=1.15、WACC=7.64%、g=1.0%**

| 年度 | 売上高 | 営業利益率 | NOPAT | D&A | Capex | ΔNWC | FCF |
|------|--------|---------|-------|-----|-------|------|-----|
| FY3/2027E | 3,150 | ▲1.3%（赤字） | ▲29 | 82 | ▲160 | +20 | **▲87** |
| FY3/2028E | 3,000 | +1.0% | 21 | 85 | ▲140 | 0 | **▲34** |
| FY3/2029E | 3,200 | +3.5% | 78 | 88 | ▲140 | ▲10 | **16** |
| FY3/2030E | 3,400 | +5.0% | 119 | 90 | ▲130 | ▲10 | **69** |
| FY3/2031E | 3,500 | +5.5% | 134 | 92 | ▲130 | ▲10 | **86** |

**ターミナルバリュー（TV）:**
```
TV = 86 × 1.01 / (7.64% - 1.0%) = 86.86 / 6.64% = 1,308億円
PV(TV) = 1,308 / (1.0764)^5 = 904億円
```

**企業価値（EV）:**
```
PV(FCF) = -87/(1.0764) + -34/(1.0764)² + 16/(1.0764)³ 
         + 69/(1.0764)⁴ + 86/(1.0764)⁵
        = -80.8 - 29.3 + 12.8 + 51.4 + 59.4
        = 13.5億円

EV = 13.5 + 904 = 917.5億円
```

**理論株価:**
```
ネットキャッシュ（現金734.70+有価証券105.00+投資有価証券340.12 - 負債合計704.65）
= 734.70 + 105.00 + 340.12 - 704.65 = 475.17億円

株主価値 = 917.5 + 475.2 = 1,392.7億円
理論株価 = 1,392.7億 ÷ 1.0268億株（流通株式数） ≒ ¥1,357
```

> **深井コメント**: DCFベースの理論株価は**¥1,357**。現値¥1,827を下回る水準。来期赤字を含む5年間のFCFが大幅マイナスとなるため、企業価値の大部分がターミナルバリュー（2030年以降の正常化）に依存する構造。これは不確実性が高い。

## 6. 推奨アクション（決算後 判断）

### 旧DRエントリー分の判断

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
旧DR 第1トランシェ（¥1,800-¥1,850で入れていた場合）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

損切りトリガー確認:
□ EPS < ¥130 → ¥112.56 ✅ 成立
□ 増配なし → 逆に減配 ✅ 条件近似
□ 来期ガイダンス大幅減益(EPS<¥100) → EPS 0円 ✅ 成立

→ 3条件すべて抵触。原則は損切り。

ただし考慮すべき反論:
・5/1〜自社株買い27億円（BPS継続上昇の支え）
・PBR0.85倍の下値硬直性
・BPS¥2,157に対して現値は▲15%乖離
・総還元性向25〜30%の初明示（中長期ポジ）

▶ 深井の結論: 
  損切り¥1,600ラインは維持。ただし自社株買い開始（5/1〜）の
  需給確認後まで実行を数日待つ選択肢も合理的。
  来期ガイダンスが予想以上に市場に織り込まれていない場合、
  本日引け後〜翌日に株価が下落する可能性があり、
  その後5/1の自社株買い開始を確認してからの判断でも遅くない。
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
