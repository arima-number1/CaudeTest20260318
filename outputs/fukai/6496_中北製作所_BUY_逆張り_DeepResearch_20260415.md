## 1. 企業概要・ビジネスモデル

**株式会社中北製作所**（6496.T）は、創業1930年・設立1937年の老舗産業機器メーカー。
本社：大阪市。東証スタンダード市場上場。

### 主要製品
| 製品カテゴリ | 用途 | 市場ポジション |
|---|---|---|
| **自動調節弁** | 船舶・プラントの流体制御 | 国内最大手 |
| **バタフライ弁** | 船舶・発電所の流路切替 | 舶用でトップシェア |
| **遠隔操作装置** | 船舶バルブの遠隔制御 | — |

### 主要顧客セグメント
- **造船業界**: 国内大手造船所（今治造船・大島造船所・JMU等）→ 1隻あたりのバルブ使用数千個
- **発電所・プラント**: 電力会社・ガス会社・化学プラント
- **海外**: 韓国（ACE VALVE子会社経由で現地対応）

### 2024年 ACE VALVE子会社化
韓国のバルブメーカーACE VALVE（에이스밸브）を子会社化。韓国・中国造船所への供給体制強化。
造船の世界市場で韓国（KSOE・POSCO Int.等）が圧倒的シェアを持つ中、現地供給体制は長期的な必須条件。

## 2. 下落の解剖【最重要】

### 1M -30.9%の正体 = 「2つの異なる要因の重なり」

#### 要因①：4Qクリフ懸念（7割の寄与度）

**事象**：2026年4月7日発表の3Q決算で「進捗率93.7%にも関わらず通期予想を据え置き」

```
FY2026（5月期）通期予想: 経常利益 ¥18億
3Q累計（6月〜2月）実績: 経常利益 ¥16.8億（進捗率93.7%）
4Q（3月〜5月）implied: ¥18億 - ¥16.8億 = ¥1.2億
 → 前年4Q比 -80.9%（投資家が「何か悪い事が起きる」と解釈）
```

**深井の解析（クリフは本物か？）**

| 可能性 | 評価 | 根拠 |
|---|---|---|
| 前倒し計上の反動（季節性） | **高い（確率45%）** | 舶用バルブは船舶納期前(秋〜冬)に集中納品→3Qに偏重、4Qは構造的に薄い |
| ACE VALVEのれん償却・統合費用 | **中程度（確率25%）** | 2024年子会社化→FY2026から本格のれん償却が始まる可能性 |
| 保守的なガイダンス | **中程度（確率20%）** | 会社は毎期保守的ガイダンスを出す傾向があれば、通期は上振れ確定 |
| 4Q本当に急減速 | **低い（確率10%）** | バタフライ弁FY2026 3Q +54.7%の勢いを考えると急減速は不自然 |

**結論**: 「4Qクリフ」の大半は季節性・ACE VALVE統合費用の一時的なもの。業績が「消えた」わけではなく「後ずれ or 前倒しの反動」。

#### 要因②：ホルムズ停戦・有事プレミアム剥落（3割の寄与度）

**事象**: 2026年4月上旬〜中旬にかけての米・イラン停戦交渉進展

**舶用バルブへの接続**:
- ホルムズ封鎖中: 迂回航路増加→タンカー・LNG船の航海時間増→船腹需要急増→造船発注加速→**バルブ受注急増**
- 停戦・開放: 迂回航路需要消滅→新規造船発注ペース鈍化→**バルブ受注の限界的需要が低下**

**本日（4/15）の追加材料**: AP通信「US, IRAN AGREE 'IN PRINCIPLE' ON EXTENDING TRUCE」
→ 停戦延長は短期的にバルブ株に追い風にならないが、**「造船の基礎需要（LNG船・コンテナ船の老朽化更新）は停戦と無関係に存在する**

## 4. 受注・業績サイクル分析

### 舶用バルブの季節性・ラグ構造

```
造船受注（N年）
  ↓ 1〜2年ラグ
船体建造開始（N+1年）
  ↓ 6〜9ヶ月
バルブ発注・受注（N+1年秋〜N+2年春）
  ↓ 3〜6ヶ月
バルブ製造・納品（N+2年冬〜N+2年夏）→ 売上計上
```

**3Q（12月〜2月）集中の理由**:
- 造船所の船台スケジュール上、12〜2月は進水・艤装前の設備完成時期
- バルブ検収タイミングが年度末（3月）納期の船舶に合わせて集中
- → 3Qに利益が集中し、4Qは構造的に軽い（季節性）

**4Q cliffの本質**: 構造的季節性 ≠ 業績悪化

| シナリオ | 舶用バルブへの影響 |

## 6. DCF・理論株価

### Method 1：ROE/Ke法

```
ROE = EPS/BPS = ¥508/¥7,944 = 6.40%
Ke = 5.5%（Rf 2.38% + β0.7 × MRP 4.5%）
Justified PBR = ROE/Ke = 6.40%/5.5% = 1.164x
FV = BPS ¥7,944 × 1.164 = ¥9,247
```

> ⚠️ ROEが6.4%と低いのは**自己資本比率71%というキャッシュ過多**が原因
> 実質事業ROEは（純利益¥17.3億/事業資産ベース）でもっと高い

### Method 2：正常化PER法

```
FY2025 EPS: ¥452（確定）
FY2026 EPS予想（ベース）: ¥508（+12.4%、3Q進捗から上振れ織り込み）
FY2026 EPS予想（強気）: ¥560（通期上方修正シナリオ）
FY2026 EPS予想（弱気）: ¥400（ACE VALVE統合費用が想定超過）
目標PER: 12x（産業機器・バリュー株）〜15x（造船ブーム成長継続）
  Base: 12x × ¥508 = ¥6,096
  Bull: 15x × ¥560 = ¥8,400
  Bear: 10x × ¥400 = ¥4,000
```

### Method 3：ネットキャッシュ・バリュエーション

```
純資産¥304億 × 自己資本比率71% ÷ 自己資本比率71% = ¥304億
有利子負債推定（低い）: 純資産¥304億 / 71% × 29% ≈ ¥124億（負債）
実質キャッシュリッチ企業 → 最低限の清算価値は BPS ¥7,944 ≈ ¥6,000〜¥8,000
現値¥5,640 ≒ BPS×0.71 → ほぼ「解散価値」水準
```

## 7. 主要株主・TOBリスク

- **自己資本比率71%** かつ **時価総額¥216億** = 財務内容の良さに比べて極端に小さい時価総額
- スタンダード市場の流動性が低い小型株（ADV¥1.9億/日）
- TOB標的として魅力的なサイズだが、造船業界との深い取引関係が障壁

> **TOBリスク/シナリオ**: 韓国・中国バルブメーカーの日本参入を防ぐ観点から、国内造船所が防衛的に「取引先保護」の出資増加を行う可能性。また、純資産¥304億に対して時価総額¥216億 = **PBR0.71x** はTOBのターゲットになりうる水準。
