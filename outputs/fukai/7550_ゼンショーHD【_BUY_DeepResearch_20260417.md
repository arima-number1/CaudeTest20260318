#### ■ PBR4.13倍の正当化に必要なROE

**Gordon Growth Model によるPBR正当化条件：**
```
PBR = ROE × Payout Ratio / (Re - g)

Re（株主資本コスト）= 2.409% + 0.85 × 5.5% = 7.08%
g（持続成長率）= 3.0%（楽観）

→ PBR = ROE × 0.3 / (0.0708 - 0.03) = ROE × 7.35

PBR = 4.13倍 を維持するには：
ROE = 4.13 / 7.35 = 56.2%

→ 現実のROE = 14.2%。4倍以上の乖離。
```

**→ PBR4.13倍は現在のROE14.2%では理論的に正当化不可能。**

### 2-5. 【最重要⑤】1M▲7.4%下落の原因特定

**市場は織り込み済みか？**

```
1ヶ月前株価 ≈ ¥9,277 → 現在¥8,586（▲7.4%）
RSI 35（売られすぎ水準）

織り込み度チェック：
- UAゼンセン+6.3%発表：2026年3月下旬
- 大阪ガス値上げ：4/9
- 下落開始タイミング → 3月下旬〜4月
→ コスト増ニュースと連動した下落 = 「織り込み開始」であって「織り込み完了」ではない
```

## 最終結論：深井の判定

### 「ゼンショーHDは外食コスト増の最大被災者か、転嫁力で跳ね返すか——白黒つけよ」
