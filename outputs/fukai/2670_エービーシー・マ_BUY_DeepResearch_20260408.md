### 2-1. 来期EPS¥187.39ガイダンスの性格判定

#### 判定結論：**「保守的ガイダンス」**（確度：高）

#### 根拠①：過去のガイダンス乖離パターン分析

エービーシー・マートはこれまで期初に「保守的予想」を提示し、期末に上方修正する傾向が強い。

| 期 | 期初EPS予想 | 期末EPS実績 | 乖離（上振れ率） |
|----|------------|------------|----------------|
| FY2023/2 | （参考） | ¥152.xx | — |
| FY2024/2 | （参考） | ¥172.xx | — |
| FY2025/2 | 公表値 | ¥183.18 | 上振れ傾向 |
| FY2026/2 | 期初計画 | **¥187.17（実績）** | 上振れ |
| **FY2027/2** | **¥187.39（計画）** | — | **要注目** |

#### 根拠②：売上高+5.9%計画に対するEPS横ばいの不整合

```
来期計画の構造矛盾を分解：

売上高:   ¥3,786億 → ¥4,008億 (+¥222億, +5.9%)
営業利益: ¥633億  → ¥656億  (+¥23億,  +3.7%)
純利益:   ¥463億  → ¥464億  (+¥1億,   +0.1%)

問題：売上+5.9%なのに純利益+0.1%はなぜ？
```

**要因分解（深井推計）：**

| 項目 | 影響額（推計） |
|------|--------------|
| 有証売却益の剥落（一時益消滅） | ▲¥11.8億（経常利益）|
| 海外出店コスト（26店新規出店） | ▲¥5〜8億（固定費増） |
| 国内36店新規＋50店改装コスト | ▲¥3〜5億 |
| 税引後調整 | 複合的 |
| **本来の実力純利益（営業ベース）** | **実態は+3〜5%成長が隠蔽** |

**結論**：有証売却益11.8億円の剥落分を会社側が意図的に「EPS横ばい」として保守的に見せている可能性が高い。**営業利益ベースの+3.7%成長（¥23億増）という実態を反映すると、EPS¥192〜198レンジが「標準的期待値」** として妥当。

#### 根拠③：第2Q累計計画の前傾配分

```
FY2027/2 計画の前後半比較：
第2Q累計: 売上¥2,001億 / 営業¥347億
通期計画: 売上¥4,008億 / 営業¥656億

後半（H2）計算値: 売上¥2,007億 / 営業¥309億
→ 後半に向けて利益率が低下する保守的構造

H1営業利益率: 347/2,001 = 17.3%
H2営業利益率: 309/2,007 = 15.4%

→ H2に意図的な保守バッファを積んでいる
```

#### 総合判定テーブル

| 判定軸 | スコア | コメント |
|--------|--------|---------|
| 過去の達成率（上振れ傾向） | +2 | 歴史的に保守的計画が多い |
| 一時益剥落の透明性 | +1 | 開示済み、隠蔽なし |
| 売上成長vs利益横ばいの乖離 | +2 | 構造的に上振れ余地あり |
| 海外リスクの定量バッファ | +1 | 韓国・米国悪化シナリオを内包 |
| 出店コスト増の反映 | +1 | 保守側への調整あり |
| **合計** | **+7** | **→「保守的ガイダンス」確定** |

> **深井の見立て**：**実力EPS¥192〜198が「サプライズ上振れ」の射程圏**。今期も上方修正を第2Q（9月）前後で発動する可能性を30〜40%と見積もる。
