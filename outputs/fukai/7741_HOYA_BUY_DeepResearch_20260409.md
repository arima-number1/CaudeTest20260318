### 【最重要A】EUVフォトマスク基板の構造的独占評価

#### A-1. HOYA+信越化学 世界シェア>95%の持続可能性

**EUVマスクブランクスの製造難易度と参入障壁：**

EUV（波長13.5nm）用マスクブランクスは、通常のArF（193nm）用と比較して製造難易度が**2桁以上**異なる。主要な参入障壁を定量化する。

```
【技術的参入障壁】
① 基板表面粗さ要件：ArF用 < 0.15nm RMS → EUV用 < 0.05nm RMS
   → 3倍以上の精度要求。既存設備の転用不可

② 多層反射膜（Mo/Si 40ペア）の成膜精度
   → 各層の厚さ制御が±0.1nm以内
   → 専用スパッタリング設備を1基導入するのに¥20〜30B（推計）

③ 欠陥ゼロ（Defect Free）要件
   → EUV用は1mm²あたり欠陥数0が求められる
   → 歩留まりを95%以上にするまでに通常5〜8年の製造ノウハウ蓄積が必要

④ ASML/TSMCとの認定プロセス
   → 新規サプライヤーの認定には通常3〜5年かかる
   → 量産移行後も「認定ウィンドウ」は極めて限定的

【設備投資バリア】
HOYAのEUV基板関連設備への累積投資（推計）：¥150〜200B
Entegrisがゼロから追いつくのに必要な投資（推計）：¥100〜150B × 5〜8年の時間
→ 単純な「金を出せば追いつける」構造ではなく、「時間の障壁」が最大の参入阻止要因

【顧客ロックインの構造】
TSMC/Samsung/Intel（ファウンドリ3強）とのサプライヤー認定
→ 一度認定されたサプライヤーを変更するコスト（クオリフィケーションコスト）が巨大
→ HOYAが主要顧客の「Qualified Vendor List（QVL）」に長年登録 = 事実上の独占
```
