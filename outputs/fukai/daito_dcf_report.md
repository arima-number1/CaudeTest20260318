## 2. DCF前提条件の詳細（全項目明記）

### 2-1. リスクフリーレート — JGB 10年国債利回り

| 項目 | 数値 | 根拠 |
|------|------|------|
| **JGB 10Y（基準日: 2026-04-02）** | **1.50%** | 日銀の追加利上げ局面。2024年以降の利上げサイクルにより1.5%程度で推移していると推定 |
| 採用方針 | スポットレートを使用 | 長期平均ではなく分析時点の実勢を採用（analyst_skills.md 規定） |

### 2-2. WACC算出の内訳

#### エクイティコスト（Ke）の計算

```
Ke = JGB 10Y + エクイティリスクプレミアム(ERP) × β

各パラメータ:
  JGB 10Y      = 1.50%
  ERP          = 5.50%  （Damodaran日本市場ERP推定値 4.5〜6.0%の中央値）
  β（大東建託） = 0.85   （住宅建設セクターβ 0.8〜1.0、安定したマスターリース
                          モデルによりマーケットβより低めに設定）

Ke = 1.50% + 5.50% × 0.85 = 1.50% + 4.675% = 6.175%
```

#### 負債コスト（Kd）の計算

```
Kd（税引前）  = 1.50%  （格付けAA相当、スプレッド最小）
実効税率      = 30.0%
Kd（税引後）  = 1.50% × (1 - 0.30) = 1.05%
```

#### WACC加重平均

```
資本構成（現行会計基準）:
  自己資本比率 E/V = 44.4%
  負債比率     D/V = 55.6%

WACC（会計上）= 6.175% × 0.444 + 1.05% × 0.556
             = 2.742% + 0.584%
             = 3.33%

⚠️ 新リース会計後の調整:
  自己資本比率が15.5%に低下した場合のWACC（参考値）:
  = 6.175% × 0.155 + 1.05% × 0.845 = 0.957% + 0.887% = 1.84%
  ※ 見かけ上WACCが下がるが、これは財務リスク上昇を反映していない
  ※ 実態リスクを保守的に反映し、本分析では WACC = 5.3% を採用
     （新リース会計後の実質的な財務レバレッジ上昇を考慮した保守的調整値）
```

### 2-3. FCF成長率の前提

| フェーズ | 期間 | 成長率 | 根拠 |
|---------|------|-------|------|
| **明示的予測期間** | 2027〜2031（5年） | **+3.0%/年** | ROE 20%超・入居率高水準・受注実質横ばいから保守的成長を想定。特別利益（FY26: 約100億円）はFCFに含めない |
| **ターミナル期間** | 2032年〜永続 | **+1.0%/年** | 日本名目GDP成長率の長期予測（内閣府: 1.0〜1.5%）の下限。人口動態・住宅ストック飽和リスクを考慮し下限採用 |

### 2-4. ターミナルバリュー（TV）の算出

```
Gordon Growth Model:
  TV = FCF_5 × (1 + g_terminal) / (WACC - g_terminal)
     = FCF_5 × 1.01 / (5.3% - 1.0%)
     = FCF_5 × 1.01 / 4.3%
     = FCF_5 × 23.49倍

FCF_5（5年後FCF）= 45,000百万円 × (1.03)^5 = 45,000 × 1.1593 ≈ 52,169百万円

TV = 52,169 × 23.49 ≈ 1,225,450百万円（割引前）
PV(TV) = 1,225,450 / (1.053)^5 = 1,225,450 / 1.2950 ≈ 946,277百万円
```

**シナリオA（楽観）: 条項が新会計基準除外合意済み**

- 大東建託がこの合意を取得済みであれば、コベナンツリスクは実質ゼロ
- 確認方法: 27/3期中の新リース会計開示時のIR・説明資料で「コベナンツへの影響なし」の記載を確認

**シナリオB（悲観）: 条項が旧基準ベースのまま**
- 自己資本比率15.5%は一般的なコベナンツ閾値（30%）を大幅に下回る
- 即時の期限の利益喪失リスクは低いが、銀行との協議が必要となりリファイナンスコストが上昇
- 社債の場合は償還・条件変更を要求されるケースも存在
- このシナリオでは純有利子負債が実質的に増加し、DCFモデルの株式価値を大幅に押し下げる

DCF法に基づく試算では理論株価は現在株価を大幅に上回る結果となったが、これはベースFCF（450億円）と発行済株式数（3,900万株）の推定値に基づくものであり、実績値との乖離がある可能性が高い。本分析の本質的な価値は**「WACCの感応度」と「コベナンツリスクの早期発見」**にある。

リスク分析の結果、最も警戒すべきは🔴 Criticalの**日銀利上げリスク**と🟠 Highの**新リース会計コベナンツ抵触リスク**の2点である。前者はJGB 10Yが2.5%を超えると理論株価が現在株価水準に収束する。後者は会社側がコベナンツ除外合意を取得済みかどうかで評価が180度変わる。この2点の確認なしに積極的な買い判断を下すことはリスク管理上許容できない。

投資判断としては、**①4月30日の4Q本決算でFCF実績・純有利子負債を確認、②5月22日の社長ミーティングでコベナンツ対応状況を確認、③DCFモデルを実績値で再構築**の3ステップを経てから最終判断を行う。現時点での推奨は「中立・要調査継続」。
