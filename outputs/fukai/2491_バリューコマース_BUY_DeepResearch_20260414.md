**日本アフィリエイト市場の現状：**

アフィリエイト広告市場は2019〜2020年にピークを形成後、以下の要因で構造的縮小フェーズに入っている：

```
市場縮小の3層構造：
━━━━━━━━━━━━━━━━━━━━━━━━━━
Layer 1: 検索エンジン変化（最急速）
  Google SGE / AI Overview → 検索流入がアフィリエイトサイトに届く前に遮断
  影響: メディア（ブロガー・比較サイト）の広告収入▲20〜40%（業界推計）
  → バリューコマース：メディア離脱 → 広告主に提供できるトラフィック減少
━━━━━━━━━━━━━━━━━━━━━━━━━━
Layer 2: 広告主ROI悪化
  CVR（転換率）低下 → CPA単価上昇圧力 → 広告主の予算削減
  影響: 中小広告主のアフィリエイト離れ加速
━━━━━━━━━━━━━━━━━━━━━━━━━━
Layer 3: 代替チャネル台頭
  TikTok/Instagram広告・インフルエンサー直接取引・Amazon広告
  → 従来型ASP経由のアフィリエイトの相対的地位低下
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 4. DCFバリュエーション

### 4-1. 前提条件（全て明記）

| パラメータ | 数値 | 根拠 |
|-----------|------|------|

| 有利子負債比率（D/E） | **5%（推計：実質無借金）** | 財務体質から推計 |
| 負債コスト（Rd） | **1.5%** | 日本優良中小企業水準 |
| 実効税率（Tc） | **30%** | 日本法定実効税率 |
| WACC | **≒ 8.5%** | 下記計算参照 |
| ターミナル成長率（g） | **▲1.0%**（基本）/ **0.0%**（強気）/ **▲3.0%**（弱気） | 衰退事業として構造的マイナス成長を前提 |

**WACC計算詳細：**
```
E/(D+E) = 95%、D/(D+E) = 5%
Re = 8.77%
Rd × (1-Tc) = 1.5% × 0.70 = 1.05%

WACC = 0.95 × 8.77% + 0.05 × 1.05%
     = 8.33% + 0.05%
     = 8.38% ≒ 8.4%
```
