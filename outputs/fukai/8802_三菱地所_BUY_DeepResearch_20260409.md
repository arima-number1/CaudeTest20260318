# 4. DCFバリュエーション

## 4-1. WACC算出

### Step 1: 株主資本コスト（Re）

```
Re = Rf + β × ERP

Rf  = 2.411%（JGB10Y 日本10年国債 確定値）
β   = 1.05（不動産デベ・市場感応度やや高め、Bloomberg推計参照）
ERP = 5.5%（日本株 Damodaran ERP 2026年推計: 5.0〜6.0%中央値）

Re = 2.411% + 1.05 × 5.5%
   = 2.411% + 5.775%
   = 8.186%
```

### Step 2: 負債コスト（Rd）

```
Rd  = 1.8%（格付A相当 長期固定借入の実績コスト想定）
Tc  = 30%（法定実効税率）
Rd × (1-Tc) = 1.8% × 70% = 1.26%（税引後）
```

### Step 3: 資本構成

```
D  = 有利子負債 ¥4.5兆円（推計中央値）
E  = 時価総額 ¥4.82兆円（¥4,772 × 1,010M株）
D+E = ¥9.32兆円

E/(D+E) = 4.82/9.32 = 51.7%
D/(D+E) = 4.50/9.32 = 48.3%
```

### Step 4: WACC

```
WACC = [51.7% × 8.186%] + [48.3% × 1.26%]
     = 4.232% + 0.609%
     = 4.841% ≒ 4.84%
```

## 4-4. DCF理論株価算出

```
【DCF計算（WACC 4.84%, g 1.5%）】

PV(FCF Year 1〜10) の累計:
  各年FCFを4.84%で割引 → PV合計 ≒ ¥8,050億円

ターミナルバリュー:
  TV = FCF(Year 10) × (1+g) / (WACC - g)
     = 1,400億円 × 1.015 / (4.84% - 1.5%)
     = 1,421億円 / 3.34%
     = ¥42,545億円（¥4.25兆円）

  = ¥42,545億円 / (1.0484)^10
  = ¥42,545億円 × 0.6255
  = ¥26,612億円（¥2.66兆円）

企業価値(EV):
  = PV(FCF) + PV(TV)
  = ¥8,050 + ¥26,612

  = ¥3.47兆 - ¥3.5兆 = ▲¥0.03兆???

⚠️ DCF(FCFベース)は不動産デベでは過小評価になる点に注意。
```

## 4-5. 修正DCF（NAV加算型）— 不動産向け適正手法

```
【不動産デベロッパー適正バリュエーション手法】
純粋DCFよりも「DCF + NAV」のハイブリッドが業界標準

修正後 理論株価:
  = (DCFバリュー × 40%) + (NAVバリュー × 60%)
  ※ 不動産デベは賃貸収益(DCF)と含み益(NAV)の双方が価値源泉

DCF寄与（FCFベース）:
  = 上記DCVが機能しない点を修正。EBITDA EV/EBITDAで補完。
  EV/EBITDA = 20倍（国内不動産デベ適正水準15〜22倍の中央）
  EBITDA ¥3,750億円 × 20倍 = EV ¥7.5兆円
  株主価値 = ¥7.5兆 - ¥3.5兆（純有利子負債）= ¥4.0兆円
  EV/EBITDA理論株価 = ¥4.0兆 / 1,010M株 = ¥3,960/株

  NAV/株 = ¥5,150（§2-1-E参照）

  = (¥3,960 × 40%) + (¥5,150 × 60%)
  = ¥1,584 + ¥3,090
  = ¥4,674/株（下限：保守的）

上限（NAV比率70%）:
  = (¥3,960 × 30%) + (¥5,762 × 70%)
  = ¥1,188 + ¥4,033
  = ¥5,221/株

【DCF + NAV ハイブリッド 理論株価レンジ】
  保守的: ¥4,674
  中央値: ¥4,950〜5,200
  楽観的: ¥5,221〜5,762
```
