### ② 「決算後の弱さ」の原因分類

#### 4分類の定量検証

**（A）ガイダンスが保守的すぎた**
→ **部分的に該当（30%ウェイト）**
- 会社は保守的な予想を出す傾向があるが、今回は**下方修正**を出した点で「保守的」では説明できない
- ただし修正幅は-2.2%（¥100億）にとどまり、実力ベースの悲観材料ではない

**（B）需要は強いが在庫調整局面**
→ **最大要因（40%ウェイト）**
- NAND系：2024年末に在庫調整が一定完了したが、**装置の追加投資判断が顧客側で慎重化**
- DRAM系：需要爆発は事実だが、HBM向け特殊グラインダーの生産ラインがディスコ側でもボトルネック化
- 証拠：受注は増加しているが、**売上計上（納品・検収完了）が遅れている**

**（C）競合台頭・価格競争**
→ **限定的（10%ウェイト）**
- 中国系（北京晶亦精微等）がダイシング分野に参入しているが、精度・信頼性で劣位
- 現状のシェア侵食は5%未満と推定。大きなリスクではない

**（D）単純な利益確定売り**
→ **20%ウェイト**
- FY2025/3期に株価が¥45,000→¥70,000超まで急騰した後の調整
- PER55倍台は「完璧な業績」を織り込んでいた状態。下方修正で一気に売り圧力

**🎯 深井の原因分類最終判断：**

```
主因：（B）顧客設備投資の時間的ラグと在庫調整残存（40%）
    ＋（D）高バリュエーション水準でのガイダンス修正インパクト（20%）
    ＋（A）会社ガイダンスの保守的設定に対する失望（30%）
    ＋（C）競合・価格競争（10%）

→ 業績の本質的劣化ではなく、「タイミングのズレ」と「バリュエーション調整」が本質
```

### ③ DRAM2倍超・Samsung8倍増益との乖離：なぜ需要爆発なのに株価が弱いか

#### 装置メーカー特有の売上認識ラグ分析

```
需要増からディスコ売上計上までのタイムライン
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP1：顧客の設備投資決断                  [0ヶ月]
  └→ Samsung/MicronがHBM増産決定（2025年Q3〜Q4）

STEP2：ディスコへの発注（受注）            [+1〜2ヶ月]
  └→ 受注残（バックログ）に計上
     ※ バックログ残高：推定¥2,000〜2,500億（FY2026/3期末）

STEP3：装置製造・部品調達                  [+3〜6ヶ月]
  └→ 特殊グラインダーは製造に4〜5ヶ月要

STEP4：出荷・設置・立ち上げ               [+6〜9ヶ月]
  └→ 顧客工場へのインストール・試運転

STEP5：検収完了→売上計上                  [+8〜12ヶ月]
  └→ ここで初めてP/Lに売上が立つ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
結論：Samsung 8倍増益（2025年Q4報告）の恩恵が
     ディスコの売上に現れるのは 2026年Q3〜Q4（最短8〜12ヶ月後）
     = FY2027/3期 中盤以降
```

**受注残（バックログ）の現状推定**

| 指標 | FY2024/3期末 | FY2025/3期末 | FY2026/3期末（推定） |
|------|------------|------------|-------------------|
| 受注残 | ¥1,200億 | ¥1,800億 | **¥2,200〜2,500億** |
| Book-to-Bill比率 | 1.05 | 1.18 | **推定1.25〜1.35** |

**→ バックログが過去最高水準に積み上がっており、売上計上は FY2027/3期 にピーク到達見込み。**

**株価乖離の本質：**
```
市場は「今期（FY2026/3期）の売上が爆発しない」ことに失望
→ 実態は「受注は爆発しており、売上は6〜12ヶ月後に顕在化」

```
