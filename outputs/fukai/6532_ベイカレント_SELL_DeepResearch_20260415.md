### 2-2. PBR7.07倍の正当性検証

#### ■ PBR正当化の理論的フレームワーク

```
PBR = ROE × PER
PBR = ROE × (1 / (Re - g))

現在のPBR: 7.07倍
BPS: ¥691
PER: 19.6倍
EPS: ¥249
```

**逆算ROE（PBR÷PERから）**:
```
ROE = PBR / PER = 7.07 / 19.6 = 36.1%
```

> ⚠️ **ROE取得失敗**（Yahoo Financeより）。上記は逆算値（参考値）。

**逆算ROE 36.1% の妥当性検証**:

| 指標 | 数値 | 評価 |
|------|------|------|
| 逆算ROE | 36.1% | **極めて高水準** |
| 日本コンサル業界平均ROE | 15〜25%程度 | ベイカレントは高い |
| アクセンチュア（参考） | 約30〜35% | 近似値 |
| PBR7倍に必要なROE（PER20倍前提） | 35%超 | **維持できるか？** |

**PBR7倍正当化の条件**:
1. ROE 35%超を**持続的に**維持すること
2. 利益成長率（g）が高水準であること
3. 投資家がプレミアム（人的資本×ブランド）を認め続けること

**正当化の崩壊条件**:
- 営業利益率が23%を下回り始める → ROE低下 → PBR圧縮
- EPS成長率が+10%以下に鈍化 → PER収縮 → 株価ダブルパンチ
- 日銀利上げでリスクフリーレート上昇 → 分母拡大 → 理論PBR低下

**深井判定**:
> **PBR7.07倍は「成長継続×高ROE維持」の両方が実現する楽観シナリオでのみ正当化される。成長鈍化の兆候が出た瞬間に急速なバリュエーション収縮が発生するリスクが高い。** 現状は「高ROEは本物だが、成長率の鈍化トレンドを市場がまだ十分に織り込んでいない」局面と判断。
