## 4. Fair Value算出と感応度分析

### 4-1. WACC計算

```
Rf     = 2.380%（固定）
β      = 1.2（海運株の典型的β。不動産セグメント約20%保有でβは大手海運より低め）
ERP    = 5.5%（前提固定）
Re（株主資本コスト）= 2.380% + 1.2 × 5.5% = 8.98%
Rd（負債コスト税引後）= 約2.0%（自己資本比率47.5%）
WACC   = 8.98% × 0.475 + 2.0% × 0.525 = 5.32%
```

### 4-2. PER法（主方法）

| シナリオ | EPS（使用） | 適用PER | FV | 現値比 |
|---------|-----------|--------|-----|------|
| 弱気（停戦・運賃急落） | ¥120（FY2027E） | 10.0x | ¥1,200 | -34.2% |
| ベース（市況回復） | ¥142（FY2027E） | 13.0x | ¥1,845 | +1.1% |
| **ベース+不動産NAV加算（SOP法）** | ¥142 + NAV¥680 | 11.0x | **¥2,242** | **+22.8%** |
| 強気（ホルムズ高原持続） | ¥174（FY2027E強気） | 11.0x | ¥1,914 | +4.9% |
| **強気+不動産NAV加算（SOP法）** | ¥174 + NAV¥680 | 11.0x | **¥2,594** | **+42.1%** |

**採用手法:** 海運事業PER × 11倍 + 不動産NAV（per share）の「Sum of Parts（SOP）法」を正値として採用。

### 4-3. PBR法（補足）

```
理論PBR = ROE / Re = 13.23% / 8.98% = 1.47x
理論FV  = BPS¥1,447 × 1.47 = ¥2,127
（現値PBR1.26 → 理論値に対し-14.3%ディスカウント）
```

### 4-4. DCF法（簡易）

```
FCF FY2027E ≈ 純利益¥155億 + 減価償却¥90億 - CAPEX¥120億 = ¥125億
成長率g      = 2.5%（長期：不動産成長+海運正常化）
Terminal FV  = ¥125億 / (5.32% - 2.5%) = ¥4,433億
PV（2年割引）= ¥3,997億
Net Debt調整 ≈ ¥3,097億
FV per share = ¥3,097億 / 1.089億株 = ¥2,844
```

### 4-5. FV統合（加重平均）

| 手法 | FV | ウェイト |
|------|-----|---------|
| PER×SOP（ベース+NAV） | ¥2,242 | 40% |
| PER×SOP（強気+NAV） | ¥2,594 | 20% |
| PBR理論値 | ¥2,127 | 20% |
| DCF | ¥2,844 | 20% |
| **加重平均FV** | **¥2,390** | 100% |

**目標株価:** ベースケース¥2,400〜¥2,550、強気ケース¥2,700（ホルムズ高原+CGコード再評価同時発動時）

## 7. 不動産NAV考察（東京都心保有物件）【飯野固有の差別化要素】

### 7-1. 保有物件概要

| 物件 | 所在地 | 特徴 |
|------|--------|------|

| 英国不動産 | ロンドン | 近年新規取得 |

### 7-2. NAV推計

```
不動産帳簿価格（連結BS推計）:
土地 + 建物（減価償却後）≈ ¥200〜¥300億（推定）

不動産時価（市場評価）:
千代田区一等地 坪単価¥3,000万〜¥5,000万/坪
飯野ビル延床面積：推定2.5万坪〜3万坪規模

時価評価：¥2.5万坪 × ¥4,000万/坪（中値）= ¥1,000億
→ 文献ベース推定: ¥1,000億超（最大¥1,310億超との試算あり）

含み益（時価 - 帳簿価格）:
  = ¥1,000億 - ¥250億（帳簿推定） = ¥750〜¥1,060億（推定範囲）
  
1株当たり不動産NAV超過分:
  ¥750億〜¥1,060億 / 1.089億株 = ¥689〜¥973/株
  
採用値（保守的）: ¥680/株
```

### 7-3. 不動産NAVと株価の関係

```
現値 ¥1,825:
  うち不動産NAV（帳簿超過分）= ¥680/株 → 37.3%が不動産含み益
  海運事業価値（残余）       = ¥1,825 - ¥680 = ¥1,145/株
  海運PER換算（EPS¥136）    = ¥1,145 / ¥136 = 8.4倍（極めて割安）

→ 「海運事業を過去最悪水準PERで買い、不動産含み益を半値以下で付いてくる」構造
```

### 7-4. 不動産NAV再評価トリガー（3点）

1. **CGコード改訂（2026年中予定）**: 不動産含み益の制度的開示義務化が近づく。東邦レマック同様の「含み益再評価ラリー」発生可能性
2. **飯野ビルテナント満床継続**: 空室率マーケット平均より低位で推移。賃料改定局面に入れば年間NOI改善
3. **J-REIT市場回復**: 東京都心ビル時価の顕在化が進む
