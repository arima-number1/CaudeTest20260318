## 2. 社長指示の重点分析：AI代替リスク × リユース盛況の両面精査

### 2-A. 🤖 AI代替リスク：メルカリは本当に脅かされるか？

#### AIが「代替可能」な部分

| リスク項目 | 代替可能性 | 時間軸 | 影響度 |
|----------|-----------|-------|-------|
| AIによる自動価格査定・買取（C2B2C） | **中〜高** | 2〜3年 | ★★★★☆ |
| AI搭載フリマ（新規参入）の競合 | **中** | 3〜5年 | ★★★☆☆ |
| AI自動出品・商品説明生成 | **低（むしろ追い風）** | 既に実装中 | ☆（メルカリが活用） |
| ChatGPT等での「商品探索」行動変容 | **低〜中** | 3〜5年 | ★★☆☆☆ |

#### AIが「代替困難」な部分（メルカリの堀）

```
① ネットワーク効果の壁
   → 2,200万MAUの出品在庫プールは一朝一夕に複製不可能
   → AIプラットフォームが参入しても「買い手と売り手の両側」を
     同時に確保するのに数年〜10年単位の時間コストが必要

② 物理的な取引インフラ
   → 梱包材・発送・決済・エスクロー・クレーム対応の
     実物流インフラはAIだけでは代替できない

③ 信頼スコア・評価システム
   → 累積取引履歴・評価データは「機密性の高い参入障壁」

④ 規制・コンプライアンス対応
   → 古物商許可・本人確認義務化への対応済みは新規参入コスト
```

#### 最大のAI代替リスクシナリオ（ブラックスワン）

```
シナリオ：Amazon・楽天・LINE（LY Corp）がAI搭載の
「自動査定＋即時買取＋再販」モデルを2026〜2027年に本格展開

影響：
- GMV成長率が現在の+10%→+3〜5%に鈍化
- テイクレート圧迫（値引き圧力）
- 国内営業利益率が22〜26%→15〜18%に低下

理論株価インパクト：▲20〜35%（¥2,400〜¥2,960）
確率評価：15〜20%（現時点では低いが監視必須）
```

#### 深井の総合判断（AI代替リスク）

> **「現在は脅威の芽、3〜5年後に脅威の花」と評価する。**
> 短期（1〜2年）はネットワーク効果で十分防衛可能。しかし中期（3〜5年）には競合AIプラットフォームの台頭により、**GMV成長率の鈍化**は避けられないリスクとして織り込むべき。
> 逆説的に、**メルカリ自身がAIを最も積極的に活用しているプレイヤー**であることを見落とすな。AI価格査定・出品補助・不正検知・レコメンド高度化は全てGMV成長とOPM改善に直結する。

## 4. DCFバリュエーション

### 4-1. WACC算出

#### インプット前提

| パラメータ | 値 | 根拠・出所 |
|----------|---|----------|
| **Rf（リスクフリーレート）** | **2.423%** | 日本10年国債（JGB10Y）/ 財務省 2026-04-14 |
| **ERP（株式リスクプレミアム）** | **5.5%** | 日本市場標準（Damodaran 2025推計） |
| **β（ベータ）** | **1.25** | 成長型テックプラットフォーム（東証グロース対比） |
| **Re（株主資本コスト）** | **2.423% + 1.25 × 5.5% = 9.30%** | CAPM |
| **Rd（有利子負債コスト）** | **1.5%** | 国内低金利環境 |
| **税率（Tc）** | **30.6%** | 日本法人税実効税率 |
| **D/E比率（負債資本比率）** | **D:E = 30:70** | 推計（軽量資産モデル） |

#### WACC計算

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.70 × 9.30%] + [0.30 × 1.5% × (1-0.306)]
     = 6.51% + 0.31%
     = 6.82%
```

| FY2035 | +6% | 307 | 0.518 | 159 |

```
ターミナルバリュー（TV）計算：
TV = FCF2035 × (1+g) / (WACC - g)
   = 307億 × 1.02 / (6.82% - 2.0%)
   = 313.1 / 4.82%
   = 6,496億円

企業価値（EV）= PV合計 + TV PV
              = 1,540 + 3,365
```
