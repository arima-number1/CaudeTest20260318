## 4. DCFバリュエーション

### 4-1. 前提条件（パラメータ設定）

```
■ マクロ前提
  リスクフリーレート (Rf)   : 2.380%（JGB10Y、社長確認値）
  エクイティリスクプレミアム : 5.50%（日本株・中小型割増）
  β（ベータ）              : 0.85（産業用電子・低ボラ中堅株）
  
■ 株主資本コスト
  Re = Rf + β × ERP
     = 2.380% + 0.85 × 5.50%
     = 2.380% + 4.675%
     = 7.055% ≈ 7.1%

■ 負債コスト
  借入金利（Rd）            : 1.80%（地銀・中堅企業水準）
  実効税率（Tc）            : 30%
  税引後負債コスト           : 1.80% × (1-0.30) = 1.26%

■ 資本構成（買収後推定）
  E/(D+E)                  : 75%（自己資本比率）
  D/(D+E)                  : 25%（有利子負債比率）

■ WACC計算
  WACC = [0.75 × 7.1%] + [0.25 × 1.26%]
       = 5.325% + 0.315%
       = 5.64% ≈ 5.6%

■ ターミナル成長率（g）
  基本シナリオ: 1.5%（国内製造業・緩やか成長）
  強気シナリオ: 2.0%
  弱気シナリオ: 0.5%
```

#### 現在価値計算（WACC=5.6%）

```
割引率: 5.6%

PV(FY2027) = 9.3 ÷ 1.056^2    = ¥8.35億円

   = 11.6 × 1.015 / (0.056-0.015)
   = 11.78 / 0.041
   = ¥287.3億円

PV(TV) = 287.3 ÷ 1.056^5      = ¥220.5億円

────────────────────────────────
企業価値（EV）= ¥42.55 + ¥220.5 = ¥263.1億円

  (-) 有利子負債（買収後推定）  : -¥28億円
```
