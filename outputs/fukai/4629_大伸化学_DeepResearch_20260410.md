## 2. 核心論点：シンナー不足 × ナフサ高騰の構造解析

### 2-1. シンナー不足がポジティブな3つの理由

**① 製造業者として「価格設定側」にいる**

```
原料（ナフサ由来BTX）→ 大伸化学が調合製造 → 製品シンナーを販売

製品価格設定のフロー:
  ナフサ高騰 → 原価上昇 → 転嫁（値上げ通知）→ 顧客へパス
                                  ↑
                      ここで大伸化学は「交渉する側」
```

業界実績：2026年3月、日本ペイント等の競合が**シンナー製品を75%値上げ**（ブルームバーグ報道）。業界全体での転嫁が始まっており、大伸化学も追随値上げが正当化されている。

**② FY2023の「転嫁実績」が証明済み**

2022〜2023年のウクライナ侵攻起因のナフサ高騰時、大伸化学は原価上昇を顧客へ転嫁し、増収増益を達成した。今回の中東危機はその再現シナリオ。転嫁能力はすでに証明済み。

**③ Q3実績：原材料コスト「低下」局面でさえOP+79.6%**

```
Q3累計（FY2026/4月〜12月）実績:
  売上: 前年比-2.0%（数量減）
  営業利益: 前年比+79.6% ← 「原材料価格低下」が主因
  経常利益: 前年比+70.4%
  純利益:   前年比+83.0%

→ 量が減っても利益が大幅増益 = 価格優位を享受
→ ナフサ上昇後に転嫁が先行すれば、さらなる利益拡大も
```

### 2-2. ナフサ高騰がリスクである理由と限界

**リスク①：転嫁ラグ（2〜3ヶ月）**

ナフサ価格が急騰しても、顧客への値上げ交渉・通知・実施には**2〜3ヶ月のタイムラグ**が発生する。この間は一時的にマージンが圧縮される。

**リスク②：需要破壊（顧客が買い控え）**

75%の大幅値上げは需要を抑制する可能性がある。特に建設・塗装向けの量需要は価格弾性が高い。

**リスク③：調達難（ホルムズ経由BTXの供給不足）**

トルエン・キシレン等のBTXはナフサ分解で製造。ホルムズ封鎖が長期化すると原料自体の入手が困難になりうる。ただし10,890KLの在庫バッファが**2〜3ヶ月分のクッション**として機能する。

### 2-3. 深井の結論：「転嫁先行型でネットポジティブ」

```
現在進行中のシナリオ:
┌─────────────────────────────────────────┐
│ ホルムズ封鎖 → シンナー需給タイト → 業界値上げ開始  │
│                                           │
│ 大伸化学: 在庫バッファ10,890KLを活用しながら        │
│          転嫁値上げを実施中（2026年4月〜5月）       │
│                                           │
│ FY2027:  転嫁後の高マージン × 量の漸回復 = 増益基調 │
└─────────────────────────────────────────┘
```

## 9. Q-1定性変化考察（ホルムズ発生後の変化）

### ポジティブ変化
- ✅ **業界全体の転嫁開始**（日本ペイント等が75%値上げ）: 大伸化学の転嫁交渉の「錦の御旗」が整った

- ✅ **PBR0.52x × 東証PBR改善要請**: 超低PBRへの市場の関心が高まる構造的背景

### ネガティブ変化

- ⚠️ **ADV¥20.4M/日の超低流動性**: 急落時に損切りできないリスク。ポジションサイズ管理が必須
