# 4. DCFバリュエーション

## 4.1 前提条件（全て明記）

### マクロ前提

| パラメータ | 値 | 根拠 |
|-----------|-----|------|
| **リスクフリーレート（Rf）** | **2.386%** | JGB10Y 2026-04-06取得（固定） |
| **ERP（株式リスクプレミアム）** | **5.5%** | 日本株標準（Damodaran 2025推計・日本新興株調整） |
| **β（ベータ）** | **0.85** | アパレル小売セクター・国内中規模（市場ベータ調整後） |
| **株主資本コスト（Re）** | **2.386% + 0.85 × 5.5% = 7.06%** | CAPM計算 |

### 資本構成前提

```
【WACC計算】
自己資本（E）   : 約¥850億円（BPS × 発行株数）
有利子負債（D） : 約¥175億円（推定中央値）
E/(D+E)         : 850 / (850+175) = 82.9%
D/(D+E)         : 175 / (850+175) = 17.1%
負債コスト（Rd）: 0.8%（低金利環境・優良企業格付け）
実効税率（Tc）  : 30%

     = 5.853% + 0.096%
```
