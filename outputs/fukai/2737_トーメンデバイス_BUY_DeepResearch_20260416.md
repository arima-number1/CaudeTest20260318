## 1. 企業概要・事業分析

### 1-1. 企業プロファイル

トーメンデバイス（2737.T）は、豊田通商（8015.T）の完全子会社として機能する半導体・電子部品専門商社である。1945年設立（旧東綿＝トーメン系譜）、東京都港区芝浦に本社を置く。時価総額は約810億円と小型株水域に位置するが、取扱品目の質・取引先の格は大型商社並みである。

**事業の本質：** 海外の一流半導体メーカー（TI・ON Semiconductor・Infineon等）と国内の自動車・産業機器メーカーをつなぐ「技術商社」。単なる物流代行ではなく、技術サポートやサプライチェーン管理まで提供するバリューアド型ビジネスモデル。

### 1-2. セグメント構成（推計ベース）

| セグメント | 売上構成比（推計） | 粗利率（推計） | 主要仕入先・特徴 |
|-----------|-----------------|-------------|----------------|
| メモリ半導体（DRAM/NAND/SSD） | 約35% | 約5〜9% | Samsung・SK Hynix・Micron。**市況感応度最大** |
| System LSI・ロジック | 約28% | 約8〜12% | TI・Infineon・ON Semi。ストック型・安定収益 |
| 表示関連（LCD/OLED） | 約15% | 約6〜10% | BOE・JDI・Sharp |
| 電子部品（MLCC・LED等） | 約15% | 約8〜13% | TDK・村田・京セラ系 |
| その他（受託開発等） | 約7% | 約15〜20% | バリューアド高付加価値 |

> **核心ポイント：** メモリセグメントはDRAM/NAND現物価格の変動に直接連動する。Q1のDRAM価格が前四半期比2倍超となった環境下では、商社の在庫利益（在庫保有中の値上がり益）が急拡大する典型パターン。

### 2-3. DRAM価格感応度 Delta（1%変動あたりの業績インパクト）

```
前提条件：
  - DRAM/NAND売上比率：35%
  - 売上規模：推計¥1,400億（FY26通期）
  - DRAM売上：推計¥490億
  - 粗利率：7%（平常時）→ 在庫効果でFY26下期は9〜11%へ改善推定

  在庫利益押し上げ効果：¥490億 × 1% × 平均在庫回転1.5カ月 ÷ 12 × 75%（在庫化比率）
  ≈ ¥0.46億（年間換算）

DRAM価格+100%（FY26実績水準）の場合：
  EPS押し上げ：推計+¥700〜900（+40〜55% vs. FY25）
```

> **Rationale：** 在庫回転が1.5カ月（年8回転）と仮定。実際には期末在庫の評価益として一括計上されるケースもあり、Q4（1〜3月）の業績に集中する可能性が高い。これが4/24決算でのサプライズの主因となりうる。

### 3-3. DDM（配当割引モデル）によるFair Value算出

```
DDMモデル（ゴードン成長モデル）：

入力値：
  直近年間配当（実績）：¥11,920 × 4.58% = ¥546/株
  配当成長率（g）：推計+8%/年（DRAM好況サイクル×増配余力）
  Rf：2.380%（固定）
  リスクプレミアム（ERP）：5.5%（日本株標準）
  Beta：0.396（実測）
  
CAPM割引率（Ke）：
  Ke = Rf + Beta × ERP
     = 2.380% + 0.396 × 5.500%
     = 2.380% + 2.178%
     = 4.558%

DDM-FV：
  FV = D1 / (Ke - g)
     = ¥546 × 1.08 / (4.558% - 8%)
  ※ g > Ke の逆転状態 → DDM単独は成立しない（成長率が高すぎる過渡期）
  
調整：成長期2段階モデルを採用
  第1フェーズ（FY26-27）：g=+15%（DRAM好況期）
  第2フェーズ（FY28以降）：g=+4%（正常化後）
  
  FV2段階 = Σ(D0×1.15^t / 1.04558^t) [t=1,2] + [D2×1.04 / (0.04558-0.04)] / 1.04558^2
  D0 = ¥546, D1 = ¥628, D2 = ¥722
  FV ≈ (¥628/1.046) + (¥722/1.093) + (¥722×1.04/0.00558) / 1.093
     ≈ ¥600 + ¥660 + (¥750 / 0.00558) / 1.093
     ≈ ¥1,260 + ¥134,409 / 1.093
  → DDMはターミナル部分が過大になるため、補完的に使用のみ
```

> **DDM補足Rationale：** 成長率gがKe（4.6%）を上回る超過成長局面はDDM単独では算出不能。このため、下記のPER倍率法・EV/EBITDA法を主軸に据える。
