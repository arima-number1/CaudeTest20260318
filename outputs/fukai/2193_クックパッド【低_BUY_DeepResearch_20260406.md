### 2.3 ③ TOPIX除外による需給インパクト試算

#### TOPIX除外の背景
クックパッドは東証スタンダード市場に属しており、TOPIXの算出対象から段階的に除外または比重縮小の影響を受けている。

#### 需給インパクト試算

```
推定TOPIX関連パッシブファンドの保有比率：約5〜10%（推定）
時価総額：約¥61.4億
パッシブ売却推定額：¥61.4億 × 5〜10% = ¥3〜6億

日次出来高（推定）：約50〜150万株 × ¥133 = 約¥0.7〜2.0億/日

売却インパクト解消に要する日数：
  ¥3〜6億 ÷ ¥1.0億/日（平均出来高想定） = 約3〜6営業日分の売り圧力

株価への影響推定：▲3〜8%程度の需給的な下押し圧力
  → 既に株価に一定程度織り込まれた可能性が高い
  → 除外完了後は需給改善となるポジティブ転換点になり得る
```

> **結論**：TOPIX除外は一時的な需給悪化要因だが、**完了後はむしろカタリストに転換**する可能性がある。

## 4. DCFバリュエーション

### 4.1 前提条件

#### 資本コスト計算

```
■ リスクフリーレート（Rf）：2.380%（JGB10Y、2026-04-06確認値）

■ 株主資本コスト（Re）
  β：約0.80〜1.00（小型・グロース系メディア株として推定）
  → 保守的β：0.90を採用
  ERP（株式リスクプレミアム）：5.5%（日本市場標準）
  Re = 2.380% + 0.90 × 5.5%
     = 2.380% + 4.950%
     = 7.330%

■ 負債コスト（Rd）：2.0%（ほぼ無借金のため形式的に設定）
  実効税率（Tc）：30%
  Rd × (1-Tc) = 2.0% × 70% = 1.4%

■ 資本構成
  E（株主資本）：¥65〜75億 → 約90%
  D（有利子負債）：約¥5〜10億 → 約10%
  
■ WACC計算
  WACC = [90% × 7.330%] + [10% × 1.4%]
        = 6.597% + 0.140%
        = 6.737% ≈ 6.74%

■ 永続成長率（g）：▲1.0%〜+1.0%（売上減少考慮）
  → 基本シナリオ：g = 0.0%
  → 強気シナリオ：g = +1.0%
  → 弱気シナリオ：g = ▲2.0%
```

### 4.2 DCFモデル（基本シナリオ）

| FY2030E | ¥100 | +5% | ¥5.0 | ¥3.5 | ¥2.5 | ▲¥2.0 | +¥4.0 |

#### DCF計算（基本シナリオ）

```
割引率（WACC）：6.74%

PV of FCF:
  FY2026: ▲¥0.5億 / 1.0674¹ = ▲¥0.47億
  FY2027: +¥1.0億 / 1.0674² = +¥0.88億
  FY2028: +¥2.4億 / 1.0674³ = +¥1.98億
  FY2029: +¥3.3億 / 1.0674⁴ = +¥2.55億
  FY2030: +¥4.0億 / 1.0674⁵ = +¥2.89億

予測期間PV合計 = ▲¥0.47 + ¥0.88 + ¥1.98 + ¥2.55 + ¥2.89
              = 約¥7.83億

ターミナルバリュー（TV）：
  TV = FCF₂₀₃₀ × (1+g) / (WACC - g)
     = ¥4.0億 × (1+0%) / (6.74% - 0%)
     = ¥4.0億 / 0.0674
     = 約¥59.3億

PV of TV = ¥59.3億 / 1.0674⁵ = 約¥42.9億

企業価値（EV）= ¥7.83億 + ¥42.9億 = 約¥50.7億

株主価値 = EV + 純現金
         = ¥50.7億 + ¥58.0億（現金¥65億 - 負債¥7億）
         = 約¥108.7億

理論株価（基本）= ¥108.7億 / 4,618万株
              ≈ 約¥235/株
```

> ⚠️ **感応度注記**：ターミナルバリューが企業価値に占める割合が非常に高い（約85%）。永続成長率・WACCの前提変化で理論株価は大きく振れる。
