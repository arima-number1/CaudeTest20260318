| パラメータ | 設定値 | 根拠 |
|-----------|--------|------|
| **Rf（リスクフリーレート）** | **2.409%** | 日本10年国債（JGB10Y）2026年4月17日財務省日次確認値 |
| **β（ベータ）** | **1.15** | 証券業は市場感応度が高い。東証証券業インデックスとの相関から推定（中堅対面証券として市場平均より高め） |
| **ERP（株式リスクプレミアム）** | **5.5%** | 日本市場の標準的ERP（Damodaran 2026年推定値：5.0〜6.0%の中央値） |
| **Re（株主資本コスト）** | **2.409% + 1.15 × 5.5% = 8.73%** | CAPM |
| **Rd（負債コスト）** | **1.5%** | 証券業の借入コスト（レポ・コール含む加重平均） |
| **Tc（実効税率）** | **30%** | 日本法人標準 |
| **資本構成（E/D+E）** | **E: 75% / D: 25%** | 証券業の典型的レバレッジ（自己資本比率高め） |
| **WACC** | **下記計算参照** | |

#### WACC計算

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.75 × 8.73%] + [0.25 × 1.5% × (1-0.30)]
     = 6.55% + 0.26%
     = 6.81%
```

**WACC = 6.81%**

### 4-2. DCF計算（残余利益モデル併用）

#### 証券業DCFの特殊性
> 証券業はフリーキャッシュフロー（FCF）の定義が複雑なため、**残余利益モデル（Residual Income Model: RIM）** を主軸とし、**配当割引モデル（DDM）** を補完として使用する。

---

#### 手法1：残余利益モデル（RIM）

```
RIM理論株価 = BPS + Σ[残余利益(t) / (1+Re)^t] + 継続価値

残余利益(t) = EPS(t) - Re × BPS(t-1)

前提：
  現在BPS     = ¥3,016
  Re          = 8.73%
  ROE想定（基本シナリオ）
    FY27      = 12.0%（速報値悪化を受け下方修正）
    FY28      = 12.5%
    FY29      = 13.0%
    FY30以降  = 11.0%（長期正常化ROE）
  配当性向    = 30%（EPS¥295〜325に対し¥100配当）
```

| 年度 | ROE | BPS推定 | EPS推定 | Re×BPS | 残余利益 | PV(8.73%) |
|------|-----|---------|---------|--------|---------|-----------|
| FY27 | 12.0% | ¥3,250 | ¥390 | ¥263 | ¥127 | ¥116.8 |
| FY28 | 12.5% | ¥3,480 | ¥406 | ¥284 | ¥122 | ¥103.3 |
| FY29 | 13.0% | ¥3,720 | ¥453 | ¥304 | ¥149 | ¥116.1 |
| FY30 | 11.0% | ¥3,900 | ¥409 | ¥409 | **¥0** | — |
| 継続価値PV | — | — | — | — | ROE→Re収束で継続価値≒0 |  |

```
RIM理論株価 = ¥3,016 + ¥116.8 + ¥103.3 + ¥116.1 + (継続価値≒小)
            ≈ ¥3,016 + ¥336 + 継続価値¥150〜200
            ≈ ¥3,502〜¥3,552
```

---

#### 手法2：配当割引モデル（DDM）

```
Gordon Growth Model（定常成長DDM）
  D1（来期配当予想）= ¥100/株（維持想定）
  g（永続成長率）   = 2.0%（名目GDP成長+インフレ）
  Re              = 8.73%

P = D1 / (Re - g) = ¥100 / (8.73% - 2.0%) = ¥100 / 6.73% = ¥1,486

⚠️ DDMは配当水準が低すぎるため過小評価。あくまで配当の「床」として参考値。
```

---

#### 手法3：PBR×ROEアプローチ（補完）

```
適正PBR = (ROE - g) / (Re - g)
        = (12.0% - 2.0%) / (8.73% - 2.0%)
        = 10.0% / 6.73%
        = 1.487倍

適正株価 = 適正PBR × BPS = 1.487 × ¥3,016 = ¥4,483

注：これは長期的な均衡値であり、現在の市場環境（ROE低下懸念）を反映すると
    FY26下方修正後ROE（11.5〜12.5%）での再計算が必要。

下方修正後ROE 11.5%での適正PBR：
  = (11.5% - 2.0%) / (8.73% - 2.0%) = 9.5% / 6.73% = 1.411倍
  適正株価 = 1.411 × ¥3,016 = ¥4,256

ROE 10.5%（悲観）：
  = (10.5% - 2.0%) / (8.73% - 2.0%) = 8.5% / 6.73% = 1.263倍
  適正株価 = 1.263 × ¥3,016 = ¥3,809（現在値とほぼ一致）
```

#### 手法4：PER比較法

```
証券業セクター平均PER（2026年4月）：推定10〜12倍
岩井コスモ 現在PER：11.0倍（EPS¥348ベース）

速報値EPS（¥295〜325）で再計算：
  EPS¥310で現在株価¥3,810 → 実質PER = 12.3倍（割高方向にシフト）
  
適正PER（セクター平均下限10倍）での理論株価：
  10倍 × EPS¥310 = ¥3,100
  11倍 × EPS¥310 = ¥3,410
  12倍 × EPS¥310 = ¥3,720
```
