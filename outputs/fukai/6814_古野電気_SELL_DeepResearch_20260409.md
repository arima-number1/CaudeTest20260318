## 4-1. WACC計算

```
【WACC計算（社長指示固定値使用）】

■ パラメータ設定
  Rf（JGB10Y）：2.374%（財務省 R8.4.8 取得値）

  ERP（日本株式リスクプレミアム）：5.5%（Damodaran推計 2026年）

■ 株主資本コスト（Re）
  Re = Rf + β × ERP
     = 2.374% + 1.35 × 5.5%
     = 2.374% + 7.425%
     = 9.80%

       = 9.34%

  【使用WACC：9.34%（Base Case）】
```
