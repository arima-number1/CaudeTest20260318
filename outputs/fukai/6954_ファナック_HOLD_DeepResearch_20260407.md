### 中国喪失シナリオの株価織り込み度推計

**アプローチ：**
- 正常ケースDCF（中国リスクなし）→ フルバリュー推計
- 中国売上50%減ケースDCF → ディスカウント後バリュー
- 現在株価との比較から「織り込み度」を逆算

```
織り込み計算（簡易）
────────────────────────────────────────────
フルバリュー（リスクなし）：  ¥8,500〜9,000（後述DCFより）
中国50%減ケース FairValue：  ¥4,800〜5,200（後述DCF悲観シナリオ）
現在株価：                   ¥5,451

織り込み度 = (フルバリュー - 現在株価) / (フルバリュー - 悲観値)
           = (8,750 - 5,451) / (8,750 - 5,000)
           = 3,299 / 3,750
           ≈ 88%
────────────────────────────────────────────
```

**→ 現在株価は「中国売上50%喪失」シナリオを約85〜90%織り込んでいると推計。**
**言い換えれば、現在の株価はほぼ「悲観シナリオ」を前提にしており、正常ケースへの回帰余地（アップサイド）は相当に大きい。**
