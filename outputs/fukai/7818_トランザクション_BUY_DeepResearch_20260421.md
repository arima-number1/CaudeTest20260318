## 公正価値算定（4手法ブレンド）

### CAPM前提

| パラメータ | 値 | 根拠 |
|---|---|---|
| Rf（10年JGB） | **2.380%** | 竹村TCプロトコル固定値 |
| Beta | **0.717** | yfinance取得（低感応度ファブレス） |
| ERP | 6.0% | 竹村TCプロトコル標準 |
| **r（株主資本コスト）** | **6.682%** | 2.380% + 0.717×6.0% |

> ⚠️ 提供分析はr=10%（市場期待資本コスト）を使用→当社プロトコルと**+3.3%ptの乖離**。低betaを反映すべきであり、FVは提供分析より有意に高くなる。

---

### 手法1：Forward P/E（FY08/26E基準）

- FY08/26E EPS：¥82.1（いちよし OP ¥6.30B→NI率73.7%→EPS換算）
- 適正P/E算定根拠：ROE21%・g=10%成長フェーズ、低betaの安定成長型
  - PBR/ROE→terminal P/E換算：(payout × (1+g_terminal)) / (r - g_terminal) = (0.40×1.03) / (0.0668-0.03) = **11.2x**（ターミナル値）
  - 成長フェーズプレミアム：中計CAGR10%×PEGレシオ1.8 = **18.0x**
  - 採用P/E：**20x**（成長フェーズ中間値として保守的）
- **FV = ¥82.1 × 20x = ¥1,642**

---

### 手法2：PBR/ROE Justified Model

- ROE：21.42%
- g（保守的terminal）：3.0%
- r：6.682%
- **Justified PBR = (ROE - g) / (r - g) = (0.2142 - 0.03) / (0.0668 - 0.03) = 0.1842 / 0.0368 = 5.00x**
- FY08/26E BPS：¥346.08 × (1 + 21.42% × 60%) = ¥346.08 × 1.1285 = **¥390.6**
  （ROE×内部留保率12.85%で1年成長）
- **FV = ¥390.6 × 5.00 = ¥1,953**
  → ただし成熟時ROE逓減を考慮し**3.50x適用**：¥390.6 × 3.50 = **¥1,367**

---

### 手法3：中計P/E逆算（FY08/30E目標値割引）

- FY08/30E OP ¥9.5B → NI ¥7.0B → EPS **¥123.8**（株数56.54M固定）
- FY30ターミナルP/E：**15x**（成熟後、r=6.68%・g=3%環境下での保守値）
- FY30 FV：¥123.8 × 15 = ¥1,857
- 4年割引（r=6.682%）：¥1,857 / (1.0668)^4 = ¥1,857 / 1.298 = **¥1,431**

---

### 手法4：Two-Stage DDM（参考値・低payout補正後）

- 現配当：¥30/share（post-split、配当性向40%）
- Phase 1（4年）：g=10%
- Phase 2（terminal）：g=3%、r=6.682%
- PV Phase 1：¥33.0/1.0668 + ¥36.3/1.0668² + ¥39.9/1.0668³ + ¥43.9/1.0668⁴ = **¥129.9**
- ターミナルPV：(¥43.9×1.03/0.0368) / 1.0668⁴ = (¥1,228) / 1.298 = **¥946**
- **FV（DDM） = ¥1,076**
- ⚠️ 低payout率40%のため、DDMは成長価値を過小評価。参考値として重みを下げる。

## リスクシナリオ定量

### リスク①：中国調達×コンテナ運賃高騰（最重要）
- **現状**：ホルムズ封鎖継続でコンテナ運賃2倍水準維持（BDI2,095、4/11時点）
- **仮定**：FY08/26通期コスト増試算
  - 輸入比率：売上¥30.2Bの想定仕入¥18B、うちアジア調達¥15B
  - 運賃コスト10%増 → ¥150M増加 → OP▲¥150M → EPS▲¥2.0/share → FV▲¥40
- **FV下方修正額**：▲¥40（▲2.8%）→ BUY判断は維持

### リスク②：USD高止まり（FX未ヘッジ FY08/26）
- **現状**：USD/JPY 159.0、FY08/27よりFX先物開始予定
- **仮定**：USD/JPY 165（円安10%進行）
  - USD建て仕入¥15Bの10%増 → OP▲¥1.5B → EPS▲¥19.6 → FV▲¥392（▲27%）
  - **BUY→HOLD転換トリガー**：USD/JPY 165超が長期固定化する場合
- **FV下方修正額**：▲¥392（最悪シナリオ、50%確率）→ ¥1,058

### リスク③：D/E比率2.33x（財務リスク）
- **注意**：yfinanceのD/Eは総負債/株主資本の可能性（実質有利子負債は低い可能性）
- 金利費用がOP→NI変換に大きく影響しない（FY25 NI/OP=73.7%は健全）
- **★IR問合せ要**：有利子負債の実額と内訳（仕入れ債務との分離）
