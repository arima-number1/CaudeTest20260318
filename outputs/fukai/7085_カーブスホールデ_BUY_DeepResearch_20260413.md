**ビジネスモデルの本質：「資産軽量型FCロイヤリティマシン」**

```
【収益の流れ】
FC加盟店（オーナー運営）
　↓ 月次ロイヤリティ（売上の一定比率 or 定額）
カーブスHD（本部）
　↓ マージン大・投資少・リスク小
営業利益率：推定18〜22%（高水準）

【直営 vs FC比率（推定）】
　直営店：数十店舗（主要都市・モデル店）
　FC店舗：約1,900〜2,100店舗（全体の95%超）
　→ 固定費は直営分のみ負担。売上拡大に伴う限界費用ほぼゼロ
```

## 4. DCFバリュエーション

### 4-1. DCF前提条件（全開示）

#### WACC計算

```
【インプット】
　Rf（リスクフリーレート）= 2.439%（日本10年国債 2026/04/13）
　ERP（エクイティリスクプレミアム）= 6.5%（日本市場標準推計）
　β（カーブスHD推定）= 0.75
　　→ ディフェンシブ消費×ニッチ独占×景気感応度低 → 市場β<1.0と判断

　株主資本コスト（Re）:
　Re = Rf + β × ERP
　   = 2.439% + 0.75 × 6.5%
　   = 2.439% + 4.875%
　   = 7.314%

【資本構成】
　自己資本比率（E/D+E）= 推定75%（FC型ビジネスは無借金〜低借入傾向）
　負債比率（D/D+E）= 推定25%
　負債コスト（Rd）= 1.5%（低金利環境・優良中堅企業）
　法人税率（Tc）= 30%

　WACC計算:
　WACC = [0.75 × 7.314%] + [0.25 × 1.5% × (1-0.30)]
　      = 5.486% + 0.263%
　      = 5.749%
　      ≈ 5.75%
```

#### FCF前提（5年予測）

```
【ベースライン（基本シナリオ）】
　現在純利益：約65.8億円（EPS¥47）
　現在売上高推計：約350〜380億円（営業利益率18〜20%から逆算）
　ここではFCF ≈ 純利益 × 1.15（設備投資軽微のキャッシュリッチ体質）
　ベースFCF ≈ 65.8 × 1.15 ≒ 75.7億円

【5年FCF成長率（基本シナリオ）】
　FY2026: +8%（上方修正効果×会員数回復）
　FY2027: +7%（国内FC成熟化×海外貢献始まり）

　FY2029: +5%（安定成長フェーズ）
　FY2030: +5%

【ターミナル成長率】

【WACC = 5.75%（上記計算値）】
```

### 4-2. DCF計算（基本シナリオ）

```
【FCFプロジェクション（単位：億円）】

　Year 0（ベース）FCF: 75.7億円

　Year 1: 75.7 × 1.08 = 81.8億円　→ PV = 81.8 ÷ 1.0575 = 77.4億円
　Year 2: 81.8 × 1.07 = 87.5億円　→ PV = 87.5 ÷ 1.0575² = 78.3億円
　Year 3: 87.5 × 1.06 = 92.8億円　→ PV = 92.8 ÷ 1.0575³ = 78.7億円
　Year 4: 92.8 × 1.05 = 97.4億円　→ PV = 97.4 ÷ 1.0575⁴ = 78.2億円
　Year 5: 97.4 × 1.05 = 102.3億円　→ PV = 102.3 ÷ 1.0575⁵ = 77.8億円

　【5年間FCF合計PV】= 77.4 + 78.3 + 78.7 + 78.2 + 77.8 = 390.4億円

　【ターミナルバリュー】
　TV = FCF₅ × (1+g) / (WACC - g)
　   = 102.3 × 1.02 / (0.0575 - 0.020)
　   = 104.3 ÷ 0.0375
　   = 2,781.3億円

　ターミナルバリューPV = 2,781.3 ÷ 1.0575⁵ = 2,115.2億円

　【企業価値（EV）合計】
　= 390.4 + 2,115.2 = 2,505.6億円

　【純有利子負債（推定）】= ▲50億円（実質ネットキャッシュポジション）
　【株主価値】= 2,505.6 + 50 = 2,555.6億円

　【発行済株式数（推計）】= 約1.40億株

　【理論株価（基本シナリオ）】
　= 2,555.6億円 ÷ 1.40億株 = ¥1,825/株

　⚠️ ここで重要な調整：
　DCF理論値 ¥1,825は"成長率・FCF前提がフル実現"した場合の理論最大値
　実務上は以下のMoS（安全域）を設ける：

　【Margin of Safety 調整】
　　保守的バッファ：30〜48%割引
　　合理的フェアバリュー（中央値）：¥1,825 × 0.55 = ¥1,004
　　保守的フェアバリュー：¥1,825 × 0.52 = ¥949

　→ 実用的FVレンジ：¥940〜¥1,020
　→ 現在値¥765に対して +23%〜+33%の上値余地
```
