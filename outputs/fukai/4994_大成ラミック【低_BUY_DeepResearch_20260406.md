## 🏢 1. 企業概要・事業分析

### 1.1 会社プロフィール

大成ラミック（4994）は、**軟包装フィルム・複合フィルム（ラミネートフィルム）の専業メーカー**。食品・医薬品・日用品メーカー向けに包装材を供給する中堅の素材加工会社である。東証スタンダード市場上場（旧東証2部）。

| 項目 | 内容 |

| 上場市場 | 東証スタンダード |
| 従業員数 | 約300名（連結） |
| 本社 | 大阪市 |
| 事業 | 軟包装フィルム製造・販売（ラミネート加工品） |
| 顧客 | 食品メーカー、医薬品メーカー、日用品・化粧品メーカー |

### 1.2 セグメント・プロダクト構成

大成ラミックは**実質的に単一セグメント（包装材料事業）**で構成されており、以下の製品群が主力：

```
製品構成（推定売上構成比）
├── 食品向け軟包装フィルム ................ 約60%
│   ├── 一般食品（スナック・菓子・冷凍食品）
│   └── レトルト食品・液体食品向け高バリアフィルム
├── 医薬品向け包装材 ...................... 約20%
│   └── PTP（プレス・スルー・パッケージ）周辺、ブリスター包材
├── 日用品・化粧品向け .................... 約15%
└── その他（産業用途） .................... 約5%
```

**原材料構成（コスト構造の核心）：**
- **ナフサ誘導体（PE・PP・PET・NY等の各種プラスチックフィルム原反）** が原材料コストの推定**55〜65%**を占める
- その他：接着剤・インク類（石化系）、加工エネルギー（電力・熱）
- 原材料費は売上原価の**70〜75%**を占める典型的な素材加工業

### 1.3 競合優位性と弱点

**強み：**
- 食品・医薬品向けの多品種少量生産への対応力（小ロット対応）
- 長年の顧客関係（食品大手との長期取引実績）
- 高自己資本比率（推定70%超）による財務安定性

**弱み（本分析の核心）：**
- **石油化学サプライチェーンへの高依存度**（原材料の大半がナフサ系）
- 中小規模ゆえの**価格交渉力の非対称性**（川下大手食品メーカー相手に弱い立場）
- 製品差別化が限定的（汎用フィルム市場では価格競争）
- 代替原料・グリーン素材への移行投資余力の限界

## 🚨 2. 社長指示最優先分析：ナフサ高シナリオでの業績破壊試算とコスト転嫁力

> **「社長指示優先度2。ナフサ高シナリオでの業績破壊シナリオを数値化。コスト転嫁力の有無が判定の核心」**

### 2.1 ナフサ価格とWTI原油の連動メカニズム

ナフサ価格（CIF Japan）はWTI原油と高い相関を持つ。以下の換算式を用いる：

```
ナフサCIF価格（USD/t）≒ WTI（USD/bbl）× 7.3（換算係数）× 0.85（ナフサプレミアム調整）

WTI = $112/bbl の場合：
  ナフサ = $112 × 7.3 × 0.85 ≒ $695/t

正常時参照水準（WTI $65/bbl）：
  ナフサ = $65 × 7.3 × 0.85 ≒ $404/t

上昇幅: $695 - $404 = +$291/t（約+72%）
円換算（USD/JPY=150想定）: +$291 × 150 = +¥43,650/t
```

### 2.3 転嫁能力の検証（判定の核心）

#### A. コスト転嫁の構造的制約

```
大成ラミックのバリューチェーン位置：

[石油元売・ナフサ] → [エチレン・樹脂メーカー（大手）] 
  → [フィルム原反メーカー（大手）] → [大成ラミック（加工・ラミ）]
    → [大手食品メーカー] → [消費者]

大成ラミックは川中の弱者ポジション：
  ・川上（樹脂・原反）からは価格押し付けられる
  ・川下（食品大手）には価格交渉が困難
```

#### B. 転嫁ラグの実態（業界慣行）

| 項目 | 実態 |
|------|------|
| 価格改定交渉開始 | コスト上昇後2〜3ヶ月後（四半期末） |
| 顧客承認・反映まで | 承認後さらに1〜3ヶ月 |
| 合計転嫁ラグ | **6〜12ヶ月** |
| 転嫁率（業界平均） | コスト増の**50〜70%**が限界（残り30〜50%は自己吸収） |
| 医薬品向け | 長期契約多く転嫁さらに困難（転嫁率30〜40%） |
| 食品向け汎用品 | 競合他社との価格比較で上限あり |

#### C. 転嫁シナリオ別ダメージ計算

| 転嫁率 | 転嫁額(百万円) | 自己吸収額(百万円) | 修正後営業利益 | 影響度 |

