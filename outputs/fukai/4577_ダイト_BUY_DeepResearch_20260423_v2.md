## 1. 会社概要・事業構造

### 何をしている会社か

医薬品製造のダイトは「原薬→製剤→販売」の一貫体制を持つ数少ない後発薬（ジェネリック）メーカー。

- **原薬内製率が高く**、原材料コスト変動への耐性あり
- 富山という医薬品産業集積地（日本製薬の都）に立地 → GMP人材・サプライチェーン優位
- 後発薬製造・受託製造（CMO）・健康食品の3事業
- 自社品約100規格 + 受託品約300規格 = **総計約400規格**（高畠IR回答確認済み）

### ビジネスモデルの核心

| セグメント | 内容 | 特徴 |
|----------|------|------|
| 後発医薬品 | 承認済み品目の製造・販売 | 薬価改定リスクあるが承認済み品目は安定需要 |
| 受託製造（CMO） | 他社製品の製造代行 | コンソーシアムで拡大方向（FY2027-2028から本格化） |
| 健康食品 | OEM・自社ブランド | 補完的収益源 |

### 2-1. 品目数の正確な読み方

**【重要修正】「品目」=「規格」（業界用語）であり、SKU（包装規格）ではない**

```
例（実在しない架空の製品）:
  アジルサルタンOD錠20mg「ダイト」          ← これが1規格（=1品目）
    |__ PTP100錠                             ← 包装規格①
    |__ バラ500錠                            ← 包装規格②
  アジルサルタンOD錠40mg「ダイト」          ← これが2規格目
    |__ PTP100錠
    |__ バラ500錠
  
→ 規格数 = 2、包装規格数（SKU）= 4
→ 498 = 498規格（≠製品数・≠SKU数）
```

| 区分 | 2025年10月 | 2026年3月 | 意味 |
|------|-----------|---------|------|
| 協議対象規格（累計） | 56 | **498** | 半年で8.9倍（日経報道値） |
| 協議継続中 | 0 | **377** | まだ方針未決定 |
| 中止代替（代替供給） | 23 | **57** | 方針決定済み① |
| 製造所統合 | 33 | **64** | 方針決定済み② |

> **方針決定済みは57+64=121規格（全体の24.3%）。残り75.7%（377規格）は今後の展開次第。**

## 4. 構造的成長ドライバー

### ドライバー①：3Q進捗91.7%による本決算上振れ

FY2026 3Q終了時点（2026年2月末）でOP目標の91.7%を達成。4Q（3〜5月）は通常の繁忙期。本決算（7月）での上振れは高確度。

### ドライバー②：コンソーシアム既存品統合（FY2027へのブリッジ）

最大¥3〜5億/年の生産効率改善効果。小さいが**毎年積み上がるボトムライン貢献**。FY2026から漸増。

### ドライバー③：コンソーシアム新規共同開発（FY2027-2028の本命）

**【IR回答原文（高畠氏）】：「新規品目の共同開発化の方が、大きな利益影響を与える」**

- マル特申請品のピーク上市: FY2027/05〜FY2028/05
- 9社×各社のパイプライン = 新規開発効率が大幅向上
- 開発費シェアリング → 単独開発比でコスト▲50%以上の可能性

### ドライバー④：薬事手続き短縮（厚労省）

コンソーシアム参加企業へのマル特制度（申請期間6ヶ月→1.5ヶ月短縮）が承認済み。これが上市ピークを2027-2028に設定する根拠。

### ドライバー⑤：原薬内製×富山立地の構造的優位

後発薬業界の品質問題（2021年以降続くGMP違反）が「製造能力のある企業」への選別を加速。ダイトは原薬内製率が高く、富山のGMP人材基盤が厚い。業界再編の受益者ポジション。
