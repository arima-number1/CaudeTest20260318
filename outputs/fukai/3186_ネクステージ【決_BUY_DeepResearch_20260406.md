## 🏢 1. 企業概要・事業分析

### 1-1. 企業プロファイル

### 2-2. 新車供給垂直化トレンドの持続性：定量化

**「垂直化」とは：** メーカー→ネクステージ直系ディーラー→販売という川上川下の垂直統合的な新車供給力強化を指す。ここでは①新車在庫正常化の影響方向性と②ネクステージ固有の垂直統合力を分析する。

#### 新車在庫正常化が中古車需要に与える影響

```
【新車供給正常化の中古車市場への波及図】

新車供給↑
    │
    ├─→（短期）新車販売増→ネクステージ新車セグメント直接寄与 ✅ プラス
    │
    ├─→（中期）下取り増加→中古車仕入れ増→在庫充実→販売増 ✅ プラス
    │
    ├─→（中期）中古車相場の緩やかな下落圧力 ⚠️ 軽微なマイナス
    │
    └─→（長期）新車普及→将来の中古車供給増→市場拡大 ✅ プラス（2〜3年後）
```

| 時間軸 | 影響方向 | 強度 | ネクステージ固有の受益度 |
|--------|---------|------|----------------------|
| 0〜6ヶ月 | **追い風** | 強 | 高（在庫充実・販売増） |
| 6〜12ヶ月 | **追い風** | 中〜強 | 中〜高（下取り活性化） |
| 12〜24ヶ月 | **追い風優勢**（但し相場下落注意） | 中 | 中 |
| 24ヶ月超 | ニュートラル〜軽微追い風 | 弱〜中 | 中 |

**定量化試算：**

```
新車在庫正常化による追加営業利益インパクト（ネクステージ）

シナリオ：新車販売台数が前年比+20%増（在庫正常化分）
  新車追加台数：約5,000台（推計）
  新車1台当たり粗利：¥110,000
  直接粗利寄与：+¥5.5億円

  下取り増加による中古車仕入れ増：+3,000台
  中古車1台当たり粗利：¥220,000
  下取り起因粗利寄与：+¥6.6億円

  F&I追加寄与（台数増分）：+¥2億円

合計追加粗利寄与：≒+¥14億円
（前年営業利益比で+8〜12%の増益要因）
```

**判定：新車供給垂直化は少なくとも今後18〜24ヶ月間は構造的追い風。一時的要因ではなく、ネクステージが戦略的に仕込んだポジショニングの成果。**
