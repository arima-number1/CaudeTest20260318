### ①事業実態の把握：売上構成・利益率・市場シェア

#### 売上構成の推計（FY2025実績ベース）

EPS=¥22、PER=12.2倍、株価=¥268、BPS=¥433から逆算する。

**時価総額推計:**
- 発行済株式数 = BPS/1株純資産 → まずBSから株式数を推計
- PBR = 株価/BPS = 268/433 = 0.619倍（提供値0.62倍と一致 ✓）
- EPS = ¥22、ROE = EPS/BPS = 22/433 = **5.08%**【注：提供ROE4.1%と若干乖離。提供値4.1%を優先使用】
  - ROE4.1%からEPS逆算: BPS×ROE = 433×4.1% = ¥17.75 ≒ ¥18【提供EPS¥22との差異は優先株・非支配持分の取り扱い等による可能性】
  - **提供値EPS=¥22、ROE=4.1%を共に所与として使用**
- 純利益 = EPS × 株式数
- 株式数推計: 時価総額 / 株価 = （BPS × 株式数）/ PBR / 株価
  - 純資産 = BPS × n株、株価 = 268、PBR = 0.62
  - 268 = 0.62 × 433 → ✓整合
  - 純資産総額 = 433 × n
  - ROE4.1% × 純資産 = 純利益
  - 純利益 = EPS × n = 22n
  - 22n / 433n = 5.08%... → ROE4.1%とは乖離
  - **この乖離は少数株主持分・優先株等の影響と解釈し、提供値をそのまま使用**

**【推計】売上規模・セグメント構成**

> **投資判断: 🟡 中立〜弱い買い（NEUTRAL to WEAK BUY）**

### 4-3. DCF計算（基本シナリオ）

**割引率 WACC = 5.80%**

| 年度 | FCF（百万円） | 割引係数 | PV（百万円） |
|------|------------|---------|------------|
| FY2026E | ¥260 | 1/(1.058)^1 = 0.9452 | ¥245.8 |
| FY2027E | ¥270 | 1/(1.058)^2 = 0.8934 | ¥241.2 |
| FY2028E | ¥278 | 1/(1.058)^3 = 0.8444 | ¥234.7 |
| FY2029E | ¥284 | 1/(1.058)^4 = 0.7981 | ¥226.7 |
| FY2030E | ¥290 | 1/(1.058)^5 = 0.7543 | ¥218.7 |
| **FCF合計PV** | | | **¥1,167.1** |

**ターミナルバリュー計算:**
- TV = FCF_2031 / (WACC - g) = ¥290 × (1+1.0%) / (5.80% - 1.0%)
- TV = ¥292.9 / 4.80% = **¥6,102百万円**
- TV の PV = ¥6,102 × 0.7543 = **¥4,602.6百万円**

**企業価値（EV）:**
- EV = FCF PV + TV PV = ¥1,167.1 + ¥4,602.6 = **¥5,769.7百万円**

**株式価値:**
- 株式価値 = EV + 現金・現預金 - 有利子負債
- = ¥5,769.7百万円 + ¥1,000百万円 - ¥600百万円 = **¥6,169.7百万円**【推計、ネットキャッシュ¥400百万円加算】

**DCF理論株価:**
- 株式数 ~1,000万株として
- **DCF理論株価 = ¥6,169.7百万円 / 1,000万株 = ¥617/株**

> ⚠️ この数値は推計FCFと推計株式数に大きく依存する。参考値として扱い、後述のマルチプル法との平均で最終判断を行う。
