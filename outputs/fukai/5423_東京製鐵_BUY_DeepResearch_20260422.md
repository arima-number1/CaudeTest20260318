## 4. DCFバリュエーション

### 4-1. 前提条件

| パラメータ | 設定値 | 根拠・出所 |
|-----------|-------|-----------|
| **Rf（リスクフリーレート）** | **2.405%** | 日本10年国債（財務省 R8.4.20）— 絶対厳守 |
| **ERP（株式リスクプレミアム）** | **5.5%** | 日本市場の標準値（Damodaran 2025ベース：新興+成熟ハイブリッド） |
| **β（ベータ）** | **1.15** | 鉄鋼セクター（景気敏感・コモディティ連動）。東証鉄鋼指数との相関から推計 |
| **Re（株主資本コスト）** | **2.405% + 1.15 × 5.5% = 8.73%** | CAPM |
| **負債比率（D/E）** | **0.15**（D/（D+E）≒13%） | 低負債体質の電炉メーカーとして保守設定 |
| **Rd（負債コスト）** | **1.5%** | 優良企業の社債スプレッド |
| **実効税率（Tc）** | **30%** | 日本法人税率 |
| **WACC** | **計算参照** | 下記算出 |
| **ターミナル成長率（g）** | **1.0%** | 国内GDP成長率（0.5–1.5%）の中間値 |

**WACC算出:**

```
E/(D+E) = 87%
D/(D+E) = 13%

WACC = [0.87 × 8.73%] + [0.13 × 1.5% × (1 - 0.30)]
     = 7.60% + 0.14%
     = 7.74%
```

### 4-2. FCF予測（5年間）

| 年度 | 売上高（億円） | 営業利益率 | NOPAT | D&A | Capex | ΔNWC | FCF（億円） |
|------|-------------|----------|-------|-----|-------|------|------------|
| FY2026E | 3,900 | 5.5% | 149 | 80 | ▲120 | ▲20 | **89** |
| FY2027E | 4,050 | 6.0% | 170 | 85 | ▲125 | ▲15 | **115** |
| FY2028E | 4,100 | 6.5% | 187 | 88 | ▲130 | ▲10 | **135** |
| FY2029E | 4,150 | 6.5% | 189 | 90 | ▲130 | ▲10 | **139** |
| FY2030E | 4,200 | 6.8% | 200 | 92 | ▲135 | ▲10 | **147** |

> ※ 売上成長は値上げ定着+インフラ需要を前提。スクラップ安継続を2026–2027年に限定して保守的に設定。

**ターミナルバリュー（TV）:**

```
TV = FCF₂₀₃₀ × (1+g) / (WACC - g)
   = 147 × 1.01 / (7.74% - 1.0%)
   = 148.5 / 6.74%
   = ¥2,203億
```

**企業価値（EV）の算出:**

```
PV(FCF) = 89/(1.0774)¹ + 115/(1.0774)² + 135/(1.0774)³ 
        + 139/(1.0774)⁴ + 147/(1.0774)⁵
        = 82.6 + 99.1 + 108.2 + 103.4 + 101.8
        = 495億円

PV(TV) = 2,203 / (1.0774)⁵ = 1,520億円

EV = 495 + 1,520 = 2,015億円
```

**理論株価の算出:**

```
純有利子負債（推計）: ▲500億円（実質ネットキャッシュ or 低負債）
株主価値 = EV + 500（ネットキャッシュ加算）= 2,515億円

発行済株数（推計）: 約1.06億株（BPS¥2,133 × 株数 = 純資産規模から逆算）
※ 純資産 = ¥2,133 × 1.06億 ≈ ¥2,261億

理論株価 = 2,515億 ÷ 1.06億株 = ¥2,372
（四捨五入で ¥2,380 と設定）
```

> ⚠️ 発行済株数は公開情報（直近有価証券報告書）での確認を推奨。本試算はBPSから逆算した推計値。
