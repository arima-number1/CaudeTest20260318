## 「薬と毒」— 二律背反の定性・定量分析

投資判断の純度を高めるため、現状の外部環境がカチタスに与える影響を二律背反の視点で整理する。結論：薬の効果が毒の副作用を3〜5倍上回り、STRONG BUY判断を補強する。

### 薬（強み・加速要因）

| 薬 | 定性 | 定量インパクト |
|---|---|---|
| **ホルムズ危機による競合駆逐** | 中小リフォーム業者が新規受注停止。フィールド2点確認済み | 仕入件数：H1平均2,333件/Q → 3Q 2,725件（+16.8%加速）。将来利益の在庫積み上げ+30.8% YoY |

| **新築マイナス継続による需要シフト** | 新築着工35ヶ月連続YoYマイナス。ホルムズ危機で新築の物理的着工不能が重なる | 新築-中古価格差：¥1,754万(2020年) → ¥1,866万(2025年) → 拡大継続。潜在市場12.7万件に対し浸透率5.8%のまま需要だけが急増 |

```

### 3Q単体での利益率向上 — 要因分解

```
営業利益率 2Q推定13.6% → 3Q確定14.5%（+0.9pt）の要因：

【主因①】固定費レバレッジ（推定寄与+0.5pt）
  販売件数+9.3%に対し、本部・店舗固定費はほぼ横ばい
  → 1件当たり固定費が圧縮 → 営業利益率が自動的に改善

【主因②】仕入価格の安定（推定寄与+0.3pt）
  地方空き家の仕入価格は市況より構造的に安定
  競合撤退で「売り急ぎ物件」の入手が増加 → 仕入コスト有利化

【主因③】広告宣伝費の効率化（推定寄与+0.1pt）
  デジタル集客の成熟でCPA（顧客獲得単価）が低下傾向
  競合不在で問合せが自然流入 → 広告費をかけなくても仕入れが増える
```

```

    3Q累計¥14,248M + 4Q推定¥4,715〜5,160M
    = **¥18,963〜19,408M（ガイダンス¥17,800Mを+6〜9%上振れ）**

FY27/3期への勾配：
  3Q単体 OP margin 14.5%が次期に維持されると仮定した場合
    FY27 売上予想（中計ペース+12%）: ¥165,000M
    FY27 OP = ¥165,000M × 14.5% = ¥23,925M

  OP¥20,000M をFY27で前倒し達成するシナリオが現実味を持つ。
