### 【核心問い①】売上高▲10.1%にもかかわらず営業利益がコンセンサスを大幅上回った理由

**考察：3つの仮説と検証**

#### 仮説A：コスト削減（最有力）
4Q（12〜2月）はヒートビジネス（鍋物・おでん等）の本格シーズンであり、**商品ミックスが高粗利品にシフト**する。加えて以下が重複した可能性が高い：

- **販管費の年度末圧縮**: 広告宣伝費・物流コストの通期予算消化が3Q末に前倒しされ、4Qの実負担が軽減
- **イオングループ仕入れコスト改善**: FY2026/02通期でグループ調達比率引き上げが効いた可能性
- **人件費抑制**: パート時給上昇圧力があるものの、4Q（年度末）は一時金支給が少ない時期

#### 仮説B：売上高▲10.1%の本質はリストラクチャリング（店舗閉鎖・統廃合）
前年比売上高▲10.1%という数字は、単純な既存店不振ではなく**不採算店舗の閉鎖・縮小**による構造改革の成果とも解釈できる。閉鎖店舗の固定費が消滅することで、売上が落ちても営業利益が改善するロジック。

**この場合、「コスト削減型の質的改善」であり、正の評価が可能。**

#### 仮説C：特別要因（一過性）
在庫評価益・リース契約修正・会計処理タイミングの変更等の可能性は完全に排除できないが、監査済み決算であれば重大な歪みは通常限定的。

**深井判定：仮説Aがメイン（70%）＋仮説Bが補助（25%）＋仮説C（5%）**

### 【核心問い②】純利益赤転の原因分析

**通期純利益 ▲¥3,185M（営業利益 ¥5,050M から大幅乖離）**

```
営業利益        : +¥5,050M
± 営業外損益    : ▲¥139M（経常利益 ¥4,911M）
経常利益        : +¥4,911M
− 特別損失等    : 推定 ▲¥8,000〜9,000M
+ 特別利益      : 推定 ¥100〜500M
= 税引前利益    : 推定 ▲¥3,000〜4,000M
− 法人税等      : ▲¥200M（最低限の税負担）
= 当期純利益    : ▲¥3,185M
```

**特別損失の構成（推定）**

| 項目 | 推定金額 | 性格 |
|------|---------|------|
| **減損損失（店舗・設備）** | ▲¥4,000〜5,000M | 主因・一部一過性 |
| **のれん償却加速** | ▲¥1,500〜2,000M | 継続的（あと数年） |
| **店舗閉鎖・撤退費用** | ▲¥1,000〜2,000M | 一過性（構造改革） |
| **退職給付・人件費精算** | ▲¥500〜1,000M | 一過性 |
| **合計（推定）** | **▲¥7,000〜10,000M** | |

> **深井の核心判断**: 特別損失の主体は**減損損失と構造改革費用**であり、その大半は**一過性（FY2026/02に集中処理）**と考えられる。この「Big Bath（お風呂掃除型）」会計処理は、来期以降ののれん償却・減損負担を軽減し、利益を「クリーンアップ」する目的に整合する。
>
> ただし、**のれん償却の継続分（年間¥1,000〜1,500M規模）は来期以降も続く**ため、経常利益→純利益の変換ロスは残存する。来期純利益計画¥150Mが非常に低いのはこの認識を会社自身が持っている証左。

**一過性 vs 継続リスクの判定**

| 項目 | 一過性 | 継続 |
|------|--------|------|
| 減損損失（集中処理分） | ✅ 主に一過性 | |
| 店舗閉鎖費用 | ✅ 一過性 | |
| のれん償却 | | ⚠️ 継続（年¥1,000〜1,500M） |
| 退職給付 | ✅ 一過性 | |

## ⑦ DCF FairValue算出（3シナリオ）

### 7-1. DCF前提条件の設定

**共通パラメータ**

| パラメータ | 値 | 根拠 |
|-----------|-----|------|

**WACC計算**
```
WACC = 60% × 6.329% + 40% × 1.5% × (1 - 30%)
     = 3.797% + 0.420%

```

**DCFモデルの前提**

- 投資予測期間: FY2027/02〜FY2031/02（5年間）
- 設備投資: 売上比約1.2〜1.5%（年¥13,000〜17,000M）
- D&A: 売上比約1.5〜1.8%（年¥17,000〜20,000M）
- 運転資本変化: ±ゼロ〜若干マイナス（食品小売は負の運転資本構造）
