## 4. DCFバリュエーション

### 4-1. 前提条件

#### CAPM・WACC 計算

| パラメータ | 数値 | 根拠・出所 |
|-----------|------|---------|
| **Rf（リスクフリーレート）** | **2.439%** | JGB10Y 財務省日次 R8.4.10（絶対遵守） |
| **β（ベータ）** | **0.85** | 東証不動産中型株平均。センチュリー21(0.75)〜ヒューリック(0.90)の中間 |
| **ERP（株式リスクプレミアム）** | **5.5%** | 日本市場標準（Damodaran 2025年推計） |
| **Re（株主資本コスト）** | **2.439% + 0.85 × 5.5% = 7.114%** | CAPM計算 |
| **Rd（負債コスト）** | **1.8%** | 不動産中堅企業の借入金利推計 |
| **Tc（法人税率）** | **30%** | 日本標準 |
| **D/（D+E）** | **55%** | 不動産開発会社の典型的資本構成 |
| **E/（D+E）** | **45%** | 同上 |

#### ターミナルバリュー計算

```
TV = FCF(FY2031) / (WACC - g)
   = FCF(FY2030) × (1+g) / (WACC - g)
   = 4,513 × 1.02 / (4.6% - 2.0%)
   = 4,603 / 2.6%
   = ¥177,038 百万円（1,770億円）

TV の現在価値（5年割引）= 177,038 / (1.046)^5
                      = 177,038 / 1.2517
                      = ¥141,440 百万円（1,414億円）
```

#### 企業価値・理論株価

```
事業価値（PV合計）= ¥16,938百万円（5年間FCF合計PV）
ターミナルPV      = ¥141,440百万円
────────────────────────────────

理論株価 = ¥744億円 / 1,700万株
```
