## 1. 企業概要・事業分析

### 1-1. 会社概要

エア・ウォーター株式会社（Air Water Inc.）は、大阪府大阪市に本社を置く**産業ガス・医療・農業・機能材料**を中核とする多角化コングロマリットである。旧称・大同酸素を起源とし、複数の産業ガスメーカーを統合・再編することで現在の規模に至った。東証プライム市場上場。時価総額は概ね**4,000〜5,000億円規模**（株価水準により変動）。

### 1-2. セグメント構成（FY2025実績推定ベース）

| セグメント | 売上高シェア（推定） | 営業利益シェア（推定） | 特徴 |
|-----------|-------------------|--------------------|------|
| **産業ガス・環境** | 約35% | 約30% | 空気分離装置・高圧ガス供給。安定収益基盤 |
| **医療** | 約20% | 約25% | 在宅酸素療法（HOT）・医療ガス。高マージン |
| **農業・食品** | 約18% | 約15% | 肥料・農薬・食品向けガス。景気耐性高い |
| **機能材料・化学** | 約15% | 約18% | 特殊化学品・半導体向け材料。成長ドライバー |
| **エネルギー・その他** | 約12% | 約12% | LPG・水素エネルギー・海外事業 |

> ※ ARR（年間経常収益）・NRR（売上維持率）はSaaS企業向け指標であり、エア・ウォーターのような製造業には直接適用されない。代替指標として**長期供給契約比率（推定60〜70%）**・**医療ガス事業の継続率（推定95%超）**を収益安定性の代替指標として活用する。

### 1-3. 競合優位性

**✅ ポジティブ要因（トリガー前評価）**

1. **地域密着型パイプラインネットワーク**: 産業ガスは導管インフラへの初期投資が参入障壁となり、既存顧客は容易に乗り換えられない。スイッチングコストが極めて高い。
2. **医療ガス・在宅酸素の規制保護市場**: 薬機法・保険点数制度に守られた安定収益。競合参入困難。
3. **農業化学品との複合提案力**: 産業ガス×肥料×農薬の一括提案は競合にない差別化。
4. **多角化による景気耐性**: セグメント分散により単一事業リスクを低減。

**❌ 今回トリガーによる毀損要因**

1. **ガバナンスプレミアム剥落**: 機関投資家のESG基準厳格化により、不正発覚企業は組み入れ除外・ウェイト低下の対象となる。
2. **管理体制への疑念**: 特別調査委（社外第三者）による報告書公表は、内部統制の機能不全を対外的に確認させるものである。
3. **関係者処分による経営陣の不安定化**: 処分対象が経営中枢に及ぶ場合、戦略の継続性・M&A実行力が低下するリスクがある。

## 2. 市場ポジションと競合分析

### 2-1. 産業ガス業界の構造

世界の産業ガス市場は**リンデ（独/米）・エア・プロダクツ（米）・エア・リキード（仏）**の3強が世界シェアの約70%を占める寡占構造である。日本国内では以下の競合が存在する。

|---------|------|----------------------|

| **岩谷産業（8088）** | LPG・水素・産業ガスの三位一体 | 水素インフラで先行 |
| **住友精化（4008）** | 高純度ガス特化 | ニッチ特化型 |
| **昭和電工ガスプロダクツ** | レゾナック傘下・半導体特殊ガス | 半導体向けで競合 |

### 2-2. エア・ウォーターの相対ポジション

- **国内産業ガス市場シェア**: 大陽日酸に次ぐ**業界第2位**（推定）
- **医療ガス分野**: 国内トップクラス。在宅酸素療法の普及率上昇が長期追い風
- **農業化学品**: 国内独自ポジション（競合少ない）

### 2-3. 今回不正発覚による競合への流出リスク

産業ガスはパイプライン供給のためスイッチングコストが高く、**短期の顧客流出リスクは限定的**。ただし、新規大型案件の受注競争では信頼性を問われる可能性があり、**中長期の新規顧客獲得に支障が出るリスク**は中程度存在する。医療ガス分野では、病院・診療所からの契約更新に影響が出るリスクも排除できない。

## 4. DCFバリュエーション

### 4-1. 前提条件（すべて明示）

#### 割引率（WACC）の算出

| パラメータ | 値 | 根拠・出所 |
|-----------|-----|----------|
| **リスクフリーレート（Rf）** | **1.5%** | JGB10年物金利（現在約1.5%）— 指定値 |
| **エクイティリスクプレミアム（ERP）** | **6.0%** | 日本株の長期ERP（Damodaran等参考、新興国より低く欧米より高め） |
| **β（レバードベータ）** | **0.85** | 産業ガス・多角化コングロマリットの典型値。景気耐性高くβ低め。不正発覚後は上昇傾向にあり0.90〜1.00に修正リスク |
| **株主資本コスト（Ke）** | **6.6%** | = 1.5% + 0.85 × 6.0% |
| **負債コスト税引後（Kd）** | **1.2%** | 長期借入金利約1.8% × (1-実効税率30%) |
| **株式時価総額（E）** | 約4,200億円 | 株価2,200円 × 発行株数約1.9億株 |
| **有利子負債（D）** | 約4,500億円 | BS推定値 |
| **E/(D+E)** | 約48% | |
| **D/(D+E)** | 約52% | |
| **WACC** | **3.80%** | = 6.6% × 48% + 1.2% × 52% |

> ⚠️ **ガバナンスリスク調整後WACC**: 不正発覚に伴うガバナンスプレミアム（+0.5〜1.0%）を加味した調整WACCは**4.30〜4.80%**。本レポートの弱気ケースではこちらを使用する。

#### FCF成長率の前提

| フェーズ | 期間 | 成長率 | 根拠 |
|---------|------|--------|------|
| フェーズ1（高成長期） | FY2026〜FY2030（5年） | **+3.0%** | 水素・医療拡大・M&A。ただし不正後の保守的オペレーションで上限を設定 |
| フェーズ2（安定成長期） | FY2031〜FY2035（5年） | **+2.0%** | 成熟化・競合激化を考慮 |
| **ターミナル成長率（g）** | FY2036〜永続 | **1.0%** | 日本の長期名目GDP成長率（低成長シナリオ）。Rf(1.5%)より低く設定 |

### 4-2. DCF計算（基本ケース）

**ベースFCF（FY2025推定）**: 200億円

| 年度 | FCF（億円） | 割引係数（WACC=3.80%） | PV（億円） |
|------|-----------|---------------------|----------|
| FY2026 | 206 | 0.9634 | 198.5 |
| FY2027 | 212 | 0.9281 | 196.8 |
| FY2028 | 219 | 0.8941 | 195.7 |
| FY2029 | 225 | 0.8613 | 193.8 |
| FY2030 | 232 | 0.8297 | 192.5 |
| FY2031 | 236 | 0.7993 | 188.6 |
| FY2032 | 241 | 0.7700 | 185.6 |
| FY2033 | 246 | 0.7418 | 182.5 |
| FY2034 | 251 | 0.7147 | 179.4 |
| FY2035 | 256 | 0.6886 | 176.3 |
| **PV合計（10年）** | | | **1,889億円** |

**ターミナルバリュー（TV）計算:**

```
TV = FCF2035 × (1+g) / (WACC - g)
   = 256 × 1.01 / (0.038 - 0.010)
   = 258.6 / 0.028
   = 9,236億円

TV現在価値 = 9,236 × 0.6886 = 6,361億円
```

**企業価値（EV）計算:**

```
EV = PV合計（10年） + TV現在価値
   = 1,889 + 6,361
   = 8,250億円

株式価値 = EV - 純有利子負債（約3,800億円）
         = 8,250 - 3,800
         = 4,450億円

理論株価 = 4,450億円 ÷ 発行株数1.9億株
         = 約2,342円
```

> **基本ケース理論株価（ガバナンスディスカウント前）: 約2,342円**

**ガバナンスディスカウント適用（▲25%）:**

不正発覚・報告書公表・関係者処分という3点同時開示は、機関投資家からの組み入れ除外圧力・信頼性プレミアムの剥落を招く。過去の日本企業不正事例（東芝・三菱自動車・日産等）を参考に、イベント直後の株価ディスカウント率として**▲25〜35%**を適用する。

```
理論株価（基本ケース・ディスカウント後）
= 2,342円 × (1 - 0.25) = 約1,757円

レンジ: 1,550〜1,750円
```

## 6. メインシナリオ vs リスクシナリオ

### 6-1. 強気ケース（確率: 15%）

**前提:**
- 不正の範囲が限定的（単一部門・個人の横領レベル）と判明
- 経営陣の主要メンバーは継続し、戦略的M&Aも続行
- 特別損失は30億円以内に収まる
- 機関投資家の一部が「割安」と判断し下値で買い支え

**シナリオ展開:**
- 月曜は▲5〜10%の初期下落後、週内に下げ止まり
- 4〜6週間でリバウンド局面入り
- 理論株価：**1,950〜2,200円**

**強気ケースでのトレード戦略:**
- 月曜の寄り成り売りは行わず、売り圧力が落ち着いた後の押し目を探す
- ただし確率15%であり、メインシナリオとしては採用しない

### 6-2. 基本ケース（確率: 55%）

**前提:**
- 不正は複数部門・複数年にまたがるが、グループ全体の構造的問題ではない
- CFO または 担当役員クラスの処分（代表取締役は継続）
- 特別損失 40〜80億円 → FY2026利益予想を10〜15%下方修正
- MSCIやTOPIX組み入れは維持されるが、機関投資家のウェイトが低下
- ガバナンス格付けの引き下げ（ISS・グラスルイス等の推奨変更）

**シナリオ展開:**
- 月曜寄り付き：▲8〜15%のギャップダウン
- 1〜2週間は売り圧力継続（▲15〜25%）
- 3〜6ヶ月かけて底値形成・再評価の機会
- 理論株価：**1,550〜1,750円**

### 6-3. 弱気ケース（確率: 30%）

**前提:**
- 不正がグループ全体に広がる構造的な問題（組織的関与）であることが判明
- 代表取締役CEO の引責辞任 → 経営の空白期間が発生
- 金融機関からの追加与信調査 → 借入コスト上昇
- 特別損失 100億円超 → 当期赤字転落の可能性
- MSCI・TOPIXのリバランス（定期見直し）での除外・ウェイト削減
- 顧客（病院・大手製造業）からの取引見直し・契約更改時の切り替え検討開始

**シナリオ展開:**
- 月曜寄り付き：▲15〜25%のギャップダウン
- 1〜2ヶ月で底値を模索（▲35〜50%）
