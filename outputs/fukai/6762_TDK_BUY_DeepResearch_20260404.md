## 2. 投資テーゼ：3つの追い風

### 追い風①：SSD価格5割高 → HDD需要回帰 → HAMR磁気ヘッドASP急騰

NAND契約価格が+55〜60% QoQ（Q1 2026）急騰したことで、SSD製品価格は5割以上上昇。クラウド事業者・エンタープライズ顧客の一部が大容量ストレージをSSDからHDDへ切り替える需要回帰が発生している。

TDKはHDD磁気ヘット世界最大手として直接受益する：

- **HAMR（熱アシスト磁気記録）ヘッド**：従来比3〜4倍の記録密度を実現する次世代技術。1枚あたりASPは従来ヘッドの1.5〜2倍。Western Digital・Seagateが増産対応のためTDKへ追加発注を行う構造。
- Q2 2026以降もNAND高止まりが続くとすれば（TrendForce予測：Q2 +20〜30%継続）、HDD回帰トレンドは少なくとも2〜3四半期持続。

### 追い風②：MS日本10兆円投資 → AIサーバー向けMLCC利益率改善

Microsoft日本国内DC・AI投資10兆円発表により、国内DCの新設・増設ラッシュが確定。TDKのMLCC（積層セラミックコンデンサ）・インダクタは全てのサーバーに搭載される電源回路・ノイズ対策の必需部品。

- AIサーバー1台あたりのMLCC搭載数は通常サーバーの**3〜5倍**（GPU電源管理・メモリ安定化に大量使用）
- DC建設増加でTDKのMLCC稼働率が急上昇 → 固定費吸収でOPM改善
- **利益率改善シナリオ**：MLCC事業OPM 12% → 16〜18%（稼働率90%超で達成可能）
- この改善だけでTDK全社のEPSを+15〜20%押し上げる効果を持つ

### 追い風③：原油高 → EV化加速 → EV電池材料・BMS部品需要増

原油$111/bblの高止まりでEV需要が構造的に加速。TDKはEV向けリチウムイオン電池材料・BMS（バッテリーマネジメントシステム）部品で恩恵を受ける。

- ただし、この分野ではパナソニック・CATLとの競争激化でASP圧力が存在
- **中期テーマ**として位置づけ。FY2027以降の収益寄与を期待

## 3. テクニカル分析：25日線乖離▲6.1%からのリバウンド期待値

```
52週高値（2026/02/26）: ¥2,498
現在株価（2026/04/03）: ¥2,011
高値からの下落率:       ▲19.5%

25日移動平均（推定）:  ¥2,142（高値水準から25日間の平均を概算）
25日線乖離:           (¥2,011 - ¥2,142) / ¥2,142 = ▲6.1%

リバウンド期待値（25日線回復）: ¥2,011 × 1.061 = ¥2,134
その先の目標（base FV）:         ¥2,380（+18%）
強気目標（bull FV）:             ¥2,800（+39%）
```

25日線乖離▲6%超は、多くの機関投資家の「ルールベース買い戻し」が発動するゾーン。NAND+60%という実需ファクトがこれに重なることで、テクニカルとファンダメンタルズが同方向に作用する局面にある。

## 7. 不労（Flow）× 豆田（Risk）コメント
