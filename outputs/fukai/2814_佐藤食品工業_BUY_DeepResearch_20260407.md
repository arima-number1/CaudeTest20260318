## ④ ROE低迷の原因と改善余地

### ROE3.4%のデュポン分解

```
【ROE デュポン分解（推計）】
────────────────────────────────────────────────────────
 ROE = 純利益率 × 総資産回転率 × 財務レバレッジ
 
 3.4% ≈ 2.4%      ×    0.65回    ×    1.2〜1.3倍
          ↑              ↑               ↑
        低マージン    資産効率低い    低レバレッジ
       （コスト高）  （遊休資産多い）（無借金経営）
────────────────────────────────────────────────────────
```

### ROE構造的低迷の原因詳細

| 原因 | 詳細 | 改善難易度 |
|------|------|----------|
| **過剰資本（分母肥大化）** | BPS¥11,948に対し株価¥3,235。大量の現預金・投資有価証券が資本効率を押し下げ | 中（意思決定次第） |
| **低営業利益率** | 米価高騰・人件費上昇・エネルギーコスト上昇が粗利を侵食 | 難（構造的） |
| **販管費の硬直性** | 全国配送網維持コスト・品質管理コストが高止まり | 中 |
| **設備過剰投資** | 新潟工場・関東工場の稼働率低迷の可能性 | 難 |
| **投資有価証券保有** | 事業に無関係な株式・債券を大量保有 | 易（売却可） |

### ROE8%達成に必要な施策

```
【ROE8%達成シナリオ試算】
────────────────────────────────────────────────────────
 【現状】
   純利益 約¥7.1億 ÷ 自己資本 約¥210億 ≈ ROE 3.4%
   （自己資本: BPS¥11,948 × 株数365万株 ≈ ¥436億）
   ※ 実際の自己資本はyfinance BPS×株数から逆算
   
 【ROE8%達成に必要な純利益】
   自己資本¥436億 × 8% ≈ ¥34.9億
   → 現状純利益¥7.1億の約4.9倍が必要
   
 【実現可能な施策】
 施策①: 自社株買い・配当増額（分母圧縮）
   → 自己資本を¥90億まで圧縮すれば現状利益でROE8%
   → 必要自社株買い額: 約¥346億（現実的でない）
   
 施策②: 収益改善（分子拡大）
   → 純利益率5%達成: 売上¥300億→純利益¥15億（ROE 3.4%→3.4%）

 ⭐ 現実的な着地点: ROE5〜6%（自社株買い+配当増+微増益）
```

# 4. DCFバリュエーション

## 4.1 前提条件の明記

### マクロ・市場前提

| パラメータ | 値 | 根拠 |
|-----------|-----|------|
| リスクフリーレート (Rf) | **2.429%** | JGB10Y 2026-04-07（財務省日次 R8.4.6） |
| エクイティリスクプレミアム (ERP) | **5.5%** | 日本市場 Damodaran推計（2025〜26年） |
| β（ベータ） | **0.55** | 食品セクター低β・小型株調整（推計） |
| 株主資本コスト (Re) | **2.429% + 0.55 × 5.5% = 5.45%** | CAPM |
| 負債コスト税引後 (Rd×(1-Tc)) | **1.0%** | 実質無借金・超低金利借入想定 |
| 資本構成 (E/D) | **E:95% / D:5%** | 高自己資本比率 |
| WACC | **5.45% × 0.95 + 1.0% × 0.05 ≈ 5.23%** | 加重平均資本コスト |
| 実効税率 (Tc) | **30%** | 日本法人税 |
| 永久成長率 (g) | 強気:2.5% / 基本:0.5% / 弱気:▲1.0% | シナリオ別 |

### WACC計算詳細

```
【WACC計算】
────────────────────────────────────────────────────────
 Re = Rf + β × ERP
    = 2.429% + 0.55 × 5.5%
    = 2.429% + 3.025%
    = 5.454%

 WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
      = [0.95 × 5.454%] + [0.05 × 1.0%]
      = 5.181% + 0.050%
      = 5.23%
────────────────────────────────────────────────────────
```
