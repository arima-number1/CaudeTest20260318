## 【配当性向97%問題の解剖】

```
配当¥76 vs EPS(岡三予想)¥78 → 配当性向97%

リスクシナリオ:
EPS実績が¥70（岡三予想▲10%）→配当性向109% → 減配検討必要
EPS実績が¥65（岡三予想▲17%）→配当性向117% → 減配確率高い
EPS¥78達成 → 配当維持、株価安定的

なぜ超高配当性向を維持するか:
・中計では「現在配当水準¥76維持+EPS成長で増配見据える」と明言
・25/12期実績ROE10.4%、自己資本¥859億と財務余裕あり
・株主還元270億円+（中計期間）のコミットメント

下落シナリオの具体的トリガー:
1. 中国事業でローカル競合の価格競争が再激化 → OP下振れ
2. 米国でマーケティング投資の費用先行で短期EPS圧迫
3. ホルムズ長期化でシンガポール事業大幅減益
```

**→ EPS¥78達成が「配当維持」の最低限ライン。岡三予想達成を基本シナリオとして投資判断。**
