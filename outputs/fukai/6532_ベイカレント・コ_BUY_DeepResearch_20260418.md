```

  28%成長想定: 22.5 ÷ 28 = **0.80倍** ← 著しい割安

岡三TP¥7,000 PEG:
  23倍 ÷ 23% = **1.00倍** ← フェアバリュー
```

```
現在値¥5,618比: +¥596（+10.6%）のアップサイド（保守的上限）
```

**深井結論**：PEGレシオ0.80-0.98倍はいずれも「割安」判定。ROEベースの適正PBR（8.06倍）でも最低+10%のアップサイド。**PER23倍は成長率を考慮すれば「チープ」な水準**。

## 4. DCFバリュエーション

### 4-1. WACC計算

**パラメーター設定**

| パラメーター | 数値 | 根拠・出典 |
|------------|------|----------|
| **Rf（リスクフリーレート）** | **2.409%** | 日本10年国債（JGB10Y）/ 財務省 R8.4.16 【絶対厳守値】 |
| **β（ベータ）** | **1.15** | コンサルティング業種（成長株・高ROE）の推計値。市場感応度やや高め |
| **ERP（株式リスクプレミアム）** | **5.50%** | 日本市場ERP（Damodaran 2025年推計・JPY建て） |
| **Re（株主資本コスト）** | **8.733%** | Rf + β × ERP = 2.409% + 1.15 × 5.50% = 2.409% + 6.325% = **8.734%** |
| **Rd（負債コスト）** | **0.8%** | コンサル業（低借入・低スプレッド） |
| **Tc（実効税率）** | **30%** | 日本法人実効税率 |
| **D/（D+E）** | **5%** | コンサル業は実質無借金。純資産比率95%と仮定 |
| **E/（D+E）** | **95%** | — |

**WACC計算**

```
WACC = E/(D+E) × Re + D/(D+E) × Rd × (1-Tc)
     = 0.95 × 8.734% + 0.05 × 0.8% × (1 - 0.30)
     = 8.297% + 0.028%
     = 8.325%
     ≈ 8.33%（切り上げ）
```
