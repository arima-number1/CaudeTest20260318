## 1. 事業概要

### 1-1. 会社プロフィール

サムコ株式会社（本社：京都市伏見区）は1979年創業の半導体製造装置メーカー。1981年に国産初の化合物半導体用MOCVD装置を開発・販売した先駆者であり、化合物半導体プロセス装置に特化した国内唯一に近い専業メーカーとして国際的なニッチトップ地位を確立している。

| 項目 | 内容 |
|------|------|
| 設立 | 1979年 |
| 上場 | 東証プライム |
| 従業員数 | 約187名（2024年7月末時点） |
| 資本金 | 約16.6億円 |
| 主要生産拠点 | 京都（本社・製造）、米国・欧州・中国（販売拠点） |

### 1-2. 製品ラインナップ

| 装置カテゴリ | 主要製品 | 対象プロセス |
|---|---|---|
| **MOCVD装置** | 有機金属気相成長装置 | InP・GaAs・GaN系結晶成長（光デバイス・LED・レーザー） |
| **CVD装置** | PECVD・ALD・LP-CVD | 絶縁膜・パッシベーション膜形成（化合物半導体・Si両対応） |
| **ドライエッチング装置** | RIE・ICP-RIE（RIE-400iP-ALE等） | 微細加工・パターニング。InP原子層エッチング対応 |
| **スパッタ装置** | DC・RF スパッタ | 電極・配線形成 |
| **UV洗浄装置** | UV-Ozone洗浄 | 表面清浄化 |

**Delta（新情報の意味）**: 大澤ANの4/16ブローカーノートによれば、4/15スモールミーティングで「個別材料への対応力に優れており、足下で需要が高まっている」と明言。これは光トランシーバー向けInP/GaAs系装置の受注が具体的に動き始めている（開示済み中間決算の数字以上に足元のパイプラインが積み上がっている）ことを示唆する重要なシグナルである。

### 1-3. 顧客層と市場ポジション

- **国内**: 国内大学・研究機関（化合物半導体・フォトニクス研究）、光通信用レーザーメーカー
- **海外**: 欧州・米国の大学・スタートアップラボ。2023年に欧州・日本の研究機関に18%精度改善のコンパクトMOCVD機を提供、導入率40%（市場調査）
- **光通信デバイスメーカー向け**: Coherent（旧II-VI）等の光トランシーバーメーカー向け装置需要が足下で急拡大中（2026年Coherent、InP 6インチファブ稼働計画）

## 3. バリュエーション

### 3-1. DCFモデル（CAPM）

| パラメータ | 値 | Rationale |
|---|---|---|
| **Rf（無リスク金利）** | 2.380% | 固定値（竹村トレードカンパニー社内規定） |
| **β（ベータ）** | 1.35 | 小型精密機械株・半導体装置セクターの典型値。サムコは月次変動の大きさからCAPM上1.2〜1.5の範囲が妥当と判断し中央値採用 |
| **ERP（市場リスクプレミアム）** | 5.5% | 社内標準採用値 |
| **WACC（≒Ke）** | **9.8%** | 2.380% + 1.35 × 5.5% = 9.805%。無借金のためWACC≒Ke |
| **ターミナル成長率** | 3.5% | 光トランシーバー市場のCAGR 13.5%（2025〜2035E）に対し、装置メーカーとして保守的に長期3.5%を採用 |

**DCF試算（5年明示、永続成長モデル）**

| 年度 | NOPAT試算 | 前提 |
|---|---|---|
| FY2026E | ¥17.5億 | 営業利益¥24.6億×(1-実効税率29%) |
| FY2027E | ¥19.5億 | 売上15%成長・OPM維持 |
| FY2028E | ¥22.0億 | 売上12%成長・OPM微低下 |
| FY2029E | ¥24.0億 | 需要成熟化・OPM25%維持 |
| FY2030E | ¥25.5億 | ターミナル移行期 |
| **ターミナルバリュー（TV）** | **(25.5×1.035)÷(9.8%-3.5%)** | **≒¥420億** |

PV計算（各年NOPATをWACC 9.8%で割引）+ TV割引:
- 明示期間PV合計: 約¥88億
- ターミナルバリューPV: 約¥419億 × (1/1.098)^5 ≒ **¥261億**
- **企業価値合計: ¥349億** ← **保守的DCF** (注: 無借金のため時価=株式価値)

*DCFでの内在価値は¥349億 / 発行済株式数（約803万株、BPS¥1,567×PBR5.83から時価総額¥810億割る株価¥10,080=約803万株相当）= **¥4,350/株** という結果になる。現値¥10,080はDCFの2.32倍。これは「成長期待の織り込み」であり、裏を返せばシナリオが崩れた際の下落耐性が極めて低い。*

## 8. Q-1（直近四半期）定性変化 — 大澤ANスモールミーティング（4/15）を踏まえた分析

### 大澤充周AN（みずほ証券）発言サマリー（4/16付けブローカーノート）

> 「個別材料への対応力に優れており、足下で需要が高まっている」

**Delta（この発言が変えるもの）**: この発言は単なるセクター全体の受注増加ではなく、サムコの「個別材料（InP・GaAs・GaN等）」への対応力が需要増加の独自要因であることを明示している。つまり：

1. **装置レシピの専有性**: 化合物半導体各材料に特化したエッチング・成膜レシピ（プロセス条件）をサムコが蓄積しており、顧客がサムコ装置でプロセス最適化した後は競合に乗り換えにくい（スイッチングコスト）
2. **足元の受注動向**: 「足下で需要が高まっている」は中間決算（H1 2026）の売上+11.3%増よりも、その後の受注パイプライン（Q3以降）が加速していることを示唆
3. **光トランシーバー・シリコンフォトニクス向け**: 外部光源としてInP/GaAs系レーザーが不可欠であり、そのファブ向け装置需要が本格化しつつある

**Q-1（FY2026 Q2＝8-1月期中間）の定性変化**:
- 売上高: ¥45.9億（前年同期比+11.3%）→ 数字以上に足元の受注モメンタムが強い可能性
- 営業利益: ¥10.3億（前年同期比+3.8%）→ 増収に対して利益率が低下気味（大型案件対応コストの前払い？）
- ガイダンス: 修正なし（通期¥102億・¥24.6億維持）→ 下期に向けた保守的スタンスを維持
