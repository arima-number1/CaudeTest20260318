## 2. 【社長指示最優先】重点分析

### 2.1 低PBRの原因解剖：「成熟化ディスカウント」か「本物のバリュー」か

#### ■ 現在の株価・純資産関係

```
現在株価（リアルタイム）：¥436
BPS（リアルタイム取得）：¥515
PBR：0.85倍
ディスカウント幅：¥515 - ¥436 = ▲¥79（純資産比▲15.3%割安）
```

**PBR 0.85倍の構造的背景を3層に分解する：**

---

**【第1層：ゲーム事業の成熟化ディスカウント】**

スマホゲーム事業は「フロー型収益」であり、タイトル寿命が限定的。コロプラの主力タイトルは全て2015〜2019年リリースの7〜10年選手であり、**ユーザー離脱加速リスク**を市場は織り込んでいる。

- 白猫プロジェクト（2014年〜）：12年目。コアファン維持も新規獲得困難
- ドラクエウォーク（2019年〜）：7年目。競合不在の地位は強みだが天井感
- ROE -0.3%という数字が示す通り、**現在の事業は純資産を増やす力を失っている**

市場の判断：「現在のBPS ¥515は将来的に毀損する」→PBRが1倍を下回る理論的根拠

---

**【第2層：資本効率の慢性的低さ】**

```
ROE：-0.3%（実質ゼロ〜マイナス）
リスクフリーレート：2.386%
超過リターン：-0.3% - 2.386% = ▲2.686%（価値破壊状態）
```

**ROEがRfを下回っている以上、理論上PBRは1倍を割り込む。**
（Gordon Growth Model的観点：P/B = (ROE - g) / (Re - g)）

現在のROEがRfすら下回る水準では、MktがPBR<1を付けるのは合理的判断である。

---

**【第3層：Cash Rich企業特有のディスカウント】**

逆説的だが、過剰現金保有企業はしばしばPBR割れを示す。

理由：
1. 現金は低利回り資産（インフレ負け）
2. 「株主に還元せず死蔵している」という市場の不信感
3. 経営者の資本配分能力への疑念

コロプラの高自己資本比率（後述）はこのパターンに合致する。

---

**【総合判定】低PBRの原因は「成熟化ディスカウント+資本非効率ディスカウント」の複合**

「本物のバリュー」である可能性は：
- ✅ 純資産の質が良い（現金多い）→NAVベースでは割安
- ❌ 収益力がROE≈0であり、純資産を増やせない
- ❌ 新規カタリストが顕在化するまでは再評価トリガーが不明確

**暫定評価：「条件付きバリュー」。還元強化・新作ヒットが条件。現状は"Cheap for a reason"。**

#### ■ NAV（Net Asset Value）算出

**不動産含み益の検討：**

コロプラは東京・大阪等に自社オフィスを保有しており、一部不動産の含み益が存在する可能性がある。ただし、コロプラは重厚な不動産保有企業ではなく、**不動産含み益はNAVの主要因ではない**点を明記する。

```
【NAV試算】

ステップ1：純資産ベース
  BPS（リアルタイム）：¥515
  ×発行株数（推計1.65億株）= 純資産約¥850億

ステップ2：含み益調整
  投資有価証券評価差額（推計）：+¥0〜30億（時価評価済み分が多く含み益は限定的）
  不動産含み益（推計）：+¥10〜20億（渋谷区オフィス等）
  NAV調整額合計：+¥10〜50億

ステップ3：NAV総額
  NAV = ¥850億 + ¥10〜50億 = ¥860〜900億

ステップ4：1株NAV
  NAV/株 = ¥860〜900億 ÷ 1.65億株 = ¥521〜545/株

【NAV対比株価乖離】
  現在株価 ¥436 vs NAV ¥521〜545
  → NAVディスカウント：▲15〜20%
```

**NAV結論：現在株価はNAVに対して15〜20%割安。純資産の質は概ね良好だが、投資有価証券（VRファンド等）の流動性・時価評価の不確実性が残存リスク。**

## 4. DCFバリュエーション

### 4.1 WACC算出

#### ■ 基本パラメータ設定

```
【リスクフリーレート】
Rf = 2.386%（JGB10Y、2026-04-07 05:44 JST 財務省日次R8.4.3）

【エクイティリスクプレミアム（ERP）】
ERP = 5.5%（Damodaran推計、日本市場2026年水準）
※ 日本株ERP：5.0〜6.0%のレンジ中央値を採用

【ベータ（β）】
β = 0.85（推計）
根拠：
- スマホゲーム業界平均β ≒ 0.8〜1.1
- コロプラはCash Richのため市場感応度やや低め
- 直近株価の低ボラティリティを反映
- 保守的推計として0.85採用

【株主資本コスト（Re）】
Re = Rf + β × ERP
Re = 2.386% + 0.85 × 5.5%
Re = 2.386% + 4.675%
Re = 7.061%

【負債コスト・資本構成】
コロプラの自己資本比率：推計75%
E/(D+E) = 0.75、D/(D+E) = 0.25
Rd（借入金利推計）= 1.5%（低金利借入）
実効税率（Tc）= 30%

【WACC算出】
WACC = E/(D+E) × Re + D/(D+E) × Rd × (1-Tc)
WACC = 0.75 × 7.061% + 0.25 × 1.5% × (1-0.30)
WACC = 5.296% + 0.263%
WACC = 5.559% ≒ 5.56%
```

#### ■ 基本シナリオ FCFプロジェクション

**ターミナルバリュー（Terminal Value）：**
```
TV = FCF(FY5) × (1+g) / (WACC - g)

```

**企業価値（EV）：**
