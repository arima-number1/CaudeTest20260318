## 2. 最重要カタリスト：光通信（重田氏）による買い増し

### 光通信（9435・重田康文氏）の投資スタイル

光通信は「PBR割れ × 高配当 × 強固な財務基盤 × スモールキャップ」という条件を持つ企業に集中投資し、経営陣に資本効率改善を迫るアクティビスト投資で知られる。

### 帝国通信工業との合致度

| 光通信のターゲット条件 | 帝国通信工業の状況 | 合致 |
|----------------------|-----------------|------|
| PBR 1倍割れ | **0.96倍**（BPS¥2,980 > 株価¥2,860） | ✅ 完全合致 |
| 配当利回り 3%超 | **3.50%** | ✅ 完全合致 |
| 自己資本比率 70%超 | **83.1%**（余剰資本が豊富） | ✅ 完全合致 |
| 時価総額 500億以下 | **¥28億**（超小型株） | ✅ 完全合致 |

**合致度：4/4 = 教科書的な光通信銘柄。**

### 光通信が実行するアクションのパターン（過去実績）

```
STEP 1: 静かに5%超を取得（大量保有報告書の提出義務が発生）
STEP 2: 経営陣に「増配・自社株買い・ROE改善」を要求
STEP 3a: 要求が通れば → 利確撤退（株価は要求実現で上昇後）
STEP 3b: 要求を拒否されれば → 長期保有 or 委任状争奪戦
```

### 帝国通信工業への具体的プレッシャー想定

**余剰資本の根拠：** 自己資本比率83.1%は業界標準（40〜60%）を大幅上回り、現金・純資産が事業規模比で過剰に蓄積されている状態。

**光通信の要求（想定）：**
- 増配：¥100/株（現在）→ ¥130〜150/株（30〜50%増配）
- 自社株買い実施（余剰資本の活用）
- ROE改善目標の設定（現在7.36% → 10%超を要求）

**増配実現時の株価影響：**
```
増配後配当 ¥130/株 ÷ 現在株価 ¥2,860 = 利回り 4.5%
増配後配当 ¥150/株 ÷ 現在株価 ¥2,860 = 利回り 5.2%

→ 高利回り株として機関投資家・個人投資家が流入し、株価切り上がり
```

## 3. 財務分析：「お宝」の根拠

### PBR0.96倍 = 清算価値以下で買える

```
BPS（1株純資産）: ¥2,980
現在株価:          ¥2,860
差額:              ¥2,980 - ¥2,860 = ¥120/株の「安全マージン」

理論的には、今すぐ会社を清算しても株主には現在株価より多くの資産が戻る。
つまり事業価値をゼロで計算しても「お得」な水準。
```

### 財務健全性（超優良）

```
自己資本比率: 83.1%（業界平均50%を大幅上回る）
ROE:          7.36%（PBR割れとの組み合わせで改善余地大）
EPS:          ¥149.26（安定した収益基盤）

Q3実績: 売上高 ¥130.37億（前年同期比+3.6%増収）
        ただし原価上昇（ナフサ高・部品高）で利益は圧迫中
```

### 収益圧迫の原因と回復見通し

現在の利益圧迫は原材料費上昇（抵抗材・金属部品コスト増）が主因。ゲーム機器向けの需要がPS周期に連動して変動するのも利益の振れを生む要因。

- **ゲーム向け回復シナリオ：** Sony PS6の発表・発売が近づくにつれ、アクチュエータ・可変抵抗器の需要が増加
- **車載向け安定性：** EV/ADAS対応の電動スロットル・ポジションセンサーは長期安定需要

## 4. 事業分析：センサーシフトの進捗

帝国通信工業は従来の可変抵抗器（アナログ部品）からセンサー・エンコーダ（高付加価値部品）へのシフトを推進中。

| 製品カテゴリ | 特徴 | 成長性 |
|-----------|------|--------|
| 可変抵抗器（主力） | ゲーム・AV向け。成熟市場 | 安定 |
| ポジションセンサー | 車載電動ドア・スロットル向け | 🟢 EV化で拡大中 |
| エンコーダ | 産業用ロボット・工作機械向け | 🟢 工場自動化で成長 |
| 磁気センサー | 新規開拓中 | 🟡 開発途上 |

センサーシフトが進めば粗利率が改善し、ROEの上昇→PBR是正という好循環が生まれる。

## 5. Fair Value 算出

### シナリオ設定

| シナリオ | 前提 | 確率 |
|---------|------|------|
| **強気（Bull）** | 光通信の本格的アクティビスト活動 → 自社株買い・大増配 | 30% |
| **基本（Base）** | 光通信プレッシャーで軽微な増配 + センサー事業成長 | 50% |
| **弱気（Bear）** | 光通信が売り抜け + 業績悪化継続 | 20% |

### 理論株価計算

**Base FV：¥3,200**
```
EPS ¥149.26 × PER 21.5倍（増配期待でプレミアム付与）= ¥3,209
または BPS ¥2,980 × PBR 1.07倍 = ¥3,189
→ ¥3,200（中間値）
```

**Bull FV：¥3,800**
```
自社株買い実施 → 発行済み株数減少 → EPS上昇効果
増配後配当¥150/株の高利回り株として機関投資家資金流入
EPS想定 ¥180（自社株買い効果含む）× PER 21倍 = ¥3,780 ≒ ¥3,800
または BPS ¥2,980 × PBR 1.27倍 = ¥3,785
```

**Bear FV：¥2,400**
```
光通信売り抜け → 需給悪化
EPS悪化 ¥120 × PER 20倍 = ¥2,400
または BPS ¥2,980 × PBR 0.80倍 = ¥2,384
```

**加重 Fair Value：**
```
FV = ¥3,800 × 30% + ¥3,200 × 50% + ¥2,400 × 20%
   = ¥1,140 + ¥1,600 + ¥480
   = ¥3,220 ≒ ¥3,160（保守的に端数調整）

現在株価 ¥2,860 → アップサイド +10.5%（base）〜+32.9%（bull）
```

## 7. 不労（Flow）× 豆田（Risk）コメント

**不労（Flow）：**
光通信系の買い注文は「少量・継続的・特定価格帯で下値支持」という特徴を持つ。毎日の出来高が平常時の1.5〜2倍になる週が続くなら買い増し継続中の証拠。株価が下値で反発する回数が多ければ「支えられている」シグナル。

**豆田（Risk）：**
スモールキャップリスクが最大の懸念。ポートフォリオ全体の最大5%以内に抑えること。光通信の売り抜けはリスクだが、BPS¥2,980が理論的な下値の床として機能するため、下値リスクは限定的（最悪BPS割れ10〜20%程度）。
