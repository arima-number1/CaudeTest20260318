#### 2-2. HPAL技術の競争優位

**HPAL（High Pressure Acid Leach）とは：**
高温高圧（240〜260℃、40〜50気圧）の硫酸環境でラテライト鉱石からNi・Coを浸出する湿式製錬技術。

```
競争優位のポイント：

【技術蓄積の深さ】
  コーラルベイ: 20年以上の操業実績（世界最長クラス）
  → オートクレーブ（反応槽）の耐久性・稼働率管理に圧倒的ノウハウ
  → 世界で成功したHPAL施設は数少なく、SMM/コーラルベイはTop3

【原料調達の安定性】
  フィリピン・パラワン島周辺の鉱区を長期確保
  → 中国・インドネシアに依存しないサプライチェーン
  → 米国IRA・欧州CBAM観点で「フレンドリー調達」として評価高

【産出物MHPの付加価値】
  MHP → NiSO4（硫酸ニッケル）→ 電池正極材（NCA/NCM）へのルートが確立
  住友金属鉱山の電池材料部門と垂直統合
```

**コスト競争力：**

| コスト指標 | SMM HPAL | インドネシアNPI（参考） | 優劣 |
|-----------|----------|---------------------|------|
| C1コスト (Ni/lb) | 推計$3.5〜4.5/lb | $3.0〜4.0/lb（NPI） | ほぼ同等 |
| 産出物品質 | Class I相当（MHP） | Class II相当（NPI） | **SMM優位** |
| 脱炭素適合性 | 高（湿式、炭素負荷低） | 低（石炭電力多用） | **SMM圧倒的優位** |
| EV電池適用性 | ◎（硫酸塩前駆体に直結） | △（精製工程追加必要） | **SMM優位** |

> **深井コメント：** インドネシアNPIとのコスト比較では差が縮まっているが、EV電池グレードのMHP産出・脱炭素適合性・IRA対応の観点で、SMMのHPALは**中長期での構造優位**を維持。特に欧米自動車OEMの「Supply Chain Due Diligence」強化の流れでフィリピン産MHPの引き合いは増加傾向。

### 4-3. DCFモデル（ベースシナリオ詳細）

**FCF推計（ベースシナリオ）**

| 年度 | FCF（億円） | 成長率 | 割引率 | PV（億円） |
|------|-----------|--------|--------|----------|
| FY2026 | 550 | Base | × (1/1.0737)^1 | 512 |
| FY2027 | 567 | +3% | × (1/1.0737)^2 | 491 |
| FY2028 | 584 | +3% | × (1/1.0737)^3 | 471 |
| FY2029 | 601 | +3% | × (1/1.0737)^4 | 451 |
| FY2030 | 619 | +3% | × (1/1.0737)^5 | 433 |
| **PV合計（5年）** | | | | **2,358** |

**ターミナルバリュー計算：**
```
TV = FCF2030 × (1 + g) / (WACC - g)
   = 619億円 × (1 + 0.015) / (0.0737 - 0.015)
   = 628億円 / 0.0587
   = 10,699億円

TV現在価値 = 10,699 / (1.0737)^5 = 7,481億円
```

**企業価値・理論株価（ベースシナリオ）：**
```
EV = PV（FCF5年分）+ PV（TV）
   = 2,358億円 + 7,481億円
   = 9,839億円

純有利子負債 = 推計▲250億円（控除）
株式価値    = 9,839 - 250 = 9,589億円

理論株価    = 9,589億円 ÷ 197百万株 = ¥48,680/株?
```

> **⚠️ Deep井注記：** この計算はFCF550億円ベースだが、これが実態FCFと乖離している可能性がある。FCF550億円は売上1.5兆円に対して約3.7%のマージンで、保守的な設定。一方、公開ROE5.0%・EPS¥88という数値が過去最高水準の金価格に比して低いことは**一時的特損の可能性**を示唆。

**→ 現実的FCFベースに補正（EPS¥88ベース）：**

```

FCF推計（純利益+D&A-CAPEX）：

  - CAPEX推計  250億円

   = 497 + 1,488 = 1,985億円
株式価値 = 1,985 - 250 = 1,735億円
→ ¥1,735億 ÷ 197百万株 = ¥8,807/株（**最保守的DCF値**）
```
