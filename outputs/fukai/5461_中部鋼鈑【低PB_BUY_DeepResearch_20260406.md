## 1-2. ビジネスモデル判定：**電炉メーカー型（スプレッドビジネス）**

> **深井判定：中部鋼鈑は純粋なメーカー型であり、流通加工型ではない。これがスプレッド感応度を最大化させる最重要ファクター。**

### ビジネスモデル構造図

```
【原料調達側】                    【販売側】
鉄スクラップ（主原料） ──→  [電気炉溶解]──→ 鋼片 ──→ [圧延]──→ 厚板・鋼板
電力（製造コスト大）          ↑                                    ↓
副原料（合金鉄等）         コスト変動                        鋼材市況価格
                                                               ↓
                              メタルスプレッド = 販売価格 - (スクラップ価格 + 電力コスト + 副原料)
```

**電炉メーカーの特徴：**
- 主原料：鉄スクラップ（コストの55〜65%を占める）

- 主製品：厚板（板厚6mm以上）が主力。造船・建機・橋梁向け

## 2-1. メタルスプレッドの定義と中部鋼鈑固有の計算式

### 電炉厚板メーカーのメタルスプレッド（MS）

```
MS（円/トン）= 厚板販売価格 − [スクラップ購入価格 × 投入比率(1.1t/t)]
                              − 電力コスト(円/t)
                              − 副原料費(円/t)
                              − 固定費配賦(円/t)
```

**実態推計ベンチマーク値（直近ベース）：**

| 項目 | 単位 | 推計値 |
|------|------|--------|
| 厚板販売価格（国内） | 円/t | 110,000〜120,000 |
| 鉄スクラップ（H2相当）購入価格 | 円/t | 52,000〜58,000 |
| スクラップ投入量（粗鋼1t生産に） | t | 1.08〜1.12 |
| スクラップコスト（換算） | 円/粗鋼t | 57,000〜63,000 |
| 電力コスト | 円/t | 18,000〜22,000 |
| 副原料・その他変動費 | 円/t | 8,000〜10,000 |
| **変動コスト合計** | **円/t** | **83,000〜95,000** |
| **推計メタルスプレッド** | **円/t** | **15,000〜37,000** |
| 固定費（設備減価償却等） | 円/t | 10,000〜13,000 |
| **営業利益換算スプレッド** | **円/t** | **2,000〜24,000** |

> **ポイント：中部鋼鈑の損益は「スプレッド×生産量」で決まる構造。生産量は年産40〜50万トン規模と推計。スプレッドが1,000円/t変動すると営業利益が約4〜5億円動く（感応度試算）。**

## 2-3. 「戦争でメタルスプレッドは拡大するか縮小するか」——深井の数理的回答

### ◆ 結論：**フェーズ依存型。開戦直後は拡大、長期化・エネルギー危機化で縮小リスク**

> **深井の数理的回答（6段論法）：**

**①鋼材価格の上昇メカニズム（有事→価格上昇）**
```
有事発生
  → 軍需（戦車・艦船・装甲車・弾薬ケース・橋梁）向け需要増加
  → インフラ再建期待（ウクライナ復興特需等）
  → 投機的買い（コモディティ全般）
  → 供給不安（ウクライナ・ロシア合計：世界鋼材輸出の約14%）
  ⇒ 鋼材価格：+20〜40%（開戦直後1〜3ヶ月）
```

**②スクラップ価格の上昇メカニズム（鋼材に遅行）**
```
鋼材価格上昇
  → 電炉各社が増産志向
  → スクラップ需要増加
  → スクラップ価格も追随上昇（タイムラグ：2〜6週間）
  ⇒ スクラップ価格：+25〜35%（鋼材の約0.85倍の上昇率）
```

**③スプレッドの純拡大計算（Phase1推計）**

```
鋼材価格の上昇率 > スクラップ価格の上昇率
⇒ スプレッドは「拡大」

定量試算：
  ベース：厚板 115,000円/t、スクラップ 55,000円/t
  有事Phase1後：厚板 145,000円/t（+26%）、スクラップ 70,000円/t（+27%）
  
  スプレッド変化：
    Before: 115,000 - (55,000×1.10) - 20,000 = 34,500円/t
    After:  145,000 - (70,000×1.10) - 23,000 = 45,000円/t（+10,500円/t、+30%）
  
  ⇒ 営業利益インパクト（年産45万t×スプレッド+10,500円/t）= +47億円
```

**④電力コスト上昇の打ち消し効果**
```
有事→エネルギー価格上昇→電力コスト増加
  日本の電炉コストへの影響：
    電力コスト：製造コストの約20%
    原油+100% → 日本の電力コスト+15〜25%推計
    ⇒ 電炉製造コスト：+3,000〜5,000円/t
    
  スプレッド縮小インパクト（電力）：-3,000〜-5,000円/t
  
  ⇒ 純スプレッド拡大（Phase1）：+10,500 - 4,000（電力）= +6,500円/t（純拡大）
```

**⑤長期化フェーズ（Phase3）での逆転リスク**
```
長期化
  → 需要失速（欧州景気悪化、中国需要低迷）
  → 鋼材価格急落（需給緩和）
  → スクラップは国内供給積み上がり→価格下落（但し電力コストは高止まり）
  
  最悪シナリオ：
    厚板 90,000円/t（-22%）、スクラップ 45,000円/t（-18%）
    電力コスト：維持（+23,000円/t）
    
    スプレッド：90,000 - (45,000×1.10) - 23,000 = 17,500円/t
    ⇒ ベースケース34,500円/tから -17,000円/t（-49%）の縮小
```

**⑥現在局面（2026年Q1-Q2）の総合判断**

```
現在の環境：
  ・WTI原油：112.06 USD/bbl（高水準）→ 電力コスト上昇圧力
  ・ウクライナ紛争：継続（停戦協議中）
  ・中東緊張：継続
  ・中国鋼材過剰：継続（押し下げ圧力）
  ・日本国内：防衛費増額、インフラ需要堅調

  ⇒ 現在フェーズの判定：「Phase2〜Phase3移行期」
    鋼材価格：高止まりしているが急騰期は過ぎた
    スクラップ：国内は50,000〜55,000円/t水準
    電力コスト：継続高止まり（23,000〜25,000円/t）
    
  ⇒ 現在のメタルスプレッド（推計）：
    115,000 - (52,000×1.10) - 24,000 = 33,800円/t
    → ベースに近い水準。「縮小でも拡大でもない」均衡状態。
```

## 2-4. 高炉vs電炉：有事スプレッド構造の違い

| 比較軸 | 高炉メーカー（日鉄・JFE） | 電炉メーカー（中部鋼鈑） |
|--------|----------------------|----------------------|
| 主原料 | 鉄鉱石＋原料炭 | **鉄スクラップ** |
| エネルギー | コークス（石炭系） | **電力** |
| 有事での原料高 | 鉄鉱石・原料炭急騰でコスト圧迫 | スクラップは鋼材に追随（遅行）でスプレッド一時拡大 |
| 有事での電力高 | 影響は電炉ほど大きくない | **電力高が直撃（コストの20%）** |
| スプレッド拡大スピード | やや遅い（長期契約多い） | **速い（スクラップはスポット多い）** |
| 有事フェーズ1 | 中程度ポジティブ | **大ポジティブ** |
| 有事フェーズ3 | 電力は相対的に軽微 | **電力高止まりでスプレッド縮小リスク大** |

> **深井コメント：電炉メーカーの中部鋼鈑は、有事Phase1（開戦直後）においてはスプレッド拡大の恩恵を最も受けやすい構造。ただしエネルギー危機長期化（Phase3）では電力コスト上昇がスプレッドを食い尽くすリスクがあり、WTI 112ドルという現在の原油高は中期的に警戒要因。**
