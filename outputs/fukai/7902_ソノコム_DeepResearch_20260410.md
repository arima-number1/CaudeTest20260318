### EF2事業の内容

マクセルのEF2事業は、半導体製造工程向け精密微細加工製品（薄板マスク、回路形成用メタルマスク等）を製造・販売する事業。ソノコムの主力製品であるスクリーンマスク・フォトマスク・メタルマスクと技術・顧客基盤の重複・補完が期待できる。

### 財務的評価

| 分析指標 | 計算値 | 評価 |
|----------|--------|------|
| **EV/Sales（取得価額÷EF2売上高）** | ¥9.3億 ÷ ¥21.7億 = **0.43x** | 極めて安い。業界平均EV/Sales 1.0x前後に対し大幅ディスカウント |
| **EPS影響（FY2027E）** | OP利益率5〜8%と想定 → OP貢献¥1.1〜1.7億 → EPS押し上げ+¥15〜+¥30 | **Accretive（増益的）** |
| **規模感** | EF2売上¥21.7億 vs ソノコム単体¥26.0億（FY2026E） | 売上高が約83%増加。事業規模の大幅拡大 |

### エクイティファイナンス（公募増資）リスク評価

**結論：エクイティファイナンスの可能性は実質ゼロ**

| 項目 | 数値 |
|------|------|
| 現預金（2025年9月末） | ¥37.5億 |
| 短期有価証券 | ¥5.1億 |
| 長期有価証券（売却可能） | ¥20.5億 |
| **流動性合計** | **¥63.1億** |
| 有利子負債 | ¥0（完全無借金） |
| M&A取得価額 | ¥9.3億 |
| M&A比率 vs 純資産 | **10.2%**（純資産¥90.8億の約1割） |

流動性資産¥63.1億は時価総額¥43.7億を大幅に上回る。EV≈▲¥19.4億（ネットキャッシュが時価総額超過）という特異な構造。公募増資を行う動機が財務的に存在しない。仮に追加M&Aを¥30〜40億規模で行っても、有価証券売却と現預金の組み合わせで自己資金調達が可能。

### 戦略的評価

**プラス面**：
- EF2との技術シナジー（精密微細加工技術は既存スクリーンマスク製造と隣接）
- 顧客基盤拡大（半導体メーカーへのクロスセル機会）
- 0.43x EV/Salesという割安価格での取得
- マクセルの「事業縮小撤退」からの事業受け皿として良好な交渉ポジション

**リスク面**：
- EF2事業の収益性が不明（マクセルが売却した背景に収益悪化が潜む可能性）
- 統合コスト（PMI）：福岡県福智町所在のノア社の管理・統合方法
- 市場の初動反応が「限定的」にとどまった
- FY2026業績予想への影響は「精査中」のまま未開示

**深井コメント**: 取得価格は割安感あり。ただしマクセルがなぜこの価格で手放したか（EF2の競争力・収益性・設備状態）の精査が必要。統合後初決算（FY2027.3期 Q1、2026年8月）の粗利率・OP率が最大のモニタリングポイント。

### WACC計算

```
WACC = JGB10Y + β × ERP
     = 2.425% + 0.129 × 5.5%
     = 2.425% + 0.710%
     = 3.135%
```

無借金・株式100%ファイナンスのためWACCは株主資本コストに等しい。

### 理論PBR

| g前提 | 理論PBR | 理論株価 | 現在PBR比 |
|-------|---------|---------|---------|
| g=0% | **(ROE-g)/(WACC-g) = 2.85%/3.135% = 0.91x** | **¥2,311** | 現在0.47xは理論値の半値以下 |
| g=1% | 0.87x | ¥2,203 | |
| g=2% | 0.75x | ¥1,903 | |

> ROE（2.85%）がWACC（3.135%）をわずかに下回る「価値破壊」状態だが、純資産の64%超がキャッシュ・有価証券であることが主因。もし余剰キャッシュを高収益M&A活用・自社株買いでROEが5〜7%に改善すれば、理論PBRは1.6〜2.2xに大幅上昇する。
