### 2-1. 選挙特需剥落後の実態EPS・FCF試算

#### 🔍 選挙サイクルと収益構造の理解

ムサシの収益は**選挙年度に大きく跳ね上がるサイクリカル構造**を持つ。


#### 📊 EPS・収益の特需剥落シナリオ試算

**基礎データ（リアルタイム確認済み）:**
- 現在株価: ¥2,786
- EPS（直近実績、選挙特需込み想定）: ¥470
- PER: 5.9倍（← これが選挙特需込みの割安水準）
- BPS: ¥5,142
- ROE: 4.8%

**【重要前提の確認】**

ROEとEPS/BPSの整合性チェック:
```
ROE = EPS / BPS = ¥470 / ¥5,142 = 9.1%
```
⚠️ **乖離あり:** 取得ROE=4.8% vs 計算値=9.1%

→ この差は**選挙特需年度の一時的高EPS**と、**通常期ROEとしての4.8%**が混在している可能性を示唆。
→ **通常期実態EPS推計:** BPS¥5,142 × ROE4.8% = **¥247**

つまり：
| シナリオ | EPS | 根拠 |
|---------|-----|------|
| 直近実績（選挙特需込み） | ¥470 | Yahoo Finance取得値 |
| 通常期実態（ROEベース） | ¥247 | BPS×ROE4.8% |
| 特需剥落後（中間推計） | ¥320〜350 | 保守サービス・情報機器継続寄与 |

**選挙特需剥落幅の試算:**
```
特需剥落幅 = ¥470 - ¥247 ≒ ▲¥223（減益率: ▲47%）
中間シナリオ = ¥470 - ¥350 ≒ ▲¥120（減益率: ▲26%）
```

- EBIT = EPS × 株数 × PER調整（概算）
- 資本的支出（CapEx）: 売上比4〜6%推定（機器製造業水準）
- 減価償却: 売上比3〜4%推定
- 運転資本変動: 選挙特需年は回収増、通常期は安定

```
【FCFマージン推計】

                    特需年度    通常期（推計）

(-) CapEx/売上       ~5%          ~5%
(+) D&A/売上         ~3%          ~3%
FCFマージン          ~13%         ~6-8%

通常期FCF想定: 売上高×6-8%
（選挙特需年の約半分水準）
```

**通常期EPS・FCF試算サマリー:**

| 指標 | 特需年度（直近） | 通常期（剥落後） | 剥落幅 |
|------|--------------|--------------|--------|
| EPS | ¥470 | ¥247〜350 | ▲25〜47% |

→ **重要結論:** PBRは収益変動に関係なく純資産ベースで算出されるため、**減益が起きてもPBR0.54倍という株価割安性は変化しない。** これが「減益でも低PBRなら買い」論拠の核心。

### 2-2. 低PBR×高自己資本比率の純資産質解剖

#### 🔬 純資産の質的分析

**BPS¥5,142の内訳推計:**

```
【純資産構成の質的解剖（推計）】

BPS ¥5,142（100%）
├── 実現済み留保利益    ~40-50%  ← 最高品質（現金化容易）
├── 資本金・資本剰余金   ~20-25%  ← 高品質
├── 有形固定資産（帳簿） ~10-15%  ← 含み益考察対象
│    └── 不動産含み益    +α      ← 下記詳述
├── 投資有価証券        ~10-15%  ← 市況依存
└── その他（繰延税等）  ~5-10%   ← 低品質
```

#### 🏢 不動産含み益の考察

ムサシは**本社・工場・倉庫等の自社保有不動産**を持つ製造業（精密機器）として、簿価と時価の乖離が生じやすい。

| 考察要素 | 分析 |
|---------|------|
| 取得年代 | 製造業としての設立年代から見て1980〜1990年代取得不動産は含み益保有の可能性 |
| 立地 | 埼玉・首都圏エリアの不動産は近年価格上昇基調 |
| 簿価乖離 | 含み益が実在する場合、**実質BPSはさらに高い可能性** |
| 保守的試算 | 開示なしのため含み益を+10〜20%と仮置き → 実質BPS: ¥5,656〜6,170 |

**含み益考慮後の実質PBR:**
```
保守的含み益試算後実質PBR = ¥2,786 / ¥5,656〜6,170 ≒ 0.45〜0.49倍
```
→ 更なる割安示唆。純資産の「下方硬直性」は強い。

#### 📈 自己資本比率の評価

ROE=4.8%、PBR=0.54倍という構造は、**高い自己資本比率（資本効率が低い）** を示す典型パターン。

推計自己資本比率:
```
製造業・精密機器セクター平均: 40〜50%
ムサシ推定: BPS¥5,142水準と収益規模から 50〜60%以上と推定
```

**高自己資本比率の二面性:**
| 側面 | 評価 |
|------|------|
| ✅ 財務安全性 | 無借金・低負債。倒産リスク極小 |
| ✅ 株価下値硬直性 | 清算価値が株価を大きく上回る |
| ✅ 還元余力 | キャッシュリッチ→自社株買い・増配の原資 |

| ❌ PBR抑制要因 | 資本を使い切れない企業との評価でディスカウント |

### 2-4. 「減益でも低PBRなら買いか」vs「減益でPBRが崩れるリスク」最終判定

#### ⚖️ 論点の整理

**【買い論拠（低PBR維持）】**

```
論拠A: 純資産の絶対的安全網
  BPS¥5,142 >> 株価¥2,786
  清算価値ベースで約2倍の安全余裕
  EPS¥247（通常期）でもPER約11.3倍 → 割安

論拠B: 東証圧力による資本政策の必然性
  PBR0.54倍は「改善要請」の明確な対象
  自社株買い実施→EPS向上・需給改善の好循環

論拠C: 次回選挙特需への「コールオプション」
  2027年以降の選挙年度でEPS¥470復活可能性
  現在のPER5.9倍はその特需を「タダで」付与されている

論拠D: ビジネスモデルの持続性
  選挙インフラとして不可欠。廃業・代替リスク極小
```

**【リスク論拠（PBR崩壊リスク）】**

```
リスクA: 減益幅が予想を超えた場合
  通常期EPS¥200以下→PER14倍超→割安感消失
  ROEさらに低下→機関投資家撤退→PBR圧縮継続

リスクB: 純資産の「張り子の虎」リスク
  不動産価格下落・投資有価証券含み損発生
  → BPS切り下がり → PBR0.54倍の前提崩壊

リスクC: 東証改革への無反応リスク
  資本政策を示さない場合、「PBR改善意欲なし」として
  アクティビスト懸念・機関投資家のアンダーウェイト継続

リスクD: 電子投票化による既存ビジネス消滅リスク
  デジタル選挙への移行で物理機器需要が中長期で消滅
  （ただし日本の政治システム的に移行は極めて遅い）
```

## 4. DCFバリュエーション

### 4-1. DCF前提条件（全開示）

#### マクロ・割引率前提

| パラメータ | 数値 | 根拠 |
