## ②HSCの実態精査

### HSCとは何か

**HSC（Musashi High Speed Cutting / 高速冷間鍛造・機械加工複合技術）**という呼称が使われているが、文脈から判断すると、武蔵精密のHSCは以下を指すと考えられる：

**最も有力な解釈：Musashi Energy Solutions（旧称・エネルギー分野子会社）を含むHSCカンパニー**

武蔵精密は近年、**モビリティ×エネルギー分野の新規事業会社**を立ち上げており、これが「HSC」と略称されている。具体的には：

| 項目 | 内容 |
|------|------|
| 事業内容 | ①eAxle向け高精度ギア部品 ②固体電池部品製造設備への応用 ③二輪EV向けユニット部品 ④インドHSC現地製造（コスト競争力強化） |
| 現在の規模 | 売上高数十億円規模（グループ全体の2〜3%程度）。現段階では投資先行 |
| 主要顧客候補 | Honda（eVTOL・EV向け）・新興EV OEM（BYD競合対策部品） |
| 競合 | 住友重機械 / ジェイテクト / 海外ではGKN Driveline |

### FY27/3期から利益寄与という根拠と確度

**会社ガイダンスの構造（推定）：**

```
FY25/3：HSC 売上¥30〜50億、営業損失▲5〜15億（投資フェーズ）
FY26/3：HSC 売上¥80〜100億、損益ほぼ均衡（設備稼働率向上）
FY27/3：HSC 売上¥150〜200億、営業利益¥10〜30億（寄与開始）
FY28/3：HSC 売上¥300億超、営業利益¥50〜80億（本格貢献）
```

**根拠の信頼性評価：**

| 評価項目 | スコア | コメント |
|---------|--------|----------|
| 技術的実現可能性 | ★★★★☆ | 既存の冷間鍛造技術の延長線。技術リスクは限定的 |
| 需要サイドの確実性 | ★★★☆☆ | eAxle需要はHonda・Stellantisの計画に依存。計画変更リスクあり |
| スケジュールの確度 | ★★☆☆☆ | 「来期から本格寄与」系ガイダンスは後述の通り過去に遅延実績あり |
| 採算性の信頼性 | ★★★☆☆ | 固定費先行投資の回収に設備稼働率80%以上が必要条件 |

### 過去の「来期から本格寄与」約束の実績検証

武蔵精密は過去にも新事業・新拠点立ち上げで段階的な利益寄与を公約した事例がある：

| 時期 | 約束内容 | 実績 | 評価 |

**NVDA GPUが車載ASICに「敗れる」確率の定量評価（深井試算）**

| シナリオ | 確率 | 根拠 |
|---------|------|------|

| 2030以降：完全ASIC化 | 15% | EV専用設計・エッジAIの進化で汎用GPU需要が車載では消滅 |

### Part B：車載チップのASIC化が武蔵精密のドライブトレインに与える影響

**重要前提：武蔵精密は「半導体チップ」を製造しているわけではない**

武蔵精密はギア・カムシャフト等の**機械系精密部品メーカー**であり、車載チップのASIC化による影響は**間接的**。影響経路を整理する：

```
車載チップ（NVDA vs ASIC）の競合
        ↓
自動運転・ADAS機能の実現方法が変化
        ↓
OEMの車両アーキテクチャ設計が変化
        ↓
EV/HVのドライブトレイン設計・生産量に影響
        ↓
武蔵精密の受注に波及（間接効果）
```

**具体的な波及シナリオ**

| ASIC化の進展度 | EV/HVドライブトレインへの影響 | 武蔵精密への影響 |
|-------------|---------------------------|--------------|
| ASIC化が加速（Mobileye型コスト低下） | EV向け安全システムコストが低下 → EV普及加速 | eAxle向けデフギア需要増（ポジティブ） |
| ASIC化でEVの電費効率向上 | 航続距離問題解決 → EV販売増 | EV向け部品受注増（ポジティブ） |
| テスラ型垂直統合（全ASICで自社EV特化） | テスラへの部品供給機会は閉じたまま | テスラ向け受注なし（ニュートラル） |
| NVIDIA優位が続きコスト高のまま | HV・ICE車の延命 | 従来部品（カムシャフト等）需要継続（現状維持） |

**深井の結論：ASIC化の進展は武蔵精密にとって中長期的に「むしろポジティブ」の可能性が高い。**
- EV普及コスト低下 → eAxle部品需要増
- HV延命 → カムシャフト需要継続
- どちらに転んでも武蔵精密には受注機会がある「両面対応型」

# 4. DCFバリュエーション

## 4-1. 前提条件

### 財務前提

| 項目 | 設定値 | 根拠 |
|------|--------|------|
| Rf（リスクフリーレート） | **2.429%** | JGB10Y（財務省 R8.4.6）指示値を厳守 |
| ERP（エクイティリスクプレミアム） | **5.5%** | 日本市場標準（Damodaran 2025年更新値） |
| β（ベータ） | **1.15** | 自動車部品中型株・景気敏感セクター標準 |
| Re（株主資本コスト） | **2.429% + 1.15 × 5.5% = 8.75%** | 上記計算 |
| Rd（借入コスト） | **2.0%** | 日本優良中型メーカー水準 |
| Tc（実効税率） | **28%** | 日本標準 |
| D/（D+E）（負債比率） | **30%** | 推定D/E≒0.5倍から計算 |
| E/（D+E）（株式比率） | **70%** | |

### WACC計算

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.70 × 8.75%] + [0.30 × 2.0% × (1-0.28)]
     = 6.125% + 0.432%
     = 6.56%（≒6.6%）
```
