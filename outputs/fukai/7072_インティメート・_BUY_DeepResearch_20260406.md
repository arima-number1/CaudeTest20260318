## 1. 企業概要・事業分析

### 1.1 インティメート・マージャーとは何者か

インティメート・マージャー（Intimate Merger、以下IM）は、**データマーケティング×DMP（Data Management Platform）**を核とするアドテク・マーテック企業である。東証グロース上場の小型株ながら、日本のデジタル広告エコシステムにおいて独自のポジションを占める。

**主要プロダクト群:**

| プロダクト | 概要 | 収益モデル |
|-----------|------|-----------|
| UM（Unified Measurement） | 1st/3rd Party データ統合DMP | SaaS月額 + 従量課金 |
| インティメート・オーディエンス | データオーディエンス配信 | CPM型レベニュー |
| Cookieレス対応ソリューション | ポストCookie時代の識別技術 | ライセンス/SaaS |
| データコンサルティング | マーケティングデータ活用支援 | プロジェクト型 |

### 1.2 競合優位性の評価

**強み（Moat分析）:**
- 日本最大級の**匿名行動データ（オーディエンスデータ）**の保有・管理

**弱み・懸念:**

- Google・メタ等プラットフォームジャイアントとの**間接競合リスク**

## 2. 🎯 社長指示最優先分析：「そんなに安くない気がする」を数値で解剖する

> **社長指示優先度2:** 「そんなに安くない気がする」直感を数値で検証。MBO候補としての現実的なプレミアム水準を試算

### 2.1 「時価総額が低い≠割安」の罠：定量的解剖

まず社長の直感を結論から言う。**正しい。**

以下の3つの角度から「安くない」を証明する。

---

#### 📌 罠①：PBR1.70倍は「バリュー株」ではない

```
PBR = 株価 / BPS
1.70 = ¥864 / ¥509  ← リアルタイム確認値で検証済み
```

**PBR比較表（データマーケティング・アドテク小型株）:**

| 指標 | IM（7072） | 同業平均水準 | 判定 |
|------|-----------|------------|------|
| PBR | 1.70倍 | 1.5〜3.0倍 | 中央値付近 |
| PER | 18.5倍 | 20〜35倍（成長株） | やや割安に見えるが… |
| ROE | 16.6% | 10〜20% | 健全 |

PBR1.70倍は**「割安のシグナル」ではなく、「正当な評価」水準**である。PBR1.0倍未満のいわゆる解散価値割れとは全く異なる。

---

#### 📌 罠②：「時価総額が小さい」は「株価が安い」と同義ではない

時価総額の絶対額（例：50〜100億円台）が小さいことは、**1株当たりの本源的価値と何ら関係がない。**

```
時価総額 = 株価 × 発行済株式数
株価の割安度 = 本源的価値（FV）との比較
```

IM社の場合：
- 時価総額が小さい理由 → **発行済株式数が少ない（浮動株が限定的）**
- 株価¥864は、EPS¥47に対してPER18.5倍を付けており、収益の約18.5年分を払っていることに変わりない

**「小さな会社の株だから安い」という錯覚は典型的な認知バイアス（サイズ錯誤）である。**

---

#### 📌 罠③：成長織り込みを加味すると「割高」に傾く可能性

**PEGレシオ（Price/Earnings to Growth）で検証:**

```
PEGレシオ = PER / 期待EPS成長率(%)
```

| 想定成長率 | PEGレシオ | 判定 |
|-----------|---------|------|
| 成長率15%/年の場合 | 18.5 / 15 = 1.23倍 | 若干割高 |
| 成長率20%/年の場合 | 18.5 / 20 = 0.93倍 | やや割安 |
| 成長率10%/年の場合 | 18.5 / 10 = 1.85倍 | 割高 |

PEGレシオが1.0倍を超えると**成長性比較で割高圏**とされる。IMが15%超の持続的成長を実現できるかは、現時点で確信を持てる根拠が乏しい。

**結論：社長の「そんなに安くない気がする」は財務指標が裏付ける正しい直感。通常の長期投資観点では割安とは言えない。**

### 2.2 MBO可能性の定量検証

#### 2.2.1 大株主・株主構成の特徴

| VC・機関投資家 | 推定20〜35% | イグジット意向あり → MBO賛成可能性 |
| 個人投資家・その他 | 推定30〜45% | プレミアム次第で賛成 |

**MBO成立の条件：**
- 経営陣が保有株 + 外部借入で買い集め可能か
- TOBプレミアム提示で一般株主の賛成を得られるか
- PE（プライベートエクイティ）ファンドとの連携（スポンサー型MBO）

#### 2.2.3 MBO後の買収採算性（買い手目線）

MBOを実行するPEファンドが要求するIRRを満たせるか：

```
想定TOB価格: ¥1,100（標準シナリオ）
時価総額ベース買収額: 仮に発行済株数 × ¥1,100

- 小型株のため**レバレッジドバイアウト（LBO）の借入余力**が限定的
- データ事業のEBITDAマージンが低い場合、LBOの採算が合わない
- 上場廃止後の**EXIT手段**（再上場 or 戦略的売却）が見えにくい

## 4. DCFバリュエーション

### 4.1 前提条件（全項目明記）

| パラメータ | 採用値 | 根拠 |
|-----------|--------|------|
| **Rf（リスクフリーレート）** | **2.380%** | JGB10Y 社長確認値 |

| **β（ベータ）** | **1.35** | 小型アドテク/データ株の市場感応度 |
| **Re（株主資本コスト）** | **10.48%** | 2.380% + 1.35 × 6.0% |

| **Tc（実効税率）** | **30%** | 日本法人税率標準 |

| **予測期間** | **5年** | |

**WACC計算詳細：**

```
Re = 2.380% + 1.35 × 6.0% = 2.380% + 8.10% = 10.48%
Rd × (1 - Tc) = 1.5% × (1 - 0.30) = 1.05%

WACC = [0.90 × 10.48%] + [0.10 × 1.05%]
     = 9.432% + 0.105%
     = 9.54%（≒9.53%）

## 5. 理論株価と投資判断

### 5.1 バリュエーションサマリー

| 手法 | 理論株価 | 現在株価比 |
|------|---------|-----------|
| DCF（基本ケース） | ¥800 | ▲7.4% |
| PBRベース（ROEアプローチ） | ¥750〜¥900 | ▲4〜▲13% |

| MBO織り込みFV（30%確率） | ¥898 | +4% |

**ROEアプローチ（PBR妥当性検証）:**

```
Ke（資本コスト） = 10.48%
ROE = 16.6%
理論PBR = (ROE - g) / (Ke - g)
        = (16.6% - 2.0%) / (10.48% - 2.0%)
        = 14.6% / 8.48%
        = 1.72倍

理論株価 = 1.72 × ¥509 = ¥875
```

→ ROEアプローチでは理論PBR1.72倍に対し、現在PBR1.70倍は**ほぼ適正水準**。つまり**現在株価はFair Value付近にある。**
