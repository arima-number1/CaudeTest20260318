## 4. フェアバリュー（FV）算出

### 4-1. 方法1: Forward P/E法

| 前提 | FY2027E純利益 | EPS | P/E倍率 | FV |
|------|-------------|-----|--------|-----|
| 保守的（関税最悪） | ¥3,200億 | ¥302 | 18x | ¥5,436 |
| **基本（ベース）** | **¥3,500億** | **¥330** | **20x** | **¥6,600** |
| 強気（コモディティ継続+アフリカ加速） | ¥3,800億 | ¥358 | 22x | ¥7,876 |

> **P/E倍率のRationale**: 豊通の過去5年平均P/E: 14〜20倍。アフリカ資産のプレミアムを考慮し20倍を基本倍率に設定。同業比（伊藤忠13倍・三菱商事10倍）より高いのはアフリカ高成長資産（+16.9%）とDRAM/メタル複合による差別化。

### 4-2. 方法2: Justified PBR法

```
Ke（資本コスト）= Rf + β × ERP
  = 2.380% + 0.817 × 5.5%
  = 2.380% + 4.494%
  = 6.87%

Justified PBR = ROE / Ke = 13.5% / 6.87% = 1.966倍

Justified FV = BPS × Justified PBR = ¥2,848 × 1.966 = ¥5,600

現在PBR 2.15倍 > Justified 1.966倍 → 9.4%の「成長プレミアム」が上乗せ済み
```

> 現在株価¥6,124はJustified FV¥5,600を+9.4%上回る。この差はアフリカ成長オプションとコモディティ高の「歪みプレミアム」として正当化可能。

### 4-3. ブレンドFV（採用）

```
ブレンドFV = Forward P/E（60%）+ Justified PBR（40%）
  = ¥6,600 × 60% + ¥5,600 × 40%
  = ¥3,960 + ¥2,240
  = ¥6,200（下限）

TP設定: ¥7,000（アナリストコンセンサスTP¥7,498より保守的に設定）
  → 保守的FV ¥6,200 と P/E強気ケース ¥7,876 の中間を採用
```

**アナリストTPコンセンサス（2026-04-06時点）**: ¥7,498（強気2・買い1・中立2）

## 9. 「ガイダンス未定」シナリオ詳細分析

### なぜ豊田通商は他社より開示確率が高いか

```
開示確率比較（深井推計）:
  自動車部品メーカー（トヨタ系）:  30〜40%（関税直撃・visibility最低）

豊通が開示確率高い理由:
  1. アフリカセグメント（+16.9%）がモビリティ減の自然ヘッジ機能
  2. 銅・アルミ・原油高が当面継続見通し（ホルムズ封鎖継続中）
  3. 過去5期連続で翌期ガイダンスを4月末に開示してきた実績
  4. 伊藤忠・丸紅・三菱商事・三井物産が全社FY2027ガイダンス開示済み（同業圧力）
```

### ガイダンスが「未定」になる最悪シナリオ

```
シナリオ: 親会社トヨタ自動車が「FY2027ガイダンス保留」を4月末決算で表明
  → 豊通もモビリティ連動でガイダンス開示困難と判断する可能性
```

## 11. 不動産含み益・隠れ資産（DeepResearch必須考察）

豊田通商は土地・建物の含み益については詳細を開示していないが：
