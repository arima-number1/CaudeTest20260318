## 1. 企業概要・事業分析

### 1-1. GLP J-REIT（3281）とは

**GLP J-REITは東京証券取引所上場の物流施設特化型J-REIT（不動産投資信託）**。スポンサーは世界最大級の物流施設デベロッパーである**GLP（Global Logistic Properties）**。2012年上場。

**特徴・強み：**
- **大型マルチテナント型物流施設に特化**：GLP浦安Ⅲ、GLP流山Ⅲ等、首都圏・関西圏の主要物流ハブに集中
- **スポンサーパイプライン優位**：GLPが開発したBTS（Built-to-Suit）案件を優先的に取得できる外部成長余力
- **テナント品質が高い**：EC大手・3PL・食品メーカー等の長期契約テナント中心
- **ESG対応**：太陽光発電設備設置による収益多様化とGreen Finance実績

### 1-2. ポートフォリオ概要（直近公開情報ベース）

| 項目 | 内容 |
|------|------|
| 保有物件数 | 約37〜40棟（継続的に入替中） |
| 資産規模（取得価格ベース） | 約7,500億円〜8,000億円 |
| 地域分布 | 首都圏60%超・関西圏20%・その他 |
| 平均築年数 | 比較的新しい（竣工10年未満が多数） |
| 稼働率 | **98%台（ほぼ満稼働）** |
| 主要テナント | ECプラットフォーム・3PL・食品・医薬品 |

### 1-3. 直近カタリスト

① **賃料改定サイクル到来**：2023〜2026年にかけて竣工時の長期契約が満期を迎え、市況賃料への改定余地が最大+15〜20%程度存在。これが内部成長の主エンジン。

② **EC物流・医薬品・冷凍冷蔵物流の需要拡大**：構造的な物流施設不足継続。空室率が歴史的低水準を維持。

③ **自己投資口取得継続**：2025〜2026年にかけて積極的な自己口取得を実施。口数減少によりDPU/NAVの1口当たり価値が向上。

④ **GLP親会社のスポンサーサポート**：親会社保有物件のパイプラインから選別取得可能。外部成長の継続性が高い。

## 4. DCFバリュエーション

### 4-1. バリュエーション前提条件

**J-REITの特性上、DCFはDPU割引モデル（Dividend Discount Model = DDM）で実施する。**

| 前提項目 | 設定値 | 根拠 |
|---------|--------|------|
| リスクフリーレート（Rf） | **2.439%** | JGB10Y（財務省 R8.4.10） |
| ERP（エクイティリスクプレミアム） | **5.5%** | 日本市場標準（Damodaran 2025-2026推定） |
| β（GLP J-REIT） | **0.65** | 物流REIT特有の低変動性（市場β推定） |
| 株主資本コスト（Re） | **Rf + β×ERP = 2.439% + 0.65×5.5% = 6.01%** | — |
| 負債コスト（Rd） | **1.5%** | 直近リファイナンス水準（変動平均） |
| 実効税率（Tc） | **0%** | REITの導管性（ほぼ非課税） |
| D/（D+E）比率（LTV） | **42%** | J-REIT業界平均・公開情報 |
| E/（D+E）比率 | **58%** | 上記より |
| **WACC** | **= 0.58×6.01% + 0.42×1.5%×(1-0) = 3.486% + 0.63% = 4.12%** | — |
| 巡航DPU（2026年8月期） | **¥2,841** | ミーティングメモ確認値 |
| 年換算巡航DPU | **¥5,682** | ¥2,841×2期 |
| 成長率g（フェーズ1：1〜5年） | **3.0%** | 内部成長+自己口取得継続 |
| 成長率g（フェーズ2：6〜10年） | **2.0%** | 賃料改定余力逓減 |
| 永続成長率gT（Terminal） | **1.5%** | 日本インフレ率+αの保守的水準 |
