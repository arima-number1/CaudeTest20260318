## 会社概要

新明和工業は4事業部門を擁する総合機械メーカー:
- **特装車**: ごみ収集車・塵芥車・脱着ボディー車。国内シェア首位級
- **航空機**: US-2飛行艇（海上自衛隊向け）。国内唯一の大型飛行艇メーカー
- **産業機械・環境システム**: 駐車機械・洗浄装置・環境関連
- **流体機械（ポンプ）**: 官公庁・インフラ向け

## IR回答 — 重要発見事項

### 1. 防衛事業目標: 500億円（IR質問で初確認）

> **「2030年度の防衛関連売上イメージとして、500億円程度（US-2：150億円程度、US-2以外：350億円程度）を目標として掲げております」**

| 項目 | 内容 |
|------|------|
| 従来中計目標（公開情報） | 防衛関連売上 2030年200億円 |
| **IR開示の実際目標** | **2030年500億円（US-2: 150億 + その他: 350億）** |
| 目標上方修正倍率 | **2.5倍** |
| 時価総額比 | **時価総額¥1,647億 vs 防衛売上目標500億円 = 30%超** |

→ **防衛単独で売上の30%を超える事業規模になる計画。防衛OPM仮定15%で¥75億OP = EPS+¥70相当（現EPS比+45%）**

### 2. 特装車: 価格改定+8%、2026年度下期から本格寄与

> **「平均して8％前後の価格改定を行っており、その価格改定効果が2026年度の下期から本格的に寄与する見込みです」**

| 項目 | 推計値 |
|------|--------|
| 受注残 | **14ヶ月**（ファクトシート8ページ） |
| 価格改定率 | **+8%** |
| 本格寄与開始 | **2026年度下期（10月〜）** |
| OP寄与試算（FY26下期のみ） | +¥7億（売上¥220億×8%×80%OP転換×0.5年分） |
| OP寄与試算（FY27通期） | **+¥14億**（EPS +¥13） |

### 3. 政府・官公庁向け売上比率（実態）

| セグメント | 官公庁比率 |
|---------|---------|
| 産業機械・環境システム | **約50%** |
| 流体（ポンプ） | **約60%** |
| 航空機 | **約50%** |

→ **全社売上の約40〜50%が官公庁向け。防衛予算増加の直接受益構造**

### 4. 独禁法リスク（審判継続中）

- 2025年3月・9月に審判関連リリース公表済み

### Bull Case（TP ¥3,800）
1. **防衛事業500億円ロードマップ進捗確認**: US-2インド輸出交渉進展、US-2以外でCBRN対応車・UAV関連受注拡大
2. **特装車マージン回復**: FY2027通期で+8%価格改定フル寄与 → OPM5.5%以上
3. **B787月産14機回復**: ボーイングの生産正常化で航空機セグメント利益急回復
4. **防衛PBR再評価**: PBR2.5x以上への収束（現PBR1.33 → 2030年防衛500億目標織り込み）

### Base Case（TP ¥3,000）
1. 特装車価格改定+8%がFY2026下期から寄与開始（IR確認済み）
2. 防衛事業は現状維持 + 緩やかな受注増加
3. 独禁法は軽微な課徴金（¥10億未満）で決着
4. 市場がPBR1.33→1.8x程度に再評価

### Bear Case（TP ¥1,700）
1. 独禁法課徴金が¥30〜50億規模 → EPS▲15〜20%打撃
2. B787生産遅延継続 → 航空機セグメント赤字転落リスク
3. 特装車価格改定効果が想定下回り（原材料高の続伸で相殺）
4. PER13x水準に売られる

## 結論

**浅井スコア: +82 BUY**

IRで確認された**防衛2030年売上目標500億円**（公開情報の2.5倍）は、時価総額¥1,647億の企業に対して変革的な規模感。PBR1.33は三菱重工の24%水準であり、防衛比率上昇に伴う再評価余地が大きい。特装車+8%価格改定の**5/8決算での初確認**が近接カタリスト。

独禁法課徴金リスクがベアケースの主因だが、現時点では規模未確定 = 過度な売り込みは不合理。**¥2,300〜2,450のMA25近傍で打診買い、5/8決算で独禁法リスク確認後にフルポジ移行**を推奨。
