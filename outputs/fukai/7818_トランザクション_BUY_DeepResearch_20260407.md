**根拠：**

| 判断材料 | 示唆 |
|----------|------|

### 3-3. 食品インフレ局面での「必需品化」検証

```
【深井の仮説検証】
「推し活」支出は食品インフレ下でも削られにくいか？

検証① エンゲル係数の上昇（食費比率）と推し活支出の相関
→ 2023〜2024年の調査では、食費上昇局面でも推し活支出は増加継続
→ 「好きなものへの支出は最後まで削らない」という心理的優先度

検証② CPI（消費者物価）との比較
→ エンタメ・趣味消費は弾力性が低い（嗜好品だが感情的必需品化）
→ むしろ「自己肯定感・コミュニティ帰属感」という精神的充足財として機能

検証③ コロナ禍（2020年）の実績
→ ライブ・イベントが停止した中でもグッズ・EC購入は維持・拡大
→ 「推し」への支出代替が起きた（ライブ費→グッズ費）

結論：食品インフレとの無相関性は高い。むしろストレス解消・精神的充足の
      観点から「逆相関」的に推し活支出が増える可能性も。
```

### 5-3. PBR・PER水準の適正性評価

```
【PBR 3.36倍の適正性】

Gordon成長モデルによるPBR正当化水準：
適正PBR = ROE × (1 − payout ratio) / (Re − g) ÷ (1/PER)
        = ROE / Re × (1 − g/ROE) ≒ ROE / (Re − g)

ROE = 22.5%
Re（株主資本コスト）= 2.429% + 0.8 × 6.0% = 7.2%（後述WACC参照）
g（持続成長率）= 3.5%（推し活市場成長率を反映）

適正PBR = 22.5% / (7.2% − 3.5%) = 22.5% / 3.7% = 6.08倍

→ 現在PBR 3.36倍は「理論適正PBR（6.08倍）」の約55%水準
→ 推し活成長株として明確に割安圏

【PER 16.4倍の適正性】
同業（消費関連エンタメ）の平均PER：20〜25倍
→ 現在PER 16.4倍はセクター平均比で20〜35%のディスカウント
→ 成長持続性が確認されれば再評価余地大
```
