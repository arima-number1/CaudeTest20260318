## 2. 社長指示最重要分析：株価不反応原因解明×ガイダンス考察

> **【社長指示最優先セクション】** 4Q Op 26.3億円・コンセンサス比+36.3%のサプライズにもかかわらず、決算前-4.6%・PTS-0.1%と株価が全く反応しない。この異常事態の原因を5要因に分解して定量的に検証する。

### 2-1. 株価不反応の「5要因仮説」定量検証

#### 【仮説A】来期ガイダンス保守的 ← **最有力：重要度★★★★★**

**論拠と検証:**

アパレル小売企業の株価は「今期実績」ではなく「来期EPS予想の変化」に反応する。
PER=15.6倍（現在）× EPS=¥187 = ¥2,919（現在株価）は整合的。

```
来期ガイダンス仮説（会社発表ベース推定）:

仮に来期（FY2027）営業利益予想を：
  ケース①：今期実績並み or 微増 → 市場は「成長ゼロ」と判断
  ケース②：保守的な前提（消費環境悪化・コスト増）で▲10〜20%
  
来期EPS予想の変化が株価に与える影響：
  来期EPS 仮に¥170（▲9%）→ PER15.6倍でも株価 ¥2,652（▲9.1%）
  来期EPS 仮に¥160（▲14%）→ 株価 ¥2,496（▲14.5%）
```

**「来期ガイダンス保守的仮説」の検証ポイント:**

| 検証項目 | 内容 | 株価インパクト |
|----------|------|--------------|
| 来期営業利益予想 vs 今期実績 | 会社ガイダンスが今期実績を下回る場合 | **ネガティブ（最大要因）** |
| 来期売上高成長率予想 | 一桁台前半なら「成長株プレミアム剥落」 | ネガティブ |
| 利益率ガイダンス | コスト増要因（人件費・家賃・物流）の織り込み | ネガティブ |
| 配当予想 | 増配 vs 据え置き | 中立〜ポジティブ |

> **深井の見解:** 4Q単独での+36.3%サプライズが株価に反映されない最大理由は、**「会社が来期ガイダンスを保守的に設定し、アナリスト・機関投資家がそれを素直に織り込んだ」**可能性が極めて高い。日本企業のIR慣行として、期末サプライズ後に翌期は保守的ガイダンスを出すパターンは頻出。

---

#### 【仮説B】信用買い残整理 ← **重要度★★★☆☆**

**論拠と検証:**

```
信用倍率の推定：
  アパレル中小型株の特性として、好業績期に信用買い残が積み上がる傾向
  
株価水準の推移（推定）：
  決算前の-4.6%という下落は、「決算プレイの手仕舞い売り」の可能性
  → 信用期日到来（3ヶ月〜6ヶ月）に伴う買い残解消が先行した可能性
  
  時価総額≒350億円に対し、信用買い残が数十億円規模なら
  需給インパクトは10〜20%程度のダウンサイドプレッシャー
```

**検証結果:** 決算前-4.6%の下落が信用買い残の整理と一致している可能性はある。ただし、PTS-0.1%（ほぼゼロ）という反応の小ささは、「決算内容を受けて信用売りが増加していない」ことも示す。**中程度の寄与要因。**

---

#### 【仮説C】時価総額・流動性の壁 ← **重要度★★★★☆**

**論拠と検証:**

```
時価総額推定：
  現在株価 ¥2,919 × 発行株式数推定（後述）
  
  BPS ¥1,772 × PBR 1.65倍 = ¥2,919（整合的）
  
  純資産総額 = BPS ¥1,772 × 発行株数
  EPS ¥187 × PER 15.6倍 = ¥2,919（整合的）
  
  ROE = EPS / BPS = ¥187 / ¥1,772 = 10.55%
  ※ yfinance提供ROE 11.7%とやや乖離あり（期首・期末平均の差による）
  
発行株式数の逆算：
  仮に純資産が300〜400億円規模なら
  株数 = 300〜400億 / ¥1,772 = 1,693〜2,258万株
  時価総額 = 1,693〜2,258万株 × ¥2,919 = 約494〜659億円
  
  ただし浮動株比率60〜70%と仮定すると
  流動性のある実質時価総額 = 296〜461億円
```

**流動性の壁の影響:**

| 機関投資家カテゴリ | 最低時価総額要件（目安） | 2685の対応可否 |
|------------------|----------------------|--------------|
| 大型アクティブファンド | 500億円〜 | **ギリギリ / 微妙** |
| 中型バリューファンド | 200〜500億円 | 対応可能 |
| 外国人機関投資家 | 通常500億円〜 | **困難** |
| 小型グロースファンド | 100〜300億円 | 対応可能 |

> **深井の見解:** 時価総額300〜500億円の「谷間」は、大型機関が入れず小型機関でも規模超過というデッドゾーン。外国人投資家の買いが入りにくく、需給改善のカタリストが限られる。**株価不反応の構造的要因。**

---

#### 【仮説D】機関投資家が入れないサイズ ← **重要度★★★☆☆**（仮説Cと重複要因）

仮説Cで既に分析済み。補足として：

```
TOPIX構成比：
  時価総額300〜500億円はTOPIX内での存在感が薄く、
  パッシブファンドによる機械的買いも限定的
  
アナリストカバレッジ：
  時価総額が小さいと証券会社のアナリストカバレッジが少なく、
  コンセンサス形成が遅れる → サプライズが市場に伝わりにくい
  （今回のコンセンサス¥19.3億円がすでに古い可能性）
```

---

#### 【仮説E】材料出尽くし ← **重要度★★★☆☆**

**論拠と検証:**

```
「材料出尽くし」パターンの典型的条件：
  ① 決算前に株価が先行上昇していた
  ② 市場がすでに好決算を織り込んでいた
  ③ 決算発表と同時に利益確定売りが出た
  
今回のケースへの適用：
  決算前-4.6%という下落は上記①②と矛盾する
  → 「先行上昇→出尽くし」のパターンではない可能性が高い
  
  ただし、浅井速報スコア+98という高い期待値が
  「既に評価された後」という状況であれば、
  サプライズの実現が「想定内」として処理された可能性
```

**検証結果:** 決算前に株価が先行上昇していないため、純粋な「材料出尽くし」ではない。ただし、中期的に市場参加者の間でサプライズが「織り込み済み」だった可能性は排除できない。**補完的要因。**

### 3-2. 通期業績推定

```

### 3-3. コンセンサス対比の意味分析

**なぜコンセンサスが¥19.3億円と低かったか？**

```
コンセンサス乖離の背景（推定）：
  1. 前四半期（3Q）の業績が想定以下だった可能性
  2. アナリストが「年末商戦不調」を織り込んでいた
  3. カバレッジアナリストが少なく、古い予想が放置されていた
  4. 暖冬リスクの保守的織り込み
  
実際に+36.3%上振れた要因（推定）：
  ① 年末・冬物需要の想定外の強さ
  ② EC比率上昇による固定費レバレッジ効果
  ③ 在庫管理の改善（バーゲン率低下→粗利率改善）
  ④ 費用のタイミング差（一部費用が翌期ずれ込みの可能性）
```

> ⚠️ **深井注記:** ④の「費用前倒しずれ込み」は要精査。来期ガイダンスに費用前倒しが含まれる場合、今期サプライズが来期負担になる「見かけ上のサプライズ」である可能性がある。

  26.3億円という利益水準は：
  売上高が仮に¥250〜300億円（Q4）とすれば
  営業利益率 = 8.8〜10.5%（かなり高水準）

```

### 6-1. 前提条件の設定

#### ■ リスクフリーレート（Rf）【固定・変更禁止】
```

#### ■ β（ベータ）
```

  参考：しまむら(8227) β≒0.7〜0.8
  参考：バロックジャパン(3548) β≒1.0〜1.2

  中小型×業績ボラティリティ→ β = 0.85（採用値）
```

#### ■ ERP（株式リスクプレミアム）
```

採用値：ERP = 5.5%（Damodaran日本株基準 / 2026年推定）
```

#### ■ 株主資本コスト（Re）
```
Re = Rf + β × ERP
   = 2.386% + 0.85 × 5.5%
   = 2.386% + 4.675%
   = 7.061% ≒ 7.06%
```

#### ■ 負債コスト（Rd）と資本構成
```
アパレル中堅企業の資本構成推定：
  有利子負債比率：推定20〜30%（設備投資は少ないが運転資本で借入）
  D/(D+E) = 25%（採用）
  E/(D+E) = 75%（採用）
  
負債コスト：
  Rd = 1.5%（国内銀行借入・低金利環境）
  実効税率 Tc = 30%（日本法人標準）
  税引後負債コスト = 1.5% × (1-30%) = 1.05%
```

#### ■ WACC計算
```
WACC = E/(D+E) × Re + D/(D+E) × Rd × (1-Tc)
     = 0.75 × 7.061% + 0.25 × 1.5% × 0.70
     = 5.296% + 0.263%
     = 5.559% ≒ 5.56%
```

#### ■ ターミナル成長率（g）
```

アパレル業界成長率：0.5〜1.5%（EC拡大で若干上振れ）
