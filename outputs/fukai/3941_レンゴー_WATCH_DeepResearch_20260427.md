**Rationale（修正前比較）**：Q3累計（4-12月）経常¥33,714M達成率84.3%の段階で、Q4にドイツ子会社減損189億円を集中計上。経常利益の会社計画比達成率が当初84%だった状況を考慮すると、国内事業自体は概ね計画通り推移していたと解釈できる。

### 下方修正の要因分析

**メイン要因（欧州・一過性）**
- ドイツ子会社の事業計画見直しによる**減損損失189億円**を特別損失計上
- 欧州自動車産業の低迷→重包装（ヘビーデューティー段ボール）事業の採算悪化
- 同事業はドイツ・欧州拠点が自動車・機械部品輸送向けに特化

**オフセット要因（一過性プラス）**
- 湘南工場敷地の一部収用補償金：**特別利益148億円**
- 政策保有株売却益：**特別利益117億円**
- 合計特別利益265億円が減損189億円を上回るが、純利益水準では複合影響

**国内事業の実態**：営業利益は370億円（▲7.5%）と修正されたが、Q1-Q3累計OP¥33,004Mに対しQ4目標¥4,000M追加（合計¥37,000M）は達成可能水準。コスト増（古紙・電力・労務）の一方、2025年10月値上げ浸透が徐々に効いている。

**判断**：下方修正の本質は欧州の会計的一過性減損であり、国内セグメントの構造劣化ではない。FY2027以降の正常化が視野に入る。

## 3. 紙容器需要増・ナフサ代替テーゼ（コアテーゼ定量分析）

### 3-1. レンゴーのビジネスポジション

- **段ボール国内シェア：約30%（業界首位）**（Rationale：楽天証券トウシル記事・日経ビジネス記事から確認）
- 事業構成：板紙・段ボール原紙→段ボール製品・紙器製品（主力）、軟包装製品（フィルム・ラベル）、重袋・その他包装
- 連結売上約¥1兆円、国内外200以上の工場

### 3-2. 代替需要の試算

**プラスチック容器市場規模（国内参考）**：
- 日本のプラスチック包装材市場：約¥3〜4兆円規模
- 食品・飲料向け容器（トレー・ボトル・フィルム）が中心

**ナフサ危機のシフト圧力**：
- 合成樹脂価格3割高騰（ポリエチレン・ポリプロピレン）
- 食品容器メーカー4割値上げ→食品メーカーが代替を検討
- EU規則（PPWR 2026年8月適用）：プラスチック輸送包装の6%が段ボールへ移行済み
- 日本でも「プリン休売・乳酸菌飲料パッケージ印字省略」等の実例が出始めている

**レンゴーへの増収インパクト試算**：

| シナリオ | 前提 | 増収インパクト |
|---------|------|-------------|
| **Base（+1.5%シフト）** | 国内プラ包材市場の1.5%が紙・段ボールへシフト、レンゴーシェア30%捕捉 | 約¥135〜180億円 |
| **Bull（+3%シフト）** | ナフサ危機長期化で加速・同30%捕捉 | 約¥270〜360億円 |
| **Bear（+0.5%シフト）** | 夏以降にナフサ代替ナフサ供給で解消 | 約¥45〜60億円 |

Rationale：国内プラ包材¥3兆円×シフト率×レンゴーシェア30%で算出。段ボール売価単価・粗利率（推定15〜20%）を適用するとOP寄与はBase+¥20〜36億円程度。

**タイムライン**：
- 2026年5月〜：食品・飲料メーカーがQ4決算前後に容器切替検討を具体化
- 2026年7〜9月：ナフサ高止まり継続なら発注実績として顕在化
- 2026年10〜12月：レンゴーQ2実績で確認可能（最短確認タイミング）

### 3-3. 2025年10月値上げの効果

- 2025年10月1日納品分より段ボール原紙等+10円/kg以上値上げ（2022年9月以来ほぼ3年ぶり）
- Q4以降に業績への本格寄与が見込まれるが、コスト高（古紙・電力・労務）との綱引き継続

## 4. コスト構造・感応度分析

### 主要コスト項目

| コスト項目 | 状況 | レンゴーへの影響 |
|-----------|------|--------------|
| **古紙価格** | 上昇基調継続。中国向け輸出増で国内需給タイト | 主原料の30〜40%コストを占める。¥1/kg上昇でOP約▲¥10〜15億円（推計） |
| **電力コスト** | 原油・LNG連動で高止まり | 製紙は電力多消費（売上比5〜8%）。ホルムズ継続なら逆風継続 |
| **労務費** | 賃上げ圧力継続・人手不足 | 固定費増要因。10月値上げで転嫁中 |
| **物流コスト** | 2024年物流改革以降コスト上昇 | トラック不足・燃料高ダブルパンチ |

**感応度分析（OP影響）**
- 古紙+¥1,000/t → OP▲約¥20〜30億円
- 電力+10% → OP▲約¥10〜15億円
- 段ボール原紙+¥1/kg値上げ実現 → OP+約¥20〜25億円

**判断**：ナフサ代替需要増×10月値上げ浸透 vs 古紙・電力・労務費の綱引き。FY2027にコスト高が一服すれば、値上げ効果が利益に落ちる「後ずれ利益化」が起きる構造。

## 6. 不動産含み益考察（恒久プロトコル）

### 含み益の推計

**工場・土地の規模**：
- レンゴーグループは国内外に200以上の工場を保有（海外含む）
- 国内主力工場：春日井（愛知）・尼崎（兵庫）・印西（千葉）・岡山等の大規模敷地
- 有価証券報告書ベースの土地簿価：推計¥1,000〜1,500億円（製紙大手水準として）

**湘南工場事例からの逆算**：
- 収用補償金148億円が計上→これは土地の一部であることを示唆
- 湘南（茅ヶ崎・平塚エリア）の工業地帯は再開発圧力高く路線価との乖離が大きい
- 仮に全国工場用地の簿価が路線価の30〜50%水準なら、含み益¥500〜1,000億円が潜在

**政策保有株式の含み益**：
- 売却益117億円は一部売却分
- 残存政策保有株は純資産比10%未満目標（現状推定15〜20%保有）→残余売却で追加¥100〜300億円期待
- Vision120で「600億円資産売却」の裏付けとしてリアリティがある

**NAV簡易試算**：

|------|--------|-----------|
| 純資産（BPS×株数） | ¥4,897億 | BPS¥1,975×248M株 |
| 不動産含み益 | +¥500〜1,000億 | 工場用地簿価vs路線価乖離の推計 |
| 政策保有株含み益 | +¥100〜300億 | 残存分推計 |
| ドイツ子会社減損リスク | ▲¥100〜200億（追加） | 欧州事業残存リスク |
| **修正NAV合計** | **¥5,297〜6,797億** | — |

| **現値対比安全域** | **▲40.7〜▲53.7%のディスカウント** | — |

## 9. 投資判断：「逆張り打診30%の好機か？」への明示回答
