|------------|-------|-----------|
| **Rf（リスクフリーレート）** | **2.380%** | JGB10Y（2026-04-06 社長確認値） |
| **ERP（エクイティリスクプレミアム）** | **5.5%** | 日本株ERP標準値（Damodaran 2025年推計） |
| **β（ベータ）** | **0.75** | 東洋製缶：ディフェンシブ製造業・低ボラティリティ推計 |
| **Re（株主資本コスト）** | **6.505%** | 2.380% + 0.75 × 5.5% |
| **D/E比（負債比率）** | **40%/60%** | 製造業一般・推計 |
| **Rd（負債コスト）** | **1.5%** | 銀行借入金利・社債水準 |
| **Tc（実効税率）** | **30%** | 日本法人税率相当 |
| **WACC** | **4.83%** | 下記計算 |
| **売上成長率（FCF成長率）g1: 1〜5年** | **3.0%** | アルミ転嫁+飲料缶需要安定 |
| **売上成長率 g2: 6〜10年** | **2.0%** | 成熟期移行 |
| **ターミナル成長率（g）** | **1.0%** | 日本経済長期成長・インフレ率 |
| **DCF計算期間** | **10年** | 標準的評価期間 |

#### WACC計算詳細

```
WACC計算:
─────────────────────────────────────────────────────────
Re = Rf + β × ERP
   = 2.380% + 0.75 × 5.5%
   = 2.380% + 4.125%
   = 6.505%

Rd × (1 - Tc) = 1.5% × (1 - 0.30) = 1.05%

WACC = E/(D+E) × Re + D/(D+E) × Rd × (1-Tc)
     = 0.60 × 6.505% + 0.40 × 1.05%
     = 3.903% + 0.420%
     = 4.323% ≈ 4.32%

※ 資本構成: E=60%, D=40%（製造業標準）
─────────────────────────────────────────────────────────
```

> ⚠️ **深井注記:** 実際のD/E比は有報負債明細で要確認。ここでは製造業標準値を採用。
