## 1. 企業概要・事業分析

### 1-1. 会社概要

日本航空（JAL）は1951年創業、2010年の会社更生法申請・再建を経て2012年に再上場した日本最大級の航空会社グループ。ANAホールディングス（9202）と並ぶ国内2大キャリアの一角を占める。

### 1-2. セグメント構成（FY2025/3期推定）

```
売上高構成（推定）
┌─────────────────────────────────────────────┐
│ 航空事業（国際線）    ：約 45%  ←最大リスク源   │
│ 航空事業（国内線）    ：約 35%                  │
│ 非航空事業（物流等）  ：約 12%                  │
│ その他・グループ      ：約  8%                  │
└─────────────────────────────────────────────┘
```

**国際線比率：約45%**（ANA比：国際線約42%）

JALはANAと比較して**国際線売上比率がやや高く**、ジェット燃料コスト感応度は同等〜やや大。特に中東・欧州路線はホルムズ・紅海複合リスクの直撃を受ける路線構成となっている。

### 1-3. 主要路線ネットワーク（国際線）

| リスク区分 | 主要路線 | 備考 |
|-----------|---------|------|
| **高リスク** | 成田〜中東（ドバイ経由等） | ホルムズ海峡リスク直撃 |
| **中〜高リスク** | 成田〜欧州（中東上空通過） | 迂回コスト+燃料増大 |
| **中リスク** | 成田〜北米 | 需要堅調だが燃油高 |
| **低リスク** | 国内線全般 | 燃料コスト転嫁可能 |

### 1-4. 競合優位性と弱点

**強み：**
- 国際線プレミアムブランド（JGC会員基盤）
- 2010年再建時に債務圧縮し財務体質は相対的健全
- 羽田・成田スロット保有

**弱み（今局面）：**
- **燃料費が売上原価の約25〜28%を占める**（業界最大コストドライバー）
- 国際線依存度が高く、地政学リスクへの感応度大
- ヘッジ比率がANAより保守的（詳細後述）
- 2024年以降の機材老朽化更新投資でCAPEX増加局面

#### 深井の結論：**「ANA(9202)より空売り難易度は高いが、条件付きで売り可能」**

**売れる理由（BEAR CASE）：**
1. 燃料コスト急騰が業績下振れ圧力として継続
2. 国際線比率高く、中東・欧州便の運航コスト増が直撃
3. FY2026/3期の業績ガイダンス次第で大幅下方修正リスク
4. 現在株価は楽観シナリオを織り込んでいる可能性（下値余地▲15〜20%）

**売れない・売りにくい理由（BULL CASE）：**
1. **財務体質がANAより格段に優れており、燃料高への耐久力が高い**
2. **高配当利回り（3.5〜4%）が下値を支える**
3. PER8〜9倍は既に割安感があり、買い主体の参入余地
4. インバウンド需要継続で旅客収入が燃料高を部分相殺
5. 信用倍率がANAより売り難い構造

**深井推奨：**
> JALはANA(9202)の「Strong Sell(-95)」と同格には扱えない。財務体力の差が決定的。**JALは「Sell(-72〜-75相当）」**として、**ANAより小ポジション・タイトなストップで空売りすべき**。単独の主力空売りとしてではなく、**ANAとのペア・トレード（ANA売り/JAL売り比率 = 2:1）**が最適と判断する。

## 4. DCFバリュエーション

### 4-1. WACC算出前提

#### リスクフリーレート（厳守）
```
Rf = 2.380%（JGB10Y、2026-04-06 05:17 JST確認値）
```

#### パラメータ設定

| パラメータ | 値 | 根拠 |
|-----------|---|------|
| **Rf** | **2.380%** | JGB10Y（社長確認値、絶対厳守） |
| **β（ベータ）** | **1.25** | 航空セクター標準（JAL実績β推定：1.20〜1.30） |
| **ERP（株式リスクプレミアム）** | **6.0%** | 日本市場長期平均（Damodaran等参照） |
| **Re（株主資本コスト）** | **9.88%** | = 2.380% + 1.25 × 6.0% |
| **Rd（負債コスト）** | **1.8%** | 有利子負債加重平均金利（推定） |
| **Tc（実効税率）** | **30%** | 日本標準的実効税率 |
| **E/(D+E)（株式比率）** | **73%** | 自己資本/総資本比率推定 |
| **D/(D+E)（負債比率）** | **27%** | 上記補数 |

#### WACC計算

```
Re = 2.380% + 1.25 × 6.0% = 2.380% + 7.500% = 9.880%

WACC = [0.73 × 9.880%] + [0.27 × 1.80% × (1 - 0.30)]
     = 7.212% + 0.340%
     = 7.552%

→ WACC ≈ 7.55%（採用値）
```
