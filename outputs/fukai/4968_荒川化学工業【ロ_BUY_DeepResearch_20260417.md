**分散効果の結論**：ロジン系と石化系は一定の分散効果があるが、輸送コスト（原油連動）という共通因子があるため完全な逆相関ではない。**ナフサ急騰＋中国ロジン規制の同時発生は荒川化学にとって最大の毒シナリオ**（EPS▲¥40〜60の毀損リスク）。

### 2-6. 【⑥】バリュエーション評価——PBR0.43倍の「正当な割安度vs構造的ディスカウントの罠」

**化学セクター平均との比較**

| 指標 | 荒川化学 | 東証プライム化学平均 | 割引率 |
|------|---------|-------------------|-------|
| PBR | **0.43倍** | 0.8〜1.0倍 | **▲50〜57%** |
| PER | 14.7倍 | 15〜18倍 | ▲3〜18% |
| ROE | **0.4%** | 5〜8% | **壊滅的な乖離** |
| 配当利回り | 3.86% | 2.5〜3.0% | 高配当 |

**PBR0.43倍の乖離要因分析**

```
理論PBR = ROE ÷ 株主資本コスト（Re）

荒川化学の理論PBR：
ROE = 0.4%
Re（推定）= 2.409% + 0.8×5.5% = 6.8%
理論PBR = 0.4% ÷ 6.8% = 0.059倍（！！）

→ 理論上はROE=0.4%なら、PBR0.43倍でも「まだ割高」という残酷な計算が成立する
→ 「PBRが低いから割安」は間違い。ROEが極度に低い場合、低PBRは正当評価の可能性がある
```

**PBR0.43倍の判定**

| 判定軸 | 評価 | 根拠 |
|--------|------|------|
| ROEベースの理論PBR | ☠️ バリュートラップ疑い | ROE0.4%では理論PBRは0.06倍——0.43倍は理論上まだ割高 |
| BPS（解散価値）ベース | 💊 割安 | BPS¥2,987に対して¥1,289は57%ディスカウント |
| ナフサ高騰の織り込み度 | 中程度 | 2022年高騰時最安値水準に既に接近 |
| **総合判定** | **要注意：ROE改善なき割安は意味がない** | ROEが2〜3%に回復しない限り、PBR0.43倍は「正当な低評価」 |
