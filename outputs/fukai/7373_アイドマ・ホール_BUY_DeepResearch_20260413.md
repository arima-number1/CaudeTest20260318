---

### 2-1. なぜ決算が悪かったか ― 3つのシナリオ

```
【シナリオA：一過性コスト集中（確率: 40%）】
  原因: 採用費・システム投資が今期に集中計上
  含意: 来期から利益が急回復する可能性
  → この場合は「バリュー投資の好機」
  → 現在PER 10.6倍は割安

【シナリオB：成長鈍化の構造的始まり（確率: 40%）】
  原因: AI代替・競合激化でARR成長が天井打ち
  含意: 売上成長率が+15%→+8%以下に定着
  → この場合は「バリュートラップ」
  → PER 10.6倍は実は割高（成長プレミアム剥落）

【シナリオC：組み合わせ（確率: 20%）】
  原因: 短期コスト増 + 中期成長鈍化の複合
  含意: 一時的回復後、再度失望売りのリスク
  → 最も厄介なパターン
```

---

### 2-2. 「買えない」理由の定量的整理

| 判断根拠 | 詳細 | 深井評価 |
|---------|------|---------|
| **決算の質が不透明** | 利益減少の原因が構造的か一過性か未確定 | ⚠️ 最重要リスク |
| **AI代替リスクが可視化されつつある** | 営業代行の人的部分がAIに置き換わるトレンド | ⚠️ 構造的脅威 |
| **来期ガイダンスの信頼性低下** | 下方修正後のガイダンスは市場が懐疑的に見る | ⚠️ 信頼回復に時間 |
| **RSI 18.4の過売り** | テクニカルには「底打ち近し」の可能性 | 💊 逆張りチャンス示唆 |
| **PER 10.6倍の低バリュエーション** | グロース株として異例の割安水準 | 💊 下値余地は限定的 |

### 2-3. 社長への直言

> **「買えないかな」は正解ですが、理由は違うかもしれません。**

**正しい「買えない」理由：**
「決算が悪かった」ことよりも、**「その悪さが構造的かどうかの答えが出るまで待つべき」**という判断が正しい。

**誤解してはいけない点：**
- PER10.6倍・ROE26.7%は「安い株」の条件を満たしている
- RSI18.4の過売りは統計的に「反発確率が高い」ゾーン
- **ただし「安い理由がなくなる前提」が必要**

**推奨スタンス：**
```
📌 「待ちだが、捨てるな」

→ 次の四半期決算（約3ヶ月後）で
  ① 利益率回復の兆候 があれば：買い検討
  ② 再度の下方修正 があれば：完全撤退
  
→ 現時点での購入は「賭け」。
  社長の「まあ買えないかな」は
  リスク管理として適切な判断。
```

| パラメータ | 値 | 根拠 |

**WACC計算：**
```
E/(D+E) = 0.909, D/(D+E) = 0.091

WACC = [0.909 × 9.86%] + [0.091 × 0.8% × (1-0.30)]
     = 8.96% + 0.051%
     = 9.01% ≒ 9.0%（切り捨て処理）
```

> ※ ほぼ無借金のため、WACCはほぼ株主資本コスト（Re）に一致。

### 4-2. FCF予測（3シナリオ）

**基本前提：**
- 現在の純利益: ¥14〜16億円（EPS¥125より）
- FCF = 純利益 × FCFコンバージョン率 約85〜90%

#### ベースケース（基本シナリオ）

| 年度 | FCF（億円） | 成長率前提 |
|------|-----------|---------|
| FY2026 | 14.0 | 利益率正常化で回復 |
| FY2027 | 16.5 | +18%（採用費正常化） |
| FY2028 | 19.0 | +15% |
| FY2029 | 21.5 | +13% |
| FY2030 | 23.5 | +9% |
| **ターミナル** | **g=2.0%** | WACC=9.0% |

**DCF計算（ベースケース）：**

```
割引率: WACC = 9.0%
ターミナルバリュー = FCF2030 × (1+g) / (WACC-g)
                  = 23.5 × 1.02 / (0.09 - 0.02)
                  = 23.97 / 0.07
                  = 342.4億円

PV（ターミナル）= 342.4 / (1.09)^5 = 342.4 / 1.539 = 222.5億円

PV（FCF5年計）:
  FY2026: 14.0 / 1.09 = 12.84
  FY2027: 16.5 / 1.188 = 13.89
  FY2028: 19.0 / 1.295 = 14.67
  FY2029: 21.5 / 1.411 = 15.24
  FY2030: 23.5 / 1.539 = 15.27
  合計 = 71.9億円

企業価値 = 71.9 + 222.5 + ネットキャッシュ12.5 = 306.9億円

発行済株式数 ≒ 1,200万株（BPS¥485・純資産58億円より逆算）

理論株価（ベース） = 306.9億円 ÷ 1,200万株

```

## 9. 金利感応度分析（Sensitivity Analysis）
