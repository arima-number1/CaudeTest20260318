## セグメント別分析

### スペース事業（成長） — 投資対象の本体

主要サービス：

- **固定衛星通信（FSS）**：放送局・通信企業への衛星帯域リース（Ku/Ka/C帯）
- **防衛・安全保障通信**：自衛隊向けX帯衛星通信（最機密扱い、開示限定）
- **宇宙状況監視（SSA）**：内閣府宇宙政策委員会との国家プロジェクト
- **海事VSAT**：船舶・海上基地の通信インフラ

**競争優位（取替不可能性）：**

- GEO衛星軌道スロット（経度位置）は物理的限界財。日本企業中最多保有
- 一度占有したスロットは法的に保護（ITU国際電気通信連合）
- JSAT衛星フリートは防衛用途を含む多目的設計 → 代替不能な国防インフラ

**Capex急増の正体：**

```
FY2022/3: ¥83.8億
FY2023/3: ¥174.2億 (+108% YoY) ← 次世代衛星打上開始
FY2024/3: ¥153.8億
FY2025/3: ¥239.5億 (Capex/EBITDA = 50%) ← 防衛対応新衛星建造
```

このCapex急増は単なる更新投資ではなく、防衛需要に対応した**新規衛星能力増強**の可能性が高い。軌道に乗った衛星は15-20年間無形の高マージン収益を生成。

### メディア事業（衰退） — 構造的衰退だが安定キャッシュ

SKYPerfecTV!の現状：

- 契約者数：約390万件（推定、減少傾向）
- OTT（Netflix・Amazon・DAZN）に継続侵食
- ただし高齢者層＋スポーツコンテンツの根強い需要
- 衰退速度：年▲2-4%程度（急崩壊ではない）

> メディアは「スペースへの転換コストを賄うキャッシュカウ」として機能

### 不動産含み益考察（NAV調整）

- 土地帳簿¥38.8億 → 地上局施設・本社（東京・港区）
- 港区・赤坂の不動産時価との差異：推計+¥20-40億（限定的）
- 真のNAVは有形固定資産内の**衛星自体の価値**が核心
- 衛星1基の軌道投入コスト：数百億円（¥200-400億/基）
- 軌道スロットの「独占的フランチャイズ価値」は帳簿外
- **調整BPS：¥1,050-1,100/株**（本質は衛星スロットの独占的フランチャイズ価値）

## TP算出根拠

### Base TP ¥4,100（+28.3%）

```
EV/EBITDA 22x × FY2027/3E EBITDA ¥520億 = EV ¥11,440億
+ ネットキャッシュ ¥234億 = Equity ¥11,674億
÷ 283.4M株 = ¥4,119/株 → ¥4,100

根拠: 防衛契約フル稼働+メディア安定でEV/EBITDA 22xは正当化可能
```

### Bull TP ¥5,200（+62.8%）

防衛省との超長期（10-20年）契約が開示 → DCF的フランチャイズ価値認識。EV/EBITDA 25x × ¥560億（防衛拡大） = EV ¥14,000 → 株価¥5,176

メディア契約者急減（年▲10%以上）でEBITDA¥420億以下。EV/EBITDA 14x = EV ¥5,880 → 株価¥2,154

## リスクシナリオ定量化

**シナリオ1（メイン55%）：防衛契約フル展開×メディア緩慢衰退**

- FY2027/3 OP ¥300-310億（+10-13%）
- 株価→TP ¥4,100（Base TP達成）

**シナリオ2（弱気20%）：FY2027/3ガイダンス保守的（NI¥190-200億）**

- コンセンサスFWD NI¥198億が事実→市場失望売り
- 現在PER 45.7x（FWD）は持続不可能
- 株価→¥2,500-2,800（PER 30-35x+ダウングレード）

**シナリオ3（強気25%）：防衛10-20年長期契約開示**

- DCF価値が市場認識される（衛星耐用年数15年×防衛収益）
- 株価→¥5,200（Bull TP）

### 最大の懸念：PBR-ROE乖離

```
現在PBR 3.19x を正当化するには ROE 15-20%が必要
現在ROE  7.9% → 理論的適正PBR = 0.8-1.0x

現在株価のPBR 3.19xは「将来ROEの急拡大」を完全織り込み済み
防衛収益が実現しないと PBR 3x は持続不可能
→ 最大リスクはROEが上がらないことへの見切り売り
```

- **コンセンサスFWD NI ¥198億**（FY2027/3）はFY2026/3推計¥220億を下回る
