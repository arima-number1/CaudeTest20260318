## 2. 社長指示の重点分析｜「電気代上昇が気になります」

### 2-1. JR東海の電力消費規模

東海道新幹線は日本の鉄道インフラ中、**最大の電力消費体**の一つ。その特性を定量的に把握する。

```
【電力消費の特性】
- 東海道新幹線（東京-新大阪）全長552.6km
- 運行速度：最高285km/h（のぞみ）
- 空気抵抗（∝速度の3乗）：高速運転は電力消費が急増
- 推定年間電力消費量：約70〜80億kWh（業界推計ベース）
- 電力調達：電力会社からの購入＋自家発電（一部）
- 電力費はJR東海の**最大固定コスト項目の一つ**（年間1,000〜1,200億円規模と推定）
```

### 2-2. LNG高騰→電力価格への伝播メカニズム

```
ホルムズ海峡緊張

LNG輸送障害・スポット価格急騰

 日本向けアジアLNGスポットは別建て。JKM（Japan Korea Marker）が
 Henry Hubの3〜4倍水準で推移する場合あり）
```

## 4. DCFバリュエーション

### 4-1. 前提条件（全項目明示）

```
【割引率前提】
- リスクフリーレート（Rf）：2.423%（JGB10Y・2026-04-15・財務省）【絶対使用】
- 株式リスクプレミアム（ERP）：5.50%（日本市場標準・Damodaran 2026参照）
- β（ベータ）：0.70（インフラ公益性・規制業種・景気安定性を考慮。鉄道業界標準）
- 株主資本コスト（Re）：Rf + β × ERP = 2.423% + 0.70 × 5.50% = 2.423% + 3.850% = 6.273%

【WACC計算】
- 資本構成：E/（D+E）= 40%、D/（D+E）= 60%
  （有利子負債約5.4兆円・時価総額約3.1兆円より算出）
- 負債コスト（Rd）：2.20%（JR東海格付AA-相当・社債利回り参照）
- 実効税率（Tc）：30%
- WACC = [0.40 × 6.273%] + [0.60 × 2.20% × (1-0.30)]
        = 2.509% + 0.924%
        = **3.433%**

【FCF前提】
- ベースFCF（FY25E）：約2,500億円
  （営業CF約7,000億円 - リニアCAPEX約4,500億円）
- 成長率フェーズ1（FY26-30）：4.5%（加算料金効果・インバウンド・個室効果）
- 成長率フェーズ2（FY31-35）：3.0%（リニア開業後の在来新幹線収益安定化）
- 永続成長率（g）：1.5%（名目GDP成長率＋インフレ。日本インフラ事業の上限）
- DCF対象期間：10年（FY26〜FY35）+ ターミナルバリュー
```

### 4-2. DCF計算（ベースシナリオ）

| 年度 | FCF（億円） | 割引係数（WACC=3.433%） | PV（億円） |
|------|-----------|----------------------|----------|
| FY26 | 2,613 | 0.9668 | 2,526 |
| FY27 | 2,730 | 0.9348 | 2,553 |
| FY28 | 2,854 | 0.9037 | 2,580 |
| FY29 | 2,983 | 0.8736 | 2,607 |
| FY30 | 3,117 | 0.8445 | 2,632 |
| FY31 | 3,211 | 0.8163 | 2,621 |
| FY32 | 3,307 | 0.7890 | 2,609 |
| FY33 | 3,406 | 0.7625 | 2,597 |
| FY34 | 3,508 | 0.7368 | 2,584 |
| FY35 | 3,613 | 0.7120 | 2,572 |
| **合計PV（FCF）** | | | **25,881** |

```
ターミナルバリュー（TV）計算：
TV = FCF_FY35 × (1+g) / (WACC - g)
   = 3,613 × 1.015 / (3.433% - 1.500%)
   = 3,667 / 1.933%
   = 189,706億円

TV現在価値：189,706 × 0.7120 = 135,070億円

企業価値（EV）= FCF合計PV + TV現在価値
             = 25,881 + 135,070
             = 160,951億円

ネット有利子負債：▲54,000億円（推定）
株主価値 = 160,951 - 54,000 = 106,951億円

発行株式数：約7.1億株（時価総額÷株価より逆算）
理論株価（DCFベース）= 106,951億円 ÷ 7.1億株
                    = 約¥15,063
```

> ⚠️ **DCF感応度注記（重要）**：上記DCFのTV部分はWACC-g差（スプレッド1.933%）に極めて敏感。このスプレッドが±0.5%変動すると理論株価が±30〜40%変動する性質あり。以下ではシナリオ別感応度を明示する。

### 4-3. バリュエーション整合性チェック（PER・PBRアプローチ）

| 手法 | 理論株価 | 根拠 |
|------|---------|------|
| DCF（ベース） | ¥15,063 | 上記計算（FCF割引） |
| PERアプローチ | ¥5,598 | EPS ¥533 × 同業平均PER 10.5倍 |
