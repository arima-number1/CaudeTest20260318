## ③ DCFによるFairValue算出と割安・割高判定

**（詳細はセクション4で展開。ここでは社長指示への直接回答として要約を提示）**

### PBR・ROE・成長率による簡易バリュエーション

**Gordon Growth Model（簡易PBR適正値計算）：**

```
理論PBR = ROE / (Re - g)

前提：
  ROE = 13.5%（リアルタイム取得値）
  Re（株主資本コスト）= 2.411% + 0.65 × 5.5% = 5.99%（後述WACCセクションで詳細計算）
  g（持続成長率）= 3.5%（基本ケース）

理論PBR = 13.5% / (5.99% - 3.5%) = 13.5% / 2.49% = 5.42倍

現在PBR = 2.35倍

→ Gordon Growthモデル上では「大幅割安」
```

**ただし重要な注意点：** Gordon Growth Modelは成長率gの仮定に極めて敏感。g=2.5%に下げると：

```
理論PBR = 13.5% / (5.99% - 2.5%) = 13.5% / 3.49% = 3.87倍 → まだ割安
g=1.5%なら：13.5% / 4.49% = 3.01倍 → 割安
g=0.5%なら：13.5% / 5.49% = 2.46倍 → ほぼ適正
```

**深井判定：** PBR 2.35倍×ROE 13.5%の組み合わせは、**成長率が2%を超える限りにおいて割安〜適正圏**。月次+16.2%が示す成長モメンタムを勘案すれば、現在株価¥2,254は**軽度の割安圏**に位置する。ただし「大幅割安」とは言えない。

**【発行株数の再確認と調整】**

```

  EPS = ¥122 → 純利益 ÷ 株数 = ¥122

  純利益 = EPS × 発行株数 = ¥122 × 発行株数

  18.5 = ¥2,254 / ¥122 = 18.47 ≈ 18.5 ✅（整合性確認）

→ 発行株数は比率計算で消えるため、
  「一株当たりFCF」ベースで理論株価を再計算する
```
