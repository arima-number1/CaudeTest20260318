## 3. 事業モデル・競争優位

### 半導体ファイナルテスト（OSAT）専業。日本国内最大規模

テラプローブはOSAT（Outsourced Semiconductor Assembly and Test）。ウエハテスト・ファイナルテストの受託実行と、テストプログラム開発を提供。製品・プロセスは自社で持たず、半導体メーカーのテスト工程を代行する「テストハウス」モデル。

| 拠点 | 内容 |
|------|------|
| **九州事業所（日本）** | ロジック・車載向け中心。国内最大規模テストハウス |
| **TeraPower Technology（台湾・新竹）** | AI・サーバー向け高付加価値テスト。AI需要急増の主戦場 |

### 売上構成（FY2025/12期）

| セグメント | 比率 | 成長性 |
|-----------|------|--------|
| **ロジック半導体** | **93%** | 車載EV×AIで急拡大 |
| **メモリ半導体** | 7% | 安定（ミックスドシグナル等） |
| **うち車載向け** | **約44%** | 中期目標：国内50%・TPW40% |
| **うちAI・サーバー向け** | 成長中 | FY2025でYoY+170%（約¥65億増加）|

### 競争優位の3層構造

1. **テストプログラムの蓄積**: 顧客別・製品別テストプログラムは長年の積み上げで代替困難。切替コストが極めて高い
2. **AI向け高単価構造**: AIサーバー向けテストタイムは車載向けの3〜4倍。同じ設備で高いスループット収益を実現
3. **PTIグループのターンキー連携**: 台湾PTIと連携してウエハテスト→後工程まで一貫受託が可能。国内OSATで希少

### 手法②：DCF法（ウェイト30%）

| パラメータ | 数値 | 根拠 |
|-----------|------|------|
| WACC | **9.4%** | Rf 2.49% + β1.2×ERP 5.5% + サイズ0.3% |
| Terminal growth（g） | **2.5%** | OSAT成熟後の安定成長 |
| 明示期間 | 5年 | FY2026〜FY2030 |

| 年 | FCF（推計） | PV |
|----|-----------|-----|
| FY2026 | ¥55億 | ¥50.3億 |
| FY2027 | ¥65億（台湾工場稼働前） | ¥54.3億 |
| FY2028 | ¥80億（台湾+九州稼働） | ¥60.9億 |
| FY2029 | ¥95億 | ¥65.8億 |
| FY2030 | ¥110億 | ¥69.3億 |
| **PV合計** | | **¥300.6億** |

Terminal Value: ¥110億 × 1.025 / (0.094 − 0.025) = ¥1,634億
PV TV: ¥1,634億 / 1.094⁵ = **¥1,030億**

EV = ¥300.6億 + ¥1,030億 = **¥1,330億**
Net Debt推計: ¥100億（有利子負債¥200億推計 − 現金¥100億推計）
Equity Value = ¥1,330億 − ¥100億 = **¥1,230億**
**Per share: ¥1,230億 / 910万株 = ¥13,516**

> **根拠Rationale**: FCFは設備投資（¥137億円進行中）控除後の保守値。FY2028以降は台湾・九州両工場フル稼働でOP急拡大を反映。WACC 9.4%はシリコンサイクルリスク・地政学（台湾リスク）・集中顧客リスクを織り込んだ高め設定

### 手法③：EV/EBITDA法（ウェイト30%）

- FY2026 EBITDA推計: OP¥105億 + D&A¥30億 = **¥135億**
- Target EV/EBITDA: **8.5x**（OSAT高成長株のプレミアム水準）
- Target EV: ¥135億 × 8.5 = **¥1,147.5億**
- Net Debt控除: ¥1,147.5億 − ¥100億 = **¥1,047.5億**
- Per share: ¥1,047.5億 / 910万株 = **¥11,511**

> **根拠Rationale**: OSAT大手（ASE/Amkor）のEV/EBITDA 5〜7xに対し、高成長・高OPM・国内希少性のプレミアムで8.5x採用。台湾有事リスクを考慮し9〜10xより保守的設定

## 10. 不動産含み益考察（全DR必須）

テラプローブは製造業（OSAT）であり、工場・土地の含み益が存在する可能性がある。

| 資産 | 内容 | 含み益評価 |
|------|------|----------|
| 九州事業所（土地・建物） | 半導体テスト工場。築年数・取得価格不明 | 工業地帯のため地価上昇限定的。**含み益は軽微** |
| 半導体テスト装置（テスタ） | 主要資産。簿価vs市場価値 | テスタは専用機で転売価値低い。**含み益なし** |
| 台湾TPW設備 | 台湾子会社保有 | 台湾不動産の含み益は連結では限定的 |
| **実質的「含み益」** | **顧客別テストプログラムの簿外価値** | テスタに蓄積されたプログラム・ノウハウは簿価に反映されない無形資産。顧客切替コストの源泉 |

**結論**: テラプローブの最大の「隠れ資産」は有形固定資産の含み益ではなく、長年蓄積された**顧客別テストプログラムと技術ノウハウ（無形の参入障壁）**。これはバランスシートに計上されていないが、顧客離脱を阻止する最大の競争優位。
