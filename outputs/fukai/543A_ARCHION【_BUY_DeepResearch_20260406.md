```
■ BPS × PBR = 理論株価チェック
  ¥375 × 1.26 = ¥472.5 ≒ ¥471 ✅ 整合

■ EPS × PER = 理論株価チェック
  ¥42 × 11.1 = ¥466.2 ≒ ¥471 ✅ 概ね整合（端数差）

■ ROE = EPS / BPS チェック
  ¥42 / ¥375 = 11.2% ← ⚠️ yfinance表示ROE 35.4%と乖離大

【重要な矛盾点】
  単純計算: ROE = EPS/BPS = ¥42/¥375 = 11.2%
  yfinance表示: ROE = 35.4%

  この乖離の解釈:
  (A) yfinanceのROEは期首BPSベース（BPSが期中に大幅増加した可能性）
      → 期首BPS = ¥42 / 0.354 = ¥118.6
      → 統合初年度でBPSが¥119→¥375に急増した可能性

      → 実際の純利益ベースROEは35.4%が正しい
      → この場合の実質EPS = ¥375 × 0.354 = ¥132.75
      → 見かけのEPS¥42との乖離 = ¥90.75/株が一過性利益

  ⚠️ 深井判断: 解釈(A)が最も整合的
     → 統合時の資産再評価・売却益でBPSが急膨張
     → EPS¥42は経常的な収益力を示す（ROE 11%水準）
     → 35.4%ROEは「統合バブル」の可能性あり
     → バリュエーションにはROE 11%ベースを保守的シナリオとして使用
```

## 4. 重点分析②：希薄化リスクの定量化

### 4-1. 希薄化のメカニズム

```
■ 現状
  浮動株比率: 17.1%
  目標浮動株比率: 50%（東証プライム基準との整合・流動性確保目的）

  (1) 親会社（トヨタ・ダイムラートラック）による持株売出し

  (3) 段階的オファリング（クォータリー）

■ 株式供給量の試算
  浮動株差分: 50% - 17.1% = 32.9%
  発行済株式数推計（EPS¥42、純利益規模から逆算）:
    → 純利益 = EPS × 株式数
    → 売上3.5兆円 × ROE実態11% × 純利益率3%推定 ≈ 1,050億円

    25億株 × 32.9% = **8.2億株**（市場への売出し相当）

    8.2億株 × ¥471 = **約3,862億円**の供給圧力
```
