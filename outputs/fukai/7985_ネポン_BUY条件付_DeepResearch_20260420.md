### 6-4. DCF（正常化・保守的）

**前提**:
- 正常化FCFF: ¥1.0億/年（FY2022-2023平均FCFベース）
- WACC: 8.5%（Rf 2.38% + β 0.137×ERP 5.50% + サイズプレミアム 5.0%）
- ターミナル成長率: 0%（農業人口減少懸念）

```
DCF FV (Equity) = FCFF / WACC × 事業価値 - 純負債
= ¥1.0億 / 0.085 = ¥11.8億（事業EV）
= ¥11.8 - ¥13.3 = ▲¥1.5億（エクイティ価値）

→ DCF上は理論的にエクイティ価値がゼロ付近！
```

**⚠️ DCF分析の意味**: 現在のFCF水準（¥0.38億/年）では純負債返済に数十年かかる。FCFが正常化（¥1.5〜2.0億）しなければ、DCFベースでの株式価値は乏しい。**本案件はDCFではなく「強制売り後の需給回復」と「P/B底値圏からの反発」で判断すべき。**
