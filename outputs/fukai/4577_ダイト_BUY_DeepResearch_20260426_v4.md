### Q6詳細分析：「最大出資者（1/3超・過半未満）」の意味

**IR高畠氏 原文：**
> リリース p2 L4 に記載のとおり、本件は3社による出資で、当社が過半とならない最大出資者となりますので、必然的に当社の出資比率は 1/3 以上となります。

**含意の分解：**
```
確定事項:
  ① ダイトの出資比率 = 1/3超（「1/3以上」という表現は「1/3より多い」を意味）
  ② ダイトが3社中で最大の出資者
  ③ ダイトは過半（50%超）を持たない → 支配はしないが最大シェアを持つ

推定される出資構造:
  ダイト:          40〜49%（最大・過半未満）
  明治Seikaファルマ: 26〜30%（2位推定）
  杏林製薬:        22〜30%（3位推定）
  ※3社合計=100%、ダイトが最大で他2社よりも多い

v3との変化:
  v3推計「1/3〜1/2（均等分割仮定）」→「1/3超・最大（より高い可能性）」
  → EPS影響の下限が上昇: +¥5〜11 → +¥6〜11（中間値+¥8.5）
```

**財務インパクト更新（v4）：**
```
持分比率の最善推計: 40〜49%（v3の1/3〜1/2から上限寄りに修正）
JV年間売上（全体推計）: ¥80億/年（変更なし）

ダイト持分売上（更新）:
  ¥80億 × 40%〜49% = ¥32〜39億/年（v3: ¥27〜40億 → 下限上昇）
  中間値: ¥35億/年（v3: ¥33億 → 微増）

OPM（統合後）: 8〜12%（変更なし）
ダイト持分OP: ¥2.6〜4.7億/年
税引後NI: ¥1.7〜3.1億/年
EPS影響: +¥6〜11/株（FY2028〜）← v3の+¥5〜11から下限引き上げ
```

### Q7詳細分析：「JVスキームを前提にコンソ活動」の戦略的含意

**IR高畠氏 原文（重要部分）：**
> このような体制を築いた以上、新コンソの活動においては、当社は、本件スキームの枠組みを前提として行うことになるかと思われます。

**この一文の戦略的含意：**

```
表面的な事実:
  JV（医薬品共創機構）とコンソーシアムは独立した活動

しかし実態は:
  ダイトはコンソーシアムの活動においてJVスキームを前提とする
  ↓
  ①コンソーシアム9社がダイト-JV枠組みを「参照モデル」として活用可能
  ②将来的に他のコンソ参加企業との資本関係構築への入り口
  ③「JV第1号=ダイト×明治×杏林」が成功すれば、第2号・第3号への布石

v4で強気シナリオの確率引き上げ根拠:
  v3強気TP¥2,400「M&A第2弾（コンソーシアム企業の資本統合）」
  → その布石となる枠組みがIRで確認された
```
