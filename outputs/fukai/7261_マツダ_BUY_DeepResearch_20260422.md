## ■ ビジネスモデル解剖

### 事業概要
マツダは広島を本拠とする独立系自動車メーカー。SUVを中核とした「鼓動デザイン」×「人馬一体」ブランドで差別化。トヨタと資本関係（3.5%持合）。

### 収益構造（FY3/26推定）
| 地域 | 販売比率 | 特徴 |
|---|---|---|
| 北米（米国）| **最大シェア約40%** | CX-5/CX-50中心、アラバマ・メキシコ工場補完 |
| 日本 | 約15% | 軽自動車スズキ・ダイハツに押され縮小傾向 |
| 欧州 | 約15% | CX-60 PHEVで存在感 |

| アジア・その他 | 約18% | 東南アジアが成長 |

|---|---|---|---|

## ■ 将来の関税再設定リスク（Rev.2追加）

> **深井さんのご指摘：「将来についてまた同種の関税が設定されるのではないか」— マツダはSUBARUよりネット現金が少ない分、この問いへの答えはより重要。**

### 関税再設定の法的・政治的経路

| リスク経路 | 発動主体 | 浅井確率 | MAZDA影響額 | SUBARU比較 |
|---|---|---|---|---|
| **IEEPA同スキーム即時再発動** | 大統領令 | **5%** | ¥120-130B | 同等 |
| **Section 232 自動車25%（最大リスク）** | 大統領（商務省勧告済み） | **30-40%** | **¥120-150B** | 同等 |
| **議会立法（Trade Act改正）** | 上下院 + 大統領 | **10-15%** | 恒久的 | 同等 |
| **日米通商協定による恒久撤廃** | 日米政府 | **40%** | ポジティブ | 同等 |

### マツダ特有の脆弱性：SUBARU比較

```
SUBARUの場合（関税再設定されても）：
  ネット現金¥1.4T = 市場価値の76%
  → EV/EBITDA 1.1x = 清算価値近傍
  → 最悪シナリオFV ¥2,800 > 現値 ¥2,570（下値限定）

マツダの場合（関税再設定されたら）：
  ネット現金 ≈ ほぼ中立（D/E 48%はファイナンス含む。自動車単体は概ね中立）
  → キャッシュバッファーが薄い分、EPS感応度が高い
  → Section 232 25%再発動なら FY3/27 OP ¥130-150B程度に後退
  → EPS ¥140-160 → P/E6.5x → FV ¥910-1,040

Section 232 発動シナリオでのマツダFV推計：
  OP ¥80-100B → EPS ¥82-100 → FV ¥700-800（P/E 8x = ROE懸念分プレミアム）
  → 現値 ¥1,082 に対して大幅な下値リスク
```

**→ マツダはSUBARUと異なり、「最悪シナリオでも現値以上」の保証がない。この非対称性がBUY止まり（STRONG BUYでない）の核心。**

### シナリオ確率の改訂（Rev.2）

| シナリオ | 旧確率 | **改訂確率** | FV | 変更理由 |
|---|---|---|---|---|
| **Bear（関税再設定・中国悪化）** | 25% | **35%** | ¥900 | Section 232自動車リスク＋中国構造問題を明示的に上乗せ |
| **Base（IEEPA撤廃継続・安定運営）** | 55% | **45%** | ¥1,800 | Bear確率増加分を移管 |
| **Bull（日米通商協定＋CX-5ヒット）** | 20% | **20%** | ¥2,600 | 変更なし |

**確率加重FV（Rev.2）：**
0.35 × ¥900 + 0.45 × ¥1,800 + 0.20 × ¥2,600 = ¥315 + ¥810 + ¥520 = **¥1,645**
→ **改訂TP：¥1,600**（保守的丸め。旧TP¥1,700から▲¥100 = 関税再設定リスクプレミアム）

**改訂アップサイド：¥1,600 / ¥1,082 - 1 = +47.9%（旧+57.1%から修正）**

## ■ バリュエーション（4手法）

### 前提パラメーター
| パラメーター | 値 | 根拠 |
|---|---|---|
| Rf（無リスク金利） | **2.397%** | Bloomberg JGB 10年（2026/04/22） |
| ERP | 6.0% | 竹村TC標準 |
| β | 1.45 | Bloomberg実績値 |
| **Ke** | **11.10%** | 2.397% + 1.45×6.0% = 11.097% |
| BPS | ¥2,847 | PBR0.38×株価¥1,082より逆算 |
| FY3/27E OP（本レポート） | ¥250B（中央値） | IEEPA後 |
| FY3/27E EPS | ¥275 | OP¥250B×税引後率65%÷631M株 |

### 手法①：PERバリュエーション
- FY3/27E EPS: ¥275
- 適正P/E: **6.5x**（低ROE・高β・中国リスク考慮、同業Honda 7x・SUBARU 9xを割引）
- **FV: ¥275 × 6.5 = ¥1,788 → ¥1,800**

### 手法②：Justified P/Bバリュエーション
- 正常化ROE（FY3/27）: **8.0%**（関税前FY3/24実績7.1%OPMをベースに試算、中国縮退で保守的設定）
- Justified P/B = ROE / Ke = 8.0% / 11.10% = **0.72x**
- FV = 0.72 × BPS ¥2,847 = **¥2,050**
- *注：現在ROE 0.49%はIEEPA関税の一時的歪み。正常化ROE8%でも同業比低水準（SUBARU 12%+）を反映してKe未調整。*

### 手法③：DCFバリュエーション
| 仮定 | 値 |
|---|---|
| FY3/27E FCF | ¥100B/年（OP ¥250B × 利益率 - Capex ¥120B） |
| WACC | 9.0%（Ke11.1% × 80% + Kd3% × 20%） |
| 永続成長率 | 0.5%（電動化移行リスク反映） |
| TV = FCF/(WACC-g) | ¥100B / (9.0%-0.5%) = ¥1,176B |
| EV | ¥1,176B |
| Net Debt | ≈ ¥0B（オートモーティブ単体） |
| 株数 | 631M株 |
| **FV（DCF）** | **¥1,863** |

### 手法④：EV/EBITDAバリュエーション
| 項目 | 計算 |
|---|---|
| FY3/27E OP | ¥250B |
| D&A（推定） | ¥100B |
| EBITDA | **¥350B** |
| 現在EV（Market Cap + Net Debt） | ¥683B + ¥0 ≈ ¥683B |
| 現在EV/EBITDA | **1.95x** |
| グローバル同業中央値 | 3.0-4.5x |
| ターゲット倍率（低ROE割引） | **3.0x** |
| Target EV | ¥350B × 3.0x = ¥1,050B |
| Net Debt控除 | ¥0 |
| 株数 | 631M株 |
| **FV（EV/EBITDA）** | **¥1,050B / 631M = ¥1,664** |

### 4手法ブレンドFV
| 手法 | FV | ウェイト |
|---|---|---|
| P/E | ¥1,800 | 30% |
| Justified P/B | ¥2,050 | 20% |
| DCF | ¥1,863 | 30% |
| EV/EBITDA | ¥1,664 | 20% |
| **ブレンドFV** | **¥1,834** | — |

## ■ Q-1定性変化（FY3/26 3Q：2025年10月-12月）

### 決算概要（Bloomberg/3Q決算短信）
- **3Q累計OP: ¥30.7B**（QoQ見ていないが全年ガイダンス¥50Bに対し61%進捗）
- **通期ガイダンス据え置き: ¥50B**（関税後）
- コスト削減: 変動費¥15B削減（年間目標¥100B）・固定費削減進捗中
- 在庫水準: 北米在庫正常化進行中（適正水準に向けて調整）

### 定性変化の注目点
1. **ホルムズ封鎖による中東向け生産停止（4月〜）:** 宇品・防府工場の中東向けラインが影響。ただし対米向けラインへの転換が可能か要確認。
2. **CX-5フルモデルチェンジ準備:** 2026年モデル（北米向け）はハイブリッドなし。競合RAV4/CR-Vに対して継続的な弱点。
3. **中国市場の構造的撤退加速:** CX-70/90の前年割れ継続（2025年11月〜2026年1月）。NEV対応は2029年以降。
4. **アラバマ工場（MTMUS）フル稼働:** トヨタとの合弁工場は米国製造扱いでIEEPA非対象。生産比率を米国に移すほど関税リスク低下。
