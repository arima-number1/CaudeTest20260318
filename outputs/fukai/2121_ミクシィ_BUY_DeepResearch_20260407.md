### PBR1倍割れの深度

```
PBR乖離額 = BPS ¥2,683 − 株価 ¥2,540 = ▲¥143（純資産に対して5.3%ディスカウント）
```

**PBR1倍割れの深度は「浅い」**。0.95倍は1倍まで残り5.3%であり、深刻な解散価値割れではない。しかし純資産の質（現預金・有価証券の厚みvs.のれん等の評価）を考慮すると**実質NAVベースではさらに割安**である可能性が高い（④NAV算出で詳述）。

**ミクシィの財務構造**は「キャッシュリッチ・無借金」が特徴。公開情報（最新有価証券報告書・決算短信ベース）から以下を推計：

| 自己資本比率 | **推計70〜80%** | IT系無借金企業として業界最高水準 |
| 現預金＋有価証券 | **推計600〜800億円** | 純資産の45〜60%をキャッシュ性資産が占める構造 |

## ③ キャッシュの使途と資本効率改善余地

### ネットキャッシュの規模と時価総額比

```
【前提推計】
　ネットキャッシュ（現預金＋有価証券−有利子負債）：550〜750億円（中央値：650億円）
　時価総額：約1,245億円
　ネットキャッシュ比率：650 ÷ 1,245 = 52%

　→ 時価総額の約半分がキャッシュ。EV/事業価値 = 1,245 − 650 = 595億円
　→ 事業価値595億円で純利益（EPS ¥251 × 4,900万株 ≒ 123億円）を稼ぐ
　→ 事業EV/利益倍率 = 595 ÷ 123 ≒ 4.8倍（割安感は極めて強い）
```

### （A）大規模自社株買いのインパクト試算

```
【シナリオ：200億円規模の自社株買い実施】

　現在発行済株式数：約4,900万株
　買い付け株価：¥2,540
　買付株式数：200億円 ÷ 2,540 = 約787万株（発行済の約16%）

　【EPS改善効果】
　純利益約123億円 ÷ 残存株式数（4,113万株）= EPS ¥299（+19%改善）
　
　【BPS・PBR改善効果】
　純資産約1,315億円 − 200億円（買付コスト）= 1,115億円
　新BPS = 1,115億円 ÷ 4,113万株 = ¥2,710（+1.0%）
　PBR改善：理論的に株価上昇でPBRが1.0倍に接近
　
　【適正株価試算（PER維持）】
　EPS ¥299 × PER 10.1倍 = 理論株価 ¥3,020（現在比+19%）

　★インパクト評価：「最も即効性が高い」★★★★★
```

# 4. DCFバリュエーション

## 4-1. WACC前提条件

### パラメータ設定

| パラメータ | 値 | 根拠・出所 |
|-----------|-----|---------|
| **Rf（リスクフリーレート）** | **2.429%** | 日本10年国債（財務省 R8.4.6） |
| **ERP（株式リスクプレミアム）** | **5.5%** | Damodaran 日本株市場標準値（2025〜2026年） |
| **β（ベータ値）** | **0.85** | IT×ゲーム株（モメンタム低下）。キャッシュリッチでβ低め |
| **Re（株主資本コスト）** | **7.10%** | Rf 2.429% + 0.85 × 5.5% = 7.104% |
| **Rd（負債コスト）** | **0.5%** | 実質無借金→超低コスト借入のみ |
| **Tc（法人税率）** | **30%** | 日本法人税率標準 |
| **E/（D+E）（株主資本比率）** | **95%** | 無借金に近い財務構造 |
| **D/（D+E）（負債比率）** | **5%** | — |
| **WACC** | **6.77%** | 下記計算 |

```
WACC計算：
　= 95% × 7.104% + 5% × 0.5% × (1−0.30)
　= 6.749% + 0.0175%
　= 6.766% ≒ 6.77%
```

### 永続成長率（Terminal Growth Rate）設定

| シナリオ | g（永続成長率） | 根拠 |
|---------|--------------|------|
| 楽観 | 1.5% | みてね成長＋スポーツ成熟期の安定 |
| 基本 | 0.5% | ゲーム縮退を新事業でほぼ相殺 |
| 悲観 | ▲0.5% | モンスト急落後の構造的縮小 |

## 4-2. 予測フリーキャッシュフロー（FCFF）

### 【シナリオA】ゲーム緩慢縮退 × スポーツ成長（基本シナリオ）

```
　　FY         売上高   営業利益  NOPAT   CAPEX    ΔWC     FCFF
　　（億円）
　　FY2026E    460      90       63     ▲25     ▲5      33
　　FY2027E    430      80       56     ▲30     ▲3      23
　　FY2028E    400      75       53     ▲35     ▲2      16
　　FY2029E    390      80       56     ▲30      0      26
　　FY2030E    400      90       63     ▲25      0      38
　
　　NOPAT = 営業利益 × (1−0.30)
　　Terminal Value（FY2030以降）：FCFF FY2030 × (1+g) ÷ (WACC−g)
　　= 38 × 1.005 ÷ (0.0677−0.005) = 38.19 ÷ 0.0627 ≒ 609億円
```

**DCF計算（基本シナリオ）**：

```
　DCF現在価値（FY2026〜2030）：
　PV = 33/(1.0677)^1 + 23/(1.0677)^2 + 16/(1.0677)^3 + 26/(1.0677)^4 + 38/(1.0677)^5
　≒ 30.9 + 20.2 + 13.1 + 19.9 + 27.2 = 111.3億円
　
　Terminal Value PV = 609 ÷ (1.0677)^5 ≒ 609 ÷ 1.388 ≒ 438.8億円
　
　事業価値（Enterprise Value）= 111.3 + 438.8 = 550.1億円
　＋ ネットキャッシュ：650億円
　＝ 株主価値：1,200.1億円
　÷ 発行済株式数：4,900万株
　＝ 理論株価：約¥2,449
```

　　（億円）
　　FY2026E    390      40       28     ▲30     ▲5      ▲7
　　FY2027E    360      35       25     ▲30     ▲3      ▲8

　　FY2030E    390      80       56     ▲20      0      36

　DCF計算：
　PV（FCF） = ▲7/1.0677 + ▲8/1.0677² + 5/1.0677³ + 22/1.0677⁴ + 36/1.0677⁵
　≒ ▲6.6 + ▲7.0 + 4.1 + 16.8 + 25.9 = 33.2億円
　Terminal Value PV = 535 ÷ 1.388 ≒ 385.4億円
　
　事業価値 = 33.2 + 385.4 = 418.6億円
　＋ ネットキャッシュ：650億円
　＝ 株主価値：1,068.6億円
　÷ 4,900万株
　＝ 理論株価：約¥2,181
