### 4-3. DDM（配当割引モデル）との統合

**Gordon Growth Model（永続成長モデル）補完計算**：

```
理論株価（DDM）= EPS × (1 - 内部留保率) / (Re - g)
= ¥185 × 0.22 / (6.539% - 1.5%)
= ¥40.7 / 5.039%
= ¥807（配当ベースの理論値：保守的下限）
```

> DDMは超低配当（利回り約1.9%）の成長企業では本源的価値を著しく過小評価する。
> **PBR回帰モデルとDCFの加重平均**で算出することが合理的。

### 4-4. PBR回帰・バリュエーション統合（メインモデル）

**Residual Income Model（残余利益モデル）**：

```
理論株価 = BPS + Σ[EPS_t - (Re × BPS_t-1)] / (1+Re)^t + TV

年間残余利益（RI）= EPS¥185 - (6.539% × ¥3,728) = ¥185 - ¥243.7 = ▲¥58.7
```

> 現状のROE（5.4%）はRe（6.539%）を下回っており、厳密なRIMでは理論株価はBPS割れになる。
> これが現在PBR0.57倍の主因（市場も同様に評価）。

**しかし以下の調整が必要**：

1. **ROE改善効果**：Q3OP+79.6%が通期に反映されればROEは5.4%→**7.5〜9.0%**に改善（EPS¥185〜¥205前提）
2. **カタリスト調整後のROE**：7.5〜9.0%はReの6.539%を上回る→残余利益がプラス転換

```
改善後ROE（7.5%）での残余利益：
RI = ¥279.6（=7.5%×¥3,728） - ¥243.7（=Re×BPS） = +¥35.9/年
```
