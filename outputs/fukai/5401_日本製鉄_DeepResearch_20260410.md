## 【セクション1：コア質問への直接回答】

### 「中国デフレ脱却は本物か」

**答えは"層別"で判断しなければならない。**

| 層 | 数値 | 解釈 |
|---|---|---|
| 中国PPI（全体） | 2026年3月：**+0.1%**（41ヶ月ぶり正転換） | コストプッシュ型（エネルギー・資源高が主因）|
| 中国PPI（鉄鋼） | 2026年2月：**-3.4%**（1月:-3.7%） | 需要不足型のデフレが鉄鋼では継続 |
| 中国鉄鋼輸出量 | 2025年：**1億1,900万トン**（過去最高、+7.5%） | デフレ輸出圧力は2025年まで拡大 |
| 中国輸出許可制 | 2026年1月1日より一部実施 | 「方向性の転換」シグナル |

**結論：全体PPIの正転換はコストプッシュ型。鉄鋼固有のデフレは未解消だが、輸出量の頭打ちと許可制導入で「底打ちは視野に入ってきた」段階。**

---

### 「それなら日本製鉄にポジティブか」

**答えは"時間軸次第"でポジ材料だが、今期（FY2026/3期）利益には未反映。**

```
鋼材市況回復ルート:
中国輸出減少 → アジア鋼材価格の底打ち
→ 日本製鉄の国内価格転嫁力強化（5月出荷+10%交渉）
→ 事業利益改善（感応度: 鋼材トン単価+¥1,000→事業利益+¥100〜150億）
```

**日本製鉄CFO 岩井武彦（2026年2月19日）:** 「中国輸出業者はアジア市場の供給過剰と各国の貿易措置により、海外販売での採算確保が困難になっている。アジア鋼材市場は**底打ちに近い**」

この発言と同時に、2026年5月出荷分から**鋼板10%値上げ**を公表したことは、市況転換に自信がある証拠。

## 【セクション5：Q-1定性変化考察（2026-04-06→04-10）】

### ポジティブ変化（3点）

**ポジ①　中国鉄鋼輸出の方向性転換シグナルが2つ揃った**
- 中国輸出許可制（2026年1月）+ CFOの「底打ち宣言」（2026年2月）が時間差で市場に浸透
- アジア鋼材価格の下落率が縮小（-10%台 → -5%台）
- 5月出荷分鋼板+10%値上げ公表は「反転攻勢の第一弾」

**ポジ②　実力事業利益（USS除く）¥6,800億は「想定超の底力」**
- 今期の損失は「USS統合費用」という一過性コストが正体
- 国内製鉄事業・インド事業は堅調 → 中長期の収益基盤は劣化していない

**ポジ③　PBR0.59xという水準は東証圧力の「射程内」**
- 東証のPBR1.0x改善要請が継続。PBR0.59xは改善要請の最前線
- 5/13決算で「追加自社株買い」「増配」「2030中計進捗」開示の可能性
- BPS¥1,018に対し株価¥600は物理的下値支持として機能

### ネガティブ変化（3点）

**ネガ①　中国鉄鋼PPIは2月でも-3.4%。「底打ち宣言」と「数字の現実」の乖離**
- CFOが「底打ちに近い」と言っても、中国の鉄鋼PPIは依然マイナス
- 中国の2025年輸出量1億1,900万トンは「過去最高」。2026年Q1の減少確認が必須
- 許可制が実効的に機能するか → 「ザル規制」の可能性は排除できない

**ネガ②　USスチール貢献「FY2026ゼロ」の事実は重い**
- 800億円の利益計画が0に修正されたのはQ3決算（2月）で明らかになった
- FY2027の貢献回復を織り込む前に「設備トラブルの根本解決」を確認する必要
- USS統合費用はFY2027以降も継続する可能性（税務・財務構造統合）

**ネガ③　EPS赤字の中での配当¥24は「財務的な無理」が発生している**
- EPS -¥13.39なのに配当¥24は配当性向計算不能（赤字期のキャッシュ払い出し）
- FY2027のEPS回復がなければ減配リスクが浮上。配当利回り4.0%の「信頼性」に疑問符

## 【セクション6：リスクシナリオ（定量ストレステスト）】

```

  → 中国輸出2026年も1億トン超
  → USS貢献 -¥500億（追加損失）
  → 連結事業利益: ¥6,800億 - ¥500億 = ¥6,300億
```
