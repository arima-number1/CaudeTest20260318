## ■ ビジネスモデル解剖

### 会社概要
トビラシステムズは名古屋発の迷惑電話・詐欺対策SaaS企業。警察庁との連携による迷惑電話DBを軸に、180M+IoTデバイスからデータ収集するネットワーク効果型ビジネスモデル。ブロック率97%という業界最高水準の精度が競争優位の核心。

### 事業セグメント詳細

| セグメント | 主要サービス | FY2025売上 | 成長率 | 収益特性 |
|---|---|---|---|---|
| **Security Business** | トビラフォン モバイル・280blocker（広告ブロック）・キャリア向けDB提供 | ~¥21億 | 安定 | **高マージン・定常収益**。キャリア3社（NTT/SB/au）との契約が基盤 |
| **Solutions Business** | トビラフォン Biz（法人PBX+詐欺対策）・クラウドサービス | ~¥7億 | **+63.2%** | **高成長SaaS**。企業顧客はLTV高・解約率低。収益化加速フェーズ |
| **New Business** | 詐欺トレ（SagiTore）・アニメ等コンテンツ | 育成中 | 立ち上げ | 将来の第3の柱 |

### 280blockerの特別分析（最大の非線形アップサイド）

```
280blocker 現状：
  iPhone有料アプリ3年連続1位（2023-2025）
  2026年4月時点：総ダウンロード数 200万件突破（日経報道）
  iOS・Androidの両プラットフォーム対応
  
収益モデル：
  月額サブスク: ~¥370/月 = ¥4,440/年
  年間パック: ¥1,900/年
  
潜在的収益試算（2M DL前提）：
  アクティブユーザー2M × 有料転換率25% = 50万人有料
  平均ARPU: ¥2,500/年（月額とパック混在）
  年間潜在収益: 50万 × ¥2,500 = ¥12.5億/年
  
現在の年間純利益ガイダンス（FY2026E）: ¥5.31億
→ 280blockerフル有料化だけで純利益の倍以上のポテンシャル
→ 現行収益見積もりに280blocker monetization upside が未反映の可能性
```

### ネットワーク効果の護城河

```
ユーザーが通報（善意のフィードバック）
         ↓
迷惑電話・詐欺DB更新（月次1,000万件+更新）
         ↓
ブロック精度向上（現在97%）
         ↓
新規ユーザー増加・キャリアへの売り込み力向上
         ↓
DBが大きくなるほど後発が追いつけない
```

NTT/SoftBank/KDDIはキャリア自身が無料で迷惑電話対策を提供しているが、それらの多くがトビラシステムズのDB技術をホワイトラベルで使用している。競合ではなく**顧客**。

## ■ バリュエーション（4手法）

### 前提パラメーター

| パラメーター | 値 | 根拠 |
|---|---|---|
| Rf | **2.397%** | Bloomberg JGB 10年（2026/04/22） |

| FY2028E EPS | **¥100**（中央値） | OPM回復後の正規化EPS |
| 発行済み株数 | **10.695M株** | |

### 手法①：PERバリュエーション（FY2028E基準）
- FY2028E EPS: ¥100（OPM 28%回復、ROE 25%+ 時代）
- 適正P/E: **18x**（高成長SaaS vs スタンダード小型株のバランス）
- FV: ¥100 × 18x = **¥1,800**
- *注：FY2026E EPS ¥50基準では P/E 27xが高く見えるが、これは一時的な投資フェーズ圧縮。FY2028E復帰後の12-18ヶ月先P/Eで評価すべき。*

### 手法②：Justified P/Bバリュエーション
- FY2028E 正常化ROE: **25%**（中期計画達成時）
- Justified P/B = 25% / 7.50% = **3.33x**
- FV = 3.33 × BPS ¥250 = **¥833**
- *注：SaaS/プラットフォーム企業はJ-PBが過小評価になりがち。無形資産（DB・ブランド・特許）がBPSに反映されていないため、このFVは参考下限値として扱う。*

### 手法③：DCFバリュエーション
| 仮定 | 値 |
|---|---|
| FY2027E FCF | ¥700M（OP ¥950M × 利益率 - Capex ¥200M）|
| FY2028E FCF | ¥950M（OPM回復） |
| FY2029-2031E FCF | ¥1,100M/年 avg |
| WACC | **7.0%**（Ke7.5% × 97%+Kd2%×3%） |
| 永続成長率 | **3.0%**（SaaS型長期成長） |
| TV = FCF2031/(WACC-g) | ¥1,100M/(7%-3%) = ¥27,500M |
| TV現在価値 | ¥27,500M / 1.07⁵ = ¥19,600M |
| FY2027-2031 FCF現在価値 | ~¥3,800M |
| Enterprise Value | ¥23,400M |
| ネット現金（＋） | ¥3,035M |
| Equity Value | **¥26,435M** |
| 1株当たり | **¥26,435M / 10.695M = ¥2,472** |

*DCFは高成長前提のため楽観的。ターミナル成長3%が鍵。参考上限値として扱う。*

### 手法④：EV/Revenue（SaaS比較法）
| 項目 | 計算 |
|---|---|
| FY2027E Revenue | ¥4,200M（+25% YoY） |
| EV = 時価総額 - ネット現金 | ¥1,530M × 10.695M - ¥3,035M = ¥15,330M - ¥3,035M ≈ **¥12,295M** |
| 現在EV/Revenue（FY2026E） | ¥12,295M / ¥3,366M = **3.65x** |
| 日本SaaS同業水準 | 3-8x（成長率20%+は4-6x帯） |
| ターゲット倍率（成長20%+・ROE25%） | **5x** |
| Target EV | ¥4,200M × 5x = ¥21,000M |
| ネット現金加算 | ¥21,000M + ¥3,035M = ¥24,035M |
| **FV（EV/Rev）** | **¥24,035M / 10.695M = ¥2,247** |

## ■ Q-1定性変化（FY2026 Q1：2025年11月-12月）

### 決算概要
- **売上+16.8%** vs FY2025 Q1: ガイダンス進捗◎
- **営業利益▲12.5%**: 計画的投資フェーズ確認。投資家は「見た目の減益」に過敏になりがちだが、SaaS投資フェーズとして正常
- **Solutions +63.2%**: 法人向けの急成長が全社を牽引。ここが鍵中の鍵

### 定性変化の注目点
1. **280blocker 200万DL突破（最新）**: アドブロック市場での存在感が急拡大。AndroidへのOEM展開やB2B（企業向けブラウザ管理）に転用可能
2. **「詐欺トレ」（SagiTore）サービス開始**: 従来の電話対策に加え、SNS詐欺・フィッシング対策に事業領域を拡張。TAMを大幅拡大
3. **採用積極化・名古屋→全国展開**: 東京オフィスの機能拡充で首都圏大企業への営業強化
4. **キャリアとの関係深化**: NTTとのホームルーター向けDB提供が継続。キャリア顧客は解約リスクが極めて低い安定収益源

## ■ 不動産含み益考察

トビラシステムズは本社・オフィス賃借型のアセットライト企業。不動産含み益は**ほぼゼロ**。

隠れ価値は**無形資産**に存在：
1. **迷惑電話DB（特許技術）**: 180M IoTデバイスのリアルタイムデータ。競合は10年かけても複製不可能
2. **280blockerブランド**: iPhone有料アプリ3年連続1位。ユーザーロイヤルティ極高
3. **警察庁・キャリアとの関係性**: 法執行機関との連携は政府許認可と同等の参入障壁
4. **ユーザーコミュニティDB**: 1.5M MAUのフィードバックループ

**推定無形資産価値: ¥50-100億（貸借対照表に記載なし）**
→ P/B 5.76xの大部分がこの無形資産プレミアムであり、正当化されると判断
