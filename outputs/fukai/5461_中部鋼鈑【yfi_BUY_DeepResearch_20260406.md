## 2. 【最優先】社長第3次是正命令対応
### ROE -0.15%の現実と「低PBR=割安」論の根本的崩壊

> **社長指示：「ROE-0.15%が最大の誤り。『低PBR=割安』論の根拠が崩れるROEゼロの現実を直視し判定し直せ」**

### 2.1 過去2次レポートの誤りの核心

本セクションは最も重要であり、深井個人として誤りを正式に認める。

| 項目 | 第1・2次レポート（誤） | 第3次レポート（正） |
|------|----------------------|-------------------|
| ROE | ≒ 10%（推計誤り） | **-0.15%（yfinance実測）** |

| 投資論理 | 「PBR0.92＝割安、ROE改善期待」| **「ROE≒0では割安論は成立しない」** |
| BPS | 誇大推計の可能性 | **¥2,733（確定）** |

**ROE 10%と-0.15%では投資判断が180度変わる。** これは単なる精度の問題ではなく、判断の方向性そのものが誤っていた。

### 2.2 「低PBR=割安」論が崩れるメカニズム

市場では「PBR1倍割れ＝解散価値以下＝割安」という単純論が流布されているが、**これはROEが正常水準にある場合にのみ成立する議論**である。

#### 📐 Gordon Growth Model によるPBRの理論的導出

$$PBR = \frac{ROE - g}{WACC - g}$$

この式にROE = -0.15%を代入する：

```
仮定: WACC = 6.5%, g = 1.0%（名目成長率）

理論PBR = (-0.15% - 1.0%) / (6.5% - 1.0%)
        = -1.15% / 5.5%
        = -0.209倍
```

**理論PBRがマイナスになる。** これは何を意味するか？

> **ROEがWACCを大幅に下回り、かつ成長率をも下回る場合、企業は事業継続によって株主価値を破壊している。**
> 理論上は「解散して純資産を株主に返却する方が、事業継続よりも価値が高い」という状態を示す。

#### ❌ 「PBR0.92 = 割安」論が成立しないケース（現状）

| 割安論の前提 | 中部鋼鈑の現実 | 判定 |
|-------------|--------------|------|
| 企業が継続的に利益を生む | ROE -0.15%（実質赤字） | **❌ 崩壊** |
| 純資産は実態を反映 | 確認必要（含み損リスク） | **⚠️ 要検証** |
| ROE > WACC（価値創造） | ROE ≒ 0% << WACC ≒ 6.5% | **❌ 崩壊** |
| PBR切り上がり余地あり | ROEゼロでは切り上がらない | **❌ 崩壊** |

#### 🔑 結論：PBR0.92は「割安」ではなく「収益力ゼロのプレミアム解消待ち」

**現在株価¥2,510がBPS¥2,733を下回っているのは、市場が正しく「ROEゼロ企業は純資産価値未満でしか評価できない」と判断しているに過ぎない。** むしろPBR0.92（8%のディスカウント）でさえ、ROEがマイナスであることを考えれば、**さらなる下値余地が理論上は存在する。**

### 2.3 ROEほぼゼロが意味する電炉メーカーとしての構造問題

```
【メタルスプレッド収益モデルの崩壊構図】

 販売単価（市況連動・価格支配力なし）
   ↓  マイナス
 スクラップ調達コスト（市況高止まり）
   ↓  マイナス
 電力コスト（原油111.61ドル水準で高騰）
   ↓  マイナス
 固定費（人件費・設備維持・減価償却）
   ↓
 【結果】営業利益≒ゼロ → ROE -0.15%

```

ROEがゼロということは、**¥2,733という純資産（BPS）を使っても株主への利益貢献がゼロ**であることを意味する。株主は資本を預けているのに、リターンが全くない状態。これは「持っているだけでコスト（機会費用）が発生している」という状態である。

#### 📋 資本構成前提

| パラメータ | 数値 | 備考 |
|-----------|------|------|
| E/（D+E）自己資本比率 | 55% | BPS・財務構造から推算 |

| 実効税率（Tc） | 30% | 日本法人税標準 |

#### 📋 WACC計算

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.55 × 8.43%] + [0.45 × 2.0% × 0.70]
     = 4.637% + 0.630%
     = 5.27%

（保守的調整後: 6.5% — ROE≒0の不確実性プレミアム+1.23%加算）
```

> ROEがゼロ水準で事業継続リスクが通常より高いため、**WACC基本値5.27%に不確実性プレミアム1.23%を加算し、保守的WACC = 6.5%** を採用する。

- 予測期間：5年
- 初年度FCF：¥5億円
- 成長率（予測期間）：3%/年（スプレッド微回復）
- ターミナル成長率（g）：**0.5%**（成熟産業・国内縮小市場）
- WACC：6.5%（保守的）

```
【FCF予測（5年間）】

PV（予測期間）合計 = ¥21.98億円

【ターミナルバリュー】
TV = FCF₅ × (1+g) / (WACC - g)
   = ¥5.63億円 × 1.005 / (0.065 - 0.005)
   = ¥5.658億円 / 0.060
   = ¥94.3億円

PV（TV）= ¥94.3億円 / (1.065)^5 = ¥68.8億円

【企業価値（EV）】

→ DCFによる収益価値は純資産の約17%しかない

【純資産（BPS）ベース混合評価】
```
