### 2-1. ナフサ価格感応度分析（直近原油$111/bbl水準ベース）

#### ■ ナフサ価格の原油連動性

ナフサ価格は原油（ブレント）に強く連動する。一般的な換算式：

```
ナフサスポット価格（$/ton） ≈ ブレント原油（$/bbl）× 7.0〜7.5

現状（ブレント $109.54/bbl 基準）:
→ ナフサ: $109.54 × 7.2 = 約 $789/ton
→ 円換算（USD/JPY = 145円想定）: 約 ¥114,400/ton

ナフサ+10%シナリオ:
→ ナフサ: $789 × 1.10 = $868/ton
→ 円換算: 約 ¥125,800/ton（+¥11,400/ton）
```

#### ■ ナフサ+10%時の営業利益インパクト試算

```
【感応度計算（フルパススルー前/後）】

前提:
  売上高: ¥19,000百万円（中央値）
  ナフサ系原材料費: 売上高の42% = ¥7,980百万円
  ナフサ+10% → 原材料費増加: ¥7,980 × 10% = +¥798百万円

▶ 転嫁ゼロ（最悪ケース）:
  営業利益への直撃: −¥798百万円
  現行営業利益想定(¥1,000百万円) → ¥202百万円（▲79.8%）
  営業利益率: 5.3% → 1.1%

▶ 50%転嫁（現実的ケース）:
  コスト増のうち半分を顧客価格に転嫁
  実質コスト負担: ¥798 × 50% = ¥399百万円
  営業利益: ¥1,000 → ¥601百万円（▲39.9%）
  営業利益率: 5.3% → 3.2%

▶ 100%転嫁（理想ケース）:
  コスト増をすべて転嫁（ただし数量減のリスクあり）
  実質コスト負担: ゼロ（ただし売上増分で原価率は変わらず）
  営業利益: 理論上維持（¥1,000百万円）
  ただし顧客離れリスクによる数量▲5〜10%の可能性
```

**【ナフサ感応度サマリーテーブル】**

| 転嫁率シナリオ | 原材料費増加 | 営業利益 | 営業利益率 | EPS影響 |
|-------------|-----------|---------|---------|--------|
| 転嫁ゼロ（最悪） | +¥798M | ¥202M | 1.1% | ▲約¥83→¥89 |
| 50%転嫁（基本） | +¥399M実質 | ¥601M | 3.2% | ▲約¥42 → ¥130 |
| 80%転嫁（楽観） | +¥160M実質 | ¥840M | 4.4% | ▲約¥17 → ¥155 |
| 100%転嫁（理想） | ゼロ | ¥1,000M | 5.3% | ほぼ維持 ¥172 |

> **深井所見**: ナフサ+10%は営業利益に対して**極めて大きなレバレッジ**を持つ。転嫁率が50%を下回ると、EPSは¥130前後に低下し、PER（11.4倍）前提での適正株価も¥1,480程度まで低下リスクがある。**コスト転嫁力の把握が投資判断のコアドライバー**。

## 4. DCFバリュエーション

### 4-1. WACC計算

#### ■ パラメータ設定

| パラメータ | 値 | 根拠・出所 |
|-----------|----|---------| 
| **Rf（リスクフリーレート）** | **2.386%** | JGB10Y / 財務省 R8.4.3（絶対厳守） |
| **ERP（市場リスクプレミアム）** | **5.5%** | 日本株ERP（Damodaran推計 / 2026年版 5.0〜6.0%中央値） |
| **β（ベータ）** | **0.75** | 化学専門商社・スタンダード市場小型株。業界β 0.7〜0.9の中央値。低流動性・ディフェンシブ性を反映して低め設定 |
| **Re（株主資本コスト）** | **Re = 2.386% + 0.75 × 5.5% = 6.511%** | |
| **Rd（負債コスト）** | **1.5%** | 短期借入中心・金融機関優遇金利（推計） |
| **Tc（実効税率）** | **30%** | 日本法人税率標準 |
| **E/(D+E)** | **85%** | 自己資本比率高い。D/E≈0.15から算出 |
| **D/(D+E)** | **15%** | 有利子負債軽微 |

#### ■ WACC計算

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.85 × 6.511%] + [0.15 × 1.5% × (1-0.30)]
     = 5.534% + 0.158%
     = 5.692%

【採用WACC: 5.69%（ベースケース）】
```

### 4-3. DCF計算（ベースケース）

```
【ベースケース DCF計算（WACC=5.69%, g=1.0%）】

ベースFCF = ¥1,300百万円（推計）

FY2026: ¥1,300 × (1-0.03) = ¥1,261 / (1.0569)^1 = ¥1,193
FY2027: ¥1,261 × (1-0.01) = ¥1,248 / (1.0569)^2 = ¥1,118
FY2028: ¥1,248 × 1.01    = ¥1,260 / (1.0569)^3 = ¥1,069
FY2029: ¥1,260 × 1.02    = ¥1,285 / (1.0569)^4 = ¥1,033
FY2030: ¥1,285 × 1.02    = ¥1,311 / (1.0569)^5 = ¥998
FY2031: ¥1,311 × 1.03    = ¥1,350 / (1.0569)^6 = ¥975
FY2032: ¥1,350 × 1.03    = ¥1,391 / (1.0569)^7 = ¥953

PV合計（Phase1+2）: ¥7,339百万円

ターミナルバリュー（TV）:
TV = ¥1,391 × (1+0.01) / (0.0569 - 0.01)
   = ¥1,405 / 0.0469
   = ¥29,957百万円

PV(TV) = ¥29,957 / (1.0569)^7 = ¥20,516百万円

企業価値（EV）= ¥7,339 + ¥20,516 = ¥27,855百万円

【株式価値の算出】
純有利子負債（推計）: ¥2,000百万円 − 現金¥5,000百万円 = ▲¥3,000百万円（ネットキャッシュ）
  → 現金リッチ企業のため、株式価値 > EV

株式価値 = EV + ネットキャッシュ
         = ¥27,855 + ¥3,000 = ¥30,855百万円

理論株価（ベース）= ¥30,855百万円 ÷ 8,000千株
               = ¥3,857/株
```
