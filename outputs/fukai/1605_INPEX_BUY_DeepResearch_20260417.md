## セクション1：企業概要・事業分析

### 1-1. INPEXとは何者か

INPEXは日本最大の石油・天然ガス開発会社（E&P）。経済産業省が約18%出資する"国策エネルギー企業"であり、エネルギー安全保障の文脈で政策的な保護も受ける。時価総額は約4.1兆円（2026年4月17日現在）。

**主要権益（地理分散）:**

| 権益 | 国・地域 | INPEXの持分 | FY2025生産寄与 | ホルムズ関連リスク |
|------|---------|------------|--------------|-----------------|
| **イクシスLNG** | 豪州 | **76%（オペレーター）** | 約40〜45% | **ゼロ（完全無関係）** |
| ADCO（アブダビ陸上） | UAE | **約2.2%** | 約20〜25% | **低（フジャイラ迂回可）** |
| カシャガン | カザフスタン | **約7.56%** | 約10〜15% | **ゼロ（内陸・完全無関係）** |
| Azadegan | イラン | 持分あり | **事実上ゼロ（制裁）** | 既にゼロ |
| その他（日本/東南亜等） | 複数 | 各種 | 約15〜20% | 低〜ゼロ |

**総生産量（FY2025）**: 約70〜75万BOE/日相当（原油換算）

### 1-2. セグメント構成

```
売上高構成（推定FY2025）
├── 天然ガス・LNG: 約45%  ← イクシスが中心、JKM連動
├── 原油: 約45%           ← Brent・WTI連動、ADCO・カシャガン
└── その他（NGL等）: 約10%
```

### 1-3. 競合優位性

1. **イクシスLNGのオペレーター権限（76%持分）**: 生産スケジュールを自社でコントロール。他社権益と異なり、相手国の政治リスクに左右されにくい
2. **地理的分散の完成度**: 中東・豪州・中央アジア・アジアに分散し、ホルムズ依存度が競合他社（1662 JAPEXなど）比で著しく低い
3. **国策企業の暗黙サポート**: 経産省出資により、危機時の資金調達・政策支援が得やすい
4. **LNG長期契約比率の高さ**: 日本・韓国・台湾向けにTOCO（テイク・オア・ペイ）契約あり、スポット安でも下支え

### 2-2. 権益別ホルムズ影響の個別分解

- 現在のJKM-LNG価格：**高止まり（推定$16〜19/MMBtu）**

**原油高・LNG高による収益増試算（Brent$91.73前提）:**

```
【LNG価格感応度】

- イクシスのスポット枠（年間約10〜15%）が高値で捌ける可能性
- 追加収益：+¥50〜100億（短期・一過性）

```

**結論**: アブダビ権益は「ほぼ安全」。迂回コスト増は軽微（▲¥10〜20億）で、原油高による増収（+¥30〜50億）が十分カバー。**ネット効果はプラス。**

## セクション4：DCFバリュエーション

### 4-1. WACC計算

**インプット:**

| パラメータ | 数値 | 根拠 |
|---------|------|------|
| リスクフリーレート（Rf） | **2.409%** | 日本10年国債（財務省 2026-04-16） |
| ERP（エクイティリスクプレミアム） | **5.5%** | 日本市場標準（Damodaran 2025年参考） |
| β（ベータ） | **0.95** | INPEXの5年月次β（東証TOPIX比、原油感応度反映） |
| **株主資本コスト（Re）** | **Rf + β×ERP = 2.409% + 0.95×5.5% = 7.634%** | |
| Rd（負債コスト） | **1.8%** | INPEXの社債・借入金平均金利（低金利日本発行） |
| 実効税率（Tc） | **38%** | E&P企業の平均実効税率（資源税含む） |
| E/(D+E) | **68%** | 時価総額÷(時価総額+純有利子負債) |
| D/(D+E) | **32%** | |
| **WACC** | **= 0.68×7.634% + 0.32×1.8%×(1-0.38)** | |
| | **= 5.19% + 0.357% = 約5.55%** | |

> 保守的に調整後WACC: **5.55%〜6.00%** を使用（基本ケース: **5.8%**）
