## 2. 竹村チェックリスト（M&A銘柄必須項目）

### ① 買収資金の調達方法：デット vs エクイティ

**結論：デット（銀行借入）が本命。エクイティ発動確率は10〜15%。**

```
【イチネンHDのM&A資金調達能力の分析】

事業構造から見た借入余力:
  自動車リース関連（売上40%）は「毎月定額のリース料が入る」
  予測可能で安定したキャッシュフロー。
  銀行にとって「担保にしやすい収益源」であり、融資を引き出しやすい。

  農業（11%）・機械工具（23%）・合成樹脂（12%）も
  景気変動はあるが実物資産を保有するセクター。
  資産担保融資（ABL）が活用可能。

実績から見た調達スタイル:
  直近ニュース確認: 増資・MSワラント・公募増資のニュースなし
  子会社2社の吸収合併（ガラス加工）= 新規資金不要の社内再編
  配当を¥80/株に上方修正 = 株主還元を優先している経営姿勢

PBR 0.73倍という数値が語る真実:
  株価¥2,136 < BPS¥2,938
  もし新株を¥2,136で発行すれば、
  帳簿価値¥2,938の資産を¥2,136で売り払うことと同義。
  1株につき ¥2,938 - ¥2,136 = ¥802 の価値を破壊する行為。
  株主価値を重視する経営陣であれば、このPBR水準での増資は回避する。

結論:
  デット調達を優先する合理性が極めて高い。
  エクイティ発動は「デットが限界に達した緊急事態」のみ。
```

### ② EPS希薄化リスクの確率と数値化

**希薄化確率：10〜15%（低リスク）**

```
【希薄化シナリオ分析】

前提: 現在の株主資本 ≒ ¥698億円、発行済み株数 ≒ 2,376万株

ケースA: 10%希薄化（公募増資）
  新規発行株数: 2,376万株 × 10% = 237.6万株
  調達金額: 237.6万株 × ¥2,136 = 約¥50.8億円
  希薄化後EPS: ¥262.16 ÷ 1.10 = ¥238.33（▲9.1%）
  希薄化後PER: ¥2,136 ÷ ¥238.33 = 8.96倍（依然割安）

ケースB: 20%希薄化（大型M&A時）
  新規発行株数: 2,376万株 × 20% = 475.2万株
  調達金額: 475.2万株 × ¥2,136 = 約¥101.5億円
  希薄化後EPS: ¥262.16 ÷ 1.20 = ¥218.47（▲16.7%）
  希薄化後PER: ¥2,136 ÷ ¥218.47 = 9.78倍（依然割安水準）

ケースC: MSワラント（最悪シナリオ）
  行使価格が株価より低く設定 → 継続的希薄化
  確率: 5%未満（非上場子会社取得には不要。MSワラントは赤字企業の資金調達手段）
  イチネンHDの収益体質からは考えにくい

重要な発見:
  PER 8.15倍という超割安水準では、
  10〜20%の希薄化が起きてもPER は8.96〜9.78倍に留まり
  「それでも割安」という逆説的な耐性がある。

  つまり イチネンHDは「希薄化耐性が高い」銘柄である。
```

### ③ M&Aが「株主還元を犠牲にするもの」か否か

**結論：否。株主還元と両立している。**

```
【証拠】
  ① 2026年3月期配当を¥80/株に上方修正（直近ニュース確認済）
     = M&A継続中にも関わらず配当を増やしている

判定: M&Aは内部留保と借入で賄われており、
      株主への配当を犠牲にしていない。健全な資本政策。
```

## 3. 財務健全性の深掘り

### キャッシュフロー構造（推定）

```
【自動車リース事業のCF特性】
  リース料は月次で安定入金。
  車両取得は初期投資だが、取得後は減価償却費 > 設備投資に移行。
  → 「キャッシュカウ（現金牛）」型の事業構造

【借入余力の推定】
  総株主資本（推定）: ¥698億
  リース会社の標準D/Eレシオ: 2〜4倍（業界特性として高い）
  現在D/Eを保守的に2倍と仮定: 有利子負債 ≒ ¥1,400億
  標準上限D/E 4倍まで: さらに¥1,400億の借入余力あり
  → 大型M&Aでも「まずデット調達」が十分可能

【自己資本比率の意味】
  リース会社はB/Sにリース資産（車両）とリース負債が両建てになるため、
  自己資本比率は製造業より低く見える。
  これは「危険」ではなく「業種の特性」。
  実態は安定した事業CF + 資産担保で支えられている。
```

### 配当政策の信頼性

```
配当: ¥80/株（上方修正後）
配当利回り: 3.75%（¥2,136で計算）
配当性向: 30.5%

3.75%という高配当は「資金に余裕があるからこそ払える」。
M&Aで資金が枯渇しているなら配当を上方修正できない。
この上方修正こそが「エクイティファイナンス不要」の最も雄弁な証拠だ。
```
