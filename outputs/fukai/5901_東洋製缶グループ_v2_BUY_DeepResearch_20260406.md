日本最大の缶容器メーカー。飲料缶（アルミ缶・スチール缶）、食品缶（ツナ、ペットフード等）、エアゾール缶を中心に、国内飲料メーカー（コカ・コーラ、サントリー、アサヒ、キリン等）への供給を一手に担う。

| エアゾール缶 | 約30〜35% | 日用品・医薬品メーカー |
| 金属王冠・クロージャー | 約50%超 | 酒類・飲料メーカー |

**事業の本質的強み：**
缶製造は「インフラ型ビジネス」である。飲料・食品メーカーが缶調達先を変更するには、設備投資・品質認証・長期契約の解除が必要であり、スイッチングコストが極めて高い。東洋製缶は主要飲料メーカーとの長期供給契約（5〜10年）を保有しており、年間需要は飲料消費動向と高い相関を示すが、景気感応度は低い（ディフェンシブ性）。

### セグメント②：包装関連事業（売上の約25%）

プラスチックフィルム・フレキシブル包装・紙容器・キャップ等。PETボトル向けプリフォームも含む。環境規制対応（リサイクルPET・バイオマスプラスチック）への転換投資が進行中。

### セグメント③：不動産・その他事業（売上の約10%）

工場跡地の不動産賃貸・開発、物流施設運営。都市部の工場跡地（東京湾岸地区、川崎、大阪等）は簿価を大幅に上回る含み益を保有。詳細はSection 5で分析。

### パス・スルーラグの構造的分析

東洋製缶のビジネスモデルにおいて、アルミ仕入コストの販売価格への転嫁には**構造的に3〜6ヶ月のラグ**が存在する。

```
【転嫁ラグのメカニズム】

時点0（アルミ価格上昇）
    ↓ 3〜6ヶ月
時点1（顧客との価格改定交渉開始）
    → 大手飲料メーカーは四半期・半期ごとの価格レビュー
    → 交渉期間：1〜2ヶ月
    ↓
時点2（新価格での販売開始）

結果：アルミ価格上昇 → コスト増加は即日
      販売価格転嫁   → 3〜6ヶ月後
      ∴ ラグ期間中：マージン圧縮（利益下押し圧力）
```

**FY2026/3の具体的影響：**

## 4.2 自社株買いの実績と継続可能性

### 年3回自社株買いの意義

東洋製缶グループは**年3回（期初・中間・期末）**の自社株買いを実施する資本政策を採用。これは日本の製造業の中でも異例に高い還元規律を示す。

| 自社株買いの評価ポイント | 内容 |
|---------------------|------|
| 実施頻度 | 年3回（業界内でも最上位クラスの頻度） |
| 1回あたり規模 | 推計50〜150億円/回 → 年間150〜400億円規模 |
| 継続可能性 | ネットキャッシュ保有・高インタレストカバレッジで持続可能 |
| EPS押し上げ効果 | 発行済株式数削減 → BPS・EPS改善 → PBR割安是正の自然解消 |
| 株価下支え効果 | 市場悲観（移行年の利益圧縮）下での安定買い需要創出 |

**PBR 0.78倍 × 年3回自社株買い = 「割安確認×資本還元」の黄金組み合わせ**

BPS¥4,426に対して¥3,438で自社株を買い続けることは、本質的価値より22%安く企業資産を取得することと等価。この行為自体が残存株主への価値移転であり、現在株価での投資は自社株買いによる「見えない配当」を受け取ることでもある。

## 8.4 深井の最終メッセージ

**「市場が『一時的な傷』を『構造的な問題』と誤読している間に、賢明な投資家は行動する。」**

東洋製缶グループHD（5901）の現在株価¥3,438は、FY2026/3における「転嫁ラグ」という一時的・構造的に予見可能なコスト圧迫によって生じた歪みである。アルミが$2,505から$3,408まで+36%急騰したとき、コストは即日増えるが、価格転嫁は3〜6ヶ月後。この時間差は会計上の利益を一時的に圧縮する。市場はこの「会計上の損」を見て株を売る。そして賢明な投資家は、この「転嫁ラグ」が解消されるタイミングを正確に測り、仕込む。我々は今まさにその仕込みの最良タイミングにいる。

PBR0.78倍という数字は冷厳な事実を語っている。東洋製缶の純資産は1株¥4,426あるのに、市場は¥3,438でしか評価していない。企業が年3回自社株買いを実施するとき、それは経営者自身が「うちの株は安い」と市場に向けて宣言するメッセージに他ならない。東
