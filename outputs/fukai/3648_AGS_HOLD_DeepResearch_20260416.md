## 3. ⚠️ AI代替リスク詳細分析（最重要）

### 3-1. セグメント別AI代替可能性マトリックス
| セグメント | 売上比率 | AI代替スコア(0-10) | 代替タイムライン | 脅威となるAIサービス |
|---|---|---|---|---|
| **情報処理サービス（BPO含む）** | 48.2% | 6 | 3〜5年（段階的） | ChatGPT Operator、Microsoft Copilot、AutomationAnywhere AI Agent |
| **ソフトウェア開発（官公庁・金融）** | 29.6% | 4 | 5年超（規制障壁） | GitHub Copilot、Cursor、Devin（AI SWE）、Claude Code |
| **その他情報サービス（RPA・セキュリティ）** | 14.3% | 7 | 1〜3年 | Microsoft Sentinel AI、Rapid7 AI、UiPath Autopilot |
| **システム機器販売** | 7.9% | 3 | 5年超（ハード依存） | クラウドへの直接移行、ベンダー直販 |

**AI代替スコア評価基準**：
- スコア4（ソフトウェア開発）：官公庁・金融機関は個人情報保護・マイナンバー法・銀行法等の厳格な規制環境下にあり、AIが自律的にシステムを設計・開発・テストする体制を当局が認可するには5年以上かかると推定。ただし開発工数そのものはAIツールで30〜50%削減される可能性（人月単価下落リスクは現実）
- スコア6（情報処理サービス）：大型汎用機（メインフレーム）の運用・コンピューティングアウトソーシングは当面継続するが、BPO（データ入力・印刷・封緘）はAI-OCR・AIエージェントによる代替が急速に進む
- スコア7（RPA・セキュリティ）：RPAはAIエージェントに急速に侵食される高リスクセグメント。WinActor販売パートナーとしてのAGSは、WinActor自体のAI化（WinActor AI接続サービス）で防衛を図っているが、プラットフォーム競争力は親会社NTTデータ依存

### 3-2. 競合AIプレイヤーの動向

**①SIer事業への直接脅威**
- **GitHub Copilot / Cursor / Claude Code（Anthropic）**：コーディング自動化が2025年に急加速。大手SIer（TIS）は2029年までに開発生産性50%向上を目標に掲げる。AGSのようなSMBサイズのSIerほど、生産性向上が即座に人員削減・コスト削減につながる一方、単価交渉力が弱く「AIで安くなった分はコスト削減益を自社に帰属させにくい」構造的ジレンマがある
- **Microsoft 365 Copilot**：企業のBPO業務（文書処理・メール対応・データ整理）をエンドユーザー自身がAIで対応し始め、BPO受注量が中期的に減少するリスク
- **Amazon Q / Google Gemini for Workspace**：クラウド大手が官公庁向けAI導入で政府調達に参入するリスク。デジタル庁のガバメントクラウド（AWS・Google Cloud推奨）が直接競合

**②BPO代替の現実**
- コールセンターBPO分野では2025年にCAT.AI等のVoice LLMが本格普及。AGSのBPO（主にデータ入力・印刷物流）は音声系ではないが、AIエージェント（ChatGPT Operator等）がフォーム入力・書類処理を自動化し始めており、単純BPO業務の市場が年3〜5%収縮と試算

**③顧客の内製化加速**
- ユーザー企業がAI駆動開発ツールを用いて「SIerに頼らずシステムを作る」内製化が加速。2024年度国内ソフトウェア企業倒産は前年比+43%の220件（過去10年最多）。中小SIerほど影響が深刻。AGSは官公庁・金融という規制に守られた顧客基盤を持つため他SIerほどの即時脅威ではないが、5年視点では内製化が進む

### 3-3. 当社のAI防衛戦略と評価

**①AI-Zanmai（2025年5月リリース）**
企業・団体向けに生成AIのセキュアな業務活用基盤を提供するクラウドサービス。独自のセキュリティポリシー（入出力をAIに学習させない設定）が特徴。顧客の生成AI活用を支援することで、AGS自身がAIの「代替」ではなく「実装パートナー」として収益を取る戦略。
評価：中長期的に有効だが、現状は直接売上への貢献は小さい（IR等で金額非開示）。

**②生成AI定着化支援アドバイザリサービス（2026年1月リリース）**
企業の生成AI定着の障壁解消を支援する有償コンサルティング。リテラシー向上研修と業種特化AI構築支援の2メニュー。
評価：AGSの業種知識（自治体・地銀）を生かした差別化が可能。ただし外資コンサル・大手SIerとの価格・ブランド競争は厳しい。

**③WinActor AI接続サービス（2026年2月リリース）**
既存RPAサービスに生成AI連携機能を付加。WinActor販売での累積顧客基盤を活用。
評価：守りの一手として有効。ただしUiPath・Automation Anywhere等のグローバルプレイヤーと比較すると技術優位性は限定的。

**④セキュリティサービス（標的型攻撃メール対応訓練）への生成AI付加（2024年6月）**
生成AIで訓練メールの文例を自動生成する機能追加。
評価：ニッチだが既存顧客基盤での付加価値向上として有効。

**AGS AI防衛戦略の総合評価**：
- 強み：50年以上の官公庁・金融向け業種知識。顧客データへのアクセス権と信頼関係は短期では代替不可能
- 弱み：AI投資額・技術人材が大手SIerに比べ圧倒的に小さい（R&D費用の開示なし）。AIサービスの多くはNTTデータ（WinActor）・AWS等のプラットフォームへの依存
- 差別化の核：自社Tier-4データセンター（さいたまiDC）によるセキュアな環境での生成AI提供は、官公庁・金融向けのコンプライアンス要件（個人情報保護、データ海外流出禁止）に適合した独自性を持つ
