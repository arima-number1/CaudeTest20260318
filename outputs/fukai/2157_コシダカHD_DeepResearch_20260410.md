```

   a. 人件費上昇（最低賃金改定・人材確保競争）

   c. カラオケの客単価 or 客数の伸び鈍化

② H1 進捗率42〜43%（通期¥112億対比）
   → 残りH2で¥61〜62億の経常利益が必要
   → FY2025 H2 ≈¥63億 → ほぼ前年並みが必要
   → 達成可能だが余裕はない

③ 追加下方修正リスク
   → H2（3〜8月期）はカラオケの閑散期
   → H2の反転力が見えないなら10月の通期決算でさらに修正も
```

## 【セクション6：Q-1定性変化考察（4/6→4/10）】

### ポジティブ変化（2点）

**ポジ①　PBR2.6xは理論PBR（ROE16%前提）3.68xを大きく下回り「逆張り割安水準」**
- 3M▲18.1%という下落で相当量の悪材料を価格に折り込んでいる
- カラオケ業界でのまねきねこの競争優位（店舗数・ブランド力）は短期業績悪化で変わらない
- 配当¥26（利回り2.36%）は修正後も維持見込み → 配当利回りが下値支持

**ポジ②　カーブス（フィットネス）は月会費型で業績安定の「バッファー」**
- 月会費型の女性専用フィットネスは会員解約率が低く、安定収益
- カラオケ事業の季節変動を吸収する役割
- 会員数増加が続く限り、フィットネスセグメントは「稼ぐ貯金」として機能

### ネガティブ変化（3点）

**ネガ①　Q1営業利益▲59.9%の衝撃は「一過性」か「構造的」かが未確認**
- Q1だけで営業利益が60%近く消えた事実は深刻
- 最低賃金改定（2025年10月）+ 新規出店コスト集中の「二重苦」が閑散期に直撃した可能性
- しかし最低賃金は毎年継続して上昇するため、コスト圧力は恒常化リスクがある
- **確認ポイント：Q3（1月発表）での営業利益改善幅が試金石**

**ネガ②　通期7%下方修正は「一回で終わるか」がわからない**
- 修正後の通期達成に必要なH2利益(¥61〜62億)は前年H2並み
- ただしカラオケ閑散期（夏場）と人件費高止まりが続くと、追加修正の可能性
- 「7%で打ち止め」か「第二弾修正がある」かは7月のQ3決算まで判断できない
- 株価±0は「まだ方向性が見えない」という市場の正直な反応

**ネガ③　FY2024以降の「増収減益」トレンドが慢性化しているリスク**
- FY2023: 純利益¥71億（ピーク）→ FY2024: ¥67億 → FY2025: ¥52億 → FY2026修正後: ¥69億
- 売上は毎年+10〜20%増収なのに、純利益がピーク比で▲3%〜▲27%
- 「増収しても利益に落ちない体質」が定着しているなら、PBR2.6xは正当化できない
- BPS改善（¥422.94）は続くため、PBRは自然と下がるが、株価維持には利益の質改善が必要

## 【セクション7：リスクシナリオ（定量ストレステスト）】

```
ストレス① Q3（3〜5月）も前年割れ（春の学生需要弱い）
  → H2進捗が前年の80%に留まる
  → 通期 経常利益: ¥112億 × 0.85 = ¥95億
  → 修正EPS: ¥95億 × 実効税率(35%) ÷ 株式数 = ¥68
  → PER13x: ¥884 → ストップロス¥950が先に発動

ストレス② 最低賃金継続上昇×電気代高止まり（構造コスト悪化）
  → 営業利益率: 16.5%→13.5% （損益分岐売上が上昇）
  → FY2027 EPS¥75以下
  → バリュエーション: PBR1.8x → 株価¥761 (-30.8%)
  → ストップロス¥950で手仕舞い必須

ストレス③ フィットネス（カーブス）会員数が頭打ち
  → 月会費収益の安定バッファーが機能しなくなる
  → カラオケ事業の季節変動のみで利益を作る構造に回帰
  → このシナリオでは楽観的な評価の根拠が失われる
```
