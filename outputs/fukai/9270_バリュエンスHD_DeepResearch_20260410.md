## 2. ビジネスモデルと構造変化の本質

### 事業概要

バリュエンスは中古高級品（ブランドバッグ・時計・宝飾品・地金）の買取・販売・オークション事業。

| セグメント | 内容 |
|---------|------|
| リテール買取・販売 | 一般消費者からの直接買取 → 店舗・EC・海外販売 |
| オークション（B2B委託） | 業者向けオークションを主催し委託落札額に応じた手数料 |
| 地金・貴金属 | 金・プラチナ等の地金買取・販売（相場連動） |

### 今期の構造変化3本柱（深井分析）

**① 地金相場の歴史的高騰**
- 金: $4,749（4/8）→ $4,792（4/9）→ 史上最高値更新中
- 金高騰 → 地金買取・販売の粗利拡大 → OP直撃ポジティブ
- 我々の市況メモ（4/8コモディティ: 金$4,749 +2%）と完全整合

**② 「粗利重視仕入方針」への転換（企業固有の変革）**
- 旧方針: 仕入量を最大化 → 在庫過剰・評価損（FY2024赤字の主因）
- 新方針: 粗利率を重視した選別仕入 → 在庫回転率改善 × 利益率急改善
- Q2 OPM7.5%はこの方針転換が実を結んでいる証拠

**③ オークション委託落札額の拡大（B2Bフロー型収益化）**
- 在庫リスクなしで手数料収益を積み上げるアセットライト型
- 業界内での存在感向上 → 出品量・落札額のネットワーク効果

## 5. 不動産含み益考察（必須項目）

バリュエンスは買取専門店のため、**自社保有不動産は限定的**。店舗は全て賃借（オペレーティングリース）。

- **主要資産**: 仕入商品在庫（ブランド品・貴金属）= 相場変動資産
- 不動産含み益: **実質ゼロ**（NAV観点での分析対象外）
- **真のバリュードライバー**: ブランドネットワーク・オークション顧客基盤（B/S非計上の無形資産）

→ **投資判断は「金価格連動EPS成長」と「事業モデル転換プレミアム」で行うべき**

## 8. リスクシナリオ（定量ストレステスト）

### シナリオA: 金価格急落（停戦×有事プレミアム消滅）（確率30%）

**これが最大リスク。我々の浅井スコア履歴でも「INPEX▲100」と同じ地政学シナリオ。**

- 停戦合意 → WTIと同様に金$4,749 → $3,000台へ急落
- 在庫評価損発生 → FY2024赤字（OP▲¥426M）の悪夢再現リスク
- FY2026 OP修正後¥2,500M → EPS¥103 × 15x = **¥1,545**（SL¥2,000割れ）

> **SL¥2,000は「金$3,800割れ」と連動させて管理推奨**

### シナリオB: 信用倍率10.91倍の反動（確率25%）

- PTS急騰→翌日寄り付きで信用買いが集中 → 短期利益確定が重なり急反落
- ¥2,200〜¥2,400レンジへの押し戻し → 逆に追加エントリー機会

### シナリオC: 中古高級品需要低迷（景気後退）（確率15%）

- 個人消費悪化 → ブランド品買取量減少 → 売上¥90,000M台に圧縮
- OP¥3,500M → EPS¥150 × 12x = ¥1,800（SL割れ）
