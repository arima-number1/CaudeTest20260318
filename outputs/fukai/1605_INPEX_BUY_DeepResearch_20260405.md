## 1. 企業概要

INPEXは日本最大の石油・天然ガス開発会社（E&P企業）。国内外の油ガス田の探鉱・開発・生産を行う資源メジャー。経済産業省が約20%を保有するが、運営は民間原理で行われている。

| 項目 | 数値 |
|------|------|
| 市場 | 東証プライム / TOPIX・MSCI Japan構成 |
| 参照株価 | 約¥2,400（2026年4月時点推定） |
| 時価総額 | 約¥3.2兆円 |
| 主要資産 | Ichthys LNG（豪州）、ADMA-OPCO（UAE）、ACG（アゼルバイジャン） |
| 配当利回り | 約3.5〜4.5%（原油高で増配期待） |
| PER（FY26E） | 約8〜10倍（資源株水準） |

## 3. 最重要カタリスト：ホルムズ封鎖とWTI$111の意味

### 3-1. なぜINPEXがこの局面で「最強」なのか

**逆説1：中東で生産しているのに、中東緊張の「受益者」になる**

- ADMA-OPCO（UAE）はホルムズ海峡より西に位置し、輸出はサウジのエムデュケースパイプライン迂回も可能
- カスピ海・アゼルバイジャンは中東と地理的に無関係
- Ichthys LNGはオーストラリア——地政学的に最安全地帯の一つ

**→ 生産が止まらない。しかし、油価だけが上がる。これがINPEXの「美しい構造」だ。**

**逆説2：中東産LNGが止まれば、Ichthys LNGの価値が上がる**

- ホルムズ封鎖 → カタールLNG（世界最大のLNG輸出国）の輸出が困難に
- アジア向けLNGスポット価格が急騰
- Ichthys LNGは豪州産 → 影響なし → スポット価格上昇の恩恵を最大限受ける
- 加えて：長期LNG契約は原油価格連動型 → 油価$111でベース収益も急増

### 3-2. 原油価格感応度の試算

INPEXの収益は原油価格と為替に強く連動する。深井モデルによる感応度分析：

| WTI価格 | 想定EPS | 想定配当 | 想定Fair Value |
|--------|--------|--------|--------------|
| $80（ベースライン） | ¥190 | ¥70 | ¥1,900 |
| **$111（現状）** | **¥280** | **¥110** | **¥2,800** |
| $130（上昇シナリオ） | ¥330 | ¥130 | ¥3,300 |
| $140（最強シナリオ） | ¥360 | ¥145 | ¥3,600 |

※為替：USD/JPY 159円を前提。1円感応度：EPS約+¥8〜10円/ドル
※EPS感応度：WTI$10/bbl = EPS約±¥40〜50（Ichthys LNG中心）

### 3-3. LNG需要の構造的追い風

**AIデータセンターの電力消費急増 → 火力発電（LNG）需要が増加**

- 日本のAIデータセンター電力需要は2030年までに3倍増予測
- 脱炭素路線でも「つなぎのエネルギー」としてLNG需要は旺盛
- Ichthys LNGの主要顧客（東ガス・関電等）はAIデータセンターの電力供給者
- **AI投資ブーム → LNG需要増 → INPEXの収益増**という連鎖が成立している

## 4. Fair Value 算定（深井モデル）

### 4-1. 算定手法：DCF + NAV（純資産価値）

E&P企業の評価は一般的なPERよりも**NAV（Net Asset Value：確認埋蔵量の現在価値）**が適切。油価シナリオ別に算定する。

### 4-2. セグメント別NAV試算

**前提条件：**
- 長期油価：$90/bbl（保守シナリオ）
- ドル円：150円（保守的な円高方向に調整）
- 割引率：8%

| 資産 | INPEX持分 | 推定NAV |
|------|---------|--------|
| Ichthys LNG（豪州） | 66.07% | **¥2.5〜3.0兆円** |
| ADMA-OPCO（UAE） | 12% | **¥0.4〜0.5兆円** |
| ACG（アゼルバイジャン） | 11% | **¥0.3〜0.4兆円** |
| カシャガン他 | 各8〜11% | **¥0.3〜0.5兆円** |
| 国内・その他 | — | **¥0.3〜0.5兆円** |
| **合計NAV（ベース）** | | **¥3.8〜4.9兆円** |

現在の時価総額：約**¥3.2兆円**

→ **保守長期油価$90でもNAVに対して18〜35%ディスカウント**で取引されている。

### 4-3. シナリオ別Fair Value

| シナリオ | 長期油価前提 | 算定方法 | Fair Value | 現在比 |
|--------|-----------|---------|-----------|------|
| ベア | $75 | NAV法（Ichthys重視） | ¥1,900 | ▲21% |
| ベース | $90 | NAV法 + PER8倍 | **¥2,900** | **+21%** |
| ブル（現状継続） | $110 | NAV法 + PER9倍 | **¥3,600** | **+50%** |
| $140シナリオ | $130（長期） | NAV法 + 地政学プレミアム | **¥4,200** | **+75%** |

**深井算定 Fair Value（12ヶ月ターゲット）：¥3,200**
**現在株価¥2,400に対する安全域：+33%**

## 6. 脱炭素・水素戦略（長期成長の芽）

INPEXは「2050年カーボンニュートラル」を掲げ、以下に投資：

- **CCUS（Carbon Capture Utilization & Storage）**：Ichthys付近の地中貯留プロジェクト
- **水素・アンモニア**：豪州での大規模グリーン水素製造（ATCO社と協業）
- **バイオ燃料・合成燃料**

これらは現在は収益に貢献しないが、**脱炭素テーマファンドが資源株でもINPEXを保有できる「免罪符」**となっており、ESGファンドの排除リスクを低減させている。

## 8. リスク要因

| リスク | 内容 | 深刻度 |
|------|------|------|
| 中東緊張の急速緩和 | ホルムズ再開 → WTI急落 | ★★★★☆ |
| 円高リスク | $10円の円高 = EPS▲¥80〜100 | ★★★☆☆ |
| Ichthys生産障害 | 機械トラブル・サイクロン等 | ★★☆☆☆ |
| 脱炭素政策加速 | 先進国のLNG規制強化 | ★★☆☆☆（長期リスク） |
| 地政学的悪化（豪州） | 低リスク地域だが | ★☆☆☆☆ |

**最大リスクは「中東和平」と「円高の同時発生」。** しかし現状のイラン・イスラエル・米国の対立構造が短期に解消されるシナリオは考えにくい。

## 9. 総合評価

信越化学（4063）が「AI素材×地政学受益」の化学版だとすれば、INPEXは「エネルギー安全保障×AI電力需要」の資源版だ。両社に共通するのは、**「地政学リスクが逆風ではなく追い風として働く」稀有な構造**である。

市場は今、INPEXを「オールドエコノミーの石油会社」として評価している。しかし実態は：
- **豪州産LNG：AI電力の燃料**
- **中東資産：ホルムズを迂回できる安全な生産地**
- **累進配当：キャッシュカウ**

この3つを組み合わせた投資テーゼは、市場がまだ完全に消化していない。

**判定：BUY / 12ヶ月Fair Value ¥3,200 / 現在比+33%**
