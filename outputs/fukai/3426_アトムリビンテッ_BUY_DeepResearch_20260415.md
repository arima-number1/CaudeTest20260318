## 4. DCFバリュエーション

### 4-1. WACC計算

**前提条件（全て明記）:**

```
【リスクフリーレート（Rf）】
  Rf = 2.423%（日本10年国債 / 2026-04-15 財務省）

【エクイティリスクプレミアム（ERP）】
  ERP = 6.0%（日本株ERP: Damodaran 2026年推計 / 新興国調整後）
  ※ 住宅リフォーム中小型株として標準的なERP適用

【ベータ（β）】
  β = 0.75（住宅リフォーム業種β / 景気感応度中程度・ディフェンシブ寄り）
  ※ 大型建設β0.9〜1.0より低い。地場密着・ストック型需要で変動小さい

【株主資本コスト（Re）】
  Re = Rf + β × ERP
  Re = 2.423% + 0.75 × 6.0%
  Re = 2.423% + 4.500%
  Re = 6.923% ≒ 6.9%

【資本構成（推計）】
  E/（D+E）= 85%（自己資本比率高く借入少ない）
  D/（D+E）= 15%

【負債コスト（Rd）】
  Rd = 1.5%（地銀・短期借入主体の中小企業金利）

【実効税率（Tc）】
  Tc = 30%

【WACC計算】
  WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
  WACC = [0.85 × 6.923%] + [0.15 × 1.5% × (1-0.30)]
  WACC = 5.885% + 0.158%
  WACC = 6.04% ≒ 6.0%
```

