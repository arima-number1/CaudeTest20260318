## 1. 企業概要・事業分析

### 1-1. 会社プロフィール

あさひ（証券コード: 3333）は、**日本最大の自転車専門チェーン**を運営する小売企業。「サイクルベースあさひ」ブランドを全国に展開し、自転車販売から修理・メンテナンスサービス、関連用品販売までをワンストップで提供する。

| 項目 | 内容 |

| 設立 | 1954年（大阪府） |
| 本社 | 大阪府大阪市 |
| 従業員数 | 約3,500名（連結） |
| 店舗数 | 約500店舗（全国） |
| 事業年度 | 2月期 |
| 市場 | 東証プライム |

### 1-2. セグメント・プロダクト構成

```
売上高構成（推定）
┌─────────────────────────────────────────────┐
│ 自転車本体販売         約55〜60%            │

│  ├─ シティサイクル                          │

│  └─ ヘルメット義務化追い風                  │
└─────────────────────────────────────────────┘
```

### 1-3. ビジネスモデルの特徴

**強み①：「販売＋修理」のフルサービスモデル**
- 修理・メンテナンス部門は高粗利（推定35〜40%）かつリピート顧客を創出
- 購入後も顧客接点を維持する「ライフサイクル型」収益構造

**強み②：全国500店舗ネットワーク**
- 競合他社（ドン・キホーテ、ホームセンター等）との差別化は「専門性」
- 地域密着型の修理受付は物理的参入障壁

**強み③：電動アシスト自転車（e-bike）シフト**
- 単価上昇（シティサイクル平均¥30,000 → e-bike平均¥80,000〜¥150,000）
- 少子高齢化・都市交通コスト意識の高まりが追い風

**強み④：ヘルメット努力義務化（2023年〜）による用品需要増**
- アクセサリー部門の収益底上げ効果継続

## 4. DCFバリュエーション

### 4-1. 前提条件（全明記）

```
【割引率の前提】
────────────────────────────────────────────
リスクフリーレート(Rf)    : 1.5%（JGB10年、2026年4月時点）
エクイティリスクプレミアム : 5.5%（日本株ERP：深井採用）
β（レバードβ）           : 0.75（小売業・低ボラ・推計）
   └─ 業界β参考値: 0.65〜0.85
CAPM によるKe             : 1.5% + 0.75 × 5.5% = 5.625%

負債コスト(Kd)            : 1.0%（低金利・健全BS）
実効税率                  : 30%
Kd after-tax              : 0.7%

資本構成（推定）
  株式時価総額            : 約320億円
  純有利子負債            : 約▲30億円（実質ネットキャッシュ）
  EV ≒ 時価総額           : 約320億円
  E/(E+D)                  : 約95%  D/(E+D): 約5%

WACC = 5.625% × 95% + 0.7% × 5%
     ≈ 5.34% + 0.04%
     ≈ 5.4%（採用: 5.5%）

【成長率前提】
フェーズ1（FY2027〜FY2031）5年間:
  売上高成長率 : 3.5%〜4.5%（e-bike需要・中計）
  EBIT成長率  : 5%〜7%（利益率改善）
  FCF成長率   : 5%〜6%

フェーズ2（FY2032以降）永続成長率 g : 1.0%
  └─ 名目GDP成長率（低め）に整合

【FCF算出ベース】
  FY2026/2月期 基準FCF : 30億円（中間値採用）
────────────────────────────────────────────
```

### 4-2. DCF計算（基本ケース）

```
FCF予測（基本ケース: g=5.5%/年、WACC=5.5%）
単位: 億円

        FCF    割引係数    現在価値
FY2027: 31.7   0.9479      30.0
FY2028: 33.4   0.8986      30.0
FY2029: 35.3   0.8516      30.1
FY2030: 37.2   0.8068      30.0
FY2031: 39.3   0.7651      30.0
─────────────────────────────────
予測期間PV合計:           150.1億円

ターミナルバリュー算出:
  TV = FCF2031 × (1+g) / (WACC - g)
     = 39.3 × 1.01 / (5.5% - 1.0%)
     = 39.7 / 4.5%
     = 882億円
  TVの現在価値 = 882 × 0.7651 = 674.9億円

EV = 150.1 + 674.9 = 825億円

株式価値の算出:
  EV         : 825億円
  + 現預金等  : +70億円
  - 有利子負債: -40億円
  = 株式価値  : 855億円

1株当たり理論株価:
  発行済株式数: 約4,600万株（推定）
  理論株価 = 855億 ÷ 4,600万株
           ≈ ¥1,859 / 株
```
