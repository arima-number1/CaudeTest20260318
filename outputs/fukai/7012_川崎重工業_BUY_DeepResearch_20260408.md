## 1. 企業概要・事業分析

### 1-1. 川崎重工業の事業構造

川崎重工業（Kawasaki Heavy Industries / KHI）は1896年創業の総合重工業メーカー。売上高は連結で概ね**2.0〜2.3兆円規模**（FY2025実績ベース）。事業は以下の6セグメントに大別される。

| セグメント | 主力製品 | 売上構成比（概算） | 特性 |
|-----------|---------|-----------------|------|
| **航空宇宙・防衛** | P-1哨戒機、潜水艦（そうりゅう/たいげい型）、戦闘ヘリCH-47、BK117、F/A-18部品 | **約23〜27%** | 防衛省・JAXA向け、高参入障壁 |
| **車両** | 新幹線（N700S等）、地下鉄、LRV、海外鉄道 | 約12〜15% | 米国NYC地下鉄・台湾THSR等 |
| **エネルギー・環境** | ガスタービン、水素液化装置、水素運搬船 | 約13〜16% | 水素サプライチェーン中核 |
| **精密機械・ロボット** | 産業用ロボット（Kawasaki Robotics）、油圧機器 | 約14〜17% | 自動車・半導体向け好調 |
| **船舶・海洋** | LNG運搬船、液化水素運搬船"すいそ ふろんてぃあ" | 約8〜10% | 水素船の世界初実証完了 |
| **モーターサイクル＆エンジン** | Ninja・Z系バイク、汎用エンジン | 約18〜22% | 北米・欧州高収益 |

> **注:** 各セグメント売上構成比は公開有価証券報告書・IR説明会資料（FY2024/2025）をベースとした参考推定値。

### 1-2. 競合優位性（Competitive Moat）

**①防衛セグメント：参入障壁は極めて高い**
- 潜水艦は川崎重工と三菱重工の**2社独占体制**（交互建造）。潜水艦建造は国家安全保障上の機密技術を含み、新規参入は事実上不可能。
- P-1哨戒機はボーイングP-8との競合を国産代替で封じた例。MTBF・航続距離ともに世界水準。
- 防衛装備庁・海上自衛隊との長期契約構造→**受注残（バックログ）の視認性が高い**。

**②水素：商業化フロントランナー**
- 液化水素の海上輸送を世界で初めて実証済み（豪州→神戸、2022年）。
- 水素液化装置・コールドエバポレーター等の基幹技術を内製保有。
- NEDOプロジェクトでの中心的役割→政府補助金フローの安定受益者。

**③ロボット：グローバルプレゼンス**
- duAro等の協働ロボットで中小製造業市場を開拓。ファナック・安川と並ぶ国内トップ3。

### 2-3. 水素事業：黒字化時期とキャッシュドレイン定量化

#### 水素事業の現状

川崎重工の水素戦略は「液化水素サプライチェーン構築」を核心に置く。豪州→日本の液化水素輸送を世界初で実証した唯一の企業。

```
水素事業ステージ（KHI）:
FY2022 ─ 液化水素輸送実証成功（すいそふろんてぃあ号）
FY2023 ─ 商業規模サプライチェーン設計開始
FY2024 ─ Hy touch 神戸（水素ステーション運営）拡大
FY2025 ─ 大型液化水素運搬船（商業1号機）設計・受注活動
FY2026〜2028 ─ 商業化本格準備フェーズ
FY2029〜2030 ─ 商業運転開始目標（豪州→日本 大規模）
FY2031〜2032 ─ 黒字化目標（深井推定）
```

|------|-------------------------------|------|
| R&D（水素関連） | **約200〜300億円/年** | 全社R&Dの30〜40%が水素関連 |
| 設備投資（水素インフラ） | **約150〜250億円/年** | 液化装置・貯蔵設備・輸送設備 |

> **重要:** 全社FCF（後述）が概ね500〜1,000億円水準であることを考慮すると、水素事業のドレインは**FCFの30〜60%**に相当する。財務的に許容限界に近いため、商業化遅延は株価下押しリスクとなる。

#### 黒字化シナリオ分析

| シナリオ | 黒字化時期 | 前提条件 |
|---------|----------|---------|
| **楽観（強気）** | **FY2030〜2031** | 豪州プロジェクト商業化成功、水素価格30円/Nm³以下達成、カーボンプライシング導入 |

## 4. DCFバリュエーション

### 4-1. 前提条件（全項目明記）

| パラメータ | 設定値 | 根拠 |
|-----------|--------|------|
| **リスクフリーレート（Rf）** | **2.411%** | 日本10年国債（JGB10Y）2026年4月7日財務省公表値 |
| **エクイティリスクプレミアム（ERP）** | **5.5%** | Damodaran日本市場ERP2026年推定（成熟市場+カントリーリスク） |
| **β（ベータ）** | **1.15** | 東証重工業セクター実績β。防衛・水素テーマで市場感応度高め |
| **株主資本コスト（Re）** | **Re = 2.411% + 1.15 × 5.5% = 8.74%** | CAPM |
| **負債コスト（Rd）** | **1.8%** | 国内銀行借入・社債加重平均金利（推定） |
| **実効税率（Tc）** | **30%** | 日本法人税+地方税 |
| **D/E比率（推定）** | **1.2** | E/(D+E)=45.5%, D/(D+E)=54.5% |
| **WACC** | **WACC = 0.455×8.74% + 0.545×1.8%×0.70 = 4.65%** | 後述詳細計算参照 |
| **ターミナル成長率（g）** | **1.5%** | 日本GDP成長率（実質1%+インフレ0.5%）、防衛・水素で若干上乗せ |
| **FCF予測期間** | **10年（FY2026〜FY2035）** | — |

### 4-2. WACC詳細計算

```
Re = 2.411% + 1.15 × 5.5%
   = 2.411% + 6.325%
   = 8.736% ≈ 8.74%

WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.455 × 8.74%] + [0.545 × 1.8% × (1-0.30)]
     = 3.977% + 0.545 × 1.26%
     = 3.977% + 0.687%
     = 4.664% ≈ 4.66%
```
