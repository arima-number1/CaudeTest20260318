## 4. DCFバリュエーション

### 4-1. WACC計算前提

#### 基本パラメータ

| パラメータ | 数値 | 根拠・出所 |
|-----------|------|----------|
| **Rf（リスクフリーレート）** | **2.409%** | 日本10年国債（JGB10Y）財務省日次 R8.4.16 ※絶対厳守 |
| **β（ベータ）** | **0.85** | 証券業の国内中堅銘柄として推計（市場β1.0より低め：対面型で安定収益） |
| **ERP（株式リスクプレミアム）** | **5.5%** | 日本株ERP標準値（Damodaran Japan ERP 2026推計） |
| **Re（株主資本コスト）** | **2.409% + 0.85×5.5% = 7.084%** | CAPM計算 |
| **Rd（負債コスト）** | **0.8%** | 証券会社の短期借入コスト（低金利環境） |
| **Tc（実効税率）** | **30%** | 日本標準法人実効税率 |
| **D/（D+E）** | **20%** | 証券業として負債比率保守推計 |
| **E/（D+E）** | **80%** | |
| **WACC** | **= 80%×7.084% + 20%×0.8%×(1-30%) = 5.667% + 0.112% = **5.78%**** | |

#### WACC詳細計算

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.80 × 7.084%] + [0.20 × 0.80% × 0.70]
     = 5.667% + 0.112%
     = 5.78%
```
