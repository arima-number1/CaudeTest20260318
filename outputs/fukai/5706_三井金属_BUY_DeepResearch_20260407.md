## 1. 企業概要・事業分析

### 1-1. 会社概要

三井金属鉱業（以下「三井金属」または「MKK」）は、1950年設立の総合非鉄金属メーカーで、鉱山開発から精錬・加工・機能材料まで垂直統合型の事業構造を持つ。東証プライム上場、時価総額は現在株価¥31,250 × 発行済株式数（約70.5百万株）＝**約2,203億円**。

> **注:** 発行済株式数は直近IR資料ベース推計値。厳密な値は最新有価証券報告書を参照のこと。

### 1-2. セグメント構成（FY2025実績ベース推計）

| セグメント | 主要製品 | 売上高構成比（推計） | 営業利益率（推計） |
|-----------|---------|-----------------|----------------|
| **銅箔・機能材料** | 電池用極薄銅箔、半導体用圧延銅箔、電解銅箔 | 約25〜28% | 高収益（推計15〜20%） |
| **非鉄金属精錬** | 亜鉛・鉛精錬、銅製錬 | 約30〜35% | 中程度（推計5〜8%） |
| **機能材料** | 酸化亜鉛、電池材料、表面処理材料 | 約15〜18% | 中高（推計10〜14%） |
| **鉱山** | 三菱マテリアルとのカミオカ鉱山他、海外権益 | 約10〜12% | 変動大 |
| **その他（不動産含む）** | 倉庫・不動産賃貸等 | 約8〜10% | 安定 |

**三井金属の核心的優位:**
1. **極薄銅箔技術（4μm以下）:** 業界最高水準の薄膜化技術。EV電池のエネルギー密度向上に直結。競合比で技術リードは推定2〜3年。
2. **垂直統合:** 鉱山権益→精錬→箔加工→顧客直販の一貫体制でマージン保護。
3. **顧客ロックイン:** トヨタ・パナソニック（プライムプラネット）との長期供給契約。韓国LG・SKとのサプライヤー認定。
4. **半導体銅箔の技術流用:** 半導体パッケージ用極薄銅箔の知見がEV電池向けにシナジー。

```

  2023Q1: USD 9.5/kg（過剰供給で下落）
  2024Q1: USD 8.8/kg（底打ち）
```

銅原料（LME銅）上昇局面でも、加工マージン（Conversion Margin）の維持が鍵。MKKは長期契約比率が高く（推定70%超）、スポット価格変動の直撃を一定程度緩衝。

#### (d) DRAM需要との相乗効果 — 定量化試算

**前提:**
- DRAMサイクル回復によりHBM（High Bandwidth Memory）需要急増
- HBM1層あたり銅箔使用量は従来DRAM比**+40〜60%**（積層数増加）
- 世界DRAM生産量：2025年→2026年 前年比+22%成長（推計）

| 需要源 | 銅箔需要増加量（推計） | MKKへの貢献（推計売上） |
|-------|-------------------|-------------------|
| EV電池向け（世界EV+28%） | +8,000〜10,000トン/年 | +¥50〜65億（年間） |
| DRAM/HBM向け半導体銅箔 | +2,000〜3,000トン/年 | +¥18〜25億（年間） |
| **合計相乗効果** | **+10,000〜13,000トン/年** | **+¥68〜90億（年間）** |

> **深井コメント:** この相乗効果（年間+¥70〜90億）は現在の市場コンセンサスに**十分織り込まれていない可能性**が高い。特にDRAM×HBMの銅箔需要増は2026年後半から顕在化するタイミング。ここに株価の非対称的アップサイドが存在する。

**TC/RC圧縮の主因:**

```
【亜鉛精錬コスト構造比較】
```

### ⑤ PBR 4.92倍の正当化：銅箔事業のグロースバリュー算定

#### 現在のBPS・PBRの確認

```
BPS（一株純資産）: ¥6,348  [リアルタイム値 / 絶対使用]
現在株価:         ¥31,250  [リアルタイム値 / 絶対使用]
PBR:              4.92倍   [リアルタイム値 / 絶対使用]
ROE:              18.4%    [リアルタイム値 / 絶対使用]
```

#### PBR正当化フレームワーク（Gordon Growth Model分解）

```
PBR = ROE × (1 - b) / (r - g)  ※簡略ゴードン展開版

あるいは

PBR = ROE / r  ※成長率考慮なし（下限）
    = 18.4% / 8.5%（WACC）
    = 2.16倍  ← これが「ゼロ成長」の最低正当PBR

成長プレミアム（銅箔グロース）:
現在PBR 4.92倍 / ゼロ成長PBR 2.16倍 = 2.28倍のグロースプレミアム

∴ 市場は銅箔事業に対して、純資産の2.28倍分の「将来超過収益」を評価している
```

#### 銅箔事業単独グロースバリュー算定

| 前提 | 値 |
|------|----|
| 銅箔セグメント現在売上（推計） | ¥400〜450億 |
| 銅箔セグメント成長率（2026〜2030E） | 年率+18〜22% |
| 銅箔セグメント営業利益率 | 16〜20% |
| EV×DRAM相乗効果反映後の2030E売上 | ¥900〜1,100億 |
| 銅箔事業EV（EV/EBITDA 20〜25倍） | ¥2,880〜5,500億 |
| 銅箔事業1株価値 | ¥8,200〜15,700/株 |

#### PBR 4.92倍の正当化結論

```
銅箔グロースバリュー（1株）:    ¥8,200〜15,700
精錬・機能材料（安定CF）バリュー: ¥12,000〜16,000

────────────────────────────
```
