### 2.4 バリュートラップ判定フレームワーク

バリュートラップの本質は「**安いが、永遠に安いまま**」の状態。以下4軸で検証：

#### 軸①：資本効率（ROE）の問題

```
ROE = 純利益 ÷ 自己資本

推定ROE: ¥7〜12億 ÷ ¥130〜160億 ≒ 4.5〜7.5%

→ Rf(2.38%)は上回るが、ERP(5.5%想定)加味したRe(≒9.5%)を下回る
→ 「資本コスト割れROE」 ⚠️ バリュートラップ要素①
```

#### 軸②：利益の成長性

```
売上高CAGR（過去5年）: ≒ 0〜+3%（横ばい〜微増）
営業利益CAGR（過去5年）: ≒ -2〜+2%（停滞）
→ 成長によるPBR改善圧力が弱い ⚠️ バリュートラップ要素②
```

#### 軸③：還元政策（カタリストの有無）

```
配当性向: 推定40〜50%（比較的高め）
自社株買い: 散発的に実施（継続性は低め）
東証PBR是正要請への対応: 開示不十分 ⚠️

→ 還元で株価を引き上げる積極的意欲がやや乏しい
→ ただし余力は十分 ✅ カタリスト待ち状態
```

#### 軸④：事業の質・永続性

```
石油ファンヒーター市場:
  国内出荷台数トレンド: ゆるやかな長期縮小傾向（年▲1〜2%）
  ただし寒冬年に急増、暖冬年に急減
  2026年現在：原油高（WTI$112）が需要に中立〜ネガティブ影響

加湿器市場:
  コロナ禍以降定着（感染症対策認知向上）
  成長余地あり → バランサー機能 ✅
```

## 4. DCFバリュエーション

### 4.1 前提条件の設定

#### マクロパラメータ

| パラメータ | 値 | 根拠 |
|----------|---|------|
| Rf（リスクフリーレート） | **2.380%** | JGB10Y（社長確認値） |
| ERP（株式リスクプレミアム） | **5.5%** | 日本市場標準（Damodaran参照） |
| β（ベータ） | **0.65** | 低ボラティリティ消費財・季節株 |
| **Re（株主資本コスト）** | **2.380% + 0.65 × 5.5% = 5.955%** | CAPM |
| 実効税率（Tc） | **30%** | 日本法人標準 |
| Rd（負債コスト） | **0.5%** | 実質無借金のためほぼゼロ |
| E/(D+E) | **98%** | 自己資本比率82〜85%→時価ベース概算 |
| D/(D+E) | **2%** | — |
| **WACC** | **5.853%** | 下記計算参照 |

#### WACC計算

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]

     = [0.98 × 5.955%] + [0.02 × 0.5% × (1-0.30)]

     = 5.836% + 0.007%

     = 5.843% ≒ 5.85%（採用値）
```

#### 原油価格感応度の考慮（コモディティ前提）

```
WTI原油直近価格: $112.16/bbl（2026/04/06）

ダイニチへの影響経路:
1. 消費者の灯油コスト意識上昇 → ファンヒーター需要に中立〜微ネガ
2. 樹脂・金属部材コスト上昇 → 製造コスト圧迫（▲0.3〜0.8%）
3. ただし電気暖房との比較で石油機器が相対的に安価維持
   （電気代高騰↔灯油高騰のバランス）

→ 原油$112水準は: 中長期的に「微ネガティブ」と判定
   売上高成長率を基本±0%（成長なし）として設定
```

### 4.3 DCF計算（基本シナリオ）

```
【ステップ1：FCFの現在価値（明示期間5年）】

割引率 WACC = 5.85%

PV(FY2025) = 7.75 ÷ 1.0585^1  = 7.32億
PV(FY2026) = 8.37 ÷ 1.0585^2  = 7.48億
PV(FY2027) = 9.10 ÷ 1.0585^3  = 7.70億
PV(FY2028) = 8.82 ÷ 1.0585^4  = 7.08億
PV(FY2029) = 8.33 ÷ 1.0585^5  = 6.33億
─────────────────────────────────
明示期間FCF現在価値合計: 35.91億

【ステップ2：ターミナルバリュー（TV）】

TV = FCF_2030 × (1+g) ÷ (WACC - g)
   = 8.33 × 1.005 ÷ (0.0585 - 0.005)
   = 8.37 ÷ 0.0535
   = 156.4億

TV現在価値 = 156.4 ÷ 1.0585^5 = 118.8億

【ステップ3：事業価値（Enterprise Value）】

EV = 35.91 + 118.8 = 154.7億

【ステップ4：株主価値（Equity Value）】

+ 現預金       = +90億（推定中間値）
+ 投資有価証券  = +25億（推定）
- 有利子負債    = -0億（実質ゼロ）
─────────────────────────────────
株主価値 = 154.7 + 90 + 25 = 269.7億

【ステップ5：1株当たり理論株価（基本シナリオ）】

発行済株数: 約18,000千株（推定）
```
