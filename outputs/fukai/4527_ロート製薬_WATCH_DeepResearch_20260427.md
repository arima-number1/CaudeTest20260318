## 5. カタリスト分析

### 正のカタリスト

**1. AVI（英Asset Value Investors）圧力 → 再生医療コスト削減**
- AVI（持分2.3%、第6位株主）が2025年4月に再生医療事業への「無謀な投資」を問題視
- 2025年5月：杉本社長退任→瀬木英俊新社長就任（実質AVI要求による人事）
- 再生医療事業は赤字・費用重荷の主因 → 縮小・撤退すれば**OPMが100〜200bp改善する可能性**
- **Rationale:** FY2027中計目標OPM12.6%（405億/3,405億=11.9%→460億/3,650億=12.6%）達成にはコスト削減が必須。AVIの要求と会社中計が一致方向にある点はポジティブ。

**2. 5月13日通期決算 + FY2027ガイダンス発表**
- 通期 OP405億円を着地・上振れ → 市場安堵
- FY2027ガイダンス: 中計目標通り460億円 → Base TP達成
- FY2027ガイダンス: 460億円超（再生医療削減効果先行） → Bull TP接近

**3. 中国小売好調・海南島免税継続**
- 宮迫AN「中国小売非常に好調」発言がロートのアジアセグメントを後押し
- EYS社（漢方・ウェルネス）への中国需要は構造的

**4. インフレ→トレードダウン受益**
- メラノCC・スキンアクアが価格帯の恩恵ゾーンに位置
- コーセー・ポーラが高コンセンサスで下方修正リスクを抱える中、相対優位

### 負のカタリスト

**1. ナフサ価格高騰（ホルムズ起因）**
- 宮迫AN「トイレタリーは原材料コスト上昇（ナフサ高）」
- 現在ナフサは前年比+70%水準（浅井スコア記録より）
- ロートの主力スキンケア・OTC製品は石油由来原材料（界面活性剤・ポリマー）を多用
- 2025年5月に価格改定実施済みだが、ナフサが高止まりなら追加値上げか原価率悪化が避けられない
- **影響度試算（深井推計）:** 売上高の原価率が1pt上昇するなら、粗利56.1%（yfinance）の観点からOP約32億円（= 3,405億円×1%）の押し下げ。全体OP405億円比で▲8%のインパクト（要注意）

**2. 訪日客（インバウンド）減少リスク**
- 宮迫AN「訪日客減少→ドラッグストア経由でロート製薬にもリスク波及」
- ただし「相対的に小さい」とも発言。ロートはインバウンド特需依存度がコーセー・ポーラより低い
- 中東情勢（ホルムズ封鎖）→ 原油高→ 航空燃油コスト上昇→ 訪日客価格上昇→ 旅行控えの連鎖リスク

**3. 東南アジア工場（サプライチェーン）リスク**
- 宮迫AN「東南アジア工場のサプライチェーンリスク」
- ミャンマー新工場（3Q連結開始）は政情リスク内包
- マレーシア・タイ等の工場がホルムズ起因の原材料逼迫に晒される可能性

**4. FY2027ガイダンス保守的リスク**
- 新社長（瀬木氏）が初年度ガイダンスを保守的に設定する可能性
- OP460億円（中計目標）未達ガイダンス（例: 420億円）→ 株価失望売り → TP Bear接近

## 7. 不動産含み益考察

### 保有不動産の状況

ロート製薬は大阪市生野区巽西1-8-1に本社・生田工場を保有。1895年創業の老舗企業で、大阪中心部周辺の工場用地は歴史的に簿価が取得原価（低廉）に固定されており、時価との乖離が存在する可能性。

| 項目 | 内容 |
|------|------|
| PBR | 1.90x → 純資産比は保守的 |
| BPS | 1,272円/株 × 22,596万株 = 純資産約2,875億円 |
| 推定含み益 | 公式開示なし。ただしBPS1,272円に対して株価2,417円のPBR1.9倍は有形資産超過分を市場が評価している形 |

**NAV試算（暫定・深井推計）:**
- 時価総額: 5,462億円
- ネットキャッシュ: +223億円（現預金741億円 - 有利子負債518億円）
- コア事業EV: 5,239億円
- 不動産含み益（推定）: 非開示のため保守的に0〜100億円（不確実性大）

**触媒:** 再生医療事業縮小に伴う関連施設・設備の売却やB/Sスリム化が進めば、不動産売却益・再評価益が発生する可能性。ただし、主要工場は継続使用が前提のため短期的なカタリストとしては弱い。
