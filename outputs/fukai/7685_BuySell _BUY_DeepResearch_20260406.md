## 1. 企業概要・事業分析

### 1-1. 会社プロフィール

BuySell Technologies株式会社は、リユース（中古品買取・販売）事業をテクノロジーで再定義する「リユースDX」企業。2002年創業、2019年東証マザーズ上場（現：東証グロース/スタンダード）。本社: 東京都新宿区。

**コアビジネス**: 出張買取を主軸とした総合リユース事業
- 着物・ブランド品・貴金属・骨董品・家電・家具など幅広いカテゴリ
- **完全自社オペレーション**（フランチャイズ非依存）

## 4. DCFバリュエーション

### 4-1. 前提条件（全項目明記）

#### マクロ前提

| パラメータ | 値 | 根拠 |
|-----------|-----|------|
| リスクフリーレート（Rf） | **2.380%** | JGB10Y（2026-04-06 社長確認値） |
| エクイティリスクプレミアム（ERP） | **6.0%** | 日本市場標準（Damodaran 2025日本ERP参照） |
| β（ベータ） | **1.45** | 東証スモールキャップ成長株・リユース業態調整値 |
| 株主資本コスト（Re） | **11.08%** | Rf + β × ERP = 2.380% + 1.45 × 6.0% |
| 負債コスト（Rd） | **1.2%** | 低金利・低Debt体質を反映 |
| 実効税率（Tc） | **30%** | 日本標準 |
| D/(D+E) | **10%** | 低Debt体質を反映 |
| E/(D+E) | **90%** | 同上 |
| **WACC** | **10.05%** | 下記計算参照 |

**WACC計算:**
```
WACC = E/(D+E) × Re + D/(D+E) × Rd × (1-Tc)
     = 0.90 × 11.08% + 0.10 × 1.2% × (1-0.30)
     = 9.972% + 0.084%
     = 10.056% ≒ 10.05%
```
