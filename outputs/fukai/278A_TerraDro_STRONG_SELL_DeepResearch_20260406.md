## 1. 企業概要・事業分析

### 1-1. 会社プロフィール

TerraDrone株式会社（東証グロース：278A）は2021年設立の国内ドローン・エアモビリティスタートアップ。2023年にIPOを実施。本社：東京。主要事業はドローンによるインフラ点検・測量・空撮・物流支援に加え、海外ドローン関連企業の買収・統合によるグローバルプラットフォーム構築を標榜。

### 1-2. セグメント・プロダクト構成

```
TerraDrone (278A)
├── [セグメント1] ドローンソリューション事業（国内）
│     ├── インフラ点検（電力・ガス・通信鉄塔）
│     ├── 測量・3Dマッピング（建設・土木）
│     ├── 農業ドローン散布
│     └── 緊急・防災活用
│
├── [セグメント2] グローバルM&A・統合事業
│     ├── 欧州ドローン企業買収（複数件）
│     ├── アジア・中東展開
│     └── ソフトウェアプラットフォーム（UTM: 無人機交通管理）
│
└── [セグメント3] エアモビリティ関連（先行投資期）
      ├── eVTOL（電動垂直離着陸機）周辺
      └── 防衛・官需向け（検討・受注開発段階）
```

### 1-3. 事業モデルの実態

- **主力収益**: ドローン測量・点検の「受託サービス」— 高度化されたSaaSではなく、労働集約型サービス業に近い
- **グローバル拡張**: 複数の海外ドローン企業をM&Aで統合する「ロールアップ戦略」を採用。のれん計上・統合コスト重く、収益貢献が限定的
- **防衛テーマとの実際の距離**: 防衛省との直接契約・大型受注は現時点で公表なし。「防衛費倍増の恩恵を受ける可能性がある」というテーマ株文脈での評価が主

### 2-2. 財務実態 vs テーマ株プレミアムの乖離

#### ❶ ROE -43.4%が示す本質

```
ROE = -43.4%
→ 株主資本を毀損し続けている企業
→ 「成長投資フェーズ」の赤字と「構造的赤字」の区別が重要
```

- EPS = ¥-61 / BPS = ¥519 → 純資産に対して年率▲12%近くを焼却中
- **成長投資型赤字（Amazon型）** であれば、売上成長率・ARR拡大・FCFマージン改善トレンドが見えるはず
- TerraDroneの公開財務では、**M&Aのれん・統合費用による赤字が主因**であり、コアサービスの収益性が見えにくい構造

#### ❷ PBR 9.36倍の異常性

```
現在株価: ¥4,855
BPS:      ¥519
PBR:      9.36倍
```

- **PBR 9.36倍** = 純資産の9倍以上を市場が評価
- ROE マイナス企業のPBR 9倍超 → **「将来の爆発的収益」への純粋期待値**であり、現在の事業価値とは無関係
- 比較: トヨタPBR 約1.1倍、ACSL（競合）PBR 約3〜5倍（赤字成長期）
- **ドローンテーマプレミアムが純資産の8倍以上乗っている**

#### ❹ ドローン×国策テーマの「虚実仕分け」

```

#### ❺ M&Aロールアップ戦略の危険性

```
