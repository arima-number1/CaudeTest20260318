## ■ 企業概要

**2607 不二製油グループ本社**（FUJI OIL GROUP INC.）

大阪本社の食品素材・特殊油脂メーカー。1950年創業。売上¥7,621億の中堅食品材料大手。主力はカカオバター代替品（CBE: Cocoa Butter Equivalent）、製菓用チョコレート素材（ブラマー）、大豆タンパク・乳化油脂。グローバル食品大手（ネスレ・モンデリーズ等）へのB2B供給が主体。

**主要セグメント**

| セグメント | 主要製品 | 競争優位 |
|------------|---------|---------|
| 油脂（主力） | CBE・チョコレート素材 | カカオ代替の独占的技術・顧客基盤 |
| 大豆素材 | 分離大豆タンパク・豆乳素材 | 欧米健康食品需要に乗る成長 |
| その他 | 業務用マーガリン・ショートニング | 安定収益源 |

**主要拠点**：大阪（本社）、東京、欧州ブラマー社（オランダ・ベルギー）、カナダ（新工場 H2 2026稼働）、フランス子会社（¥170億取得 ひまわり油供給拠点）

## ■ フェアバリュー計算

### ① Forward PER法（ベースケース）

```
Forward EPS ¥186.13 × PER 19.9x = FV ¥3,704
根拠: 食品素材セクター平均PER 18-22x
     CBE独占プレミアム(+2x) vs Net Debt重荷(▲2x) ≒ 差し引き19.9x採用
```

### ② EV/OP正規化法（12ヶ月メイン目標）

```
セクター適正 EV/OP = 14x（日系食品素材平均）
OP正規化想定: ¥510億（油脂¥270億 + その他事業 + カナダ寄与¥10億）
目標EV = ¥510億 × 14x = ¥7,140億
目標時価総額 = ¥7,140億 − Net Debt¥2,908億 = ¥4,232億
FV = ¥4,232億 ÷ 8,590万株 = ¥4,928
→ 保守調整（借入金削減タイムラグ50%反映）: ¥4,200
```

### ③ Gordon Growth Model（検証）

```
β = 0.75（食品材料+カカオコモディティリスク）
WACC = 2.392% + 0.75 × 5.5% = 6.517%（Rf=2.392%固定）
Forward ROE = 6.69%（EPS ¥186.13 ÷ BPS ¥2,780）
FV（Forward ROE） = ¥2,780 × 6.69% / 6.517% = ¥2,854

Target ROE = 10%（借入金削減後の正規化水準）
FV（Target ROE） = ¥2,780 × 10% / 6.517% = ¥4,267
```

### ④ 正規化EPS法（ブルケース）

```
正規化EPS算定:
  全社OP ¥510億 × (1−税率30%) × (1−少数株主比率) ≒ 純利益¥250億
  + 借入金¥2,000億削減完了時 金利負担▲¥60億/年 → 純利益追加¥42億
  正規化純利益 ≒ ¥292億
  正規化EPS = ¥292億 ÷ 8,590万株 = ¥340
  FV = ¥340 × PER 17.5x = ¥5,950（完全正規化時の上限）
```

| 手法 | FV | 重み | 加重値 |
|------|-----|------|--------|

**→ TP（12ヶ月）= ¥4,200**（EV/OP正規化法を主軸、借入金削減の進捗確認次第でアップサイド拡大余地）
