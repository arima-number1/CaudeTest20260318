### 2.3 有事ベータ推計（★社長指示必須項目★）

#### 通常時β vs 有事時β の推計方法論

**通常時β（平常時市場ベータ）:**
東レは素材・化学セクターの大型株であり、過去実績ベースのβ（TOPIX対比）は概ね**0.85〜1.10**の範囲に収まる。

```
通常時β推計:
 参照期間: 過去3年（2022〜2025）
 市場: TOPIX（東証プライム）
 推計β（通常時）: β_normal ≈ 0.95
 （大型バリュー・素材株として市場連動型）
```

**有事ベータの推計方法論（深井モデル）:**

有事時のβは「通常時β」と「防衛・地政学ベータ」の合成として定義する。

```
有事時β算出式（深井モデル）:
β_wartime = β_normal × (1 - w_defense) + β_defense × w_defense

変数定義:
 β_normal   = 0.95（通常市場ベータ）
 β_defense  = 炭素繊維/防衛素材の有事感応度ベータ
 w_defense  = 防衛関連事業の利益貢献比率（推定30〜40%）

β_defense の推計根拠:
 ・ロシア・ウクライナ勃発時（2022年2月）：
    防衛素材株は市場下落時に逆行高 → 実質的にβ < 0（逃避需要）
 ・ただし東レは汎用繊維も保有（市場連動部分残存）
 ・防衛素材純粋プレイ（Hexcel/Solvay）有事時実績β: -0.3〜+0.4
 ・東レの防衛比率（推定30%）を加重 → β_defense ≈ 0.10〜0.30
```

**有事ベータ推計結果:**

| シナリオ | w_defense | β_defense | β_wartime | 解釈 |
|---------|-----------|-----------|-----------|------|
| 通常時（ベースライン） | 0% | - | **0.95** | 市場追随 |
| 地政学緊張（現在） | 30% | 0.20 | **0.73** | 市場下落への耐性UP |
| 北朝鮮エスカレーション | 40% | 0.10 | **0.61** | 防衛プレミアム顕在化 |
| 台湾有事突入 | 60% | -0.10 | **0.32** | 逆相関傾向（防衛避難） |
| 極東全面有事 | 80% | -0.30 | **0.43倍（逆行）** | 逃避買い + サプライ懸念混在 |

> **深井コメント**: 有事時βの低下（0.95→0.32〜0.43）は、東レが市場暴落局面での**相対的な下落耐性**を持つことを示す。ただし「極東全面有事」では東レ自体の生産拠点リスク（日本国内工場）もあり、単純な防衛銘柄としての逆行高は限定的になる可能性に注意。

**有事ベータを用いた株主資本コスト変化:**

```
Re = Rf + β × ERP
ERP = 5.5%（日本市場 標準ERP）

通常時: Re = 2.380% + 0.95 × 5.5% = 7.605%
緊張時: Re = 2.380% + 0.73 × 5.5% = 6.395%
有事時: Re = 2.380% + 0.32 × 5.5% = 4.140%

→ 有事時はReが低下 → DCF割引率低下 → 理論株価上昇
  （安全資産的性格の強化による逆説的なバリュエーション上昇）
```
