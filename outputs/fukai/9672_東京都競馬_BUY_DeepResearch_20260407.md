### ③ 「株価が上がらない」構造的原因の解明

#### (A) 東京都の保有比率問題

```
東京都：約36%保有（筆頭株主）
　│
　├── 資本政策への介入力：絶大
　├── 自社株買い：都の意向なしには事実上不可
　├── 大規模増配：都が不必要と判断すれば阻止可能
　├── 解散・清算：都の競馬政策と直結 → 現実的でない
　└── 資産売却（土地）：競馬事業継続が前提 → 不可
```

**評価：** 東京都は株主として利益最大化よりも**「競馬を通じた都民サービス・地域貢献」を優先**する主体である。資本効率改善（ROE向上・PBR改善）に積極的に動く誘因が根本的に欠如している。これが株価上昇の最大の構造的障壁である。

**定量インパクト：** 都の保有比率が30%→15%に低下するだけで、市場の見方が劇的に変わる可能性。ただし都の政策的文脈（競馬振興・馬主産業保護）からその可能性は**極めて低い（確率5%未満）**。

#### (B) 流動性の問題

| 指標 | 数値 | 評価 |
|------|------|------|
| 時価総額 | 約318億円 | 機関投資家参入の最低水準（500億円）を下回る |
| 日次出来高（推定） | 約2,000〜5,000株 | 極めて薄い |
| 売買代金（推定） | 約1,200〜3,000万円/日 | 機関投資家が1億円規模のポジションを取れない |
| 流通時価総額 | 約200億円（都保有除く） | 実質的な機関投資家参入は不可能水準 |

**評価：** 流動性の欠如は機関投資家・ヘッジファンドの参入を物理的に阻害する。仮にアクティビストが参入しようとしても、**市場で株を買い集める過程で株価が急騰・流動性が枯渇する**。これが流動性トラップの本質。

#### (C) 成長性ゼロ問題

- 馬券市場の長期トレンド：高齢化により2030年代以降は緩やかな縮小が予想される
- 新規ファン獲得：eスポーツ・IR・スポーツベッティング等との競合激化
- 地方競馬統廃合リスク：国の財政支援政策次第では再編圧力
- **DCFバリュエーション上、成長率g＝0〜1%が上限** → PER・PBRが低く評価されるのは合理的

#### (D) 資産の不動性（最重要バリュートラップ要因）

```
大井競馬場の土地（約39万㎡）
　│
　├── 現状：競馬法上の競馬場として使用中
　├── 売却：競馬法・東京都条例により事実上不可能
　├── 再開発：競馬事業を継続しながらの部分開発は限定的
　├── 移転：代替地の確保・都の政治判断・莫大なコストが必要
　└── 廃止：都の政策転換なしにはあり得ない（黒字事業）
```

**現実的な出口シナリオ：**

| シナリオ | 実現確率 | 所要時間 |
|---------|---------|---------|
| 一部エリアの商業施設・オフィス開発 | 15〜20% | 5〜10年 |
| 競馬場移転＋全面再開発 | 3〜8% | 10〜20年以上 |
| 東京都主導の総合開発（臨海副都心連携） | 10〜15% | 5〜15年 |
| 東京都による競馬廃止・土地売却 | 2〜5% | 10年以上 |
| 現状維持（変化なし） | 55〜65% | 永続 |

**現実的な結論：** 資産の不動性は「永久に続くバリュートラップ」の核心要因。競馬が続く限り土地は解放されず、競馬が廃止されれば事業価値がゼロになる。**この構造的ジレンマこそが最大の問題。**

### ④ CGコード改訂（2026年6月）の影響

#### 4-1. 改訂の概要と本銘柄への影響

2026年6月施行予定のCGコード改訂では以下が強化される方向：

| 改訂ポイント | 東京都競馬への影響 |
|------------|----------------|
| 政策保有株の合理性説明義務強化 | 都の保有に対する説明圧力（ただし都は「競馬振興政策」で説明可能） |
| 不動産含み益の開示促進 | 大井競馬場の含み資産の開示要請強化 |
| PBR1.0倍割れ企業への改善計画義務 | PBR1.60倍（現在値）は既に1.0倍超 → 直接的プレッシャーは弱い |
| 資本コスト・ROE目標の開示義務 | ROE11.3%は相対的に高い → 直接プレッシャーは限定的 |

> **重要：** 現在のPBR1.60倍・ROE11.3%は表面上「CGコード圧力の直接対象外」の水準にある。これがCGコード改訂の実効性を大幅に制限する。

#### 4-2. 定量評価

**CGコード改訂による株価押し上げ効果の試算：**

| シナリオ | 内容 | 株価インパクト |
|---------|------|-------------|
| 含み資産開示のみ | NAVの開示→市場認知向上 | +5〜10%（¥6,100〜¥6,400） |
| 配当方針変更（利回り2.5%へ） | 増配¥50〜60 | +8〜15%（¥6,300〜¥6,700） |
| 自社株買い実施（10億円規模） | EPS向上+株式需給改善 | +10〜20%（¥6,400〜¥7,000） |
| 含み益活用の一部再開発表明 | NAV現実化への一歩 | +20〜40%（¥7,000〜¥8,200） |

**総合評価：** CGコード改訂単体でのカタリスト効果は**限定的（+5〜15%が現実的上限）**。根本的な構造変化（土地活用・都の保有引き下げ）なしには、継続的な株価上昇には繋がらない。

#### 4-3. 都が動く誘因はあるか？

```
東京都の観点から見た資本効率改善の誘因：
　・配当収入増加：誘因あり（都の歳入に直結）
　・自社株買い：都の保有割合が上昇するため誘因なし（むしろ逆効果）
　・土地売却：競馬政策と矛盾 → 誘因ゼロ
　・含み益開示：PRとして有利だが、具体的行動には繋がらない可能性大

結論：都は「増配」については前向きになる可能性があるが、
　　　根本的な資本政策転換には踏み込まない確率が高い。
```

#### 4-4. アクティビスト参入の可能性と要求内容

**参入条件：**
- 時価総額約318億円：アクティビストの最低参入規模（通常500億円以上）を下回る
- 流動性：市場での株式取得が困難（日次2,000〜5,000株）
- 支配株主（都）の存在：要求の実現可能性が根本的に制限される

**仮に参入した場合の要求内容試算：**

| 要求項目 | 実現可能性 | 株価インパクト（実現時） |
|---------|----------|----------------------|
| 配当利回り3%超へ増配 | 30〜40% | +80%（¥10,500程度） |
| 大規模自社株買い（20億円超） | 20〜30% | +15〜25% |
| 大井競馬場の一部土地活用・売却 | 5〜10% | +50〜100% |
| 東京都への株式売却（TOB） | 5〜10% | +30〜50%（プレミアム付き） |
| 解散価値の実現 | 1〜3% | +1,000%（理論値） |

**結論④：** CGコード改訂は意識改革の第一歩にはなるが、東京都という巨大支配株主が存在する限り、**アクティビストの実質的な影響力は極めて限定的**。

#### 6-2. 「PBR1.60倍で放置されている」財務的根拠

**なぜPBRが見た目より問題なのか：**

```
表面的事実：PBR 1.60倍 → 「割安ではない」と判断されがち
　　　　　　ROE 11.3% → 「資本効率は悪くない」
　　　　　　PER 14.9倍 → 「成熟株として適正」

真の問題：
　・BPS（帳簿）¥3,639 ← 大井競馬場の含み益を全く反映していない
　・実質BPS（NAVベース）：¥40,000〜¥60,000超
　・実質PBR（NAVベース）：¥5,840 ÷ ¥50,000 ≒ 0.12倍
　・→ NAVベースでは極めて割安な「隠れバリュー」
　
なぜ市場はNAVを評価しないのか：
　1. 含み資産が開示されていない → 認知されていない
　2. 開示されても実現可能性が低い → 市場は織り込まない
　3. 成長性ゼロ → 将来キャッシュフロー基準の評価が低い
　4. 流動性欠如 → 機関投資家が評価できない
　5. 支配株主（都）問題 → 含み益実現への道が閉ざされている
```

#### 6-3. ネットキャッシュポジション（推定）

| 項目 | 推定値 |
|------|--------|
| 現金・預金 | 約60〜80億円 |
| 有利子負債 | 約30〜50億円（設備投資ローン等） |

### ⑦ DCF FairValue算出

#### 7-1. 前提条件

| パラメータ | 値 | 根拠 |
|-----------|-----|------|
| **Rf（リスクフリーレート）** | **2.429%** | JGB10Y（2026-04-07取得・財務省） |
| **ERP（株式リスクプレミアム）** | **5.5%** | 日本株市場標準値（Damodaran 2026年推計）|

| **Re（株主資本コスト）** | **5.73%** | 2.429% + 0.60 × 5.5% |
| **D/(D+E)** | **20%** | 推定資本構成（有利子負債/総資本） |
| **E/(D+E)** | **80%** | 同上 |
