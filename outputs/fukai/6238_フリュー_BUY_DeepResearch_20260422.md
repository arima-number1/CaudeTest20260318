## ■ ビジネスモデル解剖

### 会社概要
フリュー株式会社は東京・渋谷本社のエンターテインメントコンテンツ企業。プリントシール機「PrintClub（プリクラ）」市場でシェア約70%を誇る独占的地位を持つ。元オムロン子会社から2007年にMBOで独立。東証プライム上場（2019年上場、小型株）。

### 事業セグメント

| セグメント | 主要事業 | FY2025売上 | YoY成長 | 特徴 |
|---|---|---|---|---|
| **Girls Trend Business** | プリントシール機「PrintClub」・Pictlink（画像配信）・スマホ向けコンテンツ | ~¥159億 | +1.1% | 安定・高マージン・国内成熟市場 |
| **World View Business** | クレーンゲーム景品（フィギュア・ぬいぐるみ）・キャラクターIP・海外販売 | ~¥233億 | **+142.6%** | **爆発的成長エンジン**。ホルムズ封鎖で輸送コスト上昇リスクあり |
| **FURYU New Business** | ゲームソフト・アニメ制作・インバウンド向けコンテンツ | 残余 | 変動 | 育成中 |

### 収益構造の変化（重要トレンド）
```
旧来（〜FY2023）：プリクラ機主体 → 国内成熟で成長限界
現在（FY2025〜）：World View（クレゲIP景品）が主柱に浮上
  ↓
「プリクラ会社」から「IP・エンタメコンテンツ会社」への変容
クレゲ景品はセカンダリー市場（フリマ・eBay）でも流通 → IP価値の増幅
海外展開（東南アジア・中国・欧米）でTAMを大幅拡大中
```

### 競合環境
- プリントシール機: DNP（大日本印刷）系フリュー vs インデックスHD vs フリュー→競合少なく独占的
- クレゲ景品: セガサミー・バンダイナムコ・タイトー等と競合するが、フリューの強みはキャラクターIPライセンス（大手漫画・ゲームIP保有会社との提携）

## ■ バリュエーション（4手法）

### 前提パラメーター

| パラメーター | 値 | 根拠 |
|---|---|---|
| Rf（無リスク金利） | **2.397%** | Bloomberg JGB 10年（2026/04/22） |
| ERP | 6.0% | 竹村TC標準 |
| β | **0.75**（推定） | 小型エンタメ株・内需中心・相場感応度低め |
| **Ke** | **6.90%** | 2.397% + 0.75×6.0% = 6.897% |
| BPS | **¥885** | PBR1.50x × ¥1,327 / 1.50 |
| FY2027E EPS | **¥100**（中央値） | FY2026E ¥75 × 成長率+33% |
| 発行済み株数 | **約28.3M株** | 時価総額¥375億 / 株価¥1,327 |

### 手法①：PERバリュエーション
- FY2027E EPS: ¥100
- 適正P/E: **16-18x**（小型成長エンタメ株、World View事業の高成長を反映）
- FV: ¥100 × 17x = **¥1,700**

### 手法②：Justified P/Bバリュエーション
- FY2027E 正常化ROE: **9.0%**（World View成長継続、オペレーティングレバレッジ発揮）
- Justified P/B = 9.0% / 6.90% = **1.304x**
- FV = 1.304 × BPS ¥885 = **¥1,154**
- *注：現在P/B 1.50xはJustified P/B 1.30xを上回っており、P/B観点ではやや割高。ROE10%達成なら1.449x（¥1,282）に改善。*

### 手法③：DCFバリュエーション
| 仮定 | 値 |
|---|---|
| FY2027E FCF | ¥25億/年（OP ¥40億 - Capex ¥10億 - 税調整） |
| WACC | 6.5%（Ke6.9% × 95%株式比率 + Kd2% × 5%） |
| 成長ステージ1（2027-2031） | 年率10%成長（World View継続） |
| 永続成長率 | 1.5%（プリクラ機安定） |
| TV = FCF2031/(WACC-g) | ¥36億 / (0.065-0.015) = ¥720億 |
| 現在価値合計 | ~¥600億 |
| 株数 | 28.3M株 |
| **FV（DCF）** | **¥600億 / 28.3M ≈ ¥2,120** |

*DCFは成長継続前提で楽観的。ターミナル価値への感応度が高い。*

### 手法④：EV/EBITDA（同業比較）
| 項目 | 計算 |
|---|---|
| FY2027E OP | ¥40億 |
| D&A（推定） | ¥5億 |
| EBITDA | ¥45億 |
| ネット現金（推定） | ¥50億（自己資本比率82.6%、無借金に近い） |
| EV = 時価総額 - ネット現金 | ¥375億 - ¥50億 = ¥325億 |
| 現在EV/EBITDA | ¥325億 / ¥45億 = **7.2x** |
| 同業比較（小型エンタメ・コンテンツ）| 8-12x |
| ターゲット倍率 | **9x** |
| Target EV | ¥45億 × 9x = ¥405億 |
| ネット現金加算 | ¥405億 + ¥50億 = ¥455億 |
| **FV（EV/EBITDA）** | **¥455億 / 28.3M ≈ ¥1,608** |

## ■ Q-1定性変化（FY2026 Q3：2025年10月-12月）

### 決算ハイライト
- **Q3累計OP**: ¥29.12億（+41.8% YoY）— 高い利益成長を維持
- **通期進捗率**: FY2026ガイダンスOP¥30億に対して97%達成（Q4はほぼ損益分岐点水準）
- **4Qは転換点**: FY2026 4Q（2026年1-3月）は例年閑散期だが、March +38%は4Q後半の急回復を示唆

### 定性変化の注目点
1. **価格標準化（スタンダード化）の浸透**: PrintClub機の料金設定統一化でARPU改善。1プレイ単価の引き上げが利益率に寄与
2. **World View Business の新規IP獲得**: 直近の大型アニメ・ゲームタイトルとのコラボ景品が好評。「推し活」需要と完全に連動
3. **海外展開（東南アジア・台湾・韓国）**: PrintClub機の海外設置台数が拡大中。FY2027以降の成長余地
4. **インバウンド需要の恩恵**: 訪日外国人（特に東アジア・東南アジア）がプリクラ機を積極利用。秋葉原・原宿・心斎橋の利用率急増

## ■ 不動産含み益考察

### フリューの不動産保有状況
フリューはアセットライト型のエンタメ企業。主要不動産は：
- 渋谷本社：賃貸のため含み益なし
- 工場・倉庫：プリクラ機部品の保管・メンテナンス施設（少数保有）

**含み益推計：極めて限定的（¥1-5億程度）**

フリューの「隠れ価値」は不動産ではなく**IPライセンス権・ブランド価値・機材ネットワーク**：
- PrintClubブランドの無形資産価値
- 全国プリクラ機設置ネットワーク（小売等への間接的価値）
- World Viewのキャラクターコンテンツ・ライセンスポートフォリオ

**→ 製造業・不動産と異なり「含み益」より「EPS成長」がFV決定要因の9割以上。**
