## 【バリュエーション：FV算出（4手法）】

**前提：** Rf=2.380%（財務省CSV）、ERP=6.0%、β=0.80（不動産デベロッパー・安定収益型）
→ **Ke=2.380%+0.80×6.0%=7.18%**

### 手法①：P/E（実力EPS法）

```
FY2026E EPS     : ¥290円 (経常140億×net margin68%÷32.73M株)
FY2027E EPS推計 : ¥260円 (若干の利益率低下想定)

Bull P/E  13x：¥290×13 = ¥3,770
Base P/E  10x：¥290×10 = ¥2,900
Bear P/E   7x：¥260×7  = ¥1,820（金利急騰シナリオ）
```

### 手法②：Justified P/B（ROE/Ke法）

```
ROE（FY2026E推計）:
  純利益95億÷平均自己資本((729+810)/2=769億) = 12.4%
  ← より正確には Yahoo Finance提供値9.2%（正規化ROE）を採用

正規化ROE 10.0%（5年平均考慮）:
Justified P/B = 10.0%/7.18% = 1.393x
FY2026E BPS推計: ¥2,452
→ P/B FV      : ¥2,452 × 1.393 = ¥3,416

BPS実績ベア（1.0x）: ¥2,452 × 1.0 = ¥2,452（下限）
```

### 手法③：DCF（不動産業簡易版）

```
FY2026E 経常利益 : ¥14,000百万
税後純利益      : ¥9,500百万（税率32%）
Capex（維持的）  : ▲¥1,000百万
FCFF推計        : ¥8,500百万
FCFF/株        : ¥8,500M ÷ 32.73M = ¥259.7/株

Terminal成長率  : 2.5%（インフレ+東京賃料上昇）
Terminal FV/株  : ¥259.7÷(7.18%-2.5%)=¥259.7÷4.68%=¥5,549
（注：不動産業のFCFFは棚卸資産変動に大きく左右。正規化Capexの不確実性高い）

修正DiscountFV  : ¥5,549 × 0.50 = ¥2,775（棚卸資産変動ペナルティ50%）
```

### 手法④：配当割引（DDM）+ 配当利回り比較法

```
FY2026E配当     : ¥66
FY2027E配当推計 : ¥70 (成長期待8%/yr)
ゴードン成長    : ¥70÷(7.18%-3.0%) = ¥70÷4.18% = ¥1,675
（DDMは保守的すぎるため参考値扱い）

配当利回り比較:
  現在利回り     : ¥66÷¥1,622 = 4.07%
  同業平均目標   : 2.5-3.0%
  FV（目標3.0%） : ¥66÷3.0% = ¥2,200
  FV（目標2.5%） : ¥66÷2.5% = ¥2,640
```

## 【Q-1定性変化：直近1Qで何が変わったか】

1. **資産運用型マンション販売が「計画を上回って推移」** — これが連続2回（2月→4月）の上方修正を生んでいる。富裕層の東京都心不動産取得ニーズが想定より強く、GALLIAシリーズの即日完売案件が増加

2. **賃料収入が「上昇傾向」** — 保有物件の賃料収入増。ストック型収益が積み上がっており、将来の安定収益基盤が強化。ただし金利上昇→cap rate上昇で保有物件の評価圧力もある

3. **3Q末自己資本比率69.1%の維持** — 売上高+17.2%成長にもかかわらず財務健全性を維持。過去の不動産デベロッパーのような過剰レバレッジリスクは現時点では低い

4. **株価+3.97%（4/21当日）** — 修正を好感も、市場反応は限定的。アナリストカバレッジが薄く、機関投資家の認知度が低い点が継続的な株価低迷の一因

5. **配当性向：** ¥66÷¥290EPS=22.8%。配当余力は十分あり、今後の還元強化余地は大きい

```

  仮定コスト基準：¥300〜500億 × 取得原価
```

**含み益を考慮したNAV観点でのFV下限は¥1,897〜2,081と試算。**
これはPBR0.70xからの解消（BPS¥2,228）より低い水準。含み益を加算した実質BPSは¥2,503〜2,687相当。
