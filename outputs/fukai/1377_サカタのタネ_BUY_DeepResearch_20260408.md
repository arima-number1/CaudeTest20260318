## 1. 企業概要・事業分析

### 1.1 企業プロファイル

サカタのタネ（株式会社サカタのタネ、証券コード1377）は1913年創業、横浜市緑区本社。野菜種子・花き種子の育種・開発・販売を主軸とするグローバル種苗企業。日本国内最大手、世界売上高ランキングはTop10に位置する（Kleffmann Group推計）。

**事業の本質**: 「育種（ブリーディング）」を核とした知的財産ビジネス。種子の品種開発に10〜15年を要し、一度市場に出た品種は5〜10年にわたって継続的な売上を生む「品種ロイヤルティ型」ビジネスモデルが参入障壁の源泉。

## 4. 為替・コスト感応度

### 4.1 円高10円進行時の営業利益影響

```
前提:
・海外売上高: 約400億円（海外比率60%×売上高660億円）
・海外営業利益率: 約11〜12%（野菜種子高マージン品が中心）
・海外売上に対する費用の現地通貨建て比率: 約70%
・海外純エクスポージャー（為替影響を受ける売上−費用純額）: 約120〜140億円

円高10円進行（例: $/¥=150→140）の場合:
・売上高への影響: ▲約25〜30億円（海外売上全体を10/150で換算）
・営業利益への影響: ▲約7〜10億円
  （純エクスポージャー130億円 × 10/150 ≒ 8.7億円）
・営業利益率への影響: ▲0.6〜1.1pt

→ 円高10円で営業利益が▲7〜10億円（▲10〜14%）の影響
```

## 9. DCFバリュエーション（3シナリオ）

### 9.1 共通前提・WACC算出

#### リスクフリーレート・WACC計算

```
【指定値厳守】
Rf = 2.411%（日本10年国債 JGB10Y / 財務省 R8.4.7）

ERP（エクイティリスクプレミアム）= 5.5%（日本株 標準的推計）
β（サカタのタネ）= 0.75（推計: 農業関連株は市場より低β）

株主資本コスト:
Re = Rf + β × ERP
   = 2.411% + 0.75 × 5.5%
   = 2.411% + 4.125%
   = 6.536%

資本構成（推計）:
・自己資本（E）: 約1,000〜1,100億円（推計時価）
・有利子負債（D）: 約50〜80億円（ネットキャッシュポジション考慮）
・E/(D+E) ≈ 92〜95%
・D/(D+E) ≈ 5〜8%

負債コスト:
Rd = 1.0%（推計、優良企業の銀行借入レート）
Tc = 30%（実効法人税率）

WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
      = [0.93 × 6.536%] + [0.07 × 1.0% × 0.70]
      = 6.078% + 0.049%
      = 約6.13%

→ WACC ≈ 6.1〜6.2% を採用
```
