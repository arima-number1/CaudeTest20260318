## 2. 社長指示重点分析：AI中抜きリスク×MCPデータレイヤー化×生成AI大手買収オプション【優先度1】

> 本セクションは社長指示「優先度1」案件として最優先で詳述する。

### 2-1. AI中抜きリスク（最大の構造的脅威）

**メカニズム：**

```
従来の消費者行動:
  消費者 → Google検索 → 価格.com/食べログ → 意思決定

AI中抜き後の消費者行動:
  消費者 → ChatGPT/Perplexity/Gemini → 直接回答（最安値・おすすめ店）→ 意思決定
  ↑ここで価格.com/食べログがバイパスされる
```

**定量的脅威シナリオ：**

| シナリオ | MAU影響 | 広告収入影響 | 売上高影響 |
|---------|---------|------------|---------|
| 楽観（AI補完）| ±0% | ±0% | ±0% |
| 基本（漸進侵食）| ▲10〜15% | ▲8〜12% | ▲5〜8% |
| 悲観（急速中抜き）| ▲25〜35% | ▲20〜30% | ▲15〜20% |

**価格.comの脆弱性：**
価格比較という行為そのものがAIに代替されやすい。「今すぐ最安のエアコンを教えて」というクエリにAIが直接回答するようになれば、**価格.comへの誘導トラフィックは構造的に減少する**。

**食べログの相対的耐性：**
口コミ・評点の「信頼性のある一次データ」としての価値は残存するが、AIがその口コミを要約・回答する場合、ユーザーが食べログサイト自体を訪問しない「ゼロクリック問題」が深刻化する。

---

### 2-2. MCPデータレイヤー化戦略（転換オプション）

**MCP（Model Context Protocol）とは：**
Anthropic（Claude開発元）が策定した、AIモデルが外部データソースと動的に接続するための標準プロトコル。生成AIが「リアルタイム・構造化データ」を必要とする際の接続仕様。

**カカクコムのMCPポジション：**

```
カカクコムが保有する資産:
  ① 価格データ: 数千万件の商品価格・価格推移（リアルタイム性あり）
  ② 口コミデータ: 食べログ4,000万件以上の飲食店評価（質的・量的優位）
  ③ 比較コンテキスト: ユーザーの検索→比較→購買行動データ

MCPレイヤー化後の収益モデル:
  AI企業（OpenAI/Google/Anthropic）がMCPでカカクコムのAPIにアクセス
  → データ使用量に応じたライセンス料を徴収
  → トラフィック減少を「データ販売」で補完
```

**類似事例（参照ベンチマーク）：**

| 企業 | データ資産 | AI企業向け収益化 | 評価インパクト |
|------|----------|----------------|-------------|
| Shutterstock | 写真・動画 | Adobe/Getty向けライセンス | 株価+30〜50% |
| Veeva Systems | 医療データ | AI創薬企業向けAPI | PER拡張 |
| 食べログ類似（参考） | 口コミデータ | 潜在的ライセンス | 未織り込み |

**深井の判断：**
カカクコムがMCPデータレイヤー戦略に成功すれば、**「トラフィック収益→データ収益」のビジネスモデル転換**が実現し、AI中抜きリスクを「AI依存収益」で相殺できる可能性がある。ただし、現時点でこの戦略の具体的進捗は未公表であり、オプション価値としての評価にとどまる。

## 4. DCFバリュエーション

### 4-1. 前提条件（全明示）

**マクロ前提：**
```
リスクフリーレート（Rf）: 2.467%（日本10年国債 / 財務省 2026-04-13）
ERP（株式リスクプレミアム）: 5.5%（Damodaran 日本市場推計2026）
β（ベータ）: 0.85（カカクコム 過去3年 / ITメディアセクター平均）
株主資本コスト（Re）: Rf + β × ERP = 2.467% + 0.85 × 5.5% = 7.14%
```

**資本構成前提：**
```
負債比率（D/E）: 実質ゼロ（無借金経営）
E/(D+E): 100%
D/(D+E): 0%
実効税率（Tc）: 30%

WACC = 100% × 7.14% + 0% = 7.14%
（無借金のためWACC = Re = 7.14%）
```

**DCF前提（基本シナリオ）：**

| 年度 | 売上高成長率 | EBIT（営業利益）率 | NOPAT | CAPEX/売上 | FCF |
|------|------------|-----------------|-------|-----------|-----|
| FY2026 | +5% | 33% | — | 2% | 推計¥16,000百万 |
| FY2027 | +5% | 33% | — | 2% | 推計¥16,800百万 |
| FY2028 | +4% | 32% | — | 2% | 推計¥17,300百万 |
| FY2029 | +4% | 32% | — | 2% | 推計¥17,900百万 |
| FY2030 | +3% | 31% | — | 2% | 推計¥18,200百万 |

**ターミナルバリュー前提：**
```
永続成長率（g）: 1.5%（国内GDP成長率＋インフレ想定）
Terminal Value = FCF₂₀₃₀ × (1+g) / (WACC - g)
             = ¥18,200百万 × 1.015 / (7.14% - 1.5%)
             = ¥18,473百万 / 5.64%
             = ¥327,553百万
```

### 4-2. DCF計算（基本シナリオ）

```
割引率（WACC）= 7.14%

PV計算:
FY2026: ¥16,000百万 / (1.0714)¹ = ¥14,933百万
FY2027: ¥16,800百万 / (1.0714)² = ¥14,641百万
FY2028: ¥17,300百万 / (1.0714)³ = ¥14,079百万
FY2029: ¥17,900百万 / (1.0714)⁴ = ¥13,578百万
FY2030: ¥18,200百万 / (1.0714)⁵ = ¥12,878百万

FCF PV合計: ¥70,109百万

ターミナルバリュー PV:
¥327,553百万 / (1.0714)⁵ = ¥231,735百万

企業価値（EV）合計: ¥70,109 + ¥231,735 = ¥301,844百万
純有利子負債（推計）: ▲¥30,000百万（実質ネットキャッシュ保有）
株主価値: ¥301,844 + ¥30,000 = ¥331,844百万

発行済株式数（推計）: 約1.45億株
理論株価（基本）: ¥331,844百万 / 1.45億株 ≈ ¥2,288/株
```

**安全域（Margin of Safety）：**
```
理論株価（基本）: ¥2,288
安全域バッファ（▲15%）: ¥1,945
→ ¥1,945以下であれば明確な割安買い圏
現在値 ¥2,078 は理論株価の91%水準（軽微な割安）
```
