### 上下振れの理由分析

**ホタテ事業の構造的毀損（主因）**
- 2023年8月の中国による日本産水産物輸入禁止措置後、ヨシムラ傘下のワイエスフーズ・マルキチ（ホタテ販売量シェア約10%）はアメリカ・アジア代替販路を開拓したが、代替市場での値決め力はホタテ売価を十分に回収できなかった。
- 2025年6月に中国が条件付き輸入再開を表明したが、対象工場の認可申請手続きが必要で、26年2月期内に実需回復は間に合わず。

**コスト増が追い打ち**
- ホルムズ緊張（2025年後半〜）に伴う食品原材料・燃料費の上昇が食品製造子会社全体に波及。
- ナフサ起源の包材・合成樹脂コスト上昇は国内食品製造各社共通の逆風。

**海外事業（シンガポール）減速**
- シンガポール景気減速・外食需要低迷により海外セグメントも減収減益。

## 事業・財務分析

### ヨシムラモデルの概要

**「連合経営」型M&A戦略**
- 後継者難・資金難・技術難の中小食品メーカーをM&Aで取得（2008年設立以降、累計M&A件数多数）
- 取得後は**バックオフィス（経理・人事・法務・調達）を本社集約**してコスト削減、製造・ブランドは維持する「ヨシムラモデル」
- 2025年8月末時点で連結子会社**38社**
- 主要セグメント: 国内食品事業（ホタテ水産加工・食品製造）・海外食品事業（シンガポール）

### 過去3期の業績推移

| 期 | 売上高 | 営業利益 | 営業利益率 | 経常利益 | EPS |
|---|---|---|---|---|---|
| **25年2月期** | 約581億円 | 約41.6億円 | **7.2%** | **42.5億円** | — |
| **26年2月期（実績）** | **574.84億円** | **15.68億円** | **2.7%** | **16.92億円** | **¥38.52** |

**Rationale（OPM急悪化）**: 25年2月期の7.2%OPMから26年2月期は2.7%へ4.5ptの大幅悪化。単一セグメント（ホタテ）の毀損が連結利益の大部分を吸収した構造的脆弱性を示す。

## バリュエーション

### DCF（Rf=2.380%固定）

**前提条件**
- ベース営業利益: 27年2月期ガイダンス（経常17億円≒営業15〜16億円）からスタート
- 正常化シナリオ: 中期計画（30年2月期営業80億円）を**50%割引**（信頼性低い）→30年2月期に40億円到達を想定
- 4年間CAGR: 約26%（15.68億→40億円）
- ターミナル成長率: 1.5%（食品製造・日本国内需要の長期縮小考慮）
- WACC: Rf 2.380% + リスクプレミアム 6.5% × β1.049 = **9.2%**

**DCF試算**

| 年度 | 営業利益（推定） | NOPAT（税後） | PV |
|---|---|---|---|
| 27年2月期 | 16億円 | 10.4億円 | 9.5億円 |
| 28年2月期 | 22億円 | 14.3億円 | 12.0億円 |
| 29年2月期 | 30億円 | 19.5億円 | 14.9億円 |
| 30年2月期 | 40億円 | 26億円 | 18.3億円 |
| ターミナル（TV） | — | 27.4億円 ÷(9.2%-1.5%) | 272億円 PV→190億円 |
| **Enterprise Value** | | | **約244億円** |
| 有利子負債控除 | | | ▲348億円 |
| **Equity Value** | | | **▲104億円（注）** |

**注意**: 標準DCFではEquity Valueがマイナスになる（有利子負債が過大）。これはM&A型の特性で「EV基準」評価が適切。

**EV/EBITDA基準TP**
- 現在EV = 493億円（yfinance）
- 現EBITDA = 61.6億円（EV/EBITDA=8.0x）
- 26年2月期EBITDA推定（営業利益15.68億 + のれん償却8.8億 + 減価償却推定10億）= 約34〜35億円
- 現実EBITDA（26年2月期）: 約34億円
- フード系M&A型企業の妥当EV/EBITDA: **8〜10x**
- 27年2月期予想EBITDA（営業16億 + のれん償却9億 + 減価償却10億）= 約35億円
- 妥当EV = 35億 × 9x = **315億円**
- 妥当時価総額 = EV315億 − 有利子負債348億 + 現金 ≒ **▲33億〜+30億円**（現金保有次第）

**PER基準TP**
- forwardEPS: ¥83.9（yfinance取得、27年2月期ベース）
- 食品M&A型の妥当PER: **10〜15x**（高成長前提だった20x超は剥落）
- TP range: ¥839〜¥1,259
- 中央値PER12.5x → **TP ¥1,049**
- ただしガイダンス信頼性低下・借入過大を考慮してディスカウント20%適用 → **TP ¥840**

**最終TP: ¥840（PER基準10x相当、保守的設定）**

**Rationale**: EV/EBITDA基準はEquity価値がほぼゼロ〜マイナスになるほど借入依存が高く、PER基準でも信頼性ディスカウントを適用。¥840は現値比+12%の限定的上値。

## 不動産含み益考察

**M&A取得資産の含み益（有報ベース推定）**

| 項目 | 評価 |
|---|---|
| 固定資産（食品製造設備・倉庫） | M&A取得価格は通常帳簿価格に近い水準。土地含み益は存在する可能性があるが、中小食品メーカーの工場・倉庫は地方立地が多く含み益は限定的 |
| ホテル関連資産 | 株主優待に「自社運営ホテル宿泊割引券」が含まれることから、一部ホテル事業を展開している可能性あり。帳簿価格と時価の乖離は詳細不明 |
| のれん（純粋含み損要因） | 約8.8億円/年の償却負担。M&A取得プレミアムの積上げであり、将来収益が下振れると減損リスクが発生する |
| **総合評価** | 不動産含み益は**軽微または調査要**。含み益よりものれん減損リスクの方が大きい可能性が高い |

**Rationale**: ヨシムラモデルは中小食品メーカー（工場・設備中心）のM&Aであり、不動産デベロッパーや卸業と異なり潜在含み益の規模は小さいと推定。のれん残高の総額（有報未取得のため推定）は累計M&A件数から見て100〜150億円規模の可能性があり、ホタテ事業の構造悪化が長期化すれば**のれん減損リスク**の方が投資家にとって重要なリスク要因となる。
