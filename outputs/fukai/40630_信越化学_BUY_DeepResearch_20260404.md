## 4. リスク要因

### 4.1 中国PVC過剰供給（最大の逆風）

```
中国PVCメーカーがコスト以下でアジア・米国に輸出継続中
→ 全体市場のPVC価格が抑制される
→ シンテックのPVC値上げが通りにくい

ただし:
→ 米国では反ダンピング関税（AD/CVD）が発動中
→ シンテックの国内シェア34%は関税の恩恵を最も受ける立場
→ 中国PVCはカーバイド製（石炭由来）でエコ規制が追い風になる可能性
```

### 4.2 為替リスク（円高）

```
FY2026 Q3累計 OPI: ¥4,980億（前年比▲14.8%）
→ 主因は円高（約11円の円高）
→ ¥1円の円高 = 約▲¥80〜100億のOPI影響（推定）
→ 現在USD/JPY ≈ 145円が基準
→ 円高が145→135に進行すると: ▲¥800〜1,000億
```

### 4.3 半導体シリコン（電子材料セグメント）

```
DRAM/NAND +90% QoQ（今回確認済み）
→ 半導体シリコンウェーハ需要に追い風
→ 電子材料セグメントの更なる利益拡大要因

ただし: ウェーハは需要増に対して新設備が必要（リードタイム2〜3年）
→ 短期（FY2026）への影響は軽微
→ FY2028〜のボリューム増として顕在化
```
