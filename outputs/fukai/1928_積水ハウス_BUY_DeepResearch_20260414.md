## セクション4：DCFバリュエーション

### 4-1. 前提条件の設定

#### WACC計算前提

| パラメータ | 値 | 根拠・出所 |
|----------|---|----------|
| **Rf（リスクフリーレート）** | **2.467%** | 日本10年国債（JGB10Y）財務省 2026/04/13 【絶対厳守】 |
| **ERP（株式リスクプレミアム）** | **5.5%** | 日本市場標準（Damodaran 2025 Japan推計） |
| **β（レバードβ）** | **0.85** | 住宅・建設セクター日本株中位。ディフェンシブ性+M/I Homes米国リスクで調整 |
| **Re（株主資本コスト）** | **2.467% + 0.85 × 5.5% = 7.14%** | CAPMベース |
| **Rd（借入コスト）** | **1.8%** | 有利子負債加重平均金利（日本大手住宅メーカー水準） |
| **Tc（実効税率）** | **30%** | 日本法人税率標準 |
| **D/(D+E)** | **30%** | 自己資本比率約70%→負債比率30%（推計） |
| **E/(D+E)** | **70%** | |

**WACC計算**
```
Re = 2.467% + 0.85 × 5.5% = 2.467% + 4.675% = 7.142%
Rd × (1-Tc) = 1.8% × (1-0.30) = 1.26%

WACC = 0.70 × 7.142% + 0.30 × 1.26%
     = 4.999% + 0.378%
     = 5.377% ≈ 5.38%
```

#### FCFプロジェクション前提

| 項目 | 前提 |
|------|------|
| **ベースFCF（FY2025推計）** | 約¥1,800億円（純利益¥2,000億円×FCF転換率90%）|
| **FY2026成長率** | +10%（M/I Homes通年寄与+価格転嫁）|
| **FY2027〜2028成長率** | +6%（米国着工回復×国内ZEH拡大）|
| **FY2029〜2030成長率** | +4%（安定成長フェーズ）|
| **ターミナル成長率（g）** | **1.5%**（日本GDP成長＋海外寄与を勘案）|
| **プロジェクション期間** | 5年（FY2026〜FY2030）|

### 4-2. DCF計算

#### FCF予測テーブル（基本ケース）

| 年度 | FCF（億円） | 成長率 | 割引係数（WACC=5.38%） | PV（億円） |
|------|-----------|-------|---------------------|-----------|
| FY2026 | 1,980 | +10% | 1/(1.0538)¹ = 0.9490 | 1,879 |
| FY2027 | 2,099 | +6% | 1/(1.0538)² = 0.9006 | 1,890 |
| FY2028 | 2,225 | +6% | 1/(1.0538)³ = 0.8547 | 1,902 |
| FY2029 | 2,314 | +4% | 1/(1.0538)⁴ = 0.8111 | 1,877 |
| FY2030 | 2,407 | +4% | 1/(1.0538)⁵ = 0.7697 | 1,852 |
| **合計PV（5年分）** | | | | **9,400** |

#### ターミナルバリュー（TV）計算

```
TV = FCF₂₀₃₀ × (1+g) / (WACC - g)
   = 2,407 × 1.015 / (0.0538 - 0.015)
   = 2,443 / 0.0388
   = ¥62,964億円

PV（TV） = 62,964 × 0.7697 = ¥48,472億円
```

#### 企業価値（EV）→ 株式価値

```
EV = PV（FCF） + PV（TV）
   = 9,400 + 48,472
   = ¥57,872億円

純有利子負債（推計）：▲¥8,500億円（M/I Homes買収負債含む）
少数株主持分等：▲¥300億円

株式価値 = 57,872 - 8,500 - 300 = ¥49,072億円

発行済み株式数：約5億5,500万株（推計）
```
