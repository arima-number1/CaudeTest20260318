## セクション2：社長指示の重点分析 — 「整備新幹線貸付料30年延長でSELL判断は数値で正当化されるか」

### 2-1. 制度概要と現行コスト

**整備新幹線貸付料制度（現行）：**
- 国・自治体が建設した整備新幹線をJR各社が国（独立行政法人鉄道建設・運輸施設整備支援機構＝JRTT）から借り受ける仕組み
- 現行：開業後**30年間の定額制**
- 算定根拠：開業前の需要予測ベースで固定（固定型）

**JR西日本が負担する主要整備新幹線区間：**

| 路線 | 区間 | 開業年 | 貸付料（推定年額） | 現行満了年 |
|------|------|-------|------------------|-----------|
| 北陸新幹線 | 金沢〜敦賀 | **2024年3月** | 推定**150〜250億円/年** | **2054年** |
| 山陽新幹線（一部）| — | — | — | — |

> **重要前提：** 北陸新幹線（金沢〜敦賀）は2024年3月開業の**最新整備新幹線**であり、30年延長（＝計60年）の対象としてJR西日本が最も強く影響を受ける。

### 2-2. 政策変更シナリオ別追加負担試算

#### シナリオA：現行定額のまま30年延長（最低悲観）
- 追加負担：現行年額150〜250億円×30年 = **4,500〜7,500億円（PV換算：WACC4.5%で2,700〜4,500億円）**
- 1株あたりPVインパクト：▲¥90〜▲¥150/株

#### シナリオB：変動制導入（不動産賃料ベース算定）
- 北陸新幹線沿線の不動産賃料ベースへの算定切替で、収益連動型の貸付料になる可能性
- 楽観的に見ても現行比**+1.5〜3倍**の水準に上昇（業績好調時に貸付料も上がる構造）
- 追加年間負担：+100〜+500億円/年（変動幅大）
- 最悪ケース（好景気×変動制確定）：+500億円×30年PV＝**+3,500〜+5,000億円**
- 1株あたりPVインパクト：▲¥117〜▲¥167/株

#### シナリオC：変動制＋期間延長の二重適用（最悪シナリオ）
- 追加年間負担：+300〜+700億円/年
- 30年PV（WACC4.5%）：**+4,700〜+10,900億円**
- 1株あたりPVインパクト：**▲¥157〜▲¥363/株**

### 2-3. SELL判断の数値的正当化

```
現在EPS（提供値）：¥282
実力EPS（毒調整後）：¥245〜¥260

貸付料影響（シナリオB・中位）での修正EPS試算：
  追加貸付料（税前）：▲300億円/年
  税後影響（実効税率28%）：▲216億円
  発行済株式数（概算）：約2.9〜3.0億株
  EPS下押し：▲¥72/株
  修正EPS：¥282 - ¥72 = 約¥210/株

修正EPS¥210 × PER 10.8倍 = 理論株価 ¥2,268
現在株価¥3,044 → 乖離率 ▲25%（SELL超過）
```

**→ 社長指示への回答：SELLは数値で正当化される。**

特にシナリオB〜C（変動制導入）では：
- 修正EPS¥210〜¥245
- 理論株価¥2,268〜¥2,646
- 現在値比▲13〜▲25%の下値余地
- **SELL判断は定量的に正当化される**

### 4-2. DCFキャッシュフロー予測（5年＋ターミナルバリュー）

**基本シナリオ（現行貸付料定額維持）**

| 年度 | FCFF推定（億円） | 成長率前提 |
|------|---------------|-----------|
| FY2026 | 800 | — |
| FY2027 | 840 | +5.0% |
| FY2028 | 870 | +3.6% |
| FY2029 | 900 | +3.4% |
| FY2030 | 920 | +2.2% |
| ターミナル成長率（g） | — | **1.5%**（インフレ+人口動態考慮） |

**DCF計算（基本シナリオ）**

```
WACC = 4.5%、g = 1.5%

現在価値（PV）計算：
  FY2026: 800億 / 1.045^1 = 765億
  FY2027: 840億 / 1.045^2 = 769億
  FY2028: 870億 / 1.045^3 = 763億
  FY2029: 900億 / 1.045^4 = 755億
  FY2030: 920億 / 1.045^5 = 739億
  5年FCFF合計PV = 3,791億円

ターミナルバリュー（TV）：
  TV = 920×(1+1.5%) / (4.5%-1.5%) = 935 / 3.0% = 31,167億円
  TV PV = 31,167 / 1.045^5 = 25,044億円

企業価値（EV）= 3,791 + 25,044 = 28,835億円

株主価値 = EV - 純有利子負債（推定20,500億円）
  = 28,835 - 20,500 = 8,335億円

発行済株式数（推定）：約3,000万株×100 ≒ 3億株
  → 理論株価 = 8,335億 ÷ 3.0億株 = ¥2,778/株

※ 安全域（MoS）10%適用後：¥2,778 × 0.9 = ¥2,500
```
