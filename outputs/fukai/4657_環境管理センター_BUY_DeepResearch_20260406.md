## 4. DCFバリュエーション

### 4.1 WACC算定

#### β・ERP・株主資本コスト

```
【WACC算定パラメータ】
─────────────────────────────────────────────────────────────
Rf（リスクフリーレート）:  2.380%  ← 社長確認値・絶対厳守
β（推計）:                0.65    ← 環境サービス業（安定収益型）の
                                     日本市場βの中央値。景気感応度低
ERP（Equity Risk Premium）: 5.50%  ← 日本株ERP標準（Damodaran推計
                                     Japan ERP 2025ベース）
Re = Rf + β × ERP:        2.380% + 0.65 × 5.50% = 5.955% ≈ 5.96%

【負債コスト・資本構成】
Rd（借入金利）:            1.20%   ← 実質無借金。既存借入の推計加重平均金利
Tc（実効税率）:            30.6%   ← 日本標準法人実効税率
D/(D+E):                  約5%    ← 有利子負債3億円 / EV約28億円
E/(D+E):                  約95%

WACC計算:
  = [0.95 × 5.96%] + [0.05 × 1.20% × (1 - 0.306)]
  = 5.662% + 0.042%
  = 5.70%（ベースケース）
─────────────────────────────────────────────────────────────
```
