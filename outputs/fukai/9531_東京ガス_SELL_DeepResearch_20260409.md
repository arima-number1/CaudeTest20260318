### 2-C｜原料費調整制度の仕組みと4〜6ヶ月ラグの財務インパクト

#### 原料費調整制度の構造

```
【仕組み】
  ・毎月、原料費（LNG・石油等）の輸入価格変動を「基準原料価格」と比較
  ・変動分を2〜3ヶ月後に「原料費調整額」として小売料金に自動加算/減算
  ・上限・下限のキャップが設定されており、急騰時は全額転嫁できない
  ・総括原価方式の規制料金（一般ガス）では政府認可が必要な場合あり

【タイムライン】
  2026年3月：カタールLNG喪失→スポット価格急騰
  2026年4月〜5月：仕入れコスト急上昇が財務に反映開始
  2026年8月〜9月：原料費調整制度による値上げ請求が顧客に適用（4〜6ヶ月後）
  
  → FY2027/3期の1Q（2026年4〜6月）：制度転嫁前のコスト増をそのまま被る
  → FY2027/3期の2Q（2026年7〜9月）：転嫁始まるが上限キャップで不完全

【上限キャップ問題】
  一般ガス（規制料金）：1ヶ月あたり±一定額の調整上限あり
  TTF急騰シナリオ（€70以上）では上限キャップが転嫁の「壁」になる
  自由料金契約（大口）は個別交渉→転嫁しやすいが時間を要する
```

### 現在のバリュエーション：LNGリスク織り込み済みか？

#### バリュエーション適正水準分析

```
【現状値】
  PBR：1.54倍（BPS¥4,743）→ 株価¥7,297
  PER：14.1倍（EPS¥518）
  ROE：12.1%

【PBR理論値（Gordon ROE-Parity）】
  理論PBR = (ROE - g) / (Re - g)
  Re（株主資本コスト）= Rf + β × ERP = 2.374% + 0.55 × 5.5% = 5.40%
  g（成長率）= 1.5%（公益株・保守的）

  ROE（FY2027想定）= 12.1% × 0.70 = 8.5%

  → BPS¥4,743 × 1.15倍 = ¥5,455

【結論：LNGリスクは「全く」織り込まれていない】
  現在PBR1.54倍は「ROE12%が継続する」前提での評価。
```

## 4｜DCFバリュエーション

### 前提条件の全明示

```
【割引率の前提】
  Rf（リスクフリーレート）：2.374%（JGB10Y・2026-04-09確認値）
  ERP（株式リスクプレミアム）：5.5%（日本株・標準的推定値）
  β（ベータ）：0.55（社長指示・公益株）
  
  Re（株主資本コスト）= 2.374% + 0.55 × 5.5% = 2.374% + 3.025% = 5.399% ≒ 5.40%
  
  Rd（負債コスト）：1.8%（公益企業・格付A相当）
  実効税率（Tc）：30%
  D/（D+E）：≒35%（負債比率）
  E/（D+E）：≒65%（株式比率）
  
  WACC = 65% × 5.40% + 35% × 1.8% × (1-30%)
       = 3.51% + 0.441%
       = 3.951% ≒ 3.95%

【ターミナル成長率（g）】
  Bear：0.5%（成熟・コスト構造悪化）
  Base：1.5%（公益株標準・緩やかな成長）
  Bull：2.5%（水素・電力事業成長が実現）

【FCF試算（5年間）】
  ベースFCF（FY2027起算）：各シナリオの純利益×FCF変換率0.85（D&A加算後）
  CAPEX：¥1,500〜2,000億/年（維持）
  運転資本変動：中立
```

### DCF計算（3シナリオ）

#### Bear（TTF€70継続×純利益▲25%×配当減額リスク）

```
FY2027 FCF：¥1,204億 × 0.85 + D&A¥2,200億 - CAPEX¥1,800億 = ≒¥1,424億
5年FCF（成長率0%〜+1%）：¥1,424 / ¥1,453 / ¥1,467 / ¥1,482 / ¥1,497 億

PV計算（WACC 3.95%）：
  Year1：¥1,424 / 1.0395 = ¥1,370億
  Year2：¥1,453 / 1.0395² = ¥1,346億
  Year3：¥1,467 / 1.0395³= ¥1,315億
  Year4：¥1,482 / 1.0395⁴= ¥1,282億
  Year5：¥1,497 / 1.0395⁵= ¥1,249億
  FCF合計PV：¥6,562億

Terminal Value = FCF₅ × (1+g) / (WACC-g)
  = ¥1,497 × 1.005 / (0.0395 - 0.005)
  = ¥1,504.5 / 0.0345 = ¥43,609億
  TV PV = ¥43,609 / 1.0395⁵ = ¥36,376億

EV = ¥6,562 + ¥36,376 = ¥42,938億
株式価値 = EV - ネットデット¥9,000 = ¥33,938億
理論株価（Bear）= ¥33,938億 / 3.34億株 = ¥10,161

※ ただしBearシナリオでは「減配」「バランスシート劣化」「リスクプレミアム上昇」により
  WACCが0.5〜1.0%上昇する可能性→WACC4.95%での修正：
  TV = ¥1,504.5 / (0.0495 - 0.005) = ¥1,504.5 / 0.0445 = ¥33,809億
  TV PV = ¥33,809 / 1.0495⁵ = ¥26,012億
  EV修正 = ¥6,362 + ¥26,012 = ¥32,374億
  理論株価（Bear修正）= (¥32,374 - ¥9,000) / 3.34 = ¥6,994 ≒ ¥7,000
  
→ ストレスWACC前提でのBear理論株価：約¥7,000（現在株価とほぼ同水準）
  = 現在株価はBearシナリオの最上限に位置している
```

```
  TV PV = ¥79,755 / 1.0395⁵ = ¥66,503億
```

## ⑥追補｜「守りのつもりが最も危険」という逆説の定量的証明

### 公益株パラドックスの構造

```
【一般的な公益株への誤解】
  「インフラ企業=ディフェンシブ=下がらない」
  「配当が安定=保有コストが低い」
  「β0.55=ボラティリティ低=リスク低」

【東京ガスの現実：3つの「隠れリスク」】

隠れリスク①：原料コスト変動リスク（半製造業的なリスク）
  都市ガスは「導管インフラ」ではなく「LNG製造・販売業」の側面が大きい
  LNG調達コストが+60%変動→純利益が▲30〜50%変動する
  これはメーカーと同等以上のコスト変動リスク

隠れリスク②：規制による「転嫁封鎖」リスク
  原料費調整制度は「いつかは転嫁できる」仕組みだが
  政府が「封鎖」した場合：東京電力の2011年以降に近い構造になる
  公益企業だからこそ「社会インフラ」として料金値上げを政府に止められる
  = 公益性が「逆回り」して株主にとっての毒になる

隠れリスク③：バリュエーションの「守りプレミアム」の崩壊
  PBR1.54倍×PER14倍は「守りのプレミアム」が乗っている
  コスト問題が顕在化すると「守りプレミアム」が消え「再評価」が起きる
  
  ROE12.1%→6%になった場合の理論PBR：
    From: (12.1%-1.5%)/(5.4%-1.5%) = 2.72倍（上限）
    To:   (6.0%-1.5%)/(5.4%-1.5%) = 1.15倍
    
    BPS¥4,743 × 1.15 = ¥5,454
    現値¥7,297 → 理論値¥5,454 = ▲25%の「プレミアム崩壊リスク」

【定量的な「逆説」の証明】

  しかし「公益株特有の構造リスク」が顕在化した時：
  β的なリスクを大幅に上回る固有リスクが炸裂する
```
