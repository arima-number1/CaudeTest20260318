### 2-3. FY2026 3Q累計の推計値（4/14発表前）

**前提**: FY2025 Q3単体（確定）: 売上¥20.70億、OP¥6.80億

```
【保守的シナリオ】 Q3成長率 = H1成長率の60%水準（成長鈍化前提）
  Q3単体売上推計: ¥20.70億 × 1.30 = ¥26.9億

  通期進捗率（売上）: 83.4 / 118.0 = 70.7%

  Q3単体OP推計: ¥6.80億 × 1.28 = ¥8.7億
  3Q累計OP: ¥21.03 + ¥8.7 = ¥29.7億
  通期進捗率（OP）: 29.7 / 40.0 = 74.3%

【基本シナリオ】 Q3成長率 = H1成長率の80%水準
  Q3単体売上推計: ¥20.70億 × 1.37 = ¥28.4億
  3Q累計売上: ¥56.51 + ¥28.4 = ¥84.9億
  通期進捗率（売上）: 84.9 / 118.0 = 72.0%

  Q3単体OP推計: ¥6.80億 × 1.38 = ¥9.4億
  3Q累計OP: ¥21.03 + ¥9.4 = ¥30.4億
  通期進捗率（OP）: 30.4 / 40.0 = 76.0%

【楽観シナリオ】 Q3成長率 = H1成長率維持
  Q3単体売上推計: ¥20.70億 × 1.46 = ¥30.2億
  3Q累計売上: ¥56.51 + ¥30.2 = ¥86.7億
```

**結論**: どのシナリオでも3Q累計OP ¥29.7〜¥31.0億 = **通期ガイダンスの74〜78%達成**。
「即逃げライン」¥27.3億（68%）を全シナリオで大幅上回ると推計。
