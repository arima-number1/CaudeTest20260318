## 3. ビジネスモデル・競合分析

### ビジネス概要

マニー株式会社（MANI Inc.）は栃木県宇都宮市に本社を置く精密医療・歯科器具の専業メーカー。1956年創業、世界60カ国以上に製品を販売。

**主要4事業セグメント:**

| セグメント | 主力製品 | 競争優位 |
|-----------|---------|---------|
| **医療用縫合針** | 外科用縫合針・縫合糸付き針 | **アジア最大手**・世界TOP3。μm精度の研磨技術 |
| **歯科用切削器具** | ダイヤモンドバー・エンドファイル・リーマー | 日本・アジア高シェア。高速回転精度が差別化 |
| **眼科用手術器具** | 白内障手術用ナイフ・マイクロニードル | 極細加工技術（直径0.1mm以下）が参入障壁 |
| **注射針・ランセット** | 採血ランセット・注射針 | コスト競争力と品質の両立 |

**競争優位の核心:**
- 製品1本あたりコストは数円〜数十円の微細部品だが、**失敗が許されない医療現場**で使用されるため価格弾力性が低い
- 栃木工場の職人的精密加工技術は模倣困難。中国等の低コスト代替品への置き換えリスクは極めて小さい
- 高い参入障壁（医療機器薬事承認・品質管理体制）が寡占を維持

## 6. バリュエーション（三手法）

**リスクフリーレート（Rf）**: 2.467%（JGB10Y、財務省CSV 2026-04-13）

### 手法①：PBR法（競合比較）

- ピア精密医療器具（テルモ・HOYA除外した中型優良株）目標PBR: **3.3x**
  - 根拠: ROE 10.2%・OP margin 29%は高品質だがROE 13-18%のメガピア比やや低い → ピアの80%水準
- BPS: ¥555.2
- **FV = ¥555.2 × 3.3 = ¥1,832**（ウェイト: 20%）

> **根拠Rationale**: PBR 3.3x採用理由は①ROE 10.2%はRf（2.47%）を大きく超える超過利益創出②無借金で財務リスクなし③ただしCapEx集中でFCFが限定的な現状に保守調整

### 手法②：フォワードPER法

- FY2026/08 EPS（yfinanceコンセンサス）: **¥74.14**
- Target PER: **27x**
  - 根拠: 国内高品質医療機器のPERレンジ25〜35x。マニーはOP margin 29%で上位品質だが小型プレミアム控除10%で27x
- **FV = ¥74.14 × 27 = ¥2,002**（ウェイト: 40%）

> **根拠Rationale**: 27x採用理由は①Gross margin 65%は医薬品並みのリカーリング収益構造②製品1個あたり失敗不可の医療現場での使用 → 価格弾力性ほぼゼロ = 安定成長プレミアム正当化③テルモ30.1x・オリンパス22.5xの中間値

### 手法③：DCF法

| パラメータ | 数値 | 根拠 |
|-----------|------|------|
| Rf | 2.467% | JGB10Y（財務省CSV） |
| β | 0.50（実測0.278にフロア適用） | 超低β銘柄はDCFで割引率過剰圧縮を防ぐため下限設定 |
| ERP | 5.5% | 日本株市場プレミアム |
| Ke | 2.467% + 0.50×5.5% = **5.217%** | 無借金のためWACC≒Ke |
| **WACC** | **5.20%** | 実質無借金（負債¥76M = リースのみ） |
| Base FCF | ¥5,200M | NOPAT（¥9,200M×70%）+ D&A（¥2,471M）- 維持CapEx（¥3,500M）- ΔWC（¥200M） |
| FCF成長率（5年） | 8%/年 | 医療機器世界需要成長×アジア高成長 |
| Terminal growth（g） | 2.5% | 医療機器インフレ率+人口動態による長期需要 |

**FCF投影（5年明示 + 永続価値）:**

| 年度 | FCF | PV（WACC 5.2%） |
|------|-----|----------------|
| Year 1 | ¥5,616M | ¥5,338M |
| Year 2 | ¥6,065M | ¥5,488M |
| Year 3 | ¥6,550M | ¥5,643M |
| Year 4 | ¥7,074M | ¥5,802M |
| Year 5 | ¥7,640M | ¥5,966M |
| **PV FCF合計** | | **¥28,237M** |

**Terminal Value:**
- TV = ¥7,640M × 1.025 / (0.052 - 0.025) = ¥7,831M / 0.027 = **¥290,037M**
- PV TV = ¥290,037M / 1.052⁵ = **¥226,470M**

**Enterprise Value** = ¥28,237M + ¥226,470M = **¥254,707M**
**Add Net Cash** = ¥17,300M（現金¥17.4B − 負債¥76M）
**Equity Value** = ¥272,007M
**Per Share** = ¥272,007M / 98.5M = **¥2,762**（ウェイト: 40%）

> **根拠Rationale**: FCF ¥5,200Mは維持CapEx¥3,500M（直近実績¥6.7Bの半分）前提。拡張CapExが一巡すれば正常化FCFは¥6,000M超。g=2.5%は医療機器の構造的需要（高齢化・アジア中間層拡大）に基づく。感応度: WACC+1%→▲¥350/株、g▲0.5%→▲¥280/株
