### 2-3. PBR0.82倍の正しい解釈

> **❌ 誤り：「PBR0.82は極度割安」**
> **✅ 正しい解釈：「PBR0.82は適度割安〜適正水準」**

PBRの文脈的解釈：

| PBRレンジ | 一般的解釈 | 東京鉄鋼（0.82）の位置 |
|-----------|-----------|----------------------|
| < 0.3 | 「極度割安」または経営危機疑念 | — |
| 0.3〜0.6 | 「強い割安」 | — |
| **0.6〜0.9** | **「適度割安〜適正水準」** | **← 東京鉄鋼（0.82）** |
| 0.9〜1.2 | 「概ね適正」 | — |
| > 1.2 | 「やや割高〜成長プレミアム」 | — |

**PBR0.82の示唆：**
- 市場は純資産の約82%の価値を付けており、完全な解散価値以下ではあるが**大幅ディスカウントではない**
- ROE15.7%という高い収益性を考慮すると、本来PBR1.0〜1.2倍程度が理論的に正当化される水準
- したがって「適度割安」は維持されるが「極度割安」の誤謬は訂正する

**ROEとPBRの整合性チェック（Gordon Growth Model派生）：**
```
理論PBR = (ROE - g) ÷ (Re - g)
  ROE = 15.7%, g = 2.5%（長期成長率推計）, Re = 8.2%（後述）
  理論PBR = (15.7% - 2.5%) ÷ (8.2% - 2.5%) = 13.2% ÷ 5.7% = **2.32倍**
```
この計算から、現在のPBR0.82は**理論値2.32倍の約35%水準**にとどまり、資本コストや収益継続性への市場の慎重な見方が反映されている。単純に「極度割安」と断じるのではなく、**収益の持続性リスクを織り込んだ上での適度割安**と解釈すべきである。

## 4. DCFバリュエーション

### 4-1. WACC前提条件の確定

**【全前提を明示】**

```
■ リスクフリーレート（Rf）
  Rf = 2.380%（JGB10Y / 2026-04-06 10:56 JST 社長確認値）

■ β（ベータ）
  東京鉄鋼（5445.T）の推計β = 1.05
  根拠：鉄鋼素材セクター・東証プライム上場
        建設需要連動の景気敏感株として市場βより若干高め
        ただし純粋な素材大手（β1.2〜1.4）より安定的

■ ERP（株式リスクプレミアム）
  ERP = 5.50%
  根拠：日本株市場の標準的ERP（Damodaran 2025年Japan推計値参照）
        2026年時点：地政学リスク・円安継続を考慮しやや高め設定

■ 株主資本コスト（Re）
  Re = Rf + β × ERP
     = 2.380% + 1.05 × 5.50%
     = 2.380% + 5.775%
     = 8.155%（≒8.2%）

■ 負債コスト（Rd）
  Rd = 1.5%（日本の優良製造業の借入金利水準）
  実効税率（Tc）= 30%（法定実効税率）
  税引後負債コスト = 1.5% × (1 - 30%) = 1.05%

■ 資本構成（Bloomberg確定財務値から逆算）
  純資産（E）= ¥500億
  有利子負債（D）= ¥125億（推計）
  E/(D+E) = 500 ÷ 625 = 80.0%
  D/(D+E) = 125 ÷ 625 = 20.0%

■ WACC計算
  WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
       = [0.80 × 8.155%] + [0.20 × 1.05%]
       = 6.524% + 0.210%
       = 6.734%（≒6.73%）

■ 永久成長率（g）
  g = 2.0%
  根拠：
    ・国土強靱化・公共投資拡大を中期追い風とするが
    ・長期的には建設投資頭打ち・人口減少を考慮
    ・名目GDP成長率（推計2〜2.5%）と整合
```

### 4-2. FCF予測モデル（5ヵ年＋永久成長）

**基点：FCF ≒ ¥67億（FY2025推計、Bloomberg確定値ベース）**

```

EBITDA ≒ 純利益÷0.80 ≒ ¥98億（税前・金融費用前・償却前）
設備投資（Capex） ≒ ¥20億（売上高比1.7%、電炉設備更新）

運転資本増減 ≒ ▲¥3億（売上拡大に伴う若干の増加）

FCFF = EBITDA × (1-Tc) + D&A × Tc - Capex - ΔWC
      ≒ ¥98億 × 0.70 + ¥22億 × 0.30 - ¥20億 - ¥3億
      = ¥68.6億 + ¥6.6億 - ¥20億 - ¥3億
      = ¥52.2億

※簡略化FCF（純利益+D&A-Capex）
  = ¥78.5億 + ¥22億 - ¥20億 - ¥3億（WC）= ¥77.5億
  → 中間値として FCF = ¥67億を採用
