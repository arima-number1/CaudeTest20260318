## 2. 株価急落の真因（改訂・最重要）

**[4月1日上場〜4月24日 ¥562（52週高値）→¥340 ▲39%の要因分解]**

### ① トヨタ・ダイムラーの持分売却需給（主因・寄与度50%推定）

JPMorganレポートで明示された最重要リスク（出所：JPMorgan 2026年4月16日付レポート）

| 項目 | 内容 |
|------|------|
| トヨタ現保有比率 | 41.4% |
| ダイムラー現保有比率 | 41.4% |
| 目標保有比率（東証プライム要件） | 各25.0% |
| 1社あたり売却規模 | 16.4% = 2,757M株 × 16.4% = **451M株** |
| 合計売却規模 | 32.8% = **903M株** |
| 売却金額概算（¥340ベース） | **約¥3,070億** |
| 現在の浮動株数 | 2,757M株 × 17.9% = 494M株 |

**903M株の売却玉は現在の浮動株数(494M株)の1.83倍。** これが株価急落の構造的主因。

### ② Bloomberg P/E 3.64xの謎が解明（「割安幻想」の訂正）

JPMorgan資料でBBG consensus EPS = ¥107.87（26/3E）が確認された。

- 4/16株価¥388 ÷ BBG EPS ¥107.87 = **3.60x ≈ Bloombergの3.64x** → 正体確認
- この¥107.87には27/3期一過性利益（羽村工場売却¥1,500億等）込みのEPS
- 正規化EPS（一過性除く）= **¥34.2**（28/3期EPSに相当）
- 正規化P/E（¥340基準）= ¥340 ÷ ¥34.2 = **9.9倍**（同業並み水準）

**「超割安P/E 3.64x」は誤読。正規化P/Eは約10倍でいすゞと同水準。**

### ③ IPO初値割高の修正（寄与度20%推定）

上場当日SmartKarmaが「ARCHION has landed – expensive」と指摘。EV/EBITDA 8.9x vs いすゞ6x。IPO初期の海外機関投資家売りが加速した。

### ④ 統合移行コスト先行認識（寄与度10%推定）

羽村工場（日野）のトヨタへの移管（4月1日実施済、¥1,500億）・米アーカンソー部品事業撤退（27年末）・5工場→3工場集約の費用先行計上。

### ⑤ 市場全体の弱地合い（寄与度20%推定）

EWJ大量解約（4月21日USD508M・3日累計▲USD525M純流出）による海外売り継続。
