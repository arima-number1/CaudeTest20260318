### 2.1 内航海運とは何か — 外航との本質的構造差

#### 【構造比較マトリクス】

| 比較軸 | 内航海運（明海の主戦場） | 外航海運（川崎汽船等） |
|--------|----------------------|-------------------|
| 運航ルート | 国内港間（日本国内のみ） | 国際航路（太平洋・アジア等） |
| 運賃決定メカニズム | **長期契約（1-3年）主体。荷主との相対交渉** | スポット市場（BDI・CCFI等）が主導 |
| 運賃の変動性 | **低ボラティリティ・遅行性（ラグ大）** | 高ボラティリティ・即時反映 |
| 競合環境 | **国内企業のみ（内航海運業法）** | グローバル競争（COSCO・MSC等） |
| 為替リスク | **ほぼゼロ（円建て契約）** | ドル建て運賃（為替エクスポージャー大） |
| 燃料費構造 | C重油。コスト転嫁に時間差 | バンカー油。GRI等で迅速転嫁 |
| 船員規制 | 日本人船員義務付け（一定比率） | 外国人船員活用可能 |
| 参入障壁 | **極めて高い（法的規制＋船腹調整）** | 高い（資本集約）だが内航より低い |
| 市況インデックス | **存在しない（非公開相対取引）** | BDI（バルチック海運指数）等 |
| 代替輸送との競合 | トラック・鉄道（陸路） | 航空貨物（超高単価品） |

**→ 深井コメント:** 内航海運は「地味だが強固」な市場構造を持つ。外航のように市況が乱高下しない反面、**運賃改定が実現すると複数年にわたり粘着的に収益貢献する**のが最大の特徴。今まさにその転換点にある。

### 2.2 「船賃上がってよさそう」仮説の数値的裏付け

#### 2.2.1 内航海運運賃改定の実態（2023-2026年動向）

```
【内航海運コスト構造の変化（推定）】

           2020年    2023年    2025年    変化率
────────────────────────────────────────
燃料費      ¥/ton  低水準    高止まり   +35〜50%
船員費      index   100       125〜135   +25〜35%
修繕費      index   100       115〜125   +15〜25%
金利コスト  %       低        上昇局面   要注意
────────────────────────────────────────
コスト合計増加率 推定 +30〜45%（2020年比）
```

**国交省・内航総連データに基づく運賃改定状況:**
- 2022年末〜2023年: 主要荷主（製鉄・電力・石油元売り）との運賃改定交渉が本格化
- 2024年: 「物流2024年問題」を背景にモーダルシフト（陸→海）需要が顕在化し、内航の価格交渉力が著しく向上
- 2025-2026年: 改定運賃が通期で寄与するフェーズへ突入（**ラグ解消による収益拡大期**）

#### 2.2.2 運賃改定の収益インパクト試算

**【前提条件】**
- 明海グループ推定売上高: ¥8,000〜10,000百万円（FY2025推定）
- 内航海運事業売上比率: 約62%
- 内航海運事業売上: 約¥5,000〜6,200百万円
- 営業利益率（現状推定）: 約3〜5%

**【運賃改定感応度テーブル】**

| 運賃改定率 | 内航売上増加額 | 営業利益増加額 | EPS増加影響 | 株価インパクト（PE×10-12倍）|
|-----------|-------------|-------------|------------|--------------------------|
| +3% | +¥150〜186M | +¥120〜149M* | +¥20〜35/株 | +¥200〜420/株 |
| +5% | +¥250〜310M | +¥200〜248M | +¥33〜58/株 | +¥330〜696/株 |
| +8% | +¥400〜496M | +¥320〜397M | +¥53〜93/株 | +¥530〜1,116/株 |
| +10% | +¥500〜620M | +¥400〜496M | +¥67〜116/株 | +¥670〜1,392/株 |
| +15% | +¥750〜930M | +¥600〜744M | +¥100〜174/株 | +¥1,000〜2,088/株 |

*変動費ゼロ仮説（売上増分がほぼ利益貢献）は内航の特性から概ね成立

> **深井所見:** 内航海運は**運賃が上がった分がほぼそのまま限界利益に乗る**（追加的な燃料消費・航行コストがほぼ固定）ため、営業レバレッジが極めて高い。+5%の運賃改定でも理論株価へのインパクトは+¥330〜700/株程度と試算。現在株価水準（推定¥1,500〜2,500レンジ）に対して20〜40%のアップサイドを示唆。

---

#### 2.2.3 「2024年問題」とモーダルシフト — 構造的需要押し上げ

```
【モーダルシフト試算】

トラック輸送能力不足（2024年問題）
→ 輸送力▲30%（ドライバー残業規制による）

不足分の海運代替想定:
- 国内貨物輸送量うち内航海運シェア: 約40%（トンキロベース）
- トラックから内航へのシフト需要: 年間+2〜5%増（試算）
- 明海グループの積取数量増加: +3〜7%（保守的シナリオ）

需要増と運賃改定の「ダブル効果」:
→ 数量×単価の両軸改善が同時進行中
```

### 2.3 内航 vs 外航：運賃感応度の決定的差異

#### 感応度比較（定量）

| 感応度指標 | 内航（明海） | 外航大手（川崎汽船等） |
|-----------|------------|-------------------|
| BDI感応度 | **ゼロ**（直接連動なし） | 高い（バルカー系は直結） |
| コンテナ運賃感応度 | **ほぼゼロ** | 極めて高い |
| 国内インフレ感応度 | **高い（コスト転嫁機会）** | 中程度 |
| 為替（USD/JPY）感応度 | **ほぼゼロ（円建て）** | 極めて高い |
| 金利感応度 | 中程度（借入金・リース） | 高い（大型資産保有） |
| 燃料（C重油）感応度 | 高い（コスト要因） | 高い（バンカー要因） |
| 運賃粘着性 | **非常に高い（契約性）** | 低い（スポット主体） |
| 収益可視性 | **高い（契約先確定）** | 低い（市況依存） |

**→ 結論（社長向け）:**
> 明海グループは外航の「市況ギャンブル」とは全く異なるビジネスモデル。船賃上昇局面での収益インパクトは「遅れて確実に来る」型であり、**一旦上昇した運賃は数年間粘着する**。外航のように「今週運賃下落」という事態は発生しない。社長の「船賃上がってよさそう」仮説は、内航においては**一度改定されれば複数年の持続的収益改善**を意味する。これは外航投資と本質的に異なるリスク/リターン構造。

## 4. DCFバリュエーション

### 4.1 WACC算定

#### ステップ1: パラメータ設定

| パラメータ | 値 | 根拠・出所 |
|-----------|---|-----------|
| **Rf（リスクフリーレート）** | **2.380%** | **JGB10Y 2026-04-05確認値（社長確認値）** |
| β（ベータ） | 0.70 | 内航海運の低市況感応度。東証スタンダード内航同業平均。外航（β≈1.0-1.3）より低い |
| ERP（株式リスクプレミアム） | 5.50% | Damodaran日本ERP推定値（2025年）。JPXデータ参照 |
| Re（株主資本コスト） | **6.230%** | = 2.380% + 0.70 × 5.50% |
| Rd（負債コスト） | 2.20% | 地方銀行・政策金融機関からの船舶ファイナンス金利 |
| Tc（実効税率） | 30.0% | 日本法人標準実効税率 |
| Rd×(1-Tc)（税引後負債コスト） | 1.540% | = 2.20% × (1-0.30) |
| E/（D+E）（株式比率） | 40% | D/E≈1.25倍想定。自己資本比率約40% |
| D/（D+E）（負債比率） | 60% | 上記の裏返し |

#### ステップ2: WACC計算

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.40 × 6.230%] + [0.60 × 1.540%]
     = 2.492% + 0.924%
     = 3.416%

→ WACC = 3.42%（ベースケース）
```
