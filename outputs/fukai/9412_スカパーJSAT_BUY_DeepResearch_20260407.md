> 📌 **BPS ¥1,035 / PBR 2.96倍は確定値として以後全分析に使用する。**

#### PBR 2.96倍の意味論的解析

PBR2.96倍は「純資産の約3倍の値段をつけている」ことを意味する。

```
ROEサステナビリティチェック（Gordons Justification）
正当化PBR = ROE ÷ Re（株主資本コスト）
Re = 2.386% + 0.65 × 6.0% = 6.29%（基本ケース）

正当化PBR = 7.9% ÷ 6.29% = 1.26倍

現在PBR（2.96倍） >> 正当化PBR（1.26倍）
→ 市場は「将来のROE改善（防衛需要・宇宙インフラ成長）」を織り込んでいる
→ もしくは成長期待プレミアム（宇宙・防衛テーマ株としての需給プレミアム）
```

**結論：現在のPBR2.96倍はROE7.9%では純粋には正当化困難。防衛・宇宙テーマプレミアムが1.7倍分（約¥2,000）乗っている計算。ROEが12〜15%水準に改善するシナリオでのみ正当化される。**

