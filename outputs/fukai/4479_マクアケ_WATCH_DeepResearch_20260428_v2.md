## Q2 QoQ減速 — 完全解剖【V2確定版】

### 季節性の証明

```
GMV四半期QoQ推移（歴史的パターン）
FY2023/9: Q2/Q1 = ▲3.4%（Q2 < Q1）← 通常
FY2024/9: Q2/Q1 = ▲13.8%（Q2 << Q1）← 大幅通常
FY2025/9: Q2/Q1 = +6.4%（Q2 > Q1）← 例外
FY2026/9: Q2/Q1 = ▲10.2%（Q2 < Q1）← 通常に回帰

結論：Q2<Q1は3年に2年起こる通常パターン。
     FY2025の例外的強さ（+6.4%）が見かけ上の比較基準を歪めた。
```

> 「１Qに引き続き、家電やガジェットジャンルを中心とした大型の案件が成長をけん引」

→ 競合激化・構造的鈍化の記述は**一切なし**。
