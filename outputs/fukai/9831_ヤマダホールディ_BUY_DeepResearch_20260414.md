## 5. DCFバリュエーション

### 5-1. WACC計算

```
【使用データ（全て規定値）】
Rf = 2.439%（日本10年国債、2026-04-14取得値）
ERP = 5.5%（日本市場エクイティリスクプレミアム標準値）
β = 0.75（家電量販・ディフェンシブ性考慮、Bloomberg推計）
Re = Rf + β × ERP = 2.439% + 0.75 × 5.5% = 2.439% + 4.125% = 6.56%

負債コスト:
Rd = 1.8%（金融機関借入金利推計）
実効税率: Tc = 30%
Rd × (1-Tc) = 1.8% × 0.70 = 1.26%

資本構成（推計）:
総資産約¥1兆8,000億
純資産（E）: ¥9,350億（51.9%）
有利子負債（D）: ¥8,650億（48.1%）

WACC計算:
= [E/(D+E) × Re] + [D/(D+E) × Rd(1-Tc)]
= [0.519 × 6.56%] + [0.481 × 1.26%]
= 3.41% + 0.61%
= 4.02%

→ WACC ≒ 4.0%（DCF計算使用値）
```
