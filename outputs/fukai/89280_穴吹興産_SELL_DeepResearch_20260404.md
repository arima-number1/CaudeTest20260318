## 1. 企業概要・事業分析

### 1.1 企業プロフィール

穴吹興産株式会社（8928、東証スタンダード市場）は、香川県高松市を本拠とする中堅総合不動産デベロッパーである。四国・西日本地盤の分譲マンションブランド「サーパス（SARPAS）」を中核に展開し、地方中核都市での存在感は高い。

| 基本情報 | 内容 |
|----------|------|
| 設立 | 1964年 |
| 本社 | 香川県高松市 |
| 従業員数 | 連結 約500〜600名（推定） |
| 市場 | 東証スタンダード |
| 決算期 | 3月期 |
| 主要株主 | 穴吹グループ関連会社 等 |

### 1.2 セグメント構成

```
売上高構成（推定・直近FY）
├── 不動産分譲事業        ～55〜60%  ← コア（サーパスブランド）
├── 不動産賃貸・管理事業  ～20〜25%  ← ストック型収益
├── 建設請負事業          ～10〜15%
└── その他（仲介・サービス）～5〜10%
```

**不動産分譲事業（最重要）**
- ブランド：「SARPAS（サーパス）」シリーズ
- 主戦場：高松・岡山・広島・松山・高知など四国・中国地方の地方中核都市
- 特徴：土地の厳選仕入れ＋地域密着型の中価格帯マンション
- 販売棟数：年間 10〜20棟程度（推定）

**不動産賃貸・管理事業**
- サーパス分譲後の管理受託（管理組合業務）がベース
- ストック型の安定収益源。ランサムウェア被害で管理データリスクが顕在化

**建設請負事業**
- 自社物件の施工管理、外部受注も一部あり

### 1.3 競合優位性の評価

## 4. DCFバリュエーション

### 4.1 前提条件の全明示

```
【割引率パラメータ】
─────────────────────────────────────
リスクフリーレート（Rf）  : 1.5%（JGB10年債利回り・現在値）
エクイティリスクプレミアム: 6.0%（Damodaran推計 Japan ERP参考）
β（アンレバード）        : 0.75（不動産デベロッパー業種β）
β（レバード、推定）      : 0.90〜1.05（財務レバレッジ調整後）
                            ※サイバー事件によるリスク増加を加味し+0.15調整
β（使用値）              : 1.10（保守的設定）

株主資本コスト（Ke）:
  Ke = Rf + β × ERP
     = 1.5% + 1.10 × 6.0%
     = 1.5% + 6.6%
     = 8.1%

負債コスト税引前（Kd）    : 2.0%（地方銀行借入金利推定）
実効税率                   : 30%
負債コスト税引後           : 1.4%

資本構成（D/V）            : 60%（不動産業の特性上高め）
資本構成（E/V）            : 40%

WACC:
  = Ke × (E/V) + Kd(税後) × (D/V)
  = 8.1% × 0.40 + 1.4% × 0.60
  = 3.24% + 0.84%
  = 4.08% ≒ 4.1%

※サイバーリスク・信用毀損プレミアム追加: +1.0%
→ 調整後WACC: 5.1%（保守的シナリオ）
─────────────────────────────────────

【成長率前提】
フェーズ1（1〜3年）  : FCF成長率 ▲15〜▲5%（被害収束期）
フェーズ2（4〜7年）  : FCF成長率 +1〜+3%（回復期）
ターミナル成長率（g） : 0.5%（人口減少・地方不動産の長期停滞）
─────────────────────────────────────

【ベースFCF】
正常化FCF（FY2024ベース）: 約4〜6億円（推定）
被害影響後FCF（FY2025）  : 約▲2〜+2億円（復旧コスト・損失含む）
```

### 4.2 DCF計算（基本シナリオ）

```
【基本シナリオ】WACC: 5.1%, g: 0.5%

年度     FCF(億円)  割引係数(5.1%)  現在価値(億円)
FY2025   ▲1.0      0.951          ▲0.95
FY2026    2.0      0.905           1.81
FY2027    3.5      0.861           3.01
FY2028    5.0      0.819           4.10
FY2029    6.0      0.780           4.68
FY2030    6.5      0.742           4.82
FY2031    7.0      0.706           4.94
────────────────────────────────
フェーズ1合計（PV）                22.41

ターミナルバリュー:
  TV = FCF_t × (1+g) / (WACC - g)
     = 7.0 × 1.005 / (0.051 - 0.005)
     = 7.035 / 0.046
     = 152.9億円
  TV現在価値 = 152.9 × 0.706 = 107.9億円

企業価値（EV）合計:
  = 22.41 + 107.9
  = 130.3億円

ネット負債調整（有利子負債160億円 - 現預金30億円）:
  = 130.3 - 130 = 0.3億円 → 株主価値: 約0〜10億円

株式数（推定）: 約10〜11百万株

理論株価（基本シナリオ）: 約 0〜1,000円
→ 現実的レンジ提示: 800〜1,050円
```

> **⚠️ 注釈:** DCFにおいて、ネット負債が企業価値に近い場合、株主価値は極めて感応度が高くなる。穴吹興産は不動産業の特性上、有利子負債が大きく、FCF悪化時の株主価値毀損が著しい。これが「強い売り」判断を支持する核心。
