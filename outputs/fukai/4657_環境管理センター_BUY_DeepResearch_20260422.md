### 2-1. 水道法改正カタリスト

#### 改正の骨子と環境管理センターへの影響

```
【水道法改正（2024年改正〜2026年施行加速）の核心】

改正前: 水道事業者が「自主的に」水質管理
改正後: 水源から蛇口まで全工程での水質検査「義務化・強化」
        + PFAS（有機フッ素化合物）検査の必須化
        + マイクロプラスチック検査の追加
        + 検査頻度の大幅引き上げ（年1回→月次・四半期次）
        + 第三者機関による確認義務
```

#### 環境管理センターへの具体的インパクト推計

| 項目 | 改正前 | 改正後（推計）| インパクト |
|------|-------|------------|---------|
| 水質検査受託件数 | ベースライン100 | 130〜150（推計） | **+30〜50%** |
| 検査単価（PFAS等新項目）| 標準単価 | +20〜40%プレミアム | **+20〜40%** |
| 対象自治体数（全国） | 約1,400水道事業者 | 同数だが検査項目・頻度増 | **件数×単価の複合増** |
| 参入業者 | 既存計量証明事業者 | 新規参入困難（設備・認定）| **同社の競争優位維持** |
| 売上への寄与（推計）| — | +¥2〜5億円/年（FY2027以降本格化）| **売上高+5〜15%相当** |

#### PFAS規制の特需性

> PFAS（ペルフルオロアルキル物質）は「永遠の化学物質」と呼ばれ、2026年の環境基準強化により全国の水道事業者が緊急対応を迫られている。専門機器（LC-MS/MS）と認定資格を持つ環境管理センターは、この特需の直接的受益者。競合が限られる超専門分野であり、受注単価は一般水質検査の3〜5倍と推計される。

### 2-2. 環境省プラスチック資源循環受託（本日S高トリガー）

#### 制度的背景

```
【プラスチック資源循環促進法（2022年施行）の執行強化】

2026年4月: 環境省が「プラスチック資源循環実態調査」委託業務を本格化
            → 全国の事業者・自治体のプラ廃棄物組成分析・追跡調査が義務化
            → 第三者認定機関による分析が必須要件
            
環境管理センターの役割:
  ✅ プラスチック種別分析（IR分光法・熱分解GC-MS）
  ✅ 再生材の品質証明発行
  ✅ 微細プラスチック（マイクロプラ）の計量証明
  ✅ 廃棄物フロー追跡レポート作成
```

## 4. DCFバリュエーション

### 4-1. 前提条件

```
【マクロ前提】
リスクフリーレート（Rf）  : 2.392%（日本10年国債 2026-04-22）
エクイティリスクプレミアム : 6.5%（日本株ERP標準値）
ベータ（β）               : 0.75（環境分析・規制業種は市場連動性低め）
株主資本コスト（Re）       : 2.392% + 0.75 × 6.5% = 2.392% + 4.875% = 7.267%

【資本構成前提】
E/(D+E)（株主資本比率）   : 70%（推計・自己資本比率高め）
D/(D+E)（負債比率）       : 30%
負債コスト（Rd）          : 1.5%（銀行借入金利・優良中小推計）
実効税率（Tc）            : 30%
WACC計算:
  = 0.70 × 7.267% + 0.30 × 1.5% × (1-0.30)
  = 5.087% + 0.315%
  = 5.40%

【成長率前提】
```
