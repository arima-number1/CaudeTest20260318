### サブリースモデルの構造と持続可能性

大東建託の収益モデルは「土地オーナーへのアパート建設請負 → 一括借上（サブリース）→ 賃貸管理」という垂直統合型の3フェーズ構造である。

**Phase 1（建設）:** 全国の専属営業担当（約3,000人）が地主・相続案件を対象に飛び込み営業。相続税節税（小規模宅地等の特例）を訴求し、建設請負契約を獲得。平均工事単価は1〜2億円。

**Phase 2（一括借上）:** 竣工後、月額賃料の90〜95%をオーナーに保証家賃として支払い、差額がサブリースマージン。オーナーは空室リスクをゼロ化できる一方、家賃は数年ごとに市況に応じて見直される。

**Phase 3（管理・仲介）:** DK SELECT（全国1,100超の提携仲介店舗ネットワーク）を通じた入居者斡旋、修繕・管理業務で追加収益。入居率97.8%は同業他社を大きく上回る。

**持続可能性の総評:**
- 管理戸数の拡大がプール効果で空室リスクを分散し、入居率の高位安定を支える構造は強固。
- ただし2025年1月の地方拠点3割削減（都市部再集中）は、人口動態悪化に伴う地方事業縮小の構造的転換点であり、フロー収益（建設受注）の成長頭打ちリスクを孕む。
- 不動産開発事業（FY2025売上+64.8%）を「第2の柱」として育成中であり、ストック比率のさらなる向上が中期の業績安定化シナリオの鍵となる。

### 過去の主要労働・社会問題

**サブリース問題（2017〜現在）:**
- 週刊ダイヤモンド（2017年）が「高齢地主への不適切な営業」を告発。サブリース被害対策弁護団が結成され、社会問題として認知された。
- 消費者機構日本による契約適正化要求、サブリース被害訴訟が複数件進行。
- **対応:** 2021年6月施行の賃貸住宅管理業法（サブリース新法）により、勧誘時の誇大広告・不当勧誘を規制。重要事項説明書の書面交付が義務化。大東建託は外部通報窓口設置・説明品質向上宣言を実施。

**営業現場の過酷な労働環境（現在も継続課題）:**
- 主力の法人営業は毎日飛び込み訪問が基本で離職率が高い。
- 成果主義の徹底により、達成できない場合のプレッシャーが強い。
- 2025年1月の拠点再編（3割削減）により、地方担当者の都市部への異動・転職が発生。

### 直近の人員・採用動向

- **2025年1月:** 全国の営業拠点を約3割削減し、大都市圏・都市部に人員を再配置（人員削減ではなく配置転換）。
- **DE&I中期計画（2024〜2026年度）:** 女性取締役比率15.3%へ向上（前年8.3%→女性社外取締役2名選任）。女性育成プログラムにより女性管理職比率が過去最高水準を更新。
- **外国人技能実習生:** 5年計画で約560名の育成推進。建設業の人手不足への対応策だが、品質管理リスクも内包。

**組織健全性の総評:**
大東建託の組織健全性は、住宅大手の中では「課題先進企業」の側面を持つ。給与水準（特に営業職の歩合給）は業界上位だが、飛び込み営業主体のビジネスモデルに伴う離職率の高さ、サブリース問題に起因する社会的評判リスクが継続している。2021年のサブリース新法施行後は制度面での整備が進んだが、口コミサイトへの投稿件数の多さ（エン カイシャで7,902件）は社会的関心の高さを示している。ESG対応は女性役員登用では前進しているが、現場の労働環境改善の持続的モニタリングが必要。

```
  JGB 10Y（日本10年国債利回り）: 1.5%（2026年4月時点）

【エクイティリスクプレミアム（ERP）】
  5.5%（Damodaran日本市場推計ベース）

【ベータ値（β）】
  0.85（住宅大手・ディフェンシブ特性を反映し市場平均より低め）

【Cost of Equity（Ke）】
  Ke = 1.5% + 5.5% × 0.85 = 6.175% ≈ 6.2%

【負債コスト（税引後）】
  Kd(after-tax) = 0.5%（有利子負債倍率0.26倍の低水準を反映）

【資本構成（時価ベース）】
  E/V = 80%、D/V = 20%

【WACC】
  WACC = 6.2% × 80% + 0.5% × 20% = 5.06% ≈ 5.0%
```

## 総合投資見解

### 投資レーティング: **強い買い（理論株価5,900〜7,200円、現在株価3,742円比+57〜92%）**

**強みのサマリー（Bull Case）:**
1. **業界最大の護城河（Moat）:** 管理戸数129万戸・29年連続首位という参入障壁は事実上崩せない水準。プール効果による97.8%の入居率は収益安定性の証左。
2. **圧倒的な資本効率:** ROE 21.5%は住宅大手（12%前後）の約2倍。この差がPBRプレミアムを正当化する。
3. **魅力的な株主還元:** 配当利回り3.82%・4期連続増配（配当性向約50%）は安定した配当投資先として評価。
4. **DCFベースの明確な割安感:** WACC5%・ターミナル成長率1%のベースケースで理論株価6,200円。現在株価との+65%の乖離はDCF投資家にとって魅力的。
5. **中計進捗の確実性:** FY2025に純利益+25.7%増を達成。FY2026目標（売上2兆円・営業利益1,400億円）はやや保守的と判断。

**懸念のサマリー（Bear Case）:**
1. **フロー受注の先行指標悪化:** Q1受注高▲11.7%は中計目標に黄信号。地方縮小の影響が想定より大きければ下方修正リスク。
2. **人口動態の逆風:** 構造的な人口減少は賃貸需要の長期減退を不可避とし、ストック収益の成長率鈍化をもたらす。
3. **評判・訴訟リスクの尾を引く構造:** サブリース問題は制度整備により収束方向だが、事業モデルの性質上、再発リスクはゼロではない。

**推奨保有期間:** 中長期（12〜24ヶ月）
**主要モニタリング指標:** ①四半期受注高（前期比）、②管理戸数と入居率、③建設事業利益率、④金利水準（JGB 10Y）
