## 【バリュエーション：FV算出（4手法）】

**前提：** Rf=2.380%（財務省CSV）、ERP=6.0%、β=1.30（精密機械×中国87%集中×同業比較）
→ **Ke=2.380%+1.30×6.0%=10.18%**

### 手法①：P/E（ピークサイクル修正型）

```
FY2026E EPS     : ¥355.90 （今回修正確定値）
FY2027E EPS推計 : ¥290円  （中国不透明感×設備投資一巡想定）

Bull P/E  20x：¥355.90×20 = ¥7,118
Base P/E  17x：¥355.90×17 = ¥6,050
Normalized ：¥290×16x  = ¥4,640（FY2027E保守）
Bear P/E  12x：¥220×12  = ¥2,640（China shock）
```

### 手法②：Justified P/B（ROE/Ke法）

```
ROE推計（FY2026E）： 16,500 / 平均親会社持分(68,486M) = 24.1%
Justified P/B    ： 24.1% / 10.18% = 2.37x
FY2026E BPS推計  ： ¥1,600
→ P/B FV        ： ¥1,600 × 2.37 = ¥3,792

（注：24%ROEは現在サイクル高の極値。正規化15%採用時）
正規化 Justified ： 15% / 10.18% = 1.47x → ¥1,600×1.47 = ¥2,352
```

### 手法③：DCF（1年フォワード）

```
FY2026E EBIT（Implied）  : ¥36,000百万
税後NOPAT（税率27%）    : ¥26,280百万
DA（推計）              : ¥2,500百万
Capex（推計）           : ▲¥3,000百万
正規化FCFF              : ¥25,780百万（推計）
FCFF/株（46.455M）     : ¥555/株

Terminal成長率          : 2.0%
Terminal FV/株          : ¥555×0.75÷(10.18%-2%)=¥416÷8.18%=¥5,086
PV(Terminal, 1yr)       : ¥5,086/(1.1018) = ¥4,616
PV(FCFF yr1)            : ¥555/(1.1018) = ¥504
→ DCF FV                : ¥5,120
```

### 手法④：EV/EBITDA比較法

```
EBITDA推計（FY2026E）   : 営業利益¥36,000+DA¥2,500=¥38,500百万
時価総額               : ¥2,075億
ネット現金（3Q末推計）  : +¥10,000百万以上（現金増+無借金）
EV                     : ~¥19,750億-¥1,000億=¥18,750億
EV/EBITDA              : ¥18,750億÷¥38,500百万=4.88x

同業比較：
  DMG Mori(6141)       : 8-12x
  Okuma(6011)          : 6-10x
  citizen(7762)        : 8-12x
6101のEV/EBITDAは極端に安い。ターゲット7x適用：
→ EV Target            : ¥38,500×7=¥26,950億
→ Equity Value        : ¥26,950億+¥1,000億=¥27,950億
→ 株価FV              : ¥27,950億÷4,645.5万=¥6,017
```

## 【Q-1定性変化：直近1Qで何が変わったか】

1. **中国スマートフォン精密部品需要の急拡大** — Apple/Huawei/Samsung向け中国部品加工が予想を大きく上回る。iPhone 17シリーズの複雑部品構成がスイス型旋盤需要を後押し

2. **インドセグメント黒転** — 前期損失から¥77百万の黒字転換。インド精密製造エコシステムの立ち上がりが加速。将来の中国代替候補として重要

3. **日本セグメント売上▲10.2%、利益+155.6%** — 数量減でも固定費改善で利益率が急改善。構造変革の成果が顕在化

4. **自己株式取得加速** — 1,311,478株（前期末比708,718株増）。自己株取得は継続中

5. **会社保守的ガイダンスの構造的修正** — 3Q累計純利益(119億)/前回通期予想(125億)=95.3%達成。今後、会社は保守的ガイダンス体質を修正しつつある

## 【不動産含み益考察】

ツガミは製造業・機械メーカーであり、保有不動産は主に**工場・生産設備**（上海・無錫・日本国内）。

- 中国生産拠点の土地・建物: IFRS基準で公正価値評価済み → 含み益の大部分は財務数字に反映
- 日本本社・倉庫等の不動産: 規模限定的
- **不動産含み益による追加的なNAV discount解消余地は限定的**。バリュエーションは純粋な収益力（EPS/FCFF）で評価
