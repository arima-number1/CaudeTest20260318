### 1-3. 競合優位性

**①塩ビ管・雨どいの国内シェア**
塩ビ管（給排水用）の国内市場で積水化学は**シェア約40%**のトップポジションを持つ。次点の積水化学グループ系を含めると寡占に近い構造。これが価格交渉力の源泉。

**②高機能品への継続シフト**
PVBインターレイヤー（合わせガラス中間膜）はグローバルシェアTop3。自動車の安全ガラス・建築用合わせガラスに不可欠。価格弾力性が低く、値上げが通りやすい。

**③住宅ブランド（セキスイハイム）の垂直統合**
住宅事業が原材料から完成住宅まで内製化。自社の化学素材を自社住宅に搭載することで、原材料値上げを一定程度「内部消化→住宅価格転嫁」という経路で収益化できる。

**④インフラ更新という不可避需要**
日本の老朽水道管は全国に約70万kmあり、更新需要が政府主導で加速。「国土強靭化」予算の追い風が続く。公共調達は原価上昇の転嫁が民間より容易。

## 2. 社長指示の重点分析：「ナフサ不足は心配だが、価格転嫁力は強いか」

### 2-1. ナフサと積水化学の関係：「どこが痛くてどこが強いか」

ナフサ（石脳油）は積水化学にとって、塩化ビニル樹脂（PVC）・ポリエチレン（PE）・ポリスチレン（PS）の最上流原料。影響度をセグメント別に以下に整理する。

```
ナフサ → エチレン/プロピレン/塩素 → PVC（塩ビ）→ 水道管・雨どい・防水シート
ナフサ → エチレン → PE → 容器・フィルム
ナフサ → エチレン → PVB → 合わせガラス中間膜（高機能品）
```

**最もナフサ直撃を受けるのは「環境・ライフライン（塩ビ管・建材）セグメント」**。
高機能プラスチック部門（PVBインターレイヤー等）はナフサの川下に位置するが、製品の付加価値が高く価格転嫁余地が大きい。

### 2-3. 「強い転嫁力」を支える3つの構造的理由

**① 寡占的市場地位（塩ビ管・雨どい）**
水道管・雨どいは日本市場で積水化学・積水化成品・クボタ等が寡占。競合も同じくナフサコスト上昇に苦しんでおり、「全員値上げ」の協調的状況が生まれやすい。カネカ40%・日本ペイント50%という大幅値上げが先行したことで、積水の15%は「むしろ控えめ」に見える。

**② 代替困難なインフラ素材**
塩ビ水道管はJIS規格・水道法適合品として設計されており、施工業者・工務店が簡単に仕入れ先を変更できない。スイッチングコストが高い。

**③ 競合の物量制約で積水への需要集中リスク**
ホルムズ封鎖で競合も原料不足に陥れば、むしろ積水への注文集中が起きる可能性。調達力（長期契約・多元化）が強い企業に需要が集まる逆説的効果。

### 2-4. 価格転嫁の「限界」と懸念事項

**① 住宅着工数の減少リスク**
値上げが住宅本体価格を押し上げ、着工件数が10%減少すると建材需要が▲¥200〜350億規模で減少する。特にデッキ材・雨どいは新設住宅連動型。

**② バルコニーデッキ材の代替リスク**
アルミや天然木材との代替が比較的容易なデッキ材は、15%以上の値上げで顧客が他素材にシフトするリスクがある。ただし建材全体に占めるシェアは限定的。

**③ ホルムズ長期封鎖（3ヶ月超）の物量途絶シナリオ**
価格転嫁以前に「原料が来ない」場合は生産停止リスク。ただし積水化学は中東以外の調達ルート（東南アジア・米国エタン）も持ち、完全依存ではないとみられる。
