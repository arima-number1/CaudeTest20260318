#### ■ 燃料費調整制度（FIT調整）の「恩恵が鉄道に届かない構造」

これがショートの核心論拠である。以下に構造的メカニズムを解説する：

```
【燃料費調整制度の資金フロー】

原油・LNG価格上昇
      ↓
電力会社（東京電力等）：燃料費調整制度により電気料金に転嫁 ✅
      ↓
東急（法人電力需要家）：料金転嫁を受ける側 ❌（転嫁先なし）
      ↓
問題：東急の「収益」転嫁先は？
  ├── 鉄道運賃 → 国交省認可制、改定に2〜3年
  ├── 駅ビル賃料 → テナント契約で固定（1〜3年単位）
  ├── ホテル宿泊料 → 部分的に転嫁可能（最も柔軟）
  └── 百貨店 → 消費者向け価格転嫁困難（競合多数）
```

**結論：電力コスト増加の70〜80%（鉄道+駅ビル+百貨店分）は即座の転嫁が構造的に不可能。**

## 4. DCFバリュエーション

### 4-1. WACC計算

#### ■ パラメータ設定

| パラメータ | 値 | 根拠・出典 |
|------|------|------|
| **Rf（リスクフリーレート）** | **2.467%** | JGB10Y 2026-04-14（財務省日次） |
| **β（ベータ）** | **0.85** | 東証上場鉄道不動産コングロマリット、市場β（業界中央値）。電力費感応度高く若干上方 |
| **ERP（マーケットリスクプレミアム）** | **5.5%** | 日本株ERP業界標準（Damodaran 2025年日本推計値） |
| **Re（株主資本コスト）** | **2.467% + 0.85 × 5.5% = 7.14%** | |
| **Rd（負債コスト）** | **2.8%** | JGB+スプレッド（A格鉄道債務、0.3%スプレッド）、金利上昇反映 |
| **実効税率（Tc）** | **30%** | 法人税+地方税 |
| **E/(D+E)（株式比率）** | **35%** | 鉄道不動産複合体の典型的資本構成（レバレッジ高） |
| **D/(D+E)（負債比率）** | **65%** | |

#### ■ WACC計算

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.35 × 7.14%] + [0.65 × 2.8% × (1-0.30)]
     = 2.499% + 1.274%
     = 3.773% ≒ 3.77%
```
