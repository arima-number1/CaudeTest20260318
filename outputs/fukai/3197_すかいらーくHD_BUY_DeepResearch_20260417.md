### 2-4.【最重要④】PBR3.96倍×PER44.4倍の正当性検証

#### ゴードン成長モデルによるPBR正当化条件

**PBR = ROE × (1-b) / (Re - g)**（変形式）
または、より直接的に：
**PBR = (ROE - g) / (Re - g)**

**前提：**
- ROE = 9.3%（リアルタイム取得値）
- Re（株主資本コスト）= 後述WACCより Re ≈ 8.5〜9.5%
- 現在PBR = 3.96倍

**PBR=3.96倍を正当化するために必要なROEとgの組み合わせ：**

```
3.96 = (ROE - g) / (Re - g)
Re = 9.0%として：
3.96 × (0.09 - g) = ROE - g
0.3564 - 3.96g = ROE - g
ROE = 0.3564 - 2.96g

g = 0% → ROE = 35.6%（現在9.3%と大幅乖離）
g = 5% → ROE = 20.6%（同様に乖離）
g = 7% → ROE = 14.8%（まだ乖離）
g = 8% → ROE = 12.1%（近いが非現実的な成長率）
```

**結論：現在のROE9.3%でPBR3.96倍を正当化するには、持続成長率g≈7〜8%が必要。外食チェーンでこれは非現実的。**

#### PER44.4倍を正当化する成長率の逆算

**PER = 1 / (Re - g)（Gordon Model簡略）**

```
44.4 = 1 / (Re - g)
Re - g = 2.25%
Re = 9.0%として：g = 6.75%
```

**外食チェーンで持続的なEPS成長率6.75%は正当化可能か？**

| 比較軸 | 数値 |
|--------|------|
| 国内外食市場の成長率（長期） | 0〜2% |
| すかいらーくEPS成長率（過去5年CAGR） | 約3〜5%（コロナ回復含む） |
| 人件費・光熱費増後のEPS成長率（今後） | **▲10〜▲30%（EPS圧縮局面）** |
| PER44.4倍を正当化するEPS成長率 | **6.75%（達成可能性：低）** |

**結論：PER44.4倍は過大評価。EPS圧縮局面では適正PERは20〜25倍程度が妥当。**
