### 2.2 PBR割れの本質的要因（構造・慢性的）

#### 【要因A】ROEの慢性的な低位（最重要因）

```
ROE推移（推定）：
FY2020: 4.2%
FY2021: 5.1%
FY2022: 6.8%
FY2023: 7.2%
FY2024: 6.5%（銅価格・円安効果剥落）
FY2025E: 7.0〜8.0%
```

- **PBR = ROE / (株主資本コスト Re)**の関係式が示す通り
- Re（株主資本コスト）を仮に8.5〜9%と見ると、ROE 7%では PBR = 7%/9% ≈ **0.78倍**が「適正」
- ROEが株主資本コストを恒常的に下回ってきた結果、市場がディスカウントして評価
- **古河電工のROEを押し下げる3大構造：**
  1. 過剰な自己資本（純資産が厚すぎる）＝資本効率の悪さ
  2. 低採算セグメント（自動車部品・ワイヤーハーネス）の収益ドラッグ
  3. 政策保有株式の大量保有（含み益は帳簿価値を押し上げるが収益に貢献しない）

#### 【要因B】コングロマリット・ディスカウント

- 電線・光ファイバー・銅箔・自動車部品と、多岐にわたる事業が混在
- 投資家からは「何屋か分からない」複合企業として評価されやすい
- 高成長セグメント（光ファイバー・銅箔）の価値が、低採算セグメントに埋もれる構造
- 住友電工・フジクラ・日立製作所等と比較した際に、ピュアプレイヤーへの資金移動が発生しやすい

#### 【要因C】政策保有株式の重石**

- 古河グループは伝統的な持ち合い構造を維持
- BSに内包される政策保有株（帳簿価値数百億円）が「隠れた非効率資産」として認識
- 東証のPBR1倍要請（2023年〜）に対する対応ペースが競合比で遅い印象
- **ROEへの影響**：分母（自己資本）の水増し効果でROEが実態より低く見える

#### 【要因D】銅価格ボラティリティによるEPS不安定性

- 銅価格（LME銅）の変動が売上高・仕入コスト双方に影響
- FY毎に業績のブレ幅が大きく、長期DCFの前提が立てにくい
- 機関投資家が「割引率を保守的に設定せざるを得ない」構造

#### 【要因E】自動車部品（ワイヤーハーネス）の構造的逆風

- EV化に伴いワイヤーハーネスの設計が変化（銅線量の減少・構造簡素化）
- 海外の低コスト競合（住友電工・ヤザキ・レオニ等）との価格競争
- 市場はこの事業を「バリュー破壊」事業として評価

### 2.3 PBR回復の触媒（Catalyst）分析

#### 【触媒1：最強】AI・DC需要爆発による光ファイバー・銅箔利益急拡大

```
光ファイバー市場成長予測（グローバル）：
2024: 約1.2兆円
2027E: 約1.8兆円（CAGR +14%）
2030E: 約2.6兆円（CAGR +13%）

AI DC1棟あたりの光ファイバー消費量：
従来型DC比 約3〜5倍
```

- **深井試算**：光ファイバー部門の売上が+20〜30%成長すれば、営業利益が連結ベースで+200〜400億円のインパクト
- この利益成長がROEを9〜10%台に押し上げれば、PBRの正当化水準が一気に1.0〜1.2倍へ
- **重要シグナル**：住友電工・フジクラが既に大幅株高→古河電工への「出遅れ買い」が情報機関の間で静かに蓄積（浅井コメント示唆）

#### 【触媒2：強】東証PBR改善要請×中計ROE公約

- 東証は2025年以降もPBR1倍未満企業への改善要求を継続・強化
- 古河電工の中期経営計画（2024〜2026）ではROE8%以上を公約
- 自社株買い・増配・政策保有株売却の組み合わせが「PBR修復トリガー」
- **直近確認事項（2025〜2026）**：
  - 自社株買い実施（数十億円規模）
  - 政策保有株売却の段階的実施
  - 配当性向引き上げ公表

#### 【触媒3：強】EV電解銅箔の需給逼迫

- 全固体電池移行前の現行リチウムイオン電池需要は2030年まで高水準継続
- 中国メーカー（猛図電子等）の価格攻勢はあるが、日・韓系顧客の品質要求で古河電工の優位性は保持
- 生産能力増強投資（三重工場等）の効果が2025〜2026年に本格的に売上貢献

#### 【触媒4：中】洋上風力・再エネ電力ケーブル受注

- 政府の洋上風力60GW計画（2040年）に向けた海底ケーブル需要
- 古河電工は九州沖・秋田沖案件で入札参加実績
- 受注確定ニュースが株価の即時カタリストになりやすい

#### 【触媒5：中】銅価格上昇局面でのラグ効果

- 銅価格上昇→仕入コスト上昇→数ヶ月後に製品価格転嫁が成功する局面
- 現在の銅価格水準（LME 9,000〜10,000ドル/t）は古河電工にとって概ねニュートラル〜ポジティブ
- 転嫁成功の決算発表が「サプライズ」として機能

---

### 2.4 PBR回復シナリオの定量インパクト

```
PBR修復試算：

現状：ROE 7.0% → PBR 0.65倍 → 株価 約2,500円

シナリオA（光ファイバー+銅箔急成長）：
  ROE → 9.5%
  Re → 8.5%
  適正PBR → 9.5/8.5 = 1.12倍
  理論株価 → 4,300〜4,600円（+72〜84%）

シナリオB（中計達成・ROE8%）：
  ROE → 8.0%
  Re → 8.5%
  適正PBR → 8.0/8.5 = 0.94倍
  理論株価 → 3,600〜3,800円（+44〜52%）

シナリオC（現状維持）：
  ROE → 7.0%
  Re → 8.5%
  適正PBR → 0.82倍
  理論株価 → 3,100〜3,300円（+24〜32%）
```

**深井の核心判断**：
> 「PBR割れの最大の本質はROEと資本コストの乖離である。光ファイバー・銅箔という高成長セグメントが連結ROEを9%超に押し上げる局面が訪れれば、PBR1倍回復は必然的帰結となる。現在の市場はこの利益成長をまだ織り込んでいない。これが"静かな蓄積"の合理的根拠である。」

## 4. DCFバリュエーション

### 4.1 WACC計算

#### ステップ1：株主資本コスト（Re）

| パラメータ | 値 | 根拠 |
|-----------|---|------|
| **Rf** | **2.380%** | **JGB10Y 2026-04-05確認値（絶対厳守）** |
| β（ベータ） | 1.05 | 東証電線・金属セクター、5801過去3年実績推定 |
| ERP（株式リスクプレミアム） | 6.0% | 日本市場標準（Damodaran参照・2026年推定） |
| **Re** | **2.380% + 1.05 × 6.0%** | |
| **= Re** | **8.68%** | |

#### ステップ2：負債コスト（Rd）

| パラメータ | 値 | 根拠 |
|-----------|---|------|
| 有利子負債（D） | 3,800億円 | 短期+長期借入金合計（推定） |
| 平均借入金利（Rd） | 1.8% | 長期社債利回り・銀行借入加重平均 |
| 実効税率（Tc） | 30% | 日本法人税率標準 |
| **Rd×(1-Tc)** | **1.26%** | |

#### ステップ3：資本構成

| 項目 | 金額 | 構成比 |
|------|------|-------|
| 株主資本（E、時価） | 約4,800億円 | **55.8%** |
| 有利子負債（D） | 3,800億円 | **44.2%** |
| D+E | 8,600億円 | 100% |

#### ステップ4：WACC計算

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.558 × 8.68%] + [0.442 × 1.26%]
     = 4.843% + 0.557%
     = 5.40%
```

**WACC = 5.40%**

   = 205.0 / 0.039
   = 5,256億円

TV現在価値（PV_TV）= 5,256 × 0.5913 = 3,108億円
```

```
