## 1. 財務実態の衝撃的暴露

### 1.1 旧レポート vs 実財務データの対比

旧深井BUYレポートの推定値が実財務と甚だしく乖離していた事実を最初に開示する。

| 指標 | **旧レポート推定（FY2024）** | **実財務（Kabutan/開示、FY2024）** | **乖離率** |
|------|--------------------------|--------------------------------|----------|
| 売上高 | **約420億円** | **217.4億円** | **△48%（2倍の過大推定）** |
| 営業利益 | **約38億円** | **10.3億円** ※ | ─ |
| 営業利益率 | 約9.0% | 約4.7% | ─ |
| 純利益 | **約28億円** | **約6億円（推定）** | **▲77%** |
| FCF（ベースFCF） | **約25億円** | **約5〜10億円（推定）** | **▲60〜80%** |

※ FY2024実績（2024年2月期）は10.3億円程度と推定。その後FY2025（2025年2月期）は**営業赤字（▲3億円）**に転落。

> 結論：旧レポートは架空の「成熟期高収益モデル」をベースにDCFを組んでいた。実態は受注サイクルの谷間に入り**FY2025は赤字、FY2026は超薄利（0.77%）の中堅製造業**である。

## 2. 中計の真贋検証

### 2.1 第4次中期経営計画（FY2026〜FY2028）目標

| 項目 | **中計目標（FY2027）** | **FY2026実績** | **達成に必要な変化** |
|------|---------------------|--------------|-------------------|
| 売上高 | **270億円** | 211.7億円 | **+27.5%増（1年間で）** |
| 営業利益 | **17.8億円** | 1.62億円 | **11.0倍増** |
| 営業利益率 | 6.6% | 0.77% | +5.8%pt改善 |
| 新規事業売上 | 80億円（FY2028目標） | 不明（小額） | 事実上新規事業の立ち上げ |
| ROE | 6.0%以上 | 低水準 | 大幅改善 |
| 3期累計配当 | 20億円 | 増配発表 | ─ |

**FY2027会社予想（2026年4月3日開示）**:
- 売上高: **270億円**
- 営業利益: **17.8億円（EPS予想: 49.3円）**
- 配当: **20円/株**

### 2.2 中計達成可能性の検証 ─ 「保守的 vs 楽観的」判定

#### ① 売上高+27.5%目標の実現性

```
【達成シナリオ分解（FY2027：+58.3億円増収の内訳推定）】

┌──────────────────────────────────────────────┐
│ 受注残消化（既受注分）          推定 +30〜40億円  │
│ Global South（インド等）新規    推定 +10〜15億円  │
│ 中国市場の回復                  推定 +5〜10億円   │
│ 新規事業（FY2027開拓期）        推定 +3〜5億円    │
│ 合計上振れ目標                  +48〜70億円       │
└──────────────────────────────────────────────┘

→ 「達成可能」な条件：
  ✅ 受注残高が現在（2026年4月時点）で最低50〜60億円以上積み上がっている
  ✅ インド受注が本格化している（受注確定情報は非開示のため要確認）
  ❌ イラン戦争によるSAP・ナフサ急騰で顧客（花王・ユニ・チャーム）の設備投資が凍結していない
```

**深井判定**: 「保守的」ではなく**「過度に楽観的な達成目標」**。受注サイクルの遅延リスクと顧客設備投資凍結リスク（後述）が同時に顕在化している現状では、**達成確率は30〜40%**と評価する。

#### ② 営業利益「11倍増」目標の実現性

```
【マージン回復メカニズムの検証】

FY2026実績: 売上211.7億円、OP1.62億円（0.77%）
FY2027目標: 売上270億円、OP17.8億円（6.6%）

増収分(+58.3億円)の限界利益率を推定すると：
追加OP = FY2027 OP - FY2026 OP = 17.8 - 1.62 = 16.18億円
追加売上 = 58.3億円
暗示される追加限界利益率 = 16.18/58.3 = 27.8%

→ これはコスト構造が全く変わらない前提での試算。
→ 原材料（ナフサ→SAP系）および運賃急騰下では、
  コスト削減と増収を同時達成するのは「至難の業」。
```

**深井判定**: 限界利益率27.8%の達成には、固定費吸収効果が必要。しかしイラン戦争によるコストプッシュ（後述セクション3・4）が同時に押し寄せている。**現時点でのOP17.8億円目標達成確率は25〜35%**。

#### ③ 新規事業80億円（FY2028）目標

新規事業の詳細は非開示だが、現状の売上規模（212億円）から80億円（38%相当）を新規で立ち上げるのは**極めて困難**。「Global South展開」が主軸とされるが、インフラ・ファイナンスのリードタイムを考慮するとFY2028での実現は疑問。

**中計総合判定**: ⚠️ **「楽観的ストレッチ目標」** ─ 旧来の受注積み上げに甘えた計画。イラン戦争後の世界では、この計画は「砂上の楼閣」の公算大。

## 3. 原材料高騰による「設備需要蒸発」リスク定量化

### 3.1 連鎖メカニズムの全体図

```
【コスト連鎖マップ（2026年3月〜現在）】

イラン戦争（2026年2月〜）
    ↓ ホルムズ海峡封鎖（3月4日）
原油急騰: $60/bbl → $120/bbl（2倍）→ 現在$111/bbl
    ↓
ナフサ(C&F Japan)急騰: +60%（$620/mt → $1,010〜1,050/mt）
    ↓ 三菱ケミカル等、2026年4月1日出荷分より
アクリル酸値上げ: +40円/kg以上（確定）
    ↓ 四半期ラグ（1〜2Qで価格転嫁）
高吸水性樹脂（SAP）値上げ: 推定+20〜35%
    ↓
おむつメーカー（花王・ユニ・チャーム等）の製造原価急上昇
    ↓
①価格転嫁: 消費者おむつ販売数量減少
②資本投資凍結: 瑞光への設備発注キャンセル・延期
    ↓
瑞光のFY2027売上目標（270億円）が未達リスク
```

### 3.2 SAP価格上昇がおむつメーカー採算に与える影響試算

#### SAP価格変動の定量把握

| 項目 | 数値 | 備考 |
|------|------|------|
| SAP単価（2026年3月以前） | ¥150〜180/kg（推定） | 中国安値競争後の底値圏 |
| アクリル酸値上げ（確定） | +40円/kg以上 | 2026年4月1日出荷分（三菱ケミカル） |
| ナフサ→アクリル酸の製造コスト比率 | 約70% | 業界通説、ニュースイッチ報道 |
| SAP価格への波及（推定） | **+25〜40%**（¥37〜72/kg） | ナフサ+60%を70%係数で換算 |
| 新SAP価格レンジ（推定） | ¥187〜252/kg | 従来比+25〜40% |

#### おむつメーカーへの採算影響

```
【花王・ユニ・チャームのコスト増試算】

前提:
・おむつ1枚あたりSAP使用量: 約12〜18g（乳児用）/ 30〜50g（大人用）
・乳児用SAP使用量: 15g/枚（中央値）
・大人用SAP使用量: 40g/枚（中央値）

【乳児用おむつ（乳幼児紙おむつ: 参考Goo-n価格¥1,500/32枚 ≈ ¥47/枚）】
SAPコスト上昇: 15g × +40円/kg ÷ 1,000 = +0.60円/枚
販売価格に対する原価率上昇: +0.60 / 47.0 = **+1.3%**

【大人用おむつ（参考価格: ¥80〜120/枚）】
SAPコスト上昇: 40g × +40円/kg ÷ 1,000 = +1.60円/枚
販売価格に対する原価率上昇: +1.60 / 100 = **+1.6%**

→ SAP単体の直接コスト影響は1.3〜1.6%で「軽微」に見えるが——
```

> **重要**: ナフサ急騰は同時に**ポリプロピレン（不織布原料）、ポリエチレン（バックシート）、弾性材（ゴム系）**の価格も押し上げる。おむつの素材の80%以上が石化由来。総合コスト上昇は**+8〜15%**と試算する。

```
【おむつメーカー総合原価増試算（ユニ・チャーム・花王を想定）】

おむつ製造原価内訳（推定）:
・SAP（高吸水性樹脂）: 20〜25%
・不織布（ポリプロピレン系）: 25〜30%
・ポリエチレンフィルム（バックシート）: 10〜15%
・弾性部材・粘着剤等: 10%
・加工費・減価償却: 20〜25%
・その他: 5〜10%

石化由来原料の合計比率: 約55〜70%

各素材の価格上昇推定（ナフサ+60%起点）:
・SAP: +25〜40%（アクリル酸経由）
・PP不織布: +15〜25%（エチレン→PP）
・PEフィルム: +15〜20%

総製造原価への波及（加重平均）:
= 0.225 × 32.5% + 0.275 × 20% + 0.125 × 17.5% + 0.10 × 15%
= 7.3% + 5.5% + 2.2% + 1.5%
= **+16.5%（製造原価ベース）**

おむつ業界の営業利益率（花王: 約8%, ユニ・チャーム: 約15%）:
花王: 原価率60%前後 → +16.5%原価増 → OP影響: ▲9.9%pt → 赤字転落リスク
ユニ・チャーム: 原価率55%前後 → ▲9.1%pt → 営業利益がほぼ消失
```

**深井所見**: 2022〜2025年の値上げで消費者の価格感応度が高まった現在、追加値上げは**「販売数量の減少」**につながりやすい。特に中国・東南アジアの新興国市場では、現地安値品への代替が加速する。

### 3.3 おむつ販売数量減少 → 瑞光への発注キャンセル・延期リスク定量化

```
【設備発注意思決定のロジックモデル】

おむつメーカーの設備投資判断基準（推定）:
  新設備投資 → 生産能力増強 → 販売数量拡大が前提

但し現状:
  ①SAP・PP等コスト+16.5%で採算悪化
  ②追加値上げ → 販売数量▲3〜8%と推定
  ③新興国での需要喪失（中国ローカル品代替加速）
  
  → 生産ライン増強の「正当化」ができない

【機械発注サイクルへの影響】

一般的な設備投資决策スケジュール:
  原価上昇確認 → 採算審査 → 新規投資凍結決定: 3〜6ヶ月
  既発注分のキャンセル判断: 1〜3ヶ月（違約金考慮）

影響タイミング: 2026年7〜10月頃に決定→FY2027（2027年2月期）売上に反映
```

#### シナリオ別・発注影響額試算

| シナリオ | 発注影響 | 瑞光売上へのインパクト | 発生確率 |
|---------|---------|---------------------|---------|
| **楽観** | 影響軽微（価格転嫁成功・数量維持） | ▲5〜10億円（FY2027） | 20% |
| **基本** | 新規発注凍結（既受注は維持） | **▲20〜35億円**（FY2027） | 50% |
| **悲観** | 既受注の一部キャンセル＋新規凍結 | **▲40〜60億円**（FY2027） | 30% |

**加重平均の発注影響**: ▲27億円（FY2027）

→ FY2027売上目標270億円 から ▲27億円 = **実質的達成予想: 約243億円**（目標の90%）
→ OP目標17.8億円からの利益インパクト（限界利益27.8%として）: ▲7.5億円
→ **実質OP予想: 10.3億円**（目標の58%）

**深井判定**: ⚠️ **設備発注の一部停滞はほぼ確実**。問題は「どこまで深刻か」。イラン戦争が長期化すれば悲観シナリオの確率が高まる。

```
輸送費比率（戦後・現在）: **1.5〜2.7%（+0.4〜0.9%pt）**

年間輸出本数（推定）: 10〜20ライン/年
年間追加輸送費（推定）: ¥0.4億 × 15機 = **約6億円/年の追加負担**
```

> **重要**: ¥6億円の追加輸送費は、FY2027のOP目標17.8億円の**33.7%**に相当。これだけで目標達成可能性が大幅に低下する。

### 4.5 中国向け輸出リスク ─ COSCO運航停止の衝撃

2026年3月29日、中国国営の中国遠洋海運（COSCO）のコンテナ船がホルムズ海峡を強制的に引き返した。中国の「安全通行」の取り決めが崩壊。

```
【COSCO停航による影響】
・瑞光の対中輸出に使用する船社: COSCOが主流と推定
・代替: Maersk・CMA-CGM等の欧州系（運賃+20〜30%高い）
・引き渡し遅延リスク: 2〜4週間

→ 中国顧客への装置引き渡し遅延 → 売上認識の四半期ズレ
→ 中国顧客の「受け取り拒否・検収遅延」リスクも増大
```
