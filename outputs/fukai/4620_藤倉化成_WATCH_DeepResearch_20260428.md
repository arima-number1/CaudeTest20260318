## セグメント別分析

### ①塗料セグメント（今期の主役）

- **内容**：建築用塗料（Febriller、Lapis Grande、Sinagio SG等）、リフォーム向け
- **今期動向**：リフォーム需要好調 → 上方修正の主因
- **ナフサ感応度**：**高**（溶剤（トルエン・キシレン系）を主原料とする塗料を製造）
- **転嫁力**：今回の修正でOP+15%=転嫁が現状は機能していることを示唆

### ②コーティング（プラスチック塗料）

- **内容**：自動車内外装・ライティング部品向け塗料、化粧容器・フィルム
- **自動車向け比率**：売上の約30-35%推定
- **ナフサ感応度**：**高**（溶剤・樹脂原料使用）
- **見通し**：EV化加速でプラスチック部品塗料需要は堅調見込み

### ③電子材料

- **内容**：導電性樹脂材料（DOITE）、PCB回路材料、EMIシールド材、タッチパネル絶縁ペースト
- **ナフサ感応度**：**低**（エポキシ樹脂ベースが中心）
- **成長性**：EV・xEV向けEMIシールドが高成長領域

### ④化成品

- **内容**：感圧接着剤、機能性粒子、トナー用材料、ウレタン樹脂
- **ナフサ感応度**：**中**（ウレタン系はMDI/TDI、非ナフサ直接）

### ⑤合成樹脂（高付加価値ニッチ）

- **内容**：医療用診断試薬原料（ラテックス粒子・蛍光・磁性粒子）
- **ナフサ感応度**：**最低**（スペシャリティ・高付加価値）
- **成長性**：体外診断薬市場拡大で安定成長

### 4620の耐性評価

**ポジティブ要因（大丈夫な根拠）：**
1. **現預金¥148.9億**：短期コスト急騰を吸収できる財務バッファ
2. **今期OP+76%（¥13億→¥23億）**：コスト上昇下でも増益確保 = 転嫁力あり
3. **リフォーム需要強**：塗料の価格引き上げ余地 = 最終需要が旺盛
4. **中国輸出規制 → 国内供給増**：三井化学等が国内溶剤増産（supply改善）

**ネガティブ要因（リスク）：**
1. **7月以降の旭化成「臨界点」シグナル**：同業と同じ溶剤調達問題が顕在化する可能性
2. **今期の経常利益含む一時益**：持合株売却益を除いた正常化EPSはEPS¥102より低い（≒¥65-75）
3. **転嫁ラグ**：値上げ浸透まで数ヶ月かかる構造

**総合判定：FY2026/3（現在）は大丈夫。FY2027/3 Q2（7-9月）が最大リスク窓。ただし財務バッファが厚く（¥148.9億現預金）、転嫁実績があるため致命傷にはなりにくい。**

## TP算出（Base/Bull/Bear）

### Base TP：¥1,600（+31.0%）

**PBR 1.0-1.05x アプローチ（BPS¥1,533 × 1.04x）**
- 理由：持合株解消進捗 + 総還元性向70%維持 → ROE改善でPBR パリティ回帰
- FY2027/3正常化NI¥20-25億 → ROE 7-9% → PBR 1.0-1.1xが適正
- 配当利回り 1.25%（¥20/¥1,600）でも小型割安株として許容範囲

### Bull TP：¥2,000（+63.7%）

**発動シナリオ：持合株解消完了 + ナフサ安定 + ROE改善**
- 持合株完全解消 → 追加売却益 + BS軽量化 → ROE 12-15%へ
- PBR 1.3x × BPS ¥1,533 = ¥1,993 → ¥2,000
- 電子材料（EMIシールド）がEV向けに急成長するシナリオ

### Bear TP：¥900（▲26.4%）

**発動シナリオ：ナフサ調達停止 + 建設需要急落**
