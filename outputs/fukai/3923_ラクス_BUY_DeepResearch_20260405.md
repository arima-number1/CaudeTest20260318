#### 2.1.2 チャーンレートと企業価値の関係（LTV/CAC分析）

```
【LTV（顧客生涯価値）計算】

前提：
  ARPU（顧客1社あたり月次ARR）  ≈ 20,000〜30,000円/月
  グロスマージン                ≈ 70%（SaaS部分）
  月次チャーン率（c）           ≈ 0.65%（中間値）

LTV = ARPU × グロスマージン ÷ 月次チャーン率
    = 25,000 × 70% ÷ 0.65%
    = 25,000 × 0.70 ÷ 0.0065
    ≈ 2,692,308円/顧客（≒約270万円）

【CAC（顧客獲得コスト）推計】
  売上高販管費率（マーケ＋セールス）≈ 30〜35%
  年間新規顧客獲得数               ≈ 5,000〜8,000社
  年間マーケ・セールス費用         ≈ 40〜60億円（推計）
  CAC ≈ 60億円 ÷ 6,500社 ≈ 92万円/社

【LTV/CAC比率】
  LTV/CAC ≈ 270万円 ÷ 92万円 ≈ 2.9〜3.2倍

※SaaS業界の健全基準は3倍以上。ラクスは境界線上〜健全水準。
  チャーンが0.5%に低下すれば LTV ≈ 350万円 → LTV/CAC ≈ 3.8倍（大幅改善）
```

**チャーンレート改善の企業価値インパクト（感応度）：**

| 月次チャーン率 | 年次チャーン率 | LTV（推計） | LTV/CAC | 企業価値への含意 |
|-------------|-------------|-----------|---------|--------------|
| 1.00%/月 | 11.4%/年 | 175万円 | 1.9倍 | 要注意圏 |
| 0.80%/月 | 9.3%/年 | 219万円 | 2.4倍 | 要観察 |
| **0.65%/月** | **7.6%/年** | **270万円** | **2.9倍** | **現状推計（基本） |
| 0.50%/月 | 5.8%/年 | 350万円 | 3.8倍 | 強気シナリオ |
| 0.35%/月 | 4.1%/年 | 500万円 | 5.4倍 | ベストケース |

**→ チャーンが0.15%ポイント改善するだけで、LTVが約30%上昇し、DCF上の企業価値が顕著に拡大する。**

### 2.2 自己株式取得50億円の需給インパクト定量化

#### 2.2.1 自社株買いのスペック確認

| 項目 | 内容 |
|------|------|
| 取得上限金額 | **50億円** |
| 発表時期 | 2025年11月（推定） |
| 取得期間 | 〜2026年度内（推定） |
| 目的 | 資本効率向上・株主還元・EPS希薄化防止 |
| 株価（分析日想定） | **約700〜800円/株**（東証プライム水準での推計） |

> **注意:** 分析日2026年4月5日時点の正確な株価はリアルタイムデータ未取得のため、レンジ推計（700〜800円）を使用。以下では基準値760円として試算。

#### 2.2.2 需給インパクトの数理化

```
【取得株数の推計】

自社株買い総額:   5,000,000,000円（50億円）
想定平均取得株価: 760円/株（現在値基準）
推計取得株数:     5,000,000,000 ÷ 760 ≈ 6,578,947株（約658万株）

【発行済株式数との比率】
発行済株式数（推計）: 約2億4,000万株
自社株買い比率:        658万株 ÷ 24,000万株 ≈ 2.74%

【一日あたりの買い圧力】
東証での一日平均売買代金（ラクス推計）: 約15〜25億円/日
取得期間: 約12ヶ月（240営業日）
一日当たり自社株買い額: 50億円 ÷ 240日 ≈ 2,083万円/日

需給比率: 2,083万円 ÷ 20億円（日次売買代金中間値）≈ 1.04%

→ 市場全体の日次売買の約1%を継続的に買い支える効果
```

#### 2.2.3 EPS・PER・理論株価への波及効果

```
【EPS希薄化防止効果】

現行EPS（推計）: 約22〜26円/株
自社株買い後の発行済株式数: 24,000万株 → 23,342万株（▲2.74%）

EPS改善効果:
  改善前EPS:  24円/株（中間推計）
  改善後EPS:  24円 × (24,000 ÷ 23,342) ≈ 24.68円/株
  EPS上昇率:  +(0.68円/24円) ≈ +2.8%

【理論株価への直接効果】
PER水準（市場コンセンサス）: 60〜80倍（高成長SaaS）
基準PER: 70倍
  改善前: 24.0円 × 70倍 = 1,680円
  改善後: 24.7円 × 70倍 = 1,729円
  差額: +49円（+2.9%）
```

#### 2.2.4 チャーンレート低下 × 自社株買いの「複合効果」定量化

本分析の核心。両効果のシナジーを以下のマトリクスで定量化する。

**[複合効果シナリオ・マトリクス]（理論株価に対する変化率）**

| | 自社株買い効果なし | 自社株買い50億円完遂 | 自社株買い＋追加枠 |
|---|---|---|---|
| **チャーン現状維持（0.65%/月）** | ±0%（ベース） | +2.9% | +5.0% |
| **チャーン改善（0.50%/月）** | +15〜20% | +18〜23% | +21〜27% |
| **チャーン大幅改善（0.35%/月）** | +35〜45% | +38〜49% | +42〜55% |
| **チャーン悪化（0.90%/月）** | ▲20〜30% | ▲18〜27% | ▲16〜25% |

```
【最重要ポイント】
チャーンレートが「0.65% → 0.50%/月」に改善した場合:
  - LTV改善:        +29.6%
  - ARR成長加速:    YoY成長率 +3〜5pp 改善見込み
  - DCFターミナル価値: +18〜22%上昇（成長率への好影響）
  
これに自社株買いの+2.9%効果が加わると:
  複合効果 ≈ +21〜26%の理論株価押し上げ ← 現在値比で非常に大きい
```

#### 2.2.5 需給の盾（浅井スキャン "需給の盾" 概念の定量解釈）

```
「需給の盾」の数理的定義:

  自社株買いによる下値支持力 =
    一日平均買入額（2,083万円/日）× 株価下落時の買い加速係数（1.5〜2.0倍）
    = 3,125〜4,167万円/日 の実効的な下値買い支え

  株価が760円から10%下落（684円）した場合:
    → 同予算で取得株数が増加（6,578,947 → 7,309,942株相当）
    → 株主価値集中効果が増大
    → 下値での自社株買い加速が理論的に下値抵抗線を形成

  ≫ 「需給の盾」= 10%下落圏での実効的な株価サポートライン
```

## 4. DCFバリュエーション

### 4.1 WACC算定

#### 4.1.1 パラメータ設定

| パラメータ | 値 | 根拠 |
|-----------|-----|------|
| **Rf（リスクフリーレート）** | **2.380%** | JGB10Y（社長確認値 2026-04-05 18:39 JST） |
| **ERP（株式リスクプレミアム）** | **5.5%** | 日本株市場の標準ERP（Damodaran参照、5.0〜6.0%レンジの中央値） |
| **β（ベータ）** | **1.25** | 高成長SaaS・東証プライム。市場感応度高め。過去2年レジエスト推計 |
| **Re（株主資本コスト）** | 2.380% + 1.25 × 5.5% = **9.255%** | CAPM |
| **Rd（負債コスト）** | **1.0%** | 実質無借金企業、短期借入中心の推計 |
| **Tc（実効税率）** | **30%** | 日本法人税率標準 |
| **E/（D+E）（株式比率）** | **90%** | 低負債構造 |
| **D/（D+E）（負債比率）** | **10%** | 低負債構造 |

#### 4.1.2 WACC計算

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.90 × 9.255%] + [0.10 × 1.0% × (1-0.30)]
     = 8.330% + 0.070%
     = 8.400%

■ 採用WACC = 8.40%
```
