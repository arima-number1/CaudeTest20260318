## 💰 バリュエーション・FV算出

### コスト・オブ・エクイティ（Ke）— β実測値修正

```
Rf   = 2.394%（10年JGB、財務省CSV 2026/4/21確認）
β    = 1.474（yfinance実測値）
ERP  = 6.0%（標準設定）
Ke   = 2.394% + 1.474 × 6.0% = 11.238%
内在P/E（成長ゼロ） = 1 / 0.11238 = 8.90x
内在P/E（g=5%） = 1 / (0.11238-0.05) = 16.0x
```

**※ Ichiyoshi Research のWACC 6.1%はβ実測値と大きく乖離。**
**6.1%を使うにはβ≈0.6が必要だが実測β=1.474はほぼ倍。**
**Ichiyoshi DCF ¥4,600は過大評価の可能性が高く採用せず。**

### ROE-Ke スプレッド分析

```
ROE  = 16.28%（yfinance確認）
Ke   = 11.238%（β実測値）
スプレッド = +5.04pp（正値→価値創造企業）

内在的適正PBR = ROE / Ke = 16.28% / 11.238% = 1.45x
実際のPBR = 2.8x
→ 市場はROE/Ke比を93.1%上回るプレミアムを付与
= 成長期待（Metal材料×AI）が株価に先行して反映済み
```

**⚠️ 現PBR 2.8xはゼロ成長モデルの適正PBR 1.45xの1.93倍。**
**BUYの成立条件は「成長シナリオの実現」が前提。成長失速ならPBR圧縮余地あり。**

## 🐂 BUY論拠

### BUY①：ナフサフリー＝ホルムズ×原材料高環境での決定的差別化

**主原料が金属（ハフニウム・タンタル・タングステン系）のためナフサ価格上昇の影響を受けない**（大谷AN 4/21朝会確認）。

| 比較軸 | トリケミカル | ナフサ依存化学材料メーカー |
|--------|------------|--------------------------|
| ホルムズ影響 | **軽微** | 深刻（ナフサ価格+30〜50%） |
| 原材料コスト | 金属価格（ハフニウム等） | ナフサ（石油誘導体） |
| FY01/27の圧迫源 | Hf価格×固定費増加 | ナフサ+Hf両方 |

現環境（ホルムズ封鎖継続）で化学材料同業が軒並み苦境の中、ナフサフリーは競争優位として際立つ。

### BUY②：CXMTローカルシフト影響は限定的

中国のメモリメーカーCXMTがHigh-k材料でローカルサプライヤーへの移行を進めているが、**量産化のハードルが高い**（大谷AN）。

- **超高純度化学材料の国産代替困難性**：世界最高水準の純度が求められるHigh-k材料の量産化は技術的障壁が高い
- **実績**: Q1 FY01/26に中国売上¥2.9B（+YoY大幅増）を達成 → まだ代替は進んでいない
- **リスク認識は維持**：ただし2Hにかけて中国High-kは減速しており、FY01/27への影響は「軽微だが無視できない」レベル

### BUY③：Metal材料転換 — Hf価格リスク低下×収益安定化

| 指標 | High-k材料 | Metal材料 |
|------|-----------|-----------|
| 主原料 | ハフニウム（希少×価格変動大） | タンタル・タングステン（比較的安定） |
| 市場成長 | 鈍化（既存ノードでは採用済み） | **加速（2nm以降のBarrierで必須）** |
| 2nm/1.4nm対応 | Yes（ただし競争激化） | **Yes（次世代で中心的役割）** |
| 収益変動性 | 高（Hf依存） | **低い** |
| FY01/29時点 | Metal材料の下位になる見込み | **売上比率でHigh-kを逆転** |

**Metal材料比率の上昇 = 収益安定性の向上 = 高倍率バリュエーションの正当化**

### BUY④：AI半導体向け需要の中長期増大

- 会社売上90%超が半導体向けで、特にロジック（AI向け）が主体
- TSMC 2nm/1.4nm量産加速（2026-2027）→ High-k+Metal材料の同時需要増
- AI向け半導体メーカー（TSMC・Intel・Samsung）の設備投資は2026年も最高水準継続

### BUY⑤：FY01/26実績がガイダンスを上回り通期好進捗

大谷AN（4/21朝会）：「会社予想は中国のHigh-k材料での新規参入影響や為替前提等で保守的と判断」
→ **FY01/27もガイダンスに対して保守的な前提が入っている可能性**
→ Q1 FY01/27（6月発表）でガイダンス超過が確認されれば株価の再評価材料

## 🐻 リスク一覧

### リスク①：中国High-k需要の急落（最大リスク）

- Q1 FY01/26の中国売上¥2.9B（全体の44%）
- 2H FY01/26でCXMT在庫調整+生産システム合理化が発生
- CXMTのローカルシフトが量産化に成功した場合、中国向けが構造的に縮小
- **FY01/27 EPS¥172 → ¥143 Bear シナリオの主因**

### リスク②：FY01/27 固定費増大

- 南アルプス工場増強設備の稼働開始（減価償却費増加）
- 労務費・諸経費の増大
- 「FY1/26結果がガイダンスを超過したが、固定費・原材料コストがFY1/27を圧迫」（Otani 3/13/26）

### リスク③：SK TRI CHEM 持分法利益の大幅減少

- ハフニウム系材料の原材料コスト急増
- 売値下落（競争激化）が見込まれる
- 持分法利益の減少→経常利益への圧迫

### リスク④：ハフニウム価格上昇

- High-k材料の主原料ハフニウム（Hf）は希少金属
- ホルムズ封鎖×精錬コスト上昇が重なる可能性
- Metal材料転換が完了するまでのリスク（FY01/29まで）

### リスク⑤：高β（1.474）による株価変動性

- 市場全体のリスクオフ時に大幅に下落しやすい（中小型成長株）
- ホルムズ停戦→リスクオフ解除→資金の大型株シフト時には相対的に売られやすい
