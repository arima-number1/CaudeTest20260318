## 4. DCFバリュエーション

### 4-1. WACCの算定

```
■ インプット
Rf = 2.392%（日本10年国債 2026-04-22）
β  = 0.75（炭素製品・素材セクター：景気連動性中程度、低流動性調整後）
ERP = 5.5%（日本株式リスクプレミアム：学術的コンセンサス範囲4.5〜6.0%の中央値）
Re = Rf + β × ERP = 2.392% + 0.75 × 5.5% = 2.392% + 4.125% = 6.517%

D/(D+E) = 20%（高自己資本比率：推定80%株式/20%負債）
E/(D+E) = 80%
Rd = 0.8%（日本の低金利環境・無借金同然）
Tc = 30%（実効税率）

WACC = [0.80 × 6.517%] + [0.20 × 0.8% × (1-0.30)]
      = 5.214% + 0.112%
      = 5.326%（≒ 5.3%）
```
