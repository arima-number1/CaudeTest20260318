## 5. メインシナリオ vs リスクシナリオ

### 5-1. メインシナリオ（確率: 55%）

```
【前提】
• 自動車CASE需要が継続拡大
• 採用・育成コストをコントロールしながら技術者数+10〜15%/年
• ソリューション事業のSaaS/サブスク化が進捗
• 株式分割で個人投資家層が拡大→出来高改善→機関投資家注目

【業績シナリオ】
FY2026: 売上+15%、営業利益率15〜17%維持
FY2027: 売上+13%、ソリューション比率拡大でGM改善
FY2028: 売上+10〜12%、利益成長率が売上成長率を上回る

【株価インパクト】
分割発表後1〜3ヶ月: +20〜35%（需給主導）
12ヶ月後:           +25〜40%（業績+バリュエーション双方）
理論株価:           3,200〜3,440円（分割前）
```

### 5-2. 強気シナリオ（確率: 20%）

```
【前提】
• 海外自動車メーカーへの展開成功
• 生成AI活用による開発効率化でGM+5pt改善
• 東証グロース250or他指数への採用

【株価インパクト】
12ヶ月後: +50〜80%
理論株価: 4,200〜4,800円（分割前）
```

### 5-3. リスクシナリオ（確率: 20%）

```
【前提・リスク要因】
⚠️ 自動車産業の投資抑制（EV移行停滞・関税戦争影響）
⚠️ エンジニア採用競争激化による人件費急騰
⚠️ キーパーソン離脱
⚠️ 大手ITベンダーの同領域参入

【業績シナリオ】
FY2026: 売上+5%止まり、営業利益率10%割れ
FY2027: 採用コスト先行で一時的減益

【株価インパクト】
12ヶ月後: -15〜25%
理論株価: 1,800〜2,200円（分割前）
```

### 5-4. 最悪シナリオ（確率: 5%）
