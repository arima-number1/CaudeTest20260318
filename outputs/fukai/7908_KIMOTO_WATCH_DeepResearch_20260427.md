## 4. 低PBR分析

### 4-1. 現在の指標

| 指標 | 数値 | ソース |
|------|------|-------|
| 株価 | 251円 | Yahoo Finance Japan（2026/4/24終値） |
| BPS（実績） | 434.72円 | 決算短信・Yahoo Finance |
| PBR（実績） | 0.58倍 | Yahoo Finance |
| ROE（実績） | 2.99% | Yahoo Finance |
| 自己資本比率 | 81.7% | 決算短信 |

### 4-2. 低PBRの構造的原因

1. **低ROE（2.99%）**: WACC推定8.5%（中計記載）を大幅に下回り、理論PBR（ROE/Ke）は0.35倍水準。帳簿純資産18,982百万円に対して時価総額10,960百万円（浮動株ベース）と純資産を大幅に下回る「解散価値割れ」状態。

2. **過剰な金融資産保有**: 現預金10,967M円＋長期預金2,719M円＝13,686M円を保有。これは総資産23,245M円の58.9%。ROEの分母（純資産18,982M円）を不必要に膨らませ、ROEを構造的に押し下げている。**事業に使われない金融資産が低PBRの本質的原因**。

3. **北米・欧州子会社の慢性的赤字**: 米国▲262M円、スイス▲24M円、中国▲61M円（FY2026）と海外3拠点が全赤字。グループROEのドラッグになっている。

4. **自動車・バッテリー向けのサイクル低迷**: 高収益製品（通信機器向け）の構成比が下がり、低採算品の影響が拡大。

### 4-3. PBR改善シナリオ（東証要請への対応状況）

第6次中計（2025年5月開示）で以下の対応を明示：

| KPI | 現状(FY2026) | 中計目標(FY2028) |
|-----|-------------|----------------|
| ROE | 3.0% | **8.0%** |
| WACC | 8.5% | 同水準想定 |
| 営業利益率 | 10.1% | **16.0%** |
| 配当性向 | 55.6%（実態35%） | **40.0%** |
| 成長投資 | — | **6,000百万円（3年間）** |
| 金融資産比率 | 42%（2025年3月期） | **34%**（2028年3月期目標） |

成長投資6,000M円の内訳：新規ビジネス3,500M円、環境対応投資1,300M円、その他成長投資2,250M円（中計資料）。金融資産を事業投資に転換することでROA・ROE改善、PBR改善の好循環を目指す。政策保有株式の解消（2027年まで）も掲げている。

### 7-3. シナリオ別TP（公正価値）

#### Base Case TP: **¥463**（現値比+85%）

**前提**: 中計目標ROE8%を2028年3月期に達成し、WACC8.5%を前提にPBR0.94倍に収束。
- FY2028 推定純利益: 1,400百万円（営業利益2,100M、OPM16%達成、実効税率33%）
- FY2028 推定EPS: 32円（浮動株43,666千株）
- 3年後BPS推定: 434.72円 + 32円×60%×3年 = 492円
- FV = BPS_FY28 × PBR_target = 492円 × 0.94 = **463円**

**Rationale**: WACC（8.5%、中計資料記載）とROE目標（8.0%）がほぼ一致する地点でPBR≒1倍への収束を織り込む。ただし未達リスクを0.94倍で若干ディスカウント。

#### Bull Case TP: **¥563**（現値比+124%）

**前提**: ROE8%達成＋市場がROE>WACC（再評価プレミアム）を評価。PBR1.14倍へ。
- FV = 492円 × 1.14 = **563円**

**Rationale**: 北米・欧州子会社の黒字転換・売却等でROEが目標を超過し、配当性向40%＋残余を自社株買いに振り向ける好循環が実現した場合。産業機器・半導体製造工程向け拡大が加速するシナリオ。

#### Bear Case TP: **¥252**（現値比+0%）

**前提**: 中計ROE目標未達（ROE5%止まり）、FY2027ガイダンス引き続き不開示、自動車・バッテリー向け構造低迷継続。PBR0.58倍現状維持。
- FV = 現BPS434.72円 × 0.58 = **252円**

**Rationale**: ガイダンス不開示が長引き、ROE改善の実績が乏しい場合、現在の評価水準が維持される。純現金313円/株というフロアが存在するため、理論上これ以下への下落は限定的（解散価値制約）。

### 9-1. 投資判断: **WATCH +45**（スコア）

本銘柄を「即時BUY」ではなく「WATCH（条件付き打診可）」とする理由：

**ポジティブ要因**:
- 現株価251円を上回る現預金313円/株（現金リッチ・解散価値割れ）
- 自己資本比率81.7%という化学セクター最高水準の財務健全性
- 第6次中計でROE8%目標・配当性向40%を明示

- FY2026純利益ミスは減損（一時的）が主因で実力収益力は維持

**ネガティブ要因**:
- **FY2027ガイダンス不開示**（中東リスク起因）：短期の催促材料が存在しない
- ROE3.0%はWACC8.5%を大幅に下回り、理論PBRは0.35倍水準（さらなる下落余地あり）
- 北米・欧州が全セグメント赤字（米国▲262M円）で構造改革の遅れが課題
- 自動車・バッテリー関連の需要回復見通し不透明（電動化の不確実性）
- 小型株（時価総額約110億円）でアナリストカバレッジが乏しく、情報発信力弱い
