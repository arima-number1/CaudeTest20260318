## 2. 社長指示重点分析：「割高で売れるか。優先度1。ショート判断の根拠を定量化せよ。」

### 2-1. バリュエーション異常の定量的証明

現在の財務指標（リアルタイム取得値）を整理する：

| 指標 | 数値 | 業界適正水準 | 乖離 |
|------|------|------------|------|
| PER | **63.9倍** | 小売セクター平均15〜25倍 | **+2.5〜4倍 割高** |
| PBR | **4.12倍** | ROE5.8%適正PBR: 0.8〜1.0倍 | **+3.1〜3.3倍 超割高** |
| ROE | **5.8%** | — | WACC（後述約6.5〜7%）を下回る可能性 |
| EPS | **¥27** | — | PER64倍の根拠なし |

#### ROE-PBR適正値の計算

**ゴードン成長モデルによる適正PBR：**
```
適正PBR = (ROE − g) / (Ke − g)
ROE = 5.8%
g（成長率） = 2.0%（名目GDP成長＋α、保守的）
Ke（株主資本コスト）= 後述 6.5〜7.0%（WACC算出参照）

適正PBR = (5.8% − 2.0%) / (6.7% − 2.0%)
         = 3.8% / 4.7%
         = **0.81倍**
```

**適正株価（PBRベース）：**
```
BPS ¥418 × 適正PBR 0.81 = **¥338**
```

> **現在値¥1,720 vs PBRベース適正値¥338。乖離率 ▲80.3%。**

これはやや極端な理論値であり、DCFとのブレンドが必要だが、**PBR・ROEの組み合わせが示す本質的割高性は数値として証明された。**

#### PERベース適正値

```
小売コングロマリット適正PER = 20〜25倍（楽観）
EPS ¥27 × 25倍 = ¥675（楽観上限）
EPS ¥27 × 20倍 = ¥540（適正中央）
EPS ¥22（下方リスクEPS） × 20倍 = ¥440
```

> **PERベースでも適正値は¥540〜¥675。現在値¥1,720比で▲60〜68%の割高。**

### 4-2. DCFモデル（基本ケース）

#### フェーズ1：FY2026〜2030（5年間）

| 年度 | FCF（億円） | 割引率(WACC=6.0%) | 現在価値（億円） |
|------|-----------|-----------------|---------------|
| FY2026 | 800 | 1/(1.06)^1 = 0.943 | **754** |
| FY2027 | 800 | 1/(1.06)^2 = 0.890 | **712** |
| FY2028 | 808 (+1%) | 1/(1.06)^3 = 0.840 | **679** |
| FY2029 | 820 (+1.5%) | 1/(1.06)^4 = 0.792 | **649** |
| FY2030 | 832 (+1.5%) | 1/(1.06)^5 = 0.747 | **621** |
| **フェーズ1合計** | | | **3,415億円** |

#### ターミナルバリュー（TV）

```
TV = FCF_FY2030 × (1+g) / (WACC - g)
   = 832 × 1.01 / (0.060 - 0.010)
   = 840.3 / 0.050
   = 16,806億円

TV現在価値 = 16,806 × 0.747 = 12,554億円
```

#### 企業価値（EV）→ 株主価値 → 理論株価

```
EV = フェーズ1現在価値 + TV現在価値
   = 3,415 + 12,554
   = 15,969億円

純有利子負債 = 約3.2兆円（32,000億円）
非事業資産（ASEAN持分・イオンモール保有資産等）= +約5,000億円（調整）

株主価値 = 15,969 − 32,000 + 5,000
          = **▲11,031億円**
```

> ⚠️ **重要**：純粋DCFでは株主価値がマイナスになる。これは**有利子負債3.2兆円**がFCF創出能力（800億円台）に対して過大であることを意味する。これは現実的な結果であり、**現在の株価¥1,720はFCFベース価値を完全に超えた「期待値」で成立している**ことを示す。
