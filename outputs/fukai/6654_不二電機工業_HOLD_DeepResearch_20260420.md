## 【本質的問い：なぜPBR0.547xなのか】

### 低PBRの二重構造

| 要因 | 内容 | 深井評価 |
|------|------|---------|
| **ROE 2.2%の低収益** | 純資産¥126億に対し純利益¥2.36億。Ke=5.6%に対しROE半分以下 | **本質的要因。資本効率が悪すぎる** |
| **流動性の低さ** | ADV約6,892株・日次売買代金¥758万。機関投資家参入不可 | 構造的なディスカウント要因 |
| **成長期待なし** | 2期連続減収減益。ただしFY2027は会社予想で増収回復 | カタリスト次第で変化可能 |

**逆説：自己資本比率93%・無借金 = 低ROEの証明でもある**
→ 利益を稼ぐより資本を積み上げる保守経営。アクティビストが介入すれば特別配当・自社株買いで一気に解消可能。

## 【事業モデルと競合優位性】

### 何をしている会社か

1953年創業、京都市本拠の**電気制御機器専業メーカー**。

| 項目 | 内容 |
|------|------|
| 主力製品 | カムスイッチ・補助スイッチ・端子台・表示灯・電子応用機器 |
| 主要顧客 | 電力インフラ・鉄道車両・工業・交通信号 |
| 製造拠点 | 滋賀県草津市（草津製作所） |
| 国内ポジション | カムスイッチ市場で国内上位（Kraus & Naimer・高千穂に次ぐ） |
| 耐久性 | 最大500万サイクル対応（鉄道・電力インフラ向け高信頼性品） |

### なぜ70年間生き残れるか

1. **インフラの長期サイクル**：鉄道・電力設備は30〜50年使用。一度採用されると後継機種も同社製が維持されやすい
2. **カスタム対応力**：顧客仕様への細かい対応。大手競合が手を出しにくいニッチ
3. **無借金・高自己資本**：景気後退でも倒産リスクゼロ。長取引関係の信頼性担保

### バランスシート概観（深井推計）

| 項目 | 推計値 | 算出根拠 |
|------|--------|---------|
| 時価総額 | **¥69.2億** | ¥1,099×630万株 |
| **純資産（BPS×株数）** | **¥126.6億** | ¥2,009×630万株 |
| 自己資本比率 | **93%** | 公式開示値 |
| 総資産推計 | **¥136億** | 純資産÷0.93 |
| 総負債推計 | **¥9.4億** | 総資産−純資産 |
| **推定純現金** | **¥60〜80億** | 総資産¥136億から固定資産（工場等）¥40〜50億控除 |

### 「純現金が時価総額に匹敵する」構造の含意

```
時価総額 ¥69.2億
  ↓
推定純現金 ¥60〜80億を差し引くと
  ↓
事業価値（EV） = ¥0〜10億
  ↓
FY2026 OP ¥2.48億 に対しEV/OP ≈ 0〜4倍
  ↓
→ 極めてチープなEV。問題は現金がいつ株主還元されるか
```

**→ これが今回のユニークな投資アイデアの核心。グレアム流NCAV投資の典型例。**
