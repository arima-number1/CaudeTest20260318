### 1-3. ニトリの競合優位性（モート分析）

**① 製造小売業（SPA）モデル**
ニトリの最大の競合優位性は、企画・製造・物流・販売を一気通貫で管理するSPA（Speciality store retailer of Private label Apparel）モデルを家具業界で完成させた点にある。

- 商品の約95%が自社企画のプライベートブランド
- 海外（主に中国・ベトナム・インドネシア）での製造委託（OEM）
- 独自の海外拠点（中国・上海、ベトナム・ホーチミン等）による品質管理
- 横浜・大阪等の自社物流センター（冷蔵機能なし・家具特化）

**② 「お、ねだん以上。」ブランドの圧倒的認知**
価格訴求力と品質の両立により、競合（IKEA・無印良品・ホームセンター各社）との差別化に成功。

**③ 為替ヘッジ構造**
円安時のコスト上昇を抑制するため、独自の為替予約を複数年先まで実施。**円高転換時には原価率低下による利益拡大**という特殊な受益構造を持つ。

**④ 不動産含み益**
自社保有店舗・物流拠点が多数存在し、帳簿価額に対して時価含み益が大きい（Section 2-6で詳述）。

### 2-4. 円高受益 × 中国リスクの二重構造分析

#### ニトリの為替感応度（円高受益の構造）

ニトリは輸入型小売業のため、**円高は原価を直接圧縮する受益企業**である。

```

─────────────────────────────────────────────
USD/JPY  140円  → 営業利益¥1,050億（基準）
USD/JPY  135円  → 営業利益¥1,120億（+6.7%）※5円円高

USD/JPY  125円  → 営業利益¥1,260億（+20.0%）

─────────────────────────────────────────────
※ 1円円高あたり営業利益+¥14億（公開説明会データより逆算推定）
```

#### 二重構造のパラドックス

```
円高進行時のシナリオ分岐
────────────────────────────────────────────────────────────
         円高進行（USD/JPY 130円台）
               ↓
    ┌──────────────────────────────┐
    │ 背景A: 日本景気堅調・金利差縮小  │
    │ → 円高受益そのまま享受          │
    │ → 営業利益+15〜20%増             │

    │ 背景B: 台湾有事・地政学リスク    │
    │ → 円高（安全資産としての円高）   │
    │ → 同時にサプライチェーン寸断     │
    │ → 円高受益 < 調達コスト増・損失  │
    │ → ネットネガティブシナリオ       │
    └──────────────────────────────┘
────────────────────────────────────────────────────────────
```

**重要な洞察：「円高=ニトリ買い」という単純公式が台湾有事では逆転する**

台湾有事シナリオでの円高は「リスクオフによる円高」であり、同時に：
1. 中国からの調達停止（コスト最大化）
2. 中国店舗・資産の損失
3. 消費者マインドの悪化（国内需要減少）
4. グローバルリセッションによる日本国内消費の冷え込み

上記4つの逆風が同時発生するため、**円高メリットが完全に相殺される**。

## 4. DCFバリュエーション

### 4-1. WACC計算

#### ステップ1: 株主資本コスト（Re）

```
Re = Rf + β × ERP

Rf  = 2.386%（JGB10Y 確定値 — 書き換え禁止）
β   = 0.75（東証プライム・小売セクターニトリ推定値）
     ※ 過去5年月次回帰ベース。内需防衛株として市場βより低め
ERP = 6.0%（日本市場ERP — Damodaran Japan ERP 2025参考）

Re  = 2.386% + 0.75 × 6.0%
    = 2.386% + 4.50%
    = 6.886%
```

#### ステップ2: 負債コスト（Rd）

```
Rd（税引前）= 1.2%（実質無借金企業、低金利での借入）
実効税率 Tc = 30%
Rd（税引後）= 1.2% × (1 - 0.30) = 0.84%
```

#### ステップ3: 資本構成

```
E（自己資本時価）= ¥3,800億（時価総額）
D（有利子負債）  = ¥860億
D+E              = ¥4,660億

E/(D+E) = 81.5%
D/(D+E) = 18.5%
```

#### ステップ4: WACC算出

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.815 × 6.886%] + [0.185 × 0.84%]
     = 5.612% + 0.155%
     = 5.767%

採用WACC = 5.77%（四捨五入）
```
