### 3.2 WACCの詳細計算

```
Re = 2.380% + 0.85 × 5.5%
   = 2.380% + 4.675%
   = 7.055%

WACC = [0.95 × 7.055%] + [0.05 × 1.5% × (1 - 0.30)]
     = 6.7023% + 0.0525%
     = 6.755% ≒ 6.75%（以下6.75%使用）
```

### 3.3 5年間FCF予測（Baseケース）

| 年度 | FCF成長率仮定 | FCF（M¥） | 割引係数(6.75%) | PV（M¥） |
|------|------------|---------|---------------|---------|
| FY2026 | +3.0% | 2,266 | 0.9368 | 2,123 |
| FY2027 | +3.0% | 2,334 | 0.8776 | 2,048 |
| FY2028 | +2.5% | 2,393 | 0.8220 | 1,967 |
| FY2029 | +2.0% | 2,440 | 0.7699 | 1,879 |
| FY2030 | +2.0% | 2,489 | 0.7211 | 1,794 |
| **合計PV** | | | | **9,811** |

**ターミナルバリュー（TV）計算:**
```
TV = FCF_2030 × (1+g) / (WACC - g)
   = 2,489 × 1.01 / (0.0675 - 0.010)
   = 2,514 / 0.0575
   = ¥43,722M

TV現在価値 = 43,722 × 0.7211 = ¥31,528M
```

**企業価値（EV）:**
```
EV = 事業PV合計 + TV現在価値
   = 9,811 + 31,528
   = ¥41,339M
```

**株式価値（Equity Value）:**
```
ネットキャッシュ = ¥17,500M（現預金・有価証券中央値）- ¥2,000M（有利子負債）= ¥15,500M

株式価値 = EV + ネットキャッシュ
         = 41,339 + 15,500
         = ¥56,839M

発行済株式数: 約12,500千株（推計）

理論株価（Base）= ¥56,839M ÷ 12,500千株 = ¥4,547/株
```

> ⚠️ **重要注記**: 発行済株式数・ネットキャッシュは公開情報ベースの推計値。数値精度は±15%程度の誤差を含む可能性がある。
