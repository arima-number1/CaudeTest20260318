# 4. DCFバリュエーション

## 4-1. 前提条件（全項目明記）

### 資本コスト前提

| パラメータ | 数値 | 根拠 |
|-----------|------|------|
| **Rf（リスクフリーレート）** | **2.380%** | **JGB10Y 社長確認値（絶対厳守）** |
| β（ベータ） | 0.85 | 東証上場・ディフェンシブ小売業（5年月次推計） |
| ERP（株式リスクプレミアム） | 5.50% | 日本株ERP（Damodaran 2025推計ベース） |
| **Re（株主資本コスト）** | **2.380% + 0.85 × 5.50% = 7.055%** | CAPM計算 |
| Rd（負債コスト） | 1.2% | 実質無借金だがリース負債想定 |
| Tc（実効税率） | 30.0% | 日本法人税率準拠 |
| E/（D+E）（株式比率） | 85% | BS構造から推計 |
| D/（D+E）（負債比率） | 15% | 同上 |
| **WACC** | **6.19%** | 以下で詳細計算 |

### WACC詳細計算

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]

= [0.85 × 7.055%] + [0.15 × 1.2% × (1 - 0.30)]
= [5.997%] + [0.15 × 1.2% × 0.70]
= [5.997%] + [0.126%]
= 6.123%

≒ 6.12%（四捨五入）
```

> **採用WACC: 6.12%**


### ターミナルバリュー（TV）

```
Terminal FCF = FY2034 FCF × (1+g) / (WACC - g)
             = 9,034億円 × (1 + 0.020) / (0.0612 - 0.020)
             = 9,215億円 / 0.0412
             = 223,665億円（TV gross）

TVの現在価値 = 223,665億円 / (1.0612)¹⁰
             = 223,665億円 / 1.8066
             = 123,818億円
```

### 企業価値（EV）計算

```
EV = FCF現在価値合計 + TV現在価値
   = 49,457億円 + 123,818億円
   = 173,275億円

ネットキャッシュ調整（+9,000億円想定）:
株式価値 = 173,275 + 9,000 = 182,275億円

発行済株式数: 約1億2,000万株（自己株控除後推計）

理論株価（ベースケース）= 182,275億円 ÷ 12,000万株
                      = 151,896円（株）

※ 単位換算:
  182,275億円 = 18兆2,275億円 = 18,227,500百万円
  18,227,500百万円 ÷ 12,000万株 = 151,896円
```

> **ベースケース理論株価: 約152,000円**
