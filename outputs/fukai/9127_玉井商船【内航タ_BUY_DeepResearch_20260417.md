## 4. DCFバリュエーション

### 4-1. WACC計算

#### Step 1: リスクフリーレート（確定）
```
Rf = 2.409%（日本10年国債・2026-04-17取得）
```

#### Step 2: ERP・β設定
```
ERP（日本市場リスクプレミアム）: 5.5%（Damodaran 2026年日本推計）
β（玉井商船推定）: 0.75
  - 内航海運：景気感応度低・安定収益型→β<1.0
  - 超小型株固有のリスク→やや引き上げ調整
  - 業界平均β（内航海運）: 0.65〜0.85の中央値

株主資本コスト（Re）:
  Re = Rf + β × ERP
  Re = 2.409% + 0.75 × 5.5%
  Re = 2.409% + 4.125%
  Re = 6.534%
```

#### Step 3: 負債コスト・資本構成
```
推定自己資本比率（E/D+E）: 70%
  （BPS¥4,937×194.5万株=純資産¥96億／推定総資産¥137億）
  
推定負債コスト（Rd）: 1.5%（内航海運・中小企業向け長期融資金利）
実効税率（Tc）: 30%（標準的日本法人税率）

WACC計算:
  WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
  WACC = [0.70 × 6.534%] + [0.30 × 1.5% × (1-0.30)]
  WACC = 4.574% + 0.315%
  WACC = 4.889% ≒ 4.9%
```
