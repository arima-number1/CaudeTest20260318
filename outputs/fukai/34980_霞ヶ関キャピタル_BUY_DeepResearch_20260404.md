## 1. 企業概要・事業分析

- **設立：** 2018年
- **上場：** 2021年（東証グロース → プライム移行）
- **ビジネスモデル：** アセットマネジメント型（AUM積み上げ→フィー収益）＋ Co-Invest（自己投資）型の**ハイブリッド**

## 4. DCFバリュエーション

### 4-1. WACC計算の前提条件

#### リスクフリーレート（Rf）
```
JGB10年債利回り = 1.5%（指定値・固定使用）
```

#### エクイティリスクプレミアム（ERP）
```
ERP = 5.5%
（Damodaran Japan ERP 推計ベース：約5.0〜6.0%の中央値）
```

#### ベータ（β）

```
β計算の考え方：
- 参照セクター：不動産AM・ファンド系（J-REIT隣接）
- 上場後の実績β（2年程度）：約1.4〜1.8（高成長・小型株バイアス）
- 採用β：1.6（中央値）

→ レバードβ（Levered β）= 1.6
```

#### 資本コスト（Ke）

```
Ke = Rf + β × ERP
   = 1.5% + 1.6 × 5.5%
   = 1.5% + 8.8%
   = 10.3%
```

#### 負債コスト（Kd）

```
借入金利（推定） = 2.0〜2.5%（不動産担保ローン）
実効税率 = 30%
税引後Kd = 2.3% × (1 - 0.30) = 1.6%
```

#### 資本構成

```
時価総額（E）：約420億円（株価3,800円 × 約1,100万株）
有利子負債（D）：約450億円（推計）
E/(D+E) = 420 / 870 ≒ 48%
D/(D+E) = 450 / 870 ≒ 52%
```

#### WACC計算

```
WACC = Ke × [E/(D+E)] + Kd × [D/(D+E)]
     = 10.3% × 0.48 + 1.6% × 0.52
     = 4.94% + 0.83%
     = 5.77%

→ 採用WACC：5.8%（四捨五入）
```

### 4-2. DCFモデル詳細（基本シナリオ）

#### FCF（フリーキャッシュフロー）予測

> 注：不動産ファンドのDCFは「調整後FCF」を使用。ファンドCo-Investの支出/回収を調整したノーマライズドFCF。

```
調整後FCF ≈ EBITDA × (1 - 税率) + ファンドExitキャッシュ回収 - 維持投資

実務的アプローチ：純利益ベースDDM的手法と組み合わせ
```

| 予測年度 | 売上高(億円) | 営業利益(億円) | NOPAT(億円) | 調整FCF(億円) | DCF(億円) |
|--------|-----------|-------------|-----------|------------|---------|
| **FY2026/8** | 580 | 130 | 91 | 65 | 61.4 |
| **FY2027/8** | 750 | 176 | 123 | 88 | 78.6 |
| **FY2028/8** | 940 | 226 | 158 | 113 | 95.6 |
| **FY2029/8** | 1,100 | 253 | 177 | 130 | 104.1 |
| **FY2030/8** | 1,250 | 288 | 202 | 148 | 112.2 |

#### ターミナルバリュー（TV）計算

```
ターミナル成長率（g）：
- 日本不動産AM市場の長期成長率：2.0〜3.0%
- 採用g：2.5%（保守的）

TV = FCF2030 × (1 + g) / (WACC - g)
   = 148億円 × 1.025 / (5.8% - 2.5%)
   = 151.7億円 / 3.3%
   = 4,597億円

TV現在価値 = 4,597億円 / (1 + 5.8%)^5
           = 4,597 / 1.325
           = 3,470億円
```

#### 企業価値（EV）計算

```
EV = 予測期間FCFのPV合計 + TV現在価値
   = (61.4 + 78.6 + 95.6 + 104.1 + 112.2) + 3,470
   = 451.9億円 + 3,470億円
   = 3,922億円

株主価値 = EV - 純有利子負債
         = 3,922億円 - 370億円（有利子負債450億円 - 現金80億円）
         = 3,552億円

理論株価 = 3,552億円 / 1,100万株
         = 約32,300円

⚠️ ※注意：このEVはファンドAUMの急成長シナリオを含むため
    後述するマルチプル法との折衷が適切
```

### 4-3. マルチプル法（PER・EV/EBITDA）による補正

```
PERアプローチ：
- 成長率調整PER（PEG考慮）：
  FY2026 EPS推計 = 770円
  適正PER（成長率40%超の局面）：15〜20倍
  → 理論株価：770円 × 17倍 = 13,090円（やや楽観）
  保守的（PER12〜15倍）：770円 × 13.5倍 = 10,395円

EV/EBITDA：
  EV/EBITDA（同業比較・プレミアム考慮）：12〜18倍
  EBITDA（FY2026推計）：約150億円
  EV = 150億円 × 15倍 = 2,250億円
  株主価値 = 2,250 - 370 = 1,880億円
  理論株価 = 1,880億円 / 1,100万株 = 17,090円
```
