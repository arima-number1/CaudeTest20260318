## 【事業モデルと競合優位性】

### 何をしている会社か

大伸化学は1952年創業、国内最大の**塗料用シンナー・有機溶剤調合メーカー**。

| 項目 | 内容 |
|------|------|
| 主力製品 | ラッカー・合成樹脂塗料用シンナー、インキ溶剤、洗浄溶剤、工業用有機溶剤 |
| 製品数 | 約32,000品目（多品種少量生産） |
| 年間出荷量 | 約136,100トン |
| 顧客 | 塗料メーカー・印刷インキメーカー・工業用途（建設・自動車・エレクトロニクス） |
| 国内シェア | **塗料・インキ溶剤顧客ベースで約30%（業界首位）** |

### なぜ30%シェアを維持できるか

1. **32,000品目の多品種対応力**：競合が追いつけない製品ラインナップ。顧客ごとのカスタム配合が参入障壁
2. **全国販売網**：1952年から70年以上の顧客関係。スイッチングコストが高い
3. **ニッチ支配**：シンナー・溶剤は市場規模が小さく大手化学会社が本格参入しない領域

### 原料構造（最重要）

```
ナフサ（石油精製）
  ↓
石化各社がナフサをクラッキング
  ↓
トルエン・キシレン・酢酸エチル等の有機溶剤を生産
  ↓
大伸化学が調達・調合・販売
  ↓
塗料・インキ・工業用途メーカーへ
```

**→ ナフサ価格変動が2段階遅れで大伸化学の原料費に波及する構造。石化各社の在庫・生産調整期間があるため、ホルムズ封鎖の原料コスト影響は2〜3ヶ月のタイムラグがある。**

## 【不動産含み益考察】

大伸化学は工場用地（全国複数拠点）を保有。BPS¥3,728に対し現株価¥2,150（PBR0.58x）の差分¥1,578は主に純資産（固定資産・土地）の簿価評価に由来する可能性。製造業の土地含み益は有価証券報告書の土地明細表で確認要。IRへの確認推奨。
