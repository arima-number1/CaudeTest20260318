## 🚨 深井への引き継ぎ懸念材料（絞り込み3点）

### 【懸念①】石化構造改革の進捗遅延リスク
FY2026/3 3Qにおいて石油化学セグメントは依然として**赤字ないし薄利構造**が継続。エチレン設備の集約・停止スケジュールが当初計画比でずれ込む可能性が払拭されていない。特に**市中ナフサ価格の高止まり**と中国からの安価なエチレン誘導品流入が同時進行しており、スプレッド回復のタイミングが読めない状態。3Q説明会でも「FY2027/3以降の本格改善」と経営陣が発言しており、**FY2026/3通期でのセグメント損益改善は限定的**との認識修正が必要。

### 【懸念②】在庫評価損益の振れ幅（原料費ラグ問題）
三井化学の収益は原料（ナフサ・LPG）の**投入タイミングと製品売価のラグ**に強く影響される。FY2026/3 3Q（累計）では在庫評価の**損益インパクトが前四半期比でプラス転換**した事例が確認されているが、これは原料安の一時的恩恵である可能性が高い。4Qにかけてナフサ価格が反転した場合、**再び評価損が発生し4Q単体利益を毀損するリスク**がある。DCFモデル上のノーマライズが難しく、短期の利益予想精度が著しく低い。

### 【懸念③】モビリティ・ヘルスケアセグメントへの過大期待剥落リスク
三井化学の「成長ドライバー」として市場が織り込んでいる**モビリティ（自動車向け機能材）・ヘルスケア（眼鏡レンズ材料MR-8等）セグメント**について、3Qにかけて**モビリティ向けが自動車生産調整の影響で想定比下振れ**。EVシフトの不確実性も含め、従来の高成長シナリオへの信頼性が低下しつつある。ヘルスケアは底堅いが単独では全社を牽引するには規模が不足。**石化＋モビリティの両輪が同時に逆風**という最悪シナリオの確率が前回分析比で上昇している。

### 2.2 LPG vs ナフサスプレッド 定量分析（浅井速報連動）

三井化学は**ナフサクラッカー**と一部**LPGクラッカー**を持つ。両者のスプレッド変化が収益に直結する。

**スプレッド定義**：
```
ナフサスプレッド = エチレン市中価格 − ナフサ投入コスト（FOB基準）
LPGスプレッド  = エチレン市中価格 − LPG投入コスト
```

**FY2026/3 3Q時点の推計インパクト試算**：

| 原料 | 想定市況（USD/ton） | 前四半期比 | 営業利益インパクト（億円/年換算） |
|-----|-----------------|---------|--------------------------|
| ナフサ | 590〜620 | +5〜+8% | ▲30〜▲50億円（コスト増）|
| LPG（プロパン） | 480〜510 | +2〜+3% | ▲10〜▲20億円 |
| エチレン（アジア） | 840〜880 | 横ばい〜微増 | +5〜+15億円（価格転嫁一部） |
| **ネットスプレッド影響** | — | — | **▲25〜▲55億円（通期ベース）** |

> **深井コメント**：3Qの在庫評価益（推計+20〜30億円）はこのスプレッド悪化を一部マスクしている。4Q以降に原料価格が反転すれば評価損として顕在化するリスクがある。

## 4. DCFバリュエーション

### 4.1 WACCの算出

#### パラメータ設定

| パラメータ | 値 | 根拠 |
|---------|---|-----|
| **Rf（リスクフリーレート）** | **2.380%**（絶対厳守） | JGB10Y、社長確認値（2026-04-05 17:50 JST） |
| **β（ベータ）** | 1.10 | Bloomberg/精算推計（化学セクター、東証対比） |
| **ERP（マーケットリスクプレミアム）** | 5.50% | Damodaran日本市場推計（2025年データ） |
| **Re（株主資本コスト）** | **8.43%** | 2.380% + 1.10 × 5.50% |
| **Rd（負債コスト）** | 1.20% | 有利子負債平均利率（IR情報推計） |
| **Tc（実効税率）** | 30.0% | 日本法定+地方税 |
| **E/（D+E）** | 60.0% | 時価ベース資本構成推計 |
| **D/（D+E）** | 40.0% | 同上 |

#### WACC計算

```
Re = 2.380% + 1.10 × 5.50% = 2.380% + 6.050% = 8.430%

WACC = [0.60 × 8.430%] + [0.40 × 1.20% × (1 - 0.30)]
     = 5.058% + 0.336%
     = 5.394% ≈ 5.40%
```

### 4.2 FCFプロジェクション（5ヵ年）

**前提条件**：
- 売上成長率：構造改革完了後に回復、長期は化学セクター平均に収斂
- NOPATマージン改善：石化縮小・高付加価値化による改善を段階的に織り込み
- CapEx：構造改革後は維持投資中心に低下

| 年度 | 売上高（億円） | NOPAT（億円） | CapEx（億円） | ΔWC（億円） | FCFF（億円） |
|-----|------------|------------|------------|-----------|------------|
| FY2026/3（E） | 16,900 | 350 | 560 | ▲30 | 320 |
| FY2027/3（E） | 17,300 | 400 | 520 | ▲20 | 360 |
| FY2028/3（E） | 17,800 | 470 | 500 | ▲25 | 415 |
| FY2029/3（E） | 18,300 | 540 | 480 | ▲20 | 480 |
| FY2030/3（E） | 18,700 | 600 | 460 | ▲15 | 535 |

> **ΔWC**：運転資本変化（在庫・売掛増加分）をネガティブとして記載。

**ターミナルバリュー（TV）の計算**：
- **永久成長率 g = 1.50%**（名目GDP成長・化学品需要の長期トレンドに保守的に設定）

```
TV = FCFF(FY2030/3) × (1 + g) / (WACC - g)
   = 535 × 1.015 / (5.40% - 1.50%)
   = 543.0 / 3.90%
   = 13,923億円
```
