## 2. 【社長優先指示】「来期黒字転換できるなら低PBRで買える」仮説の定量検証

> **社長指示:** 「来期黒字転換できるなら低PBRで買える」仮説の真偽を定量的に証明せよ。スタンダード流動性リスクにも言及。

### 2.1 直近赤字の構造要因分析

片倉コープアグリの赤字局面（FY2023〜FY2024）は、単純な事業競争力の喪失ではなく、**原料コストショックによるタイムラグ損失**と**統合後の固定費構造の重さ**が主因と判断する。

```
【赤字の主因分解（推定）】

① ウクライナ危機後の肥料原料価格急騰（2022〜2023年）
   ↓
   原料費（アンモニア・リン酸・カリ）が前年比+30〜50%上昇
   ↓
   販売価格への転嫁が半年〜1年ラグで遅延
   ↓
   粗利率が急激に圧縮（推定：12〜15%→7〜9%に悪化）

② 製造拠点の稼働率低下
   農家の買い控え（高騰期）→ 在庫調整→ 固定費吸収悪化

③ 統合効果の実現遅延
   PMI（統合後経営統合）コスト＋システム統合費用の残存
```

【ドライバー② 販売価格の下方硬直性】

  農家側は価格上昇を「新常態」として受容
  → 値下げ圧力は限定的（JA経由の価格交渉力が緩衝）

【ドライバー③ 固定費削減の進捗】
  統合後の製造拠点集約（一部拠点の閉鎖・集約）

  → 販管費の年間▲2〜4億円削減は射程内

### 2.3 低PBR（0.60倍）の純資産質検証

**PBRの構造分解:**

```
PBR = ROE / (株主資本コスト - 成長率)
    = ROE / (Re - g)

現状:
  現在PBR = 0.60倍
  BPS（1株純資産）= 推定 約1,100〜1,200円
  現在株価 = 約680円
  
  → 市場が「解散価値の60%しか評価していない」
  
PBRが割安かどうかの判定式:
  PBR > 1.0 ← ROE > Re（株主資本コスト）の場合
  PBR < 1.0 ← ROE < Re の場合（価値破壊懸念）
  
現状のROE（FY2024E）: ▲5%〜▲7%（赤字）
Re（後述WACC計算より）: 約7.5〜8.5%
→ PBR < 1.0は「理論的に正当」
```

**純資産の質的評価:**

| 資産項目 | 規模（推定） | 質的評価 | リスクコメント |
|---------|------------|---------|-------------|
| 現預金・短期金融資産 | 約40〜60億円 | ◎ 高品質 | 流動性の核 |
| 売掛金（JA経由） | 約80〜100億円 | ○ 良質 | 貸倒リスク低 |
| 棚卸資産（肥料・原料） | 約60〜80億円 | △ 要注意 | 原料高騰時に減損リスク |
| 有形固定資産（工場・設備） | 約150〜180億円 | △ 要注意 | 老朽化・時価<簿価の可能性 |
| 投資有価証券（持合株等） | 約20〜40億円 | ○ | 時価評価済み |
| のれん・無形資産 | 約20〜30億円 | ▲ 要警戒 | 統合時ののれん。減損リスク残存 |
| **純資産合計** | **約220〜250億円** | **△ 中程度** | |

> **深井判断:** BPS約1,100〜1,200円に対し、**実質的な清算価値は80〜85%水準**（固定資産の時価減損・棚卸在庫調整後）と推定。純資産BPS約**880〜1,020円**が「実質BPS」。現在株価680円はこれに対し約**70〜77%水準**であり、やや割安だが「劇的な割安」とは言えない。

**仮説の真偽判定:**

```
「来期黒字転換できるなら低PBRで買える」

→ 【条件付きTRUE（真）】

理由:
① 黒字転換確率75%は十分に高い
② 黒字転換 → ROE改善（▲5%→+3〜5%）→ PBR適正水準切り上がり
③ PBR0.60倍→0.80〜0.90倍への回帰だけで
   株価: 680円 → 900〜1,080円（+32〜+59%）の上昇余地
④ 黒字転換後の配当復活（DOE政策採用の場合）が
   新たな投資家層を呼び込む可能性

ただし以下の条件が崩れる場合はFALSE:
× 黒字転換しても利益が極薄（EPS+10円以下）の場合
× 純資産に隠れた減損が顕在化する場合
× 原料価格が2022〜2023年水準に再高騰する場合
```

### 2.4 スタンダード流動性リスクの詳細分析

> **社長指示への回答：流動性リスクについて明示的に言及**

**東証スタンダード上場銘柄としての流動性特性:**

```
【片倉コープアグリの流動性プロフィール】

時価総額: 約120〜130億円
→ 「小型株」カテゴリー。機関投資家の本格参入閾値（200億円）未満

日次平均出来高（直近推定）:
  通常時: 約10〜20万株/日（金額ベース: 約7,000〜1.5億円）
  モメンタム局面: 約50〜100万株/日（金額ベース: 約3〜7億円）

スプレッド: 通常時 約0.5〜1.0%（流動性低時は2%超も）
```

**流動性リスクの具体的インパクト:**

| リスク要因 | 内容 | 実務的影響 |
|----------|------|----------|
| **大口エグジットリスク** | 数億円規模の売却で需給が崩れる | 想定売却時に▲5〜15%のスリッページ発生 |
| **信用買い残の増加** | 1M +49.7%モメンタム後の信用膨張 | 需給悪化トリガーになりうる |
| **プライム市場との格差** | 機関投資家のユニバース外 | アナリストカバレッジ希薄→価格発見機能が弱い |
| **流動性プレミアム** | 理論株価に▲10〜20%の流動性ディスカウント | DCF理論値を機械的に適用できない |
| **分割エントリーの必要性** | 一括購入が市場インパクトを生む | 分割・時間分散エントリー必須 |

**流動性リスクへの定量的対応（ポジションサイジング）:**

```
【推奨最大ポジションサイズの目安】

日次出来高（通常時）: 約1〜1.5億円
ポジション構築基準: 日次出来高の10〜20%以内
→ 1日あたり約1,000〜3,000万円以内で分割購入が現実的上限

社長ポートフォリオ全体比: 個別銘柄5%以内を推奨
（流動性リスク考慮の場合は3%以内が保守的）
```

> **深井総括（2.4）:** スタンダード銘柄の流動性リスクは**「実在する重大リスク」**であり、プライム銘柄同様の流動性を期待してはならない。特に1M +49.7%のモメンタム後は信用残の膨張が予想され、**需給反転時の下落速度が急激になりうる**点を経営陣は認識すべきである。ただし、長期保有前提（12〜18ヶ月）かつ適切なポジションサイズに収めるなら、このリスクはコントロール可能と判断する。

## 4. DCFバリュエーション

### 4.1 WACC計算

**【使用パラメータ】**

```
■ リスクフリーレート（Rf）
  Rf = 2.380%
  出所: 日本10年国債（JGB10Y）
  取得: 2026-04-05 19:32 JST（社長確認値）

■ エクイティリスクプレミアム（ERP）
  ERP = 5.50%
  根拠: 日本株市場の長期平均ERP（Damodaran推定値ベース）
        新興国リスクの追加は不要（日本国内企業）
        ただし小型株プレミアム+0.5%を織り込み済み

■ ベータ（β）
  β = 0.85
  根拠: 
  ・肥料セクターは景気感応度が低い（農業は必需性高い）
  ・ただし原料価格変動で業績変動大きく素材株的側面あり
  ・東証スタンダード小型株として若干低β推定
  ・実績βは0.75〜0.95レンジと推定→中央値0.85採用

■ 株主資本コスト（Re）
  Re = Rf + β × ERP
     = 2.380% + 0.85 × 5.50%
     = 2.380% + 4.675%
     = 7.055% ≈ 7.1%
  
  ※小型株プレミアム（Liquidity Premium）: +1.0〜+1.5%
  ※流動性調整後Re = 8.1〜8.6%（保守的シナリオ）

■ 負債コスト（Rd）
  Rd = 1.20%（短期借入中心の日系銀行融資。低金利環境継続）

■ 実効税率（Tc）
  Tc = 30%（法定実効税率 30.62%を概算）
  税引後負債コスト = 1.20% × (1-0.30) = 0.84%

■ 資本構成（D/E比率）
  自己資本（E）= 約230億円 → 比率: 約68%
  有利子負債（D）= 約110億円 → 比率: 約32%
  
  E/(D+E) = 68%
  D/(D+E) = 32%
```

**WACC計算:**

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.68 × 7.055%] + [0.32 × 1.20% × 0.70]
     = 4.797% + 0.269%
     = 5.07%

流動性プレミアム調整後WACC（保守的ベース）:
WACC（adj） = 5.07% + 1.00% = 6.07%

→ 基本シナリオ DCF使用WACC: 6.0%（切り上げ）
→ 保守的シナリオ DCF使用WACC: 7.0%（流動性プレミアム1.5%上乗せ）
```
