## 4. DCFバリュエーション（正値ベース完全再計算）

### 4-1. WACC計算前提

#### ステップ①：資本構成

```
【資本構成推定（修正後）】
自己資本（E）= 200億円
有利子負債（D）= 65億円（中間値）
D+E          = 265億円

E/(D+E) = 200 ÷ 265 = 75.5%
D/(D+E) = 65 ÷ 265  = 24.5%
```

#### ステップ②：株主資本コスト（Re）

```
【パラメータ】
Rf  = 2.380%（日本10年国債 / 2026-04-06社長確認値）
β   = 0.85（鉄鋼特殊鋼メーカー、市場平均より若干低リスク）
     ※ 特殊鋼厚板専業・防衛需要安定・無借金体制→市場β<1.0
ERP = 6.0%（日本株式リスクプレミアム：標準値5.5〜6.5%の中央）

Re  = Rf + β × ERP
    = 2.380% + 0.85 × 6.0%
    = 2.380% + 5.100%
    = 7.480%
```

#### ステップ③：負債コスト（Rd）

```
Rd（税引前）= 1.5%（優良中堅製造業・低金利借入推定）
法人税率（Tc）= 30%（実効税率）
Rd × (1-Tc) = 1.5% × 0.70 = 1.050%
```

#### ステップ④：WACC

```
WACC = E/(D+E) × Re + D/(D+E) × Rd × (1-Tc)
     = 75.5% × 7.480% + 24.5% × 1.050%
     = 5.647% + 0.257%
     = 5.904%
     ≒ 5.90%（ベースケース）
```

### 7-1. メタルスプレッドとは

```
メタルスプレッド = 厚板製品販売価格 - 鉄スクラップ原料コスト
                  （製造マージンの核心指標）
```

中部鋼鈑は電炉メーカーであり、**鉄スクラップを原料とする電炉法**を採用。原料コストの大半が鉄スクラップ価格で決まるため、メタルスプレッドの動向が収益を直接支配する。

### 7-2. 有事局面でのメタルスプレッド動向

**【有事がメタルスプレッドに与える影響（二面性）】**

```
▶ POSITIVE面（需要増）
  ① 防衛・軍需向け装甲鋼板需要急増 → 販売価格に上乗せ可
  ② 造船（軍艦・輸送艦）向け厚板受注急増
  ③ エネルギーインフラ（LNGタンク・ガス配管）設備投資増
  ④ 緊急性の高い調達 → 価格交渉力の向上

▶ NEGATIVE面（コスト増）
  ① 鉄スクラップ奪い合い（米・欧・日）→ 原料コスト上昇
  ② エネルギーコスト（電力・LNG）高騰 → 電炉操業コスト増
  ③ 物流コスト増加
```
