## 1. 企業概要・ビジネスモデル

**株式会社淺沼組**（1852.T）は、創業1892年・設立1937年の関西系総合建設業（ゼネコン）。
本社：大阪市浪速区。東証プライム上場。

### 事業の特徴
- **建築主体**: 売上の大半は建築工事（商業施設・オフィス・病院・学校）

## 2. ショート仮説の検証【最重要：結論=NOT SHORT】

### ユーザー仮説
「1850はネットキャッシュ水準なのに割安、1852は高いのでショートしてペアを作る」

### 検証結果：**仮説は成立しない**

#### ❌ ショート根拠①「高バリュエーション」→ 否定

- PBR 1.63xは「正当化PBR = ROE/Ke = 10.05%/5.5% = **1.83x**」を**下回る**
- つまり現値¥966は理論値より**11%安い**水準
- ショートするには正当化PBRを「超えている」必要がある → 条件を満たさない

#### ❌ ショート根拠②「資材不足で業績が壊滅する」→ 否定

- 資材危機（TOTO・LIXIL供給制約）の直撃先は**住宅設備メーカー**と**消費者向け住宅施工会社**
- 淺沼組のような大型建築ゼネコンへの影響は「工事完成の**遅延**」のみ
- 受注は**消えない**。繰越工事高¥2,440億は工期が延びても受注残として残る
- 工事進行基準（%完成度認識）により、利益認識が**後ずれ**するがゼロにはならない

#### ❌ ショート根拠③「1850と同じビジネスだから同じ水準になるべき」→ 否定

- 1850（南海辰村建設）: **住宅系・小型案件中心・大阪南部地盤**
- 1852（淺沼組）: **大型商業・公共建築・大阪IR Block B JV**
- 同じ「建設業」でも事業特性が大きく異なる → バリュエーション差は正当化される

## 6. DCF・理論株価

### Method 1：ROE/Ke法（Justified PBR）

```
Justified PBR = ROE / Ke = 10.05% / 5.5% = 1.83x
FV = BPS(¥593) × Justified PBR(1.83) = ¥1,085
```

### Method 2：Gordon成長モデル（配当割引）

```
配当: ¥43.5 / 株価¥966 = 4.50%
ROE: 10.05%、Ke: 5.5%、内部留保率: 43.5/77 = 56.5% → 配当性向43.5%
g（サステナブル成長率） = ROE × 留保率 = 10.05% × 0.565 = 5.68%
注: g > Ke(5.5%) の場合はゼロ成長仮定に切替
FV = D/(Ke-g) は発散するため → PERモデルに切替
```

### Method 3：PER目標倍率法

```
FY2026 EPS予想: ¥83（期初比+7.4%）
目標PER: 建設業平均12x〜14x（ROE10%・増配・IR受注プレミアム）
  Bear case (10x): ¥830  ← 資材危機深刻化・利益率圧縮シナリオ
  Base case (13x): ¥1,079 ← 現状継続
  Bull case (15x): ¥1,245 ← 大阪IR本格稼働・受注絶好調
```

### FV総合

| 手法 | FV | 現値比 |
|---|---|---|
| ROE/Ke法（PBR） | **¥1,085** | **+12.3%** |
| PER Base case（13x） | **¥1,079** | **+11.7%** |
| PER Bull case（15x） | **¥1,245** | **+28.9%** |
| PER Bear case（10x） | **¥830** | **▲14.1%** |
| **FV 中央値** | **¥1,082〜¥1,090** | **+11〜13%** |

**現値¥966はFair Value比11〜13%割安水準**

## 8. リスクシナリオ定量化

### ① 大阪IR遅延・費用超過シナリオ（確率15%）

```
影響: Block B工事費超過→JV損失分担→淺沼組持分損失¥20〜50億
EPS 影響: ▲¥25〜63（-32〜-82%）
株価影響試算: PER10x × ¥50EPS = ¥500（現値比▲48%）
```
→ テール・リスクとして実在するが、大阪府市・国土交通省が支援する案件で急性破綻は考えにくい

### ② 資材危機深刻化・完成遅延シナリオ（確率20%）

```
影響: 工期6〜12ヶ月遅延 → 売上計上が後ずれ（FY2026 売上10〜15%減）
EPS 影響: ¥77 → ¥50〜60（▲22〜35%）
株価影響試算: PER10x × ¥55 = ¥550（現値比▲43%）
```
→ 遅延は「消滅」ではなく「先送り」。翌期以降に利益計上 = 一時的なEPSへの影響

### ③ 建設需要減退・受注急減シナリオ（確率10%）

```
影響: 受注高¥1,975億 → ¥1,500億に急減
繰越工事高消化で2〜3年の猶予はある（¥2,440億÷¥1,500億/年=1.6年）
EPS影響: 遅延発現はFY2028以降
```

### ④ ベースシナリオ（確率55%）

```
受注残¥2,440億をFY2026〜FY2028に順次消化
大阪IR Block B工事継続
EPS ¥83 (FY2026) → ¥90 (FY2027) の成長軌道
FV: ¥1,100〜¥1,200
```

### 加重FV（確率加重）

```
¥1,150 × 55% + ¥550 × 20% + ¥500 × 15% + ¥600 × 10%
= ¥632.5 + ¥110 + ¥75 + ¥60 = ¥877.5
```

> 加重FVでも¥877 > 現値¥966 との乖離は小さい。
> 極端なテールリスクを加味しても「割高ではない」。

## 10. 推奨アクション

### ショートとして → ❌ 見送り

**理由（3点）**:
1. 正当化PBR 1.83x に対して現値は1.63x = **11%割安**（ショートは割高でないと成立しない）
2. 大阪IR Block B JV確定受注 = バックログ2030年まで保護
3. RSI35.3 = 売られすぎ圏でのショートは不利なリスク/リワード

### ロングとして → ✅ 逆張り検討

```
エントリー目標: ¥940〜¥980（RSI33以下への押しをつかむ）
利益確定目標: ¥1,080〜¥1,150（正当化FV水準）
ロスカット: ¥900（52週安値¥705の上方サポート破り）
期待リターン: +13〜19%
リスク/リワード: 1:2.0〜2.5 ✅
```

### 1850/1852 ペアトレード → ❌ 1852ショートサイドは不成立

```
両社とも大阪IR Block B JVメンバー
1852のプレミアム（PBR差）は受注残・規模・IR受注量で正当化される
```
