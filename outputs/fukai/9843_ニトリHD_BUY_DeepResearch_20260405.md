## 2. 【社長指示最優先】円安リスクと構造的下落トレンドの継続性評価

> **社長指示:** 「浅井suggestion +75逆張り。円安リスクと構造的下落トレンドの継続性を冷静に判断」

### 2-1. 円安リスクの定量的インパクト分析

ニトリの**為替エクスポージャーは業界最高水準**である。これはSPAモデルの宿命だ。

#### 調達通貨構造（推計）

```
海外調達比率：約85〜90%（推計）
  ├── USD建て決済：約40〜45%
  ├── 人民元（CNY）建て：約35〜40%
  └── その他（VND・THB等）：約10〜15%

ヘッジ方針：一部為替予約（通常6〜12ヶ月先物）
          ただし全量ヘッジは行わない方針
```

#### 為替感応度（1円の円安が営業利益に与える影響）

| USD/JPY変動 | 売上原価への影響（推計） | 営業利益への影響（推計） |
|-------------|------------------------|------------------------|
| 1円の円安 | +約30〜40億円のコスト増 | ▲約20〜30億円 |
| 10円の円安 | +約300〜400億円のコスト増 | ▲約200〜300億円 |
| 現在（155円仮定）vs 2022年前（110円） | 約45円差異 | 累積▲約900〜1,350億円相当 |

> **深井コメント:** ニトリ自身が過去の決算説明資料で「1円の円安で約30億円の営業利益影響」と開示している。これを基準とすると、USD/JPYが140円→155円に動いた場合だけで**約450億円**の利益が吹き飛ぶ計算になる。これは営業利益の20〜25%に相当する重大なインパクトだ。

#### 円安の現状認識（2026年4月時点）

| シナリオ | USD/JPY水準 | 確率（深井推計） |
|---------|------------|----------------|
| 円高回帰（日銀利上げ加速） | 130〜140円 | 20% |
| 現状維持（レンジ） | 145〜155円 | 45% |
| 円安継続・深化 | 155〜165円超 | 35% |

**深井の見立て：** 日米金利差は依然として大きく、米国がインフレを完全制圧するまでドル高圧力は続く。日銀の追加利上げが円高要因となるが、そのペースは緩慢。**円安は「一時的リスク」ではなく「構造的コスト負担」として少なくとも1〜2年は継続する可能性が高い。**

## 4. DCFバリュエーション

### 4-1. WACC計算の前提

#### ① リスクフリーレート（Rf）
```
Rf = 2.380%（絶対固定値）
出典：JGB10Y、社長確認値、2026-04-05 17:45 JST
```

#### ② エクイティリスクプレミアム（ERP）
```
ERP = 5.50%
根拠：日本株市場の長期実績・Damodaran教授推計（日本2025〜2026年推計値）
     東証プライム全体の過去リターン実績との整合性
```

#### ③ ベータ（β）
```
β = 0.75（推計）
根拠：
 ・家具小売業は景気敏感だが必需品性もあり中程度
 ・ニトリHDの過去3年52週ベータ：約0.65〜0.85レンジ
 ・中央値として0.75を採用
 ・円安リスク加算要素を既に業績期待に織り込む前提
```

#### ④ 株主資本コスト（Re）
```
Re = Rf + β × ERP
Re = 2.380% + 0.75 × 5.50%
Re = 2.380% + 4.125%
Re = 6.505%
```

#### ⑤ 負債コスト・税率・資本構成

| パラメータ | 値 | 根拠 |
|-----------|-----|------|
| Rd（負債コスト） | 1.20% | 優良企業・低金利調達 |
| Tc（実効税率） | 30.0% | 日本法人税実効税率 |
| Rd × (1-Tc) | 0.840% | 税引後負債コスト |
| E/（D+E） | 85% | BS自己資本比率ベース |
| D/（D+E） | 15% | 同上 |

#### ⑥ WACC計算

```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
WACC = [0.85 × 6.505%] + [0.15 × 0.840%]
WACC = 5.529% + 0.126%
WACC = 5.655%
```

**→ WACC = 5.655%（ベースケース）**
