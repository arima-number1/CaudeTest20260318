### 2-1. 伊藤忠商事の戦略意図の特定【最重要①】

#### A. 21%は通過点か最終地点か——深井の判断：**「通過点」蓋然性75%**

```
【時系列と数量から読む伊藤忠の意図】

4/09：9.58%新規取得　→ 「私募REIT再編・不動産M&A戦略強化」の一環
4/16：21.18%まで一気拡大（+11.60pt）
　　　⬆ わずか7日で11.60ptの追加取得
　　　⬆ 21%は「持分法適用（20%超）」を超えた「強い持分法適用」水準

【なぜ21%で止まらないと考えるか】

根拠①：伊藤忠の不動産戦略との整合性
　→ 伊藤忠都市開発は「都市型複合開発」「ホテル・物流・オフィス複合」戦略を推進
　→ サンフロンティアの「オフィスバリューアド×ホテル×民泊」は伊藤忠の戦略と
　　 100%シナジーがある
　→ 住友不動産・三菱地所との競争において「独立系バリューアド特化プラットフォーム」
　　 を傘下に収めることは伊藤忠にとって差異化要因

根拠②：取得スピードの異常な速さ
　→ 7日間で11.60ptは「財務投資」ではなく「戦略取得」のスピード感
　→ 一般的な財務投資（持分法利益狙い）なら5〜10%で止まる
　→ 20%を超えて一気に21%まで取得した理由は「30%超えを視野に入れた段階的取得」
　　 の布石以外に合理的説明が困難

根拠③：JGB2.425%私募REIT再編文脈
　→ 金利上昇で私募REITの解約リスクが高まる局面に伊藤忠が不動産を直接抑えに来た
　→ 私募REIT崩壊シナリオが来たとき「直接保有オフィスビル」網を持つプラットフォームは
　　 最大の防衛資産になる
```

#### B. AAGS撤退と伊藤忠取得の関係——**売却疑惑：蓋然性85%**

| 指標 | 数値 | 解釈 |
|------|------|------|
| AAGS変化量 | -5.48pt（11.66%→6.18%） | 推定¥55〜65億相当の売却 |
| 伊藤忠変化量 | +11.60pt（9.58%→21.18%） | 推定¥125〜140億相当の取得 |
| 差分 | +6.12pt（伊藤忠の追加取得が上回る） | 残り6pt超は市場取得または第三者（AAGS以外）から |

**深井結論：** AAGS売却分の約半分（5pt前後）は伊藤忠へのブロックトレード（相対取引）で消化されたと推定。残り6〜7pt分は市場での取得（出来高と株価推移からも確認可能）。AAGSが撤退した理由は「伊藤忠という戦略的買い手」が現れた局面で「十分なリターンを実現して出口」という判断。ヘッジファンド的行動として合理的。


## 4. DCFバリュエーション

### 4-1. WACC計算

```
【前提パラメーター】

リスクフリーレート（Rf）：2.414%（JGB10Y・2026/04/15財務省）← 絶対使用
ERP（エクイティリスクプレミアム）：5.5%（日本市場・Damodaran推定）
β（ベータ）：0.85（不動産業・バリューアド特化・推定）
　→ 市場βではなく、業種β調整値。サンフロの売却益依存は市場感応度やや低め

株主資本コスト（Re）：
　Re = 2.414% + 0.85 × 5.5% = 2.414% + 4.675% = 7.089%

負債コスト（Rd）：1.8〜2.0%（推定：変動金利主体・現在の短期プライムレート+スプレッド）
実効税率（Tc）：30%（日本法人税等実効税率）
税引後負債コスト（Rd×(1-Tc)）：1.9% × 0.70 = 1.33%

【資本構成（D/E比率）】
　総資本：純資産¥242億 + 有利子負債¥480億 = ¥722億（推定）
　E/（D+E）= 242/722 = 33.5%
　D/（D+E）= 480/722 = 66.5%

【WACC計算】
　WACC = 0.335 × 7.089% + 0.665 × 1.33%
　　　　= 2.375% + 0.884%
　　　　= 3.26%（ベースケース）
```
