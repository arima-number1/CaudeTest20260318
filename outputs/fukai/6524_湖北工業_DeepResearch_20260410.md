## 1. 光部品ビジネスモデルと業界ポジション（核心）

### 何を作っているのか

湖北工業は光ファイバー通信の**心臓部品**メーカー。主要 4 セグメント：

1. **アルミ電解コンデンサ用リード端子** — 創業事業（FY2025 では収益ウエイト低下傾向）
2. **光通信部品・デバイス** — 成長エンジン。主要製品：
   - インライン光アイソレータ（光の逆流防止）
   - 光フィルタデバイス・光サーキュレータ
   - 可変光減衰器（VOA）
   - 偏波保持光ファイバ組立品（キャピラリ付き）
   - 多チャンネル光ファイバアレイ（特殊組立）
   - **海底ケーブル用光部品（submarine cable 専用ライン）**
3. **高純度石英ガラス（SSG®）** — 光ファイバ母材・半導体装置向け。NEC の多芯化ファイバに直結
4. **宇宙・フォトニクス電子融合デバイス** — 将来成長種

### バリューチェーン上の位置

```
光ファイバ母材(信越化学) → 線引き(住友電工/藤倉) → 【部品組立: 6524】 → ケーブル製造(NEC/古河) → 海底敷設(Google/MS/Meta)
```

6524 は**中流域の高付加価値コネクタ・部品組立**に特化。精密組立・高純度加工というニッチで優位性を維持。

### なぜ「今」なのか ― 三つの需要交点が 2025-2028 に重なる

| 需要源 | 規模感 | 6524 へのリンク |
|--------|---------|-----------------|
| 海底ケーブル超サイクル | Google/MS/Meta が各々$50-100 億規模投資。2030 年前に 1Pbit/s 容量が必要（SBI セミナー 2026-04-10） | submarine cable 用光部品（アイソレータ・光サーキュレータ）需要直撃 |
| AI データセンター光インターコネクト | 超高密度サーバ間通信が電気→光に移行加速 | 多チャンネル光ファイバアレイ・VOA 需要急増 |
| NEC マルチコア・空孔コア光ファイバ実用化 | 2030 年前稼働予定（MS が空孔コア既に実用化） | 高純度石英 SSG® が次世代ファイバ向け材料として浮上 |

### 競合ポジション

- 中国メーカーが低価格で光アイソレータに参入しているが、**海底ケーブル向け高信頼部品（過酷環境耐久性・MIL 規格等）** では日系精密メーカーが優位を維持
- 高純度石英 SSG® は独自ブランド、代替困難
- 3:1 株式分割（2024-03-28）後に流動性改善、機関投資家参入障壁が低下

## 3. バリュエーション分析

### 現在指標と同業比較

| 指標 | **6524 湖北工業** | 5803 フジクラ | 5801 古河電工 |
|------|------|------|------|
| Trailing PE | 41.0x | **71.8x** | **91.6x** |
| PBR | 5.20x | 18.4x | 8.5x |
| ROE | 12.8% | 31.1% | 14.9% |
| 売上成長 | +9.6% | +0.0% | +8.5% |
| OPM | **26.5%** | — | — |

→ ROE では劣後するが、PE は同業比 **43〜55% 割安**。光部品比率上昇に伴う ROE 改善で収斂余地大。

### WACC・理論 PBR

```
JGB10Y:  2.425%
Beta:    1.028（yfinance）
ERP:     5.5%
WACC = 2.425% + 1.028 × 5.5% = 8.08%

理論 PBR（g=5%, ROE=12.8%）
  = (12.8% - 5.0%) / (8.08% - 5.0%) = 2.52x

→ 現在 PBR 5.20x > 理論値 2.52x（プレミアム状態）
  ＝ 市場は ROE の持続的改善（g=7%想定なら理論値5.37x）を織り込み済み

g=7% 理論 PBR: (12.8%-7%) / (8.08%-7%) = 5.37x → 現 PBR と整合
  ＝ 「光部品成長率 7%/年 持続」を現価格で仮定している水準
```
