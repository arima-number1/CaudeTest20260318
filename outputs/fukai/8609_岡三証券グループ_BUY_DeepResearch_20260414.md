| **β（ベータ）** | **1.15** | 証券・金融セクター中堅（市場感応度高め。ネット開示値不在のため業界標準採用） |

| **Tc（実効税率）** | **30%** | 日本法人税標準 |

**ターミナルバリュー（TV）**

```
TV = FCF2030 × (1+g) / (WACC - g)
   = ¥34.4億 × 1.015 / (0.0410 - 0.015)
   = ¥34.9億 / 0.026
   = ¥1,342億

PV(TV) = ¥1,342億 × 0.8180 = ¥1,098億
```

**企業価値（EV）**

```
EV = PV(FCFs) + PV(TV)
   = ¥131億 + ¥1,098億
   = ¥1,229億
```

**株式価値・理論株価**

```
純有利子負債（推定）: ¥1,700億 - ¥400億（現預金推定）= ¥1,300億
株式価値 = EV - 純有利子負債

```

> **⚠️ 重要注記**: 上記DCFにより純算で負の株式価値が出るのは、**証券会社特有の高レバレッジ構造**（レポ取引・信用取引残高が「有利子負債」に計上される）に起因する。この場合、DCF法は本質的な企業価値を正確に反映しない。**証券会社のバリュエーションには PBRベース（残余利益モデル）の採用が業界標準**。

### 4-4. PBRベース理論株価（主軸バリュエーション）

証券会社のバリュエーションはPBR＋ROEを組み合わせた**残余利益モデル（Residual Income Model）**が適切。

**理論PBR算出式**:
```
理論PBR = (ROE - g) / (Re - g)
         = (6.5% - 1.5%) / (9.37% - 1.5%)
         = 5.0% / 7.87%
         = 0.635倍
```

> この「理論PBR0.635倍」はROE6.5%という現状の収益力に基づく理論上の"正当化できる"最低水準であり、**改善期待を込めたフォワードPBRとは区別が必要**。

| 弱気（ROE低下） | 5.5% | 0.51倍 | **¥559** | ▲38% |
| 保守的（ROE維持） | 6.5% | 0.64倍 | **¥702** | ▲22% |
