# 2. 社長指示の重点分析（最優先）

## 2.1 低PBRの原因解剖：「構造的衰退ディスカウント」か「本物のバリュー」か

### 現状の数値確認

```
現在株価      : ¥385
BPS（簿価）   : ¥538
PBR           : 0.72倍（=¥385/¥538）
ROE           : 1.5%
EPS           : ¥9
PER           : 45.2倍
```

**PBR < 1.0倍**は「市場が純資産価値すら認めていない」状態。ここで重要な分解フレームワーク：

```
PBR = PER × ROE
0.72 = 45.2 × 1.5%  → 0.678 ≒ 0.72（ほぼ一致）
```

この式が示す本質：

> **ROE 1.5%という資本収益性の極端な低さが、PBR割れの真因である。**

PER 45倍は「成長期待がある」と読めるが、EPS ¥9・ROE 1.5%という実態は**「成長への期待とは裏腹に、既存資産からほとんど稼げていない」**状況を示す。

### ディスカウント要因の分解

#### 要因①：ゲーム事業の構造的衰退（ウェイト：大）
- フィーチャーフォン時代に築いた収益基盤が、スマホ化・グローバル競争の激化で長期縮小
- 「ゲームはいつ終わるかわからない」という**サンセットリスク**が純資産に対する恒常的ディスカウントを生む
- 市場は「BPSの一部はいずれ毀損する可能性がある」と見ており、NAVを額面どおり評価しない

#### 要因②：資本効率の慢性的な低さ（ウェイト：大）
- ROE 1.5%は日本株全体の平均（約9〜10%）を大幅に下回る
- 大量の内部留保・投資有価証券を抱えながら、それを十分に活用できていない
- **「資産はあるのに稼げない企業」**というレッテルがPBR割れを固定化

#### 要因③：成長事業（REALITY）の不透明性（ウェイト：中）
- メタバース・VTuber配信は有望だが、現時点では赤字投資フェーズ
- 市場は「いつ黒字化するか」の見通しが立てられず、成長プレミアムを付けにくい

#### 要因④：資本政策の消極性（ウェイト：中）
- 自社株買いや増配が積極的でなければ、内部留保は「死に金」とみなされる
- ROE改善へのコミットメントが弱いと市場は判断

### 判定：「本物のバリュー」か「衰退ディスカウント」か

```
┌────────────────────────────────────────────────┐
│  暫定判定：「衰退ディスカウント要因が主因」    │
│  ただし純資産の質次第では部分的な割安あり      │
└────────────────────────────────────────────────┘
```

**完全な割安とは言い切れない理由**：ROE 1.5%が継続する限り、PBR 0.72倍は「正当なディスカウント」の側面が強い。ただし、後述するNAV分析で**投資有価証券・不動産の含み益が相当額存在**するなら、その部分は真の割安となり得る。

### 内部留保の質

グリーは過去の投資先（ゲーム系・テック系スタートアップ）のうち、上場を果たした企業の株式を保有している可能性がある。これらの**含み益（時価－簿価）**はBSには反映されていない場合があり、**「隠れた資産価値」**として機能し得る。

# 4. DCFバリュエーション

## 4.1 前提条件

### 資本コストの計算

```
【WACC計算】

① リスクフリーレート（Rf）  = 2.386%（JGB10Y / 2026-04-07）
② エクイティリスクプレミアム（ERP） = 5.5%（日本市場標準）
③ ベータ（β）              = 0.85（推計：ゲーム×IT系 / 市場感応度やや低い）
④ 株主資本コスト（Re）      = Rf + β × ERP
                             = 2.386% + 0.85 × 5.5%
                             = 2.386% + 4.675%
                             = 7.061% ≒ 7.1%

⑤ 有利子負債コスト（Rd）    = 1.0%（ほぼ無借金のため低位設定）
```
