## ■ 1. 会社概要・ビジネスモデル

### カチタスとは何者か

**中古住宅買取再販の圧倒的No.1**。年間5,597戸販売（売上¥876億）で2位の17倍。事業モデルは極めてシンプルかつ参入障壁が高い：

```
① 地方・築古・空き家を低価格で仕入れ（約80%が空き家）
      ↓
② 現代の生活動線に合わせたリフォーム実施（粗利率26.1%確保）
      ↓
③ 新築の半額程度で販売 → 住宅コスト問題を解決
```

### 構造的競争優位（モート）

| 要素 | 内容 |
|------|------|
| **スケール優位** | 全国ネットワーク × 大量仕入れ → 査定精度・コスト低下 |
| **独自データベース** | 築古物件の修繕コスト・売却実績の蓄積（AIに直結） |
| **社会的使命** | 空き家問題解決 ≒ 国土交通省・自治体との協力関係 |
| **参入障壁** | 規模の経済（仕入れ×リフォーム×販売の三位一体）を2位以下が複製不可 |

## ■ 10. NAV・含み益分析（不動産含み益ルール適用）

**カチタスはフロービジネス（buy-remodel-sell）のため、伝統的な不動産含み益は限定的。**

| 項目 | 評価 |
|------|------|
| 土地・建物含み益 | 微小（本社・事業用資産のみ。投資不動産保有なし） |
| 棚卸資産（未販売物件） | 時価 ≈ 帳簿価格（短期回転のため乖離小さい） |
| **真の含み益 = フランチャイズバリュー** | **業界独占的地位の無形資産（競合は17分の1以下）→ PBR4.9xに反映済み** |

**結論**: カチタスの「含み益」は不動産ではなく、**競争的優位性（無形資産）と成長オプション**。PBR 4.90xは過剰ではなく、ROE25% × 成長20%を反映した合理的プレミアム。
