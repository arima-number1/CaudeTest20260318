## 3. 【テーマ深掘り】AI関連で期待できるか — IT14シリーズ分析

### 3-1. 製品概要: IT14シリーズ（AI関連の核心）

```
IT14シリーズ = ヒロセ電機のAI関連戦略の筆頭製品

伝送速度: 112Gbps PAM4（業界最高水準帯域）
用途:    AIサーバーラック内GPU間・CPU間接続
採用実績: AMD GPU搭載サーバーへの採用確認済み（2026Q3）
競合製品: TE Connectivity SFP56, Amphenol ImSi
ヒロセ優位: 精密小型化（省スペース・高密度実装）×カスタム設計力
```

**なぜIT14が重要か**:
AIデータセンターは「高速・高密度・低遅延」の三原則で設計される。112Gbps PAM4はNVIDIA H100/AMD MI300x世代のGPU間接続標準規格であり、**1ラックに数百本のコネクタが使用される**。GPUが高価値であればあるほど、コネクタの品質要求は上がり、**プレミアム製品市場を形成**する。

### 3-2. 市場規模と成長性

| 項目 | 数値 | 出典 |
|---|---|---|
| AIデータセンター市場 CAGR | **25.8%** | 業界推計 |
| グローバルコネクタ市場 | 約$950億（2025） | 業界レポート |
| AI向けコネクタ市場 | 約$80億（2025） → $200億（2028E） | 業界推計 |
| ヒロセ AI関連売上（推定） | ¥80〜120億/年 | 深井試算 |

### 3-3. 競合比較

| 社名 | 強み | 弱み |
|---|---|---|
| TE Connectivity（米） | 量産・規模 | カスタム対応遅い |
| Amphenol（米） | 品種豊富 | 精密小型苦手 |
| Molex（米） | 低コスト | 高速伝送後発 |

| JAE（日本） | 国内実績 | 利益率低い（13.5%） |

**結論**: ヒロセはAI向け高付加価値ニッチ（AIサーバーラック内精密接続）で先行優位を持つ。NVIDIA/AMD/IntelのGPU採用コネクタとして実績が積み上がれば、**「AI関連コネクタブランド」としての評価プレミアム**が形成されうる。

### 3-4. AI関連売上の寄与タイミング

```
2026.3期（今期）: IT14シリーズ量産初期。採用決定フェーズ。
                 → 利益貢献は軽微（先行投資フェーズ）
2027.3期（来期）: AMD/NVIDIA向け本格量産開始。売上貢献顕在化。
                 → EPS¥1,050想定の根拠（現在¥912から+15.1%成長）
2028.3期以降:    AIデータセンター累積採用拡大。¥200億超の AI売上も視野。
                 → EPS¥1,200超シナリオへ
```

**「期待できるか」の最終答え: YES — ただしタイミングは来期（2027.3期）が本番**。今期は先行投資で利益が重く、来期から売上・利益の双方で成果が可視化される見込み。

## 5. 株主還元分析（DOEギャップが最大の触媒）

### 5-1. 配当政策（DOEターゲット）

```
現在の状況:
  BPS         = ¥11,433
  配当/株      = ¥490
  現在のDOE   = ¥490 ÷ ¥11,433 = 4.28%
  
  会社ターゲットDOE = 5.0%
  必要配当額        = ¥11,433 × 5.0% = ¥572/株
  増配必要額        = ¥572 - ¥490 = ¥82/株（+16.7%）
```

**DOEギャップ解消シナリオ（5/11決算）**:
来期ガイダンスで「DOE5.0%達成宣言＋配当¥572（+¥82増配）」が出た場合、配当利回りは2.30% → **2.69%**に上昇。高配当株として機関投資家の新規買いが入りやすくなる。

### ネガティブ変化
