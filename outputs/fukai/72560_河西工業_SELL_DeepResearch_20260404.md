## 1. 企業概要・事業分析

### 1.1 会社概要

河西工業（Kasai Kogyo Co., Ltd.）は、**自動車内装部品の専業メーカー**として東証プライムに上場する中堅部品サプライヤーである。主力製品はドアトリム・天井内張り（ヘッドライナー）・トランクトリムなど内装の「ソフト系」部品群であり、外観と乗り心地に直結する高品質要求部品を供給する。

- **設立：** 1938年
- **本社：** 神奈川県厚木市
- **連結従業員数：** 約12,000名（グローバル）
- **主要顧客：** 日産自動車（売上の60%超）、ホンダ、スバルほか

### 1.2 セグメント・プロダクト構成

```
河西工業 連結事業構成（推定）
├─ 内装部品（国内）         約45%
│   ├─ ドアトリム
│   ├─ ヘッドライナー
│   └─ トランクトリム・その他
├─ 内装部品（北米）         約30%
│   └─ 米国・メキシコ拠点
├─ 内装部品（アジア）       約20%
│   └─ 中国・タイ・インドネシア
└─ その他・エンジニアリング  約5%
```

**重要構造的特徴：**
- **日産依存度が極めて高い（推定60%以上）**：日産の生産変動がそのまま業績に直撃するビジネスモデル
- **受注生産型・高固定費モデル**：量産前の金型投資・設備投資を先行、稼働率が利益率を直接左右する

## 4. DCFバリュエーション

### 4.1 前提条件（全明記）

#### マクロ前提

| パラメータ | 数値 | 根拠 |
|-----------|------|------|
| **リスクフリーレート（Rf）** | **1.5%** | JGB10年債利回り（2026年4月時点） |
| **エクイティリスクプレミアム（ERP）** | **6.0%** | 日本株長期平均ERP（Damodaran推計ベース） |
| **β（ベータ）** | **1.35** | 自動車部品中堅・高財務レバレッジ・日産集中を反映。業界β約1.1にサイバー攻撃プレミアム0.25を加算 |
| **株主資本コスト（Ke）** | **9.6%** | Rf + β × ERP = 1.5% + 1.35 × 6.0% |
| **負債コスト（税引後）** | **2.1%** | 有利子負債の推定平均金利2.8% × (1-実効税率25%) |
| **資本構成（E/V）** | **42%** | 時価総額÷（時価総額＋純有利子負債） |
| **資本構成（D/V）** | **58%** | 純有利子負債÷（同上） |
| **WACC** | **5.3%** | Ke×E/V + Kd×D/V = 9.6%×42% + 2.1%×58% |
| **永続成長率（g）** | **0.5%** | 日本自動車内装部品の長期成長。業界縮小懸念から保守的設定 |

```
WACC計算詳細:
Ke = 1.5% + 1.35 × 6.0% = 1.5% + 8.1% = 9.6%
Kd(税引後) = 2.8% × (1 - 0.25) = 2.1%
WACC = 9.6% × 0.42 + 2.1% × 0.58
     = 4.03% + 1.22%
     = 5.25% ≈ 5.3%
```
