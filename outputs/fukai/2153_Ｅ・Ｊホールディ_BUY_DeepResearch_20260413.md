## 4. DCFバリュエーション

### 4-1. WACC計算前提

#### Step 1: リスクフリーレート（Rf）
```
Rf = 2.439%（日本10年国債・財務省 2026/4/10確認）
```

#### Step 2: マーケットリスクプレミアム（ERP）
```
ERP = 5.5%
根拠：
  - Damodaran日本株ERP推計（2026年1月更新）≈ 5.0-5.5%
  - 日本株式市場のリスクプレミアム標準値 5.5%を採用
  - 地政学リスク・インフレ不確実性を考慮し標準値を維持
```

#### Step 3: β（ベータ）
```
β = 0.75
根拠：
  - 建設コンサルティング業種のβ標準値 0.65-0.85
  - 公共案件依存度70%以上 = 景気感応度低い（ディフェンシブ性）
  - TOPIX相対▲5.2%(3M)は一時的アンダーパフォーム（β調整後）
  - 中小型株プレミアムを一部考慮して0.75採用
```

#### Step 4: 株主資本コスト（Re）
```
Re = Rf + β × ERP
   = 2.439% + 0.75 × 5.5%
   = 2.439% + 4.125%
   = 6.564% ≈ 6.56%
```

#### Step 5: 負債コスト（Rd）と資本構成
