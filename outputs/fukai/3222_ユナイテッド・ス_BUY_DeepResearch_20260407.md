#### 1-B. 【核心問い①】売上高▲10.1%なのに営業利益がコンセンサスを+29%上回った理由

**結論: コスト削減＋販管費圧縮が主因。売上減は「構造的縮小」ではなく「事業再編・不採算店舗整理」が大きい。**

```
【売上高▲10.1%（▲¥27,975M）の要因分解（推計）】

① 不採算店舗・非中核事業の整理・撤退   ▲¥15,000M〜▲¥18,000M
   → マルエツ/カスミの老朽店閉鎖加速
   → 一部テナント撤退・スクラップ&ビルド
   
② 既存店売上の軟調（食品インフレ後の需要抑制）  ▲¥8,000M〜▲¥10,000M
   → 消費者の節約志向・業務スーパー/ドンキへのシフト
   → 客数減少（来店頻度低下）

③ EC/デリバリー等の伸び（部分的相殺）   +¥2,000M前後
```

**なぜ売上が減ったのに営業利益がコンセンサスを大幅上回ったか：**

| 要因 | 定量インパクト（推計） | 性質 |
|------|-------------------|----|
| **不採算店舗閉鎖による固定費削減** | +¥600M〜+¥900M | 構造的（一部継続） |
| **人件費・物流費の見直し（イオン共同化前倒し）** | +¥300M〜+¥500M | 構造的 |
| **仕入れ条件改善（イオンPB比率向上）** | +¥200M〜+¥300M | 構造的 |
| **エネルギーコスト抑制（ヘッジ効果等）** | +¥100M〜+¥200M | 一時的 |
| **販管費の一時的抑制（広告費・投資抑制）** | +¥200M〜+¥400M | 一時的 |
| **合計コンセンサス超過分** | **+¥1,050M（=¥4,666M−¥3,616M）** | 混在 |

> **深井判断**: 超過分の**約60%が構造的コスト改善**（店舗スリム化・イオングループ共同化）。残り40%は一時的抑制（広告費・資本的支出の繰延）と判断する。来期の収益改善基盤として評価可能だが、売上回復なき利益改善の持続には限界がある。

#### 1-C. 【核心問い②】純利益が赤転した原因と一過性/継続性判断

**純利益▲¥1,317M（赤転）の構造：**

```
営業利益         ¥4,666M
経常利益         ¥4,712M  ← 営業外収益（受取配当等）が若干寄与
特別損失（推計）  ▲¥8,000M〜▲¥9,000M
  └ 内訳（推計）:
    ① 減損損失（老朽店舗・のれん等）   ▲¥5,000M〜▲¥6,000M
    ② 閉店関連費用・撤退費用           ▲¥1,500M〜▲¥2,000M
    ③ 構造改革費用（早期退職優遇等）   ▲¥500M〜¥1,000M
    ④ その他特別損失                   ▲¥500M前後
税引前純利益（推計） ▲¥3,500M〜▲¥4,500M
法人税等（繰延税金資産計上等）    +¥2,000M〜+¥3,000M
純利益           ▲¥1,317M（赤転）
```

**一過性か継続リスクか：**

| 損失項目 | 性質 | 継続リスク |
|---------|------|----------|
| 店舗減損・閉鎖費用 | **一過性**（FY2026に集中計上） | 低〜中（残存老朽店舗次第） |
| のれん償却 | **継続的**（毎期発生） | 中（残存のれん額による） |
| 早期退職費用 | **一過性**（構造改革の前払いコスト） | 低（一巡すれば消滅） |
| 繰延税金資産取崩 | **要注意**（収益性低下時に発生） | 中（来期黒字確保が必要） |

> **深井判断**: 純利益赤転は**主に一過性の特別損失集中計上**が原因。ただし、のれん残高の水準によっては継続的な下押し圧力もあり得る。来期¥150Mという薄利の純利益計画は、「特損の消滅」を前提とした保守的見積もりと解釈できる。**一過性判断: 70%の確度で一過性**。ただし残存のれん・追加減損リスクは継続監視必要。

## 5. DCFバリュエーション（⑦）

### 5.1 DCF前提条件

#### 基本前提

| 前提項目 | 数値 | 根拠 |
|---------|------|------|
| **リスクフリーレート（Rf）** | **2.429%** | 日本10年国債（財務省 R8.4.6） |
| **ERP（株式リスクプレミアム）** | **5.5%** | 日本市場標準（Damodaran 2025推計） |
| **β（ベータ）** | **0.55** | 食品スーパー業界・ディフェンシブ銘柄（低β） |
| **株主資本コスト（Re）** | **2.429% + 0.55 × 5.5% = 5.45%** | 計算値 |
| **実効税率（Tc）** | **30%** | 日本法人税標準 |
| **負債コスト（Rd）** | **1.8%** | スーパー業界・投資適格水準 |
| **D/(D+E)** | **55%** | スーパー業界標準（賃借債務含む） |
| **E/(D+E)** | **45%** | 同上 |
| **WACC** | **45% × 5.45% + 55% × 1.8% × (1-0.30)** | |
| | **= 2.45% + 0.693% = 3.14%** | **≒ 3.1%** |
| **ターミナル成長率（g）** | **0.5%** | 食品スーパー成熟市場（人口減少考慮） |

```
WACC計算詳細:
  Re = 2.429% + 0.55 × 5.5% = 2.429% + 3.025% = 5.454%
  Rd×(1-Tc) = 1.8% × 0.70 = 1.26%
  WACC = 0.45 × 5.454% + 0.55 × 1.26%
       = 2.454% + 0.693%
       = 3.147% ≈ 3.15%
```

### 5.2 FCF推計の前提

| ΔWC | 売上変化×0.5%（運転資本変化） |
