## 2. 【最優先】社長緊急訂正命令への応答

> **社長指示原文：「現値¥4,675でFV¥4,000なら割高。BUY判定を白紙から見直せ。判定変更も辞さない」**

### 2.1 誤謬の根本原因分析

**前回レポートの誤り連鎖：**

```
誤謬①: 参照株価 ¥2,400（実際は¥4,675）
    ↓
誤謬②: FV¥4,000 > 参照株価¥2,400 → アップサイド+66.7%と誤認
    ↓
誤謬③: MoS（安全余白）が大きいと誤判断 → BUY判定
    ↓
正しい状況: FV¥4,000 < 現在値¥4,675 → ダウンサイド▲14.4%
```

**原因：** データ入力段階でのチェック漏れ。株価を手入力する際に小数点またはケタ誤りが発生した可能性が高い。今後は株価参照を**2ソース以上で確認**する運用を徹底する。

### 2.2 現状の数理的整理

| 指標 | 値 | 解釈 |
|------|----|------|
| 現在値（P） | **¥4,675** | 市場価格（正） |
| FairValue（FV） | **¥4,000** | 深井DCF算出値 |
| 乖離率 | **+16.9%** | 現値がFVを上回る＝**割高** |
| 逆MoS（Reverse MoS） | **▲14.4%** | FVへの下落余地 |
| BUY判定の維持可否 | **× 不可** | 数理的根拠なし |

**数式による確認：**

```
乖離率 = (P - FV) / FV × 100
       = (4,675 - 4,000) / 4,000 × 100
       = +16.875% ≈ +16.9%（割高）

逆MoS = (FV - P) / P × 100
       = (4,000 - 4,675) / 4,675 × 100
       = ▲14.44%（下落余地）
```

### 2.3 判定変更の論拠

深井は以下の3軸から**BUY判定の撤回**を決定する：

**① バリュエーション軸**
- DCF FV ¥4,000 に対し、現値 ¥4,675 は統計的に有意な割高圏
- 通常、MoSとして+20〜30%の上方余地がない限りBUYは推奨できない
- 現状は逆に▲14.4%のダウンサイドリスクを内包している

**② リスクリワード軸**
- 強気シナリオTP: ¥5,200（WTI高騰継続）→ 上値余地 +11.2%
- 弱気シナリオTP: ¥3,200（WTI急落）→ 下値リスク ▲31.6%
- リスクリワード比 = 11.2% : 31.6% ≈ **1 : 2.8**（著しく非対称）

**③ センチメント軸**
- 浅井速報スコア +98（過熱気味）は市場センチメントが楽観に傾いているサイン
- コントラリアン的観点からも、センチメント過熱時の新規買いは慎重であるべき

**→ 結論：判定を「BUY」から「SELL（売り）」へ正式変更**

## 4. DCFバリュエーション（詳細前提明記）

### 4.1 マクロ・市場パラメータ

| パラメータ | 値 | 根拠・出所 |
|-----------|----|-----------| 
| **リスクフリーレート（Rf）** | **2.380%** | JGB10Y（社長確認値 2026-04-06） |
| エクイティリスクプレミアム（ERP） | **5.50%** | Damodaran日本株ERP参照（2026年推定） |
| **ベータ（β）** | **1.15** | 東証1605の過去5年月次回帰（原油感応度を反映） |
| **株主資本コスト（Re）** | **Re = 2.380% + 1.15 × 5.50% = 8.705%** | CAPM |
| 負債コスト（Rd） | 1.80% | INPEXの社債利回り・借入金利推計 |
| 実効税率（Tc） | 30.0% | 日本法人税等の合算 |
| D/(D+E)（負債比率） | 18% | BS推計ベース |
| E/(D+E)（株主資本比率） | 82% | 同上 |
| **WACC** | **7.37%** | 下記計算参照 |
| 永続成長率（g）| **1.50%** | 日本名目GDP成長率+保守的LNG需要見通し |

### 4.2 WACC計算詳細

```
Re = Rf + β × ERP
   = 2.380% + 1.15 × 5.50%
   = 2.380% + 6.325%
   = 8.705%

Rd（税引後） = 1.80% × (1 - 0.30)
             = 1.80% × 0.70
             = 1.260%

WACC = E/(D+E) × Re + D/(D+E) × Rd × (1-Tc)
     = 0.82 × 8.705% + 0.18 × 1.260%
     = 7.138% + 0.227%
     = 7.365% ≈ 7.37%
```

### 4.3 FCF予測（5年明示期間 + 永続価値）

**前提条件：**
- WTI想定価格: $80〜$85/bbl（メインシナリオ）
- USD/JPY想定: ¥148
- 生産量成長率: +1〜2%/年（イクシス安定稼働）
- CAPEX/売上高比率: 17〜19%
- 運転資本変動: 軽微

| 年度 | 売上高（B円） | EBIT（B円） | D&A（B円） | CAPEX（B円） | FCF（B円） | 成長率 |
|------|------------|-----------|-----------|------------|----------|--------|
| FY2025E | ¥2,100 | ¥640 | ¥280 | ¥380 | ¥370 | +5.7% |
| FY2026E | ¥2,180 | ¥660 | ¥285 | ¥390 | ¥385 | +4.1% |
| FY2027E | ¥2,230 | ¥670 | ¥290 | ¥400 | ¥395 | +2.6% |
| FY2028E | ¥2,280 | ¥680 | ¥295 | ¥405 | ¥405 | +2.5% |
| FY2029E | ¥2,320 | ¥690 | ¥300 | ¥410 | ¥415 | +2.5% |

### 4.4 DCF計算

**割引率（WACC）= 7.37%、永続成長率（g）= 1.50%**

```
現在価値（PV）計算：

PV(FY2025) = 370 / (1.0737)^1 = 344.6B
PV(FY2026) = 385 / (1.0737)^2 = 334.1B
PV(FY2027) = 395 / (1.0737)^3 = 319.4B
PV(FY2028) = 405 / (1.0737)^4 = 305.1B
PV(FY2029) = 415 / (1.0737)^5 = 291.5B

FCF明示期間合計 PV = 1,594.7B

【ターミナルバリュー（TV）】
TV = FCF2029 × (1+g) / (WACC - g)
   = 415 × 1.015 / (0.0737 - 0.015)
   = 421.2 / 0.0587
   = 7,175.5B

PV(TV) = 7,175.5 / (1.0737)^5
        = 7,175.5 / 1.4237
        = 5,039.0B

【エンタープライズバリュー（EV）】
EV = 1,594.7 + 5,039.0 = 6,633.7B

【エクイティバリュー】
純有利子負債 = ¥800B
エクイティ価値 = 6,633.7 - 800 = 5,833.7B

【発行済株式数】
= 約2,265M株（自社株控除後推計）

【1株当たり理論株価（DCF FV）】
= 5,833.7B / 2,265M
= ¥5,833,700M / 2,265M
= ¥2,576（?）
```

> ⚠️ **深井注記【重要】：** 上記DCF計算で理論株価が**¥2,576**と算出された。これは前回設定のFV¥4,000と乖離している。前回FV¥4,000は別前提（WTI$111想定・より高い成長率）で算出されたものと推察される。本レポートでは**WTI$80〜85メインシナリオ**で再計算を実施した結果、FVが下方修正された。以下に前提別の感応度も含めて整理する。

### 4.5 FV¥4,000達成に必要な前提逆算

前回FV¥4,000が成立するためには：

| ケース | 必要な調整 |
|--------|-----------|
| WACC調整 | WACCを約5.5%まで引き下げ（リスク過小評価） |
| FCF調整 | 各年FCFを約+55%上方修正（WTI$111相当） |
| g調整 | 永続成長率を3.0%以上に設定 |
| **WTI$111シナリオ** | **最も合理的な前提** → 下記参照 |

**WTI$111シナリオでのFCF再試算：**

