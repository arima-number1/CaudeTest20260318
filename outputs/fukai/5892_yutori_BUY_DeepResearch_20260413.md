## 1. 企業概要・事業分析

### 1-1. yutori とは何者か

yutori（ゆとり）は2018年創業・2023年8月東証グロース上場の**Z世代向けSNS発アパレルブランドコングロマリット**。従来型アパレルと根本的に異なるのは「広告費をほぼかけず、InstagramとTikTokのオーガニックリーチで顧客獲得」するD2Cモデル。

**主要ブランド群（公開情報ベース）**:
- **#FR2（ハッシュタグ・フランク・ライナー）** — ストリート×アート。国内外でコレクター層を獲得。最大収益源。
- **chloma（クロマ）** — テクノロジー×ファッション。ニッチ高単価。
- **its'demo** — サブカルチャー雑貨・コスメ連携。 
- その他複数のマイクロブランドを孵化・運営

**ビジネスモデルの核心**:
```
SNSバイラル → 小ロット高回転生産 → ECメイン販売 → 高粗利維持
（在庫リスク最小化 × 広告費ほぼゼロ × ファンダム形成）
```

**競合優位性（Moat分析）**:
| 要素 | 評価 | コメント |
|------|------|---------|
| ブランド力・コミュニティ | ★★★★☆ | Z世代への浸透度は同世代比圧倒的。代替困難 |
| 低コスト獲得構造 | ★★★★☆ | SNSオーガニック主体。CAC（顧客獲得コスト）低い |
| スケーラビリティ | ★★★☆☆ | D2C小規模ブランドの限界→M&Aで打破を狙う |
| 財務基盤 | ★★☆☆☆ | BPS¥281・時価総額117億円と規模小 |
| 流動性 | ★★☆☆☆ | ADV5万株は機関投資家には狭い |
