## 4. DCFバリュエーション

### 4-1. 前提条件（全明記）

#### マクロ前提

| パラメータ | 数値 | 根拠 |
|---|---|---|
| **リスクフリーレート（Rf）** | **1.5%** | **JGB10年利回り（現在水準、指定値）** |
| **ERP（株式リスクプレミアム）** | **6.5%** | 日本株市場ERP標準レンジ5.5〜7.0%の中央値。ただし不正リスク発覚により+1.0%加算 |
| **β（ベータ）** | **1.6** | 小型テック×不正疑惑プレミアム。通常ベース1.2〜1.4に不正ディスカウント+0.2〜0.4 |
| **WACC** | **12.0%** | Rf 1.5% + β1.6 × ERP 6.5% = 1.5% + 10.4% ≈ **12.0%** |
| **ターミナル成長率（g）** | **1.0%** | 名目GDP成長率（約1.0〜1.5%）の低位値。不正発覚後の成長鈍化織り込み |
| **割引期間** | 5年間（FY2025〜FY2029） | — |

> **WACC算出式**：
> ```
> WACC = Rf + β × ERP
>      = 1.5% + 1.6 × 6.5%
>      = 1.5% + 10.4%
>      = 11.9% ≈ 12.0%（端数調整）
> ```
> ※ 負債コストは自己資本比率60%超のため影響軽微。簡便上、株主資本コスト≒WACCとして処理。

### 8-1. 即時アクション（本日〜今週）

```
【STEP 1】既存ポジション保有者向け
━━━━━━━━━━━━━━━━━━━━━━━━
