```

### 4-2. PERカウンターサイクリカル原則【KEY INSIGHT】

```
E&P企業のPER特性（経験則）:
  高油価（>$95）: PER 7.5〜9x   ← 「いつか落ちる」と市場が懐疑的

確定データでの検証:
  現在WTI$97 / EPS¥439 / 株価¥4,147 → PER9.4x ✓（9〜10x帯に一致）
  
この特性により、油価が下落してもFVは非線形に安定化する。
