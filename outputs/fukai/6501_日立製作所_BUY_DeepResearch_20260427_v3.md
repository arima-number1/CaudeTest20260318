## 8-B. 新設セクション：AI代替リスク vs Physical AI優位性

### 8-B-1. Bears Side — GlobalLogic人月モデル崩壊リスク

**構造的問題の核心:**

GlobalLogicは2021年に$9.6B（約¥1.4兆）で買収した。取得時の価値の根拠は約30,000人のエンジニアによる「人月課金モデル」のITエンジニアリング事業だ。インド・東欧拠点のオフショア開発は低コスト人月の供給で成立してきた。

**AIコーディングによる生産性破壊（実測値）:**

| AIツール | 実測生産性向上 | 出典 |
|---------|-------------|------|
| GlobalLogic VelocityAI（自社データ） | **30%生産性向上・25%時間短縮** | GlobalLogic公式プレスリリース |
| GitHub Copilot（現場体感） | **1.5〜3倍の生産性向上** | 業界レポート2026年 |
| AIエージェント型（Claude Code/Cursor等）| **1リポジトリ全体をPR化する生産性** | 2026年4月AI業界調査 |
| 工数削減の意味 | **同成果物を1/3〜1/2の人月で完成** | 上記から試算（推定） |

**DSSセグメントへの財務的インパクト試算（推定）:**

```
【試算前提】（全て推定値・実績での確認要）
DSS + GlobalLogic 売上: ¥3,500〜4,000B（FY2026推計）
DSS 調整後EBITA推定: ¥464〜532B（EBITA率13.3%ベース）
     ↑ セグメント合計EBITAから逆算した参考値

【AI代替シナリオ（3〜5年で進行・推定）】
人月単価▲20%（需要量維持×単価競争）シナリオ:
→ EBITA影響: ▲¥93〜106B（▲20%）
→ SOTP Lumada評価倍率: 25x → 20x（コモディティ化でマルチプル圧縮）
→ TP下方バイアス: ¥7,500 → ¥6,500〜7,000（▲7〜13%）（推定）

人月需要量▲30%（ヘッドカウント削減・最悪シナリオ）:
→ EBITA影響: ▲¥140〜160B
→ TP下方バイアス: ¥7,500 → ¥6,000〜6,500（▲13〜20%）（推定）
```

> **重要注記:** 上記は「推定」。AI代替の速度・範囲・業界の対応（単価転嫁・新サービス移行）には不確実性が大きく、3〜5年スパンでの影響。GlobalLogic自身がVelocityAIを武器に単価維持戦略を実行中であり、単純な人月崩壊とはならない可能性が高い。

**SIer人月モデル崩壊の業界動向:**

2026年春時点で、SIer/SES業界では「コードをどれだけ書けるか」から「仕様・設計を整理してAIに渡せるか」への価値シフトが加速中。AIエージェント型の台頭で1ユーザーあたりのコンピュート消費が10〜100倍に拡大し、フラットレートの経済性が崩壊。これは人月ビジネスモデルの変容を中長期的に不可避にする。

### 8-B-2. Bulls Side — Physical AI優位性とDSS自己防衛

**「AI代替に強い」要素（反論）:**

| 分野 | AI代替困難性の根拠 | 規模感 |
|------|----------------|-------|
| **Hitachi Energy（電力グリッド）** | IEC 61850等規制認証・変電所安全基準・人間の責任体制 | 受注残$43B・FY2027影響なし |
| **鉄道CBTC（列車制御）** | 安全認証SIL-4・乗客命に関わる安全要件 | 非代替領域 |
| **プラント・工場OT** | OT（Operational Technology）は設備固有ノウハウ×物理世界接続 | HMAXの核心 |
| **GlobalLogic VelocityAI** | AI代替リスクを逆手に取り顧客に30%生産性を提供→需要創出 | 5xのSDLC展開計画 |
| **自社株買い5,000億円** | EPSを守りながらAI代替による利益減少をヘッジ | 時価総額の2.1%消却 |

**Physical AI vs LLM代替領域の非対称性:**

```
【代替リスクなし（Physical AI優位）】
Hitachi Energy（売上¥4,200B・受注残$43B）: 規制・安全認証・物理制約
鉄道CBTC・信号システム: SIL-4安全認証
プラントOT・産業IoT: HMAXによる物理世界接続

【代替リスクあり（DSSデジタル）】
GlobalLogic ITエンジニアリング: 人月コーディングの一部
SIer的フロントエンド開発: AIコーディングで代替加速
汎用データ分析・レポート自動化: LLM API化

【重要な非対称性】
代替リスク対象（DSS）: 売上の約35〜40%
代替リスクなし（Physical AI）: 売上の約40%以上（かつ高成長）
→ 全体EBITAへの影響は「限定的」と判断（推定）
ただし、マルチプルのコモディティ化懸念が先行して株価圧迫するリスクはある
```

### 8-B-3. AI代替リスクに関する結論

**「Physical AI上振れ > DSS代替リスク」か否かの判断：**

現時点での深井の評価は **「Physical AI上振れ ＞ DSS代替リスク」** で BUY維持。

根拠:
1. Hitachi EnergyのDC受注残$43B（3年で3倍）の可視性が異次元に高い
2. GlobalLogicはVelocityAIで自己防衛しており、単純崩壊とはならない
3. AI代替が本格化しても3〜5年のタイムラグがあり、その間にHitachi Energyが利益を積み上げる
4. 自社株買い5,000億円がEPSを守りながら株価のフロアを形成

**モニタリング指標（AI代替リスクの早期警戒）:**
- GlobalLogic売上成長率がFY2027で**+10%割れ**した場合→レビュー
- DSS EBITA率が**12%割れ**（v3試算13.3%から▲1.3pt超の悪化）した場合→TP引き下げ検討
- GlobalLogic人員削減（インドスタッフ目標20,000人→大幅未達）した場合→崩壊開始のシグナル
