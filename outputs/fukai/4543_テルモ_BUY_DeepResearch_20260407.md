### 2-1. 株価低迷の原因分析：4要因の構造的 vs 一時的切り分け

#### 【要因①】コスト増：原材料・物流・人件費の複合インフレ

```
構造的か一時的か？ → 【半構造的】：一部改善中だが体質転換未完

内訳分解：
  ・原材料（樹脂、ステンレス、電子部品）：WTI原油$112/bbl水準での高原材料コスト継続
    → 製品原価率を約2〜3%pt押し上げ（FY2022〜FY2025累積）
  ・物流：コロナ禍以降のサプライチェーン再構築コスト
    → 一部正常化（一時的要因解消方向）
  ・人件費：日本国内の賃上げ圧力 + 海外（特に欧米）の慢性的人件費インフレ
    → 構造的上昇圧力、短期では解消不能
  ・製造コスト：Cコロナ対応設備投資の減価償却負担（FY2027頃まで継続）

深井判定：
  コスト要因の約60%は「半構造的」（原油・人件費）
  残り40%は「一時的」（物流・旧型設備の特別償却）
  → FY2026〜FY2027にかけて段階的に改善見込み。ただし完全解消には2〜3年
```

#### 【要因②】為替：円安ピークと円高転換リスク

```
構造的か一時的か？ → 【一時的（ただし反転が業績直撃リスク）】

テルモの為替感応度（推定、公開IR資料ベース）：
  売上高感応度：USD/JPY 1円の円高 → 売上高約▲40〜50億円
  営業利益感応度：USD/JPY 1円の円高 → 営業利益約▲15〜20億円
  ※EUR/JPYも重要（売上比率約20%）：1円円高 → 営業利益約▲5〜8億円

直近為替環境（2026年4月時点推定）：
  USD/JPY：推定142〜148円水準（円高方向へ転換局面）
  → FY2022〜FY2024の円安追い風（150〜160円）の剥落が確実に業績を下押し
  → FY2025 vs FY2024比較で為替逆風インパクト：営業利益▲100〜150億円（推定）

円高感応度シミュレーション（詳細は§2-4）：
  現在水準（推定145円）→ 130円シナリオ：営業利益▲225〜300億円
  → PL上の「見かけ上の低迷」として顕在化

深井判定：
  為替要因は「一時的」に分類できるが、円高が140円以下に定着した場合、
  業績見通しの下方修正連鎖が株価を更に押し下げる「罠」となりうる
  → 最大リスク要因として要警戒
```

```
構造的か一時的か？ → 【部分的に構造的（セグメント依存）】

セグメント別競合圧力：

  ・エドワーズ vs テルモ：TAVR領域でエドワーズが圧倒的シェア

  判定：部分的に構造的。ただしTRI・カテーテルの核心競争力は維持

[BCT：血液システム]
  ・Haemonetics：血小板採血でシェア競争激化
```

```
構造的か一時的か？ → 【構造的（日本特有）+ 一時的（海外承認遅延）】

  判定：FDA/MDR対応コストは一時的（FY2025〜FY2026で峠越え）

深井判定：
  国内薬価下落は構造的（逃げられない）
  海外規制コストは一時的。FY2026以降は収束見込み
```
