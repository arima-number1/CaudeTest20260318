## 2. 社長指示の重点分析：「低PBR × 高自己資本比率」

> ⚠️ **【PBR要確認】** リアルタイムPBR取得失敗。以下は学習データ・公開IR参照値。必ず最新値を確認してください。

### 2-1. エスペックの「低PBR×高自己資本比率」の構造

**エスペックの財務優良性（推定・参考値）**

|------|------------|------------|------|

| 現金・同等物 | 約100億円超（推定） | — | **財務バッファ十分** |

### 2-2. なぜPBRが低いのか？（構造的要因の解剖）

```
PBR低迷の方程式：
PBR = ROE × PER
   ≒ 7% × 12倍 = 0.84倍

ROEが低い → PBRが低い
↑
自己資本が厚すぎる（過剰資本）
＋
利益成長率が市場期待に届いていない
```

**三層構造の「PBR1倍割れ」要因：**

1. **過剰資本問題**：自己資本比率70%超は財務健全性の証明だが、同時に資本効率（ROE）を低下させる。株主から「遊んでいるキャッシュを返せ」のプレッシャー。

2. **利益成長の不確実性**：投資サイクル依存型ビジネスゆえ、景気後退・設備投資削減局面でEPS成長が止まりやすい。市場はこのシクリカルリスクをディスカウント。

3. **市場認知度の低さ**：B2B製造装置メーカーとして機関投資家・アクティビストの関心が薄い。「放置プレミアム」が存在。

### 2-3. PBR1倍回復シナリオ：何が必要か？

| 解消策 | 必要条件 | 株価へのインパクト |
|-------|---------|----------------|
| **自社株買い（最有力）** | 100億円規模の自社株買い実施 | PBR+0.15〜0.20倍、株価+15〜20% |
| **増配（継続的）** | 配当性向50%超へ引き上げ | PBR+0.05〜0.10倍、株価+5〜10% |
| **ROE改善（8%→12%）** | 売上増×利益率改善 or 資本削減 | PBR+0.3〜0.4倍（中長期） |
| **不動産含み益顕在化** | 非中核不動産の売却・活用 | 一時的EPS押し上げ+資本効率改善 |
| **IR強化・英語開示拡充** | 海外機関投資家へのアクセス向上 | 流動性プレミアム解消 |

### 2-4. 東証「資本コスト・株価意識」との連動

東京証券取引所は2023年以降、プライム市場のPBR1倍割れ企業に対して継続的改善を要請。エスペックは**その典型的対象企業**であり、以下の圧力にさらされている：

- 東証からの開示要請（資本コスト・株価意識の改善計画開示）
- アクティビスト投資家の参入可能性（財務優良×低PBR＝格好のターゲット）
- 競合他社の株主還元強化による相対的圧力

**結論：エスペックのPBR1倍割れは「放置された割安」であり、東証圧力＋アクティビスト潜在介入＋業績回復の3重カタリストが揃う。**

| **Rd（借入コスト）** | **1.2%** | 実質無借金企業。銀行借入金利推定。 |
| **実効税率（Tc）** | **30%** | 日本法人実効税率 |
| **D/（D+E）** | **約5%** | 実質無借金。自己資本比率70%超から算出。 |
| **E/（D+E）** | **約95%** | 同上 |
| **WACC** | **6.79%** | 下記計算参照 |
| **永続成長率（g）** | **1.5%** | 日本GDP成長率（名目）・エスペックの長期成長性考慮 |
| **予測期間** | **5年（FY2026〜FY2030）** | — |

**WACC計算：**
```
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.95 × 7.098%] + [0.05 × 1.2% × (1-0.30)]
     = 6.743% + 0.042%
     = 6.785% ≒ 6.79%
```
