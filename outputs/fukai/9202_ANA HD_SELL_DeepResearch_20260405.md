# 4. DCFバリュエーション

## 4.1 WACC算出

### 前提条件（全項目明記）

| パラメータ | 値 | 根拠・出典 |
|-----------|-----|----------|
| **Rf（リスクフリーレート）** | **2.380%** | JGB10Y（社長確認値 2026-04-05 17:47 JST） |
| β（ベータ） | 1.45 | 東証5年月次、TOPIX比較（航空セクター高β）推定 |
| ERP（株式リスクプレミアム） | 6.00% | Damodaran Japan推定値（2026年概算） |
| **Re（株主資本コスト）** | **11.08%** | Re = 2.380% + 1.45 × 6.00% |
| Rd（負債コスト・税前） | 1.80% | 社債発行コスト・借入金利推定 |
| Tc（実効税率） | 30.0% | 日本法人税率 |
| Rd × (1-Tc)（税引後負債コスト） | 1.26% | |
| E/（D+E）（株式比率） | 36.0% | 時価ベース（株価2,800円 × 3.2億株 ÷ 32,100億円総資本） |
| D/（D+E）（負債比率） | 64.0% | |
| **WACC** | **4.79%** | 下記計算式参照 |

```
【WACC計算式】
WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.36 × 11.08%] + [0.64 × 1.80% × (1-0.30)]
     = 3.989% + 0.806%
     = 4.795% ≈ 4.80%

【Re計算式】
Re = Rf + β × ERP
   = 2.380% + 1.45 × 6.00%
   = 2.380% + 8.700%
   = 11.080%
```
