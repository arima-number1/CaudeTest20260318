## 3. ビジネスモデル・競合分析

### ビジネス概要

エスフーズ（S-Foods）は食肉・食品の製造・加工・卸売を中核とする総合食品企業。主力事業は：

- **食肉製品製造**: ハム・ソーセージ・ハンバーグ・焼肉用カット肉など
- **食品流通（卸売）**: スーパー・コンビニ・外食チェーンへの大量供給
- **惣菜・調理加工品**: 内食トレンド捕捉
- **全国冷蔵物流網**: 自社冷蔵倉庫・配送インフラ保有

売上高 ¥450-500B（FY2027ガイダンス ¥500B）規模の中堅〜大型食品会社。低マージン（OP率2%前後）だが規模・物量で収益を確保するモデル。

## 6. バリュエーション（三手法）

**リスクフリーレート（Rf）**: 2.467%（JGB10Y、財務省CSV 2026-04-13）

### 手法①：類似企業比較法（PBR）

- ピア平均PBR: 1.22x
- エスフーズBPS: ¥3,957
- TargetPBR: **1.00x**（FCF赤字・低ROE調整後の保守値。ピア比18%ディスカウント適用）
- **FV = ¥3,957 × 1.00 = ¥3,957**（ウェイト: 30%）

> **根拠Rationale**: PBR 1.0x採用理由は①FCF赤字継続で内部留保蓄積が限定的②ROE 6.6%はWACCと拮抗し超過リターン小さい。ただし当期の業績改善でROE向上余地あり、1.0xは下限保守値。

### 手法②：フォワードPER法

- FY2027 純利益予想: ¥7,000M（QUICK コンセンサス採用）
- EPS: ¥7,000M / 31.67M = **¥221**
- Target PER: **15.2x**（ピア平均16.9x×10%ディスカウント; 小型流動性プレミアム控除）
- **FV = ¥221 × 15.2 = ¥3,360**（ウェイト: 35%）

> **根拠Rationale**: 15.2x採用理由は①エスフーズの時価総額¥96Bはピア（日本ハム¥657B等）比10分の1以下の小型株②配当利回り3.4%は既にバリュエーション支持③OP率2%の低マージン構造は高PER正当化困難。逆に言えばQ4ペース継続でEPS ¥250超なら高い上振れ余地。

### 手法③：DCF法

| パラメータ | 数値 | 根拠 |
|-----------|------|------|
| Rf | 2.467% | JGB10Y（財務省CSV） |
| β | 0.50（フロア適用） | 実測0.216は非現実的に低いため0.50で保守フロア |
| ERP | 5.5% | 日本株市場プレミアム標準値 |
| 小型株プレミアム | 1.5% | 時価総額¥100B以下 |
| Ke（株主資本コスト） | 2.467% + 0.50×5.5% + 1.5% = **6.717%** | |
| Kd（負債コスト税後） | 1.5% × (1-0.30) = **1.05%** | 食品業界標準金利 |
| 資本構成 | E: 75% / D: 25% | BS Net Debt ¥14Bより |
| **WACC** | **0.75×6.717% + 0.25×1.05% = 5.30%** | |
| FCF Year1 | ¥6,000M | NOPAT ¥7,000M + D&A ¥4,460M - 維持CapEx ¥5,000M - ΔWC ¥500M |
| FCF成長率（5年） | 5%/年 | OP正常化×価格改定効果継続 |
| Terminal growth（g） | 1.5% | 食品インフレ長期均衡 |

**FCFキャッシュフロー投影:**

| 年度 | FCF | 割引率 | PV |
|------|-----|--------|-----|
| Year 1 | ¥6,300M | 1.053 | ¥5,983M |
| Year 2 | ¥6,615M | 1.109 | ¥5,965M |
| Year 3 | ¥6,946M | 1.168 | ¥5,948M |
| Year 4 | ¥7,293M | 1.230 | ¥5,929M |
| Year 5 | ¥7,658M | 1.295 | ¥5,913M |
| **PV FCF合計** | | | **¥29,738M** |

**Terminal Value:**
- TV = ¥7,658M × 1.015 / (0.053 - 0.015) = ¥7,773M / 0.038 = **¥204,553M**
- PV TV = ¥204,553M / 1.295 = **¥157,949M**

**Enterprise Value** = ¥29,738M + ¥157,949M = **¥187,687M**
**Net Debt** = ¥14,000M（yfinance）
**Equity Value** = ¥187,687M - ¥14,000M = **¥173,687M**
**Per Share** = ¥173,687M / 31.67M = **¥5,484**（ウェイト: 35%）

> **根拠Rationale**: DCF感応度：WACCが1%上昇すると-¥900/株、g=1.0%にすると-¥800/株。WACC 5.3%は保守的設定であり、FCF ¥6,000Mも維持CapEx按分後の保守値。
