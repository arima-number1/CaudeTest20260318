**強み（Moat分析）:**

## 4. DCFバリュエーション

### 4-1. 前提条件（全明示）

```
【WACC計算前提】
─────────────────────────────────────────────────
 リスクフリーレート (Rf)    : 1.5%（JGB10年債利回り）
 エクイティリスクプレミアム : 5.5%（国内小型株・リユース業）
 β（アンレバード）         : 0.75（消費関連・ディフェンシブ補正）
 β（レバード、推定D/E）    : 0.80（ネットキャッシュのため保守的）
 株式コスト (Ke)           : 1.5% + 0.80 × 5.5% = 5.9%
 負債コスト (Kd)           : 1.8%（銀行借入想定）
 実効税率                  : 30%
 負債比率（時価ベース）     : 約15%
 WACC                      : 5.9% × 0.85 + 1.8% × 0.70 × 0.15
                           = 5.015% + 0.189% ≈ 5.2%
─────────────────────────────────────────────────

【FCF予測前提】
 ベースFCF（FY2024）       : 3.5億円
 予測期間                  : 5年（FY2025〜FY2029）
 成長率（予測期間）        : 3.0%（リユース市場成長に連動）
 ターミナル成長率 (g)      : 0.5%（非公開化後の保守想定）
 WACC                      : 5.2%
─────────────────────────────────────────────────
```

### 4-2. DCF計算詳細

```
【FCF予測（5年）】
─────────────────────────────────────────────────────
  Year   FCF（億円）  割引率(5.2%)   PV（億円）
─────────────────────────────────────────────────────
  FY2025    3.605       0.9506         3.427
  FY2026    3.713       0.9036         3.355
  FY2027    3.824       0.8589         3.284
  FY2028    3.939       0.8165         3.215
  FY2029    4.057       0.7761         3.149
─────────────────────────────────────────────────────
  予測期間PV合計                      16.43億円
─────────────────────────────────────────────────────

【ターミナルバリュー】
 TV = FCF2029 × (1+g) / (WACC - g)
    = 4.057 × 1.005 / (0.052 - 0.005)
    = 4.077 / 0.047
    = 86.74億円

 TVのPV = 86.74 × 0.7761 = 67.33億円
─────────────────────────────────────────────────────
 事業価値合計 = 16.43 + 67.33 = 83.76億円
 + ネットキャッシュ          =  +4.00億円
─────────────────────────────────────────────────────
 エクイティ価値              = 87.76億円
─────────────────────────────────────────────────────

【発行済株式数（推定）】
 約880万株（推定）

【1株あたり理論価値】
 87.76億円 ÷ 880万株 ≈ 997円/株
─────────────────────────────────────────────────────
```

## 6. メインシナリオ vs リスクシナリオ

### 6-1. 強気シナリオ（確率: 30%）

**前提:**
- スクイーズアウトが迅速に進行（2ヶ月以内）
- TOB価格1,000円で全株強制取得完了
- 一部市場参加者の先回り買いでさや縮小が前倒し

**株価推移:**
```
現在(970円) → 990円（1ヶ月後） → 1,000円（2ヶ月後）
リターン: +3〜6%
```

**シナリオPL:**
- 入金: 1,000円 × 購入株数
- 費用: 取引コスト（往復0.5%程度）
- **ネット期待リターン: +2〜5%（年率換算: +8〜30%）**

### 6-2. 基本シナリオ（確率: 55%）

**前提:**
- スクイーズアウト手続きに2〜4ヶ月を要する
- 一時的な市場価格の揺れ（±2%程度）を経てTOB価格収束
- 特段の法的紛争なし

**株価推移:**
```
現在(960円) → 950〜970円（揺れ） → 1,000円（3ヶ月後）
リターン: +3〜5%（時間コスト考慮後）
```

### 6-3. 弱気シナリオ（確率: 15%）

**前提:**
- スクイーズアウトに予期せぬ遅延（法的異議申し立て等）
- 少数株主による価格不服申し立て（非価格下落要因だが時間リスク）
- 市場全体のリスクオフによる流動性低下
- 最悪ケース: TOB自体の何らかの条件変更（極めて低確率）

**株価推移:**
```
現在(960円) → 900〜920円（一時的下落） → 長期保有で1,000円収束
最大ドローダウン: ▲5〜8%（一時的）
```

**弱気シナリオでの留意点:**
- スクイーズアウト後の価格は原則TOB価格と同一
- 下落は時間的機会損失が主リスク

### 6-4. シナリオ統合期待値

```
期待理論株価 = 1,050 × 0.30 + 1,000 × 0.55 + 910 × 0.15
             = 315 + 550 + 136.5
             = 1,001.5円
```

→ **期待値ベースの理論株価: 約1,000〜1,002円**（TOB価格とほぼ一致）
