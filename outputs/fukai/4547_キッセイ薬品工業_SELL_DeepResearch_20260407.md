## ②日本からの症例集中（76例中66例）の意味

### 地理的偏在の構造的要因

| 要因 | 詳細 |
|------|------|
| **①投与量・用法の差異** | 日本では承認時の用量が欧米と異なるケースが多い。アバコパン（タブネオス）は体重ベース投与だが、日本人の体型・肝代謝特性でCmax上昇の可能性 |
| **②患者背景（ANCA血管炎の疾患特性）** | ANCA関連血管炎は日本において欧米より高齢者・腎機能低下者比率が高い。肝代謝能が低下した患者への曝露増大 |
| **③ファーマコビジランス体制の差異** | 日本は副作用報告制度（PMDA）が相対的に積極的な報告文化。実際の発生率差よりも、報告率の差が反映されている可能性 |
| **④市場規模・処方実績** | ANCA関連血管炎の治療において日本はグローバルで主要市場の一つ。絶対的処方数が多い分、副作用例数も多い |
| **⑤併用薬の差異** | 日本では免疫抑制剤（シクロホスファミド、アザチオプリン）との併用が多く、薬物相互作用による肝毒性増強の可能性 |

### PMDAの対応シナリオと国内販売停止リスク

```
PMDAリスクシナリオ評価：
┌─────────────────────────────────────┐
│ 症例の87%が日本由来 → PMDAとして対応不可避   │
│                                          │
│ 対応レベル想定：                           │
│ Level 1：緊急安全性情報（DSU）発出          │ ← ほぼ確実
│ Level 2：使用上の注意改訂（添付文書改訂）   │ ← 高確率
│ Level 3：使用制限（専門医・施設限定）       │ ← 中確率(40%)
│ Level 4：販売中止・承認取消               │ ← 低確率(15%)
└─────────────────────────────────────┘
```

### 深井コメント：
> 日本偏在の最大の問題は「**単なる報告バイアスではない可能性**」にある。76例中66例という集中度は統計的に有意であり、日本人特有の薬物動態（肝代謝酵素の遺伝多型、CYP3A4活性の民族差）が関与している場合、添付文書改訂や使用制限では不十分との判断にPMDAが至るリスクがある。**国内販売停止リスクは15〜20%と見積もる。**

### バリュエーション下限の計算

#### PBR下限分析

| 指標 | 数値 | 備考 |
|------|------|------|
| BPS（確認値） | **¥5,220** | 唯一の正解値 |
| 現在PBR | **0.93倍** | ¥4,830÷¥5,220 |
| 業種最低PBR水準（製薬中堅）| 約0.6〜0.7倍 | 危機時の歴史的底値 |
| **危機時下限株価** | **¥3,132〜¥3,654** | BPS×0.6〜0.7倍 |

```
PBRバリュエーション下限試算：
─────────────────────────────────
PBR 0.7倍 × BPS¥5,220 = ¥3,654  ← 通常下限
PBR 0.6倍 × BPS¥5,220 = ¥3,132  ← 危機時下限
PBR 0.5倍 × BPS¥5,220 = ¥2,610  ← 最悲観（販売停止等）
─────────────────────────────────
現在値¥4,830（PBR 0.93倍）は下限まで▲24〜46%の下振れ余地
```

#### ROE-PBR整合性

**「正当なPBR = ROE ÷ 株主資本コスト」（Implied PBR）**

```
ROE = 6.7%
株主資本コスト（Re）= 後述WACC計算に基づき推定8.0〜9.0%

Implied PBR = 6.7% ÷ 8.5% = 0.79倍
→ 理論PBR: 0.79倍
→ 理論株価 = ¥5,220 × 0.79 = ¥4,124

ただしタブネオス問題でROE低下（→4.5%に低下想定）した場合：
Implied PBR = 4.5% ÷ 8.5% = 0.53倍
→ 理論株価 = ¥5,220 × 0.53 = ¥2,767
```

**→ タブネオス問題によるROE低下を織り込むと、ROEベース理論株価は¥2,767〜¥4,124のレンジ。現在値¥4,830は上値に位置。**

# 4. DCFバリュエーション

## 4.1 前提条件（全項目明記）

| パラメータ | 値 | 根拠 |
|----------|------|------|
| **リスクフリーレート（Rf）** | **2.422%** | 日本10年国債（社長確認値・2026/4/7）|
| **ERP（株式リスクプレミアム）** | **5.5%** | 日本市場標準値（Damodaran参照） |
| **β（ベータ）** | **0.85** | 製薬中堅・守備的業種。ただし今回事件でリスク上昇→0.90に調整 |
| **株主資本コスト（Re）** | **2.422% + 0.90×5.5% = 7.37%** | |
| **負債コスト（Rd）** | **0.8%** | 低金利・財務健全 |
| **実効税率（Tc）** | **30%** | 法人税・地方税合算 |
| **資本構成 E/(D+E)** | **90%** | 無借金経営に近い |
| **資本構成 D/(D+E)** | **10%** | |
| **WACC** | **7.37%×0.90 + 0.8%×0.70×0.10 = 6.69% ≒ 6.7%** | |
| **永続成長率（g）** | **0.5%** | 成熟製薬・低成長 |

### WACC計算詳細
```
Re = Rf + β × ERP
   = 2.422% + 0.90 × 5.5%
   = 2.422% + 4.950%
   = 7.372%

WACC = [E/(D+E) × Re] + [D/(D+E) × Rd × (1-Tc)]
     = [0.90 × 7.372%] + [0.10 × 0.80% × 0.70]
     = 6.635% + 0.056%
     = 6.69%（≒6.7% として使用）
```
