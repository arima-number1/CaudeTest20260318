### キャッシュポジション分析

- **現金¥558億は時価総額¥551億を上回る（Cash>時価総額）**
- これはネットキャッシュがプラスであり、理論上「現金だけで時価総額を説明できる」状態
- 有利子負債ゼロ → EV（企業価値）= 時価総額¥551億 - 現金¥558億 = ▲¥7億（EV実質ゼロ）

> Rationale: EVがほぼゼロということは、市場はコロプラの**事業価値をゼロと評価**していることを意味する。逆に言えば「現金バリュー以上の事業価値回復があれば株価は上昇余地大」という構造。

### 隠れ資産の考察

- **保有有価証券（投資育成事業）**: コロプラネクスト経由の未上場株式投資を複数実施。B/S上の投資有価証券が時価以下で計上されている場合、含み益が存在する可能性（※公式未開示のため推計扱い）

- 重要注記: 不動産含み益については公式有報で固定資産明細を確認するまで「不明」とする。過去実績への推定値使用禁止プロトコルに基づき、本項は「※要確認」とする

### 新TOPIX第2段階の基準概要

|---|---|
| 継続採用基準 | 全上場銘柄の浮動株時価総額を大きい順に累積→**上位97%**以内に入ること |
| 新規採用基準 | 上位96%以内（より厳しい） |
| 評価基準日 | **2026年8月末日**（年次） |
| 評価結果発表 | 2026年10月頃（JPX公表） |
| 移行措置 | 除外対象銘柄は8四半期（2年間）で段階的にウェイトを12.5%ずつ低減 |
| 完全除外時期 | 2028年7月末（第8段階の最終実施） |

### ボーダーライン水準

- **2024年8月末時点のボーダー**: 約¥230億（JPX総研試算）
- **2025年8月末時点のボーダー**: 約¥280億（市場推計）
- **2026年8月末の予想ボーダー**: ¥280〜320億（株価水準次第で変動・※推計）

> Rationale: ボーダーはTOPIX全銘柄の浮動株時価総額分布の97%点であり、市場全体の株価上昇で上方シフトする傾向。2024→2025の+¥50億増加トレンドから2026年は¥280〜320億を想定。

### コロプラの浮動株時価総額の推計

```
計算根拠（2026年4月24日時点）:
  発行済み株式数: 1億2845万株
  浮動株数（yfinance floatShares）: 6550万株
  浮動株比率: 6550万÷12845万 = 51.0%

  ※注意: JPXの「浮動株」計算は「大株主上位10名保有株・政策保有株等を除く」方式
  馬場功淳社長（創業者）の保有比率: 約48.1%（大株主調査より）
  主要機関投資家等の合計: 約27.8%（上位10名合計75.9%から社長分を除く）

  JPX基準の流通株式数（概算）:
    発行済み1億2845万株 × (1 - 大株主75.9%) ≒ 3095万株

  JPX基準の流通時価総額:
    3095万株 × ¥429 ≒ ¥133億

  ※ただし上記はyfinanceとJPX計算方式の差異に注意が必要。実際のJPX判定値は東証開示データを参照すること。
```

> **重大な警告**: 上記¥133億はJPX独自の計算方式による概算であり、ボーダー¥280〜320億を大幅に下回る可能性が高い。

### 除外確率の評価

| 評価 | 判定 |
|---|---|
| **除外確率** | **HIGH（70〜80%）** |
| 根拠① | 流通時価総額の暫定試算¥133億（JPX基準）がボーダー¥280億の約47%水準 |
| 根拠② | 創業者が48%超を保有→JPX基準の「大株主控除」が大きく効き、「見かけの時価総額」(¥551億)と「流通時価総額」の乖離が大 |
| 根拠③ | 2025年9月期の売上縮小傾向→株価が今後大幅に上昇しない限りボーダー突破は困難 |
| 緩和要因 | 自社株買い実施（浮動株増加）or 馬場氏の持ち株一部放出があれば状況変化 |

> **重要**: 上記試算は著者による概算であり、JPX公式の「流通株式時価総額」とは差異が生じる可能性がある。東証が年次で開示するスクリーニング結果を必ず参照のこと。

### パッシブファンドの推定保有株数・金額

```
計算根拠:

  コロプラのTOPIXウェイト（概算）: ¥133億÷550兆円 ≒ 0.00242%
```

> Rationale: パッシブファンドの保有比率はTOPIX全体に対するウェイトに比例する。上記は概算であり、実際のウェイトはTOPIX公式データで確認要（※推計）。

### 売り圧力の推定（ADV%換算）

```
推定パッシブ保有株数: 約606万株（上記試算）
8段階分割なので1回あたりの売り:
  606万株 ÷ 8回 ≒ 75.75万株/回（1四半期に1回）

コロプラの1日平均出来高（直近）: 約6万株/日（yfinance regularMarketVolume参考）
ADV換算: 75.75万株 ÷ 6万株 ≒ 約12.6ADV分/回

  ※ただし、これは各四半期の「1日」に集中するのではなく、
  発表から実施まで数週間にわたって分散して執行される
```

> **解釈**: 1回あたりの売り圧力は約12.6日分の出来高に相当する大きな需給インパクト。2年間8回にわたって継続することで、じわじわと株価を押し下げる構造になる。

### 「除外リスクは既にPBRに織り込まれているか」の考察

- 現在のPBR 0.83倍は、過去（2018年ピーク時5〜6倍）から大幅下落した水準
- 低PBRの主因は**ゲーム事業の収益性悪化・Nintendo特許訴訟での89億円支払い（2021年和解）**であり、TOPIX除外リスクが明示的に織り込まれた形跡は限定的
- 一方で、機関投資家の多くは既にTOPIX除外リスクを認識しており、**買い控えによる間接的な割安放置**が起きている可能性が高い
- **結論: TOPIX除外リスクは「部分的には（間接的に）」織り込まれているが、実際の「除外確定後のパッシブ売り8段階」はまだ株価に織り込まれていない。除外確定の瞬間に株価は追加で5〜15%程度下落するリスクあり。**

### DCF分析

```
前提:
  Rf = 2.380%（固定・社長指示）
  β = 0.206（yfinance実績値）
  ERP = 5.5%（日本市場標準）
  WACC = Rf + β × ERP = 2.380% + 0.206 × 5.5% = 3.51%
  ※有利子負債ゼロのためWACC=株主資本コスト

  FCF想定（FY2026〜FY2028）:
    FY2026E: ¥10億（1Q実績から年率換算・大幅保守）
    FY2027E: ¥15億（XR収益化進展・ベースケース）
    FY2028E: ¥20億（安定化）
    Terminal: FY2028 FCF × (1+g) / (WACC - g) ※g = 1.0%

  DCF Value = PV(FCF) + PV(Terminal) + Net Cash

  PV(FY2026): ¥10億 / (1+0.0351) ≒ ¥9.7億
  PV(FY2027): ¥15億 / (1+0.0351)^2 ≒ ¥14.0億
  PV(FY2028): ¥20億 / (1+0.0351)^3 ≒ ¥18.0億
  Terminal: ¥20億 × 1.01 / (0.0351 - 0.01) = ¥806億
  PV(Terminal): ¥806億 / (1+0.0351)^3 ≒ ¥727億

  事業価値: ¥769億
  + Net Cash（¥558億 - 有利子負債¥0）: ¥558億
  = 総価値: ¥769 + ¥558 = ¥1,327億 → 1株¥1,033（※ベース事業収益過大評価の可能性）

  ※上記DCFは「事業が正常化する」前提での計算。
  現実的には事業縮退リスクがあるため、Total価値から
  TOPIX除外ペナルティ（▲15%）・ゲーム縮退継続（▲30%）を割り引き:
  修正FV = ¥1,033 × 70% ≒ ¥723

  → ただしこの数値は楽観的。ゲームFCFの持続性が不確実なため、
    保守的には**Net Cashベース（¥558億÷1.285億株 = ¥434/株）**を主軸に採用。
```

> Rationale: WACC 3.51%は低β（0.206）による。コロプラは株式市場との連動性が極めて低い（市場感応度が低い）ことが特徴。現金リッチ企業ゆえWACCが低く、DCFは理論上高い数字が出やすいが、事業縮退リスクで大幅割り引きが必要。

### リスク② ゲームDAU/ARPU悪化継続シナリオ

- 新タイトル（白猫GOLF、ブロックチェーンゲーム）が期待を下回れば、売上高が¥200億を下回る可能性

- XR投資の追加焼損が¥50億規模になれば、BPS¥515から¥476程度に低下し下値サポートが切り下がる
