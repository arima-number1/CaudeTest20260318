## 1. FY2025 OPM崩壊▲200bpsの解剖

### 前提：計測可能な2つの確定値

```
FY2024: Revenue ¥46,126M  OP ¥2,545M  OPM 5.52%
FY2025: Revenue ¥51,040M  OP ¥1,784M  OPM 3.50%
```

**「OPMが保たれた場合のOP」= ¥51,040M × 5.52% = ¥2,817M**
**実際との乖離（コスト超過）= ¥2,817M - ¥1,784M = △¥1,033M ← これを分解する**

---

### 分解①：一過性費用（M&A・持株会社移行）

会社は「一過性費用を除いた実態OP ≈ ¥2,400〜2,600M」と示唆。

```
一過性費用 = 実態OP中央値¥2,500M - 実際OP¥1,784M = △¥716M
OPM換算 = △¥716M / ¥51,040M = △140bps
```

**→ 全コスト超過△¥1,033Mのうち△¥716M（69%）が一過性。**

---

### 分解②：原材料コスト高（タコ・植物油・小麦）

FY2025の食材費増加を以下の前提で試算：

| 原材料 | 売上比仮定 | 前年比価格上昇 | OPインパクト |
|--------|-----------|-------------|------------|
| タコ（モーリタニア産） | 売上の12〜15% | +15〜20%（欧米需要増） | **△¥830〜¥1,530M** |
| 植物油（サラダ油） | 売上の6〜8% | +8〜12%（ホルムズ前兆） | △¥245〜¥489M |
| 小麦粉 | 売上の4〜5% | +5〜7%（昭和産業値上げ） | △¥102〜¥178M |

**原材料純増インパクト推計 = △¥200〜¥400M（中央値△¥300M）**

ただし、3年累積+16%の価格転嫁がこの一部を吸収している。転嫁前の原料高は△¥500M以上だった可能性があり、転嫁により△¥200〜300Mに圧縮。

```
原材料要因 OPM換算 = △¥300M / ¥51,040M ≈ △59bps（切り上げ△60bps）
```

---

### 分解③：人件費・その他構造的コスト増

残余 = △¥1,033M - △¥716M（一過性）- △¥300M（原材料）= **△¥17M**

人件費は最低賃金+5%（FY2025）の影響を受けたはずだが、残余がほぼゼロになるのは「原材料インパクト推計の誤差範囲内」か「人件費増が海外/新業態展開でさらに一過性要因に分類されている」からと解釈できる。

```
人件費・その他 ≈ △0〜40bps（残余）
```

---

### OPM崩壊▲200bpsの最終分解

| 要因 | OPインパクト | OPM bps | 性質 |
|------|------------|---------|------|
| **M&A・持株移行・米国先行費用** | **△¥716M** | **△140bps** | **一過性（消滅確認要）** |
| 原材料高（タコ/油/小麦）×転嫁ラグ | △¥300M | △59bps | **構造的（継続リスク）** |
| 人件費・その他 | △¥17M | △1bps | 軽微 |
| **合計** | **△¥1,033M** | **△200bps** | — |

**結論：▲200bpsの70%は一過性。実力OPMは4.7〜5.1%（ガイダンス4.3%を上回る可能性）。**
**ただし残り30%（△60bps）は構造的コスト高で回収できない可能性がある。**

## 3. 値上げ限界点の定量考察

### 価格弾力性（PED）の現在地

3年間の実績データから逆算：

```
FY2023→FY2025累積データ（全店ベース）：
  売上成長：¥38,710M → ¥51,040M = +31.8%（3年累計）
  店舗数成長：推定+8〜12%（全店売上の一部が新店由来）
  既存店売上（SSS）成長推定：+31.8% - 10%（新店）= +21.8%（3年SSS）
  
累積値上げ：+16%（ソース8個 ¥580→¥669）
  
SSS売上 = 単価 × 客数
+21.8% = (1+0.16) × (1+客数変化)
客数変化 = (1.218/1.16) - 1 = +5.0%（3年累計）→年率+1.6%
```

**→ 3年で客数は累積+5%（年率+1.6%）のプラスを維持。PED = ▲5%客数変化 / +16%値上げ = ▲0.31（強い非弾力性）**

ただし、2025年末の「¥390特価」投入は「値上げ実施後に客数が圧迫され始めたシグナル」である。

FY2025単年の値上げ幅（¥626→¥669 = +6.9%）と既存店客数の変化を推定：

```
FY2025 SSS ≈ 全店+10.7% - 新店寄与6%（推定） = +4.7%
SSS = 単価+6.9% × 客数変化
客数変化 = 1.047/1.069 - 1 = ▲2.1%（FY2025初の客数マイナス転落推定）
FY2025 PED = ▲2.1% / +6.9% = ▲0.30（依然非弾力的だが悪化）
```

**3年トレンド：PED ▲0.31 → ▲0.30（横ばい）。ただし2025年に初の客数マイナス転落。**

### ¥800の壁：収益反転の数学

| 価格水準 | 想定PED | 6%値上げ時客数変化 | 売上変化 |
|---------|---------|-----------------|---------|
| 現在¥669（ソース8個） | ▲0.30 | ▲1.8% | +4.0%（プラス維持） |
| ¥720（次回想定） | ▲0.50 | ▲3.0% | +2.8%（プラスだが縮小） |
| ¥770 | ▲0.80 | ▲4.8% | +0.8%（ほぼ限界） |
| **¥800〜¥820（限界点）** | **▲1.00〜1.20** | **▲6〜7%** | **▲0〜▲1%（収益反転）** |

**¥800突破までの値上げ余地：¥669から¥800 = +19.6%。年率5〜6%の値上げで3〜4年。**

**タコ価格が+20%跳ね上がり「転嫁必要額が年+8〜10%」になった場合：¥800到達は1.5〜2年に短縮。**

## 4. 完全希薄化後FV（Margin of Safety 厳格版）

### 使用前提（厳格化）

- **株式数：PO後26.04M株（OA全行使）**
- **実力EPS：一過性費用△¥716M除外、ただし構造的原材料コスト▲60bpsは控除**
- **実力OP：¥2,500M（FY2026ガイダンス）× 実力調整 = ¥2,500M**（ガイダンスは既に一過性除外ベースのため調整不要）

### 実力EPS（完全希薄化後）

```
FY2026 実力OP：¥2,500M（ガイダンス）
実力純利益：¥2,500M × 72.7% = ¥1,817M
完全希薄化後EPS：¥1,817M / 26.04M株 = ¥69.8
```

### FV算出（3手法）

| 手法 | FV | ウェイト | Rationale |
|------|-----|---------|-----------|
| PER法（完全希薄化EPS¥69.8×PER25x） | **¥1,745** | 40% | 外食成長株PER20〜30x。希薄化直後は低位PERが妥当。ガイダンス達成確度を反映した25x |
| EV/EBITDA法（12x×EBITDA¥3,767M÷26.04M株) | **¥1,736** | 35% | EBITDA=OP¥2,500M+D&A推定¥1,267M（売上の2.5%）=¥3,767M。外食10〜14x中央値 |
| DCF（WACC6.63%、FCF成長8%〜g1.5%） | **¥1,900〜¥2,100** | 25% | PO後成長シナリオ。FY2028 OP達成前提 |
| **加重平均FV** | **¥1,800**（レンジ¥1,650〜¥2,000） | — | Rf=2.380%（財務省CSV） |

### Margin of Safety 厳格評価

```
現値：¥1,730
FV中央：¥1,800
MoS：(¥1,800-¥1,730)/¥1,800 = 3.9%

通常必要なMoS：20〜30%
必要エントリー価格（MoS20%）：¥1,800 × 0.80 = ¥1,440
必要エントリー価格（MoS15%）：¥1,800 × 0.85 = ¥1,530
```

**結論：¥1,500以下がエントリーライン。現値¥1,730はMoS実質ゼロ。**

## 6. PO参加可否の最終答え

**PO参加：絶対見送り。理由を3行で。**

1. 公募価格¥1,644〜1,679は現時点のFV（¥1,800）を▲7〜9%下回るが、MoSはわずか3〜8%。通常要求する20%安全域に遠く及ばない。
2. 完全希薄化によりFY2026 EPSは¥87（FY2024基準）→¥69.8に**▲18%強制圧縮**。EPS回復はFY2028が最速、かつOP+40%達成が条件。
3. PO価格での投資が元本を取り戻す（IRR≥0）ためには、3年後株価が少なくとも¥1,750超（=FY2028 EPS¥85×PER20.6x）が必要。ベアシナリオ（確率35%）では¥1,006まで下落し、**PO投資額の39%が消滅する。**

**エントリー判断：¥1,500以下（MoS16.7%）で打診、¥1,300以下（MoS27.8%）でフル投資。**
