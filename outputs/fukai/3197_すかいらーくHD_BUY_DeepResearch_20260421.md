### Q1「賃上げのガイダンス織込み状況」→ **最大ポジティブ（Rev.2の最大リスク消滅）**

|------|----------------|----------------|

### ④のれんはベースFVに影響しないが、ベアケースFVを引き下げる

```
ベースケースEPS FY2027E：¥101（減損なし前提）
  → P/E 38〜40x：FV ¥3,842〜¥4,044 → 採用¥3,800

バーミヤン減損¥200億（税後▲¥62/株）が発生した年度：
  → EPS：¥101 - ¥62 = ¥39（一時減損処理）
  → 減損は「一過性費用」→ P/Eで評価するのは不適切
  → P/B・正常化EPS還元：正常FV▲¥200億÷2.275億株=▲¥88（NAV押し下げ）
  → ベアケースFV推定：¥2,200〜¥2,400

翌期以降（減損消化後）：EPS¥101程度に回帰
  → 回帰後FV¥3,800 方向
```
