## 5. ARR定義変更（Fintech追加）の戦略的意義

**変更内容**: Businessセグメントのカード・早期入金等Fintech領域をSaaS ARRに追加
- 追加ARR: ¥1,684M（Q1末時点）→ **+89.7%成長の超高速セグメント**
- FY2026ガイダンスARR修正: ¥47,500〜49,800M → **¥49,742〜52,505M**（+4.7〜5.4%上方修正）

**意義の本質:**
- Fintech（カード・早期入金）はトランザクションビジネスだが「経常的・反復的収益」として定義可能
- これにより「SaaS ARR」の成長率が実態をより正確に反映
- MFビジネスカードは前年比+200%のトランザクション売上 → Fintechとの統合で**ARPAの上限が事実上存在しない**構造へ

## 7. バリュエーション（SaaS三手法）

SaaS・成長株の損益計算書ベースPERは意味をなさない。EV/ARR・EV/Revenue・DCFで評価。

### 手法①：EV/ARR法（主要SaaS指標）

| ベンチマーク | EV/ARR | 備考 |
|-----------|--------|------|
| Sansan (4443) | ~10x | 名刺SaaS、ARR成長20%台 |
| freee (4478) | ~8x | 会計SaaS、ARR成長25% |
| **マネーフォワード（現在）** | **4.1x（Forward）** | **34%成長・Rule40=53.4 → 明確割安** |
| **マネーフォワード TP想定** | **6.5x** | 成長率・品質考慮の保守Target |

- Target EV/ARR: **6.5x**（34%成長・Rule40 53.4点・低解約率のプレミアム正当化）
- FY2026 ARR 中値: ¥51,124M
- Target EV = ¥51,124M × 6.5 = ¥332,306M
- Add Net Cash: ¥332,306M + ¥7,120M = ¥339,426M
- Per share: ¥339,426M / 55.47M = **¥6,119**（ウェイト: 40%）

> **根拠Rationale**: 6.5x採用の理由は①ARR +34.2%はfreee（25%）よりも10pt高い成長率②Rule of 40スコア53.4点は国内SaaS最高水準③法人解約率0.8%/月の堅牢なコホート品質④ただしOP赤字継続のため15〜20%プレミアムディスカウント調整

### 手法②：EV/Revenue法

- FY2026 売上 QUICK consensus: ¥56,286M
- Target EV/Revenue: **5.0x**（Gross Margin 68%のSaaSとして適正中間値）
- Target EV = ¥56,286M × 5.0 = ¥281,430M
- Add Net Cash: ¥281,430M + ¥7,120M = ¥288,550M
- Per share: ¥288,550M / 55.47M = **¥5,202**（ウェイト: 30%）

> **根拠Rationale**: EV/Revenue 5.0x採用理由は①Gross Margin 68%はSaaSとして高水準②ただしまだ赤字フェーズ（利益安定化前は売上対比割引）③日本SaaSピア（Sansan 6x、freee 4x）の中間

### 手法③：DCF法（中長期SaaS黒字化シナリオ）

| パラメータ | 数値 | 根拠 |
|-----------|------|------|
| WACC | **8.5%** | Rf 2.467% + β 1.059×ERP 5.5% + 0.2% = 8.49% |
| Terminal growth（g） | 3.0% | SaaS×Fintech長期成長率 |
| 明示期間 | 5年 | FY2026〜FY2030 |

**売上・EBITDA投影（Long Roadmap）:**

|------|----------|---------|--------|---------|

## 10. 不動産含み益考察（全DR必須）

マネーフォワードは**純粋SaaS企業**であり、製造業・小売業のような不動産含み益は基本的に存在しない。

| 資産区分 | 金額 | 備考 |
|---------|------|------|
| 使用権資産（リース） | ¥3,812M増加（Q1） | オフィス賃借（自社物件なし）。不動産含み益なし |
| 知的財産・ソフトウェア | 簿価 ≪ 市場価値 | SaaSのARR/LTV差分に体化済み |
| **実質的な「含み益」** | **SaaS ARR ¥44,303Mの簿外価値** | **簿価には計上されないARRのLTV換算額が最大の隠れ資産** |

**SaaS ARR LTV試算（隠れ資産相当）:**
- 解約率 0.8%/月 → LTV倍率: ARR ÷ 解約率 = ¥44,303M ÷ 0.096（年次12%） ≈ **¥461,490M**
- 対EV（¥208B）比: **LTV/EV = 2.22倍** → EV対比で2倍超の顧客生涯価値内包
