## 2. 社長指示重点分析（最優先）

### ① 本日決算の詳細分析

#### （A）2Q¥2,166M → 3Q¥1,395M ▲771M減少の要因分析

```
四半期営業利益推移：
1Q（25.06-08）：¥1,190M　営業利益率：5.31%（売上¥22,421M）
2Q（25.09-11）：¥2,166M　営業利益率：8.18%（売上¥26,493M）
3Q（25.12-02）：¥1,395M　営業利益率：5.80%（売上¥24,035M）
```

**▲771M減少の要因分析（複合要因）：**

**【要因①】季節性・稼働日数差（影響度：中〜大、▲300〜400M推定）**
- 12月〜2月期は年末年始休暇・工場メンテナンス停止が集中
- 自動車メーカーの年末生産調整による発注量の季節的低下
- 前年同期（FY2025 3Q）との比較が重要。前年3Q営業利益を¥1,117M（逆算：前年比+24.9%=¥1,395M÷1.249≒¥1,117M）と推計すると、実は前年比では+¥278M増（+24.9%増）であり、季節性の範囲内。

**【要因②】売上高の季節的減少（影響度：大、▲300〜400M推定）**
- 2Q売上¥26,493M → 3Q¥24,035M：▲¥2,458M（▲9.3%の売上減少）
- 固定費の吸収率低下による利益率圧迫（固定費レバレッジの逆回転）
- 売上減少を粗利率改善で一部相殺（3Q粗利率改善示唆）

**【要因③】原材料コストの季節変動（影響度：小〜中、▲100〜200M推定）**
- ABS樹脂・PP樹脂は冬季需要期に上昇する傾向
- ただし原油価格はWTI$114.96と依然高水準だが、2Q→3Q変化は限定的と推定
- 電力コストの冬季ピーク（12〜2月）が稼働コストを押し上げた可能性

**【結論】2Q→3Q減益の構造的判定：**
> ✅ **季節性・売上変動による一時的減益（構造的悪化ではない）**
> - 前年同期比+24.9%増益という事実がこれを裏付ける
> - 2Qの¥2,166Mが異常高値（好条件重複）であり、3Qの¥1,395Mが「正常化」と解釈
> - むしろ3Qのコンセンサス¥950Mを大幅超過している事実が構造的改善を示唆

```

+ EVのダッシュボードは大型・高機能化（大画面統合）でプラスチック部品ニーズ拡大
+ バッテリーモジュール周辺のエンジニアリングプラスチック需要（難燃・耐熱）

- エンジン周辺プラスチック部品（インテーク・カバー等）は需要消滅

ネット評価：▲プラス効果がマイナス効果を上回る（ネットポジティブ）
