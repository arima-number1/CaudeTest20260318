### 3-3. ROE-PBR整合性検証（デュポン簡易）

```
Gordon Growth Model による適正PBR検証:
  適正PBR = ROE / (Re - g)
  ROE = 11.3%（Yahoo Finance確定値）
  Re（株主資本コスト） = 7.4〜8.2%（後述WACC計算）
  g（持続成長率） = 3.0〜4.0%

  現在PBR = 2.57倍（Yahoo Finance確定値）

  → 現在PBR（2.57倍）< 適正PBR（2.69倍）
  → 依然として本源的価値を下回って取引されている
  → ¥613 × 2.69 ≈ ¥1,649（PBR適正水準）でも現在値¥1,576を上回る
```

### 4-1. 前提条件の設定

| **β（ベータ）** | **0.95** | 東宝の5年月次β（エンタメ/消費財セクター、景気感応度中程度） |
| **Re（株主資本コスト）** | **Re = 2.467% + 0.95 × 5.5% = 7.69%** | CAPM計算 |
| **Rd（負債コスト）** | **1.2%** | 東宝の低レバレッジ×優良格付け（AA相当推定） |
