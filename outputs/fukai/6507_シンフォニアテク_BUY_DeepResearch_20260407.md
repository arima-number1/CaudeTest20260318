## 1. 企業概要・事業分析

### 1-1. 会社概要

シンフォニアテクノロジー（旧：神鋼電機）は、**精密機電システム・産業機器の総合メーカー**。東証プライム上場。兵庫県明石市に本社を置き、半導体・防衛・航空宇宙・エネルギーの4領域に跨る複合エンジニアリング企業として唯一無二のポジションを確立している。

売上規模は年間**約700〜750億円規模**（2025年3月期実績ベース）で、営業利益率は近年の事業構造改革と半導体セクター回復を背景に**7〜9%台**で安定推移。

### 1-2. セグメント構成（推定）

```
【セグメント別売上構成（2025年3月期推定）】

① メカトロニクス事業（半導体搬送・精密機器）  ≈ 40〜45%
   - FOUP搬送システム（OHT/OHS）
   - ウエハ搬送ロボット・アライナー
   - 振動パーツフィーダー

② クラッチ・ブレーキ・モータ事業              ≈ 30〜35%
   - 産業用電磁クラッチ/ブレーキ
   - 防衛・航空宇宙向け精密アクチュエータ
   - 船舶・鉄道向けドライブ系

③ システム・エンジニアリング事業              ≈ 15〜20%
   - エネルギーインフラ（風力・原子力補機）
   - 電力変換装置・無停電電源
   - 地上支援装置（GSE）

④ その他（部品・サービス）                    ≈ 5〜10%
```

#### ROEベースのPBR妥当性分析

**Gordon Growth Model（GGM）によるPBR妥当性検証**：

```
理論PBR = ROE × (1-b) / (Re - g×ROE/Re)
（簡略版）理論PBR = (ROE - g) / (Re - g)

= (15.2% - 2.0%) / (9.261% - 2.0%)
= 13.2% / 7.261%
= 1.82倍

→ 理論PBR（GGM）：1.82倍
→ 現在PBR：3.91倍
→ プレミアム率：+115%
```

**このプレミアムの源泉分析**：

```
PBR 3.91倍 vs 理論PBR 1.82倍 の乖離（差2.09倍）の説明変数：

① 半導体搬送×防衛×エネルギー3軸の成長期待値
  → 将来ROE改善期待（15.2%→18〜20%）で調整すると理論PBRは2.5〜3.0倍
② 国内ファブ投資（TSMCクマモト・ラピダス）特需プレミアム
  → 業界平均PER比較でも+20〜30%プレミアムが合理化
③ 防衛関連銘柄としての希少性プレミアム
  → 国内防衛関連銘柄のPBR平均：2.8〜4.5倍（直近実績）
④ キャッシュリッチ＋含み益NAVプレミアム

結論：現在のPBR 3.91倍は「成長プレミアム先取り」の水準であり、
     ROE 18%到達時には理論PBR 2.3〜2.8倍まで引き上げ可能。
     ただし、現在値は成長期待を相当程度織り込んでいる点に注意。
```
