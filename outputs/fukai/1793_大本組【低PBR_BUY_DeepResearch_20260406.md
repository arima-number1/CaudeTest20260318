## 2. 【社長指示最優先】高自己資本比率×低ROEのギャップ解明とPBR改善蓋然性

> 社長指示：「**高自己資本比率の実態と低ROEのギャップを解明。PBR改善に向けた経営改革の蓋然性を検証**」

### 2-1. デュポン分解：ROE 3.3%の構造分解

ROEをデュポン式で分解することで、低収益の根本原因を特定する。

```
ROE = 純利益率 × 総資産回転率 × 財務レバレッジ

ROE = (純利益/売上) × (売上/総資産) × (総資産/自己資本)
```

**逆算による推計（提供データ固定値使用）：**

- EPS = ¥50 / BPS = ¥1,405
- ROE = EPS / BPS = 50 / 1,405 = **3.56%**（≈公式値3.3%と整合）
- PER = 33.6倍 → 時価総額 = 株価 × 発行株数で推計

| デュポン要素 | 推計値 | 業界平均比較 |
|------------|--------|------------|
| 純利益率（ROS） | 推定2〜4% | 中堅平均：3〜5% |
| 総資産回転率 | 推定0.6〜0.8回 | 中堅平均：0.8〜1.0回 |
| 財務レバレッジ | 推定1.8〜2.2倍 | 大手：2.5〜3.0倍 |

**診断結果：ROE 3.3%の主因**

```
① 財務レバレッジが低い（高自己資本比率の裏返し）
   → 自己資本比率40〜50%超であれば、レバレッジは約2倍前後
   → 大手ゼネコンは借入を積極活用し、レバレッジ2.5〜3倍でROEを嵩上げ
   
② 総資産回転率の低さ
   → 現預金・有価証券等の非稼働資産を大量保有
   → 投資効率が低い
   
③ 純利益率も中位
   → 建設業特有のコスト上昇（資材・労務）の転嫁遅延
```

### 2-2. 高自己資本比率の「実態」精査

**高自己資本比率＝美徳か、非効率か？**

建設業（特に地方中堅）が高自己資本比率を維持する背景には複数の要因がある：

| 要因 | 詳細 | 評価 |
|------|------|------|
| 公共工事入札資格の維持 | 自己資本比率・経営事項審査（経審）スコアが受注力に直結 | **合理的理由あり** |
| 繁閑サイクルへの備え | 建設受注は景気・政策連動で変動大。財務バッファが必要 | **部分的に合理的** |
| 経営者の保守的志向 | オーナー系・同族系企業に多い傾向。株主還元より内部留保優先 | **改善余地あり** |
| 非稼働資産の蓄積 | 創業以来の土地・有価証券・現預金が貸借対照表に滞留 | **株主価値毀損要因** |

**問題の本質：**
大本組の高自己資本比率は「健全な財務体質」と「資本の非効率利用」という二面性を持つ。経審評点維持に必要な最低限を超えた余剰資本が、ROEを構造的に押し下げている可能性が高い。

### 2-3. ROEとWACCの関係：価値創造か価値破壊か

| 指標 | 数値 | 判定 |
|------|------|------|
| ROE | 3.3% | 実績値 |
| 株主資本コスト（Re）推定 | 7.0〜9.0%（後述WACC参照） | 要求収益率 |
| スプレッド（ROE - Re） | **▲3.7〜5.7%** | ⚠️ **価値破壊ゾーン** |

**重要結論：** ROE（3.3%）< 株主資本コスト（推定7〜9%）であり、現在の大本組は**理論上、株主価値を毀損し続けている状態**。PBR 1.19倍はこれを一部織り込んでいるが、完全に修正されていない（PBR<1.0倍ではない）のは、純資産の帳簿価値に対する市場の一定の信頼を反映。

### 2-4. PBR改善シナリオ：経営改革の蓋然性検証

**理論上のPBR改善メカニズム：**

```
PBR = ROE × PER（理論的関係）
    = ROE / (Re - g) × （調整係数）

現在：PBR 1.19 = ROE 3.3% × PER 33.6倍（市場が将来改善を一部期待）
```

**ROE改善で何が変わるか：**

| ROE改善シナリオ | 想定PBR（PER 20倍前提） | 理論株価（BPS×PBR） | 現在値比 |
|--------------|----------------------|-------------------|---------|
| ROE 3.3%（現状） | 約0.66倍（理論）→実勢1.19 | ¥928（理論） | ▲44% |
| ROE 5.0% | 約1.0倍 | ¥1,405 | ▲16% |
| ROE 7.0% | 約1.4倍 | ¥1,967 | +18% |
| ROE 10.0% | 約2.0倍 | ¥2,810 | +69% |

※ PER仮定は建設業界平均参考。現状PER 33.6倍は市場の改善期待を反映した高水準。

**経営改革の具体的手段と蓋然性評価：**

| 改革手段 | 効果 | 蓋然性 | 理由 |
|---------|------|--------|------|
| 自社株買い | 自己資本圧縮→ROE上昇 + EPS向上 | **中〜高** | 低PBR対策として近年中堅ゼネコンで増加。東証の改善要請も追い風 |
| 増配 | 資本効率改善（余剰現金の還元） | **中** | 配当性向の引き上げ余地あり（EPS¥50 vs 配当水準要確認） |
| 非稼働資産売却 | BS圧縮・ROE改善 | **低〜中** | 土地・有価証券の売却は経営判断に委ねられ時間軸不透明 |
| 収益性改善（利益率向上） | 純利益率↑→ROE↑ | **中** | 防衛・インフラ需要増で受注増加・採算改善の余地あり |
| M&A・成長投資 | 売上拡大・規模の経済 | **低** | 中堅ゼネコンの積極M&Aは稀 |

**蓋然性の総合評価：**
東証プライム市場のPBR改善要請（2023〜）により、大本組もPBR1.0倍超への取り組みを迫られる立場にある。自社株買い・増配という「低リスク・高効果」な手段の蓋然性は**中〜高**と判断する。ただし経営陣の株主還元姿勢・具体的な数値目標の公表が確認されるまでは、確信度は中程度に留める。

## 4. DCFバリュエーション

### 4-1. WACC計算

**ステップ1：株主資本コスト（Re）**

```
Re = Rf + β × ERP

Rf = 2.380%（JGB10Y・社長確認値）
β  = 0.70（建設業中堅・推定。インフラ受注依存で景気感応度はやや低め）
ERP = 6.0%（日本市場標準・Damodaran参考）

Re = 2.380% + 0.70 × 6.0%
   = 2.380% + 4.200%
   = 6.58%
```

**ステップ2：負債コスト（Rd）**

```
Rd（税引後） = 推定1.5% × (1 - 30%) = 1.05%
※ 高自己資本比率・低借入体質のため負債コストは低水準
税率 Tc = 30%（実効税率）
```

**ステップ3：資本構成**

```
自己資本比率（E/V）= 70%（高自己資本比率を反映）
負債比率（D/V）  = 30%
```

**ステップ4：WACC算出**

```
WACC = E/V × Re + D/V × Rd × (1-Tc)
     = 0.70 × 6.58% + 0.30 × 1.05%
     = 4.606% + 0.315%
     = 4.92%（≈ 4.9%）
```

### 4-4. ハイブリッド理論株価（DCF×PBR複合）

**理論株価算出フレームワーク：**

```
理論株価 = α × DCF理論価値 + (1-α) × 純資産ベース価値

建設業（資産保有型）はPBRアプローチのウエイトを高める
α = 0.40（DCF） / (1-α) = 0.60（純資産）
```

| アプローチ | 計算 | 価値 |
|-----------|------|------|
| DCF（収益）アプローチ | EPS/Re-g = ¥50/(6.08%) | ¥823 |
| 純資産アプローチ（BPS） | BPS × 適正PBR | ¥1,405 × 1.0 = ¥1,405 |
| ハイブリッド（4:6） | 0.40×¥823 + 0.60×¥1,405 | **¥329 + ¥843 = ¥1,172** |

**安全域（Margin of Safety）考慮：**

```
理論株価レンジ：¥1,100〜¥1,650
中央値：¥1,375

安全域（15%バッファ）：¥1,375 × 0.85 = ¥1,169（買い推奨水準）
現在株価 ¥1,665 は理論株価レンジの上限付近
→ 現在株価に安全域はほぼ存在しない
```
