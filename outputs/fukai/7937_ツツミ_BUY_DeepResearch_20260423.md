## 🚨 新TOPIX除外リスク（最重要ネガ）

### 試算

| 指標 | 数値 |
|------|------|
| TOPIXウェイト | **0.0066%** |
| TOPIX連動ファンド推定保有額 | **¥36.2億** |
| ADV（日次売買代金） | **¥0.5億/日** |
| **除外時の売り圧力** | **¥36億 ÷ ¥0.5億/日 = ADVの71日分** |

### 新TOPIX基準チェック

```
新TOPIX採用基準: 流通株式時価総額100億円以上
時価総額: ¥474億
流通率（創業家保有比率次第）:
  創業家60%保有 → 流通40% × ¥474億 = ¥190億 → OK
  創業家75%保有 → 流通25% × ¥474億 = ¥119億 → OK
  創業家80%保有 → 流通20% × ¥474億 = ¥95億 → ⚠️ 除外リスク
```

**→ 創業家保有比率が80%超の場合、流通時価総額100億円割れで除外対象。**
**除外が確定すれば ADVの71日分の売り圧力 = 株価に壊滅的。**

### MBOを後押しする4要因

1. **純資産＞時価総額**: 企業価値が完全に割り引かれており、取得後即座に純資産が回収可能水準
2. **新TOPIX除外圧力**: 上場継続コスト（IR費用・開示義務）のメリットが薄れる
3. **ROE3.9%の低さ**: 上場維持のプレッシャー（PBR1.0x以上）を達成できず。非上場化で解放
4. **高配当性向67%**: 上場維持のために高配当を維持する必要があるが、非上場化で内部留保可能

### MBO確率の定性評価

| 要因 | 評価 |
