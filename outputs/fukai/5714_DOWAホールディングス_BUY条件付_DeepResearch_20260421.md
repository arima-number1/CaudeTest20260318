## 🐂 投資論拠（Bull Case）

### 1. 商品価格トリプル高：製錬利益の爆発的回復
**亜鉛¥62.8万/MT（2022年4月来高値）**：
- みずほの感応度：亜鉛+$100/MT → OP+¥250M
- 現在のLME亜鉛 ≈ $3,900/MT（建値¥62.8万/¥160換算）
- 会社前提（推定）：$2,800/MT → 差分+$1,100/MT → OP+¥2.75B/年の超過利益
- **Rationale**：ホルムズ再封鎖による精錬用硫酸原料（硫黄）不足 + 中国需要回復 + 減産鉱山

**銅¥221万/MT（史上最高値）**：
- DOWAは銅精錬も実施（小坂製錬100%子会社）
- 銅+$100/MT感応度 ≈ OP+¥150〜200M（亜鉛比やや小）
- 史上最高値水準継続で年間+¥3〜4B超過利益

**銀$35超/oz（ホルムズ工業需要 + 金連動）**：
- DOWAは国内最大の銀精錬企業（年間生産量約400トン、国内シェア≈30%）
- FY26に銀急騰でヘッジ損失が発生済み → **FY27はヘッジ再設計後にクリーンな上振れ**
- $35/oz（会社前提≈$25想定）→ +$10 × 400T × 32,150oz/T = +$128.6M ≈ +¥20.6B（粗収益ベース）

### 2. Fujita Kanko株売却 → 株主還元 + B/S強化
- 売却額：¥38.9B（2025年秋完了）
- 特別利益：¥23B（FY3/26計上済み）
- 特配：¥100/株（FY3/26末一括）
- 自社株買い：¥10B（58.55M株 × ¥1,710相当を消却 = 発行済みの2.9%）
- **合計株主還元：¥100特配 + 通常配¥218 ≈ ¥318/株（FY3/26）= 利回り3.25%**

### 3. 政府重要鉱物1兆円投資計画（2026/4/20発表方向）
- 対象：LiB・希少金属・貴金属のリサイクル国内化
- DOWAの該当事業：**LiB電池リサイクル拡張中**（小坂製錬）、貴金属（金・銀・PGM）回収
- 補助金規模感：総額¥1T/5年 × 民間CAPEX補助率30% = ¥60B/年の業界全体補助
- DOWA受益分推計：業界シェア推定10% × ¥60B = **¥6B/年の設備補助＝実質CAPEX▲6B**
- EPS効果：税後¥4.2B / 58.55M株 = **+¥72/株/年**
- **Rationale**：DOWAはLiB回収技術（湿式製錬）で国内最先端。小坂製錬が実証設備稼働中。政府「経済安保」観点でのサプライチェーン国内化で最優先受益企業候補

### 4. SMBC Nikko アップグレード：FY3/27 EPS¥987の現実性
- SMBC Nikko：Outperform TP¥11,500（FY3/27 Adj.EPS ¥987.2 × SOTP P/E 11.6x）
- ¥987.2はコンセンサス¥624と大きく乖離するが、根拠：
  - 非鉄セグメント+$PGM高騰特需
  - Fujita Kanko売却後B/S強化による財務コスト低下
  - LiB recycling量産効果（FY27H2から本格化）
- 確率30%で¥11,204のFV → 現値¥9,792でも20%の上昇余地

## 🐻 投資リスク（Bear Case）

### リスク①：PCB処理ビジネス終了（最大リスク、2027年3月確定）
- PCB廃棄物特別措置法の廃棄期限：2027年3月末（延長不可）
- DOWAのPCB収益：年間OP ¥10〜15B（推定）= 全社OP比20〜30%
- 終了後のLiBリサイクル代替：OPM 3〜5% vs PCBの15〜20% → **利益2/3喪失**
- 定量影響：FY3/28 OP ▲¥10〜15B = EPS ▲¥120〜180/株

### リスク②：電子材料セグメント赤字継続
- 銀粉末（太陽電池向け）：中国メーカーのダンピング継続（価格▲40〜50%）
- FY26E電子材料 OP：▲¥4.6B（みずほ推計）
- 回復見通し：FY27以降も低一桁利益率が続く見込み
- ヘッジ損失：銀価格急騰で既にFY26 H2に▲¥5B超のデリバティブ損発生

### リスク③：¥100B CAPEX計画（FY3/28〜）
- 老朽化設備更新（小坂製錬100年超の設備）
- 単独での投資効果不明 → 機関投資家の懸念あり
- SMBC Nikko：「投資家の不満足な反応リスク」を明記

### リスク④：銀価格急騰によるヘッジ損拡大
- DOWAは銀生産者として価格上昇時に**ショートヘッジ**を保有
- 銀$40超継続：ヘッジ損が更に拡大するリスク
- FY27: 残存ヘッジ分の評価損が判定を左右

### リスク⑤：電力コスト（製錬の直接コスト）
- 小坂製錬は年間電力消費量業界最大クラス
- 電力単価+10% → OP▲¥2〜3B（推定）
- ホルムズ継続でLNG高 → 電力高止まり
