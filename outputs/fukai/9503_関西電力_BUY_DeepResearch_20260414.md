### 2-2. 決算前買いの根拠（4軸分析）

#### ① バリュエーション軸：絶対割安

```
現在値:    ¥2,524
BPS:       ¥3,032  → PBR 0.83倍（純資産以下で買える）
EPS:       ¥389    → PER 6.5倍（電力セクター平均9〜11倍比▲40%割安）
配当利回り: 約3.4%（推計¥86/株÷¥2,524）
```

> **PBR0.83倍は「純資産の83%で会社を買える」**ということ。電力会社のROE12.4%を維持できるなら、理論的にはPBR1.0倍以上に戻るべき水準。現在は明確な割安域。

#### ② 収益モメンタム軸：追い風3重構造

```
【2026年5月決算で確認されるはずの3つのポジティブ】
A. 原子力稼働率向上 → 燃料費節約効果　▲約2,000億円/年
B. LNG価格低下（2.60$/MMBtu）→ 火力燃料費削減　▲約800億円/年
C. 燃料費調整制度逆ザヤ解消 → 収益正常化　+約1,500億円
                              ━━━━━━━━━━━━━━━━
                合計利益押し上げ効果    +約4,300億円/年ピーク
```

> これだけの追い風が同時進行している。決算数字が「ネガティブサプライズになるシナリオ」は相当限定的。

#### ③ 地政学・商品価格軸：ホルムズリスクの織り込み

**WTI原油97.32$/bbl・ブレント98.60$/bbl** という足元の高水準は、中東地政学緊張を部分的に反映している。

```
【関西電力のホルムズ感応度】
LNG調達量:    約1,500万トン/年（国内電力会社トップクラス）
中東依存比率:  約40%（カタール・UAE等）
代替調達先:    米国（ヘンリーハブ連動）、オーストラリア、マレーシア

ホルムズ封鎖（最悪ケース）時の追加調達コスト：
  スポットLNG価格急騰+20$/MMBtu想定
  → 追加コスト概算: 1,500万t × 1.03t/MMBtu × 20$/MMBtu = 約2,900億円/年
  → 現在のEPS(¥389)を▲¥350程度毀損するリスク（実質赤字転落）
```

**ただし現在のHH天然ガス2.60$/MMBtu**は封鎖顕在化を織り込んでおらず、スポットLNG市場も沈静化状態。現時点では**テールリスク**として監視対象だが、投資判断を変えるほどの即時リスクではない。

> **社長へ：「ホルムズは最悪ケースでEPS消滅リスクがあるが、現在の市場価格はその可能性を低く見ている。原子力6基稼働がLNGリスクの最大のヘッジ。」**

## 4. DCFバリュエーション

### 4-1. WACCの計算

#### インプット前提

| パラメータ | 値 | 根拠・出典 |
|-----------|---|-----------|
| **Rf（リスクフリーレート）** | **2.467%** | 日本10年国債（JGB10Y）2026-04-14確定値 |
| **β（ベータ）** | **0.62** | 電力セクター特性（規制業種・ディフェンシブ）、東電・中電等との比較 |
| **ERP（エクイティリスクプレミアム）** | **5.5%** | 日本株市場標準（Damodaran 2026推計、JPモルガン日本ERP参照） |
| **Re（株主資本コスト）** | **5.877%** | Re = 2.467% + 0.62 × 5.5% = 2.467% + 3.41% = **5.877%** |
| **Rd（負債コスト）** | **1.8%** | 関西電力社債利回り（2025年度発行実績ベース） |
| **Tc（実効税率）** | **30.6%** | 日本法人税・住民税・事業税合算 |
| **E/（D+E）（株式比重）** | **44.7%** | 時価総額÷（時価総額+有利子負債）＝1.88兆÷（1.88兆+2.8兆+0.51兆EV調整） |
| **D/（D+E）（負債比重）** | **55.3%** | 上記の残余 |

#### WACC計算

```
Re = 2.467% + 0.62 × 5.5% = 5.877%

WACC = [0.447 × 5.877%] + [0.553 × 1.8% × (1 - 0.306)]
     = [2.627%] + [0.553 × 1.249%]
     = [2.627%] + [0.691%]
     = 3.318%
     ≒ 3.32%
```

> ⚠️ 電力会社は有利子負債比重が高く、低コスト負債がWACCを引き下げる。計算値3.32%はインフラ系DCFとして整合的。
